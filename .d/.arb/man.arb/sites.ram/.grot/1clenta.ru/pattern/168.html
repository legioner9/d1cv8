Переход с бп 2.0 на 3.0
Настройка системы     23.05.2017 09:37     7915

Конфигурация "Бухгалтерия предприятия 2.0" работает на обычных формах, а "Бухгалтерия предприятия 3.0" работает на управляемых формах. Переход с конфигурации БП 2.0 на 3.0 выполняется обычным обновлением. Обновление выполняется либо в 1С:Предприятии либо в 1С:Конфигураторе. Перед любым обновлением необходимо делать бэкап информационной базы. В случаи сбоя при обновлении (отключение света, технический сбой, и т.д.) можно будет восстановить базу из бэкапа.

1) Обновление БП 2.0 в режиме 1С:Предприятие


Для запуска перехода с версии 2.0 на 3.0 нужно перейти в главном меню: Сервис->Обновление на редакцию 3.0.

Здесь предлагается выбрать откуда будет загружено обновление. При выборе "Пользовательский сайт в Интернете (рекомендуется)" обновление будет загружено через Интернет с сервера 1С. Для подключения к серверу потребуется ввести логин и пароль. При выборе "Локальный или сетевой каталог" загрузка обновления будет выполнена из каталога на компьютере.



В данном окне содержится информация о доступном на сервере обновлении.


Для подключения к серверу 1С требуется ввести логин и пароль выданные при покупке ИТС.


При выборе "Да, перезагрузить "1С:предприятие" сейчас (рекомендуется)" немедленно будет выполнен перезапуск информационной базы и начнется обновление.


После перезапуска информационной базы для обновления потребуется некоторое время. На скорость обновления влияет объем информационной базы, параметры компьютера на котором выполняется обновление, в случаи с серверной базой учитываются параметры сервера.


2) Обновление БП 2.0 в режиме 1С:Конфигуратор


Для обновления через конфигуратор потребуется дистрибутив обновления, который можно скачать с официального сайта 1С. Релиз обновления нужно скачать в зависимости от используемой конфигурации: "Бухгалтерия предприятия, редакция 3.0", "Бухгалтерия предприятия КОРП, редакция 3.0", "Бухгалтерия предприятия базовая, редакция 3.0".





После того как дистрибутив будет скачан и установлено обновление можно будет приступить непосредственно к обновлению БП 2.0 на 3.0. Процесс обновления из конфигуратора запускается через главное меню: Отладка->Поддержка->Обновить конфигурацию.

Поиск доступных обновлений.


Так как обновление было установлено ранее, то нужно выбрать "Икать в текущих каталогах шаблонов и обновлений".


Помощник предложит установить обновление для переходя на 3 версию бухгалтерии.


В информационном окне можно увидеть с какой версии на какую будет произведено автоматическое обновление конфигурации.
