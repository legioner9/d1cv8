Выгрузить, загрузить информационную базу из dt файла
Настройка системы     23.07.2018 08:51     4646

В 1С Предприятии информационную базу можно выгрузить в файл или загрузить из файла с расширением (*.dt). Данный метод позволяет организовать процесс резервного копирования. Кроме того, с помощью файла можно перенести базу, например, на другой компьютер. Файл dt содержит данные и конфигурацию информационной базы. Если данные не требуется выгружать, то можно выполнить выгрузку только конфигурации.

Выгрузка, загрузка 1С базы из dt файла


Действия по выгрузке 1С базы в файл или загрузке из файла доступны только через 1С Конфигуратор. В главном меню нужно перейти: Администрирование->Выгрузить информационную базу. Для того чтобы загрузить файл: Администрирование->Загрузить информационную базу. Если база большого объёма, то может потребоваться значительное время для выгрузки или загрузки файла.
