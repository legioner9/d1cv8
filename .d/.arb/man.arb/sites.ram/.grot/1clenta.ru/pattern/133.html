Обработка "Поиск и замена значений"
Настройка системы     16.02.2017 09:13     4944

Поиск и замена значений - это универсальная обработка для поиска и замены ссылочных значений во всей базе. Данная обработка встроена во многие типовые конфигурации. Это очень удобная обработка для решения задач связанных с поиском и заменой значений. Обработка не учитывает особенности конфигураций при замене значений, поэтому перед использованием рекомендуется сделать резервную копию информационной базы.

Запуск обработки


В обычном приложении в главном меню открыть: Операции->Обработка...->Поиск и замена значений.
В управляемом можно открыть либо через интерфейс, либо универсальным способом: Сервис->Все функции->Обработки->Поиск и замена значений.

Если встроенной обработки нет, то для обычного приложения её можно скачать с официального сайта или с диска ИТС из каталога: \1CITS\EXE\ExtReps\Unireps82\SearchAndChange\. Для управляемого приложения её можно взять из типовых конфигураций в которых она есть (сохранить как внешнюю). Запускается через главное меню: Файл->Открыть->ПоискИЗаменаЗначений.epf.

Интерфейс обработки


Перед началом замены нужно выбрать что на что будет заменено. По нажатию на кнопку "Найти ссылки" будут найдены все объекты в которых присутствует искомое значение. После чего можно выборочно отметить галками в каких объектах производить замену значений. По нажатию на кнопку "Выполнить замену значений" будет произведена замена значений в выбранных объектах.
