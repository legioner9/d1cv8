Коды валют (ISO 4217)
Настройка системы     26.08.2016 12:05     11771

Код валюты - это буквенный или цифровой код валюты. Данное обозначение используется для идентификации валют при международном обмене информацией, при совершении валютных сделок. Классификация кодов валют разработана Международной организацией по стандартизации (ISO). Согласно стандарту ISO 4217 национальной валюте страны присваивается трёх символьный буквенный код. Первые две буквы берутся из названия государства. Третья буква - это первая буква названия валюты. Так к примеру, AUD (Australian Dollar). Кроме буквенного кода валюте присваивается цифровой код.

Международный стандарт классификации кодов валют используют государства при создании собственных классификаторов. ОКВ (Общероссийский классификатор валют) - государственный классификатор иностранных валют, валютных ценностей, валютных фондов. Данный классификатор создан на основе мирового стандарта ISO 4217.
