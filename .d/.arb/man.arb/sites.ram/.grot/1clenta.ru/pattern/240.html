Активные пользователи информационной базы 1С
Настройка системы     21.11.2018 09:17     9810

Для просмотра и управления подключениями к информационной базе, в 1С Предприятиии 8 предусмотрены специальные возможности. Просмотреть активные подключения к базе 1С, можно в режиме Предприятия или через Конфигуратор. При наличии серверной версии информационной базы, управление сеансами пользователей доступно через "Консоль администрирования серверов 1С Предприятия".

1С Конфигуратор


Для работы с конфигуратором нужно запустить информационную базу в режиме 1С Конфигуратор.
Для администрирования активных подключений, в главном меню перейти Администрирование->Активные пользователи.
В списке отобразятся все подключения к информационной базе. В каждой строке указывается: пользователь, приложение, время начала работы, компьютер, сеанс.





1С Предприятие


Информационную базу нужно запустить в режиме 1С Предприятие. В данном примере используется информационная база с конфигурацией "Бухгалтерия предприятия 3.0". Для управления подключениями необходимо перейти Администрирование->Активные пользователи (расположение может отличаться в зависимости от настроек интерфейса программы). Так же обработку можно открыть через: Главное меню->Все функции->Стандартные->Активные пользователи.





Программное получение активных пользователей


С помощью специальных программных объектов можно получить активные соединения и сеансы пользователей информационной базы. Для этого предназначены глобальные методы ПолучитьСоединенияИнформационнойБазы(), ПолучитьСеансыИнформационнойБазы().
