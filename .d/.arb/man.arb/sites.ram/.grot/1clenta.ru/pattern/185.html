Как включить и актуализировать партионный учет в УТ 11
Настройка системы     05.10.2017 08:12     33302

Конфигурация "Управление торговлей 11" разработана фирмой 1С для ведения учета в торговле. Эта конфигурация сделана на управляемых формах и хорошо адаптирована для клиент-серверного варианта. УТ 11 имеет широкий набор возможностей для ведения учета на торговом предприятии. Одной из главных возможностей данной конфигурации, является адаптированность для ведения партионного учета товаров. Управление партионным учетом, в разных версиях УТ 11, отличается.

Управление торговлей 11.1


В конфигурации "Управление торговлей 11.1" партионный учет реализован, и включается в: Администрирование->Финансовый результат. После включения партионного учета документы движений товаров будут проведены по регистру накопления "Партии товаров организаций".



Для вновь созданных документов необходимо периодически проводить актуализацию партионного учета. Для этого нужно перейти в регистр накопления "Партии товаров организаций" и нажать на кнопку "Актуализировать партии". После этого в движениях документов появятся проводки по регистру "Партии товаров организаций".



Управление торговлей 11.3


В конфигурации "Управление торговлей 11.3" включение партионного учета отличается от предыдущей версии. Для включения нужно перейти: НСИ и администрирование->Финансовый результат и контроллинг->Учет товаров.



Для того чтобы сформировались проводки по регистрам партионного учета, надо запустить специальную обработку "Операции закрытия месяца". Данная обработка открывается из главного меню: Финансовый результат и контроллинг->Закрытие месяца или через меню Все функции->Обработки->Операции закрытия месяца.





Для формирования движений партионного учета нужно выбрать необходимый месяц в году и нажать "Сформировать" напротив "Формирование движений по партиям товаров". Через некоторое время обработка закончит работу и движения по регистру накопления "Партии товаров организаций" будут сформированы.
