При работе с типовыми конфигурациями бывает необходимо загрузить данные из файла. В обычном приложении для решения данной задачи есть специальная обработка "ЗагрузкаДанныхИзТабличногоДокумента.epf". Она позволяет автоматизировать процесс загрузки данных из файла. Для управляемого приложения так же существует специальная Обработка ЗагрузкаДанныхИзФайла.epf, из файлов с расширением (*.xlsx, *.xls, *.ods, *.csv, *.mxl) (управляемое приложение).

Загрузка данных из файла Excel в справочник "Причины списания ОС"


В информационной базе "Бухгалтерия предприятия 2.0" есть справочник "Причины списания ОС". Есть excel файл, который содержит список причин списания основных средств. Данные из excel файла необходимо загрузить в этот справочник. Обработка ЗагрузкаДанныхИзТабличногоДокумента.epf поставляется на дисках ИТС. Она открывается через главное меню: Файл->Открыть.

После открытия обработки, на форме доступен ряд настроек. В настройке "Режим загрузки" устанавливается тип заполняемого объекта: Загрузка в справочник, Загрузка в табличную часть, Загрузка в регистр сведений. В данном примере выбрана "Загрузка в справочник" и вид справочника "Причины списания ОС".

Новые элементы справочника будут создаваться из таблицы на вкладке "Табличный документ". Колонки таблицы соответствуют реквизитам указанным на вкладке "Настройка", строки - это добавляемые элементы. Поэтому сначала необходимо заполнить таблицу данными из файла. Загружать можно файлы с расширениями: *.mxl, *.xls, *.txt, *.dbf. В этом примере данные загружаются из excel файла с расширением (*.xls). По номерам колонок, происходит сопоставление данных файла и таблицы.



На вкладке "Настройка" можно задать опции загрузки данных из вкладки "Табличный документ".



После указания всех настроек, данные из файла загружаются в табличный документ.



Данные, помещенные в таблицу, можно подредактировать. После чего, нажав на кнопку "Загрузить", в справочнике "Причины списания ОС" будут созданы элементы.
