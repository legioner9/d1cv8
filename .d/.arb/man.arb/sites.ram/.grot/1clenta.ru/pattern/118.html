Как скачать КЛАДР (адрессный классификатор)
Настройка системы     21.01.2017 11:52     4930

КЛАДР - это классификатор адресов Российской Федерации. Аббревиатура расшифровывается как: КЛ-классификатор, АД-адресов, Р-России. Данный классификатор формируется на основе данных ФИАС (Федеральная Информационная Адресная Система).
Адресные классификаторы широко используются в базах 1С:Предприятия. В типовых базах процесс загрузки адресного классификатора автоматизирован. В различных конфигурациях процесс загрузки может незначительно отличаться. В некоторых конфигурациях классификатор можно загрузить прямо с сайта ИТС. Для этого нужна подписка на ИТС.

Диск ИТС


При наличии подписки на ИТС, файл классификатора поставляется на диске ИТС. Если диск устарел, то и версия классификатора будет не актульной.



Скачать КЛАД с сайт www.gnivc.ru


Свежую версию базы можно скачать на официальном сайте (www.gnivc.ru) компании АО "ГНИВЦ".



Скачать КЛАДР, ФИАС с сайта fias.nalog.ru


Более широкий выбор форматов файлов классификаторов доступен на официальном сайте (fias.nalog.ru) ФНС России.



Скачать ФИАС с сайта ИТС


Для некоторых конфигураций КЛАДР не подходит, им требуется ФИАС. Например для "Бухгалтерия предприятия 3.0" требуется классификатор в формате ФИАС. Скачать ФИАС можно с сайта ИТС.



Скачать КЛАДР с сайта ИТС


На сайте ИТС можно скачать КЛАДР.
