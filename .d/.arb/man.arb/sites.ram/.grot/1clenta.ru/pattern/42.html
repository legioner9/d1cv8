1С демо версии прикладных решений
О системе 1С     06.06.2016 13:36     3649

Для ознакомления с типовыми решениями, созданными на базе платформы 1С Предприятие 8, компания 1С предоставляет доступ к демо версиям. Для демонстрационного запуска доступны: "Зарплата и управление персоналом КОРП", "Управление небольшой фирмой", "1C:Документооборот 8", и другие. С полным списком, доступных типовых решений для запуска в демонстрационном режиме, можно ознакомиться на сайте demo.1c.ru. Запустить демонстрационные типовые решения можно через браузер или через 1С Предприятие, установленное на персональном компьютере.

Запуск демо версии через браузер


Для запуска демо версии необходимо перейти по ссылке на странице http://demo.1c.ru/. В браузере должны быть включены Cookies и JavaScript. Компания 1С рекомендует запускать в следующих браузерах.

  • Safari версии 4.0.5 для MacOS X версии 10.5 и Microsoft Windows
  • Google Chrome версии 4 для Microsoft Windows
  • Mozilla Firefox версии 17 или более поздней для Microsoft Windows и Linux
  • Microsoft Internet Explorer 8.0, 9.0, 10, 11 для Microsoft Windows. Рекомендуется использовать версию 9.0 или выше


Запуск демо версии через 1С Предприятие


На компьютере должно быть установлено 1С Предприятие 8.Версии платформ на веб-сервере и на локальном компьютере должны совпадать. Необходимо в список информационных баз добавить существующую базу расположенную на веб-сервере.

В окне запуска нажимаем кнопку "Добавить".



Выбираем флажок "Добавление существующей информационной базы". Затем перейти по кнопке "Далее".



Указываем имя информационной базы(на ваше усмотрение). Флафок "Выберите тип расположения информационной базы" выбрать "На веб-сервере". Затем перейти по кнопке "Далее".



В строке адреса информационной базы указывается путь к базе на сервере. Флажок "Выберите вариант использования прокси сервера" выбрать "Не использовать прокси". Флажок "Выберите способ аутентификации пользователя веб-сервера" выбрать "Выбирать автоматически". Затем перейти по кнопке "Далее".



Флажок "Вариант аунтефикации" выбрать "Выбирать автоматически". Параметр "Скорость соединения" установить в зависимости от пропускной способности интернет-соединения. Флажок "Оснонвой режим запуска" выбрать "Выбирать автоматически". Затем перейти по кнопке "Далее".



В списке информационных баз появится добавленная. Нажать кнопку "1С:Предприятие".
