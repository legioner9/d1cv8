Обработка "Универсальный обмен данными XML"
Настройка системы     09.04.2017 11:23     6227

При работе с 1С может возникнуть необходимость переноса данных между различными информационными базами. Для решения данной задачи можно написать обработку, которая возьмет данный из одной базы и запишит в другую или воспользоваться обработкой "Универсальный обмен данными XML". Эта обработка работает только с помощью правил обмена. Для создания этих правил предназначена специальная конфигурация "Конвертация данных".



Данная обработка поставляется в типовых конфигурациях и доступна через главное меню.
В обычном приложении: Операции->Обработка->Универсальный обмен данными XML.
В управляемом приложении: Все функции->Обработки->Универсальный обмен данными XML.
Так же её можно скачать с сайта ИТС или диска ИТС. Для скачивания необходима платная подписка.
