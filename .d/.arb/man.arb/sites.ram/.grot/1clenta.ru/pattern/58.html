Очистить КЭШ 1С
Настройка системы     21.07.2016 18:33     6142

При запуске информационной базы 1С могут появляться ошибки. Такие ошибки могут быть связаны с ошибками при работе с КЭШем 1С.
КЭШ (cache) - это буфер для хранения часто используемых данных. Объем такого хранилища имеет ограниченный размер. Поэтому в КЭШе хранится информация небольшого объема. Главным преимуществом КЭШа, является быстрота доступа к данным.

1C Предприятие как и многие другие программные продукты, для увеличения производительности использует КЭШ. Со временем при использовании кэшируемой информации могут появится ошибки. Для решения проблем можно очистить КЭШ 1С. Результата можно добиться несколькими способами.

1) удалить файлы КЭШа


Папки с файлами КЭШа можно найти в определенных директориях в зависимости от операционной системы. В зависимости от версии платформы 1С могут быть директории с именами 1Cv8 или 1Cv82.

В WINDOWS 7:
C:\Users\Администратор\AppData\Local\1C\1Cv8
C:\Users\Администратор\AppData\Roaming\1C\1Cv8

В WINDOWS XP:
C:\Users\Администратор\Local Settings\Application Data\1C\1Cv8
C:\Users\Администратор\Application Data\1C\1Cv8

Отличить папки КЭШа можно по названиям:
4bdfd5c9-cca2-4f77-9afa-2b476e835ada
31faa8b7-1a58-4312-9995-c0b41d10bcf2
d346aa8a-30a6-4faa-994e-afa4ffedbe95



2) выполнить команду /ClearCache


Данная команда прописывается в свойствах информационной базы, в поле "Дополнительные параметры запуска". В этом случае очистка КЭШа будет выполняться при каждом запуске информационной базы.
