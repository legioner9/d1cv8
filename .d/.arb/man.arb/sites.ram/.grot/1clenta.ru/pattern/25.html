Добавить внешнюю печатную форму в управляемое приложение для конфигурации Бухгалтерия государственного учреждения 2.0
Настройка системы     12.05.2016 16:46     4039

В типовых конфигурациях возможно добавление внешних печатных форм для справочников и документов. Процесс добавления похож во многих типовых конфигурациях на управляемом приложении. В данном примере рассматривается добавление внешней печатной формы в информационную базу типовой конфигурации "Бухгалтерия государственного учреждения 2.0".

Для добавления внешней печатной формы необходимо открыть Администрирование->Печатные формы, отчеты и обработки. Отметить галочку "Дополнительные отчеты и обработки" и перейти по ссылке "Дополнительные отчеты и обработки".



Откроется список "Дополнительные отчёты и обработки". В этот список необходимо добавить внешнюю печатную форму.



Откроется форма добавления нового элемента и форма выбора файла внешней обработки. Для выбора доступны файлы с расширением (*.erf, *.epf). Файл внешней обработки(печатной формы) с расширением (*.epf).



После выбора файла с расширением (*.epf) форма заполнится данными печатной формы.



После записи нового элемента в списке "Дополнительные отчёты и обработки" появится добавленная печатная форма.



Печатная форма в примере подключена к документу "Авансовый отчет". Если открыть документ, то в меню "Печать" появится кнопка запуска добавленной формы.



Внешние печатные формы для управляемого приложения имеют определенную структуру. Существуют учебные статьи для того чтобы самостоятельно сделать внешнюю печатную форму для управляемого приложения.
