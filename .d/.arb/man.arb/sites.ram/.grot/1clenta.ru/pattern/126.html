В типовых базах есть готовые функции для формирования списка периодов.
В бухгалтерии предприятия 2.0 - функции находятся в общем модуле РаботаСДиалогами.

1) ПериодРегистрации - создать реквизит в который будет помещаться выбранное значение
2) Период- на форме создать элемент "Поле ввода" с типом "Строка"

3) У периода определить событие "НачалоВыбораИзСписка"
Процедура ПериодНачалоВыбораИзСписка(Элемент, СтандартнаяОбработка)

	РаботаСДиалогами.НачалоВыбораИзСпискаПредставленияПериодаРегистрации(Элемент, 
		СтандартнаяОбработка, ПериодРегистрации, ЭтаФорма);
	
КонецПроцедуры

4) При открытии формы заполнить "Период"
Процедура ПриОткрытии()

	ПериодРегистрации = ТекущаяДата();
	Период = РаботаСДиалогами.ДатаКакМесяцПредставление(ПериодРегистрации);

КонецПроцедуры
