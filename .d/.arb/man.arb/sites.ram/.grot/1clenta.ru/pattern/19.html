Добавить обработку заполнения табличной части в обычное приложение для конфигурации Бухгалтерия государственного учреждения 1.0
Настройка системы     06.05.2016 17:43     4166

При работе с типовыми конфигурациями может возникнуть необходимость автоматизировать процесс заполнения таблиц в документах или справочниках. В типовых конфигурациях есть возможность подключать обработки заполнения табличных частей. Рассмотрим пример подключения на примере типовой конфигурации "Бухгалтерия государственного учреждения 1.0" (БГУ 1.0).

Обработки заполнения табличной части добавляется в режиме 1С:Предприятие. Для этого необходимо перейти Сервис->Дополнительные отчеты и обработки->Дополнительные внешние обработки табличных частей.



Откроется список внешних обработок заполнения табличных частей. Для добавления новой обработки необходимо нажать кнопку "Добавить новый элемент".



Откроется окно добавления нового элемента. Нужно выбрать файл внешней обработки. Для этого нажимаем на пиктограмму с изображением папки.



Откроется окно выбора файла внешней обработки заполнения табличной части. Для выбора доступны файлы с расширением (*.epf)



После выбора файла заполнится поле "Наименование". Если в обработке есть макет "Параметры_Авторегистрации", то появится окно с вопросом:"Указанная внешняя печатная форма содержит параметры авторегистрации. Использовать их при регистрации?". Обработка содержит дополнительную информацию, которая(имя документа, имя табличной части) может быть выведена на форму.



Не все документы и табличные части поддерживают подключение обработок. Наименование документа или справочника и табличной части устанавливается в таблице "Принадлежность обработки по заполнению табличных частей".



Выбрать табличную часть для которой предназначена внешняя обработка.



После добавления внешней обработки заполнения табличной части она появится в списке.



Данная обработка была подключена к документу "Ввод остатков ОС и НМА" к табличной части "Основные средства". Если открыть документ то рядом с табличной частью появится кнопка "Заполнить". В подменю будет доступна кнопка запуска добавленной обработки.
