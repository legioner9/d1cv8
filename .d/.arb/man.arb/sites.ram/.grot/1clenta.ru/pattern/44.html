Файл настроек списка информационных баз 1С Предприятие
Настройка системы     08.06.2016 10:55     5982

При старте программы 1С Предприятие 8 открывается окно запуска. В окне отображается список информационных баз. Это редактируемый список, в котором можно добавлять, редактировать, удалять элементы. А также есть возможность настроить порядок и задать иерархический вид отображения. При переносе 1С Предприятие 8 на новый компьютер или при переустановке операционной системы, настройки списка и сам список не сохраняются.



Настройки хранятся в специальном файле. Файл расположен по пути:
C:\Users\Администратор\AppData\Roaming\1C\1CEStart\ibases.v8i.
Используя этот файл можно восстановить настройки списка информационных баз.

В некоторых случаях, после замены файла настроек, базы могут перестать запускаться.
При этом вылетает окно ошибки Ошибка формата потока.
Для устранения данной проблемы необходимо из списка удалить базу, а затем заново добавить.

Файл ibases.v8i


В операционной системе Windows, файл настроек с расширением (*.v8i), открывается встроенным текстовым редактором.
