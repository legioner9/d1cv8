Редактор картинок в 1С Конфигураторе
О системе 1С     27.04.2018 11:50     8326

В 1С Предприятии для создания и редактирования графических файлов реализован простой редактор. Он доступен в режиме 1С Конфигуратора. Особенностью данного графического редактора, является возможность создания коллекции картинок. Редактор содержит основной набор инструментов для работы с изображениями. В меню "Действия" и на панели инструментов доступен набор функций для редактирования параметров картинки. Для создания графического файла необходимо перейти в главном меню Файл->Новый->Картинка.





Меню "Действия"


  • Очистить
  • Отразить слева направо
  • Отразить сверху вниз
  • Сетка
  • Режим коллекции
  • Загрузить палитру
  • Сохранить палитру
  • Установить параметры картинки


Панель инструментов


  • Выделение прямоугольником
  • Выделение произвольной формы
  • Выбор цвета
  • Ластик
  • Заливка
  • Масштаб
  • Вырезать / Копировать / Вставить
  • Карандаш
  • Кисть
  • Аэрограф
  • Замена цветов
  • Линия
  • Кривая
  • Прямоугольник
  • Закругленный прямоугольник
  • Текст
  • Параметры картинки
