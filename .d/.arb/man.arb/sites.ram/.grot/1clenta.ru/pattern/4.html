Возможности учебного продукта


Демо версия платформы не накладывает ограничений на сложность реализации разрабатываемых конфигурации. В описании демо платформы указано, что для работы с платформой не требуется программная лицензия или ключ аппаратной защиты.

Доступны следующие возможности:
  • Создавать собственные конфигурации любой сложности
  • Изучать любую конфигурацию
  • Дорабатывать конфигурации, созданные для решения реальных задач автоматизации

Ограничения учебного продукта


  • максимальное количество записей в таблицах счетов 2000
  • максимальное количество записей в главных таблицах объектов 2000
  • максимальное количество записей в табличных частях объектов 1000
  • максимальное количество записей в наборах записей 2000
  • максимальное количество записей из внешних источников данных 200
  • нет возможности работать в клиент-серверном варианте
  • не поддерживается работа РИБ
  • нет возможности работать с COM-соединением
  • нет возможности установить пароль и аутентификацию операционной системы для пользователя
  • только в конфигураторе можно сохранять и печатать табличные документы
  • для табличного документа в режиме "1С:Предприятие" максимальный размер области копирования 1 ячейка
  • быстродействие учебного продукта ниже чем у коммерческого
  • нет возможности использовать хранилище конфигурации
  • нет возможности создавать поставки конфигурации
  • допускается только 1 сеанс работы с ИБ

Данная версия программного продукта предназначена для обучения программированию и настройке конфигураций в системе 1С:Предприятие 8. Пробная версия имеет ряд ограничений, которые не позволяют использовать программный продукт в полной мере для ведения реального учета.
