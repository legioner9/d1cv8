Добавить базу или группу в список баз в 1С:Предприятии
Настройка системы     12.08.2016 13:45     7005

При работе с 1С:Предприятием может потребоваться создать новую информационную базу. Для добавления базы необходимо запустить стартовое окно 1С.

В стартовом окне нажать кнопку "Добавить"



Создание новой информационной базы


Для создания новой базы надо выбрать "Создание новой информационной базы"



Для создания типовой базы необходимо выбрать "Создание информационной базы из шаблона". Для того чтобы добавить пустую базу необходимо выбрать "Создание информационной базы без конфигурации для разработки новой конфигурации или загрузки выгруженной ранее информационной базы".



Определить имя и тип расположения информационной базы.



Задать параметры расположения информационной базы



Задать параметры запуска информационной базы.



Добавление с список существующей информационной базы


Для добавления существующей базы необходимо выбрать соответствующий пункт "Добавление в список существующей информационной базы".



Определить имя и тип расположения информационной базы.



Задать параметры расположения информационной базы



Задать параметры запуска информационной базы.



Создание новой группы


Для добавления группы в дерево списка информационных баз необходимо выбрать "Создание новой группы".



Задать имя группы.
