Установка, обновление 1С:Предприятия 8.2, 8.3
Настройка системы     14.08.2016 10:15     3082

Все прикладные решения 1С работают на базе платформы 1С:Предприятие. Каждая версия платформы устанавливается в операционную систему как отдельная программа. Таким образом понятие "Обновление платформы" означает установку новой версии 1С:Предприятия. 1С:Предприятие - это платный коммерческий продукт, поэтому для работы с программой необходимо иметь лицензию.

Установка 1С:Предприятия


1 - скачать дистрибутив платформы 1С:Предприятие
Для установки или обновления платформы потребуется скачать дистрибутив "Технологической платформы 8.3". Данный дистрибутив доступен на сайте ИТС. Для доступа к данному сайту необходимо иметь действующую подписку на ИТС (Информационно-технологическое сопровождение).

2 - распаковать дистрибутив
Для распаковки дистрибутива на компьютере должно быть установлено программное обеспечение для архивации файлов. После разархивирования в полученной папке необходимо запустить файл setup.exe.



3 - после запуска установщика необходимо следовать простым инструкциям.



Если требуется установить только клиентскую часть выбираем компоненты как указано на скриншоте.













После окончания установки на рабочем столе появится ярлык для запуска программы.
