Доступность объектов, их свойств и методов

При исполнении встроенного языка на сервере 1С:Предприятия не все объекты, а также их свойства и методы могут быть доступны. Например, не доступны объекты глобального контекста, предназначенные для управления интерфейсом, за исключением функции Сообщить().

Также возможна ситуация, когда сам объект доступен для использования на сервере, а его свойства или методы нет. Например, свойство РабочаяДата изменяется конкретным пользователем в текущем сеансе и предназначено для удобства заполнения форм, поэтому его использование на сервере теряет смысл.

Для сессии внешнего соединения доступен весь глобальный контекст, за исключением объектов, жестко связанных с клиентским приложением (ТекстовыйДокумент, ТабличныйДокумент, ...).

Если объект, его свойство или метод не доступен на сервере, то специального упоминания об этом производиться не будет. Исключения составляют значения системных перечислений. В них возможность использования указывается только для самих объектов. Подразумевается, что значения перечислений "наследуют" эту возможность.

Аналогичное правило вводится для возможности использования объектов, их свойств и методов в сессии внешнего соединения.

Доступность объектов, их свойств, методов на тонком и веб-клиентах указывается в описании Синтакс-Помощника. Для показа описаний механизмов по определенным режимам запуска используйте настройку (пункт меню "Сервис - Параметры" закладка "Справка").
