Конец периода

Конец периода (необязательный)

Тип параметра: Дата, МоментВремени, Граница.
Указывается конец периода расчета оборотов. Значение этого параметра по умолчанию включается в период расчета оборотов. Для исключения значения из периода его следует задавать объектом Граница с исключением граничного значения.
Если параметр не задан, обороты рассчитываются по самую последнюю запись.
