Порядок (Order)

Тип: Строка.
Хранит строку с порядком, по которому можно выполнить упорядочивание выборки.
Свойство предназначено для сортировки счетов в случаях, когда порядок следования по полю Код или Наименование не приемлем. Если длина поля Порядок > 0, то при формировании выборок будет автоматически устанавливаться сортировка по полю Порядок. При этом основное представление по полю Код или Наименование влиять на упорядочивание счетов не будет.
Если при конфигурировании у плана счетов установлено свойство "Автопорядок по коду", то при формировании результатов запроса будет отмечено, что сортировка выполняется по полю Код, однако на самом деле будет использоваться сортировка по полю Порядок.
