Конец периода

Конец периода (необязательный)

Тип параметра: Дата, МоментВремени, Граница.
Указывается конец периода расчета итогов. Значение конца периода по умолчанию включается в период расчета итогов. Для исключения значения из периода его следует задавать объектом Граница с исключением граничного значения.
Если параметр не задан, итоги рассчитываются по самую последнюю запись.
