ЧтениеXML.Имя (XMLReader.Name)

ЧтениеXML (XMLReader)

Имя (Name)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Имя текущего узла XML.
Содержит квалифицированное имя для текущего узла. Имя зависит от типа текущего узла:
  • Атрибут - имя атрибута;
  • ОпределениеТипаДокумента - имя типа документа;
  • НачалоЭлемента - имя элемента;
  • КонецЭлемента - имя элемента;
  • СсылкаНаСущность - имя ссылки на сущность;
  • ИнструкцияОбработки - адрес инструкции обработки;
  • ОбъявлениеXML - строка xml.
Для узлов, для которых имя не определено, свойство содержит пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
