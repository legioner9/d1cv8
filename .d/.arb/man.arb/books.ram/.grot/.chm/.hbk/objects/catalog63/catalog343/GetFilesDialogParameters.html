ПараметрыДиалогаПолученияФайлов (GetFilesDialogParameters)

Доступен, начиная с версии 8.3.15.

Свойства:

ВыборКаталога (ChooseDirectory)
Заголовок (Title)
Каталог (Directory)

Конструкторы:

По заголовку, выбору каталога и каталогу
По умолчанию

Описание:

Содержит настройки диалога получения файлов.
Используется в качестве параметра <ПараметрыДиалога> для методов НачатьПолучениеФайлаССервера и НачатьПолучениеФайловССервера.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
