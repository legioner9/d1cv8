ВедущиеВидыРасчета.<Имя плана видов расчета> (LeadingCalculationTypes.<Chart of calculation types name>)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все строки табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данный метод следует использовать осмотрительно в том случае, если в табличной части есть предопределенные строки (т.е. имеются строки, для которых свойство Предопределенный имеет значение Истина). Если удалить все, в том числе, предопределенные строки табличной части, то не будет возможности записать такой вид расчета.
Использование метода допустимо только в том случае, если табличная часть получена из свойства объекта. Если табличная часть получена из свойства ссылки (или выборки), то использование этого метода будет вызывать ошибку выполнения.
