РежимРабочейДаты (WorkingDateMode)

Доступен, начиная с версии 8.0.

Значения

ИспользоватьТекущуюДату (UseCurrentDate)
Назначать (Assign)

Описание:

Определяет набор вариантов использования рабочей даты в глобальном контексте. Используется для определения свойства ИспользованиеРабочейДаты глобального контекста.

Доступность:

Толстый клиент.

См. также:

Глобальный контекст, свойство ИспользованиеРабочейДаты

Использование в версии:

Доступен, начиная с версии 8.0.
