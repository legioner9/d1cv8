Переключаемый (Switchable)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если это свойство установлено в значение Истина, то можно управлять отображением данного интерфейса. В противном случае данный интерфейс будет показываться всегда.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
