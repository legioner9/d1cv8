Расширение формы списка видов расчета (Calculation type list form extension)

Расширение формы списка видов расчета (Calculation type list form extension)

Доступен, начиная с версии 8.0.

Свойства:

ПараметрТекущаяСтрока (CurrentLineParameter)

Описание:

Дополнительные свойства объекта Форма списка видов расчета. Используется в том случае, если список видов расчета является основным реквизитом формы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
