МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ИсключитьИзАдминистраторовАбонента (ExcludeFromSubscriberAdministrators)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ИсключитьИзАдминистраторовАбонента(<Пользователь>, <КодДоступа>)

Параметры:

<Пользователь> (обязательный)

Тип: ИдентификаторПользователяСистемыВзаимодействия.
Пользователь, который перестанет быть администратором абонента.

<КодДоступа> (необязательный)

Тип: Строка, Неопределено.
Ключ администрирования абонента или временный код администрирования.
Значение по умолчанию: Неопределено.

Описание:

Исключает пользователя из списка администраторов абонента.
Если код доступа не задан, то текущий пользователь должен быть администратором абонента. В противном случае, генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
