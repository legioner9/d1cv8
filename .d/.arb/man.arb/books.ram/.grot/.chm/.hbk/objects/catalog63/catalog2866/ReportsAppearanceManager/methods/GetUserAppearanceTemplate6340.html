МенеджерОформленияОтчетов (ReportsAppearanceManager)

ПолучитьМакетОформленияПользователя (GetUserAppearanceTemplate)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьМакетОформленияПользователя(<ИмяПользователя>)

Параметры:

<ИмяПользователя> (необязательный)

Тип: Строка.
Имя пользователя, для которого требуется получить макет оформления. При отсутствии передаваемого значения параметра возвращается макет оформления для текущего пользователя.

Возвращаемое значение:

Тип: Строка.
Если макет оформления не задан – пустая строка.

Описание:

Возвращает имя макета оформления для пользователя. Для доступа к параметрам другого пользователя требуется наличие права "Администрирование данных".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.22.
