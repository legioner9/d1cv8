ОбъектМетаданных: Ресурс (MetadataObject: Resource)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.
Свойство определяет возможность заполнения ресурса из данных заполнения. Используется для заполнения ресурсов из значений текущего отбора списков. Если установлено Истина, то процедура стандартной обработки заполнения будет заполнять ресурс из данных заполнения. Если в данных заполнения нет данных для заполнения этого ресурса или свойство имеет значение Ложь, то ресурс будет заполнен из свойства ЗначениеЗаполнения, если оно установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
