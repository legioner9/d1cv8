Картинка.ФильтрИменФайлов (Picture.FileNameFilter)

Картинка (Picture)

ФильтрИменФайлов (FileNameFilter)

Доступен, начиная с версии 8.2.

Синтаксис:

ФильтрИменФайлов(<ВсеВарианты>, <ПодобратьВариант>)

Параметры:

<ВсеВарианты> (необязательный)

Тип: Булево.
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки. Если значение Ложь - фильтр будет подготовлен для картинки варианта. Если значение Истина - для архива.
Значение по умолчанию: Ложь.

<ПодобратьВариант> (необязательный)

Тип: Соответствие, Неопределено.
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки, и <ВсеВарианты> установлен в значение Ложь.
Пары "Ключ-Значение" соответствуют xml-тегам в манифесте вариантов картинки. Если параметры выбора варианта не заданы, будет выбран вариант по умолчанию. Если параметры указывают на несуществующий в архиве вариант, то будет возвращено Неопределено.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Строка.
Фильтр для диалога выбора файлов.

Описание:

Получает фильтры для диалога выбора файлов, соответствующие формату картинки.

Доступность:

Тонкий клиент, толстый клиент, внешнее соединение.

Пример:

ФильтрДиалога = Картинка.ФильтрИменФайлов();

Использование в версии:

Доступен, начиная с версии 8.2.
