Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ОткрытьВыпадающийСписок (OpenDropList)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОткрытьВыпадающийСписок()

Описание:

Открывает выпадающий список, если такая возможность есть. В противном случае генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
