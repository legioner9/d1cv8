ЗадачиМенеджер (TasksManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЗадачаМенеджер.<Имя задачи>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры задач.
Возможно обращение к менеджеру задачи посредством оператора [...]. В качестве аргумента передается имя задачи, как оно задано в конфигурации.

Свойства:

<Имя задачи> (<Task name>)

Методы:

ТипВсеСсылки (AllRefsType)

Описание:

Предназначен для управления задачами и предоставляет доступ к значениям типа ЗадачаМенеджер.<Имя задачи>. Доступ к объекту осуществляется через свойство глобального контекста Задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.

См. также:

Глобальный контекст, свойство Задачи

Использование в версии:

Доступен, начиная с версии 8.0.
