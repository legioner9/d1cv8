Файл.ИнициализироватьАсинх (File.InitializeAsync)

Файл (File)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Полное имя файла или каталога, с которым будет связан объект.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - инициализированный объект типа Файл.

Описание:

Асинхронная функция. Начинает инициализацию объекта Файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Файл, метод НачатьИнициализацию

Использование в версии:

Доступен, начиная с версии 8.3.18.
