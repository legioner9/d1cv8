ОтображениеПредупрежденияПриРедактировании (WarningOnEditRepresentation)

ОтображениеПредупрежденияПриРедактировании (WarningOnEditRepresentation)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
НеОтображать (DontShow)
Отображать (Show)

Описание:

Содержит варианты отображения предупреждения перед началом редактирования поля формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПолеФормы, свойство ОтображениеПредупрежденияПриРедактировании

Использование в версии:

Доступен, начиная с версии 8.2.
