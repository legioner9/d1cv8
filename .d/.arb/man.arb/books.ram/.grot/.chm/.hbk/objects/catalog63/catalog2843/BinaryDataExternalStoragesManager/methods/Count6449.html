МенеджерВнешнихХранилищДвоичныхДанных.Количество (BinaryDataExternalStoragesManager.Count)

МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Количество (Count)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.

Описание:

Возвращает количество внешних хранилищ.

Доступность:

Сервер.

Пример:

cp = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cp.URL = "https://bucket.example.com";
cp.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

ap = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных();
ap.ИдентификаторДоступа = "EXAMPLE";
ap.СекретныйКлюч = "SecretKey";
ap.Регион = "US";

ВнешниеХранилищаДвоичныхДанных.Добавить("manager1", cp, ap);

cp2 = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cp2.URL = "https://bucket2.example.com";
cp2.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

ВнешниеХранилищаДвоичныхДанных.Добавить("manager2", cp2, ap);

// Вернет число 2

ВнешниеХранилищаДвоичныхДанных.Количество();
