ТипСвязиНаборовДанныхКомпоновкиДанных (DataCompositionDataSetsLinkType)

Доступен, начиная с версии 8.1.

Значения

Внешняя (Outer)
Внутренняя (Inner)

Описание:

Содержит возможные типы связи наборов данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/composition-template}. Имя типа XDTO: DataCompositionDataSetsLinkType.
