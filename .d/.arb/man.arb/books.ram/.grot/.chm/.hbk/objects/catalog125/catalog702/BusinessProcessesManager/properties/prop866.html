БизнесПроцессыМенеджер (BusinessProcessesManager)

<Имя бизнес-процесса> (<Business process name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: БизнесПроцессМенеджер.<Имя бизнес-процесса>.
Набор свойств содержит менеджеры отдельных бизнес-процессов. Имена свойств совпадают с именами бизнес-процессов, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
