Описание:

Менеджер получения программных лицензий на запуск платформы "1С:Предприятие" средствами конфигурации.
Объект может быть получен только из свойства глобального контекста ПолучениеЛицензий.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

См. также:

Глобальный контекст, свойство ПолучениеЛицензий
