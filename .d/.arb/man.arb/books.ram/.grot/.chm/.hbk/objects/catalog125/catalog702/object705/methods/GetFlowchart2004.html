БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ПолучитьКартуМаршрута (GetFlowchart)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьКартуМаршрута()

Возвращаемое значение:

Тип: ГрафическаяСхема.

Описание:

Получает копию карты маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Получение карты маршрута у менеджера бизнес-процесса

КартаМаршрута = БизнесПроцессы.ОформлениеОтпуска.ПолучитьКартуМаршрута();
ЭлементыФормы.ПолеГрафическойСхемы.УстановитьСхему(КартаМаршрута);

Использование в версии:

Доступен, начиная с версии 8.0.
