Описание:

Специализированный узел DOM. Может появляться только в результате вычисления выражения XPath. Представляет собой узел объявления соответствия URI пространства имен. Узел пространства имен является узлом только для чтения. Все попытки изменения, добавления и удаления узла приводят к вызову исключения.
Все атрибуты узла DOM, не описанные ниже, имеют либо значение Ложь, либо Неопределено:

  • ПространствоИменXPath.ДокументВладелец - (одноименные атрибуты разных объектов) соответствует документу-владельцу элемента-владельца данного узла;
  • ПространствоИменXPath.ИмяУзла - (одноименные атрибуты разных объектов) всегда равен "#namespace";
  • ПространствоИменXPath.Префикс - (одноименные атрибуты разных объектов) префикс пространства имен, представляемого данным узлом;
  • ПространствоИменXPath.ЛокальноеИмя - (одноименные атрибуты разных объектов) то же, что и ПространствоИменXPath.Префикс;
  • ПространствоИменXPath.ТипУзла - (одноименные атрибуты разных объектов) ПространствоИменXPath;
  • ПространствоИменXPath.URIПространстваИмен - URI пространства имен, представляемого данным узлом;
  • ПространствоИменXPath.ЗначениеУзла - (одноименные атрибуты разных объектов) то же, что и ПространствоИменXPath.URIПространстваИмен (одноименные атрибуты разных объектов);
  • АдаптироватьУзел, ПространствоИменXPath.КлонироватьУзел и ИмпортироватьУзел - не поддерживаются и вызывают исключения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
