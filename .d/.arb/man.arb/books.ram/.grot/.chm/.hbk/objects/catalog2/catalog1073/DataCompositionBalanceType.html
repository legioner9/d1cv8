ТипОстаткаКомпоновкиДанных (DataCompositionBalanceType)

Доступен, начиная с версии 8.1.

Значения

КонечныйОстаток (ClosingBalance)
НачальныйОстаток (OpeningBalance)

Описание:

Содержит возможные типы остатка.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/common}. Имя типа XDTO: DataCompositionBalanceType.
