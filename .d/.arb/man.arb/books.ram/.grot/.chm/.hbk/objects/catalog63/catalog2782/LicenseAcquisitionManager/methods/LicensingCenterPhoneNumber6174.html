МенеджерПолученияЛицензий (LicenseAcquisitionManager)

НомерТелефонаЦентраЛицензирования (LicensingCenterPhoneNumber)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НомерТелефонаЦентраЛицензирования(<СтранаЦентраЛицензирования>)

Параметры:

<СтранаЦентраЛицензирования> (необязательный)

Тип: Строка.
Код локализации, на основании которого определяется центр лицензирования, используемый для получения лицензии:
  • ru, ru_RU – Российская Федерация;
  • uk, uk_UA, ru_UA – Украина;
  • kk, kk_KZ – Казахстан;
  • пустая строка – центр лицензирования определяется кодом локализации системы;
  • другой текст – все остальные страны.

Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: Строка.

Описание:

Позволяет получить номер телефона центра лицензирования для получения лицензии.
Может использоваться для получения лицензии по телефону и для обращений за консультациями.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
