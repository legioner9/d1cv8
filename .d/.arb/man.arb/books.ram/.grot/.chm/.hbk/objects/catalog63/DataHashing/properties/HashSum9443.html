ХешированиеДанных (DataHashing)

ХешСумма (HashSum)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: Число, ДвоичныеДанные.
Содержит текущее значение хеш-суммы. Для функций MD5, SHA1, SHA256 и SHA512 – значение типа ДвоичныеДанные, для функции CRC32 – Число.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
