Диаграмма.УстановитьТочку (Chart.SetPoint)

Диаграмма (Chart)

УстановитьТочку (SetPoint)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТочку(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.
Точка, которую требуется найти или создать.

Возвращаемое значение:

Тип: ТочкаДиаграммы.
Найденная точка. Если указанной точки нет, создается новая точка, для нее заполняется свойство Значение, и эта точка возвращается.

Описание:

Предназначен для создания и доступа к точкам по ключевому значению. Позволяет выполнять заполнение диаграммы в произвольном порядке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Важно! Во время заполнения диаграммы свойство АвтоТранспонирование не должно быть установлено. Его можно устанавливать после заполнения.

Использование в версии:

Доступен, начиная с версии 8.0.
