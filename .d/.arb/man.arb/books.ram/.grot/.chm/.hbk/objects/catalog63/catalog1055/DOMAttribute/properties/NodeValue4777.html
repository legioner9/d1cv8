АтрибутDOM.ЗначениеУзла (DOMAttribute.NodeValue)

АтрибутDOM (DOMAttribute)

ЗначениеУзла (NodeValue)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит значение узла DOM. Значение узла присутствует не для каждого типа узла DOM.
Узел Значение узла
  • Атрибут - Значение;
  • СекцияCDATA - ТекстDOM.Данные, КомментарийDOM.Данные, СекцияCDATADOM.Данные, содержит CDATA секцию;
  • Комментарий - ТекстDOM.Данные, КомментарийDOM.Данные, СекцияCDATADOM.Данные, содержит комментарий;
  • Документ - Неопределено;
  • ФрагментДокумента - Неопределено;
  • ОпределениеТипаДокумента - Неопределено;
  • Элемент - Неопределено;
  • Сущность - Неопределено;
  • СсылкаНаСущность - Неопределено;
  • Нотация - Неопределено;
  • ИнструкцияОбработки - Данные;
  • Текст - ТекстDOM.Данные, КомментарийDOM.Данные, СекцияCDATADOM.Данные, содержит текст узла.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
