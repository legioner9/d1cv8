Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: Строка.
Идентификатор поля.

Возвращаемое значение:

Тип: ОграничениеИспользованияДоступногоПоляКомпоновкиДанных, Неопределено.

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
