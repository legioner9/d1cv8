ШкалаДиаграммы.Отображение (ChartScale.ShowInChart)

ШкалаДиаграммы (ChartScale)

Отображение (ShowInChart)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: ОтображениеВДиаграмме.
Управляет отображением шкалы диаграммы.
Если установлено значение Отображать и нет ни одной серии, которая выводится на этой шкале, то шкала дублирует основную.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
