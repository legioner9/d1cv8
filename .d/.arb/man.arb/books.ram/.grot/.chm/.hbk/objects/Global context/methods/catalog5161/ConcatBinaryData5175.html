Глобальный контекст (Global context)

СоединитьДвоичныеДанные (ConcatBinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

СоединитьДвоичныеДанные(<МассивДвоичныхДанных>)

Параметры:

<МассивДвоичныхДанных> (обязательный)

Тип: Массив.
Массив объектов типа ДвоичныеДанные.

Возвращаемое значение:

Тип: ДвоичныеДанные.

Описание:

Объединяет несколько объектов типа ДвоичныеДанные в один. Новый объект будет содержать данные всех объектов из массива в том порядке, в котором они были в массиве.
Метод по смыслу аналогичен методу ОбъединитьФайлы, но во многих случаях применение данного метода более эффективно за счет того, что нет необходимости предварительно сохранять двоичные данные в файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Необходимо иметь ввиду, что при использовании объекта ДвоичныеДанные значительный объем данных может храниться в оперативной памяти (до 1 Мб на экземпляр). Одновременное использование большого количества экземпляров ДвоичныеДанные может привести к проблемам с недостачным количеством оперативной памяти.

Особую осторожность необходимо соблюдать при работе с двоичными данными на веб-клиенте, т.к. в этом случае все данные объекта ДвоичныеДанные полностью хранятся в оперативной памяти.

Использование в версии:

Доступен, начиная с версии 8.3.10.
