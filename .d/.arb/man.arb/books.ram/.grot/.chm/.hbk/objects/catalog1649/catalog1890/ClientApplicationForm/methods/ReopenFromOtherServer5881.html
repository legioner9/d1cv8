ФормаКлиентскогоПриложения (ClientApplicationForm)

ПереоткрытьСДругогоСервера (ReopenFromOtherServer)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПереоткрытьСДругогоСервера()

Описание:

Переоткрывает форму, получая ее с другого сервера.
Имеет смысл только при работе мобильного клиента с возможностью автономной работы. При вызове в другом режиме работы будет сгенерировано исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
