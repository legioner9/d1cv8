МакетыТелаТаблицыМакетаКомпоновкиДанных (DataCompositionTemplateTableBodyTemplates)

Доступен, начиная с версии 8.1.

Элементы коллекции:

МакетТелаТаблицыМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются макеты.
Возможно обращение к макету посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит макеты тела таблицы макета компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
