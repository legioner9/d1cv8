СоставКоманднойПанелиФормыНаМобильномУстройстве (MobileDeviceFormCommandBarContent)

Найти (Find)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Идентификатор элемента.

Возвращаемое значение:

Тип: ГруппаФормы, КнопкаФормы.

Описание:

Осуществляет поиск группы или кнопки в составке командной панели формы на мобильном устройстве.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
