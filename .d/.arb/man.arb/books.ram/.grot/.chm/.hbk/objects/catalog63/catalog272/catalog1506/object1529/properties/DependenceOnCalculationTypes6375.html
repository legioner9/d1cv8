ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ЗависимостьОтВидовРасчета (DependenceOnCalculationTypes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: ИспользованиеБазыПланаВидовРасчета.
Если это свойство установлено в значение НеИспользовать, то виды расчета такого плана видов расчета не могут зависеть по базовому периоду от других видов расчета.
Если это свойство установлено в значение ПоПериодуДействия или ПоПериодуРегистрации, то видам расчета такого плана видов расчета можно задать список базовых видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
