СтраницаСканированияДокументов (DocumentScanningPage)

ДанныеИзображения (ImageData)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: ДанныеМультимедиа.
Изображение полученного документа, если его граница найдена или задана пользователем. Если граница не найдена, то возвращается фотография, полученная во время сканирования.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
