РежимВводаСтрокТабличногоПоля (TableBoxRowInputMode)

РежимВводаСтрокТабличногоПоля (TableBoxRowInputMode)

Доступен, начиная с версии 8.0.

Значения

ВКонецОкна (EndOfWindow)
ВКонецСписка (EndOfList)
ПередТекущейСтрокой (BeforeCurrentRow)
ПослеТекущейСтроки (AfterCurrentRow)

Описание:

Содержит варианты режимов интерактивного ввода строк в табличный документ. Используется для определения свойства РежимВводаСтрок табличного поля.

Доступность:

Толстый клиент.

См. также:

ТабличноеПоле, свойство РежимВводаСтрок

Использование в версии:

Доступен, начиная с версии 8.0.
