МенеджерКриптографии (CryptoManager)

ПоказатьСписокСертификатов (ShowCertificateList)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоказатьСписокСертификатов(<ОписаниеОповещенияОЗавершении>)

Параметры:

<ОписаниеОповещенияОЗавершении> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после закрытия окна управления сертификатами со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
  • Если параметр не указан, то по завершении никакая процедура вызвана не будет.

Описание:

Вызывает окно для интерактивной работы с хранилищами сертификатов приложения, в котором пользователь может посмотреть информацию об установленных в хранилищах сертификатов и контейнеров ключей криптографии, удалить их и добавить новые.
Работает только на мобильном клиенте.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе под управлением Windows в хранилище "Персональные Сертификаты" пользователь может удалить только сертификаты с истекшим сроком действия.

Использование в версии:

Доступен, начиная с версии 8.3.10.
