СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяСканированиеДокументов (DocumentScanningSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяСканированиеДокументов()

Возвращаемое значение:

Тип: Булево.
Возвращает значение Истина в следующих случаях:
  • на Android - при наличии камеры на устройстве,
  • на iOS - если система поддерживает видеозапись.
Возвращает значение Ложь на устройствах под управлением ОС семейства Windows.

Описание:

Определяет, поддерживается ли сканирование документов.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
