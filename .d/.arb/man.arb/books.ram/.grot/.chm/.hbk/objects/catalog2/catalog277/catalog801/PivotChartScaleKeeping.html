ПоддержкаМасштабаСводнойДиаграммы (PivotChartScaleKeeping)

Доступен, начиная с версии 8.0.

Значения

ВсеЗначения (AllValues)
КоличествоЗначений (ValuesCount)
МинимальнаяШирина (MinimumWidth)

Описание:

Содержит варианты способов поддержки масштаба (количества одновременно отображаемых значений) при изменении размеров области построения.
Вариант ВсеЗначения предусматривает представление в области построения всех значений. Прокрутка не применяется.
Вариант КоличествоЗначений предусматривает представление области построения определенного количества значений, заданное соответствующим свойством. Если реальных значений больше, применяется прокрутка.
Вариант МинимальнаяШирина предусматривает задание минимальной ширины значения (в символах). Если размер области построения не позволяет отобразить все значения с шириной каждого, не меньше заданной, применяется прокрутка.

Доступность:

Тонкий клиент, сервер, толстый клиент.

См. также:

СводнаяДиаграмма, свойство ГоризонтальнаяПоддержкаМасштаба

Доступен, начиная с версии 8.0.
