СтандартнаяКомандаСистемыВзаимодействия (CollaborationSystemStandardCommand)

КопироватьВложение (CopyAttachment)

Доступен, начиная с версии 8.3.18.

Описание:

Команда копирования вложения системы взаимодействия.
Добавляется в список команд по умолчанию, передаваемых в обработчик формирования команд, если источником команд является Вложение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
