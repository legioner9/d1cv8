МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьОписаниеВнешнейСистемы (GetExternalSystemDescription)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьОписаниеВнешнейСистемы(<ТипВнешнейСистемы>)

Параметры:

<ТипВнешнейСистемы> (обязательный)

Тип: Строка.
Тип внешней системы (например, Telegram-бот или сообщество ВКонтакте).

Возвращаемое значение:

Тип: ОписаниеВнешнейСистемыСистемыВзаимодействия.

Описание:

Возвращает описание внешней системы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
