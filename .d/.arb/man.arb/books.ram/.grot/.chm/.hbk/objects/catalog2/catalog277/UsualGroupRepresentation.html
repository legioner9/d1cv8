ОтображениеОбычнойГруппы (UsualGroupRepresentation)

Доступен, начиная с версии 8.2.

Значения

Нет (None)
ОбычноеВыделение (NormalSeparation)
СильноеВыделение (StrongSeparation)
СлабоеВыделение (WeakSeparation)

Описание:

Содержит варианты отображения группы формы клиентского приложения вида ОбычнаяГруппа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение группы формы для обычной группы, свойство Отображение

Использование в версии:

Доступен, начиная с версии 8.2.
