МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПодключитьОбработчикПослеОтправкиСообщения (AttachAfterMessageSendHandler)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПодключитьОбработчикПослеОтправкиСообщения(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после интерактивной отправки сообщения в обсуждение со следующими параметрами:
  • <Сообщение> - тип СообщениеСистемыВзаимодействия. Сообщение. В обработчик передается копия объекта. В качестве даты сообщения устанавливается текущая дата компьютера. В качестве автора используется текущий пользователь системы взаимодействия;
  • <Обсуждение> - тип ОбсуждениеСистемыВзаимодействия. Передается копия значения;
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Подключает процедуру-обработчик, вызываемую после интерактивной отправки сообщения в обсуждение.
Допускается наличие нескольких обработчиков. В этом случае после отправки сообщения будут вызваны все обработчики. Порядок вызова обработчиков не определен.
Вызов метода с тем же обработчиком замещает обработчик.
Сравнение обработчиков выполняется по равенству значений свойств:

  • ИмяПроцедуры,
  • ИмяПроцедурыОбработкиОшибки,
  • Модуль,
  • МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.11.
