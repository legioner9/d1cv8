РезультатЧтенияДанных (ReadDataResult)

НачатьПолучениеДвоичныхДанных (BeginGetBinaryData)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьПолучениеДвоичныхДанных(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <Результат> - тип ДвоичныеДанные. Результат получения данных,
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Начинает получение результата чтения в виде двоичных данных.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В веб-клиенте данный метод загружает все данные в оперативную память. Чтобы этого избежать, следует использовать метод ОткрытьПотокДляЧтения.

См. также:

РезультатЧтенияДанных, метод ОткрытьПотокДляЧтения

Использование в версии:

Доступен, начиная с версии 8.3.18.
