Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПередЗагрузкойПользовательскихНастроекНаСервере (BeforeLoadUserSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗагрузкойПользовательскихНастроекНаСервере(<Настройки>, <ИспользуютсяСтандартныеНастройки>)

Параметры:

<Настройки>

Тип: ПользовательскиеНастройкиКомпоновкиДанных.
Загружаемые настройки.

<ИспользуютсяСтандартныеНастройки>

Тип: Булево.
Если имеет значение Истина, настройки были сформированы на базе основных настроек отчета. Если Ложь - настройки загружены из хранилища.

Описание:

Вызывается при восстановлении пользовательских настроек, перед помещением настроек в компоновщик настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
