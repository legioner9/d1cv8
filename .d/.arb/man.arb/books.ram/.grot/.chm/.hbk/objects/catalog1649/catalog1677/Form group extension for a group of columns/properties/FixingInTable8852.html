Расширение группы формы для группы колонок (Form group extension for a group of columns)

ФиксацияВТаблице (FixingInTable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ФиксацияВТаблице.
Определяет, будет ли фиксироваться группа в таблице.
Если значение свойства установлено в Лево, то группа прижимается к левому краю таблицы и фиксируется.
Если значение установлено в Право, то группа прижимается к правому краю и фиксируется.
Если значение установлено в значение Нет, то группа выводится обычным образом.
Если свойство установлено в Лево или Право для вложенной группы, то группа переносится на верхних уровень таблицы и фиксируется так, как описано выше.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
