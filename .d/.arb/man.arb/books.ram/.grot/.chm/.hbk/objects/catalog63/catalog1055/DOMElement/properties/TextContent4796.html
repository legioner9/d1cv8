ЭлементDOM.ТекстовоеСодержимое (DOMElement.TextContent)

ЭлементDOM (DOMElement)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Текстовое содержимое узла DOM.
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство ЭтоПробельныеСимволыЭлемента).
Принцип формирования текстового значения узла:
Элемент, Атрибут, Сущность, СсылкаНаСущность, ФрагментДокумента - объединение текстового содержимого каждого из дочерних узлов, исключая Комментарий; ИнструкцияОбработки. Пустая строка, если дочерних узлов нет;
Текст, СекцияCDATA, Комментарий; ИнструкцияОбработки - значение свойства ЭлементDOM.ЗначениеУзла;
Документ, ОпределениеТипаДокумента, Нотация - Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
