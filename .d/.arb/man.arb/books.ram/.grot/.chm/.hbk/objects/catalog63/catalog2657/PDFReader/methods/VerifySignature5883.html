ЧтениеPDF.ПроверитьПодпись (PDFReader.VerifySignature)

ЧтениеPDF (PDFReader)

ПроверитьПодпись (VerifySignature)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПроверитьПодпись(<Имя>, <Сертификат>, <МенеджерКриптографии>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя подписи для проверки.

<Сертификат> (необязательный)

Тип: СертификатКриптографии.
В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).

<МенеджерКриптографии> (обязательный)

Тип: МенеджерКриптографии.
Менеджер криптографии.

Описание:

Проверяет действительность присоединенной к PDF-документу подписи с указанным именем.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Если проверка подписи завершилась неудачно, генерируется исключение.

См. также:

ЧтениеPDF, метод ПроверитьПодписьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.16.
