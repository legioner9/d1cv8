РежимАвтоВремя.ТекущееИлиПоследним (AutoTimeMode.CurrentOrLast)

РежимАвтоВремя (AutoTimeMode)

ТекущееИлиПоследним (CurrentOrLast)

Доступен, начиная с версии 8.0.

Описание:

Если дата объекта (документа, бизнес-процесса, задачи) совпадает с текущей, то будет установлено текущее время. Если дата объекта не совпадает с текущей, то устанавливается такое время, чтобы объект оказался последним за день.

Использование в версии:

Доступен, начиная с версии 8.0.
