КонтейнерКлючейКриптографии (CryptoKeysContainer)

НачатьИнициализацию (BeginInitialization)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяФайла>, <Пароль>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения загрузки контейнера ключей криптографии из файла со следующими параметрами:
  • КонтейнерКлючейКриптографии - созданный КонтейнерКлючейКриптографии.
  • ДополнительныеПараметры – значение, которое было указано при создании объекта ОписаниеОповещения.

<ИмяФайла> (необязательный)

Тип: Строка.
Имя файла, из данных которого будет инициализирован контейнер ключей криптографии.

<Пароль> (необязательный)

Тип: Строка.
Пароль контейнера ключей криптографии.

Описание:

Начинает инициализацию контейнера.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

КонтейнерКлючейКриптографии, метод ИнициализироватьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
