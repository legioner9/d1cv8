Расширение поля формы для поля географической схемы (Form field extension for a geographical schema field)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: РежимИспользованияДиалогаПечати.
Способ вызова печати:
  • НеИспользовать - если выбрана команда "Печать с текущими настройками",
  • Использовать - если выбрана команда "Печать".

<Отказ>

Тип: Булево.
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение Истина, то печать выполнена не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
