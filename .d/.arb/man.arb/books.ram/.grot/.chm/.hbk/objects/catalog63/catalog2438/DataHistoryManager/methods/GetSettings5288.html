МенеджерИсторииДанных.ПолучитьНастройки (DataHistoryManager.GetSettings)

МенеджерИсторииДанных (DataHistoryManager)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьНастройки(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: ОбъектМетаданных: Справочник, ОбъектМетаданных: Документ, ОбъектМетаданных: РегистрСведений, ОбъектМетаданных: БизнесПроцесс, ОбъектМетаданных: Задача, ОбъектМетаданных: ПланСчетов, ОбъектМетаданных: ПланВидовХарактеристик, ОбъектМетаданных: ПланОбмена, ОбъектМетаданных: ПланВидовРасчета, ОбъектМетаданных: Константа.
Объект метаданных.

Возвращаемое значение:

Тип: НастройкиИсторииДанных, Неопределено.
Неопределено - если настройки не установлены.

Описание:

Возвращает настройки истории данных для объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Вызов метода выполняет обращение к серверу.

Примечание:

Результат выполнения кэшируется при первом обращении до изменения конфигурации или версии платформы.

Пример:

Настройки = ИсторияДанных
    .ПолучитьНастройки(Метаданные.Документы.Заказ);

Использование в версии:

Доступен, начиная с версии 8.3.10.
