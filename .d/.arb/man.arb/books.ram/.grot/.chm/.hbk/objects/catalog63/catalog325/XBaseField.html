ПолеXBase (XBaseField)

Доступен, начиная с версии 8.0.

Свойства:

Длина (Length)
Имя (Name)
Тип (Type)
Точность (Precision)

Описание:

Представляет собой поле таблицы базы данных в формате DBF. Используется для получения свойств поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.
