Расширение формы документа (Document form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.
Признак отказа от записи документа. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись документа выполнена не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события ПриЗаписи объекта ДокументОбъект.<Имя документа>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

См. также:

ДокументОбъект.<Имя документа>, событие ПриЗаписи

Использование в версии:

Доступен, начиная с версии 8.0.
