РезультатXPath.НеверноеСостояниеИтератора (XPathResult.InvalidIteratorState)

РезультатXPath (XPathResult)

НеверноеСостояниеИтератора (InvalidIteratorState)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит признак недействительного состояния итератора.
Истина, если итератор становится недействительным (тип результата является НеупорядоченныйИтераторУзлов или УпорядоченныйИтераторУзлов).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Устанавливается в Истина, если документ был изменен после получения данного результата.

Использование в версии:

Доступен, начиная с версии 8.1.
