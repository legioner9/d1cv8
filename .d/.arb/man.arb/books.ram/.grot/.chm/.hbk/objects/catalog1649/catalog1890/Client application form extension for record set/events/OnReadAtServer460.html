Расширение формы клиентского приложения для набора записей (Managed form extension for record set)

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: РегистрСведенийНаборЗаписей.<Имя регистра сведений>, РегистрНакопленияНаборЗаписей.<Имя регистра накопления>, РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии>, РегистрРасчетаНаборЗаписей.<Имя регистра расчета>.
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
