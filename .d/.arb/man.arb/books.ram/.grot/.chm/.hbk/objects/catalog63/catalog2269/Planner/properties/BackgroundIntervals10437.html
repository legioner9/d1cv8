Планировщик (Planner)

ИнтервалыФона (BackgroundIntervals)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: КоллекцияИнтерваловФонаПланировщика.
Предоставляет доступ к интервалам фона планировщика. Представляет собой коллекцию элементов типа ИнтервалФонаПланировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
