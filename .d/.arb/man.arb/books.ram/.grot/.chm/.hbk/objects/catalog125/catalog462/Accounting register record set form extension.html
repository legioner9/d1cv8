Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

Доступен, начиная с версии 8.0.

Методы:

ЗаписатьВФорме (WriteInForm)
Обновить (Refresh)

События:

ПередЗаписью (BeforeWrite)
ПослеЗаписи (AfterWrite)
ПриЗаписи (OnWrite)

Описание:

Предоставляет доступ к дополнительным методам и событиям объекта Форма в том случае, если основной реквизит формы имеет тип РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
