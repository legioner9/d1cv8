ТочкаДиаграммы.Значение (ChartPoint.Value)

ТочкаДиаграммы (ChartPoint)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Содержит значение, идентифицирующее точку в методе диаграммы УстановитьТочку. Строковое представление значения используется для представления точки, если свойство Текст не установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство Текст может быть сформировано программой автоматически, если свойство АвтоУстановкаТекстаТочек = Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
