СерииДиаграммыГанта (GanttChartSeriesCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СерияДиаграммыГанта
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются серии коллекции.
Возможно обращение к серии коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) серии.

Свойства:

Родитель (Parent)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Используется для доступа к свойствам и методам коллекции серий диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
