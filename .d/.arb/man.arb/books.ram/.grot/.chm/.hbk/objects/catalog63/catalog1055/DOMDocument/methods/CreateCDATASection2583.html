ДокументDOM.СоздатьСекциюCDATA (DOMDocument.CreateCDATASection)

ДокументDOM (DOMDocument)

СоздатьСекциюCDATA (CreateCDATASection)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьСекциюCDATA(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: Строка.
Данные секции CDATA.

Возвращаемое значение:

Тип: СекцияCDATADOM.

Описание:

Создает узел DOM - секция CDATA.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
