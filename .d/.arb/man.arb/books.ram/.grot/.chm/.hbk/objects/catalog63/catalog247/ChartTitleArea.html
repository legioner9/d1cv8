ОбластьЗаголовкаДиаграммы (ChartTitleArea)

Доступен, начиная с версии 8.1.

Свойства:

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)
ПрозрачныйФон (Transparent)
Рамка (Border)
Расположение (Placement)
Текст (Text)
ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области заголовка диаграммы для настройки параметров оформления. В области заголовка может содержаться название диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
