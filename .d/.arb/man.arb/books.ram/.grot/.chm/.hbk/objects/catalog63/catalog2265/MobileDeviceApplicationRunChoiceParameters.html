Содержит данные для уточнения фильтров, которые будут использованы для поиска приложения мобильного устройства при запуске с помощью методов ЗапуститьБезОжидания и ЗапуститьАсинх.

Мобильный клиент, мобильное приложение (клиент).

Доступен, начиная с версии 8.3.23.
