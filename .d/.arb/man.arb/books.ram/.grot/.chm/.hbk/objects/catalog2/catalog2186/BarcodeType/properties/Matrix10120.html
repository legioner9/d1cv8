ТипШтрихКода (BarcodeType)

Двухмерный (Matrix)

Доступен, начиная с версии 8.3.5.

Описание:

При вызове сканирования система должна распознавать двухмерные штрихкоды.
Поддерживаются штрихкоды следующих типов: QRCode, DataMatrix, PDF417, Aztec, MaxiCode.

Использование в версии:

Доступен, начиная с версии 8.3.5.
