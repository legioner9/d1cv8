ТипПодписиPDF (PDFSignatureType)

Сертифицирующая (Certifying)

Доступен, начиная с версии 8.3.16.

Описание:

Позволяет внести допустимые типы изменений, при которых документ останется достоверным. Например, заполнение формы или установка подписей.

Использование в версии:

Доступен, начиная с версии 8.3.16.
