Запрос (Query)

ТребуемоеВремяАктуальностиДанных (RequiredDataRelevanceTime)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: Число, Дата, СтандартнаяДатаНачала.
Минимальное допустимое время актуальности данных, которые необходимо получить. Свойство имеет смысл только если ТребуемаяАктуальностьДанных равна Авто или Любые.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.16.
