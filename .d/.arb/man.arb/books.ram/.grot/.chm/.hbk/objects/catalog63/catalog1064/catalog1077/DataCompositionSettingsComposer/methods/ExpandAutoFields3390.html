КомпоновщикНастроекКомпоновкиДанных.РазвернутьАвтоПоля (DataCompositionSettingsComposer.ExpandAutoFields)

КомпоновщикНастроекКомпоновкиДанных (DataCompositionSettingsComposer)

РазвернутьАвтоПоля (ExpandAutoFields)

Доступен, начиная с версии 8.1.

Синтаксис:

РазвернутьАвтоПоля()

Описание:

Преобразует все автополя (АвтоВыбранноеПолеКомпоновкиДанных, АвтоПолеГруппировкиКомпоновкиДанных, АвтоЭлементПорядкаКомпоновкиДанных), содержащиеся в настройках, в наборы соответствующих элементов.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
