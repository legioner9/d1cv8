ТокенДоступа (AccessToken)

По заголовкам и полезной нагрузке

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый ТокенДоступа(<Заголовки>, <ПолезнаяНагрузка>)

Параметры:

<Заголовки> (обязательный)

Тип: Соответствие.
Заголовки токена доступа.

<ПолезнаяНагрузка> (обязательный)

Тип: Соответствие.
Полезная нагрузка токена доступа.

Использование в версии:

Доступен, начиная с версии 8.3.21.
