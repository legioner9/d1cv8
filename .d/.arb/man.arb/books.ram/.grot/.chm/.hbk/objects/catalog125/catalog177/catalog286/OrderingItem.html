ЭлементПорядка (OrderingItem)

Доступен, начиная с версии 8.0.

Свойства:

Данные (Data)
Имя (Name)
Направление (Direction)
Представление (Presentation)
ПутьКДанным (DataPath)

Описание:

Используется для доступа к параметрам порядка по одному элементу порядка в различных выборках, наборах записей и визуальных представлениях.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Порядок
Порядок, метод Получить
Порядок, метод Найти
Порядок, метод Индекс

Использование в версии:

Доступен, начиная с версии 8.0.
