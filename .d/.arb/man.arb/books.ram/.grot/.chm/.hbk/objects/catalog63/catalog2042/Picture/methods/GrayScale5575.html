Картинка.ОттенкиСерого (Picture.GrayScale)

Картинка (Picture)

ОттенкиСерого (GrayScale)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ОттенкиСерого()

Возвращаемое значение:

Тип: Булево, Неопределено.
Если картинка создана с вариантами, то возвращает информацию по текущему варианту.
На мобильной платформе и мобильных клиентах всегда возвращает Неопределено.

Описание:

Возвращает признак того, что изображение хранится в виде оттенков серого.
Может быть определен только, если формат изображения предполагает хранение этой информации (PNG и TIFF).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.19.
