ПараметрыФормированияКомандПланировщика (PlannerCommandGenerateParameters)

Источник (Source)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

  • Источник

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: ИсточникКомандПланировщика.
Содержит информацию о том, на какой объект было произведено нажатие.
В зависимости от того, какой элемент содержит форматированную строку с гиперссылкой, заполняются разные свойства объекта типа ПараметрыФормированияКомандПланировщика.
Если источник равен НавигационнаяСсылка, тогда:
  • Заполнено свойство либо Элементы, либо ЭлементИзмерения, либо Измерение,
  • если навигационная ссылка находится в заголовке элемента измерения, то свойство ЗначенияИзмерений, либо ЭлементШкалыВремени заполнен, иначе содержит Неопределено,
  • Дата содержит Неопределено,
  • НавигационнаяСсылка заполнено,
  • Действие содержит Неопределено,
  • Команды содержат стандартные команды ПерейтиПоНавигационнойСсылке, КопироватьНавигационнуюСсылку,
  • КомандаПоУмолчанию заполнено командой ПерейтиПоНавигационнойСсылке.
Если источник равен Действие тогда:
  • Элементы заполнено,
  • свойства ЭлементИзмерения, Измерение, ЗначенияИзмерений, содержит Неопределено, Дата, ЭлементШкалыВремени, НавигационнаяСсылка содержат Неопределено,
  • Список команд не содержит стандартных команд,
  • <КомандаПоУмолчанию> заполнено командой ВыполнитьДействие.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.
