РазрешенноеВнешнееПриложение (AllowedExternalApplication)

СтрокаКоманды (CommandString)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит шаблон строки запуска приложения. Состоит из последовательности шаблонных слов, разделенных пробелами.
Шаблонное слово - произвольная последовательность символов. Может содержать символы шаблонов. Если содержит пробелы, то должно быть заключено в кавычки. В этом случае кавычки в тексте экранируются. Кавычки в начале и в конце шаблонного слова также экранируются (удваиваются).
Шаблонные символы:
  • % - произвольная последовательность символов;
  • _ - один произвольный символ;
  • * - имя файла. Если шаблонное слово начинается с символа "*", то параметр является путем от виртуального каталога. После проверки соответствия параметра шаблонному слову перед выполнением команды виртуальный каталог будет заменен на физический каталог;
  • \ - префикс символа. Его запись перед шаблонным символом делает его нешаблонным.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
