ПоложениеЗаголовка (TitleLocation)

Доступен, начиная с версии 8.0.

Значения

ЗаголовокСлева (TitleLeft)
ЗаголовокСправа (TitleRight)

Описание:

Содержит варианты расположения заголовка относительно отметок для элементов управления Флажок, Переключатель.

Доступность:

Толстый клиент.

См. также:

Флажок, свойство ПоложениеЗаголовка
Переключатель, свойство ПоложениеЗаголовка

Использование в версии:

Доступен, начиная с версии 8.0.
