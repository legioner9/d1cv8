ТабличноеПоле (TableBox)

АвтоВводНовойСтроки (AutoInsertNewRow)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак автоматического ввода новой строки. В этом режиме новая строка вводится в табличное поле автоматически после окончания редактирования последнего элемента предыдущей строки.
Истина - новая строка вводится автоматически.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
