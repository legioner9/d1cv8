ОбработкаРасшифровкиКомпоновкиДанных.Расшифровать (DataCompositionDetailsProcess.DrillDown)

ОбработкаРасшифровкиКомпоновкиДанных (DataCompositionDetailsProcess)

Расшифровать (DrillDown)

Доступен, начиная с версии 8.1.

Синтаксис:

Расшифровать(<ИдентификаторРасшифровки>, <Поле>)

Параметры:

<ИдентификаторРасшифровки> (обязательный)

Тип: ИдентификаторРасшифровкиКомпоновкиДанных.
Идентификатор обрабатываемой расшифровки.

<Поле> (обязательный)

Тип: ПолеКомпоновкиДанных.
Поле, по которому должна быть произведена расшифровка.

Возвращаемое значение:

Тип: НастройкиКомпоновкиДанных.

Описание:

Позволяет получить более детальную информацию, развернув ее по заданному полю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
