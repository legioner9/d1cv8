ПолеHTMLДокумента (HTMLDocumentField)

Документ (Document)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ВнешнийОбъект.
Предоставляет доступ к HTML-документу.
Описание объекта, в целом, определяется стандартом World Wide Web Consortium (W3C) Document Object Model (DOM) Level 1 (http://www.w3.org/TR/2000/WD-DOM-Level-1-20000929/).
Но объект имеет особенности, связанные с реализацией. ПолеHTMLДокумента реализовано с использованием библиотеки WebKit.

Доступность:

Толстый клиент.

Примечание:

В веб-клиенте из-за ограничений политики безопасности браузеров, невозможно получить объект Документ, если страница расположена на другом сервере. Поэтому в этом случае значение поля Документ будет Неопределено.

Использование в версии:

Доступен, начиная с версии 8.0.
