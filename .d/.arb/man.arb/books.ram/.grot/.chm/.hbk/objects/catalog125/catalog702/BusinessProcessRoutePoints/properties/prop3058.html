ТочкиМаршрута (BusinessProcessRoutePoints)

<Имя точки маршрута бизнес-процесса> (<Business process route point name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Набор свойств содержит точки маршрута бизнес-процесса. Доступ к точке маршрута осуществляется по имени, как оно задано в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
