Описание:

Тип: КолонкаТабличногоПоля.
Набор свойств содержит значения полей колонок табличного поля. Доступ к значению осуществляется по имени колонки. Имена свойств совпадают с именами колонок табличного поля.

Доступность:

Толстый клиент.
