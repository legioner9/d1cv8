ОбновлениеТекстаРедактирования (EditTextUpdate)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
Всегда (Always)
НеИспользовать (DontUse)
ПриИзмененииЗначения (OnValueChange)

Описание:

Содержит варианты обновления текста редактирования в поле ввода формы клиентского приложения при установке значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

См. также:

Расширение поля формы для поля ввода, свойство ОбновлениеТекстаРедактирования

Использование в версии:

Доступен, начиная с версии 8.2.
