ЖурналыДокументовМенеджер (DocumentJournalsManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЖурналДокументовМенеджер.<Имя журнала документов>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры журналов документов.
Возможно обращение к менеджеру журнала документов посредством оператора [...]. В качестве аргумента передается имя журнала документов, как оно задано в конфигурации.

Свойства:

<Имя журнала документов> (<Document journal name>)

Описание:

Предназначен для управления журналами документов и предоставляет доступ к значениям типа ЖурналДокументовМенеджер.<Имя журнала документов>. Доступ к объекту осуществляется через свойство глобального контекста ЖурналыДокументов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

Пример:

// менеджер журналов документов предоставляет доступ к 

// менеджеру журнала Реализация.

Выборка = ЖурналыДокументов.Реализация.Выбрать();

См. также:

Глобальный контекст, свойство ЖурналыДокументов

Использование в версии:

Доступен, начиная с версии 8.0.
