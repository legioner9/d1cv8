ЗаписьТекста.Формирование неинициализированного объекта

ЗаписьТекста (TextWriter)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЗаписьТекста()

Описание:

Формирует неинициализированный объект ЗаписьТекста. Для инициализации объекта используется метод Открыть.

Пример:

Текст = Новый ЗаписьТекста;

Использование в версии:

Доступен, начиная с версии 8.0.
