ВариантЗаписиДатыJSON (JSONDateWritingVariant)

Доступен, начиная с версии 8.3.6.

Значения

ЛокальнаяДата (LocalDate)
ЛокальнаяДатаСоСмещением (LocalDateWithOffset)
УниверсальнаяДата (UniversalDate)

Описание:

Определяет варианты записи даты JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
