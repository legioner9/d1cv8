Разделитель.Ориентация (Splitter.Orientation)

Разделитель (Splitter)

Ориентация (Orientation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Ориентация.
Содержит вариант ориентации разделителя. Горизонтальный разделитель делит окно формы на верхнюю и нижнюю части, вертикальный - на правую и левую.

Доступность:

Толстый клиент.

Примечание:

При ориентации Авто разделитель отображается горизонтально, если его горизонтальный размер больше или равен вертикальному. Если вертикальный размер больше горизонтального - разделитель отображается вертикально.

Использование в версии:

Доступен, начиная с версии 8.0.
