КоллекцияИнформацииОЗаписиВерсииИсторииДанных (WriteDataHistoryVersionInformationCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.15.

Синтаксис:

Индекс(<ИнформацияОЗаписиВерсииИсторииДанных>)

Параметры:

<ИнформацияОЗаписиВерсииИсторииДанных> (необязательный)

Тип: ИнформацияОЗаписиВерсииИсторииДанных.
Версия истории данных.

Возвращаемое значение:

Тип: Число.

Описание:

Получает индекс версии в коллекции версий истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
