Внешний компонент.AddInHash (ISecurityProfileAddIn.AddInHash)

Внешний компонент (ISecurityProfileAddIn)

AddInHash (AddInHash)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит контрольную сумму разрешенного внешнего компонента, вычисленную алгоритмом SHA-1 и преобразованную к строке base64. Для построения контрольной суммы можно использовать:
  • объект ХешированиеДанных с указанием хеш-функции SHA1;
  • метод глобального контекста Base64Строка.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
