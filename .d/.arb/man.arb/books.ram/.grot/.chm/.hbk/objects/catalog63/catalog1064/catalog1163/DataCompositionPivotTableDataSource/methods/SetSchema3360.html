ИсточникДанныхСводнойТаблицыКомпоновкиДанных (DataCompositionPivotTableDataSource)

УстановитьСхему (SetSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьСхему(<Схема>)

Параметры:

<Схема> (обязательный)

Тип: СхемаКомпоновкиДанных.
Устанавливаемая схема компоновки данных.

Описание:

Устанавливает схему компоновки данных в источник сводной таблицы.
Схема компоновки данных определяет перечень ресурсов, измерений и реквизитов, которыми можно будет манипулировать в сводной таблице.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод должен быть вызван до установки источника данных в сводную таблицу.

Использование в версии:

Доступен, начиная с версии 8.1.
