Требование назначения сервисов (IAssignmentRule)

Доступен, начиная с версии 8.3.1.

Свойства:

ApplicationExt (ApplicationExt)
InfoBaseName (InfoBaseName)
ObjectType (ObjectType)
Priority (Priority)
RuleType (RuleType)

Описание:

Определяет одно из множества правил, в соответствии с которыми кластер серверов распределяет соединения с информационными базами и сервисы кластера по рабочим серверам и отдельным процессам.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
