Глобальный контекст (Global context)

ДокументЗащищенПаролемАсинх (DocumentIsPasswordProtectedAsync)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: Из файла

Синтаксис:

ДокументЗащищенПаролемАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла для проверки наличия защиты паролем.

Вариант синтаксиса: Из потока

Синтаксис:

ДокументЗащищенПаролемАсинх(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: ФайловыйПоток, ПотокВПамяти, Поток.
Поток для проверки наличия защиты паролем.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания – значение типа Булево:
Истина – у файла установлена защита паролем.

Описание:

Асинхронная функция. Проверяет наличие защиты паролем у файла. Применяется для файлов форматов:

  • ДокументPDF.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
