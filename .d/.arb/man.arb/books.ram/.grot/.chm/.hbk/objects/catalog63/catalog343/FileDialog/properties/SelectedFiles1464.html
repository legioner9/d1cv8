ДиалогВыбораФайла.ВыбранныеФайлы (FileDialog.SelectedFiles)

ДиалогВыбораФайла (FileDialog)

ВыбранныеФайлы (SelectedFiles)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Массив.
Содержит массив строк с полными именами файлов, выбранных в диалоге.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В режиме выбора каталога содержит пустой массив.
В веб-клиенте, если расширение работы с файлами не установлено, то массив содержит пустые строки.

Использование в версии:

Доступен, начиная с версии 8.0.
