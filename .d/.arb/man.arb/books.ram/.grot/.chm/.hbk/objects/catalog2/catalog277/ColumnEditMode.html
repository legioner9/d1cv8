РежимРедактированияКолонки (ColumnEditMode)

Доступен, начиная с версии 8.0.

Значения

Вход (Enter)
ВходПриВводе (EnterOnInput)
Непосредственно (Directly)

Описание:

Определяет варианты редактирования содержимого колонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.
