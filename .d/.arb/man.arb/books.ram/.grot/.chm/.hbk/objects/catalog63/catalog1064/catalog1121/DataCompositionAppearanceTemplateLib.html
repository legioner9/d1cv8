Элементы коллекции:

ЭлементБиблиотекиМакетовОформленияКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы библиотеки макетов оформления.
Возможно обращение к элемементам библиотеки макетов оформления посредством оператора [...]. В качестве аргумента передается .

Описание:

Содержит предопределенные макеты оформления компоновки данных, а также макеты оформления, определенные в общих макетах дерева конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
