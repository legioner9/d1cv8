ИнтернетТекстыПочтовогоСообщения (InternetMailTexts)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: С указанием типа текста

Синтаксис:

Добавить(<Текст>, <ТипТекста>)

Параметры:

<Текст> (необязательный)

Тип: Строка.
Добавляемый текст.

<ТипТекста> (необязательный)

Тип: Строка, ТипТекстаПочтовогоСообщения.
Указывает тип добавляемого текста.
Если Строка, то свойство ТипТекста устанавливается в ПроизвольныйТекст.
При генерации отправляемого текста письма, в поле Content-Type письма добавляется (или заменяется) параметр charset, получаемый из кодировки, указанной в свойстве Кодировка.
Значение по умолчанию: ПростойТекст.

Вариант синтаксиса: Без указания типа текста

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.
Добавляемый текст.

Возвращаемое значение:

Тип: ИнтернетТекстПочтовогоСообщения.

Описание:

Добавляет текст в коллекцию почтовых текстов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если для всех параметров метода используются значения по умолчанию, то ТипТекста добавленного объекта ИнтернетТекстПочтовогоСообщения будет равен HTML. Если же указан первый параметр <Текст>, то в качестве типа текста будет установлен ПростойТекст.

Использование в версии:

Доступен, начиная с версии 8.0.
