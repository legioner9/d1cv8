ПланСчетовВыборка.<Имя плана счетов>.ПолучитьОбъект (ChartOfAccountsSelection.<Chart of accounts name>.GetObject)

ПланСчетовВыборка.<Имя плана счетов> (ChartOfAccountsSelection.<Chart of accounts name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: ПланСчетовОбъект.<Имя плана счетов>, Неопределено.
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для модификации и записи элемента, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
