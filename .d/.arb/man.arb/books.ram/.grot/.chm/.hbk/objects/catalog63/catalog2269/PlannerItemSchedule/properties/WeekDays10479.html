РасписаниеЭлементаПланировщика.ДниНедели (PlannerItemSchedule.WeekDays)

РасписаниеЭлементаПланировщика (PlannerItemSchedule)

ДниНедели (WeekDays)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Массив.
Массив номеров дней недели, по которым повторяется интервал (понедельник - 1, вторник - 2 и т.д.).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
