Соединение с агентом сервера.GetClusterManagers (IServerAgentConnection.GetClusterManagers)

Соединение с агентом сервера (IServerAgentConnection)

GetClusterManagers (GetClusterManagers)

Доступен, начиная с версии 8.2.

Синтаксис:

GetClusterManagers(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер серверов. Он может быть элементом массива кластеров, зарегистрированных в текущем центральном сервере, полученного методом GetClusters. Также кластер может быть создан методом CreateClusterInfo с последующим заполнением его свойств.

Возвращаемое значение:

Тип: COMSafeArray.
Массив объектов Менеджер кластера. Один из них представляет главный менеджер кластера, а остальные – дополнительные менеджеры, зарегистрированные в кластере.

Описание:

Получает список менеджеров, зарегистрированных в кластере.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
