ЗаписьДанных.ЗаписатьАсинх (DataWriter.WriteAsync)

ЗаписьДанных (DataWriter)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Запись двоичных данных

Синтаксис:

ЗаписатьАсинх(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: ДвоичныеДанные.
Записать экземпляр объекта ДвоичныеДанные в поток.

Описание варианта метода:

Записывает в поток данные из объекта типа ДвоичныеДанные.

Вариант синтаксиса: Запись результата чтения данных

Синтаксис:

ЗаписатьАсинх(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: РезультатЧтенияДанных.
Записать экземпляр объекта РезультатЧтенияДанных в поток.

Описание варианта метода:

Записывает в целевой поток данные из объекта типа РезультатЧтенияДанных.

Возвращаемое значение:

Тип: Обещание.

Описание:

Асинхронная функция. Записывает данные в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.
