ИсточникДанныхСводнойТаблицыКомпоновкиДанных (DataCompositionPivotTableDataSource)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьНастройки()

Возвращаемое значение:

Тип: НастройкиКомпоновкиДанных.

Описание:

Получает текущее состояние сводной таблицы в виде настроек системы компоновки данных.
Настройки содержат таблицу, группировки которой соответствуют выбранным измерениям сводной таблицы, выбранные поля группировок соответствуют выбранным реквизитам.
Выбранные ресурсы сохраняются как выбранные поля в настройках.
Также настройки могут содержать отбор и упорядочивание, которые сохранены после последней установки настроек в сводную таблицу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод может быть вызван после того, как в источник была установлена схема компоновки данных (метод УстановитьСхему) и источник был установлен в сводную таблицу.

Использование в версии:

Доступен, начиная с версии 8.1.
