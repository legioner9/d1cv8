Виртуальный каталог (ISecurityProfileVirtualDirectory)

PhysicalPath (PhysicalPath)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Физический URL каталога на сервере для размещения данных виртуального каталога. Может включать макросы. В процессе выполнения файловых операций логический URL заменяется на физический с подстановкой всех макросов. При этом каждая последовательность символов, не допустимых в URL, заменяется на один символ "_" (подчеркивание). Возможны следующие макросы:
  • %r – ссылочное имя информационной базы;
  • %i – идентификатор информационной базы;
  • %z – строковое представление текущих значений разделителей в формате значения параметра /Z;
  • %s – номер сеанса;
  • %p – идентификатор безопасного режима исполнения кода;
  • %e – каталог загрузочных модулей 1С:Предприятия;
  • %t – текущий каталог временных файлов операционной системы;
  • %u – каталог данных приложений текущего пользователя операционной системы, от имени которого запущен процесс rphost;
  • %a – каталог данных приложений всех пользователей;
  • %n – имя текущего пользователя информационной базы;
  • %% – символ процент.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
