ЭлементЛегендыГеографическойСхемы (GeographicalSchemaLegendItem)

ОтображатьШкалу (ShowScale)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ТипОтображенияШкалыЭлементаЛегендыГеографическойСхемы.
Определяет способ отображения шкалы значений для элемента легенды географической схемы.
Имеет смысл только в том случае, если с элементом связана серия слоя географической схемы и тип её отрисовки установлен как один из перечисленных - Гистограмма, РазмерФигуры, ОттенокЦветаФигуры.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
