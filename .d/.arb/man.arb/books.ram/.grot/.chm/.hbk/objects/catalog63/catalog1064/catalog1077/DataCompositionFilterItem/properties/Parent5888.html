ЭлементОтбораКомпоновкиДанных.Родитель (DataCompositionFilterItem.Parent)

ЭлементОтбораКомпоновкиДанных (DataCompositionFilterItem)

Родитель (Parent)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Неопределено, ГруппаЭлементовОтбораКомпоновкиДанных.
Родительская группа элементов отбора, Неопределено для элементов отбора верхнего уровня.
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
