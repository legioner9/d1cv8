ОтчетыМенеджер (ReportsManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ОтчетМенеджер.<Имя отчета>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры отчетов.
Возможно обращение к менеджеру отчета посредством оператора [...]. В качестве аргумента передается имя отчета, как оно задано в конфигурации.

Свойства:

<Имя отчета> (<Report name>)

Описание:

Предназначен для управления отчетами и предоставляет доступ к значениям типа ОтчетМенеджер.<Имя отчета>. Доступ к объекту осуществляется через свойство глобального контекста Отчеты.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

Глобальный контекст, свойство Отчеты

Использование в версии:

Доступен, начиная с версии 8.0.
