ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ДоступноеПолеСхемыЗапроса, ДоступнаяВложеннаяТаблицаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля и доступные вложенные таблицы.
Возможно обращение к доступным полям и доступным вложенным таблицам посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Коллекция доступных полей запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
