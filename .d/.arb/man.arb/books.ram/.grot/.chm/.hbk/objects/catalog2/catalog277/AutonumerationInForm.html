АвтонумерацияВФорме (AutonumerationInForm)

Доступен, начиная с версии 8.1.

Значения

Авто (Auto)
НеИспользовать (DontUse)

Описание:

Режим автонумерации в форме определяет, получать или нет автоматически номер при открытии формы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение формы элемента справочника, свойство Автонумерация
Расширение формы документа, свойство Автонумерация
Расширение формы элемента вида характеристик, свойство Автонумерация
Расширение формы объекта бизнес-процесс, свойство Автонумерация
Расширение формы задачи, свойство Автонумерация
Расширение табличного поля списка справочника, свойство Автонумерация
Расширение табличного поля списка видов характеристик, свойство Автонумерация
Расширение табличного поля списка задач, свойство Автонумерация

Использование в версии:

Доступен, начиная с версии 8.1.
