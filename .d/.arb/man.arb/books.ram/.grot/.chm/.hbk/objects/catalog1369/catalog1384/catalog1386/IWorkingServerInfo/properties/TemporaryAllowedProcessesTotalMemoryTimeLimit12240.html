Рабочий сервер.TemporaryAllowedProcessesTotalMemoryTimeLimit (IWorkingServerInfo.TemporaryAllowedProcessesTotalMemoryTimeLimit)

Рабочий сервер (IWorkingServerInfo)

TemporaryAllowedProcessesTotalMemoryTimeLimit (TemporaryAllowedProcessesTotalMemoryTimeLimit)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Определяет длительность периода превышения временно допустимого объема памяти процессами кластера, после которого рабочие процессы перезапускаются.
Параметр будет действовать, если его значение отлично от 0, и значение параметра ВременноДопустимыйОбъемПамятиПроцессов отлично от -1.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
