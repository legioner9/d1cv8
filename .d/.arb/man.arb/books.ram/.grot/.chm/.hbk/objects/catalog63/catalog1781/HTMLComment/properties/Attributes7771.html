КомментарийHTML.Атрибуты (HTMLComment.Attributes)

КомментарийHTML (HTMLComment)

Атрибуты (Attributes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: КоллекцияАтрибутовHTML.
Содержит коллекцию атрибутов узла. Коллекция атрибутов доступна только для узла Element.
Узел Атрибуты
  • Атрибут - Неопределено;
  • Комментарий - Неопределено;
  • Документ - Неопределено;
  • ФрагментДокумента - Неопределено;
  • ОпределениеТипаДокумента - Неопределено;
  • Элемент - КоллекцияАтрибутовHTML;
  • Текст - Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
