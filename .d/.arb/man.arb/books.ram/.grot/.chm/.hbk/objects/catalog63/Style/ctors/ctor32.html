Стиль (Style)

Конструктор по умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Стиль()

Описание:

Создает значение, которое будет восприниматься как стиль по умолчанию (стиль "Авто").

Использование в версии:

Доступен, начиная с версии 8.0.
