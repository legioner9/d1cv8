Расширение поля формы для поля планировщика (Form field extension for a planner)

ПроверкаПеретаскиванияВнутри (InsideDragCheck)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверкаПеретаскиванияВнутри(<ПараметрыПеретаскивания>, <ВариантИзмененияГраниц>, <ДействиеПеретаскивания>, <Отказ>)

Параметры:

<ПараметрыПеретаскивания>

Тип: ФиксированныйМассив.
Фиксированный массив из объектов типа ПараметрПеретаскиванияВнутриПланировщика. Описывает параметры перетаскивания внутри поля планировщика.
Возвращаемое значение.

<ВариантИзмененияГраниц>

Тип: ВариантИзмененияГраницПеретаскиванияВнутриПланировщика.
Описывает вариант изменения границ перетаскивания внутри поля планировщика.

<ДействиеПеретаскивания>

Тип: ДействиеПеретаскиванияВнутриПланировщика.
Фиксированный массив из объектов типа ПараметрПеретаскиванияВнутриПланировщика. Описывает параметры перетаскивания внутри поля планировщика.
Возвращаемое значение.

<Отказ>

Тип: Булево.
Если Ложь, то при выходе из события поле планировщика изменится в соответствии со значениями в фиксированном массиве <ПараметрыПеретаскивания>. Если Истина, то при выходе из события:
  • в случае создания элемента планировщика путем выделения пустой области границы области выделения не будут изменены,
  • в случае выделения элементов планировщика границы области выделения не будут изменены,
  • в случае редактирования элементов планировщика путем перетаскивания границы элементов не будут изменены,
  • в случае копирования существующих элементов планировщика границы создаваемых элементов не будут изменены.

Значение по умолчанию: Ложь.

Описание:

Возникает в следующих ситуациях:

  • Выделение пустой области планировщика для создания элемента планировщика.
  • Выделение элементов планировщика.
  • Перетаскивание или изменение границ одного или нескольких элементов планировщика.
  • Копирование одного или нескольких элементов планировщика.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
