НаборДанныхЗапросСхемыКомпоновкиДанных (DataCompositionSchemaDataSetQuery)

ИспользоватьГруппировкиЗапросаЕслиВозможно (UseQueryGroupIfPossible)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает на возможность использования группировки запроса для расчета агрегатных функций схемы компоновки данных.
Рекомендуется устанавливать значение Ложь, если в запросе используются данные, группировка которых не имеет смысла и может привести к снижению производительности, например, из-за того, что данные уже уникальны.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
