ИнтернетПочта (InternetMail)

ПолучитьКоличествоСообщений (GetMessageCount)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьКоличествоСообщений()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество сообщений IMAP в текущем почтовом ящике (ТекущийПочтовыйЯщик) или количество сообщений в ящике POP3.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
