ПараметрыСеанса (SessionParameters)

Доступен, начиная с версии 8.0.

Элементы коллекции:

Произвольный
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры сеанса.

Свойства:

<Имя параметра сеанса> (<Session parameter name>)

Методы:

Очистить (Clear)

Описание:

Является значением свойства глобального контекста ПараметрыСеанса и служит для доступа к значениям параметров сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

Пример:

// Установка значения параметра сеанса.

ПараметрыСеанса.ТекущийОтдел = "Производство";
// Получение значения параметра сеанса.

МойОтдел = ПараметрыСеанса.ТекущийОтдел;
// Обращение к параметру сеанса из ограничения доступа к данным.

ГДЕ Отдел = &ТекущийОтдел
