Только чтение.

Описание:

Тип: МенеджерОкнаВнешнегоСайта.
Используется для доступа к функциям окна внешнего сайта, когда веб-клиент интегрирован в него.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
