XBase.Переиндексировать (XBase.Reindex)

XBase (XBase)

Переиндексировать (Reindex)

Доступен, начиная с версии 8.0.

Синтаксис:

Переиндексировать()

Описание:

Выполняет пересчет индекса таблицы базы данных. Объекты XBase автоматически выполняют индексацию при изменениях в таблице базы данных, однако при работе более чем с одним индексным файлом или после аварийного завершения может потребоваться принудительная переиндексация.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

Если ФлагАварийногоЗавершения Тогда
    xB.Переиндексировать();
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
