Глобальный контекст (Global context)

ПолучитьСкоростьКлиентскогоСоединения (GetClientConnectionSpeed)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьСкоростьКлиентскогоСоединения()

Возвращаемое значение:

Тип: СкоростьКлиентскогоСоединения.

Описание:

Возвращает скорость клиентского соединения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме запуска Толстый клиент и Внешнее соединение всегда содержит Обычная.

Использование в версии:

Доступен, начиная с версии 8.2.
