ДоступныеПоляСхемыЗапроса.Добавить (QuerySchemaAvailableFields.Add)

ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Имя>, <Тип>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого поля.

<Тип> (необязательный)

Тип: Тип.
Содержит описание типа добавляемого элемента.
Значение по умолчанию: Неопределено.

Описание:

Добавляет новый элемент коллекции. Метод доступен только для доступных полей описания временной таблицы (ДоступныеПоля).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
