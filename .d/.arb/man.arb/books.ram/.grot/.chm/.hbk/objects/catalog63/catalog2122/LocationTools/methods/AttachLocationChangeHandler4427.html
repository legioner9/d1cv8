СредстваГеопозиционирования (LocationTools)

ПодключитьОбработчикИзмененияМестоположения (AttachLocationChangeHandler)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПодключитьОбработчикИзмененияМестоположения(<Процедура>, <ИмяПровайдера>, <МинимальноеРасстояние>, <МинимальныйПериодВремени>)

Параметры:

<Процедура> (обязательный)

Тип: ОписаниеОповещения, Строка.
Вызываемая процедура.
Процедура должна иметь параметры:
  • ИмяПровайдера - имя провайдера, тип Строка;
  • ДанныеМестоположения - данные местоположения: тип ДанныеМестоположения;
  • если используется описание оповещения, могут быть переданы дополнительные параметры.

<ИмяПровайдера> (обязательный)

Тип: Строка.
Содержит имя провайдера.

<МинимальноеРасстояние> (необязательный)

Тип: Число.
Минимальное расстояние от координат предыдущего местоположения в метрах.

<МинимальныйПериодВремени> (необязательный)

Тип: Число.
Минимальный период времени от времени получения предыдущего местоположения в секундах.

Описание:

Подключает вызов указанной процедуры при изменении местоположения.
Параметры <МинРасстояние> и <МинПериодВремени> используются для определения частоты обновления местоположения.
Если параметры не указаны, события будут порождаться с частотой, максимально возможной для данного провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
