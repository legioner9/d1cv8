КатегорияСодержимогоHTML (HTMLContentCategory)

StyleТеги (StyleTags)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все элементы <style> с атрибутом type, значение которого отличается от text/css, т.е.:
<style type="text/html" src="some/link/to/src"> будет удален, а <style type="text/css" src="some/link/to/src"> удален не будет.

Использование в версии:

Доступен, начиная с версии 8.3.13.
