МетодКластеризации.БлижняяСвязь (ClusterizationMethod.NearestNeighbor)

МетодКластеризации (ClusterizationMethod)

БлижняяСвязь (NearestNeighbor)

Доступен, начиная с версии 8.0.

Описание:

Метод "Ближней связи". Метод кластеризации, в котором объект присоединяется к той группе, для которой расстояние до ближайшего объекта минимально.

Использование в версии:

Доступен, начиная с версии 8.0.
