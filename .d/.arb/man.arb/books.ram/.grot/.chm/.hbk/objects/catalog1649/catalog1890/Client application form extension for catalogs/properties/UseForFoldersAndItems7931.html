Расширение формы клиентского приложения для справочника (Client application form extension for catalogs)

ИспользованиеДляГруппИЭлементов (UseForFoldersAndItems)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеГруппИЭлементов.
Указывает, как следует использовать форму для редактирования групп или элементов.
В случае открытия формы средствами встроенного языка с помощью данного свойства можно указать, для чего данная форма будет использована. По значению свойства будут автоматически сформирована командная панель и сгенерированы управляющие элементы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может принимать только значения Элементы или Группы.

Использование в версии:

Доступен, начиная с версии 8.2.
