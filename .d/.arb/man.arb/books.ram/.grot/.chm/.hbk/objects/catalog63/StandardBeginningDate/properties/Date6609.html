СтандартнаяДатаНачала.Дата (StandardBeginningDate.Date)

СтандартнаяДатаНачала (StandardBeginningDate)

Дата (Date)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Дата.
В случае, если вариант даты - ПроизвольнаяДата, содержит дату установленную для даты компоновки.
В случае, если вариант даты отличен от ПроизвольнаяДата, содержит дату, которая будет использоваться в качестве значения даты компоновки, если выполнить компоновку сейчас.
При присвоении значения этому свойству вариант даты автоматически устанавливается в ПроизвольнаяДата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
