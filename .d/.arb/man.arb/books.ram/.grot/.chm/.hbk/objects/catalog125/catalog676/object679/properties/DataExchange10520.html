ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: ПараметрыОбменаДанными.
Используется для управления обменом данных.
С помощью данного свойства устанавливается режим "Загрузка", при котором выполняется перенос информации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
