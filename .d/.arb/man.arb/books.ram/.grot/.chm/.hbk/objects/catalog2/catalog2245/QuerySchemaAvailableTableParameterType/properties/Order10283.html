ТипПараметраДоступнойТаблицыСхемыЗапроса (QuerySchemaAvailableTableParameterType)

Порядок (Order)

Доступен, начиная с версии 8.3.5.

Описание:

В параметре указывается список полей для упорядочивания с указанием типа упорядочивания.
Поля, которые можно использовать в параметре содержатся в свойстве ДоступныеПоля объекта типа ПараметрДоступнойТаблицыСхемыЗапроса.

Использование в версии:

Доступен, начиная с версии 8.3.5.
