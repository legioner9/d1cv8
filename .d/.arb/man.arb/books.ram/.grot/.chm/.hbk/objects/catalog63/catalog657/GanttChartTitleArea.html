ОбластьЗаголовкаДиаграммыГанта (GanttChartTitleArea)

Доступен, начиная с версии 8.1.

Свойства:

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)
ПрозрачныйФон (Transparent)
Рамка (Border)
Расположение (Placement)
Текст (Text)
ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области заголовка диаграммы Ганта для настройки параметров оформления. В области заголовка может содержаться название диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ДиаграммаГанта, свойство ОбластьЗаголовка

Использование в версии:

Доступен, начиная с версии 8.1.
