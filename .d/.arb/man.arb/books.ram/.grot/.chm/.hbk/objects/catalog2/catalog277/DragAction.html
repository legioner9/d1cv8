ДействиеПеретаскивания (DragAction)

Доступен, начиная с версии 8.0.

Значения

Выбор (Choice)
Копирование (Copy)
Отмена (Cancel)
Перемещение (Move)

Описание:

Указывает тип действия перетаскивания.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
