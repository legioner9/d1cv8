ЧтениеXML.ИгнорироватьПробелы (XMLReader.IgnoreWhitespace)

ЧтениеXML (XMLReader)

ИгнорироватьПробелы (IgnoreWhitespace)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак учета пробельных значений узлов (значение содержит только символы пробел, табуляция, перевод строки и т. п.).
Если значение свойства Истина, то узлы, содержащие пробельные значения, пропускаются при чтении. Если Ложь, то такие узлы не пропускаются.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение свойства можно менять при чтении XML. При этом вновь установленное значение будет использовано при очередном обращении к методу Прочитать().
Значение по умолчанию Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
