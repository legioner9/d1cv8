ЧтениеZipФайла (ZipFileReader)

Доступен, начиная с версии 8.0.

Свойства:

Комментарий (Comment)
Элементы (Items)

Методы:

Закрыть (Close)
Извлечь (Extract)
ИзвлечьВсе (ExtractAll)
Открыть (Open)

Конструкторы:

На основании имени файла
На основании потока
Формирование неинициализированного объекта

Описание:

Предназначен для чтения ZIP файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
