ЭлементПланировщика.Родитель (PlannerItem.Parent)

ЭлементПланировщика (PlannerItem)

Родитель (Parent)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: ЭлементПланировщика, Неопределено.
Если элемент является замещающим в расписании, свойство содержит оригинальный элемент, в котором задано расписание и один из повторенных элементов которого замещает данный элемент. Иначе - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЭлементПланировщика, свойство Расписание
ЭлементПланировщика, свойство ДатаЗамещения
ЭлементПланировщика, свойство ЗамещающиеЭлементы
ЭлементПланировщика, свойство Удаленный

Использование в версии:

Доступен, начиная с версии 8.3.6.
