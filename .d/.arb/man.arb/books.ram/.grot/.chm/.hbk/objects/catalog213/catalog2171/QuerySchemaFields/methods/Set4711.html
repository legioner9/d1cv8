ПоляСхемыЗапроса.Установить (QuerySchemaFields.Set)

ПоляСхемыЗапроса (QuerySchemaFields)

Установить (Set)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Установить(<Индекс>, <Выражение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс элемента коллекции.

<Выражение> (обязательный)

Тип: ВыражениеСхемыЗапроса.
Выражение, которое требуется установить.

Описание:

Устанавливает элемент коллекции по индексу. Работает аналогично оператору []. В случае ошибки установки выражения вызывает исключение, содержащее сообщение об ошибке в выражении. По переданному индексу должен находиться элемент типа ВыражениеСхемыЗапроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
