Глобальный контекст (Global context)

ЗаписатьДатуJSON (WriteJSONDate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗаписатьДатуJSON(<Дата>, <Формат>, <ВариантЗаписиДаты>)

Параметры:

<Дата> (обязательный)

Тип: Дата.
Дата, которую надо преобразовать в формат JSON.

<Формат> (обязательный)

Тип: ФорматДатыJSON.
Формат записываемой даты.

<ВариантЗаписиДаты> (необязательный)

Тип: ВариантЗаписиДатыJSON.
Указывает используемый вариант записи даты JSON.
Значение по умолчанию: ЛокальнаяДата.

Возвращаемое значение:

Тип: Строка.

Описание:

Преобразовывает дату в строку для последующей записи в формат JSON.
Если запись выполняется в формате ISO и в качестве варианта записи даты используется ЛокальнаяДатаСоСмещением, то в силу особенностей формата будут опущены секунды смещения местного времени относительно UTC.
Пример: если смещение местного времени относительно UTC составляет 02:30:17, в результате смещение будет записано в виде: +02:30.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.23.
