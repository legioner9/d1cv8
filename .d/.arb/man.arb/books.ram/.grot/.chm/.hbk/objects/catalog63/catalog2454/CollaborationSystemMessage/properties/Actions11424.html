Тип: КоллекцияДействийСообщенияСистемыВзаимодействия.
Содержит список действий, которые будут выведены в виде гиперссылок при отображении сообщения:
  • Текст ссылки – значение свойства Текст объекта типа ДействиеСообщенияСистемыВзаимодействия. Если Текст содержит пустую строку, то гиперссылка для этого элемента не отображается.
  • В обработчик, подключенный при помощи вызова ПодключитьОбработчикДействияСообщения передается значение, заданное в свойстве Значение объекта типа ДействиеСообщенияСистемыВзаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
