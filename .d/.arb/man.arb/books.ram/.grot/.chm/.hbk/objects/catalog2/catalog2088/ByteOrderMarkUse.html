ИспользованиеByteOrderMark (ByteOrderMarkUse)

Доступен, начиная с версии 8.3.6.

Значения

Авто (Auto)
Использовать (Use)
НеИспользовать (DontUse)

Описание:

Содержит варианты использования метки порядка байтов ( от англ. Byte Order Mark (BOM)) при кодировании строк.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
