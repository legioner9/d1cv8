Доступен, начиная с версии 8.1.

Элементы коллекции:

ПолеНабораДанныхМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля.
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция полей набора данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
