СвязиНаборовДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSetLinks)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СвязьНаборовДанныхМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются связи.
Возможно обращение к связи посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Описание связей наборов данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МакетКомпоновкиДанных, свойство СвязиНаборовДанных

Использование в версии:

Доступен, начиная с версии 8.1.
