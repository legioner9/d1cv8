ЧтениеУзловDOM.ПервоеОбъявление (DOMNodeReader.FirstDeclaration)

ЧтениеУзловDOM (DOMNodeReader)

ПервоеОбъявление (FirstDeclaration)

Доступен, начиная с версии 8.1.

Синтаксис:

ПервоеОбъявление()

Возвращаемое значение:

Тип: Булево.
Истина - позиционирование на объявление прошло удачно, Ложь - в противном случае.

Описание:

Осуществляет переход к первому определению DTD.
В случае, если текущий узел является ОпределениеТипаДокумента (DTD), текущий узел позиционируется на первое объявление DTD.
Вызов метода ЧтениеУзловDOM.Прочитать прерывает процесс чтения DTD и позиционирует текущий узел на следующий после DTD.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
