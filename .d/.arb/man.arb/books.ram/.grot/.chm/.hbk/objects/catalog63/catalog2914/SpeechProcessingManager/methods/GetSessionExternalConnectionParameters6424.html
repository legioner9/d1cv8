МенеджерРаботыСРечью.ПолучитьПараметрыВнешнегоПодключенияСеанса (SpeechProcessingManager.GetSessionExternalConnectionParameters)

МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьПараметрыВнешнегоПодключенияСеанса (GetSessionExternalConnectionParameters)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьПараметрыВнешнегоПодключенияСеанса()

Возвращаемое значение:

Тип: ПараметрыВнешнегоПодключенияРаботыСРечью.
Неопределено, если параметры не установлены.

Описание:

Получает параметры доступа к облаку работы с речью для текущего сеанса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
