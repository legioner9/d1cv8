РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: Число, РегистрРасчетаЗапись.<Имя регистра расчета>.
Индекс удаляемой записи либо сама запись.

Описание:

Удаляет запись из набора по указанному индексу записи или объекту записи регистра.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
