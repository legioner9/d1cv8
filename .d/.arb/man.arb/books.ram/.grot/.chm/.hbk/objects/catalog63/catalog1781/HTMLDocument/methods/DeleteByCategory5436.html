ДокументHTML.УдалитьПоКатегории (HTMLDocument.DeleteByCategory)

ДокументHTML (HTMLDocument)

УдалитьПоКатегории (DeleteByCategory)

Доступен, начиная с версии 8.3.13.

Синтаксис:

УдалитьПоКатегории(<КатегорияСодержимого>)

Параметры:

<КатегорияСодержимого> (обязательный)

Тип: КатегорияСодержимогоHTML, Массив.
Параметр определяет категорию элементов или массив элементов типа КатегорияСодержимогоHTML, которые будут удалены из документа.
Если передан массив, то из дерева документа будут удалены все элементы, которые удовлетворяют хотя бы одному из переданных типов элементов.
Примеры:
  • Если в параметре передать "КатегорияСодержимогоHTML.JavascriptТеги", то из дерева будут удалены все элементы вида <script>…</script>.
  • Если передан [КатегорияСодержимогоHTML.JavascriptТеги, КатегорияСодержимогоHTML.IframeТеги] будут удалены элементы вида <script>…</script> и элементы вида <iframe>…</iframe>.

Описание:

Очищает HTML-документ от возможных вставок на языкеJavaScript. В процессе очистки удаляются узлы из DOM-дерева документа.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
