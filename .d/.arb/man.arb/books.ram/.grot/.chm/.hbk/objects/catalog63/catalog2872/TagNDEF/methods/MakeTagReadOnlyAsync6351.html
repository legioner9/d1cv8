МеткаNDEF.ЗаблокироватьЗаписьНаМеткуАсинх (TagNDEF.MakeTagReadOnlyAsync)

МеткаNDEF (TagNDEF)

ЗаблокироватьЗаписьНаМеткуАсинх (MakeTagReadOnlyAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗаблокироватьЗаписьНаМеткуАсинх()

Описание:

Блокирует запись физической метки.
После вызова данного метода физическая метка НЕОБРАТИМО становится неперезаписываемой: операции чтения будут доступны, а операции записи будут недоступны. Если произошла ошибка, то будет сгенерировано исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерМетокNDEF, метод ПоддерживаетсяЗаписьНаМетку

Использование в версии:

Доступен, начиная с версии 8.3.23.
