ПоложениеШкалыДиаграммы (ChartScaleLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Положение выбирается автоматически:
  • для шкалы значений - Край,
  • для шкалы точек в точечной и пузырьковой диаграммах - Край,
  • во всех остальных случаях - БазовоеЗначение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
