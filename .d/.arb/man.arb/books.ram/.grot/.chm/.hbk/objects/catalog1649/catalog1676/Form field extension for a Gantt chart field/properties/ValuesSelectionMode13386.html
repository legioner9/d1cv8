Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

РежимВыделенияЗначений (ValuesSelectionMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: РежимВыделенияЗначенийДиаграммыГанта.
Определяет режим выделения значений в диаграмме. Значение по умолчанию - Авто.
Если установлено значение Нет, то свойства РежимВыделения таблицы принимает значение Одиночный.
Если установлено значение Авто, то свойство РежимВыделения в режиме совместимости с Версия8_3_22 и ниже устанавливается значение Одиночный. В остальных случаях устанавливается значение {ссылка:Атрибуты; 13383; Множественный}.
Если данное свойство принимает значение Множественный, то свойство РежимВыделения таблицы также принимает значение Множественный, и наоборот.
Если данное свойство принимает значение Одиночный, то и свойство РежимВыделения таблицы также принимает значение Одиночный, и наоборот.
Если установлено значение Нет, то свойство РежимВыделения таблицы принимает значение Одиночный.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
