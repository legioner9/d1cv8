ЧтениеДанных (DataReader)

ПрочитатьБайт (ReadByte)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьБайт()

Возвращаемое значение:

Тип: Число.
Неопределено, если не удалось прочитать байт поскольку был достигнут конец потока.

Описание:

Выполняет чтение одного байта из потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ЧтениеДанных, метод НачатьЧтениеБайта
ЧтениеДанных, метод ПрочитатьБайтАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
