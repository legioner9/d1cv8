Диапазон.Начало (Range.Begin)

Диапазон (Range)

Начало (Begin)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Дата, Число, Неопределено.
Нижняя граница диапазона. Значение нижней границы диапазона должно быть не больше значения верхней границы. Неопределено - диапазон не ограничен снизу.
Тип значения должен совпадать с типом значения верхней границы.
Если свойство Конец имеет тип значения, отличный от Неопределено, то данное свойство должно иметь такой же тип значения или Неопределено.
Если свойство Конец имеет тип значения Неопределено, то данное свойство не может иметь такой же тип значения.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
