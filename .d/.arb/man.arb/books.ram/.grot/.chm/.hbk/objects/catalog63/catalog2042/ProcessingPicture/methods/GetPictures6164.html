ОбрабатываемаяКартинка (ProcessingPicture)

ПолучитьКартинки (GetPictures)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьКартинки()

Описание:

Возвращает массив картинок с кадрами. Если картинка содержит 1 кадр, возвращает массив с 1 элементом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

На мобильной платформе поддерживаются только форматы JPEG и PNG. Если в результате получается картинка, не поддерживаемого формата, будет выдано сообщение об ошибке.

Использование в версии:

Доступен, начиная с версии 8.3.15.
