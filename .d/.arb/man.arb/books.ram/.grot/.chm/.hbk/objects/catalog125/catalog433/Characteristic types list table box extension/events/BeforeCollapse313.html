Расширение табличного поля списка видов характеристик.ПередСворачиванием (Characteristic types list table box extension.BeforeCollapse)

Расширение табличного поля списка видов характеристик (Characteristic types list table box extension)

ПередСворачиванием (BeforeCollapse)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСворачиванием(<Строка>, <Отказ>)

Параметры:

<Строка>

Строка табличного поля.

<Отказ>

Тип: Булево.
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение Истина, то стандартной обработки события не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает перед закрытием узла табличного поля, отображаемого в виде дерева.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
