МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

НачатьПроверку (BeginVerification)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьПроверку(<Способ>, <СообщениеПользователю>, <ОбработчикРезультатовПроверки>)

Параметры:

<Способ> (обязательный)

Тип: СпособДополнительнойПроверкиПользователя.
Способ проверки пользователя.

<СообщениеПользователю> (обязательный)

Тип: Строка.
Текст сообщения, отображаемый пользователю при проверке доступа.

<ОбработчикРезультатовПроверки> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после выполнения проверки пользователя со следующими параметрами:
  • <ОтмененоПользователем> - тип Булево. Признак того, что проверка пользователя была отменена пользователем.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
При возникновении ошибки вызвается обработчик ошибки.

Описание:

Начинает проверку дополнительную проверку пользователя указанным способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерДополнительнойПроверкиПользователя, метод ПроверитьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
