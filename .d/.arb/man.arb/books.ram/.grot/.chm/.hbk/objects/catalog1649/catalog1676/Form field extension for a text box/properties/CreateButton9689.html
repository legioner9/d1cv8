Расширение поля формы для поля ввода (Form field extension for a text box)

КнопкаСоздания (CreateButton)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Содержит режим отображения кнопки "Создание" в выпадающем списке поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Если значение Неопределено, то видимость кнопки определяет значение свойства СозданиеПриВводе реквизита.

Использование в версии:

Доступен, начиная с версии 8.3.3.
