ТипПодписчикаДоставляемыхУведомлений (DeliverableNotificationSubscriberType)

Доступен, начиная с версии 8.3.6.

Значения

APNS (APNS)
FCM (FCM)
GCM (GCM)
HPK (HPK)
WNS (WNS)

Описание:

Определяет сервис доставки уведомлений, используемый данным подписчиком.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.24.
