Описание:

Тип: ОбъектМетаданных: Измерение.
Объект метаданных, описывающий измерение регистра сведений. Этот регистр сведений хранит информацию о соответствии исполнителей всем остальным реквизитам адресации задач.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Описывает реквизиты адресации задачи.

Использование в версии:

Доступен, начиная с версии 8.1.
