Форма.Стиль (Form.Style)

Форма (Form)

Стиль (Style)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Стиль.
Определят стиль отображения формы. Может быть установлено в одно из значений коллекции стилей глобального контекста или в значение "Авто".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
