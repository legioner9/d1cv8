ИнформацияОПроблемеОтправкиДоставляемогоУведомления (DeliverableNotificationSendingIssueInformation)

Доступен, начиная с версии 8.3.11.

Свойства:

Описание (Description)
Получатели (Recipients)
ТипОшибки (ErrorType)
Уведомление (Notification)

Описание:

Объект содержит информацию о проблеме, возникшей при запросе на отправку конкретного уведомления конкретному получателю или получателям.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
