РегистрРасчетаМенеджер.<Имя регистра расчета> (CalculationRegisterManager.<Calculation register name>)

СоздатьНаборЗаписей (CreateRecordSet)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьНаборЗаписей()

Возвращаемое значение:

Тип: РегистрРасчетаНаборЗаписей.<Имя регистра расчета>.

Описание:

Создает набор записей регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Набор записей создается пустым.

Пример:

// Выполнение перерасчета по заданному списку физлиц

// основные начисления

РегНачисления = РегистрыРасчета.ОсновныеНачисленияСотрудников;
НаборЗаписей = РегНачисления.СоздатьНаборЗаписей();
// установим Отбор

НаборЗаписей.Отбор.Регистратор.Значение = Ссылка;
НаборЗаписей.Отбор.ФизЛицо.Использование = Истина;
Для Каждого ФизЛицо из ФизЛица Цикл
    НаборЗаписей.Отбор.ФизЛицо.Значение = ФизЛицо;
    НаборЗаписей.Прочитать();
    // расчет записей

    РассчитатьЗаписиРегистра(НаборЗаписей); 
    НаборЗаписей.Записать(Истина, Истина);
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
