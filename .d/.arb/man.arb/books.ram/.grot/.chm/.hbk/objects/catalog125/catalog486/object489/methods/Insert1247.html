РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс в наборе, куда необходимо вставить новую запись.

Возвращаемое значение:

Тип: РегистрРасчетаЗапись.<Имя регистра расчета>.
Созданная запись регистра.

Описание:

Вставляет новую запись в набор на указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
