HTTPЗапрос.ПолучитьТелоКакДвоичныеДанные (HTTPRequest.GetBodyAsBinaryData)

HTTPЗапрос (HTTPRequest)

ПолучитьТелоКакДвоичныеДанные (GetBodyAsBinaryData)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьТелоКакДвоичныеДанные()

Возвращаемое значение:

Тип: ДвоичныеДанные, Неопределено.
Неопределено - тело не было установлено или был установлен файл тела.

Описание:

Получает тело запроса как ДвоичныеДанные.

В веб-клиенте в браузерах Internet Explorer и Microsoft Edge (до версии 18) метод не поддерживается - возвращает значение Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.22.
