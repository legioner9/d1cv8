УсловноеОформление.Индекс (ConditionalAppearance.IndexOf)

УсловноеОформление (ConditionalAppearance)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Оформление>)

Параметры:

<Оформление> (обязательный)

Тип: ЭлементУсловногоОформления.
Искомое оформление.

Возвращаемое значение:

Индекс элемента коллекции, или -1, в случае, если элемент отсутствует в коллекции.

Описание:

Получить индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
