АдминистрированиеКластер.ПолучитьСоединения (AdministrationCluster.GetConnections)

АдминистрированиеКластер (AdministrationCluster)

ПолучитьСоединения (GetConnections)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСоединения()

Возвращаемое значение:

Тип: Массив.
Массив элементов типа АдминистрированиеСоединение.
Для каждого соединения выдается только информация, не требующая аутентификации пользователей информационных баз.

Описание:

Получает массив описаний соединений кластера. Выполнение метода требует аутентификации одного из администраторов кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
