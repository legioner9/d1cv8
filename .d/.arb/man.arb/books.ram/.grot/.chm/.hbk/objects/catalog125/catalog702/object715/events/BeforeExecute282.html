ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередВыполнением (BusinessProcessRoutePointRef.<Business process name>.BeforeExecute)

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередВыполнением (BeforeExecute)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса вида Действие, в которой производится проверка выполнения.

<Задача>

Тип: ЗадачаСсылка.<Имя задачи>.
Выполняемая задача.

<Отказ>

Тип: Булево.
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то выполнение задачи произведено не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает при вызове метода ВыполнитьЗадачу у задачи и после обработки задачей события ПередВыполнением.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида Действие.
В этом обработчике не допускаются интерактивные вызовы (открытие форм, предупреждений).

Использование в версии:

Доступен, начиная с версии 8.0.
