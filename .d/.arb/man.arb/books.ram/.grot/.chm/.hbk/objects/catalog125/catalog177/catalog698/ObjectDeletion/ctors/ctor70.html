УдалениеОбъекта (ObjectDeletion)

На основании ссылки

Доступен, начиная с версии 8.0.

Синтаксис:

Новый УдалениеОбъекта(<Ссылка>)

Параметры:

<Ссылка> (обязательный)

Ссылка на объект базы данных, который должен быть удален.

Описание:

Создает экземпляр объекта с заданным значением ссылки на объект базы данных.
Также может быть создан с использованием функциональной формы оператора Новый.

Пример:

Массив = Новый Массив;
Массив.Добавить(Справочники.Валюты.НайтиПоКоду("999"));
УдалениеОбъекта = Новый (Тип("УдалениеОбъекта"),Массив);
УдалениеОбъекта.Записать();

Использование в версии:

Доступен, начиная с версии 8.0.
