ТипМерыРасстоянияАнализаДанных (DataAnalysisDistanceMetricType)

МетрикаГорода (CityBlock)

Доступен, начиная с версии 8.0.

Описание:

Расстояние между двумя объектами вычисляется по формуле:
D = sum (Wi * | Xi - Yi | ), где Xi, Yi - значения атрибутов объектов, Wi - весовой коэффициент. i - номер атрибута, от 1 до n, где n - число атрибутов.

Использование в версии:

Доступен, начиная с версии 8.0.
