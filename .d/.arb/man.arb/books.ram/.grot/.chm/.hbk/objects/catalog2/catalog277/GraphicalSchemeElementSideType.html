ТипСтороныЭлементаГрафическойСхемы (GraphicalSchemeElementSideType)

Доступен, начиная с версии 8.1.

Значения

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)
Центр (Center)

Описание:

Содержит варианты сторон, к которой может быть присоединена линия.

Доступность:

Сервер, толстый клиент, внешнее соединение.
