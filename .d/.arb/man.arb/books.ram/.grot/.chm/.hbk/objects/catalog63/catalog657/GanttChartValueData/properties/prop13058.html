ДанныеЗначенияДиаграммыГанта (GanttChartValueData)

<Имя колонки> (<Column name>)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Набор свойств содержит значения колонок для данного значения диаграммы Ганта. Доступ к значению осуществляется по имени колонки. Имена свойств совпадают с именами колонок.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
