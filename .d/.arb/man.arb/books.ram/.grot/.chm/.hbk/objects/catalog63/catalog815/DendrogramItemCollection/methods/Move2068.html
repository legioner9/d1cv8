ЭлементыДендрограммы.Сдвинуть (DendrogramItemCollection.Move)

ЭлементыДендрограммы (DendrogramItemCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: ЭлементДендрограммы.
Сдвигаемый элемент.

<Смещение> (обязательный)

Тип: Число.
Количество точек, на которое необходимо переместить указанную. Положительное значение означает, что точка будет передвинута ближе к концу диаграммы (вниз), отрицательное - ближе к началу (вверх).

Описание:

Сдвигает элемент коллекции на заданное количество позиций вперед или назад.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
