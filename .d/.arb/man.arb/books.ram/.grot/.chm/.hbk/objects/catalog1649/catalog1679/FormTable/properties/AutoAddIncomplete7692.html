ТаблицаФормы (FormTable)

АвтоВводНезаполненного (AutoAddIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Содержит признак автоматического ввода новой строки.
Истина - при активации таблицы (мышкой или кнопкой TAB), если в поле нет ни одной строки, происходит ввод новой строки.
Если значение свойства Неопределено (в Конфигураторе выбрано значение Авто), то при автоматическом определении значения, помимо значения свойства АвтоОтметкаНезаполненного, используется также значение свойства ОтметкаНезаполненного.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
