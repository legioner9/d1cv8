МакетОбластиКомпоновкиДанных (DataCompositionAreaTemplate)

МакетОбластиКомпоновкиДанных (DataCompositionAreaTemplate)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СтрокаТаблицыОбластиКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы макета области.
Возможно обращение к элемент макета области посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Данный объект представляет собой коллекцию объектов типа СтрокаТаблицыОбластиКомпоновкиДанных и содержит описание выводимой области компоновки данных (выводимые поля, цвета, шрифты и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: AreaTemplate.
