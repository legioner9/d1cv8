Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаЭлементеИзмерения (DimensionItemClick)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НажатиеНаЭлементеИзмерения(<ЭлементИзмерения>, <ЗначенияИзмерений>, <СтандартнаяОбработка>)

Параметры:

<ЭлементИзмерения>

Тип: ЭлементИзмеренияПланировщика.
Элемент измерения, на который было произведено нажатие.

<ЗначенияИзмерений>

Тип: ФиксированноеСоответствие.
Фиксированное соответствие, однозначно определяющее заголовок измерения, на котором было произведено нажатие.

<СтандартнаяОбработка>

Тип: Булево.
Признак выполнения стандартной обработки.
Если Истина, то показывается значение элемента измерения.

Описание:

Возникает при нажатии левой кнопкой мыши на элементе измерения в случае, если значение свойства поля планировщика ГиперссылкаЭлементаИзмерения имеет значение Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
