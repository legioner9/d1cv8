ЗначениеСерииСлояГеографическойСхемы (GeographicalSchemaLayerSeriesValue)

Доступен, начиная с версии 8.1.

Свойства:

Значение (Value)
Объект (Object)
Подсказка (ToolTip)
Расшифровка (Details)
Серия (Series)

Описание:

Предоставляет доступ к свойствам значения серии слоя географической схемы. Значение серии слоя характеризует некий топологический объект.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Получим значение объекта для серии названия

ЗначениеСерии = ОбъектКарты.Слой.ПолучитьЗначение(ОбъектКарты, 
                        СерияНазваний);
// Если есть такое значение

Если Не ЗначениеСерии = Неопределено Тогда
    // Получим реальное название

    НазваниеМестоположения = ЗначениеСерии.Значение;
КонецЕсли;

См. также:

СлойГеографическойСхемы, метод ПолучитьЗначение

Использование в версии:

Доступен, начиная с версии 8.1.
