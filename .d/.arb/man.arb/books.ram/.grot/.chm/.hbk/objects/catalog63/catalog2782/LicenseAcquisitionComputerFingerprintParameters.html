ПараметрыПривязкиККомпьютеруПолученияЛицензий (LicenseAcquisitionComputerFingerprintParameters)

Доступен, начиная с версии 8.3.20.

Свойства:

ИмяКомпьютера (ComputerName)
ОперативнаяПамять (RAM)
ОС (OS)
Представление (Presentation)
Процессор (Processor)
СписокОборудования (EquipmentList)

Описание:

Данные о привязке программной лицензии на платформу к оборудованию компьютера. Может быть получен только с помощью метода ПолучитьПараметрыПривязкиККомпьютеру.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
Сериализуется.
