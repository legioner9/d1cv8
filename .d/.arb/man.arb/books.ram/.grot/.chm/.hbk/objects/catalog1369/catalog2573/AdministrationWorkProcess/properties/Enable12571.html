АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Резервный (Enable)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: Булево.
Определяет, является ли рабочий процесс резервным:
  • Истина - процесс является резервным.
  • Ложь - процесс не является резервным.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
