Описание:

Тип: ОписаниеТипов.
Позволяет задать набор типов, доступных для выбора. Так же задается ограничение для квалификаторов примитивных типов Число, Строка, Дата.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
