РежимОткрытияФайла (FileOpenMode)

Создать (Create)

Доступен, начиная с версии 8.3.9.

Описание:

Создание нового файла. Если указанный файл существует, данные в нем будут полностью перезаписаны.

Использование в версии:

Доступен, начиная с версии 8.3.9.
