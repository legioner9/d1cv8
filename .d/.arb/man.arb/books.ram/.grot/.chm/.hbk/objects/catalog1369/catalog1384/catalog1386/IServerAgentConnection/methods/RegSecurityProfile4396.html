Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfile (RegSecurityProfile)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfile(<Кластер>, <Профиль>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, в котором должен быть зарегистрирован новый профиль безопасности.

<Профиль> (обязательный)

Тип: Профиль безопасности.
Регистрируемый профиль безопасности.

Описание:

Выполняет регистрацию нового профиля безопасности кластера или обновление существующего. Изменения профиля безопасности начинают действовать после отсоединения от информационной базы последнего соединения.

Доступность:

Интеграция.

Примечание:

Если профиль безопасности кластера существует, то выполняется обновление, в противном случае - добавление.

Использование в версии:

Доступен, начиная с версии 8.3.3.
