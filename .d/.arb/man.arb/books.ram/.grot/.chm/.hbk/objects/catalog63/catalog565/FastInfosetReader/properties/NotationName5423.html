ЧтениеFastInfoset.ИмяНотации (FastInfosetReader.NotationName)

ЧтениеFastInfoset (FastInfosetReader)

ИмяНотации (NotationName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Содержит имя нотации для узлов типа Сущность и СсылкаНаСущность, определяющих внешнюю неразбираемую сущность (external unparsed entity).
Для типов узлов, для которых идентификатор не определен, содержит пустую строку.
Если имя нотации не пустая строка и текущий узел Сущность или СсылкаНаСущность, то это означает, что сущность (ссылка на сущность) неразбираемая (external unparsed entity) и также должны быть непустыми ЧтениеFastInfoset.ПубличныйИдентификатор или ЧтениеFastInfoset.СистемныйИдентификатор.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
