ТабличныйДокумент (SpreadsheetDocument)

НачатьАвтогруппировкуКолонок (StartColumnAutoGrouping)

Доступен, начиная с версии 8.0.

Синтаксис:

НачатьАвтогруппировкуКолонок()

Описание:

Начинает автоматическую группировку колонок. При автоматической группировке учитываются значения уровней, заданные при выводе колонок методом Присоединить.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ТабличныйДокумент, метод ЗакончитьАвтогруппировкуКолонок

Использование в версии:

Доступен, начиная с версии 8.0.
