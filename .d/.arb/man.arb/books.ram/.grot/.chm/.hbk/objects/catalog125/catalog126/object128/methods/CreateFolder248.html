СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

СоздатьГруппу (CreateFolder)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьГруппу()

Возвращаемое значение:

Тип: СправочникОбъект.<Имя справочника>.

Описание:

Создает новую группу справочника.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

НовыйОбъект = Справочники["Контрагенты"].СоздатьГруппу();
НовыйОбъект.Наименование = "Покупатели";
НовыйОбъект.Записать();

Использование в версии:

Доступен, начиная с версии 8.0.
