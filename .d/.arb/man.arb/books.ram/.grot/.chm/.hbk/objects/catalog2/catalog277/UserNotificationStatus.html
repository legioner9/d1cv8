СтатусОповещенияПользователя (UserNotificationStatus)

Доступен, начиная с версии 8.3.10.

Значения

Важное (Important)
Информация (Information)

Описание:

Определяет набор статусов сообщений, которые влияют на поведение оповещения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Глобальный контекст, метод ПоказатьОповещениеПользователя

Доступен, начиная с версии 8.3.10.
