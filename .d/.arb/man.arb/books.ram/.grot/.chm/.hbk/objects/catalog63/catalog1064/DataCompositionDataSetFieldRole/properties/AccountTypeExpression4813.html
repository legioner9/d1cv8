РольПоляНабораДанныхКомпоновкиДанных (DataCompositionDataSetFieldRole)

ВыражениеВидаСчета (AccountTypeExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Выражение, при помощи которого будет определяться тип счета. Результатом выражения должно быть значение типа ВидСчета, либо Число. Значению числа 0 соответствует активный счет, 1 – пассивный, 2 – активно пассивный. Выражение записывается на языке выражений системы компоновки данных. Данная информация используется при расчете по полям бухгалтерского остатка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
