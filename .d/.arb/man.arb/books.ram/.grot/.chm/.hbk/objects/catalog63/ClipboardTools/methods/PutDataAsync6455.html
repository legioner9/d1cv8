СредстваБуфераОбмена.ПоместитьДанныеАсинх (ClipboardTools.PutDataAsync)

СредстваБуфераОбмена (ClipboardTools)

ПоместитьДанныеАсинх (PutDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПоместитьДанныеАсинх(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: ЭлементБуфераОбмена, Массив.
Элемент буфера обмена или массив элементов типа ЭлементБуфераОбмена.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Булево.

Описание:

Асинхронная функция. Помещает переданные данные в буфер обмена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

При помещении в буфер обмена данных формата HTML часть оформления HTML-текста может быть утеряна.

В веб-клиенте:
  • может появиться запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается Обещание, результат которого – Булево.
  • значение параметра <Формат> типа Строка должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Использование в версии:

Доступен, начиная с версии 8.3.24.
