НастройкаОтбора.Добавить (FilterSettings.Add)

НастройкаОтбора (FilterSettings)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя элемента отбора.

<Доступность> (обязательный)

Тип: Булево.
Определяет доступность элемента для редактирования.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: ЭлементУправленияОтбором.

Описание:

Добавляет элемент управления отбором в коллекцию.
Внимание! Метод не доступен для настройки отбора динамических списков (свойство НастройкаОтбора объектов СправочникСписок.<Имя справочника>, ДокументСписок.<Имя документа> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
