ВидСравненияКомпоновкиДанных (DataCompositionComparisonType)

ВИерархии (InHierarchy)

Доступен, начиная с версии 8.1.

Описание:

Проверка того, что сравниваемое значение принадлежит иерархии заданного значения. Имеет смысл при сравнении для иерархических объектов.

Использование в версии:

Доступен, начиная с версии 8.1.
