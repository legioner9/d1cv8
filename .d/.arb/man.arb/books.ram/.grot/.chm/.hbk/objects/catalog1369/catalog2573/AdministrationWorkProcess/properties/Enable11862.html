АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Включен (Enable)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.
Устанавливается кластером при необходимости запустить или остановить рабочий процесс:
  • Истина - процесс должен быть запущен и будет запущен при возможности.
  • Ложь - процесс должен быть остановлен и будет остановлен после отсоединения всех пользователей или после истечения времени, заданного свойством ВремяПринудительногоЗавершения объекта АдминистрированиеКластер.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
