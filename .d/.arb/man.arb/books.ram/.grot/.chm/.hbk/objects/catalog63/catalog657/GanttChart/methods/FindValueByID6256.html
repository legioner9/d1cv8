ДиаграммаГанта.НайтиЗначениеПоИдентификатору (GanttChart.FindValueByID)

ДиаграммаГанта (GanttChart)

НайтиЗначениеПоИдентификатору (FindValueByID)

Доступен, начиная с версии 8.3.21.

Синтаксис:

НайтиЗначениеПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: ИдентификаторЗначенияДиаграммыГанта.
Идентификатор искомого значения.

Возвращаемое значение:

Тип: ЗначениеДиаграммыГанта, Неопределено.
Если значение не найдено, возвращает Неопределено.

Описание:

Осуществляет поиск значения в диаграмме по идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.21.
