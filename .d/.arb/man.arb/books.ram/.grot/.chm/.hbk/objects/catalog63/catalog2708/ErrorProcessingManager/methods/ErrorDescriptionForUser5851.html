МенеджерОбработкиОшибок.ПредставлениеОшибкиДляПользователя (ErrorProcessingManager.ErrorDescriptionForUser)

МенеджерОбработкиОшибок (ErrorProcessingManager)

ПредставлениеОшибкиДляПользователя (ErrorDescriptionForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПредставлениеОшибкиДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: ИнформацияОбОшибке.
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: ФорматированнаяСтрока.

Описание:

Возвращает текст ошибки, понятный пользователю.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МенеджерОбработкиОшибок, метод КраткоеПредставлениеОшибки
МенеджерОбработкиОшибок, метод ПодробноеПредставлениеОшибки

Использование в версии:

Доступен, начиная с версии 8.3.17.
