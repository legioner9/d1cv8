КолонкиСписка.Установить (ListColumns.Set)

КолонкиСписка (ListColumns)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Колонки>)

Параметры:

<Колонки> (обязательный)

Тип: Строка.
Имена устанавливаемых колонок, разделенные запятыми. Зависят от прикладного объекта.

Описание:

Устанавливает новый набор колонок списка, который замещает старый набор колонок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Установим новые колонки в списке справочника,

// связанном с табличным полем:

Список.Колонки.Установить("Код, Наименование");

Использование в версии:

Доступен, начиная с версии 8.0.
