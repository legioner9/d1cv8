Расширение тестируемого поля формы для поля надписи (Tested form field extension for a label field)

Нажать (Click)

Доступен, начиная с версии 8.3.2.

Синтаксис:

Нажать()

Описание:

Эмулирует нажатие для поля ввода в режиме отображения надписи гиперссылкой.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.2.
