ПоложениеКартинкиКнопки (ButtonPictureLocation)

Доступен, начиная с версии 8.0.

Значения

Лево (Left)
Право (Right)

Описание:

Содержит варианты положений картинки кнопки относительно текста. Используется при определении свойства ПоложениеКартинки для элемента управления Кнопка.

Доступность:

Толстый клиент.

См. также:

Кнопка, свойство ПоложениеКартинки

Использование в версии:

Доступен, начиная с версии 8.0.
