НеобходимостьЗавершенияСоединения (ConnectionStopRequest)

ПринудительноеЗавершение (DisconnectionTime)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Дата.
Момент времени, когда рабочий процесс текущего соединения будет завершен принудительно. По достижении этого времени работа текущего соединения завершится аварийно.
Нулевая дата (1/1/0001 0:0:0), если принудительное завершение рабочего процесса не планируется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В файловом варианте всегда равно нулевой дате.

Использование в версии:

Доступен, начиная с версии 8.1.
