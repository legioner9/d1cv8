ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (необязательный)

Тип: Произвольный.
Значение, на основании которого выполняется заполнение.

Описание:

Заполняет элемент данными из какого-либо другого объекта. При этом в модуле плана видов характеристик вызывается процедура ОбработкаЗаполнения. В ней может размещаться алгоритм, заполняющий данные элемента из переданного в параметре значения.
Для заполнения может использоваться значение любого типа. Разумеется, для этого типа в процедуре ОбработкаЗаполнения должен быть описан алгоритм заполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
