ГруппаКоманд (CommandGroup)

Доступен, начиная с версии 8.2.

Методы:

Получить (Get)

Описание:

Содержит ссылку (внутренний идентификатор) на предопределенную или пользовательскую группу команд, созданную в конфигурации.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ОбъектМетаданных: Команда, свойство Группа
ОбъектМетаданных: ОбщаяКоманда, свойство Группа

Использование в версии:

Доступен, начиная с версии 8.2.
