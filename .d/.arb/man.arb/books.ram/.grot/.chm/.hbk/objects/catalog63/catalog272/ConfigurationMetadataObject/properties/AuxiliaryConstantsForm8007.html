ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ДополнительнаяФормаКонстант (AuxiliaryConstantsForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Неопределено.
Описывает форму, которая будет вызвана системой для редактирования констант в случае, если основная форма не установлена или не может быть использована в конкретном варианте использования 1С:Предприятия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
