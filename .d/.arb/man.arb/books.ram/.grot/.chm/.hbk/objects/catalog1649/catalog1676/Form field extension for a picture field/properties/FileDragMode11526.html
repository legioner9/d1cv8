Расширение поля формы для поля картинки (Form field extension for a picture field)

СпособПеретаскиванияФайлов (FileDragMode)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: СпособПеретаскиванияФайлов.
Указывает способ перетаскивания файлов. Если установлен способ КакФайл, то при перетаскивании файла (файлов) на элемент управления в качестве значения параметра перетаскивания будет передан объект типа Файл (массив объектов типа Файл), а если установлен способ КакСсылкаНаФайл – объект типа СсылкаНаФайл (массив объектов типа СсылкаНаФайл).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В веб-клиенте при перетаскивании файла (файлов) как СсылкаНаФайл обработчик события ПроверкаПеретаскивания у элементов управления вызывается один раз в момент отпускания кнопки мыши перед обработчиком события Перетаскивание в отличие от тонкого клиента, в котором обработчик события ПроверкаПеретаскивания вызывается в процессе перетаскивания файла (файлов).
В веб-клиенте, если установлен способ перетаскивания файлов КакФайл, то перетаскивание файлов игнорируется, и обработчики событий ПроверкаПеретаскивания и Перетаскивание не вызываются.

Использование в версии:

Доступен, начиная с версии 8.3.12.
