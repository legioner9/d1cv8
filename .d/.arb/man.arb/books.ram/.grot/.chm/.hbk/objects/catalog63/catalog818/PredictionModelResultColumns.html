КолонкиРезультатаМоделиПрогноза (PredictionModelResultColumns)

Доступен, начиная с версии 8.0.

Элементы коллекции:

КолонкаРезультатаМоделиПрогноза
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.
Возможно обращение к колонкам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)
Установить (Set)

Описание:

Содержит колонки, выводимые в результат модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.

МодельПрогнозаДеревоРешений, свойство КолонкиРезультата

Использование в версии:

Доступен, начиная с версии 8.0.
