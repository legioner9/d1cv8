РежимИспользованияТабличныхПространствБазыДанных (DatabaseTablespacesUseMode)

Доступен, начиная с версии 8.3.23 (в режиме совместимости с версией 8.3.1 и последующими).

Использование:

Только чтение.

Описание:

Тип: РежимИспользованияТабличныхПространствБазыДанных.
Задает режим использования табличных пространств базы данных.
В режиме совместимости конфигурации Версия8_3_22 и ниже устанавливается в значение НеИспользовать.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23 (в режиме совместимости с версией 8.3.1 и последующими).
