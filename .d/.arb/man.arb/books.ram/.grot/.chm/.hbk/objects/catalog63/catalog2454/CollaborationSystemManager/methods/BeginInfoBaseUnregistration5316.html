МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьОтменуРегистрацииИнформационнойБазы (BeginInfoBaseUnregistration)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьОтменуРегистрацииИнформационнойБазы(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Отменяет регистрацию информационной базы на сервере взаимодействия.
При выполнении данного метода удаляются криптографические ключи, но сохраняется идентификатор информационной базы в сервисе. При повторной регистрации информационной базы с использованием ранее указанного адреса электронной почты, данные системы взаимодействия будут восстановлены.
Требуется наличие у текущего пользователя информационной базы права "РегистрацияИнформационнойБазыСистемыВзаимодействия".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
