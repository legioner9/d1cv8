ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

Начало (Begin)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Начало интервала.
Допускаются следующие типы значений:
  • для информационного интервала в значениях серий диаграммы (свойство диаграммы ИнформационныеИнтервалыЗначений) - Число;
  • для информационного интервала в значениях точек диаграммы (свойство диаграммы ИнформационныеИнтервалыТочек):
    • если на оси точек откладываются точки - ТочкаДиаграммы;
    • если на оси точек откладываются числа (например, в точечной и пузырьковой диаграммах) - Число;
    • если на оси точек откладываются даты (например, в точечной и пузырьковой диаграмме) - Дата;
  • для информационного интервала, расположенного в параметрах вывода компоновки данных:
    • ПолеКомпоновкиДанных;
    • для информационного интервала значений - Число;
    • для информационного интервала точек - тип, установленный в свойстве Точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
