Определяет набор типов поведения формы при нажатии клавиши Enter. Используется для определения свойства ПоведениеКлавишиEnter.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
