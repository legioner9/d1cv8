WSПрокси (WSProxy)

<Имя метода> (<Method name>)

Доступен, начиная с версии 8.1.

Синтаксис:

<Имя метода>()

Описание:

Доступ к операциям веб-сервиса. Имя, параметры и возвращаемое значение метода должны соответствовать операции, описанной в интерфейсе веб-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
