СредстваГеопозиционирования (LocationTools)

ПолучитьОтслеживаемуюГеозону (GetMonitoredGeofence)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьОтслеживаемуюГеозону(<ИдентификаторГеозоны>)

Параметры:

<ИдентификаторГеозоны> (обязательный)

Тип: Строка.
Идентификатор отслеживаемой геозоны.

Возвращаемое значение:

Тип: Геозона.
Возвращает значение Неопределено, если геозона с указанным идентификатором отсутствует.

Описание:

Получает информацию об отслеживаемой геозоне по указанному идентификатору для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
