КонстантаМенеджер.<Имя константы> (ConstantManager.<Constant Name>)

СоздатьКлючЗначения (CreateValueKey)

Доступен, начиная с версии 8.3.13.

Синтаксис:

СоздатьКлючЗначения(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (необязательный)

Тип: Структура, Неопределено.
В качестве ключа структуры используется имя общего реквизита, в качестве значения – необходимое значение этого поля.
Если параметр не задан или равен Неопределено, будет создан пустой ключ константы.

Возвращаемое значение:

Тип: КонстантаКлючЗначения.<Имя константы>.

Описание:

Создает ключ значения константы на основе переданных параметров.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

// см. пример объекта КонстантаМенеджерЗначения.<Имя константы>


Использование в версии:

Доступен, начиная с версии 8.3.13.
