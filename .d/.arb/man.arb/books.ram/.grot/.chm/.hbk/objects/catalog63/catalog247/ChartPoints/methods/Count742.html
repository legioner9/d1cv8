ТочкиДиаграммы.Количество (ChartPoints.Count)

ТочкиДиаграммы (ChartPoints)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество точек диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ВсегоТочек = Диаграмма.Точки.Количество();

Использование в версии:

Доступен, начиная с версии 8.0.
