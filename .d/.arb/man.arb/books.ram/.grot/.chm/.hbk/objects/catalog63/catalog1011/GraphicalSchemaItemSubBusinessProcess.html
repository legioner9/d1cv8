Описание:

Элемент графической схемы "Вложенный процесс". Отображает точку маршрута бизнес-процесса, в которой запускается на исполнение вложенный бизнес-процесс. Исполнение основного (родительского) процесса возобновляется только после завершения вложенного процесса.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
