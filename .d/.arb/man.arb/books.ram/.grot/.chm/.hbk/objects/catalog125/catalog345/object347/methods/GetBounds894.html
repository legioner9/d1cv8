ПоследовательностьМенеджер.<Имя последовательности> (SequenceManager.<Sequence name>)

ПолучитьГраницы (GetBounds)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьГраницы(<Разрез>, <Отбор>)

Параметры:

<Разрез> (обязательный)

Тип: Строка.
Имена измерений последовательности, по которым нужно получить границы, перечисленные через запятую.

<Отбор> (необязательный)

Тип: ТаблицаЗначений.
Набор комбинаций значений измерений, по которым надо получить границы. Задается как таблица значений, в колонках которой содержатся соответствующие значения измерений. Имена колонок должны соответствовать именам измерений последовательности. Если значения измерений не указаны (параметр опущен), то по всем значениям измерений.

Возвращаемое значение:

Тип: ТаблицаЗначений.
Таблица значений, в первой колонке которой содержится момент времени границы, а в последующих - значения измерений, в разрезе которых выбирались границы.

Описание:

Получает границы последовательности в разрезе заданных измерений последовательности и с отбором по заданным значениям измерений последовательности.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Анализируются все границы, без учета наличия или отсутствия зарегистрированных документов по данным границам.

Пример:

Отбор = Новый ТаблицаЗначений;
Отбор.Колонки.Добавить("Товар");
Разрез = "Товар";
Последовательность = Последовательности.УчетТоваров;
Границы = Последовательность.ПолучитьГраницы(Разрез, Отбор);

Использование в версии:

Доступен, начиная с версии 8.0.
