Необходимость завершения соединения

При работе с клиент-серверной информационной базой может возникнуть необходимость завершения текущего соединения с сервером 1С:Предприятия, если рабочий процесс, с которым установлено текущее соединение выключен. Выключенный рабочий процесс может быть завершен принудительно, и все его соединения завершатся аварийно. Это может стать причиной потери несохраненных данных клиентского приложения. Для предотвращения аварийного завершения клиентского приложения в конфигурации может быть предусмотрена реакция на случай выключения рабочего процесса, например, сохранение данных и перезапуск клиентского приложения.
