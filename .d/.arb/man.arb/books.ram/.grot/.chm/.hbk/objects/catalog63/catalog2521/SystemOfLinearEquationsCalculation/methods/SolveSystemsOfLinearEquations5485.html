РасчетСистемЛинейныхУравнений.РассчитатьСистемыЛинейныхУравнений (SystemOfLinearEquationsCalculation.SolveSystemsOfLinearEquations)

РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

РассчитатьСистемыЛинейныхУравнений (SolveSystemsOfLinearEquations)

Доступен, начиная с версии 8.3.14.

Синтаксис:

РассчитатьСистемыЛинейныхУравнений(<РазрешитьРедуцированиеГрафа>)

Параметры:

<РазрешитьРедуцированиеГрафа> (необязательный)

Тип: Булево.
Истина - механизм решения использует все возможные оптимизации решения.
Ложь - механизм решения разделяет граф на компоненты и начинает решать их, пропуская этап редуцирования графа.
Данный параметр не влияет на процесс выбора алгоритма решения, определяемого свойством ГраницаИзмененияАлгоритмаРешения, а только отключает дополнительные оптимизации.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: ТаблицаЗначений.
Таблица значений содержит следующие столбцы:
  • столбец с номерами переменных (НомерУзла),
  • К столбцов с решениями (Решение1(Solution1),..., РешениеК(SolutionK))
где К - количество систем описанных в ОписанияСистемЛинейныхУравнений.

Описание:

Решает системы уравнений и возвращает объект ТаблицаЗначений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
