ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageAccessParameters)

Доступен, начиная с версии 8.3.24.

Свойства:

ИдентификаторДоступа (AccessID)
Регион (Region)
СекретныйКлюч (SecretKey)

Описание:

Параметры доступа внешнего хранилища двоичных данных.

Доступность:

Сервер.

Пример:

пд = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных();
пд.ИдентификаторДоступа = "EXAMPLE";
пд.СекретныйКлюч = "SecretKey";
пд.Регион = "US";
