ОписаниеОтображаемогоОбъектаPDF (PDFRepresentationObjectDescription)

Верх (Top)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит положение верхней границы объекта (в миллиметрах) на странице PDF-документа.
Начало координат страницы – левый верхний угол. Если не указано, то объект будет расположен автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
