ИспользованиеГруппИЭлементов (FoldersAndItemsUse)

Доступен, начиная с версии 8.0.

Значения

Группы (Folders)
ГруппыИЭлементы (FoldersAndItems)
Элементы (Items)

Описание:

Определяет способ использования групп и элементов справочника.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы;

См. также:

Расширение поля ввода справочника, свойство ВыборГруппИЭлементов
Расширение поля ввода вида характеристик, свойство ВыборГруппИЭлементов
Расширение формы списка справочника, свойство ПараметрВыборГруппИЭлементов
Расширение формы элемента справочника, свойство ИспользованиеДляГруппИЭлементов
Расширение формы элемента вида характеристик, свойство ИспользованиеДляГруппИЭлементов
Расширение формы списка видов характеристик, свойство ПараметрВыборГруппИЭлементов
Расширение табличного поля списка справочника, свойство ПросмотрГруппИЭлементов
Расширение табличного поля списка видов характеристик, свойство ПросмотрГруппИЭлементов

Использование в версии:

Доступен, начиная с версии 8.0.
