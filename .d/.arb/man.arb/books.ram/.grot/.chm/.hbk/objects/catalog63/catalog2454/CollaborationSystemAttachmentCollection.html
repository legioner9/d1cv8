КоллекцияВложенийСистемыВзаимодействия (CollaborationSystemAttachmentCollection)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

ВложениеСистемыВзаимодействия
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Предоставляет методы для работы с вложениями системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.25.
