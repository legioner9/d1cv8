ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ДобавитьДополнительныеДанные (AddAdditionalData)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ДобавитьДополнительныеДанные(<Имя>, <Значение>, <Представление>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого поля или табличной части.

<Значение> (обязательный)

Тип: Произвольный.
Значение добавляемого поля или табличной части.

<Представление> (обязательный)

Тип: Соответствие, Строка.
Если передана строка, то представление, используемое по умолчанию. При отсутствии представления на необходимом языке, поведение отлично от конфигуратора, в котором нет возможности указать представление, используемое по умолчанию.
Если передано соответствие, то представления на различных языках, где:
  • Ключ – код языка,
  • Значение - представление.

Описание:

Добавляет данные и представление данных в параметры записи истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
