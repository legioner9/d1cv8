Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает, что будут выбраны только те записи таблицы, на которые есть права у текущего пользователя.
Соответствует ключевому слову РАЗРЕШЕННЫЕ языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
