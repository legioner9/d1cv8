ДиалогВыбораТипаДиаграммы.ВыбратьАсинх (ChartTypeChooseDialog.ChooseAsync)

ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

ВыбратьАсинх (ChooseAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВыбратьАсинх()

Возвращаемое значение:

Тип: Обещание.
Выбранный тип диаграммы.

Описание:

Асинхронная функция. Открывает немодальный диалог выбора типа диаграммы.
По умолчанию будет установлен тип диаграммы, установленный в свойстве ТипДиаграммы.
Для выбора будут доступны типы диаграммы, установленные в свойстве ДоступныеТипыДиаграммы

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
