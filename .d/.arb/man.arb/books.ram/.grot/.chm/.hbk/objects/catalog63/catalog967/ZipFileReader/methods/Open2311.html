ЧтениеZipФайла (ZipFileReader)

Открыть (Open)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть для чтения из файла

Синтаксис:

Открыть(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя ZIP-файла, который требуется открыть для чтения.

<Пароль> (необязательный)

Тип: Строка.
Пароль для доступа к ZIP-файлу, если файл зашифрован.

Описание варианта метода:

Открыть ZIP-файл для чтения.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Пароль>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток, из которого будут прочитаны данные архива.

<Пароль> (необязательный)

Тип: Строка.
Пароль для доступа к ZIP-файлу, если файл зашифрован.

Описание варианта метода:

Устанавливает поток для чтения архива.

Описание:

Открывает ZIP файл для чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
