АдминистрированиеКластер (AdministrationCluster)

ВремяПринудительногоЗавершения (ForceTerminationTime)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Время принудительного завершения процесса.
Интервал времени (в секундах), по истечении которого выключенный процесс будет завершен принудительно даже, если еще имеются установленные с ним соединения. Если процесс выключен автоматически по достижении времени перезапуска, то после принудительного завершения процесс будет удален из кластера.
Если значение равно 0, то принудительное завершение не выполняется.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
