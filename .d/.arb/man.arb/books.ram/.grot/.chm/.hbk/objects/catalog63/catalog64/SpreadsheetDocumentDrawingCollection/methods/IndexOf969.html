КоллекцияРисунковТабличногоДокумента.Индекс (SpreadsheetDocumentDrawingCollection.IndexOf)

КоллекцияРисунковТабличногоДокумента (SpreadsheetDocumentDrawingCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Рисунок>)

Параметры:

<Рисунок> (обязательный)

Тип: РисунокТабличногоДокумента.
Рисунок табличного документа, индекс которого нужно определить.

Возвращаемое значение:

Тип: Число.
Индекс указанного рисунка в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс рисунка в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
