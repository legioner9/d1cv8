ВидПериодаРегистраРасчета (CalculationRegisterPeriodType)

Доступен, начиная с версии 8.0.

Значения

БазовыйПериод (BasePeriod)
ПериодДействия (ActionPeriod)
ПериодРегистрации (RegistrationPeriod)
ФактическийПериодДействия (ActualActionPeriod)

Описание:

Определяет вид периода регистра расчета - базовый период, период действия или период регистрации.
Данное перечисление используется как параметр метода регистра расчета, предназначенного для получения данных графика, связанного с регистром расчета.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

См. также:

РегистрРасчетаМенеджер.<Имя регистра расчета>, метод ПолучитьДанныеГрафика
РегистрРасчетаЗапись.<Имя регистра расчета>, метод ПолучитьДанныеГрафика

Использование в версии:

Доступен, начиная с версии 8.0.
