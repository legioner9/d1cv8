ИнформацияОСетевомАдаптере (NetworkAdapterInformation)

Доступен, начиная с версии 8.3.18.

Свойства:

IPАдрес (IPAddress)
MACАдрес (MACAddress)
ИмяАдаптера (AdapterName)
ЭтоТекущийАдаптер (IsCurrentAdapter)

Описание:

Объект содержит информацию о сетевом адаптере (для тонкого, толстого и веб-клиентов).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
