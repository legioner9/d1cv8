СредстваГеопозиционирования (LocationTools)

ОтключитьОтслеживаниеГеозон (DisableGeofencesMonitoring)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОтключитьОтслеживаниеГеозон(<Геозона>)

Параметры:

<Геозона> (обязательный)

Тип: Геозона, Строка, Массив.
Параметр может содержать идентификатор геозоны, объект или массив объектов типа Геозона.

Описание:

Отключает отслеживание одной или нескольких геозон для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
