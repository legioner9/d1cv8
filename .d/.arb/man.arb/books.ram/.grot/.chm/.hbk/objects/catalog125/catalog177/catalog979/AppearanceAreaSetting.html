НастройкаОбластиОформления (AppearanceAreaSetting)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияНастройкойОбластиОформления
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы управления областями оформления.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Набор элементов управления областями оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля настроек оформления, свойство НастройкаНастройкиОформления
Расширение табличного поля области оформления, свойство НастройкаОбластиОформления

Использование в версии:

Доступен, начиная с версии 8.0.
