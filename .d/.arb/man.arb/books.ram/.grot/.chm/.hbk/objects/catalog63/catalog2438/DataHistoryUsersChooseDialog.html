ДиалогВыбораПользователейИсторииДанных (DataHistoryUsersChooseDialog)

Доступен, начиная с версии 8.3.13.

Свойства:

Данные (Data)
Пользователи (Users)

Методы:

ОткрытьАсинх (DoAsync)
Показать (Show)

Конструкторы:

По умолчанию

Описание:

Диалог выбора пользователей-авторов версий истории данных, вызывается из формы отбора версий. Отображает полные имена пользователей авторов версий по указанным данным.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Пример:

Диалог = Новый ДиалогВыбораПользователейИсторииДанных(
                            Параметры.Данные);
Диалог.Пользователи = Пользователи;
Диалог.Показать(Новый ОписаниеОповещения
            ("ВыполнитьПослеЗакрытияОтбора", ЭтотОбъект));

Использование в версии:

Доступен, начиная с версии 8.3.13.
