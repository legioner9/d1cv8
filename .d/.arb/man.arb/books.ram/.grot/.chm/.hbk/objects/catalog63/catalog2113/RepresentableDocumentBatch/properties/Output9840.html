ПакетОтображаемыхДокументов.Вывод (RepresentableDocumentBatch.Output)

ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

Вывод (Output)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеВывода.
Содержит ограничение вывода на печать, сохранения. Используется только из пакета, у которого вызываются методы печати и сохранения. У вложенных пакетов игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойства Вывод табличных документов, находящихся в пакете, игнорируются при печати или сохранении пакета.

Использование в версии:

Доступен, начиная с версии 8.3.3.
