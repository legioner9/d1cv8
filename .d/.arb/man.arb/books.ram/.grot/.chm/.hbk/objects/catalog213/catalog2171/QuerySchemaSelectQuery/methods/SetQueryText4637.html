ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

УстановитьТекстЗапроса (SetQueryText)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТекстЗапроса(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Текст запроса выбора. Если в параметре передан пакетный запрос, запрос уничтожения таблицы или некорректный запрос, будет сгененировано исключение.

Описание:

Устанавливает текст запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
