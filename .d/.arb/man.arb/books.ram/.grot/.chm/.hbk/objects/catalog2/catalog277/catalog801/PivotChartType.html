ТипСводнойДиаграммы (PivotChartType)

ТипСводнойДиаграммы (PivotChartType)

Доступен, начиная с версии 8.0.

Значения

Гистограмма (Column)
ГистограммаОбъемная (Column3D)

Описание:

Содержит варианты типов сводных диаграмм. Используется для определения свойства ТипСводнойДиаграммы объекта СводнаяДиаграмма.

Доступность:

Тонкий клиент, сервер, толстый клиент.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: PivotChartType.

См. также:

СводнаяДиаграмма, свойство ТипСводнойДиаграммы

Использование в версии:

Доступен, начиная с версии 8.0.
