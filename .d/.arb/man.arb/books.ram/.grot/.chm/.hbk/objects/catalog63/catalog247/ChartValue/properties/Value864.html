ЗначениеДиаграммы.Значение (ChartValue.Value)

ЗначениеДиаграммы (ChartValue)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Содержит собственно значение диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При построении диаграммы значения с типами, отличными от Число, будут интерпретироваться как числа.
