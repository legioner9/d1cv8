ЭлементПланировщика.Выгрузить (PlannerItem.Unload)

ЭлементПланировщика (PlannerItem)

Выгрузить (Unload)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: Структура.

Описание:

Создает структуру и копирует в нее соответствующие свойства элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЭлементПланировщика, метод Загрузить

Использование в версии:

Доступен, начиная с версии 8.3.6.
