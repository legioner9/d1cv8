ТабличноеПоле.Перетаскивание (TableBox.Drag)

ТабличноеПоле (TableBox)

Перетаскивание (Drag)

Доступен, начиная с версии 8.0.

Синтаксис:

Перетаскивание(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Строка>, <Колонка>)

Параметры:

<ПараметрыПеретаскивания>

Тип: ПараметрыПеретаскивания.
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.
Значение по умолчанию: Истина.

<Строка>

Строка, над которой находится объект.

<Колонка>

Тип: КолонкаТабличногоПоля.
Колонка, над которой находится объект.

Описание:

Возникает при окончании перетаскивания в табличном поле - приемнике данных.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
