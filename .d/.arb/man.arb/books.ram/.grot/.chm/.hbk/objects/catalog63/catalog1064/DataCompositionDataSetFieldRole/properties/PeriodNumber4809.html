РольПоляНабораДанныхКомпоновкиДанных.НомерПериода (DataCompositionDataSetFieldRole.PeriodNumber)

РольПоляНабораДанныхКомпоновкиДанных (DataCompositionDataSetFieldRole)

НомерПериода (PeriodNumber)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Номер периода в случае, если поле является полем–периодом.
Значение 0 означает, что данное поле периодом не является. Самый старший период должен иметь номер 1. Его дочерний период – 2 и т.д.
Например, для запроса, получающего данные из виртуальной таблицы остатки и обороты, поле Период должно иметь номер периода = 1, поле Регистратор – 2, поле НомерСтроки – 3. На основании данной информации механизм будет рассчитывать итоги по полям остатка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
