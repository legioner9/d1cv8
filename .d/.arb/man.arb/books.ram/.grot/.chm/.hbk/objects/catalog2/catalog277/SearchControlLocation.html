ПоложениеУправленияПоиском (SearchControlLocation)

Доступен, начиная с версии 8.3.5.

Значения

Авто (Auto)
КоманднаяПанель (CommandBar)
Нет (None)

Описание:

Определяет положение автоматически размещенного элемента управления поиском в таблице.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
