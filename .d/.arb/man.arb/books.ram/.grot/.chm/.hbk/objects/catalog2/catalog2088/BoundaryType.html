ВидГраницы (BoundaryType)

ВидГраницы (BoundaryType)

Доступен, начиная с версии 8.0.

Значения

Включая (Including)
Исключая (Excluding)

Описание:

Определяет набор видов границ по отношению к граничному значению. Используется для определения значения свойства ВидГраницы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

Граница, свойство ВидГраницы
Граница, конструктор На основании значения и вида

Использование в версии:

Доступен, начиная с версии 8.0.
