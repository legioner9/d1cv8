ТабличноеПоле.ПриАктивизацииЯчейки (TableBox.OnActivateCell)

ТабличноеПоле (TableBox)

ПриАктивизацииЯчейки (OnActivateCell)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриАктивизацииЯчейки()

Описание:

Возникает при активизации ячейки табличного поля.

Доступность:

Толстый клиент.

Примечание:

Ячейка активизируется при изменении текущей строки и/или колонки. Последовательность возникновения событий: ПриАктивизацииСтроки, ПриАктивизацииКолонки, ПриАктивизацииЯчейки.

См. также:

ТабличноеПоле, событие ПриАктивизацииКолонки

Использование в версии:

Доступен, начиная с версии 8.0.
