МенеджерКриптографии.ПолучитьСертификатыИзПодписиАсинх (CryptoManager.GetCertificatesFromSignatureAsync)

МенеджерКриптографии (CryptoManager)

ПолучитьСертификатыИзПодписиАсинх (GetCertificatesFromSignatureAsync)

Не рекомендуется использовать, начиная с версии 8.3.24.

Рекомендуется использовать:

  • ПолучитьСертификаты
  • ПолучитьСертификатыАсинх

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьСертификатыИзПодписиАсинх(<ИсходныеДанные>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: Строка, ДвоичныеДанные.
Исходные данные, из которых будет извлечен сертификат.
Данные могут размещаться в файле (в этом случае указывается имя файла) или представлены как ДвоичныеДанные.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - массив объектов СертификатКриптографии.

Описание:

Асинхронная функция. Извлекает массив сертификатов из данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:
