БизнесПроцессОбъект.<Имя бизнес-процесса>.Удалить (BusinessProcessObject.<Business process name>.Delete)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет бизнес-процесс из базы данных.
Важно! В отличие от метода УстановитьПометкуУдаления, данный метод производит непосредственное удаление бизнес-процесса без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

ТекстВопроса = "Удалить бизнес-процесс без возможности восстановления";
Ответ = Вопрос(ТекстВопроса, РежимДиалогаВопрос.ДаНет);
Если Ответ = КодВозвратаДиалога.Да Тогда
    БизнесПроцесс.Удалить();
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
