УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

Доступен, начиная с версии 8.3.9.

Значения

Авто (Auto)
ИспользоватьЕслиВозможно (UseIfPossible)
НеИспользовать (DontUse)
Требовать (Require)
ТребоватьДляУправления (RequireForControl)

Описание:

Определяет требования клиента по использованию защищенного соединения с сервером FTP.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
