Объект внешней компоненты (Add-in object)

Доступен, начиная с версии 8.3.5.

Методы:

<ИмяМетода>Асинх (<MethodName>Async)
НачатьВызов<ИмяМетода> (BeginCalling<MethodName>)
НачатьПолучение<ИмяСвойства> (BeginGetting<PropertyName>)
НачатьУстановку<ИмяСвойства> (BeginSetting<PropertyName>)
Получить<ИмяСвойства>Асинх (Get<PropertyName>Async)
Установить<ИмяСвойства>Асинх (Set<PropertyName>Async)

Описание:

Объект, который создается во внешней компоненте.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
