ТипИмпортаСерийСлояГеографическойСхемы (GeographicalSchemaLayerSeriesImportModeType)

Доступен, начиная с версии 8.1.

Значения

ИмпортироватьВсе (ImportAll)
НеИмпортировать (DontImport)

Описание:

Определяет тип импорта серий слоя географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XML.

См. также:

СлоиГеографическойСхемы, метод Добавить

Использование в версии:

Доступен, начиная с версии 8.1.
