Форма.Открыта (Form.IsOpen)

Форма (Form)

Открыта (IsOpen)

Доступен, начиная с версии 8.0.

Синтаксис:

Открыта()

Возвращаемое значение:

Тип: Булево.
Истина - открыта; Ложь - не открыта.

Описание:

Определяет, открыта ли данная форма.

Доступность:

Толстый клиент.

Пример:

Если Не Форма.Открыта() Тогда
    Форма.Открыть();
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
