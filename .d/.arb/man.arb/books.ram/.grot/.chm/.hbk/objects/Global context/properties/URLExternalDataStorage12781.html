Глобальный контекст (Global context)

ХранилищеВнешнихДанныхНавигационныхСсылок (URLExternalDataStorage)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: СтандартноеХранилищеНастроекМенеджер, ХранилищеНастроекМенеджер.<Имя хранилища>.
Предоставляет доступ к хранилищу внешних данных навигационных ссылок.
При использовании стандартного хранилища, все пользователи имеют доступ ко всем настройкам на чтение.
При использовании стандартного хранилища для каждой настройки, сохраняемой в хранилище, записывается дата сохранения настройки. Ее можно получить через свойство ДатаИзменения. В режиме совместимости Версия8_3_18 и ниже это свойство всегда содержит значение Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
