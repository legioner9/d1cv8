ФорматированныйДокумент (FormattedDocument)

Удалить (Delete)

Доступен, начиная с версии 8.2.

Синтаксис:

Удалить(<Начало>, <Конец>)

Параметры:

<Начало> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, начиная с которой начинается удаляемый фрагмент текста. Если закладка не задана, то удаляется с начала документа.

<Конец> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Задает конец удаляемого фрагмента текста. Если параметр не задан, то удаляет текст до конца документа.

Описание:

Удаляет заданный фрагмент текста.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
