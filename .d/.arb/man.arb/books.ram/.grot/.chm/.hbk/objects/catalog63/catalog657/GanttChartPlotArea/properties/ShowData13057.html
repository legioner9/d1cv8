ОбластьПостроенияДиаграммыГанта (GanttChartPlotArea)

ОтображатьДанные (ShowData)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: ОтображениеВДиаграммеГанта.
Содержит признак отображения в области построения таблицы с данными значений. Если включено, то в области построения, после точек отображается таблица. Эта таблица доступна только для просмотра (по аналогии с таблицей значений в обычной диаграмме). Данное свойство рекомендуется включать, если диаграмма расположена в табличном документе.
Значение Авто интерпретируется следующим образом: если у диаграммы как элемента формы включено отображение таблицы в качестве предопределенного элемента (значение свойства ПоложениеТаблицы поля диаграммы не равно Нет), то НеОтображать, иначе - Отображать.
Значение по умолчанию - Авто.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
