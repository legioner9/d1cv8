КоллекцияМетокИнтервалаФонаПланировщика (PlannerBackgroundIntervalLabelCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.17.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс удаляемой метки в коллекции.

Вариант синтаксиса: По метке

Синтаксис:

Удалить(<Метка>)

Параметры:

<Метка> (обязательный)

Тип: МеткаИнтервалаФонаПланировщика.
Удаляемая метка в коллекции.

Описание:

Удаляет метку из коллекции меток интервала фона планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
