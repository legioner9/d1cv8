ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик> (ChartOfCharacteristicTypesManager.<Chart of characteristic types name>)

СоздатьЭлемент (CreateItem)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьЭлемент()

Возвращаемое значение:

Тип: ПланВидовХарактеристикОбъект.<Имя плана видов характеристик>.

Описание:

Создает новый элемент вида характеристики.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.0.
