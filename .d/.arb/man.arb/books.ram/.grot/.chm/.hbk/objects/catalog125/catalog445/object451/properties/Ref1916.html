ПланСчетовВыборка.<Имя плана счетов> (ChartOfAccountsSelection.<Chart of accounts name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовСсылка.<Имя плана счетов>.
Содержит ссылку на счет. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство используется, если ссылку на созданный и записанный элемент плана счетов нужно присвоить реквизиту другого плана счетов.

Использование в версии:

Доступен, начиная с версии 8.0.
