ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Добавить (Add)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: ОписаниеСистемыЛинейныхУравнений.
У нового созданного объекта ОписаниеСистемыЛинейныхУравнений свойства КолонкаКоэффициентовВСвязях и КолонкаКоэффициентовВУзлах имеют значение Неопределено.

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
