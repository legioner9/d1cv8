ИнтернетПочта (InternetMail)

ОтменитьПодпискуНаПочтовыйЯщик (UnsubscribeFromMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

ОтменитьПодпискуНаПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.
Имя IMAP почтового ящика.

Описание:

Отозвать подписку на IMAP почтовый ящик.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не используется для POP3 соединения.

Использование в версии:

Доступен, начиная с версии 8.2.
