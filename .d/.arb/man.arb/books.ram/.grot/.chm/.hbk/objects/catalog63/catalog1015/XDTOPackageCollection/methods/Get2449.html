КоллекцияПакетовXDTO.Получить (XDTOPackageCollection.Get)

КоллекцияПакетовXDTO (XDTOPackageCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс элемента коллекции.

Вариант синтаксиса: По URI пространства имен

Синтаксис:

Получить(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен пакета.

Возвращаемое значение:

Тип: ПакетXDTO.

Описание:

Получает пакет XDTO по индексу или имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
