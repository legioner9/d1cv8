ВыраженияИндексаСхемыЗапроса (QuerySchemaIndexExpressions)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ВыражениеИндексаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения индекса.
Возможно обращение к выражению индекса посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Объектная модель выражений индекса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
