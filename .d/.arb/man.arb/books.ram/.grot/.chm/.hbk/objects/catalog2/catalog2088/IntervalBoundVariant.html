ВариантГраницыИнтервала (IntervalBoundVariant)

ВариантГраницыИнтервала (IntervalBoundVariant)

Доступен, начиная с версии 8.0.

Значения

БезОграничения (WithoutRestriction)
Год (Year)
Квартал (Quarter)
КонкретнаяДата (SpecificDate)
Месяц (Month)
Неделя (Week)
РабочаяДата (WorkingDate)
Смещение (BeforeAfter)

Описание:

Определяет набор вариантов границы интервала, используемых объектом НастройкаПериода.

Доступность:

Толстый клиент.

См. также:

НастройкаПериода, свойство ВариантНачала
НастройкаПериода, свойство ВариантОкончания

Использование в версии:

Доступен, начиная с версии 8.0.
