РежимДиалогаВопрос (QuestionDialogMode)

Доступен, начиная с версии 8.0.

Значения

ДаНет (YesNo)
ДаНетОтмена (YesNoCancel)
ОК (OK)
ОКОтмена (OKCancel)
ПовторитьОтмена (RetryCancel)
ПрерватьПовторитьПропустить (AbortRetryIgnore)

Описание:

Определяет набор вариантов состава кнопок окна диалога вопроса. Используется в качестве параметра функции Вопрос.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Глобальный контекст, метод Вопрос

Использование в версии:

Доступен, начиная с версии 8.0.
