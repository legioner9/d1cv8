ВКонцеЭлемента (EndOfItem)

Доступен, начиная с версии 8.3.18.

Описание:

Действие отображается в области у правого или нижнего края элемента планировщика для горизонтального и вертикального положения шкалы времени соответственно.

Использование в версии:

Доступен, начиная с версии 8.3.18.
