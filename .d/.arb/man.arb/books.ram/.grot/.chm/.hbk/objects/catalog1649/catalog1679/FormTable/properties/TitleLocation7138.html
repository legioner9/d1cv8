ТаблицаФормы.ПоложениеЗаголовка (FormTable.TitleLocation)

ТаблицаФормы (FormTable)

ПоложениеЗаголовка (TitleLocation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ПоложениеЗаголовкаЭлементаФормы.
Управляет показом в форме заголовка к таблице. Если заголовок к таблице не нужен, следует установить данное свойство в Нет, в то время как пустая строка в свойстве Заголовок означает автоматическое формирование заголовка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
