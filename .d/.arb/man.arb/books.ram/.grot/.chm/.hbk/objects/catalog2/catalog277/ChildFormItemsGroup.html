ГруппировкаПодчиненныхЭлементовФормы (ChildFormItemsGroup)

ГруппировкаПодчиненныхЭлементовФормы (ChildFormItemsGroup)

Доступен, начиная с версии 8.2.

Значения

Вертикальная (Vertical)
Горизонтальная (Horizontal)
ГоризонтальнаяВсегда (AlwaysHorizontal)
ГоризонтальнаяЕслиВозможно (HorizontalIfPossible)

Описание:

Содержит варианты группировки подчиненных элементов в обычной группе и странице.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ФормаКлиентскогоПриложения, свойство Группировка
Расширение группы формы для страницы, свойство Группировка
Расширение группы формы для обычной группы, свойство Группировка

Использование в версии:

Доступен, начиная с версии 8.2.
