ПроцессорКомпоновкиДанных.Следующий (DataCompositionProcessor.Next)

ПроцессорКомпоновкиДанных (DataCompositionProcessor)

Следующий (Next)

Доступен, начиная с версии 8.1.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: ЭлементРезультатаКомпоновкиДанных, Неопределено.
Если все элементы результата получены, возвращает Неопределено.

Описание:

Получает следующий элемент результата компоновки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
