ПоддержкаМасштабаДиаграммыГанта (GanttChartScaleKeeping)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
ВсеДанные (AllData)
Период (Period)
Фиксированная (Fixed)

Описание:

Содержит варианты поддержки масштаба шкалы времени в видимой области диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChartScaleKeeping.

См. также:

ДиаграммаГанта, свойство ПоддержкаМасштаба

Использование в версии:

Доступен, начиная с версии 8.0.
