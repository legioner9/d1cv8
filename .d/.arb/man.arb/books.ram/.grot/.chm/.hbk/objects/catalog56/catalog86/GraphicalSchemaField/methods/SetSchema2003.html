ПолеГрафическойСхемы.УстановитьСхему (GraphicalSchemaField.SetSchema)

ПолеГрафическойСхемы (GraphicalSchemaField)

УстановитьСхему (SetSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьСхему(<ГрафическаяСхема>)

Параметры:

<ГрафическаяСхема> (обязательный)

Тип: ГрафическаяСхема.
Копия графической схемы.

Описание:

Устанавливает элементу управления "Поле графической схемы" значение "Графическая схема" для отображения.

Доступность:

Толстый клиент.

Пример:

Процедура ПриОткрытии()
    КартаМаршрута = ЭтотОбъект.ПолучитьКартуМаршрута();
    ЭлементыФормы.ПолеСхемы1.УстановитьСхему(КартаМаршрута);
КонецПроцедуры

Использование в версии:

Доступен, начиная с версии 8.1.
