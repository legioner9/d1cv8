РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: РазмерКартинки.
Содержит способ приспособления оригинальных размеров картинки к размеру рисунка.
Значение АвтоРазмерБезУчетаМасштаба интерпретируется как АвтоРазмер.
Значение РеальныйРазмерБезУчетаМасштаба интерпретируется как РеальныйРазмер.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунков типа Картинка.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
