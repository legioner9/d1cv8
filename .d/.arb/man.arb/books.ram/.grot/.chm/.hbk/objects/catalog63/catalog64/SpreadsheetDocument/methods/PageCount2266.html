ТабличныйДокумент.КоличествоСтраниц (SpreadsheetDocument.PageCount)

ТабличныйДокумент (SpreadsheetDocument)

КоличествоСтраниц (PageCount)

Доступен, начиная с версии 8.0.

Синтаксис:

КоличествоСтраниц()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество страниц в документе с учетом текущих настроек печати.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При возникновении проблем с получением информации о текущем принтере (например, в системе не установлено ни одного принтера), будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.0.
