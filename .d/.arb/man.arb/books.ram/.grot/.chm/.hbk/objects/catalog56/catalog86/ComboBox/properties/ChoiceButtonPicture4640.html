КартинкаКнопкиВыбора (ChoiceButtonPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Картинка.
Содержит картинку, отображаемую в кнопке выбора.

Доступность:

Толстый клиент.

Примечание:

Оптимальный размер картинки определяется размером элемента управления и размером его шрифта.
Если картинка задается программно, может быть установлена картина вида Пустая или ИзБиблиотеки.

Использование в версии:

Доступен, начиная с версии 8.0.
