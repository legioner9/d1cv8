Порядок.УстановитьДоступныеПоля (Order.SetAvailableFields)

Порядок (Order)

УстановитьДоступныеПоля (SetAvailableFields)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьДоступныеПоля(<Поля>)

Параметры:

<Поля> (необязательный)

Тип: ПоляНастройки.
Коллекция доступных полей.

Описание:

Устанавливает доступные для упорядочивания поля.
Внимание! Метод не доступен для упорядочивания динамических списков (свойство Порядок объектов СправочникСписок.<Имя справочника>, ДокументСписок.<Имя документа> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// см. пример метода ПолучитьДоступныеПоля


Использование в версии:

Доступен, начиная с версии 8.0.
