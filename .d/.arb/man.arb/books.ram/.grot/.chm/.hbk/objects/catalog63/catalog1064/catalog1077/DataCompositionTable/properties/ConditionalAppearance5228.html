Только чтение.

Описание:

Тип: УсловноеОформлениеКомпоновкиДанных.
Содержит условное оформление, применяемое в таблице.
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
