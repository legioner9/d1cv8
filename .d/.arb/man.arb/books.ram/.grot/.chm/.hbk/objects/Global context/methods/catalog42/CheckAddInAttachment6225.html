Глобальный контекст (Global context)

ПроверитьПодключениеВнешнейКомпоненты (CheckAddInAttachment)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПроверитьПодключениеВнешнейКомпоненты(<Местоположение>, <Имя>)

Параметры:

<Местоположение> (необязательный)

Тип: Строка.
Строка, определяющая местоположение внешней компоненты.

<Имя> (необязательный)

Тип: Строка.
Символическое имя подключаемой внешней компоненты.

Описание:

Выполняет проверку подключения внешней компоненты.
Перед вызовом метода компоненты рекомендуется проверить работоспособность самой компоненты с помощью данного метода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
