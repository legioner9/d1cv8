БизнесПроцессОбъект.<Имя бизнес-процесса>.Завершен (BusinessProcessObject.<Business process name>.Completed)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Завершен (Completed)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак завершения бизнес-процесса. Этот признак устанавливается системой автоматически при переходе бизнес-процесса на точку завершения при условии отсутствия невыполненных задач.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - бизнес-процесс завершен; Ложь - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.0.
