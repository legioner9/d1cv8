ИнтернетПрокси.Пароль (InternetProxy.Password)

ИнтернетПрокси (InternetProxy)

Пароль (Password)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Пароль пользователя для аутентификации на прокси-сервере.
Внимание! Данное свойство признано устаревшим. Вместо него рекомендуется использовать метод Пароль.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
