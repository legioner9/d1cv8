Описание:

Предназначен для операций с наборами записей таблицы внешнего источника данных.
Используется только для таблиц с типом данных НеобъектныеДанные.

Сервер, толстый клиент, внешнее соединение.

Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise/current-config}.
