ОбластьЗаголовкаДендрограммы (DendrogramTitleArea)

ЦветРамки (BorderColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Цвет.
Содержит цвет рамки вокруг области заголовка.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если рамка имеет стиль Выпуклая или Вдавленная - в этом случае рамка рисуется притененным цветом от цвета фона.

Использование в версии:

Доступен, начиная с версии 8.0.
