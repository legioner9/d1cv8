ОтображениеЛинийСеткиДиаграммы (ChartGridLinesShowMode)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Способ отображения линий определяется автоматически:
  • Для шкалы значений - Отображать.
  • Для шкалы точек в точечной и пузырьковой диаграммах - Отображать.
  • Во всех остальных случаях - НеОтображать.

Использование в версии:

Доступен, начиная с версии 8.3.11.
