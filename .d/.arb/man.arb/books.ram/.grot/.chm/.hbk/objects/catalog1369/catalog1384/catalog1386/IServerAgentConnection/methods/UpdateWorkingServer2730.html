Соединение с агентом сервера.UpdateWorkingServer (IServerAgentConnection.UpdateWorkingServer)

Соединение с агентом сервера (IServerAgentConnection)

UpdateWorkingServer (UpdateWorkingServer)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateWorkingServer(<Кластер>, <Сервер>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Определяет кластер, в котором должно быть модифицировано описание рабочего сервера. Список кластеров может быть получен при помощи метода GetClusters.

<Сервер> (обязательный)

Тип: Рабочий сервер.
Содержит модифицированное описание сервера. Его значение должно быть получено методом GetWorkingServers, после чего в нем может быть изменен список диапазонов IP-портов для динамического распределения при помощи методов InsertPortRange и ErasePortRange.

Описание:

Модифицирует описание зарегистрированного в кластере рабочего сервера. Можно модифицировать только список диапазонов портов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
