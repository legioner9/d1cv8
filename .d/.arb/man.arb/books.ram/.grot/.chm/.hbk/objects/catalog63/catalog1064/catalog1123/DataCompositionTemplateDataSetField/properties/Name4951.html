ПолеНабораДанныхМакетаКомпоновкиДанных.Имя (DataCompositionTemplateDataSetField.Name)

ПолеНабораДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSetField)

Имя (Name)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Имя поля набора данных, для которого делается описание. Имя, под которым данное поле будет фигурировать в макете компоновки данных, составляется из имени набора данных и имени поля, объединенных через точку. Например, к полю Номенклатура набора данных Остатки необходимо будет обращаться по имени Остатки.Номенклатура.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
