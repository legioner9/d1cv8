ТипСУБДКопииБазыДанных (DatabaseCopyDBMSType)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Значения

MSSQLServer (MSSQLServer)
OracleDatabase (OracleDatabase)
PostgreSQL (PostgreSQL)

Описание:

Содержит варианты поддерживаемых типов СУБД копии базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
