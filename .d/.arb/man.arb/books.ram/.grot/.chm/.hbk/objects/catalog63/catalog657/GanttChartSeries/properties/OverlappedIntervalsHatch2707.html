СерияДиаграммыГанта (GanttChartSeries)

ШтриховкаПерекрывающихсяИнтервалов (OverlappedIntervalsHatch)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если установлено Истина, то перекрывающиеся интервалы выводятся штриховкой, а не сплошным цветом. Цвет выбирается из свойства ДополнительныйЦвет с учётом свойства ПриоритетЦвета.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

СерияДиаграммыГанта, свойство ПриоритетЦвета
СерияДиаграммыГанта, свойство ДополнительныйЦвет

Использование в версии:

Доступен, начиная с версии 8.0.
