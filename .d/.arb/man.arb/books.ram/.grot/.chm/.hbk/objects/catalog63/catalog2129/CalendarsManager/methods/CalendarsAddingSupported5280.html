МенеджерКалендарей (CalendarsManager)

ПоддерживаетсяДобавлениеКалендарей (CalendarsAddingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяДобавлениеКалендарей()

Возвращаемое значение:

Тип: Булево.
На устройствах под управлением Windows Phone возвращает Ложь.
Возвращает Истина на устройствах под управлением:
  • Android – начиная с версии 4.1,
  • iOS – всегда.

Описание:

Определяет возможность добавления календарей.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерКалендарей, метод ДобавитьКалендарь

Использование в версии:

Доступен, начиная с версии 8.3.10.
