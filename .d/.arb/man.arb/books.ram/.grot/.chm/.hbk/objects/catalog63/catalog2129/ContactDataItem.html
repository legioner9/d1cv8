ЭлементДанныхКонтакта (ContactDataItem)

ЭлементДанныхКонтакта (ContactDataItem)

Доступен, начиная с версии 8.3.4.

Свойства:

Значение (Value)
ТипДанных (DataType)

Конструкторы:

По умолчанию

Описание:

Описывает элемент данных контакта. Например, номер телефона или адрес электронной почты.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/data/mob. Имя типа XDTO: ContactDataElement.

Использование в версии:

Доступен, начиная с версии 8.3.4.
