НаборСхемXML (XMLSchemaSet)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СхемаXML
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются схемы XML.
Возможно обращение к схеме XML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) схемы.

Методы:

Добавить (Add)
Количество (Count)
Получить (Get)
Проверить (Validate)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Набор схем XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
