Доступен, начиная с версии 8.3.5.

Свойства:

Источник (Source)
Соединения (Joins)

Описание:

Описание источника запроса.
В качестве источника может выступать таблица информационной базы, вложенный запрос, временная таблица.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
