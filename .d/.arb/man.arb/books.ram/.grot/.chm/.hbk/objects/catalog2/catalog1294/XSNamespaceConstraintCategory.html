КатегорияОграниченияПространствИменXS (XSNamespaceConstraintCategory)

Доступен, начиная с версии 8.1.

Значения

Кроме (Not)
Любое (Any)
Набор (Set)

Описание:

Содержит категории ограничения использования пространств имен.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МаскаXS, свойство КатегорияОграниченияПространствИмен

Использование в версии:

Доступен, начиная с версии 8.1.
