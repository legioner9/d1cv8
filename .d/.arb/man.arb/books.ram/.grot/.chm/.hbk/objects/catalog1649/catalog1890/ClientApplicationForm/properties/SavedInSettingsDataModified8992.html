ФормаКлиентскогоПриложения (ClientApplicationForm)

СохраняемыеВНастройкахДанныеМодифицированы (SavedInSettingsDataModified)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак модифицированности сохраняемых в настройках формы данных.
Если Истина и у формы при конфигурировании установлена необходимость автоматически сохранять данные формы, то при закрытии формы данные формы сохраняются с вызовом обработчика.
При изменении сохраняемых данных формы платформа автоматически помещает значение Истина в свойство.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
