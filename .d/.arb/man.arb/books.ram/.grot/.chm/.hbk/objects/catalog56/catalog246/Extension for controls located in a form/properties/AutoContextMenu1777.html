Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

АвтоКонтекстноеМеню (AutoContextMenu)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Установка данного свойства разрешает использование для элемента управления автоматического контекстного меню, иначе будет использовано контекстное меню, выбранное в свойстве КонтекстноеМеню.

Доступность:

Толстый клиент.

Примечание:

Свойство используется только для следующих типов элементов управления: ПолеВвода, ПолеВыбора, ПолеСписка, ТабличноеПоле, ПолеТекстовогоДокумента, ПолеТабличногоДокумента.

ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.
