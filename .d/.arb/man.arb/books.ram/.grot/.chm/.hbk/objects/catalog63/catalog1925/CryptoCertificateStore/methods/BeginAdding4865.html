ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьДобавление (BeginAdding)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьДобавление(<ОписаниеОповещения>, <Сертификат>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения добавления со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Сертификат> (обязательный)

Тип: СертификатКриптографии, КонтейнерКлючейКриптографии.
Добавляемый сертификат криптографии.
На мобильной платформе вместо сертификата может быть использован контейнер ключей криптографии.

Описание:

Начинает добавление сертификата в хранилище.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе под управлением iOS нельзя несколько раз добавить один и тот же сертификат или контейнер ключей криптографии, даже в хранилища с разным типом.

См. также:

ХранилищеСертификатовКриптографии, метод Добавить

Использование в версии:

Доступен, начиная с версии 8.3.6.
