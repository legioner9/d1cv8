СредстваБуфераОбмена.ПолучитьДанныеАсинх (ClipboardTools.GetDataAsync)

СредстваБуфераОбмена (ClipboardTools)

ПолучитьДанныеАсинх (GetDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьДанныеАсинх(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: СтандартныйФорматДанныхБуфераОбмена, Строка.
Формат получаемых данных.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания – данные типа Строка, ДвоичныеДанные, Картинка или Неопределено.
Если формат не известен, то возвращается Неопределено.
Если значения, соответствующего формату, не найдено, то возвращается Неопределено.

Описание:

Асинхронная функция. Получает из буфера обмена данные указанного формата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте:
  • может появиться запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается Обещание, результат которого – Неопределено.
  • значение параметра <Формат> типа Строка должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Использование в версии:

Доступен, начиная с версии 8.3.24.
