МеткаNDEF.ЗаписатьСообщениеАсинх (TagNDEF.WriteMessageAsync)

МеткаNDEF (TagNDEF)

ЗаписатьСообщениеАсинх (WriteMessageAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗаписатьСообщениеАсинх(<Сообщение>)

Параметры:

<Сообщение> (обязательный)

Тип: СообщениеNDEF.
Полезная нагрузка (фактические данные метки).

Описание:

Записывает сообщение NDEF на физическую метку.
Перед записью рекомендуется проверить, доступна ли запись с помощью метода ПоддерживаетсяЗаписьНаМеткуАсинх.
Если произошла ошибка записи, то будет сгенерировано исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
