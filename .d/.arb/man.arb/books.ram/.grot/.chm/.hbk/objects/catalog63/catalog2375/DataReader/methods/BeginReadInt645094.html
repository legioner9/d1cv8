ЧтениеДанных.НачатьЧтениеЦелого64 (DataReader.BeginReadInt64)

ЧтениеДанных (DataReader)

НачатьЧтениеЦелого64 (BeginReadInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтениеЦелого64(<ОписаниеОповещения>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <Результат> - тип Число. Результат чтения.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, используемый при чтении числа.
Если не задан, используется порядок, определенный для текущего экземпляра ЧтениеДанных.
Значение по умолчанию: Неопределено.

Описание:

Начинает чтение 64-битного целого числа из потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ЧтениеДанных, метод ПрочитатьЦелое64
ЧтениеДанных, метод ПрочитатьЦелое64Асинх

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
