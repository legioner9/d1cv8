НаборыДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSets)

Доступен, начиная с версии 8.1.

Элементы коллекции:

НаборДанныхЗапросСхемыКомпоновкиДанных, НаборДанныхОбъектСхемыКомпоновкиДанных, НаборДанныхОбъединениеСхемыКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы данных.
Возможно обращение к набору данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит наборы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
