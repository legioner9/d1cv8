Набор свойств содержит значения измерений регистра накопления. Доступ к значению осуществляется по имени, как оно задано в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

В качестве типа допускается использовать любые типы, кроме УникальныйИдентификатор, ДвоичныеДанные и строки неограниченной длины.
