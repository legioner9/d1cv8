КнопкаФормы.ПропускатьПриВводе (FormButton.SkipOnInput)

КнопкаФормы (FormButton)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Указывает необходимость пропуска поля при вводе данных.
Если Неопределено, то это Истина для кнопки по умолчанию и Ложь для всех остальных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
