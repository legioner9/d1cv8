ШкалаДиаграммы (ChartScale)

ШагШкалы (ScaleStep)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Значение шага шкалы. Если на шкале откладываются дискретные значения, то значение шага округляется до целого. Если шаг шкалы = 1, выводится каждый шаг. Если шаг шкалы = 2, выводится каждый второй шаг, и т. д.
"0" - шаг рассчитывается автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
