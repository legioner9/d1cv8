Описание:

Определяет набор картинок, используемых в конфигурации. Значения этого набора имеют тип Картинка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

// Для обычных форм

ЭлементыФормы.Надпись.Картинка = БиблиотекаКартинок.ИнтервалДат;
// Для управляемых форм

Элементы.Надпись.КартинкаКнопкиВыбора = БиблиотекаКартинок.ИнтервалДат;

См. также:

Глобальный контекст, свойство БиблиотекаКартинок
