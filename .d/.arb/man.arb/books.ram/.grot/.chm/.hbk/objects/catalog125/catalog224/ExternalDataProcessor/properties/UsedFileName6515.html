ВнешняяОбработка (ExternalDataProcessor)

ИспользуемоеИмяФайла (UsedFileName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Содержит полное имя файла внешней обработки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если внешняя обработка открыта в веб-клиенте, то содержит имя файла (без пути).

Использование в версии:

Доступен, начиная с версии 8.1.
