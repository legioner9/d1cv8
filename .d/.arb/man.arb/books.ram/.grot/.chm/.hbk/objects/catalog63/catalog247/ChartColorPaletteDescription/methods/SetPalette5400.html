ОписаниеПалитрыЦветовДиаграммы (ChartColorPaletteDescription)

УстановитьПалитру (SetPalette)

Доступен, начиная с версии 8.3.12.

Синтаксис:

УстановитьПалитру(<Палитра>)

Параметры:

<Палитра> (обязательный)

Тип: Массив.
Массив цветов палитры. Используются только элементы типа Цвет. Элементы другого типа игнорируются.

Описание:

Устанавливает свойство ПалитраЦветов в значение Произвольная. Использует цвета из переданного массива в качестве палитры. Цвета записываются в палитру в том же порядке, что и в переданном массиве.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
