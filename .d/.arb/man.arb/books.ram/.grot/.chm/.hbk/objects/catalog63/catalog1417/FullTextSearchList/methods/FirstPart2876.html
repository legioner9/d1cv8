ПерваяЧасть (FirstPart)

Доступен, начиная с версии 8.1.

Синтаксис:

ПерваяЧасть()

Описание:

Выполняет поиск и получает первую часть результатов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если строка поиска содержит ошибки, или если по каким-либо причинам, поиск не может вернуть результаты поиска, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
