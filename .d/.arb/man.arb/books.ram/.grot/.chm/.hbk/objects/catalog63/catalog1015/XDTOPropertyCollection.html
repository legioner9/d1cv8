КоллекцияСвойствXDTO (XDTOPropertyCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СвойствоXDTO
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются свойства объекта XDTO.
Возможно обращение к свойству объекта XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Количество (Count)
Получить (Get)

Описание:

Коллекция свойств XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
