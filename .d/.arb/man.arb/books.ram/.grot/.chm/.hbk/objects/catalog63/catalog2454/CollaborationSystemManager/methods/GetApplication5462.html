МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьПриложение (GetApplication)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьПриложение(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: ИдентификаторПриложенияСистемыВзаимодействия.
Идентификатор приложения.

Возвращаемое значение:

Тип: ПриложениеСистемыВзаимодействия.

Описание:

Возвращает приложение системы взаимодействия по переданному идентификатору приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
