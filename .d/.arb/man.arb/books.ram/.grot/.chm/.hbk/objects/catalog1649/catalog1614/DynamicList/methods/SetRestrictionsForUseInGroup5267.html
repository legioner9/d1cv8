ДинамическийСписок (DynamicList)

УстановитьОграниченияИспользованияВГруппировке (SetRestrictionsForUseInGroup)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьОграниченияИспользованияВГруппировке(<Поля>)

Параметры:

<Поля> (необязательный)

Тип: Массив.
Пути к полям компоновки, на которые требуется установить ограничение.

Описание:

Запрещает использование указанных полей и всех их дочерних полей в настройках группировки.
Поля, на которые установлено ограничение, не входят в коллекцию доступных полей.
После вызова метода список полей с ограничениями замещается указанным.
Вызов метода с пустым списком полей отменяет ранее установленные ограничения.

Доступность:

Сервер.

Примечание:

Если текст запроса не меняется динамически, или если вместе с изменением текста запроса меняются описания полей набора данных, рекомендуется использовать:
  • Поля,
  • ОграничениеИспользования,
  • ОграничениеИспользованияРеквизитов,
  • ОграничениеИспользования.

Пример:

Список.Поля.Найти("Флаг").ОграничениеИспользования.Группировка = Истина;

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
