СтандартнаяКомандаВходящегоЗапросаПоделиться (IncomingShareRequestStandardCommand)

КопироватьВБуферОбмена (CopyToClipboard)

Доступен, начиная с версии 8.3.24.

Описание:

Доступно только при запросе "Поделиться" для одной строки.
При выполнении команды строка будет сохранена в буфер обмена.
В контекстном меню команда будет иметь представление "Копировать в буфер обмена".

Использование в версии:

Доступен, начиная с версии 8.3.24.
