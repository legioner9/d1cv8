Глобальный контекст (Global context)

ASin (ASin)

Доступен, начиная с версии 8.0.

Синтаксис:

ASin(<Число>)

Параметры:

<Число> (обязательный)

Тип: Число.
Аргумент функции. Определен в диапазоне -1 ... 1.

Возвращаемое значение:

Тип: Число.
Результат вычисления арксинуса в радианах.

Описание:

Вычисляет арксинус от аргумента <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если аргумент выходит за пределы диапазона, то происходит ошибка "Неправильное значение аргумента встроенной функции (ASin)".

Пример:

Pi = 3.1415926535897932;
Аргумент = Sqrt(2)/2;
Попытка
    УголВГрадусах = 360*ASin(Аргумент)/Pi/2;
    Сообщить("Значение угла: " + Окр(УголВГрадусах,2));
Исключение
    Сообщить("При значении аргумента Sin()" + Аргумент
             + " выдается сообщение об ошибке:"
             + Символы.ПС + """" 
             + ОписаниеОшибки() + """");
КонецПопытки;
