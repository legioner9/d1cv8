СтандартноеХранилищеНастроекВыборка.Следующий (StandardSettingsStorageSelection.Next)

СтандартноеХранилищеНастроекВыборка (StandardSettingsStorageSelection)

Следующий (Next)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: Булево.

Описание:

Получает следующий элемент из выборки. Для обхода хранилища настроек нужно после получения выборки вызвать данный метод для позиционирования на первый элемент и далее вызывать до тех пор, пока не будет возвращено значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При выполнении данного метода не выполняется получение данных выборки.
Чтобы получить данные выборки, следует использовать обращение к свойствам объекта СтандартноеХранилищеНастроекВыборка. При возникновении ошибки во время получения значения свойства будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
