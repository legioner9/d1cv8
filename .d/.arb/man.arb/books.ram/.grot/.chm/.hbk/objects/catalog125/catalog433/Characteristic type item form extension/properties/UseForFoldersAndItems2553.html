Расширение формы элемента вида характеристик (Characteristic type item form extension)

ИспользованиеДляГруппИЭлементов (UseForFoldersAndItems)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеГруппИЭлементов.
Если в конфигурации не разрабатывались формы для элементов, групп, форм списка или выборки, 1С:Предприятие создает их самостоятельно. В случае открытия формы средствами встроенного языка с помощью данного свойства можно указать, для чего данная форма будет использована. По значению свойства будут автоматически сформирована командная панель и сгенерированы управляющие элементы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
