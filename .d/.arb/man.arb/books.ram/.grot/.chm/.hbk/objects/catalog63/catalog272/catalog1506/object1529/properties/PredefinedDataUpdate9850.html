Описание:

Тип: ОбновлениеПредопределенныхДанных.
Содержит режим обновления предопределенных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для режимов Авто и ОбновлятьАвтоматически дополнительно учитывается флаг, установленный в базе данных с помощью метода УстановитьОбновлениеПредопределенныхДанных.
