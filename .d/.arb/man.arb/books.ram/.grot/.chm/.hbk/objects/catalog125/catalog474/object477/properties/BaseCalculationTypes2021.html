ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

БазовыеВидыРасчета (BaseCalculationTypes)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: БазовыеВидыРасчета.<Имя плана видов расчета>.
Предопределенная табличная часть вида расчета, которая содержит список базовых видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определено только для объектов тех планов видов расчета, у которых установлено свойство "Зависимость от базы".

Использование в версии:

Доступен, начиная с версии 8.0.
