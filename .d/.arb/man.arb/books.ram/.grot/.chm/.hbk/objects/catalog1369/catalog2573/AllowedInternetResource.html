РазрешенныйИнтернетРесурс (AllowedInternetResource)

Доступен, начиная с версии 8.3.14.

Свойства:

Адрес (Address)
Имя (Name)
Описание (Description)
Порт (Port)
Протокол (Protocol)

Методы:

Записать (Write)
Удалить (Delete)

Описание:

Предоставляет доступ к свойствам и методам интернет-ресурса.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

См. также:

Использование в версии:

Доступен, начиная с версии 8.3.14.
