ОбработкаТекстаИнтернетПочтовогоСообщения (InternetMailTextProcessing)

Доступен, начиная с версии 8.0.

Значения

НеОбрабатывать (DontProcess)
Обрабатывать (Process)

Описание:

Определяет набор действий над текстами почтового сообщения перед отправкой.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

ИнтернетПочтовоеСообщение, метод ОбработатьТексты
ИнтернетПочта, метод Послать

Использование в версии:

Доступен, начиная с версии 8.0.
