ГрафическаяСхема.НачатьЗапись (GraphicalSchema.BeginWriting)

ГрафическаяСхема (GraphicalSchema)

НачатьЗапись (BeginWriting)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения записи файла со следующими параметрами:
  • <Результат> - результат операции:
    • Истина - если запись была инициирована с помощью расширений для браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
    • Ложь - если запись была инициирована с помощью расширений для браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
    • Неопределено - если запись была инициирована скачиванием файла средствами браузера и не известно - был ли файл записан или нет.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла, в котором сохраняется графическая схема.

Описание:

Вызывает начало записи графической схемы в файл.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:
