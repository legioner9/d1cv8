ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

КартаМаршрута (Flowchart)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Неопределено.
Содержит карту маршрута бизнес-процесса, которая описывает логику бизнес-процесса от точки старта до точки завершения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство всегда имеет значение Неопределено, т.к. во встроенном языке не предусмотрена работа со значением данного свойства.
Для получения карты маршрута можно использовать метод ПолучитьКартуМаршрута() менеджера соответствующего бизнес-процесса.

Использование в версии:

Доступен, начиная с версии 8.1.
