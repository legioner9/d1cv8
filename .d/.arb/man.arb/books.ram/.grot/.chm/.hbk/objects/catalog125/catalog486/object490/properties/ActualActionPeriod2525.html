РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

ФактическийПериодДействия (ActualActionPeriod)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ФиксированныйМассив.
Фактический период действия.
Отличается от периода действия, задаваемого датами начала и окончания периода действия.
Период действия - это свойство учетной записи, которое отражает тот факт, что данная учетная запись пытается получить такой период действия.
Фактический период действия - это тот период, который сложился за счет правил взаимной конкуренции учетных записей за период действия, определяемых списками вытесняющих видов расчета плана расчетов данного регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определен для регистров расчета с признаком "Период действия".

Использование в версии:

Доступен, начиная с версии 8.0.
