ПолеФормы.ПриИзменении (FormField.OnChange)

ПолеФормы (FormField)

ПриИзменении (OnChange)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриИзменении()

Описание:

Вызывается при изменении данных в поле.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Примечание:

Если Вид=ПолеВвода и данные, редактируемые в поле, имеют составной тип, то также возникает при смене типа.

Использование в версии:

Доступен, начиная с версии 8.2.
