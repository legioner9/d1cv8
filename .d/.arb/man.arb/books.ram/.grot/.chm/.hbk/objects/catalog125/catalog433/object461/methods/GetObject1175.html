ПланВидовХарактеристикВыборка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesSelection.<Chart of characteristic types name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: ПланВидовХарактеристикОбъект.<Имя плана видов характеристик>, Неопределено.
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект, предназначенный для модификации элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
