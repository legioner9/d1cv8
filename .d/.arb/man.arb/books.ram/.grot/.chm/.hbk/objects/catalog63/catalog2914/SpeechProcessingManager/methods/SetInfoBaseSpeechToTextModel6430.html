МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьМодельРаспознаванияДляИнформационнойБазы (SetInfoBaseSpeechToTextModel)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьМодельРаспознаванияДляИнформационнойБазы(<Источник>)

Параметры:

<Источник> (обязательный)

Тип: Строка, ФайловыйПоток,
Поток, ПотокВПамяти, ХранилищеЗначения.
Источник данных модели в виде пути к файлу или потока.
Может быть указан путь к файлу или адрес файла во временном хранилище.

Возвращаемое значение:

Тип: ОписаниеМоделиРаспознаванияРечи.

Описание:

Устанавливает модель распознавания речи, сохраняет запись о модели в служебной таблице в базе данных с учетом установленных разделителей.
Если модель сохраняется в области данных, для которой не установлены разделители, то модель становится доступной при работе без учета установленных разделителей.
В общем случае, модели, загруженные в область данных, применяются только к этой области и ко всем областям, включенных в область данных.
Все установленные модели будут загружены в память при первом входе в информационную базу с учетом установленных разделителей.

Вызов метода требует наличия у текущего пользователя права "Администрирование данных".
Внимание! В мобильной платформе работа с локальными моделями распознавания не поддерживается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.25.
