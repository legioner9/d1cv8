СредстваМультимедиа (MultimediaTools)

ПолучитьПоддерживаемыеРазрешенияКамеры (GetSupportedCameraResolutions)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьПоддерживаемыеРазрешенияКамеры(<ТипКамеры>)

Параметры:

<ТипКамеры> (необязательный)

Тип: ТипКамерыУстройства.
Показывает, для какой камеры будут получены разрешения.
Значение по умолчанию: Авто.

Возвращаемое значение:

Тип: Массив, Неопределено.
Массив РазрешениеКамерыУстройства. Для iOS будет возвращен тип Неопределено.

Описание:

Получает массив доступных разрешений для выбранной камеры.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяФотоснимок.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваМультимедиа, метод ПоддерживаетсяФотоснимок

Использование в версии:

Доступен, начиная с версии 8.3.6.
