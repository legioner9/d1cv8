КонтекстПространствИменXML.URIПространствИмен (XMLNamespaceContext.NamespaceURIs)

КонтекстПространствИменXML (XMLNamespaceContext)

URIПространствИмен (NamespaceURIs)

Доступен, начиная с версии 8.1.

Синтаксис:

URIПространствИмен()

Возвращаемое значение:

Тип: Массив.

Описание:

Получает массив URI пространств имен, определенных в данном контексте.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
