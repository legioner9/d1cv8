Глобальный контекст (Global context)

НачатьУстановкуРасширенияПолученияИнформацииОКомпьютере (BeginInstallingComputerInformationExtension)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьУстановкуРасширенияПолученияИнформацииОКомпьютере(<ОписаниеОповещенияОЗавершении>)

Параметры:

<ОписаниеОповещенияОЗавершении> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения установки расширения.
Процедура будет вызвана со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Описание:

Устанавливает расширение для работы с информацией о компьютере в браузер клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.
