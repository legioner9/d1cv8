Планировщик (Planner)

АвтоМинимальнаяШиринаКолонки (AutoColumnMinWidth)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет использование автоматического ограничения минимальной ширины колонки.
Если значение Ложь, автоматическое ограничение не используется, и минимальная ширина определяется свойством МинимальнаяШиринаКолонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
