МенеджерОтображенияРекламы (AdRepresentationManager)

ПолучитьСтатусРекламы (GetAdStatus)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСтатусРекламы(<ИдентификаторРекламногоБлока>)

Параметры:

<ИдентификаторРекламногоБлока> (обязательный)

Тип: Строка.
Идентификатор рекламного блока, для которого нужно получить статус.

Возвращаемое значение:

Тип: СтатусРекламы.

Описание:

Возвращает статус рекламы для конкретного идентификатора.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.14.
