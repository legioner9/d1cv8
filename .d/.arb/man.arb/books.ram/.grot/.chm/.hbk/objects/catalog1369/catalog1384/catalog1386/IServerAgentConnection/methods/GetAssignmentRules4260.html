Соединение с агентом сервера (IServerAgentConnection)

GetAssignmentRules (GetAssignmentRules)

Доступен, начиная с версии 8.3.1.

Синтаксис:

GetAssignmentRules(<Кластер>, <Сервер>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, из которого получается массив требований.

<Сервер> (обязательный)

Тип: Рабочий сервер.
Рабочий сервер, из которого получаются требования.

Возвращаемое значение:

Тип: COMSafeArray.
Массив объектов Требование назначения сервисов.

Описание:

Получает массив требований назначения сервера, указанного в параметре <Сервер>, и кластера, указанного в параметре <Кластер>. Требования в массиве располагаются в том же порядке, в котором они будут подбираться при назначении новых сервисов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
