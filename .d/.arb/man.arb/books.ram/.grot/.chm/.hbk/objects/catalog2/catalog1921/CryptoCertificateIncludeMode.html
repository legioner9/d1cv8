РежимВключенияСертификатовКриптографии (CryptoCertificateIncludeMode)

Доступен, начиная с версии 8.2.

Значения

ВключатьПолнуюЦепочку (IncludeWholeChain)
ВключатьСертификатСубъекта (IncludeSubjectCertificate)
ВключатьЦепочкуБезКорневого (IncludeChainWithoutRoot)
НеВключать (DontInclude)

Описание:

Содержит варианты режимов включения сертификатов криптографии в подпись.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
