СервисИнтеграцииМенеджер.<Имя сервиса интеграции> (IntegrationServiceManager.<Integration service name>)

Доступен, начиная с версии 8.3.17.

Свойства:

<Имя канала сервиса интеграции> (<Integration service name>)

Методы:

ПолучитьАктивность (GetActive)
ПолучитьНастройки (GetSettings)
СоздатьСообщение (CreateMessage)
УстановитьАктивность (SetActive)
УстановитьНастройки (SetSettings)

Описание:

Описывает сервис интеграции. Позволяет настроить сервис интеграции, создать сообщение, получить каналы сервиса интеграции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
