МенеджерОкнаВнешнегоСайта.ОтправитьСообщение (ExternalSiteWindowManager.PostMessage)

МенеджерОкнаВнешнегоСайта (ExternalSiteWindowManager)

ОтправитьСообщение (PostMessage)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОтправитьСообщение(<Сообщение>)

Параметры:

<Сообщение> (обязательный)

Тип: СообщениеВнешнемуСайту.
Сообщение внешнему сайту.

Описание:

Отправляет сообщение внешнему сайту в случае, если веб-клиент интегрирован в него.
Если окно внешнего сайта не доступно (свойство Доступно имеет значение Ложь), тогда при вызове данного метода будет сгенерировано исключение "Окно внешнего сайта не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
