Глобальный контекст (Global context)

ПолучитьПериодЖурналаРегистрации (GetEventLogPeriod)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьПериодЖурналаРегистрации()

Возвращаемое значение:

Тип: Диапазон.
Начало - дата самого старого события журнала регистрации,
Конец - дата самого нового события журнала регистрации.
Если и Начало, и Конец имеют значение Неопределено, то журнал не содержит ни одного события.

Описание:

Возвращает начало и конец периода, за который журнал регистрации хранит события.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Глобальный контекст, метод СократитьЖурналРегистрации

Использование в версии:

Доступен, начиная с версии 8.3.20.
