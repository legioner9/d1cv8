ПолеКартинки.Масштабировать (PictureBox.Zoomable)

ПолеКартинки (PictureBox)

Масштабировать (Zoomable)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если свойство установлено (Истина), то для картинки, не помещающейся в области размещения, появляются линейки прокрутки.
Изменить размер картинки можно с помощью команд контекстного меню.

Доступность:

Толстый клиент.

Примечание:

Для картинок, размещенных в режиме РеальныйРазмер.

Использование в версии:

Доступен, начиная с версии 8.0.
