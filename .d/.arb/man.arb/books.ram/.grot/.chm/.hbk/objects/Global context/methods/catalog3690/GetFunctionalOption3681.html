Глобальный контекст (Global context)

ПолучитьФункциональнуюОпцию (GetFunctionalOption)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьФункциональнуюОпцию(<Имя>, <Параметры>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя функциональной опции в соответствии с установленными языковыми настройками.

<Параметры> (необязательный)

Тип: Структура.
Параметры функциональных опций, необходимые для получения значения функциональной опции.
Имеет смысл только для функциональных опций, источниками значения которых является реквизит справочника или ресурс регистра сведений. Для опций констант этот параметр игнорируется.
Ключ элемента структуры задает имя параметра, которое должно соответствовать имени одному из объектов метаданных типа ОбъектМетаданных: ПараметрФункциональныхОпций. Значение элемента структуры должно содержать значение параметра.

Возвращаемое значение:

Тип: Произвольный.

Описание:

Получает значения функциональной опции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметры переданы не полностью, то для функциональной опции с параметрами типа Булево учитываются отобранные для указанных параметров значения. Если хотя бы одно из них равно Истина, то и значение опции равно Истина. Для функциональной опции остальных типов генерируется исключительная ситуация.
Если параметры переданы полностью, но подходящего значения не найдено, то значение функциональной опции будет Ложь.
Если в качестве параметра функциональной опции была передана ссылка на группу, а реквизит, в котором хранится значение функциональной опции, не используется для группы, то при получении значения функциональной опции будет возращено Null.

Использование в версии:

Доступен, начиная с версии 8.2.
