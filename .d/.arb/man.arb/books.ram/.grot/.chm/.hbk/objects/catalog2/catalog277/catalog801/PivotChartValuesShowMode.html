ОтображениеЗначенийСводнойДиаграммы (PivotChartValuesShowMode)

ОтображениеЗначенийСводнойДиаграммы (PivotChartValuesShowMode)

Доступен, начиная с версии 8.0.

Значения

ВсеЗначения (AllValues)
ЗначенияПоследнегоУровня (LastLevelValues)

Описание:

Содержит варианты способов представления иерархических измерений в сводной диаграмме.
Вариант ВсеЗначения предусматривает представление в области построения всех значений измерения, независимо от "раскрытости" веток.
Вариант ЗначенияПоследнегоУровня предусматривает представление в области построения значений только последнего раскрытого уровня.
Пример: для измерения "Номенклатура" имеются значения: "Продукты" и подчиненные ему значения "Сыр охотничий" и "Морковь". Значение "Продуты" раскрыто. В случае варианта ЗначенияПоследнегоУровня оно не будет представляться самостоятельно. Это может быть полезно для обеспечения отображения данных сопоставимого (в смысле величины) порядка. Если значение измеримого "Количество" для "Продуктов" равно 1000, а соответствующие значения "Сыра", "Моркови" и т.д. не превышают 10 каждое, то одновременное отображение таких данных в диаграмме не наглядно (в отличие от таблицы).

Доступность:

Тонкий клиент, сервер, толстый клиент.
