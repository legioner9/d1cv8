ВариантыПользовательскогоПоляВыборКомпоновкиДанных.ПолучитьИдентификаторПоОбъекту (DataCompositionUserFieldsCaseVariants.GetIDByObject)

ВариантыПользовательскогоПоляВыборКомпоновкиДанных (DataCompositionUserFieldsCaseVariants)

ПолучитьИдентификаторПоОбъекту (GetIDByObject)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИдентификаторПоОбъекту(<Объект>)

Параметры:

<Объект> (необязательный)

Тип: ВариантПользовательскогоПоляВыборКомпоновкиДанных.
Объект, для которого требуется получить идентификатор.

Возвращаемое значение:

Тип: ИдентификаторКомпоновкиДанных, Неопределено.
Неопределено - переданный объект не является частью данного объекта.

Описание:

Получает идентификатор для переданного объекта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
