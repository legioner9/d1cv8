ПараметрСхемыКомпоновкиДанных.Использование (DataCompositionSchemaParameter.Use)

ПараметрСхемыКомпоновкиДанных (DataCompositionSchemaParameter)

Использование (Use)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеПараметраКомпоновкиДанных.
Если свойство имеет значение Авто, необходимость использования данного параметра получается из настроек компоновки данных.
Если свойство имеет значение Всегда, то параметр всегда считается используемым и всегда используется его значение, указанное в настройках компоновки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
