КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияИнтерфейсом
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Свойства:

<Имя интерфейса> (<Interface name>)

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
ПереключитьИнтерфейс (SwitchInterface)
Получить (Get)

Описание:

Предназначен для управления видимостью командных интерфейсов.

Доступность:

Толстый клиент.

См. также:

Глобальный контекст, свойство ГлавныйИнтерфейс
ЭлементУправленияИнтерфейсом, свойство Переключаемый

Использование в версии:

Доступен, начиная с версии 8.0.
