ПоложениеОпорнойТочкиОтрисовки (PaintingReferencePointPosition)

ПоложениеОпорнойТочкиОтрисовки (PaintingReferencePointPosition)

Доступен, начиная с версии 8.0.

Значения

ЛевоВерх (LeftTop)
ЛевоНиз (LeftBottom)
ЛевоЦентр (LeftCenter)
ПравоВерх (RightTop)
ПравоНиз (RightBottom)
ПравоЦентр (RightCenter)
Центр (Center)
ЦентрВерх (CenterTop)
ЦентрНиз (CenterBottom)

Описание:

Определяет, относительно какой точки будет рисоваться картинка, если точечный объект отображается в режиме картинки.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/geo}. Имя типа XDTO: PaintingReferencePointPosition.
