РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: РазмерКартинки.
Свойство используется, чтобы подогнать размеры картинки под размеры ячейки. Если установлено значение Черепица или Растянуть, то значение свойства ПоложениеТекстаОтносительноКартинки игнорируется и считается равным Поверх. Значение по умолчанию – Пропорционально.
Значение АвтоРазмерБезУчетаМасштаба интерпретируется как АвтоРазмер.
Значение РеальныйРазмерБезУчетаМасштаба интерпретируется как РеальныйРазмер.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.16.
