РезультатЧтенияДанных.ИндексМаркера (ReadDataResult.MarkerIndex)

РезультатЧтенияДанных (ReadDataResult)

ИндексМаркера (MarkerIndex)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: Число.
Индекс найденного маркера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если маркер не найден или операция не предполагает поиска маркера, значение свойства будет равно -1.

Использование в версии:

Доступен, начиная с версии 8.3.9.
