ДокументPDF.ДобавитьОтображаемыйОбъектАсинх (PDFDocument.AddRepresentationObjectAsync)

ДокументPDF (PDFDocument)

ДобавитьОтображаемыйОбъектАсинх (AddRepresentationObjectAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ДобавитьОтображаемыйОбъектАсинх(<ОписаниеОтображаемогоОбъекта>)

Параметры:

<ОписаниеОтображаемогоОбъекта> (обязательный)

Тип: ОписаниеОтображаемогоОбъектаPDF.
Объект, содержащий описание отображения объекта в PDF-документе.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Располагает отображение объекта (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
