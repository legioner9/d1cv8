СвязьНаборовДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetLink)

УсловиеСвязи (LinkConditionExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит выражение, результат которого проверяется перед выполнением связи. Связь будет выполнена, если результат выражения Истина.
Условие связи может быть пустым. В этом случае связь будет выполняться всегда.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
