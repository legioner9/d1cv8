ВложенныеНаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateNestedDataSets)

Доступен, начиная с версии 8.2.

Элементы коллекции:

ВложенныйНаборДанныхМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы.
Возможно обращение к набору посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Вложенные наборы данных макета компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
