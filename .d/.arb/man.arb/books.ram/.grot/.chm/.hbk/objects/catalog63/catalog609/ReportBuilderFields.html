ПоляПостроителяОтчета (ReportBuilderFields)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ПолеПостроителяОтчета
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля.
Возможно обращение к полям посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Поля построителя отчетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ПостроительОтчета, свойство ВыбранныеПоля
