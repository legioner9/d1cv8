ОписаниеКомандыСистемыВзаимодействия (CollaborationSystemCommandDescription)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый ОписаниеКомандыСистемыВзаимодействия(<Команда>, <Представление>)

Параметры:

<Команда> (обязательный)

Тип: Произвольный.
Устанавливает команду.
См. описание свойства Команда.

<Представление> (необязательный)

Тип: Строка.
Представление команды.

Описание:

Создает объект на основании команды и представления.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.24.
