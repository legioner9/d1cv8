ЭлементПланировщика.Загрузить (PlannerItem.Load)

ЭлементПланировщика (PlannerItem)

Загрузить (Load)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Загрузить(<Источник>)

Параметры:

<Источник> (обязательный)

Тип: Структура.
Структура, из которой необходимо заполнить свойства элемента планировщика.

Описание:

Заполняет соответствующие свойства элемента планировщика из значений переданной структуры.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЭлементПланировщика, метод Выгрузить

Использование в версии:

Доступен, начиная с версии 8.3.6.
