АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ДоступныеДляРасширенияМодули (ModulesAvailableForExtension)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Используется, когда расширение всех модулей не разрешено.
Содержит список полных имен объектов конфигурации или модулей, расширение которых разрешено, разделенных ";".
Указание полного имени объекта конфигурации разрешает расширение всех модулей объекта.
Указанием полного имени модуля разрешает расширение конкретного модуля.
Примеры:
ОбщийМодуль.ОбщийМодуль1,
Справочник.Справочник1.Форма.ФормаЭлемента,
Справочник.Справочник1.МодульОбъекта,
Конфигурация.ДемоКонфигурация.МодульСеанса

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
