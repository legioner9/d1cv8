ТипДвустороннейПечати (DuplexPrintingType)

ПереворотВверх (FlipPagesUp)

Доступен, начиная с версии 8.3.1.

Описание:

Использовать двусторонняя печать таким образом, чтобы страницу нужно было переворачивать вверх (переворот выполняется относительно горизонтальной оси листа).

Использование в версии:

Доступен, начиная с версии 8.3.1.
