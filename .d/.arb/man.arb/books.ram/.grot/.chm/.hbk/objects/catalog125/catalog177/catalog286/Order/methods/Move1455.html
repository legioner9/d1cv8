Порядок.Сдвинуть (Order.Move)

Порядок (Order)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс сдвигаемого элемента.

<Смещение> (обязательный)

Тип: Число.
Размер смещения. Положительные значения производят смещение вперед, отрицательное - назад.

Описание:

Сдвигает элемент порядка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
