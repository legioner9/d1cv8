НаборСхемXML.Получить (XMLSchemaSet.Get)

НаборСхемXML (XMLSchemaSet)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс схемы в наборе.

Вариант синтаксиса: По URI пространства имен

Синтаксис:

Получить(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен, для которого нужно получить схему из набора.

Возвращаемое значение:

Тип: СхемаXML.

Описание:

Получает схему из набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
