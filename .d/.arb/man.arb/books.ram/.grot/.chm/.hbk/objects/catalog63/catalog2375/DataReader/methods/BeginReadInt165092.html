ЧтениеДанных.НачатьЧтениеЦелого16 (DataReader.BeginReadInt16)

ЧтениеДанных (DataReader)

НачатьЧтениеЦелого16 (BeginReadInt16)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтениеЦелого16(<ОписаниеОповещения>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <Результат> - тип Число. Результат чтения.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, используемый для чтения числа.
Значение по умолчанию: Неопределено.

Описание:

Начинает чтение целого 16-разрядного цисла.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

ЧтениеДанных, метод ПрочитатьЦелое16
ЧтениеДанных, метод ПрочитатьЦелое16Асинх
