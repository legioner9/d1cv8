ПланыОбменаМенеджер (ExchangePlansManager)

ПрочитатьОписаниеИзменений (ReadChangesDescription)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПрочитатьОписаниеИзменений(<ЧтениеСообщенияОбмена>)

Параметры:

<ЧтениеСообщенияОбмена> (обязательный)

Тип: ЧтениеСообщенияОбмена.
Объект, через который осуществляется чтение сообщения обмена данными. У этого объекта должен быть выполнен метод НачатьЧтение, но еще не выполнен метод ЗакончитьЧтение.
После вызова метода объект не может быть использован в вызове ПрочитатьИзменения.

Возвращаемое значение:

Тип: ОписаниеИзмененийКонфигурацииВСообщенииОбмена.

Описание:

Определяет, какие изменения конфигурации и расширений конфигурации произойдут при чтении сообщения обмена данными.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПланыОбменаМенеджер, метод ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации

Использование в версии:

Доступен, начиная с версии 8.3.22.
