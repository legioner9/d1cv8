ПостроительЗапроса (QueryBuilder)

На основании текста запроса

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ПостроительЗапроса(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.
Исходный текст запроса.

Описание:

Создает объект по тексту запроса построителя запроса.

Использование в версии:

Доступен, начиная с версии 8.0.
