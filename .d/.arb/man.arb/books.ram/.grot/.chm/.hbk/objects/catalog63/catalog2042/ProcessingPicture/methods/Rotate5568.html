ОбрабатываемаяКартинка (ProcessingPicture)

Повернуть (Rotate)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Повернуть(<Угол>)

Параметры:

<Угол> (необязательный)

Тип: Число.
Угол, под которым будет повернута картинка.

Описание:

Поворачивает картинку на заданный угол.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
