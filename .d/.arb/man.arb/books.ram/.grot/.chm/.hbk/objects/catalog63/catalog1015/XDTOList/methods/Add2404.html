СписокXDTO.Добавить (XDTOList.Add)

СписокXDTO (XDTOList)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: ЗначениеXDTO, ОбъектXDTO.
Элемент данных, устанавливаемый в список.

Возвращаемое значение:

Тип: ЗначениеXDTO, ОбъектXDTO.
Добавленный элемент данных.

Описание:

Добавляет элемент данных в конец списка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
