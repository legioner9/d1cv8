Диапазон (Range)

Доступен, начиная с версии 8.1.

Свойства:

Конец (End)
Начало (Begin)

Конструкторы:

По описанию диапазона

Описание:

Используется для задания диапазона блокируемых значений менеджеру блокировок.
Диапазон может быть открыт с одной стороны - для этого одну из границ можно задать значением Неопределено.
Если диапазон не является открытым, то тип значений верхней и нижней границы должен совпадать. Верхняя граница не должна быть меньше, чем нижняя.
Может быть использован:

  • для указания диапазона блокируемых значений менеджеру блокировок,
  • в качестве периода времени, за который журнал регистрации хранит события.

Доступность:

Сервер, внешнее соединение.
Возможен обмен с сервером. Сериализуется.
