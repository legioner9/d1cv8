АвтоПереносСтрок (Wrap)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак автоматического переноса длинных строк текста.
Истина - выполняется автоперенос строк.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, только если свойство МногострочныйРежим имеет значение Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
