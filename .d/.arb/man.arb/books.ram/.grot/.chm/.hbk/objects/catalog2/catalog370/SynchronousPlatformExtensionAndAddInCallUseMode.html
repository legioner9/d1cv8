Доступен, начиная с версии 8.3.8.

Значения

Использовать (Use)
ИспользоватьСПредупреждениями (UseWithWarnings)

Описание:

Содержит варианты проверки использования синхронных методов в конфигурации.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
