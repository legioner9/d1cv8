Администратор (IRegUserInfo)

Доступен, начиная с версии 8.1.

Свойства:

Descr (Descr)
Name (Name)
Password (Password)
PasswordAuthAllowed (PasswordAuthAllowed)
SysAuthAllowed (SysAuthAllowed)
SysUserName (SysUserName)

Описание:

Содержит информацию об администраторе центрального сервера или кластера серверов. Используется для получения информации о администраторах методом GetClusterAdmins или для регистрации новых администраторов кластера методами CreateClusterAdminInfo и RegClusterAdmin.
Представляет собой объект с интерфейсом IRegUserInfo.

Доступность:

Интеграция.

См. также:

Соединение с агентом сервера, метод RegAgentAdmin
Соединение с агентом сервера, метод RegClusterAdmin
Соединение с агентом сервера, метод UnregClusterAdmin

Использование в версии:

Доступен, начиная с версии 8.1.
