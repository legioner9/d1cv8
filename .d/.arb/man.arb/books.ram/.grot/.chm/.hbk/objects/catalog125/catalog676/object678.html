Описание:

Предназначен для управления планом обмена, как объектом конфигурации. С помощью этого объекта осуществляется поиск узлов, создание новых узлов, работа с формами и макетами плана обмена. Доступ к объекту осуществляется через свойства объекта ПланыОбменаМенеджер. Полное имя типа объекта определяется с учетом имени плана обмена конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.
