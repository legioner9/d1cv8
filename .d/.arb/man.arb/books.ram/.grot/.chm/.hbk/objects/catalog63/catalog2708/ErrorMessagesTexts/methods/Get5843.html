Получить (Get)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Получить(<Категория>)

Параметры:

<Категория> (обязательный)

Тип: КатегорияОшибки.
Категория ошибки.

Возвращаемое значение:

Тип: ТекстыСообщенияОбОшибке, Неопределено.
Если значение не найдено, возвращает Неопределено.

Описание:

Получает значение по ключу.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
