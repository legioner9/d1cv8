МетодКластеризации (ClusterizationMethod)

Доступен, начиная с версии 8.0.

Значения

БлижняяСвязь (NearestNeighbor)
ДальняяСвязь (FurthestNeighbor)
КСредних (KMeans)
ЦентрТяжести (Centroid)

Описание:

Содержит варианты методов кластеризации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: ClusterizationMethod.

Использование в версии:

Доступен, начиная с версии 8.0.
