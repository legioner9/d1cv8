КоллекцияВложенийPDF (PDFAttachmentCollection)

Доступен, начиная с версии 8.3.16.

Элементы коллекции:

ВложениеPDF
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементам коллекции посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Предоставляет методы для работы с вложениями в формате PDF.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.17.
