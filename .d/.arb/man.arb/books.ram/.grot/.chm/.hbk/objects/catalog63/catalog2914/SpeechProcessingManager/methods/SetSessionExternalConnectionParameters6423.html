МенеджерРаботыСРечью.УстановитьПараметрыВнешнегоПодключенияСеанса (SpeechProcessingManager.SetSessionExternalConnectionParameters)

МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьПараметрыВнешнегоПодключенияСеанса (SetSessionExternalConnectionParameters)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПараметрыВнешнегоПодключенияСеанса(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: ПараметрыВнешнегоПодключенияРаботыСРечью.
Параметры доступа к облачному сервису распознавания в информационной базы (для текущей области, если для сеанса определены разделители).
Поле Адрес должно быть заполнено обязательно.
Если Токен не установлен, будет получен список публичных моделей.

Описание:

Сохраняет параметры доступа к облачному сервису работы с речью для текущего сеанса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
