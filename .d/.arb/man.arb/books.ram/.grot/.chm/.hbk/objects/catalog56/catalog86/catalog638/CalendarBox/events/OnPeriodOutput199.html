ПолеКалендаря.ПриВыводеПериода (CalendarBox.OnPeriodOutput)

ПолеКалендаря (CalendarBox)

ПриВыводеПериода (OnPeriodOutput)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриВыводеПериода(<ОформлениеПериода>)

Параметры:

<ОформлениеПериода>

Тип: ОформлениеПериода.
Содержит оформление периода.

Описание:

Вызывается в момент, когда в календаре появляется для отображения месяц, для которого оформление еще не запрашивалось.
С помощью параметра можно изменить стандартное оформление.

Доступность:

Толстый клиент.

Примечание:

Для принудительной инициации события для всех видимых месяцев существует метод Обновить.

Использование в версии:

Доступен, начиная с версии 8.0.
