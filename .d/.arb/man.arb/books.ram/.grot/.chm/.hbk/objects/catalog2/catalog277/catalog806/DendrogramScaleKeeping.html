ПоддержкаМасштабаДендрограммы (DendrogramScaleKeeping)

ПоддержкаМасштабаДендрограммы (DendrogramScaleKeeping)

Доступен, начиная с версии 8.0.

Значения

ВсеЭлементы (AllItems)
КоличествоЭлементов (ItemCount)
МинимальнаяШирина (MinimumWidth)

Описание:

Содержит варианты способов поддержки масштаба (количества одновременно отображаемых значений) при изменении размеров области построения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: DendrogramScaleKeeping.

См. также:

Дендрограмма, свойство ПоддержкаМасштаба

Использование в версии:

Доступен, начиная с версии 8.0.
