ЭлементФормаHTML.ТипСодержимого (HTMLFormElement.EncType)

ЭлементФормаHTML (HTMLFormElement)

ТипСодержимого (EncType)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Указывает тип содержимого, используемый для отправки формы HTML на сервер.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
