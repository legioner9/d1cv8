СредстваМультимедиа (MultimediaTools)

ВоспроизвестиЗвуковоеОповещение (PlaySoundAlert)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ВоспроизвестиЗвуковоеОповещение(<Звук>, <Вибрация>)

Параметры:

<Звук> (необязательный)

Тип: ЗвуковоеОповещение, Строка.
Файл звукового оповещения, указанный одним из следующих способов:
  • один из стандартных звуков оповещения системы,
  • текстовая строка, содержащая имя звукового файла без расширения - файл должен входить в состав пакета мобильного приложения (для проекта Anroid mp3-файл должен располагаться в каталоге "assets', для проекта iOS - caf-файл). Если файл с указанным именем отсутствует в пакете приложения, будет сгенерировано исключение о том, что файл не найден.
один из стандартных звуков оповещения системы,
текстовая строка, содержащая имя звукового файла без расширения - файл должен входить в состав пакета мобильного приложения (для проекта Anroid mp3-файл должен располагаться в каталоге "assets', для проекта iOS - caf-файл). Если файл с указанным именем отсутствует в пакете приложения, будет сгенерировано исключение о том, что файл не найден.

Значение по умолчанию: ПоУмолчанию.

<Вибрация> (необязательный)

Тип: Булево, Массив.
Использовать вибрацию. Если установлено Истина, воспроизводится вибрация длительностью 400 мс. Если указан массив целых чисел, то:
  • в Android – массив чисел интерпретируется как шаблон вибрации: длительность пауз и вибраций в миллисекундах (первый элемент – пауза),
  • в iOS – будет воспроизведена стандартная вибрация.

Значение по умолчанию: Ложь.

Описание:

Воспроизвести краткий звуковой сигнал или вибрацию.
В iOS сначала будет подан вибросигнал, а затем воспроизведен звук.
Если в данный момент запущено стороннее приложение, воспроизводящее звук, то мобильное приложение передаст операционной системе, что в течение краткого периода времени будет воспроизведен звук уведомления (реализация зависит от конкретной мобильной операционной системы, реакция других приложений также зависит от их конкретной реализации), после чего начнется воспроизведение.
Если звук в мобильном приложении уже воспроизводится (выполняется метод ВоспроизвестиАудио()), то он будет прерван и будет воспроизведен звук оповещения, после чего воспроизведение звука будет продолжено с места остановки.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
