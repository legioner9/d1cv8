Проведение (Posting)

Доступен, начиная с версии 8.0.

Значения

Запретить (Deny)
Разрешить (Allow)

Описание:

Определяет возможность/невозможность проведения документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
