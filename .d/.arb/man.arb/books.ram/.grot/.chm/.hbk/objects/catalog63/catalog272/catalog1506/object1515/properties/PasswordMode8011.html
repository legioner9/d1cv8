ОбъектМетаданных: Константа.РежимПароля (MetadataObject: Constant.PasswordMode)

ОбъектМетаданных: Константа (MetadataObject: Constant)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.
Устанавливает отображение данных константы в режиме пароля (символы заменяются звездочками при отображении).
Истина - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
