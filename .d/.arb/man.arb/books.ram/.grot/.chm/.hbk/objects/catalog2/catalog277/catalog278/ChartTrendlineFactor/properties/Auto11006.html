ФакторЛинииТрендаДиаграммы (ChartTrendlineFactor)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Фактор устанавливается автоматически в зависимости от значения всех точек диаграммы.
Если значения всех точек являются датами или числами, то используется ЗначениеТочки.
В противном случае используется НомерТочки.

Использование в версии:

Доступен, начиная с версии 8.3.10.
