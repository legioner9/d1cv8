Расширение поля формы для поля текстового документа (Form field extension for a text document)

Ширина (Width)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит ширину текстового документа. Задается в символах.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
