Значение (Value)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Представляет значение атрибутов fixed или default объявления атрибута (http://www.w3.org/TR/xmlschema-1/#a-value_constraint).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
