HTTPСервисОтвет.ПолучитьТелоКакПоток (HTTPServiceResponse.GetBodyAsStream)

HTTPСервисОтвет (HTTPServiceResponse)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: Поток.

Описание:

Получает поток для записи тела ответа.
Полученный поток можно использовать как для чтения тела ответа, если тело уже было установлено с помощью других методов, так и для записи тела.

Доступность:

Сервер, толстый клиент.

Доступен, начиная с версии 8.3.10.
