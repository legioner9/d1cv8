Расширение формы клиентского приложения для отчета (Client application form extension for reports)

УстановитьТекущиеПользовательскиеНастройки (SetCurrentUserSettings)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьТекущиеПользовательскиеНастройки(<КлючПользовательскихНастроек>)

Параметры:

<КлючПользовательскихНастроек> (обязательный)

Тип: Произвольный.
Ключ пользовательских настроек, которые нужно загрузить.

Описание:

Загружает пользовательские настройки для текущего варианта. Заполняет свойства КлючТекущихПользовательскихНастроек, ПредставлениеТекущихПользовательскихНастроек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для использования метода требуется, чтобы для пользователя было доступно право СохранениеДанныхПользователя.

Использование в версии:

Доступен, начиная с версии 8.2.
