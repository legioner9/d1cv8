Расширение таблицы формы для вариантов пользовательского поля выбора компоновки данных (Form table extension for data composition options of user choice field)

Доступен, начиная с версии 8.2.

Описание:

Дополнительные свойства элемента управления ТаблицаФормы, применимые для вариантов пользовательского поля выбора компоновки данных.
В качестве значений для свойств ТекущаяСтрока, ТекущийРодитель и элемента коллекции, содержащейся в свойстве ВыделенныеСтроки, используется идентификатор типа ИдентификаторКомпоновкиДанных.
Свойство ТекущиеДанные и метод ДанныеСтроки возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
