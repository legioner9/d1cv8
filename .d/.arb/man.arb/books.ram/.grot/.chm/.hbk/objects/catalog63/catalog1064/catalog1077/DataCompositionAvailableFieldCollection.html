КоллекцияДоступныхПолейКомпоновкиДанных (DataCompositionAvailableFieldCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ДоступноеПолеКомпоновкиДанных, ДоступноеПолеОтбораКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля.
Возможно обращение к доступному полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Коллекция доступных полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ДоступноеПолеКомпоновкиДанных, свойство Элементы
ДоступныеПоляКомпоновкиДанных, свойство Элементы
ДоступноеПолеОтбораКомпоновкиДанных, свойство Элементы

Использование в версии:

Доступен, начиная с версии 8.1.
