ВложенныеСхемыКомпоновкиДанных (NestedDataCompositionSchemas)

ВложенныеСхемыКомпоновкиДанных (NestedDataCompositionSchemas)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ВложеннаяСхемаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются вложенные схемы.
Возможно обращение к вложенной схеме посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит вложенные схемы компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

СхемаКомпоновкиДанных, свойство ВложенныеСхемыКомпоновкиДанных

Использование в версии:

Доступен, начиная с версии 8.1.
