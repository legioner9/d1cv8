Глобальный контекст (Global context)

ПодключитьРасширениеПолученияИнформацииОКомпьютереАсинх (AttachComputerInformationExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПодключитьРасширениеПолученияИнформацииОКомпьютереАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - значение типа Булево, содержащее результат подключения.

Описание:

Асинхронная функция. Подключает расширение для работы с информацией о компьютере (MAC-адреса) в браузер.
В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
