ПоложениеОтметокШкалыДиаграммы (ChartScaleMarkLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Положение отметок определяется автоматически:
  • если подписи диаграммы отображаются снаружи -- Снаружи,
  • если подписи диаграммы отображаются внутри -- Внутри.

Использование в версии:

Доступен, начиная с версии 8.3.11.
