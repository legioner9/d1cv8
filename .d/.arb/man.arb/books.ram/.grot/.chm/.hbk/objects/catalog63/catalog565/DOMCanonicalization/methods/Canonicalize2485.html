КаноническийDOM.Канонизировать (DOMCanonicalization.Canonicalize)

КаноническийDOM (DOMCanonicalization)

Канонизировать (Canonicalize)

Не рекомендуется использовать, начиная с версии 8.3.9.

Доступен, начиная с версии 8.1.

Синтаксис:

Канонизировать()

Описание:

Записывает узел DOM в каноническом виде.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.9.
