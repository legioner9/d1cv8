ИнтернетТекстПочтовогоСообщения (InternetMailText)

Кодировка (Encoding)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит кодировку текстовых данных сообщения. Если кодировка не указана, будет использоваться значение кодировки из свойства Кодировка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
