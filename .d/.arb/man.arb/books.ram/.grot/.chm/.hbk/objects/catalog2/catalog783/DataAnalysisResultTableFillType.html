ТипЗаполненияТаблицыРезультатаАнализаДанных (DataAnalysisResultTableFillType)

Доступен, начиная с версии 8.0.

Значения

ВсеПоля (AllFields)
ИспользуемыеПоля (UsedFields)
КлючевыеПоля (KeyFields)
НеЗаполнять (DontFill)

Описание:

Содержит варианты типов заполнения результирующей таблицы анализа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: DataAnalysisResultTableFillType.

Использование в версии:

Доступен, начиная с версии 8.0.
