МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ПолучитьРежимЧтенияЗаписиХранилищаДвоичныхДанных (GetBinaryDataStorageReadWriteMode)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьРежимЧтенияЗаписиХранилищаДвоичныхДанных()

Возвращаемое значение:

Тип: РежимЧтенияЗаписиХранилищаДвоичныхДанных.

Описание:

Возвращает режим работы (только для чтения, чтение и запись) хранилища двоичных данных.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
