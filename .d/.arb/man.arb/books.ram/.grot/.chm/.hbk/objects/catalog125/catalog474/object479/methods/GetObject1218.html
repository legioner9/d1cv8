ПланВидовРасчетаВыборка.<Имя плана видов расчета> (ChartOfCalculationTypesSelection.<Chart of calculation types name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: ПланВидовРасчетаОбъект.<Имя плана видов расчета>, Неопределено.
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает текущий для выборки объект базы данных, предназначенный для чтения, изменения, добавления и удаления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
