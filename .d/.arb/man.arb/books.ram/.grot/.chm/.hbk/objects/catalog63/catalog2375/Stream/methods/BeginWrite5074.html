Поток.НачатьЗапись (Stream.BeginWrite)

Поток (Stream)

НачатьЗапись (BeginWrite)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <Буфер>, <ПозицияВБуфере>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения записи потока со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Буфер> (обязательный)

Тип: БуферДвоичныхДанных.
Буфер, данные которого получаются для записи.

<ПозицияВБуфере> (обязательный)

Тип: Число.
Позиция в буфере, начиная с которой требуется получить данные для записи в поток.

<Количество> (обязательный)

Тип: Число.
Количество байт, которые требуется записать.

Описание:

Начинает запись в поток заданного количества байтов из буфера по заданному смещению.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Запись возможна только если поток поддерживает запись. В противном случае при вызове метода будет вызвано исключение.
