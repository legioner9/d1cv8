ГруппаФормы.ОтображениеПодсказки (FormGroup.ToolTipRepresentation)

ГруппаФормы (FormGroup)

ОтображениеПодсказки (ToolTipRepresentation)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: ОтображениеПодсказки.
Задает способ отображения подсказки у элемента формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.1.
