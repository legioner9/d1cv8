МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ПолучитьУниверсальнуюДатуСозданияРезервнойКопии (GetBackupCreationUniversalDate)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьУниверсальнуюДатуСозданияРезервнойКопии(<Путь>)

Параметры:

<Путь> (обязательный)

Тип: Строка.
Путь к файлу с резервной копией.

Возвращаемое значение:

Тип: Дата.
Дата и время начала создания резервной копии в формате UTC.

Описание:

Получает дату начала создания указанной резервной копии хранилища двоичных данных.

Доступность:

Сервер.

Примечание:

Исключения возможны в следующих случаях:
  • При вызове из сеанса, в котором определены разделители,
  • При попытке выполнения на версиях продукта, не поддерживающих хранилище двоичных данных,
  • Ошибка хранилища двоичных данных,
  • Хранилище двоичных данных выключено. Включите хранилище двоичных данных, чтобы операции с хранилищем были доступны.

Использование в версии:

Доступен, начиная с версии 8.3.24.
