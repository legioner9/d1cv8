БотСистемыВзаимодействия (CollaborationSystemBot)

Записать (Write)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Записать()

Описание:

Записывает бота в информационную базу.
При записи существующего бота (полученного с помощью методом ПолучитьБота или ПолучитьБотов), если изменено свойство Пользователь, то вначале удаляется запись существующего бота.
Требует наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
