Лево (Left)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит положение левой границы.
Может содержать значения от 0 до 1.
0 - привязана к левому краю, 1 - к правому.
Используется вместе со свойством УказываетсяЛевоИВерх.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
