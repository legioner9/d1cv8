Описание:

Содержит варианты типов единицы интервала времени. Значение интервала времени определяется по указанному значению времени с учетом кратности.
Типы единицы интервала времени условно делятся на две группы. К первой относятся типы, в названии которых имеется префикс "Текущий", "Текущая" или "Текущее"; ко второй - остальные.
Для первой группы начало интервала определяется как первая секунда момента времени, а конец интервала - как последняя первая секунда момента времени, определенного в соответствии с выбранным типом и указанной кратностью. Например, для типа ТекущийЧас, кратности 2 (часа) и указанного момента времени 12.05.2004 12:34:11 начало интервала будет 12.05.2004 10:00:00, а окончание 12.05.2004 13:59:59.
Для второй группы начало интервала определяется как момент времени, вычисленный как разность значения указанного момента и размера кратности в секундах. Окончание интервала определяется как момент времени, вычисленный как сумма значения указанного момента и размера кратности в секундах. Например, для типа ТекущаяМинута, кратности 4 (минуты) и указанного момента времени 12.05.2004 12:34:11 начало интервала будет 12.05.2004 12:30:11, а окончание 12.05.2004 12:38:11.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: DataAnalysisTimeIntervalUnitType.

Использование в версии:

Доступен, начиная с версии 8.0.
