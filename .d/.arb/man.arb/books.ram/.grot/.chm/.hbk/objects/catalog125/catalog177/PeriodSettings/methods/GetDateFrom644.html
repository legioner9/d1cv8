НастройкаПериода.ПолучитьДатуНачала (PeriodSettings.GetDateFrom)

НастройкаПериода (PeriodSettings)

ПолучитьДатуНачала (GetDateFrom)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДатуНачала()

Возвращаемое значение:

Тип: Дата.

Описание:

Получает дату начала периода.

Доступность:

Толстый клиент.

Примечание:

Если граница интервала задана как БезОграничения, то будет возвращена пустая дата.

Пример:

НастройкаПериода = Новый НастройкаПериода;
НастройкаПериода.УстановитьПериод(ДатаНачалаПериода, ДатаОкончанияПериода);
Если НастройкаПериода.Редактировать() Тогда
    ДатаНачалаПериода = НастройкаПериода.ПолучитьДатуНачала();
    ДатаОкончанияПериода = НастройкаПериода.ПолучитьДатуОкончания();
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
