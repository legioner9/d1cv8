СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ДатаУстаревания (ExpirationDate)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: Дата.
Дата и время, после которого считается, что сообщение устарело и не должно обрабатываться.
Можно устанавливать только при отправке.
Значение свойства устанавливается в методе, создающем сообщение.
При попытке изменить в полученном сообщении генерируется исключение.
Хранится и передается в формате UTC. При получении преобразуется во время в часовом поясе сеанса, при установке преобразуется из времени в часовом поясе сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.20.
