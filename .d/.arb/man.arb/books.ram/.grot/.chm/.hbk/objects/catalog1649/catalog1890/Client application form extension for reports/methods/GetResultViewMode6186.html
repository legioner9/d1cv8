Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПолучитьРежимОтображенияРезультата (GetResultViewMode)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПолучитьРежимОтображенияРезультата()

Возвращаемое значение:

Тип: РежимОтображенияРезультатаОтчета.

Описание:

Возвращает значение режима отображения с учетом настроек, сохраняемых в системном хранилище.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
