ТокенДоступа.Получатели (AccessToken.Recipients)

ТокенДоступа (AccessToken)

Получатели (Recipients)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Массив.
Массив строк, который содержит идентификаторы получателей токена. Соответствует ключу 'aud'.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
