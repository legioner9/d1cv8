ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunAdditionalData)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Ключ>, <Значение>, <ИмяТипа>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.
Указывает ключ добавляемого элемента данных.

<Значение> (обязательный)

Тип: Строка, Число, Булево, Картинка, УчетнаяЗаписьКонтактов, ДанныеАдреса, ДанныеМестоположения, Массив, ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства.
Добавляемое значение дополнительных данных.
Массив может содержать элементы одного из следующих типов: Строка, Булево, Число.

<ИмяТипа> (необязательный)

Тип: Строка.
Указывает имя типа добавляемого данного.

Возвращаемое значение:

Тип: ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства.

Описание:

Добавляет элемент дополнительных данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
