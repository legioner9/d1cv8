ПодписьКриптографии (CryptoSignature)

АрхивныеМеткиВремени (ArchiveTimestamps)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: Массив.
Массив элементов типа МеткаВремениКриптографии.

Содержит архивные метки времени подписи - необходимый для CAdES-A атрибут.
Для CAdESAv2 и CAdESAv3 в данном поле содержатся различные типы меток времени.

Если метка отсутствует значение поля - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
