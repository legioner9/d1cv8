Глобальный контекст (Global context)

ОбработкаОтключенияВнешнейКомпонентыПриОшибке (AddInDetachmentOnErrorProcessing)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ОбработкаОтключенияВнешнейКомпонентыПриОшибке(<Местоположение>, <Имя>)

Параметры:

<Местоположение>

Тип: Строка.
Строка, определяющая местоположение внешней компоненты.
Соответствует первому параметру метода ПодключитьВнешнююКомпоненту.

<Имя>

Тип: Строка.
Символическое имя подключаемой внешней компоненты.
Соответствует второму параметру метода ПодключитьВнешнююКомпоненту.

Описание:

Вызывается при падении процесса внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
