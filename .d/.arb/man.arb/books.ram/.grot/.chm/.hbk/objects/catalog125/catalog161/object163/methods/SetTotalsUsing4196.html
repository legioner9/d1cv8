РегистрСведенийМенеджер.<Имя регистра сведений> (InformationRegisterManager.<Information register name>)

УстановитьИспользованиеИтогов (SetTotalsUsing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УстановитьИспользованиеИтогов(<Признак>)

Параметры:

<Признак> (обязательный)

Тип: Булево.
Признак использования итогов. Ложь - пересчет итогов не производится.

Описание:

Устанавливает признак использования итогов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
