CreateSecurityProfileUnSafeExternalModule (CreateSecurityProfileUnSafeExternalModule)

CreateSecurityProfileUnSafeExternalModule()

Тип: Внешний модуль.

Создает новый внешний модуль для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступен, начиная с версии 8.3.3.
