Расширение группы формы для страницы (Form group extension for a page)

ГоризонтальныйИнтервал (HorizontalSpacing)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: ИнтервалМеждуЭлементамиФормы.
Устанавливает горизонтальный интервал между элементами.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
