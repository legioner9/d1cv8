БизнесПроцессМенеджер.<Имя бизнес-процесса>.ПустаяСсылкаНаТочкуМаршрута (BusinessProcessManager.<Business process name>.EmptyRoutePointRef)

БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ПустаяСсылкаНаТочкуМаршрута (EmptyRoutePointRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылкаНаТочкуМаршрута()

Возвращаемое значение:

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.

Описание:

Получает пустое значение ссылки на точку маршрута бизнес-процесса данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
