ОписанияМакетовСхемыКомпоновкиДанных (DataCompositionSchemaTemplateDescriptions)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ОписаниеМакетаСхемыКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания макета.
Возможно обращение к описанию посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция описаний макетов схемы компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

СхемаКомпоновкиДанных, свойство Макеты

Использование в версии:

Доступен, начиная с версии 8.1.
