МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьДоступныеКлючи (GetAvailableKeys)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьДоступныеКлючи(<IPАдресАгента>, <Порт>)

Параметры:

<IPАдресАгента> (необязательный)

Тип: Строка.
Адрес агента сервера 1С:Предприятия.
Если пустая строка, то текущий процесс. В этом случае значение параметра <Порт> игнорируется.
Значение по умолчанию: Пустая строка.

<Порт> (необязательный)

Тип: Число.
IP-порт агента сервера 1С:Предприятия.
Если 0, то используется значение "1540".
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: Массив.
Массив элементов типа ПараметрыПривязкиККомпьютеруПолученияЛицензий.

Описание:

Получает список ключей защиты программы, доступных на текущем компьютере или на компьютере с указанным адресом процесса агента сервера 1С:Предприятия.
В случае ошибки обращения к агенту сервера генерируется исключение.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
