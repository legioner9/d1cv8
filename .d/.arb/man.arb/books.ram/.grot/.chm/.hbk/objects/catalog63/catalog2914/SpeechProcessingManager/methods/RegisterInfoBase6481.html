МенеджерРаботыСРечью (SpeechProcessingManager)

ВыполнитьРегистрациюИнформационнойБазы (RegisterInfoBase)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВыполнитьРегистрациюИнформационнойБазы(<Пользователь>, <Пароль>, <АдресСервера>)

Параметры:

<Пользователь> (обязательный)

Тип: Строка.
Логин пользователя для аутентификации в сервисе работы с речью.

<Пароль> (обязательный)

Тип: Строка.
Пароль пользователя для аутентификации в сервисе работы с речью.

<АдресСервера> (необязательный)

Тип: Строка.
Адрес сервиса работы с речью.
Значение по умолчанию зависит от региона использования.

Описание:

Формирует данные подключения к внешнему сервису работы с речью и сохраняет их в информационной базе.
Требуется наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
