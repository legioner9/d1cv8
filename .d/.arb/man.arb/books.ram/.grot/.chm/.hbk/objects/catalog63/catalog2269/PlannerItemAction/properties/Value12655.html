ДействиеЭлементаПланировщика (PlannerItemAction)

Значение (Value)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Содержит значение действия. Используется для поиска с помощью метода Найти коллекции действий элемента планировщика.
Значение по умолчанию: Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
