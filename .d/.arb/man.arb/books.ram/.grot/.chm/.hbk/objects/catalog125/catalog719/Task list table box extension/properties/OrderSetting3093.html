Расширение табличного поля списка задач (Task list table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: НастройкаПорядка.
Предоставляет доступ к настройкам порядка списка задач. Представляет собой коллекцию значений типа ЭлементУправленияПорядком. Свойства коллекции содержат объекты для управления элементом порядка. Имена свойств совпадают с именами доступных для задачи элементов упорядочивания.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
