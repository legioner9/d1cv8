Расширение поля формы для поля диаграммы (Form field extension for a chart field)

ПриАктивизации (OnActivate)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПриАктивизации()

Описание:

Возникает при изменении выделенных элементов диаграммы с помощью мыши.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
