ОбластьЛегендыСводнойДиаграммы (PivotChartLegendArea)

Доступен, начиная с версии 8.1.

Свойства:

ПрозрачныйФон (Transparent)
Прокрутка (Scrolling)
Рамка (Border)
Расположение (Placement)
ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области легенды сводной диаграммы для настройки параметров оформления. Легенда сводной диаграммы предназначена для описания серий сводной диаграммы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
