ТипОбъектаXDTO.Смешанный (XDTOObjectType.Mixed)

ТипОбъектаXDTO (XDTOObjectType)

Смешанный (Mixed)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит признак наличия у объектов данного типа смешанного содержания (текст и элементы).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - тип смешанный.

Использование в версии:

Доступен, начиная с версии 8.1.
