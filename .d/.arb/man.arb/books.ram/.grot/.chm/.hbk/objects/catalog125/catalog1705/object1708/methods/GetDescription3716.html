ХранилищеНастроекМенеджер.<Имя хранилища>.ПолучитьОписание (SettingsStorageManager.<Storage name>.GetDescription)

ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ПолучитьОписание (GetDescription)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОписание(<КлючОбъекта>, <КлючНастроек>, <ИмяПользователя>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: Произвольный.
Ключ объекта настройки.

<КлючНастроек> (необязательный)

Тип: Произвольный.
Ключ настроек.

<ИмяПользователя> (необязательный)

Тип: Строка.
Имя пользователя, описание настроек которого получается.
Если не указан, то получается описание настроек текущего пользователя.

Возвращаемое значение:

Тип: ОписаниеНастроек.

Описание:

Получает описание настроек в хранилище.
Вызывает обработчик ОбработкаПолученияОписания.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
