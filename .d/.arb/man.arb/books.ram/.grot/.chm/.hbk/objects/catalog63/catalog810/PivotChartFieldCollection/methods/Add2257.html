КоллекцияПолейСводнойДиаграммы.Добавить (PivotChartFieldCollection.Add)

КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, КоллекцияПолейСводнойДиаграммы, ПолеСводнойДиаграммы.
Добавляемое поле.

Возвращаемое значение:

Тип: КоллекцияПолейСводнойДиаграммы, ПолеСводнойДиаграммы.

Описание:

Добавляет поле или другую коллекцию в конец коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
