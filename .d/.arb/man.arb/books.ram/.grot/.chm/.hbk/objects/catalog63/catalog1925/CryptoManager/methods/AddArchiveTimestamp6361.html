МенеджерКриптографии.ДобавитьАрхивнуюМеткуВремени (CryptoManager.AddArchiveTimestamp)

МенеджерКриптографии (CryptoManager)

ДобавитьАрхивнуюМеткуВремени (AddArchiveTimestamp)

Доступен, начиная с версии 8.3.20.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ДобавитьАрхивнуюМеткуВремени(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как ДвоичныеДанные, Поток, ПотокВПамяти или ФайловыйПоток.

Описание варианта метода:

Данные подписи возвращаются методом как ДвоичныеДанные.

Вариант синтаксиса: Данные подписи сохраняются в файл

Синтаксис:

ДобавитьАрхивнуюМеткуВремени(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как ДвоичныеДанные, Поток, ПотокВПамяти или ФайловыйПоток.

<ВыходныеДанные> (обязательный)

Тип: Строка, Поток, ПотокВПамяти, ФайловыйПоток.
Имя файла или поток, в который будут выгружены данные подписи.
Если Неопределено, то результат работы метода вернется в возвращаемом значении типа ДвоичныеДанные.

Описание варианта метода:

Данные подписи записываются в файл. Возвращает Неопределено.

Возвращаемое значение:

Тип: Неопределено, ДвоичныеДанные.
Двоичные данные подписи с добавленной архивной меткой времени (если параметр <ВыходныеДанные> имеет значение Неопределено) иначе Неопределено.

Описание:

Добавляет в подпись архивную метку времени версии 3.
При добавлении метки выполняется обращение к серверам меток времени, указанным в поле АдресаСерверовМетокВремени.

Для добавления архивной метки времени подпись должна иметь тип CAdES-BES или выше.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

МенеджерКриптографии.Подписать(
    ИсходныеДанные, РезультатПодписи, Сертификат, ТипПодписиКриптографии.CAdEST); 
МенеджерКриптографии.ДобавитьАрхивнуюМеткуВремени(РезультатПодписи, ПодписьCAdESA);

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
