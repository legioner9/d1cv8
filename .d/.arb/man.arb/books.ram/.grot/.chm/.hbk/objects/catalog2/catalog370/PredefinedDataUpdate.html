ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Доступен, начиная с версии 8.3.3.

Значения

Авто (Auto)
НеОбновлятьАвтоматически (DontAutoUpdate)
ОбновлятьАвтоматически (AutoUpdate)

Описание:

Содержит варианты режимов обновления предопределенных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
