ДокументDOM.СоздатьВыражениеXPath (DOMDocument.CreateXPathExpression)

ДокументDOM (DOMDocument)

СоздатьВыражениеXPath (CreateXPathExpression)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьВыражениеXPath(<Выражение>, <РазыменованиеПространствИмен>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.
Строка, определяющая выражение XPath.

<РазыменованиеПространствИмен> (обязательный)

Тип: РазыменовательПространствИменDOM.
Объект для разрешения пространств имен в выражении XPath.

Возвращаемое значение:

Тип: ВыражениеXPath.

Описание:

Создание выражения XPath.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если выражение содержит ошибки, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
