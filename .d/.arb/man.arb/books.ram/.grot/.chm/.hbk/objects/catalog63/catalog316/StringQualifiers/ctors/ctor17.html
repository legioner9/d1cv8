КвалификаторыСтроки (StringQualifiers)

На основании описания строки

Доступен, начиная с версии 8.0.

Синтаксис:

Новый КвалификаторыСтроки(<ДлинаСтроки>, <ДопустимаяДлина>)

Параметры:

<ДлинаСтроки> (необязательный)

Тип: Число.
Допустимая длина строки. Если параметр не указан, то строка по длине не ограничивается.
Значение по умолчанию: 0.

<ДопустимаяДлина> (необязательный)

Тип: ДопустимаяДлина.
Вариант ограничения длины строки, если задана длина строки, отличная от 0.
Значение по умолчанию: Переменная.

Описание:

Создает квалификатор строки заданием длины строки.

Пример:

ПараметрыСтроки = Новый КвалификаторыСтроки(50);

Использование в версии:

Доступен, начиная с версии 8.0.
