ЗаписьТекста (TextWriter)

Доступен, начиная с версии 8.0.

Методы:

Закрыть (Close)
Записать (Write)
ЗаписатьСтроку (WriteLine)
Открыть (Open)

Конструкторы:

На основании потока
По имени файла
Формирование неинициализированного объекта

Описание:

Предназначен для последовательной записи в текстовые файлы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Текст = Новый ЗаписьТекста("d:\win.txt", КодировкаТекста.ANSI);
Текст.ЗаписатьСтроку("Добро пожаловать!");
Текст.ЗаписатьСтроку("Посторонним вход воспрещен.");
Текст.Закрыть();
