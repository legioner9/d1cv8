ЧтениеДанных (DataReader)

ПрочитатьАсинх (ReadAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьАсинх(<Количество>)

Параметры:

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется прочитать. Если не задано, то выполняется чтение всех данных до конца потока.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа РезультатЧтенияДанных.

Описание:

Асинхронная функция. Читает двоичные данные заданного размера из потока. Если размер не задан, то будут прочитаны все данные до конца потока.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

ЧтениеДанных, метод Прочитать
ЧтениеДанных, метод НачатьЧтение

Доступен, начиная с версии 8.3.18.
