ОписанияХарактеристик (CharacteristicsDescriptions)

Доступен, начиная с версии 8.2.

Элементы коллекции:

ОписаниеХарактеристик
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания характеристик.
Возможно обращение к описанию характеристики посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Индекс (IndexOf)
Количество (Count)
Получить (Get)

Описание:

Содержит коллекцию объектов ОписаниеХарактеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
