МенеджерСистемыВзаимодействия (CollaborationSystemManager)

АдресСервера (ServerAddress)

Доступен, начиная с версии 8.3.23.

Синтаксис:

АдресСервера()

Возвращаемое значение:

Тип: Строка.

Описание:

Возвращает адрес сервера системы взаимодействия, на котором зарегистрирована информационная база.
Если система взаимодействия не зарегистрирована, то возвращает пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
