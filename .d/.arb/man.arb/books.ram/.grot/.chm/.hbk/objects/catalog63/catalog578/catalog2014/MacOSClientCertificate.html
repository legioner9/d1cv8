СертификатКлиентаMacOS (MacOSClientCertificate)

Доступен, начиная с версии 8.3.13.

Свойства:

СпособВыбораСертификатаMacOS (MacOSCertificateSelectMode)

Конструкторы:

По способу выбора сертификата
По умолчанию

Описание:

Объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы macOS.
Поддерживаются хранилища сертификатов КриптоПро CSP. Поиск библиотек КриптоПро выполняется в стандартных каталогах. Также в файле conf.cfg, в параметре CryptoAPILibraryLocation можно указать пути к файлу библиотеки или список путей, разделенных точками с запятой.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
