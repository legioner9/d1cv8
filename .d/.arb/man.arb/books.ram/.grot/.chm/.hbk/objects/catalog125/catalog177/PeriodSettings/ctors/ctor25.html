НастройкаПериода (PeriodSettings)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый НастройкаПериода()

Описание:

Создает новую настройку периода. По умолчанию у созданного объекта свойства определены следующим образом:

  • РедактироватьКакИнтервал - Истина;
  • РедактироватьКакПериод - Истина;
  • ВариантНачала- БезОграничения;
  • ВариантОкончания - БезОграничения;
  • ВариантНастройки - Интервал.
Также может быть создан с использованием функциональной формы оператора Новый.

Пример:

Период = Новый (Тип("НастройкаПериода"));
Период.ВариантНастройки = ВариантНастройкиПериода.Интервал;
Период.УстановитьПериод(НачалоДня(ДатаНач), КонецДня(ДатаКон)));

Использование в версии:

Доступен, начиная с версии 8.0.
