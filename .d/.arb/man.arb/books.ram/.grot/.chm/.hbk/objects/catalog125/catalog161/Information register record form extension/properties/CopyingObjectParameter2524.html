Расширение формы записи регистра сведений (Information register record form extension)

ПараметрОбъектКопирования (CopyingObjectParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: РегистрСведенийМенеджерЗаписи.<Имя регистра сведений>.
Параметр заполняется значением типа РегистрСведенийМенеджерЗаписи.<Имя регистра сведений> для копируемой строки регистра сведений.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
