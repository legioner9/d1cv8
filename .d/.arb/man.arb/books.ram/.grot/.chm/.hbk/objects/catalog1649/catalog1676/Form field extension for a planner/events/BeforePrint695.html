Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: РежимИспользованияДиалогаПечати.
Указывает способ вызова печати:
  • НеИспользовать - Если вызвана команда "Печать с текущими настройками",
  • Использовать - если вызвана команда "Печать".

<Отказ>

Тип: Булево.
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение Истина, печать выполнена не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает на клиенте перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.19.
