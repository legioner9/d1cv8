РазрешенныйCOMКласс (AllowedCOMClass)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового разрешенного COM-класса или обновление существующего. COM-класс определяется своим идентификатором, по которому он может быть создан операционной системой, и/или текстовым представлением объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
