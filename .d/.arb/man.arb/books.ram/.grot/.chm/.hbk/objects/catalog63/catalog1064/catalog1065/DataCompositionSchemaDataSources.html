ИсточникиДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSources)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ИсточникДанныхСхемыКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются источники.
Возможно обращение к источнику посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

<Имя источника> (<Source name>)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция источников данных схемы компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

СхемаКомпоновкиДанных, свойство ИсточникиДанных

Использование в версии:

Доступен, начиная с версии 8.1.
