Глобальный контекст (Global context)

СоздатьОбъектВнешнейКомпонентыАсинх (CreateAddInObjectAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СоздатьОбъектВнешнейКомпонентыАсинх(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: Строка, Тип.
Имя типа или значение типа Тип.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания – объект внешней компоненты.

Описание:

Асинхронная функция. Создает объект внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
