СписокЗначений (ValueList)

НайтиПоИдентификатору (FindByID)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: Число.
Идентификатор элемента списка значений.

Возвращаемое значение:

Тип: ЭлементСпискаЗначений, Неопределено.
Неопределено - элемент не найден.

Описание:

Получает элемент списка значений по идентификатору.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
