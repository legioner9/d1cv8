ЧтениеДанных.Пропустить (DataReader.Skip)

ЧтениеДанных (DataReader)

Пропустить (Skip)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Пропустить(<Количество>)

Параметры:

<Количество> (обязательный)

Тип: Число.
Количество байтов, которые требуется пропустить.

Возвращаемое значение:

Тип: Число.
Количество фактически пропущенных данных.

Описание:

Пропускает при чтении указанное количество байтов в потоке.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ЧтениеДанных, метод НачатьПропуск
ЧтениеДанных, метод ПропуститьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
