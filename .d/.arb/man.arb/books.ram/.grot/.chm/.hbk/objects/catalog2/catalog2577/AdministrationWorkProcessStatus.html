Доступен, начиная с версии 8.3.14.

Значения

Используется (Used)
НеИспользуется (NotUsed)

Описание:

Определяет использование рабочего процесса кластером.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
