ТипШтрихКода.Codabar (BarcodeType.Codabar)

ТипШтрихКода (BarcodeType)

Codabar (Codabar)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Codabar. Позволяет кодировать цифры от 0 до 9, символы -, $, :, /, ., + и четыре буквы (A, B, C, D).

Использование в версии:

Доступен, начиная с версии 8.3.25.
