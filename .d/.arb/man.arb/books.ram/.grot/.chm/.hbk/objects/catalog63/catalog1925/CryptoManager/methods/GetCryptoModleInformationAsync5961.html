МенеджерКриптографии (CryptoManager)

ПолучитьИнформациюМодуляКриптографииАсинх (GetCryptoModleInformationAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьИнформациюМодуляКриптографииАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа ИнформацияМодуляКриптографии.

Описание:

Асинхронная функция. Получает информацию о модуле криптографии, если была выполнена инициализация объекта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
