Доступен, начиная с версии 8.1.

Значения

ИсключающийКаноническийXML (XMLExclusiveCanonicalization)

Описание:

Содержит типы алгоритма приведения XML к каноническому виду.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
