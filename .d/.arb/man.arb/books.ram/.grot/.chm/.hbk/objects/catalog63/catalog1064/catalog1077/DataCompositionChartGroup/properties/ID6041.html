ГруппировкаДиаграммыКомпоновкиДанных (DataCompositionChartGroup)

Идентификатор (ID)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Идентификатор. Используется при обработке расшифровки для идентификации группировки в данных расшифровки.
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
