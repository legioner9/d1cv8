Диаграмма.НачальныйУгол (Chart.BeginAngle)

Диаграмма (Chart)

НачальныйУгол (BeginAngle)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: Число, Неопределено.
Угол начала отображения секторов по часовой стрелке от вертикальной оси.
Используется в круговой и кольцевой диаграммах.
Значение Неопределено интерпретируется в зависимости от типа диаграммы: для круговой и кольцевой диаграммы - 0.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.24.
