ОриентацияЭлементаФормы (FormItemOrientation)

Доступен, начиная с версии 8.2.

Значения

Вертикально (Vertical)
Горизонтально (Horizontal)

Описание:

Содержит варианты ориентации элемента формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение поля формы для поля индикатора, свойство Ориентация
Расширение поля формы для поля полосы регулирования, свойство Ориентация

Использование в версии:

Доступен, начиная с версии 8.2.
