ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ОбъектМетаданныхКонфигурация(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: ДвоичныеДанные.
Двоичные данные файла, из которого будет сформирован объект ОбъектМетаданныхКонфигурация.

Описание:

Создает объект на основе двоичных данных файла в формате *.cf или *.cfe.

См. также:

РасширениеКонфигурации, метод ПолучитьДанные

Использование в версии:

Доступен, начиная с версии 8.3.13.
