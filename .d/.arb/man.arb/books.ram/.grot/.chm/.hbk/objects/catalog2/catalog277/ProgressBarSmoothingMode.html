РежимСглаживанияИндикатора (ProgressBarSmoothingMode)

РежимСглаживанияИндикатора (ProgressBarSmoothingMode)

Доступен, начиная с версии 8.0.

Значения

Плавный (Smooth)
Прерывистый (Broken)
ПрерывистыйНаклонный (BrokenTilt)

Описание:

Содержит варианты стилей заполнения индикатора. Используется для определения свойства СтильОтображения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

См. также:

Индикатор, свойство СтильОтображения
Расширение поля формы для поля индикатора, свойство Отображение

Использование в версии:

Доступен, начиная с версии 8.0.
