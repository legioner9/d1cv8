ИнтернетПочта.Послать (InternetMail.Send)

ИнтернетПочта (InternetMail)

Послать (Send)

Доступен, начиная с версии 8.0.

Синтаксис:

Послать(<Сообщение>, <ОбрабатыватьТексты>, <ПротоколОтправкиПочты>)

Параметры:

<Сообщение> (обязательный)

Тип: ИнтернетПочтовоеСообщение.
Почтовое сообщение.

<ОбрабатыватьТексты> (необязательный)

Тип: ОбработкаТекстаИнтернетПочтовогоСообщения.
Управляет обработкой текстов перед отправкой:
  • Обрабатывать - Локальные картинки из текста, помещаются в сообщение в виде вложений. Содержимое текста письма изменяется таким образом, чтобы теги <img> указывали на картинки из вложений.
  • НеОбрабатывать - Текст посылается без внутренней обработки.
  • Загрузка картинок при отправке HTML-документов отключена.

Значение по умолчанию: Обрабатывать.

<ПротоколОтправкиПочты> (необязательный)

Тип: ПротоколИнтернетПочты.
Определяет тип протокола отправки. Допустимые варианты IMAP или SMTP.
Указание POP3 приведет к возникновению исключения.
Значение по умолчанию: SMTP.

Возвращаемое значение:

Тип: Соответствие.
Если почтовый сервер не принял какой-нибудь из адресов получателей, то возвращаемое значение содержит соответствие ошибочный получатель - сообщение об ошибке.
При возникновении других ошибочных ситуаций, генерируется исключение.

Описание:

Посылает сообщение типа ИнтернетПочтовоеСообщение.
При отправке сообщения, которое содержит html-текст (свойство ТипТекста установлено в значение HTML) также формируется простой текст. Этот текст извлекается из html-текста c некоторыми особенностями. Например, командой перевода строки считается тег <DIV>. Тег <BR> как перевод строки не трактуется и игнорируется.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если используется IMAP протокол, сообщение будет послано в почтовый ящик IMAP сессии, который задан свойством ТекущийПочтовыйЯщик.

Пример:

Почта = Новый ИнтернетПочта;
Сообщение = Новый ИнтернетПочтовоеСообщение;
Почта.Подключиться(Профиль);
ОшибочныеПолучатели = Почта.Послать(Сообщение);
Почта.Отключиться();

Использование в версии:

Доступен, начиная с версии 8.0.
