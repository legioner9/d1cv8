ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionFilterExpressions)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

ВыражениеОтбораКомпоновкиДанныхСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения отбора.
Возможно обращение к выражению отбора посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Индексированная коллекция элементов типа ВыражениеОтбораКомпоновкиДанныхСхемыЗапроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
