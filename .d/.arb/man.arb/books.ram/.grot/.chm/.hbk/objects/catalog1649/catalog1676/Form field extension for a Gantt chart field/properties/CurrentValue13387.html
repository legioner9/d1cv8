Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ТекущееЗначение (CurrentValue)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: ИдентификаторЗначенияДиаграммыГанта.
Содержит идентификатор выделенного значения диаграммы. Если устанавливаемый идентификатор значения не является допустимым для данной диаграммы, то такая установка игнорируется.
Значение по умолчанию - Неопределено.

При изменении значения данного свойства меняется значение свойства ТекущаяСтрока таблицы формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
