СоставФункциональнойОпции (FunctionalOptionContent)

Доступен, начиная с версии 8.2.

Элементы коллекции:

ЭлементСоставаФункциональнойОпции
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (начиная с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)
Содержит (Contains)

Описание:

Коллекция, cодержащая информацию о назначенных функциональной опции объектах метаданных. Элемент коллекции представляет собой специальный тип. Значение этого типа содержит ссылку на объект метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ОбъектМетаданных: ФункциональнаяОпция, свойство Состав

Использование в версии:

Доступен, начиная с версии 8.2.
