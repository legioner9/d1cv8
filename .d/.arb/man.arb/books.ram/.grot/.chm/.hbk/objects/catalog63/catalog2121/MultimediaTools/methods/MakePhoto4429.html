СредстваМультимедиа.СделатьФотоснимок (MultimediaTools.MakePhoto)

СредстваМультимедиа (MultimediaTools)

СделатьФотоснимок (MakePhoto)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СделатьФотоснимок(<ТипКамеры>, <Разрешение>, <Качество>, <ЧерноБелый>, <ТипПодсветки>, <Отметка>, <АвтоПоворот>)

Параметры:

<ТипКамеры> (необязательный)

Тип: ТипКамерыУстройства.
Показывает, какой камерой будет сделан снимок.
Значение по умолчанию: Авто.

<Разрешение> (необязательный)

Тип: РазрешениеКамерыУстройства.
Определяет разрешение, с которым будет сделан снимок.
Значение по умолчанию: 0;0 (параметр не будет использован при открытии камеры):
  • если задано разрешение, не поддерживаемое устройством, будет использован ближайшее подходящее разрешение.
  • если задано отрицательное разрешение, то будет инициировано исключение.
В устройствах под управлением iOS параметр игнорируется.

<Качество> (необязательный)

Тип: Число.
Определяет качество снимка в процентах.
Допустимый интервал значений: 1 - 100.
По умолчанию - 100 % (съемка в максимальном качестве).
Если значение не входит в допустимый интервал, будет вызвано исключение.

<ЧерноБелый> (необязательный)

Тип: Булево.
Если параметр установлен в значение "Да" (Истина), фотоснимок будет сделан монохромным. В противном случае, монохромность не используется.
По умолчанию - "Нет".
В устройствах под управлением iOS при создании фотоснимка эффект монохромности на камеру не накладывается, но фотография будет получена в монохромном формате.

<ТипПодсветки> (необязательный)

Тип: ТипПодсветкиКамеры.
Определяет начальное использование подсветки камеры во время создания снимка.
Значение по умолчанию: Авто.

<Отметка> (необязательный)

Тип: ОтметкаНаФотоснимке.
Если передано значение Неопределено, то на фото не будет отображена подпись. Если передан объект типа ОтметкаНаФотоснимке, то в нижнем правом углу фотографии (согласно ее ориентации) будет размещена подпись.
Значение по умолчанию: Неопределено.

<АвтоПоворот> (необязательный)

Тип: Булево.
Если параметр установлен в значение Ложь, фотоснимок останется в том виде, в каком его предоставила камера мобильного устройства. Если параметр установлен в значение Истина, полученный фотоснимок будет автоматически поворачиваться под текущее положение экрана устройства.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: ДанныеМультимедиа, Неопределено.
Неопределено - запись не выполнена.

Описание:

Выполняет фотоснимок встроенной фото-видео камерой.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяФотоснимок.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:
