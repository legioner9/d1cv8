ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: ПланВидовРасчетаСсылка.<Имя плана видов расчета>.

Описание:

Получает пустое значение ссылки на план видов расчета данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, в тех случаях, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
