Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ВыполнитьВыборИзМенюРасшифровки (ExecuteChoiceFromDetailsMenu)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ВыполнитьВыборИзМенюРасшифровки(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Строка, Число.
Представление выбираемого значения или номер значения (индекс строки, нумерация с 0).

Описание:

Выбирает значение из открытого меню расшифровки по его представлению в меню.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
