ИспользованиеСреза (SliceUse)

Доступен, начиная с версии 8.0.

Значения

НеИспользовать (DontUse)
Первые (First)
Последние (Last)

Описание:

Определяет набор вариантов формирования среза периодического регистра сведений. Используется для заполнения свойства ВыбиратьСрез.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

Расширение табличного поля списка записей регистра сведений, свойство ВыбиратьСрез

Использование в версии:

Доступен, начиная с версии 8.0.
