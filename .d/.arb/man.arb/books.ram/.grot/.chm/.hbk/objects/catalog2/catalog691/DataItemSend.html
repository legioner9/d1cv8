ОтправкаЭлементаДанных (DataItemSend)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
Игнорировать (Ignore)
Удалить (Delete)

Описание:

Содержит варианты значений выходных параметров обработчиков событий ПриОтправкеДанныхГлавному и ПриОтправкеДанныхПодчиненному объектов ПланОбменаОбъект.<Имя плана обмена>. Данное значение определяет действия системы относительно элемента данных при формировании сообщения, содержащего изменения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

См. также:

ПланОбменаОбъект.<Имя плана обмена>, событие ПриОтправкеДанныхПодчиненному
ПланОбменаОбъект.<Имя плана обмена>, событие ПриОтправкеДанныхГлавному

Использование в версии:

Доступен, начиная с версии 8.0.
