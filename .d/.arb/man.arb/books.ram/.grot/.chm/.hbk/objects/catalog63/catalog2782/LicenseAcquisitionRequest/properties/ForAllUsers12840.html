ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

ДляВсехПользователей (ForAllUsers)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Управляет размещением файла программной лицензии:
  • Истина - файл программной лицензии будет записан для всех пользователей. Обычно используются каталоги:
    • Windows - "C:\ProgramData\1C\licenses"
    • Linux - "/var/1C/licenses".
  • Ложь - файл будет записан в каталог поиска лицензии текущего пользователя. Обычно используются каталоги:
    • Windows - "C:\Users\"Имя пользователя"\AppData\Local\1C\1cv8\conf",
    • Linux - "~\1C\1cv8\conf".

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
