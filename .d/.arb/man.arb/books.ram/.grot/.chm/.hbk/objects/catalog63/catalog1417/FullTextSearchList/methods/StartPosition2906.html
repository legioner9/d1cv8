НачальнаяПозиция (StartPosition)

Доступен, начиная с версии 8.1.

Синтаксис:

НачальнаяПозиция()

Возвращаемое значение:

Тип: Число.

Описание:

Начальная позиция поиска. После вызова метода ПерваяЧасть равна 0. При вызове методов СледующаяЧасть, ПредыдущаяЧасть, начальная позиция будет меняться.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
