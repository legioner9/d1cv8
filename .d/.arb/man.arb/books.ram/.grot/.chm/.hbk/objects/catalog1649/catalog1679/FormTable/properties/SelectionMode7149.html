ТаблицаФормы.РежимВыделения (FormTable.SelectionMode)

ТаблицаФормы (FormTable)

РежимВыделения (SelectionMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: РежимВыделенияТаблицы.
Определяет режим выделения строк в таблице.
Для таблицы, связанной с диаграммой Ганта:
  • если установлено значение Множественный, то РежимВыделенияЗначений диаграммы Ганта принимает значение Множественный, и наоборот;
  • если свойству установлено значение Одиночный, то свойство РежимВыделенияЗначений диаграммы Ганта принимает значение Одиночный, и наоборот.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
