ОбластьОформления (AppearanceArea)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементОбластиОформления
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы области оформления.

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
ПолучитьДоступныеПоля (GetAvailableFields)
Удалить (Delete)
УстановитьДоступныеПоля (SetAvailableFields)

Конструкторы:

По умолчанию

Описание:

Содержит описание области, к которой применяется оформление.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Сериализуется.
