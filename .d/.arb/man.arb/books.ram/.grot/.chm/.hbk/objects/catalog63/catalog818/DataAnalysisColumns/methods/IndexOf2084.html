КолонкиАнализаДанных.Индекс (DataAnalysisColumns.IndexOf)

КолонкиАнализаДанных (DataAnalysisColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: КолонкаАнализаДанных.
Искомая колонка.

Возвращаемое значение:

Тип: Число.
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
