СоставКопииБазыДанных (DatabaseCopyContent)

Получить (Get)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс элемента состава.

Возвращаемое значение:

Тип: ЭлементСоставаКопииБазыДанных.

Описание:

Получает элемент состава копии по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
