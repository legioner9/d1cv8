ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ЗапрещатьНезаполненныеЗначения (DenyIncompleteValues)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если свойство установлено в Истина, то запись движений регистра с пустым значением этого измерения будет невозможна.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для измерений регистра бухгалтерии, регистра накопления, регистра расчета, регистра сведений.

Использование в версии:

Доступен, начиная с версии 8.1.
