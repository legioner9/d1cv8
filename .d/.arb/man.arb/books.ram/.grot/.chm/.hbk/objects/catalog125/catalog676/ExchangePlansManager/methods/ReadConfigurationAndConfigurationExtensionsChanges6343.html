ПланыОбменаМенеджер (ExchangePlansManager)

ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации (ReadConfigurationAndConfigurationExtensionsChanges)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации(<ЧтениеСообщенияОбмена>)

Параметры:

<ЧтениеСообщенияОбмена> (обязательный)

Тип: ЧтениеСообщенияОбмена.
Объект, через который осуществляется чтение сообщения обмена данными. У этого объекта должен быть выполнен метод НачатьЧтение, но еще не выполнен метод ЗакончитьЧтение.
После вызова метода объект может быть использован в вызове ПрочитатьИзменения, если вызов не привёл к изменениям конфигурации, расширений конфигурации или возникновению исключения.

Возвращаемое значение:

Тип: ОписаниеИзмененийКонфигурацииВСообщенииОбмена.
Для этого метода флаг РасширенияКонфигурацииИзменены означает, что расширения конфигурации были изменены и требуется перезапуск сеанса для их применения
Если флаг ИзмененныеМетаданныеРасширенийКонфигурацииИзменяютСтруктуруДанных установлен, а РасширенияКонфигурацииИзменены – нет, это значит, что расширения содержали изменения структуры и не смогли быть применены либо из-за невозможности установить монопольную блокировку, либо из-за запуска метода в фоновом задании.

Описание:

Выполняет чтение из сообщения обмена данными изменений конфигурации и расширений конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПланыОбменаМенеджер, метод ПрочитатьОписаниеИзменений

Использование в версии:

Доступен, начиная с версии 8.3.22.
