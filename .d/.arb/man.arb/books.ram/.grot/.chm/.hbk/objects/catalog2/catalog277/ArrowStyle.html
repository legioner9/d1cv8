СтильСтрелки (ArrowStyle)

Доступен, начиная с версии 8.0.

Значения

Заполненная (Filled)
Незаполненная (Blank)
Нет (None)

Описание:

Содержит варианты стилей стрелки на конце или в начале декоративной линии графической схемы.

Доступность:

Тонкий клиент, сервер, толстый клиент.

См. также:

ЭлементГрафическойСхемыДекоративнаяЛиния, свойство СтрелкаНачала
ЭлементГрафическойСхемыДекоративнаяЛиния, свойство СтрелкаКонца

Использование в версии:

Доступен, начиная с версии 8.0.
