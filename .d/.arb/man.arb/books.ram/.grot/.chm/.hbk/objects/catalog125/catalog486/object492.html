РегистрРасчетаКлючЗаписи.<Имя регистра расчета> (CalculationRegisterRecordKey.<Calculation register name>)

Доступен, начиная с версии 8.0.

Свойства:

НомерСтроки (LineNumber)
Регистратор (Recorder)

Описание:

Представляет собой набор значений, однозначно идентифицирующих запись регистра расчета. Он используется в тех случаях, когда необходимо сослаться на определенную запись.
Например, этот объект выступает в качестве значение свойства ТекущаяСтрока табличного поля отображающего список записей регистра расчета.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise/current-config}. Имя типа XDTO: CalculationRegisterRecordKey.<Имя регистра расчета>.

См. также:

РегистрРасчетаМенеджер.<Имя регистра расчета>, метод СоздатьКлючЗаписи

Использование в версии:

Доступен, начиная с версии 8.0.
