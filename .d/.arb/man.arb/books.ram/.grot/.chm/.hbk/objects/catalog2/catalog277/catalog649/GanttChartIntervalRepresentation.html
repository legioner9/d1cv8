ОтображениеИнтервалаДиаграммыГанта (GanttChartIntervalRepresentation)

Доступен, начиная с версии 8.0.

Значения

Градиент (Gradient)
Объемный (ThreeDimensional)
Плоский (Flat)
Ромб (Rhomb)

Описание:

Содержит варианты видов интервалов при отображении.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChartIntervalRepresentation.

См. также:

ДиаграммаГанта, свойство ОтображениеИнтервала

Доступен, начиная с версии 8.0.
