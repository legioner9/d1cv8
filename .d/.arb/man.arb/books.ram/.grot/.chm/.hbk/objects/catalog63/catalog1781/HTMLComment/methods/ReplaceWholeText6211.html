КомментарийHTML.ЗаменитьПолныйТекст (HTMLComment.ReplaceWholeText)

КомментарийHTML (HTMLComment)

ЗаменитьПолныйТекст (ReplaceWholeText)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.2.

Синтаксис:

ЗаменитьПолныйТекст(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: Строка.
Новый текст.

Возвращаемое значение:

Тип: ТекстDOM, Неопределено.
Неопределено, если замещающий текст пустой.
Созданный узел:
  • данный узел, исключая случай, когда данный узел в режиме Только для чтения;
  • новый текстовый узел того же типа, что и данный.

Описание:

Производит замену текста у данного узла и логически смежных текстовых узлов. При этом удаляются все логически смежные текстовые узлы, включая данный. Создается новый текстовый узел с типом, соответствующим данному, ему устанавливается новый текст, и вставляется в дерево документа на место данного.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.2.
