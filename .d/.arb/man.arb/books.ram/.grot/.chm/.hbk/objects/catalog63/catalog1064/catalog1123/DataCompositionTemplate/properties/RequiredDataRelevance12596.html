Описание:

Тип: ТребуемаяАктуальностьДанных.
Актуальность данных, которые необходимо получить.
Если свойство ИспользованиеКопийБазыДанных имеет значение ИспользоватьТолькоКопии, то свойство может влиять только на то, какая именно копия будет выбрана.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
