Описание:

Содержит варианты стандартной навигационной ссылки формы. Используется в качестве значений ключа вариантов стандартных навигационных ссылок, передаваемых обработчику ОбработкаПолученияСпискаНавигационныхСсылок.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/managed-application/logform. Имя типа XDTO:
