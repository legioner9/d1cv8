КнопкаПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButton)

Доступность (Enabled)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если свойство имеет значение Ложь, тогда такая кнопка отображается недоступной.
Если свойство имеет значение Ложь и кнопка используется в интеграциях с мессенджерами, где не поддерживается возможность сделать кнопку недоступной, тогда такая кнопка не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
