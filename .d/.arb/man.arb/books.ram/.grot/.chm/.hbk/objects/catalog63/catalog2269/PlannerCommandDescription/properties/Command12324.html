ОписаниеКомандыПланировщика (PlannerCommandDescription)

Команда (Command)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

  • Команда

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: СтандартнаяКомандаПланировщика, ОписаниеОповещения, Массив, Неопределено.
Содержит описание команды:
  • СтандартнаяКомандаПланировщика - стандартная команда;
  • ОписаниеОповещения – процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
    • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения;
  • Массив – задает подменю. Массив может содержать элементы обозначенных выше типов;
  • Неопределено – разделитель;
  • Любой другой тип - пункт меню не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
