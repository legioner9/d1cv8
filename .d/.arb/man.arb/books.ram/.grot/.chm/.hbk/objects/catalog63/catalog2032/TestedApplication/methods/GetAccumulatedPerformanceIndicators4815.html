ТестируемоеПриложение (TestedApplication)

ПолучитьНакопленныеПоказателиПроизводительности (GetAccumulatedPerformanceIndicators)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьНакопленныеПоказателиПроизводительности(<ОчиститьНакопленные>)

Параметры:

<ОчиститьНакопленные> (необязательный)

Тип: Булево.
Очищает информацию о накопленных вызовах в клиенте тестирования перед возвратом.
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: Структура.
Структура, содержащая следующие поля:
  • Вызовы (Calls) – Тип значения – Число. Количество вызовов сервера с момента последней очистки истории накопленных.
  • Длительность (Duration) – Тип значения – Число. Время, затраченное на все вызовы с момента последней очистки истории накопленных.
  • Отправлено (Sent) – Тип значения – Число. Количество отправленных байт с момента последней очистки истории накопленных.
  • Принято (Received) – Тип значения – Число. Количество принятых байт с момента последней очистки истории накопленных.

Имена полей зависят от варианта встроенного языка, выбранного на стороне менеджера тестирования.

Описание:

Получает информацию о производительности тестируемого клиента.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
