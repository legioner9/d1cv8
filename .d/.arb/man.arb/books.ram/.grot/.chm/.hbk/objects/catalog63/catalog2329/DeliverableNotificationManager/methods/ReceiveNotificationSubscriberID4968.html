МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ПолучитьИдентификаторПодписчикаУведомлений (ReceiveNotificationSubscriberID)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьИдентификаторПодписчикаУведомлений(<НомерПриложенияGoogleCloud>)

Параметры:

<НомерПриложенияGoogleCloud> (необязательный)

Тип: Строка, Массив.
Номер приложения или приложений Google Cloud Apps, который используется для реализации отправителей push-уведомлений для сервиса Firebase Cloud Messaging (GCM).
Если массив содержит более 100 элементов, также будет выдана ошибка.
На платформе Android параметр является необязательным.
На платформах iOS и Windows Phone параметр игнорируется.

Возвращаемое значение:

Тип: ИдентификаторПодписчикаДоставляемыхУведомлений.
На Android, если задан, возвращается идентификатор GCM (Google Cloud Messaging), иначе - FCM (Firebase Cloud Messaging).

Описание:

Получает уникальный идентификатор подписчика. При использовании следует учитывать, что идентификатор при определенных условиях может быть изменен. Рекомендуется получать его регулярно, например, в начале каждой сессии.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
