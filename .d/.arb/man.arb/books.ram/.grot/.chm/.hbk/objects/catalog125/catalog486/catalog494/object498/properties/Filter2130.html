ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Отбор.
Предназначено для установки фильтрации записей при считывании или записи набора. Для набора записей перерасчета возможен отбор только по ОбъектПерерасчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для установки фильтрации используется стандартный объект, через который устанавливается предопределенный элемент отбора.

Использование в версии:

Доступен, начиная с версии 8.0.
