ИнтернетПочта.ОбъединитьЧастичные (InternetMail.CombinePartial)

ИнтернетПочта (InternetMail)

ОбъединитьЧастичные (CombinePartial)

Доступен, начиная с версии 8.1.

Синтаксис:

ОбъединитьЧастичные(<МассивСообщений>)

Параметры:

<МассивСообщений> (обязательный)

Тип: Массив.
Массив, содержащий частичные сообщения для объединения.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ИнтернетПочтовоеСообщение.

Описание:

Объединяет частичные сообщения. Если в переданном массиве находятся не все части, то сообщение не объединяется. В передаваемом массиве могут находится части нескольких сообщений. После объединения они возвратятся в массиве. У каждого объединенного сообщения свойство Идентификатор будет содержать несколько значений, соответствующих взятым из частичных сообщений при объединении.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Профиль = Новый ИнтернетПочтовыйПрофиль;
ЗаполнитьПрофиль(Профиль);

Почта = Новый ИнтернетПочта;
Почта.Подключиться(Профиль);

Масс = Почта.Выбрать(Ложь);

МассЧ = Новый Массив;
Для Ин = 0 По Масс.Количество() - 1 Цикл
    
    Если Масс[Ин].Частичное Тогда
        
        МассЧ.Добавить(Масс[Ин]);
        
    КонецЕсли;
        
КонецЦикла;

Сообщения = Почта.ОбъединитьЧастичные(МассЧ);
Сообщить(" **************************************** ");
Для Инд = 0 По Сообщения.Количество() - 1 Цикл
    
    Для Каждого Идентификатор Из Сообщения[Инд].Идентификатор Цикл
        Сообщить("Серверный идентификатор " + Строка(Инд) + ": " + Идентификатор);
    КонецЦикла;
    
КонецЦикла;

Почта.Отключиться();
