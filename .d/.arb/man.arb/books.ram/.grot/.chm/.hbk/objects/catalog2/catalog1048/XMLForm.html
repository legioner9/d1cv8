ФормаXML (XMLForm)

Доступен, начиная с версии 8.1.

Значения

Атрибут (Attribute)
Текст (Text)
Элемент (Element)

Описание:

Содержит варианты форм представления свойства объекта XDTO в формате XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/xdto}. Имя типа XDTO: Form.
