ПоказываемаяОбластьГеографическойСхемы (GeographicalSchemaShowedArea)

Доступен, начиная с версии 8.1.

Свойства:

ДолготаВосточнойГраницы (EastBorderLongitude)
ДолготаЗападнойГраницы (WestBorderLongitude)
ШиротаСевернойГраницы (NorthBorderLatitude)
ШиротаЮжнойГраницы (SouthBorderLatitude)

Описание:

Предоставляет доступ к границам показываемой области, в режиме масштабирования ЗаданнаяОбласть.

Доступность:

Толстый клиент.

Пример:

// Получим ограничивающий прямоугольник объекта

ОграничивающийПрямоугольник = Объект.ОграничивающийПрямоугольник;

ПоказываемаяОбласть = 
    ЭлементыФормы.ПолеГеографическойСхемы.ПоказываемаяОбласть;

// Установим у схемы новые границы ограничивающего прямоугольника

ПоказываемаяОбласть.ДолготаВосточнойГраницы = 
    ОграничивающийПрямоугольник.ДолготаВосточнойГраницы;
ПоказываемаяОбласть.ДолготаЗападнойГраницы = 
    ОграничивающийПрямоугольник.ДолготаЗападнойГраницы;
ПоказываемаяОбласть.ШиротаСевернойГраницы = 
    ОграничивающийПрямоугольник.ШиротаСевернойГраницы;
ПоказываемаяОбласть.ШиротаЮжнойГраницы = 
    ОграничивающийПрямоугольник.ШиротаЮжнойГраницы;

// Установим новый режим масштабирования

Схема.ПоддержкаМасштаба = 
    РежимОтображенияГеографическойСхемы.ЗаданнаяОбласть

См. также:

РежимОтображенияГеографическойСхемы, свойство ЗаданнаяОбласть
ПолеГеографическойСхемы, свойство ОтображаемаяОбласть

Использование в версии:

Доступен, начиная с версии 8.1.
