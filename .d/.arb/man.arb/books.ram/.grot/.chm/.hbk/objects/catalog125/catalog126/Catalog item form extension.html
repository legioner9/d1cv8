Расширение формы элемента справочника (Catalog item form extension)

Доступен, начиная с версии 8.0.

Свойства:

Автонумерация (Autonumeration)
ИспользованиеДляГруппИЭлементов (UseForFoldersAndItems)
ПараметрОбъектКопирования (CopyingObjectParameter)
ПараметрОснование (BasisParameter)

Методы:

ЗаписатьВФорме (WriteInForm)

События:

ПередЗаписью (BeforeWrite)
ПослеЗаписи (AfterWrite)
ПриЗаписи (OnWrite)
ПриИзмененииДанных (OnDataChange)

Описание:

Дополнительные свойства, методы и события объекта Форма элемента или группы справочника.

Доступность:

Толстый клиент.

Пример:

Процедура ПередЗаписью(Отказ)
    ТипыНоменклатуры = Перечисления.ТипыНоменклатуры;
    Если ТоварСсылка.ТипНоменклатуры=ТипыНоменклатуры.ПустаяСсылка() Тогда
        Отказ = Истина;
        Предупреждение(НСтр("ru = 'Не выбран тип номенклатуры';"
          + " en = 'Nomenclature type is not selected'"),60);
    КонецЕсли;
КонецПроцедуры

Использование в версии:

Доступен, начиная с версии 8.0.
