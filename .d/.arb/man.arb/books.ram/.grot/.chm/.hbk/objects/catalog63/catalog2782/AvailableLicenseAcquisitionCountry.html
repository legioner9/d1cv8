ДопустимаяСтранаПолученияЛицензий (AvailableLicenseAcquisitionCountry)

Доступен, начиная с версии 8.3.20.

Свойства:

Код (Code)
Представление (Presentation)

Описание:

Содержит код и локализованное наименование страны.
Рекомендуется использовать двухбуквенный код страны.
При повторном получении лицензии, полученной платформой версии 8.3.10 и ниже (в 2017 году и раньше), требуется указание наименования страны, как оно было указано при первоначальном получении лицензии.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

Глобальный контекст, метод ПолучитьДопустимыеСтраны

Использование в версии:

Доступен, начиная с версии 8.3.20.
