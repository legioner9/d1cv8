Соединение с агентом сервера.UpdateInfoBase (IServerAgentConnection.UpdateInfoBase)

Соединение с агентом сервера (IServerAgentConnection)

UpdateInfoBase (UpdateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateInfoBase(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, в котором должны быть изменены параметры информационной базы.

<ИнформационнаяБаза> (обязательный)

Тип: Описание информационной базы.
Новое описание информационной базы. Изменено может быть только значение свойства Descr.

Описание:

Позволяет изменить параметры информационной базы, зарегистрированной в кластере. Изменять можно только словесное описание информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
