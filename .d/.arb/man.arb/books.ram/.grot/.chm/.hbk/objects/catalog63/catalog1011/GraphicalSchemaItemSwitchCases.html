Элементы коллекции:

ВариантЭлементаГрафическойСхемыВыборВарианта
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы (варианты) элемента графической схемы "Выбор варианта".
Возможно обращение к элементу (варианту) элемента графической схемы "Выбор варианта" посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента в коллекции.

Свойства:

<Имя элемента> (<Element name>)

Методы:

Количество (Count)
Получить (Get)

Описание:

Представляет коллекцию вариантов элемента графической схемы ЭлементГрафическойСхемыВыборВарианта.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
