РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

ДобавитьКредит (AddCredit)

Доступен, начиная с версии 8.0.

Синтаксис:

ДобавитьКредит()

Описание:

Добавляет запись с установленным видом движения "Кредит".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл только для регистров без корреспонденции.

Использование в версии:

Доступен, начиная с версии 8.0.
