ДействиеПостроителяDOM (DOMBuilderAction)

Доступен, начиная с версии 8.1.

Значения

ВставитьПеред (InsertBefore)
ВставитьПосле (InsertAfter)
ДобавитьКакДочерние (AppendAsChildren)
Заменить (Replace)
ЗаменитьДочерние (ReplaceChildren)

Описание:

Содержит варианты действий, которые можно выполнить при создании узла в контексте документа DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПостроительDOM, метод ПрочитатьВКонтексте

Использование в версии:

Доступен, начиная с версии 8.1.
