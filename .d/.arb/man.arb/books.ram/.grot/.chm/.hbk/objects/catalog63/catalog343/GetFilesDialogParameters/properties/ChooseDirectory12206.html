ПараметрыДиалогаПолученияФайлов (GetFilesDialogParameters)

ВыборКаталога (ChooseDirectory)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет, будет ли открыт диалог для выбора каталога. Если значение Ложь, то диалог будет открыт для получения каждого файла.
В веб-клиенте значение Истина игнорируется.
Значение по умолчанию: Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
