XBase.КоличествоЗаписей (XBase.RecCount)

XBase (XBase)

КоличествоЗаписей (RecCount)

Доступен, начиная с версии 8.0.

Синтаксис:

КоличествоЗаписей()

Возвращаемое значение:

Тип: Число.
Количество записей в базе.

Описание:

Получает количество записей в таблице базы данных вместе с помеченными на удаление записями.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

Предупреждение(НСтр("ru='Записей - '; en='Records - '")+Строка(xB.КоличествоЗаписей()));

Использование в версии:

Доступен, начиная с версии 8.0.
