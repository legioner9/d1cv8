ПараметрПеретаскиванияВнутриПланировщика (PlannerInsideDragParameter)

Элемент (Item)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: ЭлементПланировщика, Неопределено.
Содержит ссылку на редактируемый или копируемый элемент планировщика или Неопределено в случае, когда происходит выделение области планировщика.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
