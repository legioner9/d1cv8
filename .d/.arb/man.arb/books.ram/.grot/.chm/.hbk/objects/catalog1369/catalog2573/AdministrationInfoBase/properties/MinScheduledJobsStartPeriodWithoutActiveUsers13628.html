АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

МинимальныйПериодЗапускаРегламентныхЗаданийБезАктивныхПользователей (MinScheduledJobsStartPeriodWithoutActiveUsers)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Определяет минимальный период запуска каждого регламентного задания информационной базы в секундах, если в информационной базе нет ни одного не спящего сеанса пользователей.
Число в диапазоне: 0…4294967.
Значение по умолчанию: 0.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
