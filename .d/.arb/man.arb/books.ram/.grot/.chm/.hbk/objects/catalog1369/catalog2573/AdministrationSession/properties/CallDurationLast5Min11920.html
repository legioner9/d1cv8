АдминистрированиеСеанс (AdministrationSession)

ДлительностьВызововЗа5Мин (CallDurationLast5Min)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит время исполнения вызовов сервера "1С:Предприятия" от имени данного сеанса за последние 5 минут, в миллисекундах.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
