МеткаNDEF (TagNDEF)

ПоддерживаетсяЗаписьНаМеткуАсинх (WriteOnTagSupportedAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяЗаписьНаМеткуАсинх()

Возвращаемое значение:

Тип: Булево.
Истина - физическая метка перезаписываемя и запись на метку не заблокирована,
Ложь - в противном случае.

Описание:

Проверяет, что физическая NFC-метка доступна для записи. Если произошла ошибка, генерируется исключение категории ОшибкаСредствМультимедиа.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерМетокNDEF, метод ПоддерживаетсяЗаписьНаМетку

Использование в версии:

Доступен, начиная с версии 8.3.23.
