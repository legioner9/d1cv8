ПараметрыЗаписиJSON.ЭкранироватьОдинарныеКавычки (JSONWriterSettings.EscapeSingleQuotes)

ПараметрыЗаписиJSON (JSONWriterSettings)

ЭкранироватьОдинарныеКавычки (EscapeSingleQuotes)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Булево.
Определяет, будет ли экранирован знак одинарной кавычки при записи документа JSON.
Имеет значение Истина, если ИспользоватьДвойныеКавычки установлен Ложь.
После создания объекта данное свойство имеет значение Ложь

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
