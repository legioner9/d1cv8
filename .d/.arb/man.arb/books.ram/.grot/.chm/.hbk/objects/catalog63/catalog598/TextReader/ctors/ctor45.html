ЧтениеТекста (TextReader)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЧтениеТекста()

Описание:

Формирует неинициализированный объект ЧтениеТекста. Для инициализации объекта используется метод Открыть.

Пример:

Текст = Новый ЧтениеТекста;

Использование в версии:

Доступен, начиная с версии 8.0.
