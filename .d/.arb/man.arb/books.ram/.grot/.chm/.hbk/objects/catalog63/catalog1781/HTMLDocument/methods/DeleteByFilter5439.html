ДокументHTML.УдалитьПоФильтру (HTMLDocument.DeleteByFilter)

ДокументHTML (HTMLDocument)

УдалитьПоФильтру (DeleteByFilter)

Доступен, начиная с версии 8.3.13.

Синтаксис:

УдалитьПоФильтру(<Фильтр>)

Параметры:

<Фильтр> (обязательный)

Тип: Строка.
Строка, содержащая текст JSON-конфигурации, описывающей узлы, которые подлежат удалению из дерева.
Описание структуры JSON-конфигурации для фильтра

Описание:

Выполняет разбор текста формата HTML согласно фильтру, указанному в виде конфигурации формата JSON, и удаляет узлы, удовлетворяющие условиям фильтра.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
