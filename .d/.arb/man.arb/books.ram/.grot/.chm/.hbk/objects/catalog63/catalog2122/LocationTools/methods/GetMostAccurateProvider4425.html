СредстваГеопозиционирования (LocationTools)

ПолучитьСамогоТочногоПровайдера (GetMostAccurateProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьСамогоТочногоПровайдера(<ТолькоДоступного>)

Параметры:

<ТолькоДоступного> (необязательный)

Тип: Булево.
Если параметр содержит Истина, то информация о самом точном провайдере будет возращена только если провайдер является доступным. Если самый точный провайдер недоступен, будет возвращено Неопределено.
Если параметр содержит Ложь, то будет возвращена информация о самом точном провайдере на устройстве.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: ИнформацияПровайдераГеопозиционирования, Неопределено.
Неопределено - ни один провайдер не найден, или если параметр <ТолькоДоступно> содержал значение Истина, а самый точный провайдер не доступен.

Описание:

Получает информацию о самом точном провайдере геопозиционирования.
Если параметр <ТолькоДоступно> содержит значение Истина, информация о самом точном провайдере будет возвращена только в случае, если провайдер доступен.
Если параметр <ТолькоДоступно> содержит значение Ложь, информация о самом точном провайдере будет возвращена независимо от доступности провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
