МенеджерСистемыВзаимодействия (CollaborationSystemManager)

АдресЭлектроннойПочтыАбонента (SubscriberEmail)

Доступен, начиная с версии 8.3.23.

Синтаксис:

АдресЭлектроннойПочтыАбонента()

Возвращаемое значение:

Тип: Строка.

Описание:

Возвращает адрес электронной почты абонента (адрес электронной почты, с помощью которого выполнена регистрация информационной базы на сервере системы взаимодействия).
Если приложение не зарегистрировано, то возвращает пустую строку.
Если текущей пользователь не является администратором абонента, то генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
