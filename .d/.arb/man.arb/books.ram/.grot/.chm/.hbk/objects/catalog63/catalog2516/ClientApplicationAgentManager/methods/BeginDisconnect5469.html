МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

НачатьОтключение (BeginDisconnect)

Доступен, начиная с версии 8.3.13.

Синтаксис:

НачатьОтключение()

Описание:

Начинает отключение от агента клиентского приложения.
Результат выполнения действия (успешное выполнение или ошибка выполнения) можно получить, подключив обработчик с помощью метода ПодключитьОбработчикИзмененияСостояния.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
