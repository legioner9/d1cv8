ВариантМасштабаФормКлиентскогоПриложения (ClientApplicationFormScaleVariant)

Доступен, начиная с версии 8.3.4.

Значения

Авто (Auto)
Компактный (Compact)
Обычный (Normal)

Описание:

Содержит варианты масштаба отображения форм в клиентском приложении.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
