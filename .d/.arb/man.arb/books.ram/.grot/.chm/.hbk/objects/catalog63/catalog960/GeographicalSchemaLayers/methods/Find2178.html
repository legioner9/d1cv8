СлоиГеографическойСхемы.Найти (GeographicalSchemaLayers.Find)

СлоиГеографическойСхемы (GeographicalSchemaLayers)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Текст, по которому будет искаться слой в коллекции.

Возвращаемое значение:

Тип: СлойГеографическойСхемы, Неопределено.
Если не найден, возвращается Неопределено.

Описание:

Осуществляет поиск объекта по указанному тексту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
