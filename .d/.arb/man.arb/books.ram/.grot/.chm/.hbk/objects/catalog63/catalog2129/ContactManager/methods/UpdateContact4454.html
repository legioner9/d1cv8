МенеджерКонтактов (ContactManager)

ИзменитьКонтакт (UpdateContact)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ИзменитьКонтакт(<Контакт>, <Данные>)

Параметры:

<Контакт> (обязательный)

Тип: ЛокальныйКлючКонтакта.
Локальный ключ контакта.

<Данные> (обязательный)

Тип: ДанныеКонтактаУчетнойЗаписи.
Данные контакта учетной записи.

Описание:

Изменяет контакт по локальному ключу в соответствии с переданными данными.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяИзменениеКонтакта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерКонтактов, метод ПоддерживаетсяИзменениеКонтакта

Использование в версии:

Доступен, начиная с версии 8.3.4.
