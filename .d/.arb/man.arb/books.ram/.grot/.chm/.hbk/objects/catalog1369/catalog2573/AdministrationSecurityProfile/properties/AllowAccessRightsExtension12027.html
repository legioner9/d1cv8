АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

РазрешитьРасширениеПравДоступа (AllowAccessRightsExtension)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет, разрешено ли расширять права доступа на объекты конфигурации расширениями, ограниченными профилем безопасности:
  • Истина – расширение прав разрешено.
  • Ложь – расширение прав не разрешено. Если указан список ролей расширяемой конфигурации, допускается расширение права, если хотя бы одна роль из списка включает в себя требуемое право.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
