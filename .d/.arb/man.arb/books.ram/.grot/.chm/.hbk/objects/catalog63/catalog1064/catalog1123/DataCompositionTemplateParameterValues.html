ЗначенияПараметровМакетаКомпоновкиДанных (DataCompositionTemplateParameterValues)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ЗначениеПараметраМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметра.
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция значений параметра макета компоновки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
