Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.2.

Описание:

Гиперссылка. Данный вид недоступен для кнопок расположенных в командной панели или в группах, расположенных в командной панели.

Использование в версии:

Доступен, начиная с версии 8.2.
