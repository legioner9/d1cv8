ТочкиДиаграммыГанта (GanttChartPointCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ТочкаДиаграммыГанта
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются точки коллекции.
Возможно обращение к точке коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) точки.

Свойства:

Родитель (Parent)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Точки диаграммы Ганта представляют собой иерархическую структуру.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
