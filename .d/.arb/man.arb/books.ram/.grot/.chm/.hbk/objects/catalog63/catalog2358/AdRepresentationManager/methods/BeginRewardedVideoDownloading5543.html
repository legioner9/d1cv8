МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуВидеообъявленияСВознаграждением (BeginRewardedVideoDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуВидеообъявленияСВознаграждением(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Указывает обработчик, вызываемый при завершении загрузки регламы из сети Интернет.
Процедура должна иметь следующие параметры:
  • Статус (Status) – тип СтатусРекламы. Показывает статус рекламного блока по результатам загрузки.
  • Идентификатор (ID) – тип Строка. Рекламный идентификатор для которого закончилась загрузка.
  • <ДополнительныеПараметры> – Значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Загружает из сети видеообъявление. По завершению загрузки вызывается обработчик.
После загрузки, при вызове метода ПоказатьВидеообъявлениеСВознаграждением видеообъявление будет показано без задержки на загрузку.
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
