ПоложениеИтоговКолонокСводнойТаблицы (PivotTableColumnTotalPosition)

Доступен, начиная с версии 8.0.

Значения

Лево (Left)
Право (Right)

Описание:

Содержит варианты положения итоговых колонок в колонках сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.

См. также:

СводнаяТаблица, свойство ПоложениеИтоговКолонок

Использование в версии:

Доступен, начиная с версии 8.0.
