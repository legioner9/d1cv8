Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ОбъектОтбораВидовХарактеристик (CharacteristicTypesFilterObject)

Доступен, начиная с версии 8.2.

Описание:

Тип: Произвольный.
Содержит объект отбора видов характеристик.
Если значение данного параметра не равно Неопределено, то определяется тип значения.
Если типом является ссылка на объектный объект метаданных, то у данного объекта ищутся в метаданных описания характеристик, виды которых получаются из того плана видов характеристик, для которого получается форма (на основе основного реквизита). Если описания характеристик найдены, то из каждого описания получаются поле отбора вида характеристик, значение отбора и в отбор добавляется элемент отбора по этому полю, с этим значением.
Если на одно поле имеется несколько различных значений, то добавляется отбор с видом сравнения ВСписке со списком, содержащем все различные значения отбора. Если значения имеются для нескольких полей, то отборы добавляются для всех полей.
Если в параметре указан массив или список значений, то описания характеристик ищутся для каждого значения в массиве или списке значений.

Использование в версии:

Доступен, начиная с версии 8.2.
