Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ВысотаЭлемента (ItemHeight)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит высоту каждого элемента. Указывается в строках. 0 - высота определяется автоматически.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

При виде переключателя отличном от Тумблер управляет высотой заголовка элемента.

Использование в версии:

Доступен, начиная с версии 8.2.
