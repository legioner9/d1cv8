Рабочий сервер (IWorkingServerInfo)

ClusterMainPort (ClusterMainPort)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит номер порта главного менеджера кластера, запущенного на данном рабочем сервере. Используется, если MainPort = Истина.

Доступность:

Интеграция.

Доступен, начиная с версии 8.3.1.
