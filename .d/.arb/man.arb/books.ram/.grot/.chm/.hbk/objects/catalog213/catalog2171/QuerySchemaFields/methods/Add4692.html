ПоляСхемыЗапроса.Добавить (QuerySchemaFields.Add)

ПоляСхемыЗапроса (QuerySchemaFields)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: Из строки

Синтаксис:

Добавить(<Выражение>, <Колонка>, <КолонкаВложеннойТаблицы>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.
Строка, на основе которой будет создано выражение.

<Колонка> (необязательный)

Тип: Число.
Индекс колонки, в которую нужно поместить добавленное поле.

<КолонкаВложеннойТаблицы> (необязательный)

Тип: Число.
При добавлении поля из вложенной таблицы указывает, в какую колонку будет помещена новая вложенная таблица. Если вложенная таблица уже существует, параметр игнорируется.

Описание варианта метода:

Создает и добавляет новое выражение из переданной строки.

Вариант синтаксиса: Из доступного поля

Синтаксис:

Добавить(<ДоступноеПоле>, <Колонка>, <КолонкаВложеннойТаблицы>)

Параметры:

<ДоступноеПоле> (обязательный)

Тип: ДоступноеПолеСхемыЗапроса.
Доступное поле, из которого будет создано выражение схемы запроса.

<Колонка> (необязательный)

Тип: Число.
Индекс колонки, в которую нужно поместить добавленное поле.

<КолонкаВложеннойТаблицы> (необязательный)

Тип: Число.
При добавлении поля из вложенной таблицы указывает, в какую колонку будет помещена новая вложенная таблица. Если вложенная таблица уже существует, параметр игнорируется.

Описание варианта метода:

Создает и добавляет новое выражение из доступного поля схемы запроса.

Вариант синтаксиса: Из доступной вложенной таблицы

Синтаксис:

Добавить(<ДоступнаяВложеннаяТаблица>)

Параметры:

<ДоступнаяВложеннаяТаблица> (обязательный)

Тип: ДоступнаяВложеннаяТаблицаСхемыЗапроса.
Доступная вложенная таблица, из которой будет создана вложенная таблица схемы запроса.

Описание варианта метода:

Создает и добавляет вложенную таблицу схемы запроса из доступной вложенной таблицы.

Возвращаемое значение:

Тип: ВыражениеСхемыЗапроса, ВложеннаяТаблицаСхемыЗапроса.

Описание:

Добавляет новый элемент коллекции. Если тип нового элемента не подходит данной коллекции, элемент не добавляется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
