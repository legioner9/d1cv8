ОбластьЗаголовкаСводнойДиаграммы (PivotChartTitleArea)

Доступен, начиная с версии 8.1.

Свойства:

ПрозрачныйФон (Transparent)
Рамка (Border)
Расположение (Placement)
Текст (Text)
ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области заголовка сводной диаграммы для настройки параметров оформления. В области заголовка может содержаться название сводной диаграммы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
