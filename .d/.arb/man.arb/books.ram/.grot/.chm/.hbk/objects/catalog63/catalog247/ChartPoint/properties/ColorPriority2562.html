ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Вместе со свойством ПриоритетЦвета соответствующей серии определяет цвет значения. При равных значениях свойства используется Цвет серии.
Истина - будет использоваться значение Цвет точки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для следующих типов диаграмм:
  • Гистограмма;
  • ГистограммаОбъемная;
  • ГистограммаГоризонтальная;
  • ГистограммаГоризонтальнаяОбъемная;
  • Изометрическая;
  • ИзометрическаяПирамида.

Использование в версии:

Доступен, начиная с версии 8.0.
