НастройкаПериода.УстановитьПериод (PeriodSettings.SetPeriod)

НастройкаПериода (PeriodSettings)

УстановитьПериод (SetPeriod)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПериод(<ДатаНачала>, <ДатаОкончания>, <ПредпочтительноИспользоватьРабочийПериод>)

Параметры:

<ДатаНачала> (необязательный)

Тип: Дата.
Дата начала периода.

<ДатаОкончания> (необязательный)

Тип: Дата.
Дата окончания периода.

<ПредпочтительноИспользоватьРабочийПериод> (необязательный)

Тип: Булево.
Если задано Истина, то при установке периода будет произведена попытка назначить рабочий период (проявляется как установка флажка "Рабочий период" в форме диалога).
Значение по умолчанию: Ложь.

Описание:

Устанавливает период редактирования. Если это возможно, объекту автоматически подбирается вариант периода и значение периода.

Доступность:

Толстый клиент.

Пример:

НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод;
НастройкаПериода.УстановитьПериод('20020401','20020531');

Использование в версии:

Доступен, начиная с версии 8.0.
