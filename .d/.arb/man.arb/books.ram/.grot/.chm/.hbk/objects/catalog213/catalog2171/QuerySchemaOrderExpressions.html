ВыраженияПорядкаСхемыЗапроса (QuerySchemaOrderExpressions)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ВыражениеПорядкаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения порядка.
Возможно обращение к выражению порядка посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит объектную модель выражений порядка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
