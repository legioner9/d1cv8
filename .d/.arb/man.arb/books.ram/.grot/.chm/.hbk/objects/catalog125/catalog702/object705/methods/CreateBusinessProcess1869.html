БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

СоздатьБизнесПроцесс (CreateBusinessProcess)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьБизнесПроцесс()

Возвращаемое значение:

Тип: БизнесПроцессОбъект.<Имя бизнес-процесса>.

Описание:

Создает новый бизнес-процесс.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Создание, запись и старт нового бизнес-процесса

БизнесПроцесс = 
    БизнесПроцессы.СогласованиеДокумента.СоздатьБизнесПроцесс();
БизнесПроцесс.Дата = ТекущаяДата();
БизнесПроцесс.Записать();
БизнесПроцесс.Старт();

Использование в версии:

Доступен, начиная с версии 8.0.
