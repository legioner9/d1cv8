Рекомендуется использовать:

  • Вставить

Доступен, начиная с версии 8.3.18.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Текст>, <Картинка>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Позиция, в которую необходимо вставить действие сообщения системы взаимодействия.

<Значение> (необязательный)

Тип: Произвольный.
Значение действия сообщения системы взаимодействия.
Значение по умолчанию: Неопределено.

<Текст> (необязательный)

Тип: Строка, ФорматированнаяСтрока.
Текст, отображаемый в действии сообщения системы взаимодействия.
Значение по умолчанию: Пустая строка.

<Картинка> (необязательный)

Тип: Картинка.
Картинка, отображаемая в действии сообщения системы взаимодействия.
Значение по умолчанию: Пустая картинка.

Возвращаемое значение:

Тип: ДействиеСообщенияСистемыВзаимодействия.

Описание:

Вставляет действие в коллекцию действий по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.
