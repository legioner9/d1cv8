ЧтениеJSON.Закрыть (JSONReader.Close)

ЧтениеJSON (JSONReader)

Закрыть (Close)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Закрыть()

Описание:

Завершает чтение текста JSON из файла или строки.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.ОткрытьФайл("c:/docs/data.json");
// Чтение текста JSON

ЧтениеJSON.Закрыть();

Использование в версии:

Доступен, начиная с версии 8.3.6.
