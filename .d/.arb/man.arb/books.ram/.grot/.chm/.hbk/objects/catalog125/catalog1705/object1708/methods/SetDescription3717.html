ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

УстановитьОписание (SetDescription)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьОписание(<КлючОбъекта>, <КлючНастроек>, <ОписаниеНастроек>, <ИмяПользователя>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: Произвольный.
Ключ объекта настройки.

<КлючНастроек> (необязательный)

Тип: Произвольный.
Ключ настроек.
Если имеет значение Неопределено, то будет сгенерирован и возвращен новый ключ настроек.

<ОписаниеНастроек> (обязательный)

Тип: ОписаниеНастроек.
Описание настроек.

<ИмяПользователя> (необязательный)

Тип: Строка.
Имя пользователя, описание настроек которого устанавливается.
Если не указан, то устанавливается описание настроек текущего пользователя.

Возвращаемое значение:

Тип: Произвольный.

Описание:

Устанавливает описание настроек в хранилище.
Вызывает обработчик ОбработкаУстановкиОписания.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
