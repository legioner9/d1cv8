ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

РежимИзмененияРазмераКолонки (ColumnSizeChangeMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: РежимИзмененияРазмера.
Указывает возможность изменения размера колонки, в которой находится ячейка.
При подведении указателя мыши к правой границе ячейки, у которой данное свойство установлено в БыстроеИзменение, изображение указателя меняется на изображение указателя для изменения размеров. С помощью мыши можно изменять размер колонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
