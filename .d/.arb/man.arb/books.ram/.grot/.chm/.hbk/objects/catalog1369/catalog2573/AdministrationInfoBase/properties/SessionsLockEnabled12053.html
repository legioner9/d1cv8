АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ВключенаБлокировкаНачалаСеансов (SessionsLockEnabled)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Управляет режимом блокировки сеансов.
Если установлено значение Истина и текущее время находится в диапазоне, определяемом свойствами ВремяНачалаБлокировки и ВремяОкончанияБлокировки, то начало новых сеансов с информационной базой запрещено. Если при этом попытаться установить новый сеанс, то будет сгенерировано сообщение об ошибке, содержащее текст из значения свойства СообщениеБлокировки.
Для начала сеанса вопреки блокировке сеансов необходимо указать код разрешения, определяемый свойством КодРазрешенияНачалаСеансов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
