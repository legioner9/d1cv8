ОписаниеОповещения (CallbackDescription)

ИмяПроцедурыОбработкиОшибки (ErrorHandlerProcedureName)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Имя экспортируемой процедуры модуля, которая будет вызвана при возникновении ошибки во время вызова асинхронных методов.
Процедура принимает следующие параметры:
  • объект ИнформацияОбОшибке,
  • <Стандартная обработка> - Булево. Если Истина - исключение будет отображаться для пользователя,
  • <Дополнительные параметры> - Значение, которое было указано при создании объекта ОписаниеОповещения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Не используется при обработке завершения модальных диалогов и закрытия форм.

Использование в версии:

Доступен, начиная с версии 8.3.5.
