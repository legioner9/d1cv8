ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ФорматСтрок (FormatOfRows)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: ФорматСтрокТабличногоДокумента.
Содержит формат строк, который используется в области.
Для областей с типом Строки, Прямоугольник и Таблица, если в области находится несколько форматов строк, то свойство содержит Неопределено. При присвоении свойству нового значения всем строкам в области устанавливается формат строк.
Для области с типом Колонки всегда содержит формат строк, из которого получена колонка и является свойством только для чтения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
