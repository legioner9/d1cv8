ТипРазмещенияРеквизитовИзмерений (DimensionAttributePlacementType)

Доступен, начиная с версии 8.0.

Значения

Вместе (Together)
ВместеСИзмерениями (WithDimensions)
Отдельно (Separately)

Описание:

Содержит варианты расположения реквизитов измерений в построителе отчета и сводной таблице.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.
