ПоляНастройки.Вставить (CustomFields.Insert)

ПоляНастройки (CustomFields)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Имя>, <Представление>, <Тип>, <СписокДопустимыхЗначений>, <Позиция>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя поля.

<Представление> (обязательный)

Тип: Строка.
Представление поля.
Если не задано, то выбирается из параметра <Имя>.

<Тип> (необязательный)

Тип: ОписаниеТипов.
Тип поля.
Значение по умолчанию: Произвольный.

<СписокДопустимыхЗначений> (необязательный)

Тип: СписокЗначений.
Список допустимых значений для поля.

<Позиция> (обязательный)

Тип: Число.
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: ПолеНастройки.

Описание:

Вставляет поле в указанную позицию коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
