РазмерСтраницы (PageSize)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Размер страницы, используемый при разбиении табличного документа на страницы.

Если содержит значение Неопределено, то размер бумаги не установлен.
