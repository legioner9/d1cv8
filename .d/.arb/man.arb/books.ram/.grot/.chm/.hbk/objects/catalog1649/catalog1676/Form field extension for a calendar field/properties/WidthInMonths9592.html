Расширение поля формы для поля календаря (Form field extension for a calendar field)

ШиринаВМесяцах (WidthInMonths)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Ширина календаря, заданная в месяцах. Если равна 0, то используется свойство Ширина. Значение по умолчанию 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Имеет смысл только для режимов совместимости выше Версия8_3_2. В противном случае свойство доступно, но его изменение не производится.

Использование в версии:

Доступен, начиная с версии 8.3.3.
