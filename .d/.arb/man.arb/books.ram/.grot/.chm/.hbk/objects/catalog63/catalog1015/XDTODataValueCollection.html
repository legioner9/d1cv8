КоллекцияЗначенийXDTO (XDTODataValueCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ЗначениеXDTO
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения XDTO.
Возможно обращение к значению XDTO посредством оператора [...]. В качестве аргумента передается индексу.

Методы:

Количество (Count)
Получить (Get)

Описание:

Коллекция значений простого типа XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЗначениеXDTO, свойство Список

Использование в версии:

Доступен, начиная с версии 8.1.
