РасширениеКонфигурации.ОсновныеРоли (ConfigurationExtension.DefaultRoles)

РасширениеКонфигурации (ConfigurationExtension)

ОсновныеРоли (DefaultRoles)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: ФиксированныйМассив.
Массив содержит объекты типа ОбъектМетаданных: Роль), установленные в свойстве ОсновныеРоли расширения конфигурации. Указанные роли будут определять доступность объектов расширения, если в расширяемой конфигурации отсутствуют пользователи.
Для ролей, которые добавляются в качестве основной роли расширения действуют следующие ограничения:
  • Роль должна быть только собственной ролью расширения. Не должна быть заимствованной.
  • Роль предоставляет доступ только к собственным объектам расширения. Не может предоставлять доступ к заимствованным объектам расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.25.
