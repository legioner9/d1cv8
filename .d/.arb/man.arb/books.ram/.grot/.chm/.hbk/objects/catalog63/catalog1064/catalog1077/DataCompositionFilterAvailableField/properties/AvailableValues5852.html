ДоступноеПолеОтбораКомпоновкиДанных.ДоступныеЗначения (DataCompositionFilterAvailableField.AvailableValues)

ДоступноеПолеОтбораКомпоновкиДанных (DataCompositionFilterAvailableField)

ДоступныеЗначения (AvailableValues)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: СписокЗначений, Неопределено.
Содержит список значений, которые может принимать поле.
Например, для поля "Оценка" (тип значения Число) доступные значения только 1,2,3,4,5.
Неопределено - значения поля неограничены.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
