МенеджерПолитикПаролейПользователей (UserPasswordPolicyManager)

ПроверитьСоответствиеПароляПолитике (CheckPasswordComplianceWithPolicy)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПроверитьСоответствиеПароляПолитике(<Пароль>, <Политика>, <ПользовательИнформационнойБазы>)

Параметры:

<Пароль> (обязательный)

Тип: Строка.
Пароль пользователя.

<Политика> (необязательный)

Тип: ПолитикаПаролейПользователей.
Политика паролей пользователей.
По умолчанию: Неопределено.
Значение по умолчанию: Неопределено.

<ПользовательИнформационнойБазы> (необязательный)

Тип: ПользовательИнформационнойБазы.
Пользователь информационной базы.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Массив.
Массив значений типа РезультатПроверкиСоответствияПароляПолитике.

Описание:

Проверяет, соответствует ли пароль политике паролей пользователей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
