БизнесПроцессОбъект.<Имя бизнес-процесса>.Метаданные (BusinessProcessObject.<Business process name>.Metadata)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: БизнесПроцесс.

Описание:

Предоставляет доступ к объекту описания метаданных бизнес-процесса. Другой путь получения того же значения - через свойство глобального контекста Метаданные. Например: Метаданные.БизнесПроцессы.ВыпискаСчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Для каждого Реквизит Из БизнесПроцесс.Метаданные().Реквизиты Цикл
    Сообщить(Реквизит.Представление());
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
