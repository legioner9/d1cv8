Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: КоллекцияТиповЗначенийXDTO.
Коллекция типов значений, участвующих в объединении.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Существует только для типа, представляющего собой объединение простых типов.

Использование в версии:

Доступен, начиная с версии 8.1.
