НастройкаПорядка.Удалить (OrderSetting.Delete)

НастройкаПорядка (OrderSetting)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: ЭлементУправленияПорядком.
Удаляемый элемент управления порядком.

Описание:

Удаляет элемент из коллекции.
Внимание! Метод не доступен для настройки порядка динамических списков (свойство НастройкаПорядка объектов СправочникСписок.<Имя справочника>, ДокументСписок.<Имя документа> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
