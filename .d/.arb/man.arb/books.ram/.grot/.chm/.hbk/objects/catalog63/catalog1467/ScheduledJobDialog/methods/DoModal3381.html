ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: Булево.

Описание:

Открывает модально диалог расписания.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

Примечание:

Если для конфигурации свойство РежимИспользованияМодальности установлено в НеИспользовать, следует использовать метод Показать.

См. также:

ДиалогРасписанияРегламентногоЗадания, метод ОткрытьАсинх

Использование в версии:

Доступен, начиная с версии 8.1.
