ФорматированныйДокумент (FormattedDocument)

ПолучитьЭлементы (GetItems)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьЭлементы(<Начало>, <Конец>)

Параметры:

<Начало> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на начало диапазона, элементы которого надо вернуть в массиве.
Если закладка не задана, то элементы будут выбраны с самого начала документа.

<Конец> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на конец диапазона, элементы которого надо вернуть в массиве.
Если закладка не задана, то элементы будут выбраны до самого конца документа.

Возвращаемое значение:

Тип: Массив.
Массив найденных элементов.

Описание:

Получает набор элементов, входящих в диапазон между заданными закладками, включая не полностью входящие.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
