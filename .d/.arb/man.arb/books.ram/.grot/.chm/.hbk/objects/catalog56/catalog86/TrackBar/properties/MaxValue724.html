ПолосаРегулирования (TrackBar)

МаксимальноеЗначение (MaxValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит максимальное значение диапазона изменения. Это значение устанавливается при перемещении "движка" в крайнее правое (нижнее) положение.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
