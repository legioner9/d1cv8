ТипТекстаПочтовогоСообщения (InternetMailTextType)

Доступен, начиная с версии 8.0.

Значения

HTML (HTML)
ПроизвольныйТекст (CustomText)
ПростойТекст (PlainText)
РазмеченныйТекст (RichText)

Описание:

Определяет набор допустимых типов текста почтового сообщения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.
