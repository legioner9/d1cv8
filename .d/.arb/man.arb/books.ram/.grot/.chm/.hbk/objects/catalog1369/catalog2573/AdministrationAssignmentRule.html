АдминистрированиеТребованиеНазначения (AdministrationAssignmentRule)

Доступен, начиная с версии 8.3.14.

Свойства:

ДополнительныйПараметр (AdditionalParameter)
ИдентификаторТребования (RuleID)
ИмяИнформационнойБазы (InfoBaseName)
Приоритет (Priority)
ТипОбъекта (ObjectType)
ТипТребованияНазначения (AssignmentRuleType)

Методы:

Записать (Write)
Удалить (Delete)

Описание:

Определяет одно из множества правил, в соответствии с которыми кластер серверов распределяет соединения с информационными базами и сервисы кластера по рабочим серверам и отдельным процессам.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
