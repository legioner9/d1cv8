МенеджерКонтактов (ContactManager)

ПоддерживаютсяГлобальныеКлючиКонтактов (GlobalContactsKeysSupported)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПоддерживаютсяГлобальныеКлючиКонтактов()

Возвращаемое значение:

Тип: Булево.
Всегда возвращает Ложь.

Описание:

Определяет, поддерживаются ли системой глобальные ключи контактов.
Предназначен для проверки возможности отправки контакта на сервер и обратно.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
