ЧтениеFastInfoset.Пропустить (FastInfosetReader.Skip)

ЧтениеFastInfoset (FastInfosetReader)

Пропустить (Skip)

Доступен, начиная с версии 8.1.

Синтаксис:

Пропустить()

Описание:

Пропускает дочерние узлы текущего узла и позиционируется на следующий узел.
Если текущий узел не содержит подчиненных узлов, то вызов метода аналогичен ЧтениеFastInfoset.Прочитать, в противном случае осуществляется пропуск всех дочерних узлов и осуществляется позиционирование на следующем узле.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
