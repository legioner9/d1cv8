СхемаXML.ДокументDOM (XMLSchema.DOMDocument)

СхемаXML (XMLSchema)

ДокументDOM (DOMDocument)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: ДокументDOM.
Документ DOM, соответствующий схеме XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Наличие документа DOM не является обязательным. Для его создания можно воспользоваться методом СхемаXML.ОбновитьЭлементDOM.

Использование в версии:

Доступен, начиная с версии 8.1.
