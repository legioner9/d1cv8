HTTPСервисОтвет.КодСостояния (HTTPServiceResponse.StatusCode)

HTTPСервисОтвет (HTTPServiceResponse)

КодСостояния (StatusCode)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Код состояния (ответа), возвращаемый HTTP-сервисом.

Доступность:

Сервер, толстый клиент.

Примечание:

HTTP-сервер Apache не поддерживает, в отличие от IIS, произвольные коды ответа. При использовании сервера Apache стоит ограничиться кодами ответа, указанными в документе http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html.

Использование в версии:

Доступен, начиная с версии 8.3.5.
