ПланОбменаСсылка.<Имя плана обмена>.Ссылка (ExchangePlanRef.<Exchange plan name>.Ref)

ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланОбменаСсылка.<Имя плана обмена>.
Содержит ссылку на узел. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Так как ПланОбменаОбъект.<Имя плана обмена> и ПланОбменаВыборка.<Имя плана обмена> имеют одноименное свойство, то при обращении в программном модуле к данному свойству переданного в процедуру или функцию параметра не потребуется определять тип параметра: ПланОбменаСсылка.<Имя плана обмена>, ПланОбменаОбъект.<Имя плана обмена> или ПланОбменаВыборка.<Имя плана обмена>.

Использование в версии:

Доступен, начиная с версии 8.0.
