ОтчетОбОшибке (ErrorReport)

По умолчанию

Доступен, начиная с версии 8.3.17.

Синтаксис:

Новый ОтчетОбОшибке(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: ИнформацияОбОшибке.
Информация, на основании которой инициализируется отчет.

Описание:

Создает и возвращает отчет об ошибке, инициализированный информацией об ошибке. Отчет автоматически инициализируется параметрами отправки отчетов об ошибках для данного пользователя.

Использование в версии:

Доступен, начиная с версии 8.3.17.
