ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

Отображаемое (Displayed)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Устанавливает видимость сообщений:
  • Истина - возвращаются обсуждения, у которых свойство Отображаемое имеет значение Истина,
  • Ложь - возвращаются обсуждения, у которых свойство Отображаемое имеет значение Ложь,
  • Неопределено - возвращаются все обсуждения в соответствии с другими значениями отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: Неопределено.

Использование в версии:

Доступен, начиная с версии 8.3.11.
