ДиалогВыбораФайла.Расширение (FileDialog.DefaultExt)

ДиалогВыбораФайла (FileDialog)

Расширение (DefaultExt)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит расширение файла по умолчанию, которое будет использоваться системой при записи файла.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В веб-клиенте, если расширение работы с файлами не установлено, то данное свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.0.
