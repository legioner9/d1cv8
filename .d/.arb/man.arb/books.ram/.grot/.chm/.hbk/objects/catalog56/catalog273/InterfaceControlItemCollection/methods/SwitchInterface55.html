КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

ПереключитьИнтерфейс (SwitchInterface)

Доступен, начиная с версии 8.0.

Синтаксис:

ПереключитьИнтерфейс(<ИменаИнтерфейсов>)

Параметры:

<ИменаИнтерфейсов> (необязательный)

Тип: Строка.
Перечисленные через запятую имена интерфейсов, которые следует сделать видимыми.

Описание:

Способ группового управления видимостью командных интерфейсов для режима запуска ОбычноеПриложение. При вызове данного метода сначала становятся невидимыми все интерфейсы, кроме имеющих значение Ложь у свойства Переключаемый, а затем становятся видимыми перечисленные в параметре интерфейсы.
При работе в режиме УправляемоеПриложение вызов данного метода игнорируется.

Доступность:

Толстый клиент.

Пример:

ГлавныйИнтерфейс.ПереключитьИнтерфейс("Основной,Маркетинг");

Использование в версии:

Доступен, начиная с версии 8.0.
