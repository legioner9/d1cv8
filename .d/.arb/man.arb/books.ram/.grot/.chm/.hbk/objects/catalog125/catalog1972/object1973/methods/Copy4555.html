Скопировать (Copy)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: ВнешнийИсточникДанныхТаблицаОбъект.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>.

Описание:

Создает новый объект копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.3.5.
