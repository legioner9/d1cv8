ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередИнтерактивнымВыполнением (BusinessProcessRoutePointRef.<Business process name>.BeforeExecuteInteractively)

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередИнтерактивнымВыполнением (BeforeExecuteInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИнтерактивнымВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса вида Действие, в которой выполняется обработка метода ВыполнитьЗадачуИнтерактивно у задачи.

<Задача>

Тип: ЗадачаСсылка.<Имя задачи>.
Выполняемая задача.

<Отказ>

Тип: Булево.
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то выполнение задачи произведено не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает при вызове метода ВыполнитьЗадачуИнтерактивно у задачи. Этот обработчик является единственным местом в котором можно разместить интерактивное взаимодействие с пользователем перед переходом выполнения бизнес-процесса дальше по маршруту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида Действие.
Допускаются интерактивные вызовы (открытие форм, предупреждений).
В режиме запуска УправляемоеПриложение не вызывается.

Использование в версии:

Доступен, начиная с версии 8.0.
