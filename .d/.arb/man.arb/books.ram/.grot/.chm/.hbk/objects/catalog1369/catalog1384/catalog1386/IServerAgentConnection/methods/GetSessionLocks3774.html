Соединение с агентом сервера.GetSessionLocks (IServerAgentConnection.GetSessionLocks)

Соединение с агентом сервера (IServerAgentConnection)

GetSessionLocks (GetSessionLocks)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessionLocks(<Кластер>, <Сеанс>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер серверов, из которого должны быть получены блокировки. Должен быть получен методом GetClusters объекта Соединение с агентом сервера.

<Сеанс> (обязательный)

Тип: Сеанс.
Сеанс, от имени которого установлены блокировки объектов. Должен быть получен методом GetSessions или GetInfoBaseSessions объекта Соединение с агентом сервера.

Возвращаемое значение:

Тип: COMSafeArray.
Массив элементов Блокировка.

Описание:

Получает объектные блокировки кластера, установленные от имени заданного сеанса.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
