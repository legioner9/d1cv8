Расширение поля формы для поля флажка (Form field extension for a check box field)

ВидФлажка (CheckBoxType)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: ВидФлажка.
Определяет режим отображения поля.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Режим Тумблер доступен, если поле связано с реквизитом типа Булево и поле не размещено в таблице.

Использование в версии:

Доступен, начиная с версии 8.3.2.
