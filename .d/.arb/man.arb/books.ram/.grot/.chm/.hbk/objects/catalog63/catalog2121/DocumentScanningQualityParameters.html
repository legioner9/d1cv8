ПараметрыКачестваСканированияДокументов (DocumentScanningQualityParameters)

Доступен, начиная с версии 8.3.23.

Свойства:

КачествоБелого (WhitePaperQuality)
КачествоГраницы (BorderQuality)
КонтрастностьТекста (TextContrast)
ОбобщенноеКачество (GeneralizedQuality)
ПерспективныеИскажения (PerspectiveDistortion)

Описание:

Содержит результат выполнения методов ПоказатьСканированиеДокументов и СканироватьДокументыАсинх, возвращаемый в свойстве объекта СтраницаСканированияДокументов для контроля уровня качества полученной страницы документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/data/ext. Имя типа XDTO: DocumentScanningQualityParameters.

Использование в версии:

Доступен, начиная с версии 8.3.23.
