ДиалогОтбораВерсийИсторииДанных (DataHistoryVersionsFilterDialog)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ДиалогОтбораВерсийИсторииДанных(<Данные>, <Отбор>)

Параметры:

<Данные> (обязательный)

Тип: СправочникСсылка.<Имя справочника>, ДокументСсылка.<Имя документа>, РегистрСведенийКлючЗаписи.<Имя регистра сведений>, БизнесПроцессСсылка.<Имя бизнес-процесса>, ЗадачаСсылка.<Имя задачи>, ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>, ПланСчетовСсылка.<Имя плана счетов>.
Данные, по которым отбираются версии. Используется для формирования доступных для отбора полей и пользователей-авторов версий.

<Отбор> (необязательный)

Тип: Структура.
Используется для установки параметров отбора при открытии диалога.
Структура идентична структуре, содержащейся в параметре <Отбор> метода ВыбратьВерсии.
Если диалог был закрыт пользователем нажатием кнопки "OK", содержит параметры отбора, установленные пользователем.
Значение по умолчанию: пустая структура.
Значение по умолчанию: Структура.

Описание:

Создает диалог отбора версий указанных данных.

Использование в версии:

Доступен, начиная с версии 8.3.13.
