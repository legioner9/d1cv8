Объект внешней компоненты (Add-in object)

Установить<ИмяСвойства>Асинх (Set<PropertyName>Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Установить<ИмяСвойства>Асинх(<НовоеЗначение>)

Параметры:

<НовоеЗначение> (необязательный)

Тип: Произвольный.
Устанавливаемое значение свойства.

Возвращаемое значение:

Результат выполнения обещания - Неопределено.

Описание:

Асинхронно устанавливает значение свойства внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Объект внешней компоненты, метод НачатьУстановку<ИмяСвойства>
