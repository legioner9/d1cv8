СписокРасширенныхИменXML (XMLExpandedNameList)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Имя>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс - местоположение вставляемого расширенного имени.

<Имя> (обязательный)

Тип: РасширенноеИмяXML.
Расширенное имя XML.

Описание:

Вставляет расширенное имя в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
