ГруппаЭлементовОтбораКомпоновкиДанных (DataCompositionFilterItemGroup)

ТипГруппы (GroupType)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: ТипГруппыЭлементовОтбораКомпоновкиДанных.
Содержит тип группы элементов отбора.
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
