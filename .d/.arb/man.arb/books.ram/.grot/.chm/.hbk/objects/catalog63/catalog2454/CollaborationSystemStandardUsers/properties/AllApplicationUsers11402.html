СтандартныеПользователиСистемыВзаимодействия (CollaborationSystemStandardUsers)

ВсеПользователиПриложения (AllApplicationUsers)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: ИдентификаторПользователяСистемыВзаимодействия.
Идентификатор пользователя системы взаимодействия, представляющего собой всех пользователей приложения.
Если список участников неконтекстного обсуждения содержит значение "ВсеПользователиПриложения", обсуждение доступно всем пользователям информационной базы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
