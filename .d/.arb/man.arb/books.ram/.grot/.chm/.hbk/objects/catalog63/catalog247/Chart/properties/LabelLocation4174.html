Диаграмма.ПоложениеПодписей (Chart.LabelLocation)

Диаграмма (Chart)

ПоложениеПодписей (LabelLocation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ПоложениеПодписейКДиаграмме.
Определяет положение подписей к значениям гистограмм.
Значение по умолчанию: Авто. В режиме совместимости Версия8_3_23 и ниже - Край.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
