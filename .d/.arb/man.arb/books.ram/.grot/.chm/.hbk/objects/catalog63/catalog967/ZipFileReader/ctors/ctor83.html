ЧтениеZipФайла (ZipFileReader)

На основании имени файла

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЧтениеZipФайла(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя ZIP-файла, который требуется открыть для чтения.

<Пароль> (необязательный)

Тип: Строка.
Пароль для доступа к ZIP файлу, если файл зашифрован.

Описание:

Открывает для чтения указанный ZIP файл. Аналогичен методу Открыть.

Использование в версии:

Доступен, начиная с версии 8.0.
