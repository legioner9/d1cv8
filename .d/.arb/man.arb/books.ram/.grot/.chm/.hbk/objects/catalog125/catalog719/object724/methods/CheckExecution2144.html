ЗадачаОбъект.<Имя задачи>.ПроверитьВыполнение (TaskObject.<Task name>.CheckExecution)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПроверитьВыполнение (CheckExecution)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьВыполнение()

Возвращаемое значение:

Тип: Булево.
Истина, если все достаточные условия выполнения задачи соблюдены, Ложь – в противном случае.
Если обработчики не определены, возвращает Ложь.

Описание:

Показывает результат вызова обработчиков события ОбработкаПроверкиВыполнения у задачи и обработчика события ОбработкаПроверкиВыполнения у точки. Если обработчик задачи вернул Ложь, то обработчик у точки не вызывается. Используется, например, для автоматизированного выполнения задач роботами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
