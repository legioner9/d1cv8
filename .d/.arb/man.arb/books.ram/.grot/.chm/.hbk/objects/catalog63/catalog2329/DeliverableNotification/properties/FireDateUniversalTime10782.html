ДатаПоявленияУниверсальноеВремя (FireDateUniversalTime)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Дата.
Время первого срабатывания локального уведомления. Указывается по UTC. Если не указано, то локальное уведомление будет показано мгновенно. Если указано "пустое" время или уже прошедшее время, то локальное уведомление будет отображено в области уведомлений операционной системы без задержки, не зависимо от активности приложения в данный момент. Такое уведомление не будет повторено, даже если для него указан повтор.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

По умолчанию - пустая дата.

Использование в версии:

Доступен, начиная с версии 8.3.6.
