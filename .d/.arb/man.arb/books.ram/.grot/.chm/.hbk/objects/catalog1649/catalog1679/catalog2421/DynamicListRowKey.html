КлючСтрокиДинамическогоСписка (DynamicListRowKey)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

Произвольный
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа КлючИЗначение, где Ключ - путь ключевого поля, Значение - значение этого ключевого поля.
Возможно обращение к значению ключевого поля посредством оператора [...]. В качестве аргумента передается строка (путь компоновки данных соответствующего ключевого поля).

Свойства:

<Путь ключевого поля> (<Key field path>)

Методы:

Количество (Count)
Получить (Get)

Конструкторы:

На основе путей и значений полей
На основе структуры, фиксированной структуры, соответствия или фиксированного соответствия

Описание:

Набор значений, однозначно идентифицирующих запись динамического списка.
Используется для динамических списков с типом ключа Составной. Объекты данного типа используются в таблицах формы, связанных с такими списками, в качестве значения свойства ТекущаяСтрока.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен . Имя типа XDTO:
