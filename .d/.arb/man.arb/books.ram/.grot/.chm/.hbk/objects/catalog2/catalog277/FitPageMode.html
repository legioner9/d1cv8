РежимРазмещенияНаСтранице (FitPageMode)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
ПоШиринеСтраницы (PageWidth)
Пропорционально (Proportionally)

Описание:

Содержит варианты режима размещения на странице графической схемы при печати и просмотре.

Доступность:

Тонкий клиент, сервер, толстый клиент.

См. также:

ГрафическаяСхема, свойство РежимРазмещенияНаСтранице

Использование в версии:

Доступен, начиная с версии 8.0.
