РасширениеКонфигурации (ConfigurationExtension)

ИспользуетсяВРаспределеннойИнформационнойБазе (UsedInDistributedInfoBase)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если это свойство установлено в значение Истина, то данное расширение конфигурации будет передаваться в распределенных информационных базах, организованных планами обмена, у которых свойство РаспределеннаяИнформационнаяБаза установлено в значение Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Изменение свойства на клиенте требует обращения к серверу.

См. также:

РасширениеКонфигурации, свойство ГлавныйУзел
ОбъектМетаданных: ПланОбмена, свойство ВключатьРасширенияКонфигурации

Использование в версии:

Доступен, начиная с версии 8.3.12.
