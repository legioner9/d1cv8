МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьДанныеРаботыСРечьюИнформационнойБазы (SetInfoBaseSpeechProcessingData)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьДанныеРаботыСРечьюИнформационнойБазы(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: ДанныеИнформационнойБазыРаботыСРечью.
Данные информационной базы в сервисе распознавания речи.

Описание:

Устанавливает данные информационной базы в сервисе распознавания речи (для текущей области, если для сеанса установлены разделители).
Если передано Неопределено, то настройка информационной базы для работы с сервисом распознавания речи отменяется.
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
