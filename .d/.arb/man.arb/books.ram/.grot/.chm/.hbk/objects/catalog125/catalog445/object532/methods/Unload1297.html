ПланСчетовВидыСубконто.<Имя плана счетов>.Выгрузить (ChartOfAccountsExtDimensionTypes.<Chart of accounts name>.Unload)

ПланСчетовВидыСубконто.<Имя плана счетов> (ChartOfAccountsExtDimensionTypes.<Chart of accounts name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.0.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: ТаблицаЗначений.

Описание:

Создает таблицу значений и копирует в нее табличную часть. Имена колонок таблицы значений совпадают с именами колонок табличного поля. Колонка, содержащая номер строки, выгружается в колонку с именем НомерСтроки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

ТаблицаСостава = Состав.Выгрузить();

Использование в версии:

Доступен, начиная с версии 8.0.
