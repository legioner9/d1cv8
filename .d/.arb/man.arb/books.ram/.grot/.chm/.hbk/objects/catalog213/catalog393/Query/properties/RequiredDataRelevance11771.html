Запрос (Query)

ТребуемаяАктуальностьДанных (RequiredDataRelevance)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: ТребуемаяАктуальностьДанных.
Актуальность данных, которые необходимо получить. Значение по умолчанию - Авто .
Для запроса Авто равно значению Актуальные.
Если у запроса свойство ИспользованиеКопийБазыДанных имеет значение ИспользоватьТолькоКопии, то свойство может влиять только на то, какая именно копия будет выбрана.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.19.
