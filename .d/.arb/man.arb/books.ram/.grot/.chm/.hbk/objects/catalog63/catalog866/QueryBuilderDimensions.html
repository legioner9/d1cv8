ИзмеренияПостроителяЗапроса (QueryBuilderDimensions)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ИзмерениеПостроителяЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются измерения.
Возможно обращение к измерению посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция измерений построителя запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ПостроительЗапроса, свойство Измерения

Использование в версии:

Доступен, начиная с версии 8.0.
