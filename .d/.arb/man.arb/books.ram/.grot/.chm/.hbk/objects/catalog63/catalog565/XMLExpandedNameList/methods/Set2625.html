СписокРасширенныхИменXML.Установить (XMLExpandedNameList.Set)

СписокРасширенныхИменXML (XMLExpandedNameList)

Установить (Set)

Доступен, начиная с версии 8.1.

Синтаксис:

Установить(<Индекс>, <Имя>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс - позиция для установки имени.

<Имя> (обязательный)

Тип: РасширенноеИмяXML.
Устанавливаемое расширенное имя XML.

Описание:

Устанавливает расширенное имя XML в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
