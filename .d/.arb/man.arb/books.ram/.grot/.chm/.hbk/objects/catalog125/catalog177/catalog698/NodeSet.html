НаборУзлов (NodeSet)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПланОбменаСсылка.<Имя плана обмена>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа ПланОбменаСсылка.

Свойства:

АвтоЗаполнение (AutoFill)

Методы:

Добавить (Add)
Заполнить (Fill)
Количество (Count)
Очистить (Clear)
Содержит (Contains)
Удалить (Delete)

Описание:

Представляет собой значение реквизита НаборУзлов объекта типа ПараметрыОбменаДанными, который, в свою очередь является значением свойства ОбменДанными, объектов, представляющих данные, по которым ведется учет изменений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПараметрыОбменаДанными, свойство Получатели

Использование в версии:

Доступен, начиная с версии 8.0.
