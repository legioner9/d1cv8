ПараметрыДоступа (AccessParameters)

Доступность (Accessibility)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит информацию о наличии права текущего пользователя на заданный объект метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - права есть, Ложь - в противном случае.

См. также:

МенеджерПользователейИнформационнойБазы, метод ТекущийПользователь

Использование в версии:

Доступен, начиная с версии 8.0.
