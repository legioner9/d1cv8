РежимВосстановленияПутейФайловZIP (ZIPRestoreFilePathsMode)

РежимВосстановленияПутейФайловZIP (ZIPRestoreFilePathsMode)

Доступен, начиная с версии 8.0.

Значения

Восстанавливать (Restore)
НеВосстанавливать (DontRestore)

Описание:

Содержит варианты восстановления путей файлов при извлечении из архива.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЧтениеZipФайла, метод Извлечь
ЧтениеZipФайла, метод ИзвлечьВсе

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
