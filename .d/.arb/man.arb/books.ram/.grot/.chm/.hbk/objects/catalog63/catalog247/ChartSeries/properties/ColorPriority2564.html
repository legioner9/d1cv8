СерияДиаграммы.ПриоритетЦвета (ChartSeries.ColorPriority)

СерияДиаграммы (ChartSeries)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Всегда используется для вывода цвета серии.
Если значение свойства равно Ложь, а у соответствующей точки свойство ПриоритетЦвета = Истина, то будет использоваться значение Цвет точки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для следующих типов диаграмм:
  • Гистограмма;
  • ГистограммаОбъемная;
  • ГистограммаГоризонтальная;
  • ГистограммаГоризонтальнаяОбъемная;
  • Изометрическая;
  • ИзометрическаяПирамида.

Использование в версии:

Доступен, начиная с версии 8.0.
