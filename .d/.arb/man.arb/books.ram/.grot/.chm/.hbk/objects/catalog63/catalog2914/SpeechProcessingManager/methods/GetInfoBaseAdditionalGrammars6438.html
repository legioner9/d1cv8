МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьДополнительныеГрамматикиИнформационнойБазы (GetInfoBaseAdditionalGrammars)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДополнительныеГрамматикиИнформационнойБазы()

Возвращаемое значение:

Тип: Массив.
Массив элементов типа Строка.

Описание:

Возвращает ключи всех доступных дополнительных грамматик информационной базы или области, если вызов выполнен из области.
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
