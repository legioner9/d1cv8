ВытесняющиеВидыРасчета (DisplacingCalculationTypes)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ВытесняющиеВидыРасчета.<Имя плана видов расчета>.
Содержит список вытесняющих видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство определено только для объектов тех планов видов расчета, которые имеют признак "Использует период действия".

Использование в версии:

Доступен, начиная с версии 8.0.
