Глобальный контекст (Global context)

ПолучитьСоставСтандартногоИнтерфейсаOData (GetStandardODataInterfaceContent)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьСоставСтандартногоИнтерфейсаOData()

Возвращаемое значение:

Тип: Массив.
Массив объектов метаданных.

Описание:

Возвращает массив объектов метаданных, включенных в стандартный OData API.
Объекты, доступные для включения в стандартный OData API:

  • Справочник.<ИмяСправочника>
  • Документ.<ИмяДокумента>
  • ПланОбмена.<ИмяПланаОбмена>
  • ПланСчетов.<ИмяПланаСчетов>
  • ПланВидовРасчета.<ИмяПланаВидовРасчета>
  • ПланВидовХарактеристик.<ИмяПланаВидовХарактеристик>
  • РегистрБухгалтерии.<ИмяРегистраБухгалтерии>
  • РегистрСведений.<ИмяРегистраСведений>
  • РегистрРасчета.<ИмяРегистраРасчета>
  • РегистрНакопления.<ИмяРегистраНакопления>
  • ЖурналДокументов.<ИмяЖурналаДокументов>
  • Перечисление.<ИмяПеречисления>
  • Задача.<ИмяЗадачи>
  • БизнесПроцесс.<ИмяБизнесПроцесса>
  • Константа.<ИмяКонстанты>

Доступность:

Сервер, толстый клиент, внешнее соединение, интеграция.

Примечание:

В режиме совместимости конфигурации с версией 8.3.4 возвращает все объекты конфигурации, доступные для включения в стандартный OData API.

Пример:

Массив =  ПолучитьСоставСтандартногоИнтерфейсаOData();
   
Для Каждого ОбъектМетаданных Из Массив Цикл
    Сообщить(ОбъектМетаданных.ПолноеИмя());
КонецЦикла;
