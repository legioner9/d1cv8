РежимОтображенияЭлементаНастройкиКомпоновкиДанных (DataCompositionSettingsItemViewMode)

Обычный (Normal)

Доступен, начиная с версии 8.1.

Описание:

Указывает на обычный режим отображения.
Он будет доступен в табличном поле, если у расширения табличного поля отбора, свойство РежимОтображения будет установлено в значение Все.

Использование в версии:

Доступен, начиная с версии 8.1.
