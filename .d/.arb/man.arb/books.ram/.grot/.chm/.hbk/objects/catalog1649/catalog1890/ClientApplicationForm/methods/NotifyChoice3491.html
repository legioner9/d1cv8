ФормаКлиентскогоПриложения (ClientApplicationForm)

ОповеститьОВыборе (NotifyChoice)

Доступен, начиная с версии 8.2.

Синтаксис:

ОповеститьОВыборе(<ЗначениеВыбора>, <ВыборДобавлением>)

Параметры:

<ЗначениеВыбора> (обязательный)

Тип: Произвольный.
Результат выбора.

<ВыборДобавлением> (необязательный)

Тип: Булево, Неопределено.
Определяет, как поле ввода обработает выбранные значения при вводе множественных значений:
  • Ложь - выбранное значение заменяет текущее значение реквизита.
  • Истина - выбранное значение добавляется как новый элемент списка в значение реквизита.
  • Неопределено - Автоматическое определение. Используется значение одноименного параметра формы <ВыборДобавлением>.

Если поле ввода находится в режиме ввода единственного значения, то параметр игнорируется.
Значение по умолчанию: Неопределено.

Описание:

Посылает оповещение владельцу формы о выполнении выбора или подбора, передает ему выбранное значение. Закрывает форму в соответствии со значением свойства ЗакрыватьПриВыборе. Данное действие аналогично выполнению выбора в форме, открытой для выбора в поле ввода или ячейке табличного поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Может применяться в случае, когда открытие формы выбора или подбора выполнено средствами языка.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
