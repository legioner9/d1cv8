ПолеФормы.ПоложениеЗаголовка (FormField.TitleLocation)

ПолеФормы (FormField)

ПоложениеЗаголовка (TitleLocation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ПоложениеЗаголовкаЭлементаФормы.
Определяет положение заголовка относительно поля в макете формы. Следует заметить, что для отключения вывода заголовка следует установить это свойство в значение Нет. Свойство Заголовок, содержащее пустую строку, означает автоматическое определение заголовка, а не ее отключение.
Для полей (колонок), размещенных в таблице, любое значение данного свойства, кроме Нет, означает отображение заголовка в заголовке колонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
