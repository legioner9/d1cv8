Категория (Category)

Доступен, начиная с версии 8.1.

Тип: КатегорияОграниченияИдентичностиXS.
Содержит категорию ограничения идентичности.

Представляет свойство identity constraint category (http://www.w3.org/TR/xmlschema-1/#identity-constraint_name) информационного пространства определения.
