ЭлементОбъектHTML.ТипКода (HTMLObjectElement.CodeType)

ЭлементОбъектHTML (HTMLObjectElement)

ТипКода (CodeType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.
Тип носителя кода, на который ссылается атрибут classid объекта.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
