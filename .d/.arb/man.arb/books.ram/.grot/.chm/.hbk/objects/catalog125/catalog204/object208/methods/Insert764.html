РегистрНакопленияНаборЗаписей.<Имя регистра накопления> (AccumulationRegisterRecordSet.<Accumulation register name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс позиции, на которую надо вставить новую запись.

Возвращаемое значение:

Тип: РегистрНакопленияЗапись.<Имя регистра накопления>.
Созданная запись регистра.

Описание:

Вставляет новую запись в набор на указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
