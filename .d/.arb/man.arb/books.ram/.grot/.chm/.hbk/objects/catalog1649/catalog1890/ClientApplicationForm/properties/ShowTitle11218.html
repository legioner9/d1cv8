ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтображатьЗаголовок (ShowTitle)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Управляет отображением заголовка формы (включая звездочку модификации).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

  • Свойство применяется только в интерфейсе Такси.
  • В режиме основного окна приложения Обычный свойство применяется только для форм, расположенных на начальной странице.
  • В режиме основного окна приложения РабочееМесто, ПолноэкранноеРабочееМесто и Киоск свойство применяется для всех управляемых форм.
  • В интерфейсе Такси, если на начальной странице располагается одна форма и свойство <ОтображатьЗаголовок> для этой формы установлено в значение Истина, то заголовок выводится вместо надписи "Начальная страница".
