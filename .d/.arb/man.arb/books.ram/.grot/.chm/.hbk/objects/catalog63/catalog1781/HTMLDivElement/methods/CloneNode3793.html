ЭлементБлокHTML.КлонироватьУзел (HTMLDivElement.CloneNode)

ЭлементБлокHTML (HTMLDivElement)

КлонироватьУзел (CloneNode)

Доступен, начиная с версии 8.2.

Синтаксис:

КлонироватьУзел(<ГлубокоеКопирование>)

Параметры:

<ГлубокоеКопирование> (обязательный)

Признак "глубокого" копирования.

Возвращаемое значение:

Тип: АтрибутHTML, ЭлементHTML, ЭлементКнопкаHTML, ЭлементВводаHTML, ЭлементЗаголовокHTML, ЭлементРазметкаHTML, ЭлементПлавающийФреймHTML, ЭлементВставкаHTML, ЭлементФреймHTML, ЭлементНаборФреймовHTML, ЭлементМетаHTML, ЭлементОбъектHTML, ЭлементСкриптHTML, ЭлементТаблицаHTML, ЭлементСтрокаТаблицыHTML, ЭлементЯчейкаТаблицыHTML, ЭлементКолонкаТаблицыHTML, ЭлементЗаголовокТаблицыHTML, ЭлементБлокHTML, ЭлементЛинияHTML, ЭлементФорматированногоТекстаHTML, ЭлементКартинкаHTML, ЭлементСвязьHTML, ЭлементЯкорьHTML, ЭлементАплетHTML, ЭлементФормаHTML, ДокументHTML, ТекстHTML, КомментарийHTML.
Копия данного узла.

Описание:

Копирует (клонирует) данный узел.
Для копии не определены родительский узел и пользовательские данные.
Копирование узла Элемент копирует все атрибуты и их значения, включая сгенерированные XML процессором значения по умолчанию.
Копирование дочерних узлов производится только в случае установленного признака "глубокого" копирования. "Глубокое" копирование включает текст, содержащийся в узле Элемент, если текст содержится как дочерний узел Текст.
Копирование узлов Атрибут отличается от копирования их как части узла Элемент тем, что признак явного указания свойства Указан будет установлен в значение Истина. Копирование узлов Атрибут всегда выполняется с копированием дочерних узлов (представляющих значение атрибута), вне зависимости от режима копирования.
Копирование узла автоматически конструирует поддерево, если соответствующий узел доступен, вне зависимости от режима копирования.
Копирование других типов узлов просто возвращает копию данного узла.
В случае копирования поддерева в режиме только чтения создает редактируемую копию, но дочерние узлы копии узла СсылкаНаСущность останутся в режиме только чтения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
