СегментПолилинейногоОбъектаГеографическойСхемы (GeographicalSchemaPolylineObjectSegment)

СегментПолилинейногоОбъектаГеографическойСхемы (GeographicalSchemaPolylineObjectSegment)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ГеографическиеКоординаты
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Предоставляет доступ к коллекции координатных точек, из которых состоит сегмент полилинейного объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.
