Доступен, начиная с версии 8.1.

ПриоритетЦвета (ColorPriority)

Текст (Text)
Цвет (Color)

Описание:

Используется для доступа к свойствам точки диаграммы. Позволяет установить название, расшифровку точки и другие свойства.
Если количество точек диаграммы уменьшилось и объект не вошел в измененный диапазон, при обращении к нему будет сгенерирована ошибка. Если для диаграммы интерактивно запрещена настройка точек (снят флаг "Настройка точек"), то при попытке изменить любые свойства, будет сгенерирована ошибка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
