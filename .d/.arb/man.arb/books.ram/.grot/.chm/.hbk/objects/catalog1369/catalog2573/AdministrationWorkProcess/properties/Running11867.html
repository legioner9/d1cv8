АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Активен (Running)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.
Определяет активность рабочего процесса:
  • Ложь – процесс неактивен (либо не загружен в память, либо не может выполнять клиентские запросы);
  • Истина – процесс активен (работает).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
