ОбрабатываемаяКартинка (ProcessingPicture)

УстановитьГлубинуЦвета (SetColorDepth)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьГлубинуЦвета(<ГлубинаЦвета>)

Параметры:

<ГлубинаЦвета> (необязательный)

Тип: ГлубинаЦвета.
Устанавливаемая глубина цвета картинки.

Описание:

Устанавливает новую глубину цвета.
Формат обрабатываемой картинки (результата) должен поддерживать глубину цвета. Например:

  • Формат JPEG не поддерживает использование 256 цветов.
  • Формат GIF поддерживает только использование 256 цветов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.16.
