Глобальный контекст (Global context)

ПобитовоеИНе (BitwiseAndNot)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПобитовоеИНе(<Число1>, <Число2>)

Параметры:

<Число1> (обязательный)

Тип: Число.
Первый операнд побитовой операции.
Значение должно быть целым числом в диапазоне от 0 до 2^32-1. Если число не целое или не укладывается в данный диапазон - генерируется исключение.

<Число2> (обязательный)

Тип: Число.
Второй операнд побитовой операции.
Значение должно быть целым числом в диапазоне от 0 до 2^32-1. Если число не целое или не укладывается в данный диапазон - генерируется исключение.

Возвращаемое значение:

Тип: Число.

Описание:

Возвращает результат логической побитовой операции И НЕ для заданной пары чисел. Эквивалентно вызову ПобитовоеИ(число1, ПобитовоеНе(число2)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.11.
