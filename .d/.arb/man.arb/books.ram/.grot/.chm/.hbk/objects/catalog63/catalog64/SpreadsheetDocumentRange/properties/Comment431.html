ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Примечание (Comment)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: РисунокТабличногоДокумента.
Содержит примечание к области.
Обратите внимание: для установки текста примечания следует использовать свойство Текст (например: Область.Примечание.Текст = "Щелкните левой кнопкой мыши для увеличения").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.0.
