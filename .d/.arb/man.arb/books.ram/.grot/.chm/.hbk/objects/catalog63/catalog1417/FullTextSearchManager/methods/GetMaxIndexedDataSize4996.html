МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ПолучитьМаксимальныйРазмерИндексируемыхДанных (GetMaxIndexedDataSize)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьМаксимальныйРазмерИндексируемыхДанных()

Возвращаемое значение:

Тип: Число.

Описание:

Получает максимальный размер индексируемых данных в байтах.
Требует наличия у пользователя права "Администрирование". Отсутствие права приводит к возникновению исключения.
Метод должен быть вызван из сеанса, в котором все разделители не используются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если возвращен 0, индексируются все данные.
Данные, размер которых превышает <МаксимальныйРазмер>, не индексируются.

Использование в версии:

Доступен, начиная с версии 8.3.7.
