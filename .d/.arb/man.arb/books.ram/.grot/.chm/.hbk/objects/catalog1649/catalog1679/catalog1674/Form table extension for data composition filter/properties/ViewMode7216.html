Расширение таблицы формы для отбора компоновки данных (Form table extension for data composition filter)

РежимОтображения (ViewMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: РежимОтображенияНастроекКомпоновкиДанных.
Свойство появляется у таблицы, если она связана с данными типа ОтборКомпоновкиДанных.
Свойство определяет режим отображения таблицы. Показываются все элементы отбора или только те, которые имеют режим отображения БыстрыйДоступ.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
