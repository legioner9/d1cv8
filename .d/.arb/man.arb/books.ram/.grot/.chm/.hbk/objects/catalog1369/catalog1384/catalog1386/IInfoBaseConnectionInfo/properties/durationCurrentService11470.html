Соединение.durationCurrentService (IInfoBaseConnectionInfo.durationCurrentService)

Соединение (IInfoBaseConnectionInfo)

durationCurrentService (durationCurrentService)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Число.
Время в миллисекундах, в течение которого соединение с информационной базой выполняет текущий вызов сервиса кластера.
0, если в данный момент вызов сервиса кластера не выполняется.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.12.
