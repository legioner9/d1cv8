Описание:

Тип: Произвольный.
Набор свойств содержит значения измерений регистра бухгалтерии. Доступ к значению осуществляется по имени, как оно задано в конфигурации.
Набор свойств используется для небалансовых измерений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

В качестве типа допускается использовать любые типы, кроме УникальныйИдентификатор, ДвоичныеДанные и строки неограниченной длины.
