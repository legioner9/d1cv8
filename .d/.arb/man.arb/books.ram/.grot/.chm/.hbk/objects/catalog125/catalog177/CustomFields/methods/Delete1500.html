ПоляНастройки (CustomFields)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: ПолеНастройки.
Удаляемое поле.

Описание:

Удаляет поле из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
