Глобальный контекст (Global context)

ПолучитьБуферДвоичныхДанныхИзСтроки (GetBinaryDataBufferFromString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьБуферДвоичныхДанныхИзСтроки(<Строка>, <КодировкаТекста>, <ЗаписатьBOM>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Строка, которую требуется преобразовать в буфер двоичных данных.

<КодировкаТекста> (необязательный)

Тип: КодировкаТекста, Строка.
Кодировка текста может быть задана как в виде значения перечисления КодировкаТекста, так и в виде строки с указанием названия кодировки.
Поддерживаемые названия кодировок:
  • UTF-8
  • UTF-16
  • UTF-16BE
  • UTF-16LE
  • ISO-8859-1
  • US-ASCII
  • gb18030
  • iso-8859-2
  • iso-8859-3
  • iso-8859-4
  • iso-8859-5
  • iso-8859-6
  • iso-8859-7
  • iso-8859-8
  • iso-8859-9
  • iso-8859-13
  • iso-8859-15
  • Shift_JIS
  • EUC-JP
  • Big5
  • Big5-HKSCS
  • GBK
  • GB2312
  • GB_2312-80
  • EUC-KR
  • KSC_5601
  • windows-949
  • windows-874
  • cp866
  • KOI8-R
  • KOI8-U
  • windows-1250
  • windows-1251
  • windows-1252
  • windows-1253
  • windows-1254
  • windows-1255
  • windows-1256
  • windows-1257
  • windows-1258
  • macintosh
  • x-mac-cyrillic
  • ISO-2022-JP
  • ISO-2022-KR
  • ISO-2022-CN
  • ISO-2022-CN-EXT
  • HZ-GB-2312

Значение по умолчанию: UTF8.

<ЗаписатьBOM> (необязательный)

Тип: Булево.
Определяет, будет ли добавлена метка порядка байт (BOM) кодировки текста в начало данных.
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: БуферДвоичныхДанных.

Описание:

Преобразует строку в буфер двоичных данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
