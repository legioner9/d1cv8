СерияДиаграммы (ChartSeries)

ФорматЗначенийВПодписях (LabelValueFormat)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Задает формат значений в подписях этой серии. Если формат пуст, то используется общий для диаграммы формат.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
