Индекс (IndexOf)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Индекс(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: ВнешнийИсточникДанныхТаблицаЗапись.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>.
Запись набора, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: Число.
Индекс указанной записи в наборе. Если не найдено, то возвращается -1.

Описание:

Получает индекс записи в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
