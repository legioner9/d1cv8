МенеджерКриптографии.ПроверитьНаличиеДанныхВПодписиАсинх (CryptoManager.CheckForDataInSignatureAsync)

МенеджерКриптографии (CryptoManager)

ПроверитьНаличиеДанныхВПодписиАсинх (CheckForDataInSignatureAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПроверитьНаличиеДанныхВПодписиАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Электронная подпись, в которой определяется наличие данных.
Данные подписи могут размещаться в файле (в этом случае указывается имя файла) или представлены в явном виде как ДвоичныеДанные, Поток, ПотокВПамяти или ФайловыйПоток.

Возвращаемое значение:

Тип: Обещание.
Результатом выполнения обещания является параметр типа Булево, содержащий признак наличия данных: Истина – если заданная подпись содержит данные и Ложь – в противном случае.

Описание:

Определяет наличия данных в электронной подписи: присоединенная подпись или отсоединенная.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
