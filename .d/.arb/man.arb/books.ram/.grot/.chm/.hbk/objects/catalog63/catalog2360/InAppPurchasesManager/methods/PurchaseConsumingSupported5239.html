МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПоддерживаетсяРасходованиеПокупок (PurchaseConsumingSupported)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоддерживаетсяРасходованиеПокупок()

Возвращаемое значение:

Тип: Булево.
  • Истина - для Windows Phone и для Android при наличии установленных служб Google Play Services телефонного модуля,
  • Ложь - для iOS.

Описание:

Определяет, доступно ли "расходование" покупок на текущей платформе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерВстроенныхПокупок, метод ИзрасходоватьПокупку

Использование в версии:

Доступен, начиная с версии 8.3.8.
