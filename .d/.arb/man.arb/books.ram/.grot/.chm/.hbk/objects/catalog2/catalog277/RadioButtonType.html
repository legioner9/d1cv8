ВидПереключателя (RadioButtonType)

Доступен, начиная с версии 8.3.2.

Значения

Авто (Auto)
Переключатель (RadioButton)
Тумблер (Tumbler)

Описание:

Содержит варианты режимов переключателя поля вида ПолеПереключателя в форме клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.2.
