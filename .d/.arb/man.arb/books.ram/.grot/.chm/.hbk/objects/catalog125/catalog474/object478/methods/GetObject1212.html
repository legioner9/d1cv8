ПланВидовРасчетаСсылка.<Имя плана видов расчета> (ChartOfCalculationTypesRef.<Chart of calculation types name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: ПланВидовРасчетаОбъект.<Имя плана видов расчета>, Неопределено.
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект базы данных, который можно изменить, записать или удалить.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
