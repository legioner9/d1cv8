ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Доступен, начиная с версии 8.3.8.

Значения

Авто (Auto)
НеОбновлятьАвтоматически (DontAutoUpdate)
ОбновлятьАвтоматически (AutoUpdate)

Описание:

Определяет режим обновления предопределенных данных.
Используется в методах &ПолучитьОбновлениеПредопределенныхДанных, &УстановитьОбновлениеПредопределенныхДанных объектов СправочникМенеджер.<Имя справочника>, ПланВидовРасчетаМенеджер.<Имя плана видов расчета>, ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик>, ПланСчетовМенеджер.<Имя плана счетов>, а также в методах ПолучитьОбновлениеПредопределенныхДанныхИнформационнойБазы и УстановитьОбновлениеПредопределенныхДанныхИнформационнойБазы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.1/data/enterprise. Имя типа XDTO: PredefinedDataUpdate.

Использование в версии:

Доступен, начиная с версии 8.3.8.
