ЧтениеДанных.ПрочитатьЦелое16Асинх (DataReader.ReadInt16Async)

ЧтениеДанных (DataReader)

ПрочитатьЦелое16Асинх (ReadInt16Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьЦелое16Асинх(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, используемый при чтении числа.
Если не задан, используется порядок, определенный для текущего экземпляра ЧтениеДанных.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - результат чтения типа Число.
Если не удалось прочитать число из-за того, что был достигнут конец потока, результатом обещания будет значение Неопределено.

Описание:

Асинхронная функция. Считывает 16-битное целое число из потока.

Доступность:

Тонкий клиент, толстый клиент.
