МенеджерКалендарей.ИзменитьКалендарь (CalendarsManager.UpdateCalendar)

МенеджерКалендарей (CalendarsManager)

ИзменитьКалендарь (UpdateCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ИзменитьКалендарь(<Календарь>, <Данные>)

Параметры:

<Календарь> (обязательный)

Тип: ДанныеКалендаряУчетнойЗаписи.
Данные календаря учетной записи.

<Данные> (обязательный)

Тип: ДанныеКалендаряУчетнойЗаписи.
Данные календаря учетной записи.

Описание:

Изменяет календарь в соответствии с переданными данными.
Данный метод поддерживается не для любого календаря. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяИзменениеКалендаря.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерКалендарей, метод ПоддерживаетсяИзменениеКалендаря

Использование в версии:

Доступен, начиная с версии 8.3.4.
