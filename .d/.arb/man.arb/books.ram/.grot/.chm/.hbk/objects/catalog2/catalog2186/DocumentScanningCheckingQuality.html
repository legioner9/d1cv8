ПроверкаКачестваСканированияДокументов (DocumentScanningCheckingQuality)

Доступен, начиная с версии 8.3.23.

Значения

НеПроверять (DontCheck)
ПредупреждатьНижеВысокого (WarnBelowHigh)
ПредупреждатьНижеСреднего (WarnBelowMedium)
ТребоватьВысокое (RequireHigh)
ТребоватьСреднееПредупреждатьНижеВысокого (RequireMediumWarnBelowHigh)

Описание:

Определяет уровень проверки качества документа после его сканирования в методах ПоказатьСканированиеДокументов и СканироватьДокументыАсинх.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
