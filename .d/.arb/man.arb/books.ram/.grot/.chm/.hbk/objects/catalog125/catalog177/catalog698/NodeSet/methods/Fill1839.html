НаборУзлов.Заполнить (NodeSet.Fill)

НаборУзлов (NodeSet)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить()

Описание:

Предназначен для заполнения набора узлов. Объект НаборУзлов принадлежит определенному объекту ПараметрыОбменаДанными, принадлежащему определенному объекту данных, который, в свою очередь, относится к объекту метаданных, входящему в состав хотя бы одного плана обмена. Обращение к данному методу вызывает заполнение набора узлов ссылками на все узлы всех планов обмена, в состав которых входит объект метаданных, для которых указан признак АвтоРегистрация.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Об.ОбменДанными.Получатели.АвтоЗаполнение = Ложь;
Об.ОбменДанными.Получатели.Заполнить();

Использование в версии:

Доступен, начиная с версии 8.0.
