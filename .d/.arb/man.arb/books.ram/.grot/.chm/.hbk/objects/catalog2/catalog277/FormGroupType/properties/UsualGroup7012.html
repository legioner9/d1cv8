ОбычнаяГруппа (UsualGroup)

Доступен, начиная с версии 8.2.

Описание:

Обычная группа. Группа данного вида может содержать поля, таблицы, кнопки и группы видов ОбычнаяГруппа, КоманднаяПанель, Страницы.

Использование в версии:

Доступен, начиная с версии 8.2.
