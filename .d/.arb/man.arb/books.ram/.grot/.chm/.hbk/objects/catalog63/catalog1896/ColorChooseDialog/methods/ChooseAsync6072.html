ДиалогВыбораЦвета.ВыбратьАсинх (ColorChooseDialog.ChooseAsync)

ДиалогВыбораЦвета (ColorChooseDialog)

ВыбратьАсинх (ChooseAsync)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Цвет - выбранный цвет;

Описание:

Асинхронная функция. Открывает диалог выбора цвета.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
