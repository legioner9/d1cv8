ПланСчетовОбъект.<Имя плана счетов>.Удалить (ChartOfAccountsObject.<Chart of accounts name>.Delete)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет элемент плана счетов из базы данных.
Важно! В отличие от метода УстановитьПометкуУдаления, данный метод производит непосредственное удаление элемента плана счетов без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Также удаляются все подчиненные элементы.

Пример:

Объект.Удалить();

Использование в версии:

Доступен, начиная с версии 8.0.
