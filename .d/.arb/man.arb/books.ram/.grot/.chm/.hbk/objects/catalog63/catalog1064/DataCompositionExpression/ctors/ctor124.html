ВыражениеКомпоновкиДанных (DataCompositionExpression)

На основе строки

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ВыражениеКомпоновкиДанных(<Строка>)

Параметры:

<Строка> (необязательный)

Тип: Строка.
Строка, используемая в качестве выражения.
Значение по умолчанию: Пустая строка.

Описание:

Создает выражение компоновки данных на основе строки.

Использование в версии:

Доступен, начиная с версии 8.1.
