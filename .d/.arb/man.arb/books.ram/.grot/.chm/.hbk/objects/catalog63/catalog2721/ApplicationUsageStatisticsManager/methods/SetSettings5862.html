МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

УстановитьНастройки (SetSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьНастройки(<ТекстНастройки>)

Параметры:

<ТекстНастройки> (обязательный)

Тип: Строка, Неопределено.
Текст новых настроек регистрации статистики использования приложения или значение Неопределено, отменяющее установку настройки регистрации из языка.

Описание:

Устанавливает новые настройки регистрации событий/экранов статистики использования приложения. Данные настройки регистрации имеют наивысший приоритет.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
