ПолеВвода.АвтоОтметкаНезаполненного (TextBox.AutoMarkIncomplete)

ПолеВвода (TextBox)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает, нужно ли формировать автоматически состояние свойства ОтметкаНезаполненного в зависимости от устанавливаемого или формируемого в поле ввода значения. Если Истина, тогда при установке и формировании значения в поле ввода будет меняться состояние свойства ОтметкаНезаполненного: если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то ОтметкаНезаполненного будет выставлено в Истина, иначе - в Ложь.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
