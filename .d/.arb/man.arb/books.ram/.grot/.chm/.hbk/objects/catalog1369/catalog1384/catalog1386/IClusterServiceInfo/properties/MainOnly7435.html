Сервис кластера.MainOnly (IClusterServiceInfo.MainOnly)

Сервис кластера (IClusterServiceInfo)

MainOnly (MainOnly)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит признак поддержки главным и дополнительным менеджером кластера.
1, если сервис поддерживается только главным менеджером кластера и не может быть перенесена в процесс дополнительного менеджера кластера.
0, если сервис может поддерживаться как главным, так и дополнительным менеджером кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
