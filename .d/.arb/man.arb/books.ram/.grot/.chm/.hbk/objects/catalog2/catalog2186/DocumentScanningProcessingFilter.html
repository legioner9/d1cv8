ФильтрОбработкиСканированияДокументов (DocumentScanningProcessingFilter)

Доступен, начиная с версии 8.3.23.

Значения

Нет (None)
Текст (Text)
ТекстСКартинками (TextWithPictures)

Описание:

Содержит варианты фильтров для обработки обнаруженного документа в методах ПоказатьСканированиеДокументов и СканироватьДокументыАсинх.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
