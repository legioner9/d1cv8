ГеографическаяСхема.ПолучитьКартинку (GeographicalSchema.GetPicture)

ГеографическаяСхема (GeographicalSchema)

ПолучитьКартинку (GetPicture)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьКартинку(<Ширина>, <Высота>, <ФорматКартинки>, <ЧерноБелая>)

Параметры:

<Ширина> (обязательный)

Тип: Число.
Ширина картинки.

<Высота> (обязательный)

Тип: Число.
Высота картинки.

<ФорматКартинки> (необязательный)

Тип: ФорматКартинки.
Требуемый формат картинки.
Допустимые значения для растровых картинок: BMP, PNG и исходный тип. Для векторных картинок допускается указывать только исходный тип.
Значение по умолчанию: BMP.

<ЧерноБелая> (необязательный)

Тип: Булево.
Признак получения черно-белой картинки.
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: Картинка.

Описание:

Получает картинку заданных размеров в указанном формате.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
