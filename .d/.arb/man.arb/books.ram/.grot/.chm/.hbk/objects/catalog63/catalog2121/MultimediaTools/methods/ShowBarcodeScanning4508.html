ПоказатьСканированиеШтрихКодов (ShowBarcodeScanning)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоказатьСканированиеШтрихКодов(<Заголовок>, <Обработчик сканирования>, <Обработчик закрытия>, <ТипШтрихКода>)

Параметры:

<Заголовок> (необязательный)

Тип: Строка.
Текст заголовка, который будет показан в окне сканирования штрихкодов.

<Обработчик сканирования> (обязательный)

Тип: ОписаниеОповещения.
Указывает обработчик, вызываемый при успешном сканировании.
Процедура обработчика должна иметь следующие параметры:
  • ШтрихКод – Cканированный код. Тип значения – Строка;
  • Результат – Возвращаемый параметр. Указывает, успешно ли сканирование. Тип значения – Булево;
  • Сообщение – Возвращаемый параметр. Это сообщение будет отображено пользователю, чтобы проинформировать его о результатах последнего сканирования. Тип значения – Строка:
    • Если сканирование не успешно, сообщение отображается красным цветом.
    • Если текст сообщения не задан, поведение различается в зависимости от успешности сканирования:
      • Сканирование успешно – ничего не отображается.
      • Сканирование не успешно - отображается сообщение "Ошибка обработки штрих-кода!"
  • <ДополнительныеПараметры>.

<Обработчик закрытия> (необязательный)

Тип: ОписаниеОповещения.
Указывает обработчик, вызываемый при закрытии окна сканирования.
Процедура обработчика должна иметь параметры:
  • Дополнительные параметры.

<ТипШтрихКода> (необязательный)

Тип: ТипШтрихКода, Массив, Неопределено.
Указывает, какие типы штрихкодов должны распознаваться. Содержит один объект или массив типов штрихкода. Если Неопределено, то распознаются все штрихкоды.

Описание:

Выводит на экран интерфейс сканирования.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяСканированиеШтрихКодов.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
