ЭлементТаблицаHTML.УстановитьУзелАтрибутаИдентификатора (HTMLTableElement.SetIdAttributeNode)

ЭлементТаблицаHTML (HTMLTableElement)

УстановитьУзелАтрибутаИдентификатора (SetIdAttributeNode)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьУзелАтрибутаИдентификатора(<Атрибут>, <ЭтоИД>)

Параметры:

<Атрибут> (обязательный)

Тип: АтрибутHTML.
Атрибут элемента.

<ЭтоИД> (обязательный)

Тип: Булево.
Признак идентификатора.

Описание:

Устанавливает атрибуту признак идентификатора.
Если значение параметра <ЭтоИД> = Истина, указанный атрибут становится "идентификатором, определенным пользователем".
Причины вызова исключений:

  • данный узел в режиме Только для чтения;
  • указанный атрибут не является атрибутом данного элемента.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не меняет установленной в документе схемы.

Использование в версии:

Доступен, начиная с версии 8.2.
