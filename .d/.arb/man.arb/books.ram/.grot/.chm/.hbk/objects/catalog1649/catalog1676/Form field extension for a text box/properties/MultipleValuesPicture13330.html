Расширение поля формы для поля ввода (Form field extension for a text box)

КартинкаМножественныхЗначений (MultipleValuesPicture)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Картинка.
Используется, если для получения картинки множественных значений в свойстве поля ввода ПутьКДаннымКартинкиМножественногоЗначения может быть указана колонка c типом Число или Булево:
  • Для колонки с типом Число, значение в колонке является индексом элемента в наборе картинок, задаваемых данным свойством.
  • Для колонки с типом Булево, если Истина используется индекс 0, а если Ложь - используется индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
