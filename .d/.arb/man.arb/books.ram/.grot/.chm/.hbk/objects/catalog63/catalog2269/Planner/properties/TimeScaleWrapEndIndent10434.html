Планировщик.ОтступСКонцаПереносаШкалыВремени (Planner.TimeScaleWrapEndIndent)

Планировщик (Planner)

ОтступСКонцаПереносаШкалыВремени (TimeScaleWrapEndIndent)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Устанавливает отступ с начала отображения после переноса шкалы времени в единицах, указанных в свойстве ЕдиницаПериодическогоВарианта.
Пример: рабочий день начинается в 9:00 и заканчивается в 17:00, но на шкале времени отображается полный диапазон времени – с 00:00 до 24:00.
Данное свойство и свойство ОтступСНачалаПереносаШкалыВремени предназначены для настройки диапазона отображаемого времени – так, чтобы на календаре отображался только рабочий день в указанном выше диапазоне. Для этого данному свойству следует установить значение "7", а свойство ОтступСНачалаПереносаШкалыВремени – "9".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
