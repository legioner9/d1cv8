ХешированиеДанных (DataHashing)

По указанной хеш-функции

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый ХешированиеДанных(<ХешФункция>)

Параметры:

<ХешФункция> (обязательный)

Тип: ХешФункция.
Тип хеш-функции.

Описание:

Создает объект, реализующий инкрементальный расчет хеш-суммы по добавленным данным. Способ расчета и тип вычисляемого значения определяются указанным типом хеш-функции.

Использование в версии:

Доступен, начиная с версии 8.3.1.
