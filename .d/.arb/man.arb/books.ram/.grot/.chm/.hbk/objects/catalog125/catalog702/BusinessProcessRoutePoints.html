ТочкиМаршрута (BusinessProcessRoutePoints)

ТочкиМаршрута (BusinessProcessRoutePoints)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются точки маршрута бизнес-процесса.
Возможно обращение к точке маршрута бизнес-процесса посредством оператора [...]. В качестве аргумента передается индекс точки маршрута бизнес-процесса в коллекции (нумерация с 0).

Свойства:

<Имя точки маршрута бизнес-процесса> (<Business process route point name>)

Методы:

Индекс (IndexOf)
Количество (Count)
Получить (Get)

Описание:

Представляет коллекцию точек маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Количество = БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута.Количество();

Для каждого Точка Из БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута Цикл
    Сообщить(НСтр("ru='Точка маршрута: ';en='Route point:'") + Точка.Имя);
    Index = БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута.Индекс(Точка);
    Сообщить(НСтр("ru='Индекс точки: ';en='Point index: '") + Index);
КонецЦикла;
