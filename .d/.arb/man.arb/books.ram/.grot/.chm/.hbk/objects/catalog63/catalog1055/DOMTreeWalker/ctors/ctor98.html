ОбходДереваDOM (DOMTreeWalker)

На основании документа DOM

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ОбходДереваDOM(<Документ>, <Отбор>, <Узел>, <ОбходитьСсылкиНаСущности>)

Параметры:

<Документ> (обязательный)

Тип: ДокументDOM.
Документ DOM, которому принадлежат узлы.

<Отбор> (необязательный)

Тип: ФильтрУзловDOM.
Отбор узлов DOM. Если отбор не указан, то будут проитерированы все узлы.

<Узел> (необязательный)

Тип: АтрибутDOM, ДокументDOM, ЭлементDOM, ОпределениеТипаДокументаDOM, НотацияDOM, СущностьDOM, ФрагментДокументаDOM, ТекстDOM, КомментарийDOM, СекцияCDATADOM, ИнструкцияОбработкиDOM, СсылкаНаСущностьDOM, ПространствоИменXPath.
Если узел не указан, то в качестве корневого узла будет использован документ DOM.

<ОбходитьСсылкиНаСущности> (необязательный)

Тип: Булево.
Признак обхода ссылок на сущности DOM. Истина - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.
Значение по умолчанию: Истина.

Описание:

Создает экземпляр объекта.

Использование в версии:

Доступен, начиная с версии 8.1.
