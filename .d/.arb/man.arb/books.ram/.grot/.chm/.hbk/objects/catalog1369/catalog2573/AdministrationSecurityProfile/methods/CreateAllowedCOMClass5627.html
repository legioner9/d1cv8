АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

СоздатьРазрешенныйCOMКласс (CreateAllowedCOMClass)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьРазрешенныйCOMКласс()

Возвращаемое значение:

Тип: РазрешенныйCOMКласс.

Описание:

Создает новый разрешенный COM-класс для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
