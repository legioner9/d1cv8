Порт (Port)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Число.
IP-порт агента сервера 1С:Предприятия. От 0 по 65535.
Значение 0 предполагает использование порта агента по умолчанию - 1540.
Не используется, если IPАдресАгента - пустая строка.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
