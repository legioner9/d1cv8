Рекомендуется использовать:

  • ПреобразованиеККаноническомуXML

Доступен, начиная с версии 8.1.

Конструкторы:

По умолчанию

Описание:

Предназначен для организации последовательной записи документов и фрагментов XML в каноническом виде (см. w3c стандарты - Canonical XML, Exclusive XML Canonicalization).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
