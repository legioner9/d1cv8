Соединение с агентом сервера.GetInfoBaseLocks (IServerAgentConnection.GetInfoBaseLocks)

Соединение с агентом сервера (IServerAgentConnection)

GetInfoBaseLocks (GetInfoBaseLocks)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBaseLocks(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, для которого необходимо получить список блокировок.

<ИнформационнаяБаза> (обязательный)

Тип: Описание информационной базы.
Описание информационной базы, для которой необходимо получить список блокировок.

Возвращаемое значение:

Тип: COMSafeArray.
Массив блокировок объектов информационной базы. Каждая блокировка представлена объектом с интерфейсом Блокировка.

Описание:

Получает массив блокировок объектов информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
