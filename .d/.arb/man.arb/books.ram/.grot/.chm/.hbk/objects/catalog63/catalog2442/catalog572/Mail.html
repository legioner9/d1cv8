Почта (Mail)

Доступен, начиная с версии 8.0.

Методы:

Выбрать (Get)
Отключиться (Logoff)
Подключиться (Logon)
Послать (Send)

Конструкторы:

По умолчанию

Описание:

Предназначен для доступа к почтовой системе, установленной на компьютере. Используется технология MAPI.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

Почта = Новый Почта;

Использование в версии:

Доступен, начиная с версии 8.0.
