СпособКодированияНеASCIIСимволовИнтернетПочтовогоСообщения (InternetMailMessageNonASCIISymbolsEncodingMode)

Доступен, начиная с версии 8.1.

Значения

MIME (MIME)
QuotedPrintable (QuotedPrintable)
БезКодирования (None)

Описание:

Содержит варианты способов кодирования не ASCII символов в почтовом сообщении.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
