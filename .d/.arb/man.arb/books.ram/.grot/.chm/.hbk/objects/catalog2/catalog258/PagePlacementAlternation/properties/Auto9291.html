ЧередованиеРасположенияСтраниц (PagePlacementAlternation)

Авто (Auto)

Доступен, начиная с версии 8.3.1.

Описание:

Если свойство ДвусторонняяПечать (для ТабличныйДокумент или ГрафическаяСхема) имеет значение Нет, то используется обычный вывод полей и колонтитулов, если ДвусторонняяПечать имеет значение ПереворотВлево, то используется вывод со значением ЗеркальноСлева, если ДвусторонняяПечать имеет значение ПереворотВверх, то используется вывод со значением ЗеркальноСверху.

Использование в версии:

Доступен, начиная с версии 8.3.1.
