ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Удалить (Delete)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: Число, ИспользуемаяКопияБазыДанных.
Используемая копия базы данных или позиция копии в коллекции.

Описание:

Удаляет используемую копию из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
