МенеджерКалендарей.ПолучитьОсновнойКалендарь (CalendarsManager.GetDefaultCalendar)

МенеджерКалендарей (CalendarsManager)

ПолучитьОсновнойКалендарь (GetDefaultCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьОсновнойКалендарь()

Возвращаемое значение:

Тип: ЛокальныйКлючКалендаря.

Описание:

Получает основной календарь.

  • Для устройств на базе Android - определяется не локальный календарь с наибольшим количеством записей. Если найдены календари с одинаковым количеством записей, используется календарь с наименьшим идентификатором. Если таковой не найден, выполняется поиск локального календаря с наибольшим количеством записей. Если найдены календари с одинаковым количеством записей, будет выбран календарь с наименьшим идентификатором. Если календарей нет, то будет создан календарь с именем "LOCAL" для фиктивной учетной записи (с именем и типом "LOCAL").
  • Для устройств на базе iOS - используется штатный системный метод.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
