ОбъектМетаданных: ПараметрWebСервиса (MetadataObject: WebServiceParameter)

ВозможноПустое (Nillable)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если это свойство установлено в значение Истина, то для операции Web-сервиса это означает, что она может возвращать неопределенное значение, а для параметра операции Web-сервиса это означает, что параметр может принимать неопределенное значение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
