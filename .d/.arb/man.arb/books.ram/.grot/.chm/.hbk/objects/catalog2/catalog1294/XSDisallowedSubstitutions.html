НедопустимыеПодстановкиXS (XSDisallowedSubstitutions)

Доступен, начиная с версии 8.1.

Значения

Все (All)
Ограничение (Restriction)
Подстановка (Substitution)
Расширение (Extension)

Описание:

Содержит виды запрета подстановки.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
