ДвоичныеДанные.ОткрытьПотокДляЧтения (BinaryData.OpenStreamForRead)

ДвоичныеДанные (BinaryData)

ОткрытьПотокДляЧтения (OpenStreamForRead)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ОткрытьПотокДляЧтения()

Возвращаемое значение:

Тип: Поток.
Возвращает Поток, позиция которого установлена в начало двоичных данных.

Описание:

Открывает поток для чтения двоичных данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

С помощью данного метода можно открыть несколько независимых потоков для одного экземпляра объекта ДвоичныеДанные. При этом позиция каждого экземпляра потока не зависит от позиции других открытых потоков.

Использование в версии:

Доступен, начиная с версии 8.3.9.
