КоллекцияИзмеренийПланировщика.Удалить (PlannerDimensionCollection.Delete)

КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число, ИзмерениеПланировщика.
Индекс удаляемого измерения или само измерение.

Описание:

Удаляет измерение в заданной позиции коллекции измерений планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
