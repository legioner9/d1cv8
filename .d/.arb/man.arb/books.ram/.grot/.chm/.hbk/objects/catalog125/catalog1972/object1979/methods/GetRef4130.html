ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьСсылку(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.
Значение, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: ВнешнийИсточникДанныхТаблицаСсылка.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>.

Описание:

Формирует ссылку из значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных ОбъектныеДанные.

Доступен, начиная с версии 8.2.
