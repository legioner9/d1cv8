АдминистраторАбонента (SubscriberAdministrator)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Если содержит значение типа Булево, то метод ПолучитьПользователей возвращает соответственно только администраторов абонента или не администраторов абонента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
