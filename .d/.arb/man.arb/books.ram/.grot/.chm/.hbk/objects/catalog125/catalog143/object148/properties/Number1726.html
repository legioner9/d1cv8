ДокументВыборка.<Имя документа>.Номер (DocumentSelection.<Document name>.Number)

ДокументВыборка.<Имя документа> (DocumentSelection.<Document name>)

Номер (Number)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Число, Строка.
Содержит номер документа. Тип значения зависит от установки в Конфигураторе представления номера документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
