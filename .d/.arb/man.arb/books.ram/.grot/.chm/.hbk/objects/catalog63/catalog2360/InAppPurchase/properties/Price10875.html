ВстроеннаяПокупка (InAppPurchase)

Цена (Price)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит численное значение стоимости покупки. При использовании необходимо учитывать значение свойства Валюта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
