ЭлементЯчейкаТаблицыHTML.НеПереносить (HTMLTableCellElement.NoWrap)

ЭлементЯчейкаТаблицыHTML (HTMLTableCellElement)

НеПереносить (NoWrap)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает, что в данной ячейке текст не должен переноситься по строкам.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
