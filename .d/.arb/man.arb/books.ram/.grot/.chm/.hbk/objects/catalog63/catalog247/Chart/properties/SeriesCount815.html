Диаграмма.КоличествоСерий (Chart.SeriesCount)

Диаграмма (Chart)

КоличествоСерий (SeriesCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит количество серий диаграммы.
Важно! Нельзя изменять данное значение, когда диаграмма находится в режиме связи с источником данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
