ИнформацияОСетевомАдаптере (NetworkAdapterInformation)

ЭтоТекущийАдаптер (IsCurrentAdapter)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: Булево, Неопределено.
Определяет, является ли данный сетевой адаптер адаптером, через который установлено соединение в данный момент:
  • Истина - соединение клиента установлено через данный адаптер,
  • Ложь - в противном случае.
  • Для веб-клиента всегда содержит Неопределено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
