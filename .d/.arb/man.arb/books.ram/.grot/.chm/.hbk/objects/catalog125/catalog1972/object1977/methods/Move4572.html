Сдвинуть (Move)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Сдвинуть(<Запись>, <Смещение>)

Параметры:

<Запись> (обязательный)

Тип: Число, ВнешнийИсточникДанныхТаблицаЗапись.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>.
Индекс сдвигаемой записи либо сама запись.

<Смещение> (обязательный)

Тип: Число.
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца списка записей набора (увеличение индекса), отрицательное - к началу.

Описание:

Сдвигает запись в наборе на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
