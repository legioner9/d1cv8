HTTPЗапрос.ПолучитьИмяФайлаТела (HTTPRequest.GetBodyFileName)

HTTPЗапрос (HTTPRequest)

ПолучитьИмяФайлаТела (GetBodyFileName)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИмяФайлаТела()

Возвращаемое значение:

Тип: Строка, Неопределено.
Неопределено - имя файла не было установлено.

Описание:

Получает имя файла, откуда будет считано тело запроса при отправке.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
