КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Доступен, начиная с версии 8.0.

Элементы коллекции:

СводнаяТаблица
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются СводнаяТаблица.
Возможно обращение к коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) или имя таблицы.

Свойства:

<Имя встроенной таблицы> (<Embedded table name>)

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Содержит коллекцию сводных таблиц, встроенных в табличный документ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ТабличныйДокумент, свойство ВстроенныеТаблицы

Использование в версии:

Доступен, начиная с версии 8.0.
