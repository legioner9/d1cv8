ЗаписьXML.ОткрытьФайл (XMLWriter.OpenFile)

ЗаписьXML (XMLWriter)

ОткрытьФайл (OpenFile)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени и типу кодировки

Синтаксис:

ОткрытьФайл(<ИмяФайлаXML>, <ТипКодировки>, <ДобавлятьBOM>)

Параметры:

<ИмяФайлаXML> (обязательный)

Тип: Строка.
Имя файла, в который будет записываться текст XML.

<ТипКодировки> (необязательный)

Тип: Строка.
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае файл будет записан в соответствующей кодировке. При этом в объявлении XML-документа (если таковое будет записано) будет присутствовать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то для записи файла будет использована кодировка UTF8, а атрибут 'encoding' не будет присутствовать в объявлении XML-документа.

<ДобавлятьBOM> (необязательный)

Тип: Булево.
Определяет, будет ли добавлена метка порядка байт (BOM) к результирующему файлу XML.
Значение по умолчанию: Истина.

Описание варианта метода:

Позволяет указать тип кодировки, который будет использован для записи файла XML.

Вариант синтаксиса: С параметрами записи

Синтаксис:

ОткрытьФайл(<ИмяФайлаXML>, <ПараметрыЗаписи>, <ДобавлятьBOM>)

Параметры:

<ИмяФайлаXML> (обязательный)

Тип: Строка.
Имя файла, в который будет записываться текст XML.

<ПараметрыЗаписи> (обязательный)

Тип: ПараметрыЗаписиXML.
Параметры записи XML.

<ДобавлятьBOM> (необязательный)

Тип: Булево.
Определяет, будет ли добавлена метка порядка байт (BOM) к результирующему файлу XML.
Значение по умолчанию: Истина.

Описание варианта метода:

Позволяет указать параметры записи XML.

Описание:

Открывает файл для записи XML. Позволяет указать тип кодировки, который будет использован для записи файла XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ЗаписьXML = Новый ЗаписьXML;
ЗаписьXML.ОткрытьФайл("c:\docs\data.xml", "UTF-8", Истина);

Использование в версии:

Доступен, начиная с версии 8.1.
