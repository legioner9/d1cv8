РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

КоличествоИтераций (IterationCount)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Число итераций, которые выполняет алгоритм для расчета решения крупных полных подматриц.
При достижении требуемого числа итераций расчет останавливается.
Допускается целое число, не меньше 1.
Значение по умолчанию – 200. Оптимальные значения: от 25 до 200.
Не рекомендуется устанавливать слишком малое значение, так как это снижает точность расчета.
Не рекомендуется устанавливать слишком большое значение, так как на поздних итерациях точность решения увеличивается незначительно, а каждая итерация – операция, требующая дополнительных расходов ресурсов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
