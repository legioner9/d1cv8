МенеджерОтображенияРекламы (AdRepresentationManager)

ПоказатьПолноэкраннуюРекламу (ShowFullScreenAd)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоказатьПолноэкраннуюРекламу(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Указывает обработчик, вызываемый при завершении показа рекламного баннера.
Процедура обработчика должна иметь следующие параметры:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Отобразить полноэкранный баннер.
Если отображение рекламы отключено либо у устройства нет доступа к сети Интернет, метод вызовет процедуру-обработчик и баннер показан не будет.
На платформе Android при использовании сервиса AdMob при загрузке баннера используется идентификатор, который был задан в результате вызова метода УстановитьИдентификаторПолноэкраннойРекламы. Если идентфикатор баннера был изменен во время отображения баннера, то новый баннер будет загружен и отображен при повторном вызове метода. Если идентификатор баннера не установлен, то будет сгенерировано исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
