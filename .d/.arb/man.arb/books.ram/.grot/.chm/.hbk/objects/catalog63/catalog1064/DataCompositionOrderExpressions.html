ВыраженияУпорядочиванияКомпоновкиДанных (DataCompositionOrderExpressions)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ВыражениеУпорядочиванияКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения упорядочивания.
Возможно обращение к выражению упорядочивания посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция выражений упорядочивания.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
