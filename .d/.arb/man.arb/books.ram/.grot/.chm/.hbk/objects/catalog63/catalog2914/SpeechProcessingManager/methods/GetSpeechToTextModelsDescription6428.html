МенеджерРаботыСРечью.ПолучитьОписаниеМоделейРаспознавания (SpeechProcessingManager.GetSpeechToTextModelsDescription)

МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьОписаниеМоделейРаспознавания (GetSpeechToTextModelsDescription)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьОписаниеМоделейРаспознавания(<ПараметрыМодели>)

Параметры:

<ПараметрыМодели> (необязательный)

Тип: ПараметрыМоделиРаспознаванияРечи, Неопределено.
Если Неопределено - будет получено описание всех доступных моделей.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ОписаниеМоделиРаспознаванияРечи.

Описание:

Возвращает описания всех доступных моделей распознавания. Если задан отбор в виде параметров модели, то возвращает описания всех доступных моделей с учетом отбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
