Описание:

Содержит описание временной таблицы. Используется для описания временной таблицы, которая не описана в пакетном запросе. Например, для таблицы из менеджера временных таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
