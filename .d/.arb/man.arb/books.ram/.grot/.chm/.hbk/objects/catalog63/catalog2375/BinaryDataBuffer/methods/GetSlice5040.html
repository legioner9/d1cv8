БуферДвоичныхДанных.ПолучитьСрез (BinaryDataBuffer.GetSlice)

БуферДвоичныхДанных (BinaryDataBuffer)

ПолучитьСрез (GetSlice)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьСрез(<Позиция>, <Количество>)

Параметры:

<Позиция> (обязательный)

Тип: Число.
Позиция, начиная с которой будет создан новый буфер.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется отобразить в срезе. Если на задано, то отображаются все байты от начала среза до конца исходного буфера.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: БуферДвоичныхДанных.

Описание:

Создает новый буфер, использующий заданное количество байтов из исходного буфера, начиная с заданной позиции (нумерация с 0). Если количество не задано, то новый буфер является представлением элементов текущего буфера, начиная с заданного индекса и до конца.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Копирование данных не выполняется, изменения в общих элементах будут доступны в обоих буферах.

Использование в версии:

Доступен, начиная с версии 8.3.9.
