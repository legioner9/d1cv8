Колонка (Column)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Свойство содержит имя колонки, в которую будет выведено значение. Процессор вывода использует данное свойство для вывода значения в соответствующую колонку. Если свойство содержит пустую строку, то процессор вывода выводит значение в следующую по порядку колонку текущей строки. Если свойство содержит имя несуществующей колонки, процессор вывода генерирует исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
