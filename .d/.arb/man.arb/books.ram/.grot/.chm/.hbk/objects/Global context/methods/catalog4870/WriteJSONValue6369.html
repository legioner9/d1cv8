Глобальный контекст (Global context)

ЗаписатьЗначениеJSON (WriteJSONValue)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗаписатьЗначениеJSON(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Структура, ФиксированнаяСтруктура, Массив, ФиксированныйМассив, Число, Строка, Булево, Неопределено.
Значение, которое требуется сериализовать в JSON.
Если тип параметра Структура, ФиксированнаяСтруктура, Массив, или ФиксированныйМассив, то значения, содержащиеся в них, должны быть одного из следующих типов:
  • Структура,
  • ФиксированнаяСтруктура,
  • Массив,
  • ФиксированныйМассив,
  • Число,
  • Строка,
  • Булево,
  • Неопределено.
При этом на значения типов Структура, ФиксированнаяСтруктура, Массив, или ФиксированныйМассив распространяются такие же ограничения, что и для параметра <Значение>.
Значение типа Неопределено преобразуется в NULL.

Если <Значение> не может быть преобразовано в/из JSON из-за неподдерживаемых типов: будет сгенерировано исключение: "Значение содержит данные недопустимых типов."

Если преобразование в строку невозможно из-за циклической зависимости объектов, генерируется исключение: "Значение содержит циклические зависимости".

Возвращаемое значение:

Тип: Строка.
JSON-строка

Описание:

Преобразует значение в JSON-строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
