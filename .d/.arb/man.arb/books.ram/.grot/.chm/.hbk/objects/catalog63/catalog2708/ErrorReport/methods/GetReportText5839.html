ОтчетОбОшибке.ПолучитьТекстОтчета (ErrorReport.GetReportText)

ОтчетОбОшибке (ErrorReport)

ПолучитьТекстОтчета (GetReportText)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТекстОтчета()

Возвращаемое значение:

Тип: Строка.

Описание:

Формирует и возвращает текст отчета. Этот текст представляет собой JSON-данные. Свойства, для отправки которых нужно разрешение пользователя, или которые не разрешены к отправке, будут не инициализированы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
