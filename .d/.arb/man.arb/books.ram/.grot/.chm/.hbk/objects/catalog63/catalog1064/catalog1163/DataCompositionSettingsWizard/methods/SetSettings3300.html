КонструкторНастроекКомпоновкиДанных.УстановитьНастройки (DataCompositionSettingsWizard.SetSettings)

КонструкторНастроекКомпоновкиДанных (DataCompositionSettingsWizard)

УстановитьНастройки (SetSettings)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьНастройки(<Настройки>, <ОписаниеОшибки>)

Параметры:

<Настройки> (обязательный)

Тип: НастройкиКомпоновкиДанных.
Редактируемый объект.

<ОписаниеОшибки> (обязательный)

Тип: Строка.
После выполнения метода параметр содержит строку с описанием уменьшения функциональности схемы. Если ошибок нет, параметр содержит пустую строку.

Возвращаемое значение:

Тип: Булево.
Ложь - есть возможность редактирования настроек без ограничения их функциональности. Истина, если придется усекать функциональность настроек.

Описание:

Устанавливает настройки в конструктор и проверяет возможность редактирования этих установок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
