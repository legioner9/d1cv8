СерияДиаграммыГанта.Значение (GanttChartSeries.Value)

СерияДиаграммыГанта (GanttChartSeries)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Ключевое значение, которое используется для идентификации серии в методе УстановитьСерию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Уникальность значения автоматически не обеспечивается.
Строковое представление значения используется, если не задано свойство Текст.

Использование в версии:

Доступен, начиная с версии 8.0.
