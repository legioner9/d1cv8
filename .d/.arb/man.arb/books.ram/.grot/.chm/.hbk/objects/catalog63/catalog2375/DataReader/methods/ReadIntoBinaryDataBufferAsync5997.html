ЧтениеДанных.ПрочитатьВБуферДвоичныхДанныхАсинх (DataReader.ReadIntoBinaryDataBufferAsync)

ЧтениеДанных (DataReader)

ПрочитатьВБуферДвоичныхДанныхАсинх (ReadIntoBinaryDataBufferAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Чтение в новый буфер

Синтаксис:

ПрочитатьВБуферДвоичныхДанныхАсинх(<Количество>)

Параметры:

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется прочитать. Если не задано, то выполняется чтение всех байтов до конца потока.
Значение по умолчанию: Неопределено.

Описание варианта метода:

Прочитать заданное количество байтов в новый БуферДвоичныхДанных.
Результат выполнения обещания - экземпляр БуферДвоичныхДанных, содержащий прочитанные байты.

Вариант синтаксиса: Чтение в существующий буфер

Синтаксис:

ПрочитатьВБуферДвоичныхДанныхАсинх(<Буфер>, <ПозицияВБуфере>, <Количество>)

Параметры:

<Буфер> (обязательный)

Тип: БуферДвоичныхДанных.
Буфер двоичных данных, в который требуется поместить прочитанные байты.

<ПозицияВБуфере> (обязательный)

Тип: Число.
Позиция в буфере, начиная с которой требуется записать прочитанные данные.

<Количество> (обязательный)

Тип: Число.
Количество байтов, которые требуется прочитать.

Описание варианта метода:

Прочитать из потока заданное количество байтов и поместить в заданный буфер, начиная с заданной позиции.
Результат Обещания - число фактически прочитанных байтов.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа БуферДвоичныхДанных.

Описание:

Асинхронная функция. Читает байты из потока в БуферДвоичныхДанных.

Доступность:

Тонкий клиент, толстый клиент.
