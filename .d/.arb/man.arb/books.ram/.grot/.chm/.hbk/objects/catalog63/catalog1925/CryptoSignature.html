ПодписьКриптографии (CryptoSignature)

Доступен, начиная с версии 8.3.20.

Свойства:

АлгоритмПодписи (SignAlgorithm)
АлгоритмХеширования (HashAlgorithm)
АрхивныеМеткиВремени (ArchiveTimestamps)
Данные (Data)
Индекс (Index)
МеткаВремениДанныхПроверкиПодписи (SignatureVerificationDataTimestamp)
МеткаВремениПодписи (SignatureTimestamp)
НеподтвержденноеВремяПодписи (UnverifiedSignatureDate)
СертификатПодписи (SignatureCertificate)
СертификатыПроверкиПодписи (SignatureVerificationCertificates)
ТипПодписи (SignatureType)

Описание:

Содержит данные одной подписи, полученной при разборе двоичных данных контейнера подписей с помощью метода ПолучитьКонтейнерПодписейКриптографии.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

КонтейнерПодписей = МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии(ДвоичныеДанныеПодписи);
Для каждого подпись из КонтейнерПодписей.Подписи Цикл
    Сообщить("Индекс: " + Подпись.Индекс);  
    Сообщить("ТипПодписи: " + Подпись.ТипПодписи);
    Сообщить("АлгоритмХэширования: " + Подпись.АлгоритмХэширования);
    Сообщить("АлгоритмПодписи: " + Подпись.АлгоритмПодписи);
    Сообщить("СертификатПодписи.ДатаОкончания: " + Подпись.СертификатПодписи.ДатаОкончания);
    Для каждого Сертификат из Подпись.СертификатыДляПроверкиПодписи Цикл
        Сообщить("ДатаОкончания: " + Сертификат.ДатаОкончания);
    КонецЦикла;
    Сообщить("НеподтвержденноеВремяПодписи: " + Подпись.НеподтвержденноеВремяПодписи);
    Если (Подпись.МеткаВремениПодписи <> Неопределено) Тогда
        Сообщить("МеткаВремениПодписи.Дата: " + Подпись.МеткаВремениПодписи.Дата);
    КонецЕсли;
    Если (Подпись.МеткаВремениДанныхПроверкиПодписи <> Неопределено) Тогда
        Сообщить("МеткаВремениДанныхПроверкиПодписи.Дата: " + Подпись.МеткаВремениДанныхПроверкиПодписи.Дата);
    КонецЕсли;
    Для каждого Метка из Подпись.АрхивныеМеткиВремени Цикл
        Сообщить("АрхивныеМеткиВремени.Дата: " + Метка.Дата);
    КонецЦикла;
КонецЦикла
