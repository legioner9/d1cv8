ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ОбработкаИнтерактивнойАктивации (InteractiveActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаИнтерактивнойАктивации(<ТочкаМаршрута>, <Задача>, <СтандартнаяОбработка>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса, связанная с задачей, на которой происходит интерактивная активация задачи.

<Задача>

Тип: ЗадачаСсылка.<Имя задачи>.
Задача, которую активирует пользователь.

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, обработка интерактивной активации производиться не будет.
Значение по умолчанию: Истина.

Описание:

Возникает при интерактивной активации пользователем задач, соответствующих данной точке маршрута бизнес-процесса. Вызов этого обработчика происходит при вызове метода АктивироватьИнтерактивно у задачи или метода АктивироватьИнтерактивно у бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида Действие.
В режиме запуска УправляемоеПриложение не вызывается.

Использование в версии:

Доступен, начиная с версии 8.0.
