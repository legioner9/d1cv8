ПланСчетовМенеджер.<Имя плана счетов>.ПустаяСсылка (ChartOfAccountsManager.<Chart of accounts name>.EmptyRef)

ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: ПланСчетовСсылка.<Имя плана счетов>.

Описание:

Получает пустое значение ссылки на план счетов данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Пример:

// Счет - реквизит отчета (имеет тип ПланСчетовСсылка)

СписокСчетов = Новый Массив;
Пока Счет <> ПланыСчетов.Основной.ПустаяСсылка() Цикл
    СписокСчетов.Добавить(Счет);
    Счет = Счет.Родитель;
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
