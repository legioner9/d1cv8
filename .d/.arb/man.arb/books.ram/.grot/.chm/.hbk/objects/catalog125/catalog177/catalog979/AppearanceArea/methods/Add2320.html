ОбластьОформления.Добавить (AppearanceArea.Add)

ОбластьОформления (AppearanceArea)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Представление>, <ТипПоля>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: Строка.
Путь к данным добавляемого поля.

<Представление> (необязательный)

Тип: Строка.
Представление добавляемого поля.
Значение по умолчанию: Пустая строка.

<ТипПоля> (необязательный)

Тип: ТипОбластиОформления.
Тип добавляемого поля.
Значение по умолчанию: Поле.

Возвращаемое значение:

Тип: ЭлементОбластиОформления.

Описание:

Добавляет поле в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
