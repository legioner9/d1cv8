СоставКоманднойПанелиФормыНаМобильномУстройстве (MobileDeviceFormCommandBarContent)

Доступен, начиная с версии 8.3.13.

Элементы коллекции:

ГруппаФормы, КнопкаФормы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит коллекцию элементов формы, входящих в командную панель мобильного клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
