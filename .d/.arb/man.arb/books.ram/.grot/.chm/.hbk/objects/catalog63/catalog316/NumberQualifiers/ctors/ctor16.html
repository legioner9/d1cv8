КвалификаторыЧисла (NumberQualifiers)

На основании описания числа

Доступен, начиная с версии 8.0.

Синтаксис:

Новый КвалификаторыЧисла(<ЧислоРазрядов>, <ЧислоРазрядовДробнойЧасти>, <ДопустимыйЗнак>)

Параметры:

<ЧислоРазрядов> (необязательный)

Тип: Число.
Общее число разрядов числа.
Значение по умолчанию: 0.

<ЧислоРазрядовДробнойЧасти> (необязательный)

Тип: Число.
Число разрядов дробной части числа.
Значение по умолчанию: 0.

<ДопустимыйЗнак> (необязательный)

Тип: ДопустимыйЗнак.
Допустимый знак числа.
Значение по умолчанию: Любой.

Описание:

Создает квалификатор числа заданием общего числа разрядов, числа разрядов дробной части и допустимого знака.

Пример:

Знак  = ДопустимыйЗнак.Неотрицательный;
ПараметрыЧисла = Новый КвалификаторыЧисла(10, 2, Знак);

Использование в версии:

Доступен, начиная с версии 8.0.
