ЭлементДанныхКонтактаМгновенныеСообщения (ContactDataItemInstantMessaging)

По умолчанию

Доступен, начиная с версии 8.3.4.

Синтаксис:

Новый ЭлементДанныхКонтактаМгновенныеСообщения(<ТипДанных>, <ТипМгновенныхСообщений>, <Идентификатор>)

Параметры:

<ТипДанных> (обязательный)

Тип: Строка, ТипАдресаМгновенныхСообщенийДанныхКонтакта.
Указывает, как интерпретировать значение.
Например, тип данных Рабочий показывает, что в значении хранится рабочий адрес.

<ТипМгновенныхСообщений> (обязательный)

Тип: Строка.
Тип мгновенных сообщений (например: "Skype").

<Идентификатор> (обязательный)

Тип: Строка.
Идентификатор учетной записи мгновенных сообщений.

Описание:

Создает элемент данных контакта мгновенных сообщений.

Использование в версии:

Доступен, начиная с версии 8.3.4.
