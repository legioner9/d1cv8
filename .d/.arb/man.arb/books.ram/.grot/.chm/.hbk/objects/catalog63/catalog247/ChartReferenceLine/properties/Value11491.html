ИнформационнаяЛинияДиаграммы.Значение (ChartReferenceLine.Value)

ИнформационнаяЛинияДиаграммы (ChartReferenceLine)

Значение (Value)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Число, Дата, ТочкаДиаграммы, Произвольный.
Значение, по которому выводится линия.
Допускаются следующие типы значений:
  • для информационной линии, расположенной в значениях серий диаграммы (свойство диаграммы ИнформационныеЛинииЗначений) - Число;
  • для информационной линии, расположенной в значениях точек диаграммы (свойство диаграммы ИнформационныеЛинииТочек:
    • если на оси точек откладываются точки - ТочкаДиаграммы;
    • если на оси точек откладываются числа (например, в точечной и пузырьковой диаграммах) - Число;
    • если на оси точек откладываются даты (например, в точечной и пузырьковой диаграмме) - Дата;
  • для информационной линии, расположенной в параметрах вывода компоновки данных:
    • ПолеКомпоновкиДанных;
    • для информационной линии значений - Число;
    • для информационной линии точек - тип, установленный в свойстве Точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
