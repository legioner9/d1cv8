HTTPСервисЗапрос.ПараметрыURL (HTTPServiceRequest.URLParameters)

HTTPСервисЗапрос (HTTPServiceRequest)

ПараметрыURL (URLParameters)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: ФиксированноеСоответствие.
Содержит части URL-адреса, которые были параметризованы в шаблоне.

Доступность:

Сервер, толстый клиент.

Примечание:

Например, если шаблон URL-адреса был "/{Doc}/ByNum/{Num}", а URL запрос - "Накладная/ByNum/НО-2345", то соответствие будет содержать две пары:
  • Doc - Накладная,
  • Num - НО-2345.

Использование в версии:

Доступен, начиная с версии 8.3.5.
