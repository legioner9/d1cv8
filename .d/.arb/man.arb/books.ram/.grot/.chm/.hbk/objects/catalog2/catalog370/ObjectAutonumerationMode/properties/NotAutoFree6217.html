РежимАвтонумерацииОбъектов (ObjectAutonumerationMode)

НеОсвобождатьАвтоматически (NotAutoFree)

Доступен, начиная с версии 8.1.

Описание:

Номер вне транзакции не освобождается автоматически. При откате транзакции номера освобождаются автоматически вне зависимости от режима автонумерации объектов.

Использование в версии:

Доступен, начиная с версии 8.1.
