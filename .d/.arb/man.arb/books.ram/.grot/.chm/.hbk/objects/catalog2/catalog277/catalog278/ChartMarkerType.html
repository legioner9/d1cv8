ТипМаркераДиаграммы (ChartMarkerType)

ТипМаркераДиаграммы (ChartMarkerType)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
Квадрат (Rect)
Круг (Circle)
Нет (None)
Ромб (Rhomb)
Чередование (Alternation)

Описание:

Содержит типы маркера, которым отмечаются значения в точках для диаграммы График.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartMarkerType.

См. также:

СерияДиаграммы, свойство Маркер

Использование в версии:

Доступен, начиная с версии 8.0.
