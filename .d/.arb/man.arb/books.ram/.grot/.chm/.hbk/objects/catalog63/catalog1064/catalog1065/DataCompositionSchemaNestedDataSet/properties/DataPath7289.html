ВложенныйНаборДанныхСхемыКомпоновкиДанных (DataCompositionSchemaNestedDataSet)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит путь к данным, под которым вложенный набор данных будет использоваться в схеме и настройках компоновки данных.

Доступность:

Сервер, внешнее соединение.

Примечание:

В системе компоновки данных не допускается использовать в качестве пути к данным имена, совпадающие с ключевыми словами:
ВЫБОР (CASE)
ЕСТЬ (IS)
ИНАЧЕ (ELSE)
КОГДА (WHEN)
КОНЕЦ (END)
НЕ (NOT)
ПОДОБНО (LIKE)
РАЗЛИЧНЫЕ (DISTINCT)
СПЕЦСИМВОЛ (ESCAPE)
ТОГДА (THEN)

Использование в версии:

Доступен, начиная с версии 8.2.
