ЗаписьPDF.ЗаписатьПодпись (PDFWriter.WriteSignature)

ЗаписьPDF (PDFWriter)

ЗаписатьПодпись (WriteSignature)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ЗаписатьПодпись(<МенеджерКриптографии>, <ПредставлениеПодписи>, <ОписаниеПодписи>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: МенеджерКриптографии.
Менеджер криптографии.

<ПредставлениеПодписи> (необязательный)

Тип: Картинка, ТабличныйДокумент.
Объект, представляющий собой оформление (штамп) электронной подписи PDF-документа. Если параметр не указан, оформление не создается (подпись не отображается).

<ОписаниеПодписи> (обязательный)

Тип: ОписаниеПодписиPDF.
Объект, содержащий описание электронной подписи PDF-документа.

Описание:

Создает присоединенную к PDF-документу подпись. Располагает оформление подписи (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

После создания подписи в документе модификация содержимого и вложений не допускается.

См. также:

ЗаписьPDF, метод ЗаписатьПодписьАсинх
ЗаписьPDF, метод НачатьЗаписьПодписи

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
