Расширение группы формы для группы кнопок (Form group extension for a button group)

Доступен, начиная с версии 8.3.8.

Свойства:

Отображение (Representation)

Описание:

Содержит свойства, специфичные для группы вида ГруппаКнопок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
