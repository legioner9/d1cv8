РезультатВыбораДействияРасшифровкиКомпоновкиДанных (DataCompositionDetailsActionChoiceResult)

Доступен, начиная с версии 8.3.18.

Свойства:

ВыбранноеДействие (ChosenAction)
ПараметрВыбранногоДействия (ChosenActionParameter)

Описание:

Содержит результат выбора действия расшифровки компоновки данных. Результат включает две составляющие, находящиеся в свойствах объекта: <ВыбранноеДействие> и <ПараметрВыбранногоДействия>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ОбработкаРасшифровкиКомпоновкиДанных, метод ВыбратьДействие

Использование в версии:

Доступен, начиная с версии 8.3.18.
