Дендрограмма.ПоддержкаМасштаба (Dendrogram.ScaleKeeping)

Дендрограмма (Dendrogram)

ПоддержкаМасштаба (ScaleKeeping)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ПоддержкаМасштабаДендрограммы.
Указывает вариант поддержки масштаба шкалы дендрограммы по шкале элементов.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
