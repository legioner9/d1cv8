КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных (DataCompositionAvailableSettingsObjectCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ДоступныйОбъектНастройкиКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные объекты настройки.
Возможно обращение к доступному объекту настройки посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Коллекция доступных объектов настройки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
