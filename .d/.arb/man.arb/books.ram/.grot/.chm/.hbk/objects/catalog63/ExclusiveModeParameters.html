ПараметрыМонопольногоРежима (ExclusiveModeParameters)

Доступен, начиная с версии 8.3.19.

Свойства:

РазрешитьЗавершатьПриНачалеСеанса (AllowTerminationAtSessionStart)
Сообщение (Message)

Конструкторы:

По умолчанию

Описание:

Параметры монопольного режима базы данных или области данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

Пример:

ПараметрыМонопольного = Новый ПараметрыМонопольногоРежима(
    "Вход в ИБ запрещен. Идёт пересчёт итогов.",
    Истина);
УстановитьМонопольныйРежим(Истина, ПараметрыМонопольного);
