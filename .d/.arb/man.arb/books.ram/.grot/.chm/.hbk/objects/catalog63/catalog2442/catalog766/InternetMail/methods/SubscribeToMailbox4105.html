ИнтернетПочта (InternetMail)

ПодписатьсяНаПочтовыйЯщик (SubscribeToMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

ПодписатьсяНаПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.
Имя IMAP почтового ящика.

Описание:

Подписать почтовый ящик IMAP для дальнейшей работы с ним.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется для POP3 соединения.

Использование в версии:

Доступен, начиная с версии 8.2.
