ПараметрВыбора (ChoiceParameter)

Доступен, начиная с версии 8.2.

Свойства:

Значение (Value)
Имя (Name)

Конструкторы:

По умолчанию

Описание:

Элемент фиксированного массива свойства ПараметрыВыбора.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

НовыйПараметр = Новый ПараметрВыбора("Отбор.Наименование", "МойСправочник");

Использование в версии:

Доступен, начиная с версии 8.2.
