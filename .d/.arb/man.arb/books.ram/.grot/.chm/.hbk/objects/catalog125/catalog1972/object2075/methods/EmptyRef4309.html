ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: ВнешнийИсточникДанныхКубТаблицаИзмеренияСсылка.<Имя внешнего источника>.<Имя куба>.<Имя таблицы измерения>.

Описание:

Получает пустое значение ссылки на таблицу измерения куба внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.3.3.
