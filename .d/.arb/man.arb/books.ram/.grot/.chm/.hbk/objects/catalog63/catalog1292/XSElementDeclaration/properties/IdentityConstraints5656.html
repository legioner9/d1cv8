ОбъявлениеЭлементаXS.ОграниченияИдентичности (XSElementDeclaration.IdentityConstraints)

ОбъявлениеЭлементаXS (XSElementDeclaration)

ОграниченияИдентичности (IdentityConstraints)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: СписокКомпонентXS.
Содержит список компонентов схемы, определяющих ограничения идентичности.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Представляет свойство identity constraint definitions http://www.w3.org/TR/xmlschema-1/#identity-constraint_definitions информационного пространства объявления элемента.

Использование в версии:

Доступен, начиная с версии 8.1.
