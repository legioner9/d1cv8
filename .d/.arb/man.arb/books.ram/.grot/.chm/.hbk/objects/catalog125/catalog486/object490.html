Описание:

Выборка формируется методом Выбрать или ВыбратьПоРегистратору и представляет собой специализированный способ перебора записей регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

РегистрРасчетаМенеджер.<Имя регистра расчета>, метод Выбрать
РегистрРасчетаМенеджер.<Имя регистра расчета>, метод ВыбратьПоРегистратору

Использование в версии:

Доступен, начиная с версии 8.0.
