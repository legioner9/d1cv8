Сеанс.durationCurrentDBMS (ISessionInfo.durationCurrentDBMS)

Сеанс (ISessionInfo)

durationCurrentDBMS (durationCurrentDBMS)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит интервал времени в миллисекундах, прошедший с момента начала выполнения запроса, в случае, если сеанс выполняет запрос к СУБД. Иначе – 0.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
