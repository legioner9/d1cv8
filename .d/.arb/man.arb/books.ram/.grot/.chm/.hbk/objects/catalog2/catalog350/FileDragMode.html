СпособПеретаскиванияФайлов (FileDragMode)

Доступен, начиная с версии 8.3.13.

Значения

КакСсылкаНаФайл (AsFileRef)
КакФайл (AsFile)

Описание:

Содержит режим перетаскивания файлов на элементы управления.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML.

Использование в версии:

Доступен, начиная с версии 8.3.13.
