Файл.НачатьПолучениеРазмера (File.BeginGettingSize)

Файл (File)

НачатьПолучениеРазмера (BeginGettingSize)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеРазмера(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:
  • <Размер> - размер файла, тип Число.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Начинает получение размера файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Файл, метод РазмерАсинх

Использование в версии:
