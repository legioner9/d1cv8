ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПолучитьИсходящуюТочку (BusinessProcessRoutePointRef.<Business process name>.GetOutgoingPoint)

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПолучитьИсходящуюТочку (GetOutgoingPoint)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Для точки вида "ВыборВарианта"

Синтаксис:

ПолучитьИсходящуюТочку(<Вариант>)

Параметры:

<Вариант> (обязательный)

Тип: ВариантТочкиМаршрутаБизнесПроцесса.
Вариант.

Описание варианта метода:

Для точки вида ВыборВарианта возможны несколько точек перехода. Число точек определяется числом вариантов.

Вариант синтаксиса: Для точки вида "Условие"

Синтаксис:

ПолучитьИсходящуюТочку(<Условие>)

Параметры:

<Условие> (обязательный)

Тип: Булево.
Указывает точку маршрута бизнес-процесса. Если значение параметра Истина, то требуется получить исходящую точку выхода по "Да", Ложь - по "Нет".

Описание варианта метода:

Для точки вида Условие возможны две исходящие точки: при значении условия "Да" и "Нет". Параметр уточняет, какую точку требуется получить.

Вариант синтаксиса: Для всех точек, кроме точек вида "Разделение", "Условие", "ВыборВарианта".

Синтаксис:

ПолучитьИсходящуюТочку()

Описание варианта метода:

Для всех точек, кроме точек вида Разделение, Условие и ВыборВарианта существует только одна следующая точка.

Возвращаемое значение:

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.

Описание:

Получает точку маршрута бизнес-процесса, следующую за указанной точкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для всех точек маршрута бизнес-процесса, кроме точек вида Разделение.

Использование в версии:

Доступен, начиная с версии 8.0.
