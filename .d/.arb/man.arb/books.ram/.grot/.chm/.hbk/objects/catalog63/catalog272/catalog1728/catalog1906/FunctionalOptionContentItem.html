ЭлементСоставаФункциональнойОпции (FunctionalOptionContentItem)

Доступен, начиная с версии 8.2.

Свойства:

Объект (Object)

Описание:

Элемент коллекции типа СоставФункциональнойОпции. Каждому элементу соответствует конкретная функциональная опция, назначенная объекту метаданных. Доступ к объекту ОбъектМетаданных: ФункциональнаяОпция осуществляется через свойство Значение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
