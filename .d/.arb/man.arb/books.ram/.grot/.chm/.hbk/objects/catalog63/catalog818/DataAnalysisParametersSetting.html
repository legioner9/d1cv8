НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияПараметрамиАнализаДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки параметров.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Коллекция элементов управления параметрами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля списка параметров анализа данных, свойство НастройкаПараметров

Использование в версии:

Доступен, начиная с версии 8.0.
