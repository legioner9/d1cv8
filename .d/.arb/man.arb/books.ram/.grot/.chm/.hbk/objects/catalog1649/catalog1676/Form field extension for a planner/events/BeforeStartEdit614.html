Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередНачаломРедактирования (BeforeStartEdit)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередНачаломРедактирования(<НовыйЭлемент>, <СтандартнаяОбработка>)

Параметры:

<НовыйЭлемент>

Тип: Булево.
Признак указывает, что создается новый элемент, а не редактируется существующий.

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события - отображение большой формы полного редактирования элемента - производиться не будет.
Элементы, над которыми выполнялось действие, можно получить с помощью свойства ВыделенныеЭлементы поля.
Значение по умолчанию: Истина.

Описание:

Возникает при нажатии кнопки "Изменить" в окне быстрого редактирования, при выборе команды "Изменить" в контексном меню, после события Выбор, если разрешена стандартная обработка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
