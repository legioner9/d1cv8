Планировщик (Planner)

КратностьПериодическогоВарианта (PeriodicVariantRepetition)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Используется вместе со свойством ЕдиницаПериодическогоВарианта для указания длительности шкалы времени до её переноса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Планировщик, свойство ЕдиницаПериодическогоВарианта
