Стиль (Style)

Стиль (Style)

Доступен, начиная с версии 8.0.

Конструкторы:

Конструктор по умолчанию

Описание:

Представляет собой стиль, определенный в конфигурации. Может быть ссылкой на конкретный стиль конфигурации или значением, созданным конструктором стиля без параметров (стиль по умолчанию).
Редактор стиля позволяет модифицировать для обычного приложения все предопределенные элементы стиля, а для приложения только следующие предопределенные цвета стиля:

  • ЦветАктивности,
  • ЦветАкцента,
  • ЦветНавигации,
  • ЦветДополнительнойНавигации,
  • ЦветВажного.
Также можно редактировать значения для собственных элементов стиля вида Цвет. Вне зависимости от возможности редактирования, существующие предопределенные элементы стиля разыменовываются в реальные цвета, используемые платформой. Для оформления элементов формы рекомендуется использовать именно цвета стиля, а не конкретные значения цвета. Другими словами, если для фона какого-либо элемента требуется использовать тот же цвет, что и цвет формы, то свойству, отвечающему за цвет фона элемента, следует установить значение стиля ЦветФонаФормы (значение из системного набора значений ЦветаСтиля), а не конкретное значение цвета. В этом случае, при изменении цвета фона формы, цвет фона выбранного элемента также изменится.
Если необходимо отключить используемый стиль и установить стиль оформления платформы по умолчанию, следует использовать конструкцию вида:
ГлавныйСтиль = Новый Стиль();

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
