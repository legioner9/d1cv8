ПараметрыАнализаДанных (DataAnalysisParameters)

<Имя параметра> (<Parameter name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПараметрАнализаДанных.
Набор свойств содержит параметры, используемые в анализе данных. Доступ к параметру осуществляется по имени. Имена свойств совпадают с именами параметров.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
