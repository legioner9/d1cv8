ПараметрыПеретаскивания (DragParameters)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Содержит перетаскиваемое значение. Тип значения зависит от типа элемента управления, предоставившего значение.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
