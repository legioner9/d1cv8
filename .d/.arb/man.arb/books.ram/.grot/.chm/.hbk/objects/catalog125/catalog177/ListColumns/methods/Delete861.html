КолонкиСписка.Удалить (ListColumns.Delete)

КолонкиСписка (ListColumns)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: Строка.
Имя колонки списка. Зависит от прикладного объекта.

Описание:

Удаляет колонку из списка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Удалим из списка справочника, с которым связано табличное поле,

// колонку с кодом элемента:

Список.Колонки.Удалить("Код");

Использование в версии:

Доступен, начиная с версии 8.0.
