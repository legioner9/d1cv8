ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

ТранзакцияАктивна (TransactionActive)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ТранзакцияАктивна()

Возвращаемое значение:

Тип: Булево.
  • Истина – если внешний источник находится в транзакции,
  • Ложь – в противном случае.

Описание:

Возвращает текущее состояние транзакции внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.8.
