ВедущиеВидыРасчета.<Имя плана видов расчета> (LeadingCalculationTypes.<Chart of calculation types name>)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Значение>, <Колонки>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.
Искомое значение.

<Колонки> (необязательный)

Тип: Строка.
Строка имен колонок, разделенных запятыми, в которых будет осуществляться поиск.
Если параметр не указан, поиск осуществляется по всем колонкам табличной части.
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: ВедущиеВидыРасчетаСтрока.<Имя плана видов расчета>, Неопределено.
Строка табличной части, которая содержит искомое значение. Если значение не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск значения в указанных колонках табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод эффективно использовать для поиска уникальных значений.

Использование в версии:

Доступен, начиная с версии 8.0.
