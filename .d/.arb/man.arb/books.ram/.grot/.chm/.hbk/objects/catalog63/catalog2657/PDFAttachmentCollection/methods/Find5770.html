КоллекцияВложенийPDF (PDFAttachmentCollection)

Найти (Find)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Найти(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: Строка.
Имя файла-вложения.

Возвращаемое значение:

Тип: ВложениеPDF.

Описание:

Осуществляет поиск вложения по имени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
