ЭлементУсловногоОформленияКомпоновкиДанных (DataCompositionConditionalAppearanceItem)

ИспользоватьВЗаголовкеПолей (UseInFieldsHeader)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеУсловногоОформленияКомпоновкиДанных.
Указывает необходимость применения условного оформления к заголовку полей. Значение по умолчанию Использовать.
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
