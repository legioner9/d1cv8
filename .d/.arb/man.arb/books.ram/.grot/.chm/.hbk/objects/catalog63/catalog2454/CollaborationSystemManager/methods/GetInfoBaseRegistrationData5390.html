МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьДанныеРегистрацииИнформационнойБазы (GetInfoBaseRegistrationData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДанныеРегистрацииИнформационнойБазы()

Возвращаемое значение:

Тип: ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия, Неопределено.
Если у пользователя нет права "АдминистрированиеДанных", генерируется исключение.

Описание:

Возвращает данные регистрации информационной базы на сервере системы взаимодействия (для текущей области данных) или Неопределено, если нет данных регистрации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только пользователям с правом "АдминистрированиеДанных".

Использование в версии:

Доступен, начиная с версии 8.3.10.
