SMSСообщение (SMSMessage)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый SMSСообщение()

Описание:

Создает SMS-сообщение на основе записи журнала SMS. Объект создается с незаполненными свойствами.

Использование в версии:

Доступен, начиная с версии 8.3.13.
