Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ШиринаЭлемента (ItemWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Определяет ширину элемента. 0 - определять автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Используется при виде переключателя Тумблер.
Не используется, если итоговое значение свойства ОдинаковаяШиринаКолонок = Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.2.
