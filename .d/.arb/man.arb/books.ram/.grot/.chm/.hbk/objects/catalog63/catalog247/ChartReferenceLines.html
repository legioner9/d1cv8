Доступен, начиная с версии 8.3.12.

Методы:

Вставить (Insert)

Количество (Count)

Описание:

Представляет собой коллекцию информационных линий диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartReferenceLines.

Использование в версии:

Доступен, начиная с версии 8.3.12.
