КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Таблица>)

Параметры:

<Таблица> (обязательный)

Тип: СводнаяТаблица.
Удаляемая таблица.

Описание:

Удаляет встроенную таблицу из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
