Описание:

Элемент графической схемы "Завершение". Отображает точку маршрута бизнес-процесса, в которой завершается бизнес-процесс. В графической схеме может быть несколько точек этого вида.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
