ФильтрОбработкиСканированияДокументов (DocumentScanningProcessingFilter)

ТекстСКартинками (TextWithPictures)

Доступен, начиная с версии 8.3.23.

Описание:

Фильтр для обработки документа, содержащего текст с картинками.
Данный фильтр улучает качество отсканированных изображений.
Если предполагается, что документ будет обработан вручную, этот фильтр можно установить в качестве фильтра по умолчанию. Он улучшит вид документа, сохранит качество фотографий и других картинок, и сэкономит тонер при печати.
Рекомендуется добавлять Нет в качестве дополнительного варианта, если качество картинок сохранить при сканировании не удалось.

Использование в версии:

Доступен, начиная с версии 8.3.23.
