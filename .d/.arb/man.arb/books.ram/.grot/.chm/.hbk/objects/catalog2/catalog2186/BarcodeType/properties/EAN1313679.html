ТипШтрихКода.EAN13 (BarcodeType.EAN13)

ТипШтрихКода (BarcodeType)

EAN13 (EAN13)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа European Article Number (полный). Кодируется 13 цифр (12 значащих + 1 контрольная сумма).

Использование в версии:

Доступен, начиная с версии 8.3.25.
