ФорматированныйДокумент (FormattedDocument)

Записать (Write)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла, в который будет записан форматированный документ.

<ТипФайла> (необязательный)

Тип: ТипФайлаФорматированногоДокумента.
Указывает тип файла.

Описание варианта метода:

Запись форматированного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для записи документа.

<ТипФайла> (необязательный)

Тип: ТипФайлаФорматированногоДокумента.
Указывает формат записи документа.

Описание варианта метода:

Запись форматированного документа в поток.

Описание:

Записывает форматированный документ в файл.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ФорматированныйДокумент, метод ЗаписатьАсинх
ФорматированныйДокумент, метод НачатьЗапись

Использование в версии:

Доступен, начиная с версии 8.2.
