Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Чтение и запись.

Описание:

Тип: Булево.
Истина - отображает элемент графической схемы как гиперссылку. При выборе элемента с установленным свойством будет порождаться событие Выбор.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
