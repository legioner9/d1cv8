ФайловыйПоток.КопироватьВ (FileStream.CopyTo)

ФайловыйПоток (FileStream)

КопироватьВ (CopyTo)

Доступен, начиная с версии 8.3.9.

Синтаксис:

КопироватьВ(<ЦелевойПоток>, <Количество>)

Параметры:

<ЦелевойПоток> (обязательный)

Тип: Поток.
Поток, в который будет выполняться копирование.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется скопировать в целевой поток. Если не задано, то будут скопированы все данные до конца текущего потока.

Возвращаемое значение:

Фактическое количество скопированных байтов.

Описание:

Копирует данные из текущего потока в другой поток.
Дополнительно можно установить количество байтов, которые будут скопированы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

См. также:

Поток, метод КопироватьВАсинх
Поток, метод НачатьКопированиеВ
