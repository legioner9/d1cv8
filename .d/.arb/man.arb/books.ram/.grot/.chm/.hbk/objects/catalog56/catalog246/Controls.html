ЭлементыФормы (Controls)

Доступен, начиная с версии 8.0.

Свойства:

<Имя элемента управления> (<Control name>)

Методы:

Добавить (Add)
ДобавитьActiveX (AddActiveX)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Используется для доступа к элементам управления, расположенным на форме. Типы элементов управления описаны в Элементы управления.

Доступность:

Толстый клиент.

Пример:

Сообщить(ЭлементыФормы.ТаблПоле.Колонки.Количество());

См. также:

Форма, свойство ЭлементыФормы

Использование в версии:

Доступен, начиная с версии 8.0.
