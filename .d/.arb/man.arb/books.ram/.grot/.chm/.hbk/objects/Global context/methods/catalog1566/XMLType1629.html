Глобальный контекст.XMLТип (Global context.XMLType)

Глобальный контекст (Global context)

XMLТип (XMLType)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLТип(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.
Тип значения, для которого необходимо определить соответствующий XML тип.

Возвращаемое значение:

Тип: ТипДанныхXML.
Неопределено, если никакой тип данных XML не соответствует указанному типу.

Описание:

Получает тип данных XML, соответствующий переданному в качестве параметра типу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
