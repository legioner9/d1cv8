БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ОбработкаИнтерактивнойАктивации (InteractiveActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаИнтерактивнойАктивации(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.
Значение по умолчанию: Истина.

Описание:

Возникает при выполнении метода АктивироватьИнтерактивно. Если стандартная обработка не изменена, то после этого события будет вызван обработчик события ОбработкаИнтерактивнойАктивации для всех невыполненных задач по данному бизнес-процессу для текущего исполнителя. Текущий пользователь определяется параметром сеанса, который установлен в соответствующем свойстве задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме запуска УправляемоеПриложение не вызывается.

См. также:

БизнесПроцессОбъект.<Имя бизнес-процесса>, метод АктивироватьИнтерактивно

Использование в версии:

Доступен, начиная с версии 8.0.
