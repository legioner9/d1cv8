ДанныеКонтакта (ContactData)

АдресаМгновенныхСообщений (InstantMessagingAddresses)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: Массив.
Содержит массив адресов мгновенных сообщений в виде объектов типа ЭлементДанныхКонтактаМгновенныеСообщения.
Свойство ТипДанных содержит значение перечисления ТипАдресаМгновенныхСообщенийДанныхКонтакта или строку. Свойство ТипМгновенныхСообщений содержит строку (например, Skype, ICQ и т.д.). Свойство Идентификатор содержит строку – идентификатор пользователя в системе мгновенных сообщений.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
