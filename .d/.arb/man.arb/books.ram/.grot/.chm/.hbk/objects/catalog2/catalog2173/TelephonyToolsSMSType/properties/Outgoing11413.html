Исходящее (Outgoing)

Доступен, начиная с версии 8.3.11.

Описание:

Исходящее сообщение. Данный статус является временным. При отправке статус сообщения изменится на один из статусов: ОшибкаОтправки, ВОчереди, Отправленное.

Использование в версии:

Доступен, начиная с версии 8.3.11.
