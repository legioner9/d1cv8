Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ВертикальныеЛинии (VerticalLines)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Управляет отображением горизонтальных линий в области построения диаграммы.
Если содержит Ложь, то все горизонтальные линии в области построения диаграммы не отображаются. Если содержит Истина, то все горизонтальные линии в области построения диаграммы отображаются.
Если содержит Неопределено, то все горизонтальные линии в области построения диаграммы отображаются.
Значение по умолчанию - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
