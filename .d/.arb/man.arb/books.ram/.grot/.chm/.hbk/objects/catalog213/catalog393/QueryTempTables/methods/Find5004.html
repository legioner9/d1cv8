ВременныеТаблицыЗапроса.Найти (QueryTempTables.Find)

ВременныеТаблицыЗапроса (QueryTempTables)

Найти (Find)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Найти(<НаименованиеТаблицы>)

Параметры:

<НаименованиеТаблицы> (обязательный)

Тип: Строка.
Имя таблицы для поиска.

Возвращаемое значение:

Тип: ВременнаяТаблицаЗапроса, Неопределено.
Если временная таблица с таким именем отсутствует, возвращает Неопределено.

Описание:

Возвращает временную таблицу по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Если РезультатЗапроса.Колонки.Найти("Цена") <> Неопределено Тогда
    ЦеныВЗапросе = Истина;
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.3.8.
