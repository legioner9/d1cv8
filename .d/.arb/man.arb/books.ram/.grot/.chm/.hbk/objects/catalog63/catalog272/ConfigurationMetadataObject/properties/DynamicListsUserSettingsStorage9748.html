ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ХранилищеПользовательскихНастроекДинамическихСписков (DynamicListsUserSettingsStorage)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: СтандартноеХранилищеНастроекМенеджер, ХранилищеНастроекМенеджер.<Имя хранилища>.
Устанавливает хранилище настроек, присутствующее в конфигурации. Если хранилище не выбрано, то будет использоваться стандартное хранилище настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
