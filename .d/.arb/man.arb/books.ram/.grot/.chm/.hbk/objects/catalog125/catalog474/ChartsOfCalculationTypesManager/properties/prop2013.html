<Имя плана видов расчета> (<Chart of calculation types name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланВидовРасчетаМенеджер.<Имя плана видов расчета>.
Набор свойств содержит менеджеры отдельных планов видов расчета. Доступ к менеджеру осуществляется по имени, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
