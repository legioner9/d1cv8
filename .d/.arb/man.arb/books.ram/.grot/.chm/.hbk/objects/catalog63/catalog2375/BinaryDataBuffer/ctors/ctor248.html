БуферДвоичныхДанных (BinaryDataBuffer)

По умолчанию

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый БуферДвоичныхДанных(<Размер>, <Порядок байтов>)

Параметры:

<Размер> (обязательный)

Тип: Число.
Размер буфера в байтах.

<Порядок байтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов.
Значение по умолчанию: LittleEndian.

Использование в версии:

Доступен, начиная с версии 8.3.9.
