ОбластьЯчеекТабличногоДокумента.Значение (SpreadsheetDocumentRange.Value)

ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Содержит значение ячеек области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство доступно, только если ячейка содержит значение (свойство СодержитЗначение равно Истина). Свойство применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.0.
