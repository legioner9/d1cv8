ПредыдущаяЧасть (PreviousPart)

Доступен, начиная с версии 8.1.

Синтаксис:

ПредыдущаяЧасть(<ТекущаяНачальнаяПозиция>)

Параметры:

<ТекущаяНачальнаяПозиция> (необязательный)

Тип: Число.
Если указан, то производится установка текущей позиции в указанное в параметре значение, а затем выполняется метод.

Описание:

Получает предыдущую порцию результатов поиска.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если строка поиска содержит ошибки, или если по каким-либо причинам, поиск не может вернуть результаты поиска, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
