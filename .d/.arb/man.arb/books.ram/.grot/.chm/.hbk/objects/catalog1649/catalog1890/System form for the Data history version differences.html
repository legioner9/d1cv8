Системная форма различий версий истории данных (System form for the Data history version differences)

Доступен, начиная с версии 8.3.13.

Параметры формы:

Данные (Data)
НомерВерсииДоИзменения (VersionNumberBeforeChange)
НомерВерсииПослеИзменения (VersionNumberAfterChange)

Описание:

Форма различий версий истории данных, отображает отчет о различиях версий.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.21.
