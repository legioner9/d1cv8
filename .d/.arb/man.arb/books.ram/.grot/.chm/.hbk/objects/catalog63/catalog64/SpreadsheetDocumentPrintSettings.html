НастройкиПечатиТабличногоДокумента (SpreadsheetDocumentPrintSettings)

Доступен, начиная с версии 8.2.

Описание:

Содержит настройки печати табличного документа.
Платформа самостоятельно создает этот объект при сохранении настроек печати и помещает его в хранилище системных настроек.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
