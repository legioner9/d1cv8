Описание:

Предоставляет доступ к функциям системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

Глобальный контекст, свойство СистемаВзаимодействия

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.25.
