Диаграмма.Транспонировать (Chart.Transpose)

Диаграмма (Chart)

Транспонировать (Transpose)

Доступен, начиная с версии 8.0.

Синтаксис:

Транспонировать()

Описание:

Меняет местами серии и точки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Недоступен в режиме источника данных.
Следует учитывать, что для диаграмм с использованием маркеров после транспонирования будут использованы маркеры по умолчанию, т.к. у ТочкаДиаграммы свойство Маркер отсутствует.

Использование в версии:

Доступен, начиная с версии 8.0.
