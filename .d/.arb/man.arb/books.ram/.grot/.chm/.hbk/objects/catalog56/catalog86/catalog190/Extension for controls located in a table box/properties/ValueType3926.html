Расширение элементов управления, расположенных в табличном поле (Extension for controls located in a table box)

ТипЗначения (ValueType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ОписаниеТипов.
Содержит тип значения, редактируемого в элементе управления.

Доступность:

Толстый клиент.

Примечание:

Запись в свойство возможна только для элементов управления, вставленных в колонки, не связанные с данными.

Использование в версии:

Доступен, начиная с версии 8.0.
