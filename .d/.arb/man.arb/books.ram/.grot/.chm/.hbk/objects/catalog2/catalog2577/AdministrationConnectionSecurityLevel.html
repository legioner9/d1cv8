АдминистрированиеУровеньБезопасностиСоединений (AdministrationConnectionSecurityLevel)

Доступен, начиная с версии 8.3.14.

Значения

Защищенное (Secure)
ЗащищенноеПриУстановкеСоединения (SecureOnConnect)
Незащищенное (Unsecure)

Описание:

Определяет уровень безопасности соединений.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
