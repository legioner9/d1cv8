ОбъектМетаданных: РегламентноеЗадание (MetadataObject: ScheduledJob)

ИнтервалПовтораПриАварийномЗавершении (RestartIntervalOnFailure)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.
Интервал времени ( в секундах), по истечении которого регламентное задание будет запущено повторно, если предыдущий его запуск завершился аварийно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
