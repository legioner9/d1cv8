УсловноеОформление.Добавить (ConditionalAppearance.Add)

УсловноеОформление (ConditionalAppearance)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Представление>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя области оформления.

<Представление> (необязательный)

Тип: Строка.
Представление добавляемой области.

Возвращаемое значение:

Тип: ЭлементУсловногоОформления.

Описание:

Добавить область оформления в конец коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
