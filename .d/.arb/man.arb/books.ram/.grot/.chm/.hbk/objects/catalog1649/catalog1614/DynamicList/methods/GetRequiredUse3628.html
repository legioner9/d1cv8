ДинамическийСписок (DynamicList)

ПолучитьОбязательноеИспользование (GetRequiredUse)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбязательноеИспользование(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: Строка.
Имя поля, для которого необходимо получить признак необходимости выборки.

Возвращаемое значение:

Тип: Булево.
Истина - поле обязательно для выборки.
Ложь - поле может не выбираться.

Описание:

Получает признак обязательности выборки значения поля для клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
