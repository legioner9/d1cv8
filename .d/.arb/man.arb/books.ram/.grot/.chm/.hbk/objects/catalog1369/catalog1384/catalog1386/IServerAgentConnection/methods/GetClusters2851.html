GetClusters (GetClusters)

Доступен, начиная с версии 8.1.

Синтаксис:

GetClusters()

Возвращаемое значение:

Тип: COMSafeArray.
Массив объектов Кластер серверов.

Позволяет получить список кластеров, зарегистрированных на центральном сервере.

Доступность:

Интеграция.

Доступен, начиная с версии 8.1.
