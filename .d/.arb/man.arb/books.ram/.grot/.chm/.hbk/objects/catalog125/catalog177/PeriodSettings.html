НастройкаПериода (PeriodSettings)

Доступен, начиная с версии 8.0.

Свойства:

ВариантНастройки (SettingsMode)
ВариантНачала (BeginningVariant)
ВариантОкончания (EndingVariant)
ВариантПериода (PeriodVariant)

ДатаОкончания (EndDate)
ЗначениеПериода (PeriodValue)
РедактироватьКакИнтервал (EditAsInterval)
РедактироватьКакПериод (EditAsPeriod)
СмещениеНачала (DaysBefore)
СмещениеОкончания (DaysAfter)

Методы:

ПолучитьДатуНачала (GetDateFrom)
ПолучитьДатуОкончания (GetDateTo)
Редактировать (Edit)
УстановитьПериод (SetPeriod)

Конструкторы:

По умолчанию

Описание:

Предназначен для ввода периодов. При помощи объекта можно настраивать интервалы просмотра в различных хронологических списках.
Также используется для установки интервала просмотра средствами встроенного языка.

Доступность:

Толстый клиент.
Сериализуется.

Пример:

НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод;
НастройкаПериода.ВариантНачала = ВариантГраницыПериода.Месяц;
НастройкаПериода.ВариантОкончания = ВариантГраницыПериода.БезОграничения;
НастройкаПериода.РедактироватьКакИнтервал = Истина;
НастройкаПериода.РедактироватьКакПериод = Ложь;
