МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

Добавить (Add)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Добавить(<ИмяПространства>, <ПутьДоПространства>)

Параметры:

<ИмяПространства> (обязательный)

Тип: Строка.
Имя пространства. Должно быть уникальным, может содержать только буквы латинского алфавита и цифры. Допускается длина не более 16 символов.

<ПутьДоПространства> (обязательный)

Тип: Строка.
Путь к каталогу табличного пространства.

Возвращаемое значение:

Тип: МенеджерТабличногоПространстваБазыДанных.

Описание:

Добавляет табличное пространство.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
