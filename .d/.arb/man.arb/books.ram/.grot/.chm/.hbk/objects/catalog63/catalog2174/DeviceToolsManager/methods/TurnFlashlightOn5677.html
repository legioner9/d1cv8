МенеджерСредствУстройства.ВключитьФонарик (DeviceToolsManager.TurnFlashlightOn)

МенеджерСредствУстройства (DeviceToolsManager)

ВключитьФонарик (TurnFlashlightOn)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ВключитьФонарик(<Включить>)

Параметры:

<Включить> (необязательный)

Тип: Булево.
Определяет, будет ли включен фонарик:
  • Истина - включить фонарик;
  • Ложь - выключить фонарик.

Описание:

Управляет состоянием фонарика.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом метода следует использовать метод ПоддерживаетсяФонарик.

Доступность:

Толстый клиент.

См. также:

МенеджерСредствУстройства, метод ПоддерживаетсяФонарик

Использование в версии:

Доступен, начиная с версии 8.3.13.
