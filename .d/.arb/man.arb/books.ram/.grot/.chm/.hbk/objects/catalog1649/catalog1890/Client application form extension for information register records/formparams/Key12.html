Расширение формы клиентского приложения для записи регистра сведений (Client application form extension for information register records)

Ключ (Key)

Доступен, начиная с версии 8.2.

Описание:

Тип: РегистрСведенийКлючЗаписи.<Имя регистра сведений>.
Тип значения - набор ключевых полей, однозначно идентифицирующих запись, редактируемых в форме.

Примечание:

В случае отсутствия параметра, а также когда параметр содержит пустое значение, создается новая запись.

Использование в версии:

Доступен, начиная с версии 8.2.
