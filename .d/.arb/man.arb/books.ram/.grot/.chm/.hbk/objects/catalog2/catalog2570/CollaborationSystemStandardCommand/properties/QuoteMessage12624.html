ЦитироватьСообщение (QuoteMessage)

Описание:

Команда цитирования сообщения.
Добавляется в список команд по умолчанию, передаваемых в обработчик формирования команд, если источником команд является Сообщение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
