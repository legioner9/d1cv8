СерииСлояГеографическойСхемы.Найти (GeographicalSchemaLayerSeries.Find)

СерииСлояГеографическойСхемы (GeographicalSchemaLayerSeries)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяСерии>)

Параметры:

<ИмяСерии> (обязательный)

Тип: Строка.
Имя искомой серии.

Возвращаемое значение:

Тип: СерияДанныхСлояГеографическойСхемы, Неопределено.
Если не найдена, возвращается Неопределено.

Описание:

Осуществляет поиск серии в коллекции по переданному имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
