СсылкаНаФайл.ПолучитьКакСтрокуАсинх (FileRef.GetAsStringAsync)

СсылкаНаФайл (FileRef)

ПолучитьКакСтрокуАсинх (GetAsStringAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьКакСтрокуАсинх(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: КодировкаТекста, Строка.
Кодировка текста. Если кодировка не установлена, то используется UTF8.

Возвращаемое значение:

Тип: Обещание.
Результат обещания - Строка.

Описание:

Асинхронная функция. Возвращает содержимое файла в виде строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.24.
