ИспользованиеАгрегатаРегистраНакопления (AccumulationRegisterAggregateUse)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
Всегда (Always)

Описание:

Определяет варианты использования агрегата регистра накопления при выполнении запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.
