ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ЗапросВыбораСхемыЗапроса, ЗапросУничтоженияТаблицыСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются запрос выбора или запрос уничтожения таблицы.
Возможно обращение к запросу выбора или запросу уничтожения таблицы посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
ДобавитьКопию (AddCopy)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция запросов пакета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
