Планировщик.ПоведениеЭлементовПриНедостаткеМеста (Planner.ItemsBehaviorWhenSpaceInsufficient)

Планировщик (Planner)

ПоведениеЭлементовПриНедостаткеМеста (ItemsBehaviorWhenSpaceInsufficient)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: ПоведениеЭлементовПланировщикаПриНедостаткеМеста.
Определяет поведение элементов планировщика при недостатке места в поле планировщика.
Свойство используется при положении шкалы времени Верх или Низ.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение по умолчанию - СворачиватьЭлементы.

Использование в версии:

Доступен, начиная с версии 8.3.15.
