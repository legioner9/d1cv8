ПоложениеСтрокиПоиска (SearchStringLocation)

Доступен, начиная с версии 8.3.5.

Значения

Авто (Auto)
Верх (Top)
ЗаголовокФормы (FormCaption)
КоманднаяПанель (CommandBar)
Низ (Bottom)
ПотянутьСверху (PullFromTop)

Описание:

Определяет положение автоматически размещенной строки поиска таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
