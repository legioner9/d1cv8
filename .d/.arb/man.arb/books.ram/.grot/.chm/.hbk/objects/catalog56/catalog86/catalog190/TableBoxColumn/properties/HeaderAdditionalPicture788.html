КолонкаТабличногоПоля (TableBoxColumn)

ДополнительнаяКартинкаШапки (HeaderAdditionalPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Картинка.
Содержит дополнительную пиктограмму, расположенную в шапке колонки. Картинка располагается с правой стороны колонки.

Доступность:

Толстый клиент.

Примечание:

Рекомендуемый размер картинки 14х14 пикселей.
Если картинка задается программно, может быть установлена картина вида Пустая или ИзБиблиотеки.

Использование в версии:

Доступен, начиная с версии 8.0.
