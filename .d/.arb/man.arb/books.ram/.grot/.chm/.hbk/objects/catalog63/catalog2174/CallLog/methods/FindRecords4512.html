ЖурналЗвонков.НайтиЗаписи (CallLog.FindRecords)

ЖурналЗвонков (CallLog)

НайтиЗаписи (FindRecords)

Доступен, начиная с версии 8.3.5.

Синтаксис:

НайтиЗаписи(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: ОтборКомпоновкиДанных.
Задает условия поиска.
В качестве условий отбора могут быть указаны следующие поля:
  • НомерТелефона - тип Строка. Допускаются виды сравнения: Равно, НеРавно, Содержит, НеСодержит.
  • Дата - тип Дата. Допускаются виды сравнения: Больше, Меньше, Равно, БольшеИлиРавно, МеньшеИлиРавно, НеРавно.
  • ТипЗвонка - тип ТипЗвонкаЖурналаЗвонков. Допускаются виды сравнения: Равно, НеРавно.
  • Продолжительность - тип Число. Допускаются виды сравнения: Больше, Меньше, Равно, БольшеИлиРавно, МеньшеИлиРавно, НеРавно.
Элементы отбора могут быть сгруппированы по "И" или по "ИЛИ". Группировка по "НЕ" в отборе не поддерживается.

Возвращаемое значение:

Тип: Массив.
Элементом возвращаемого массива является ЗаписьЖурналаЗвонков.

Описание:

Осуществляет поиск записей в журнале звонков.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
