ЗаписьPDF.НачатьЗаписьУдаленияВложений (PDFWriter.BeginWriteAttachmentsDeletion)

ЗаписьPDF (PDFWriter)

НачатьЗаписьУдаленияВложений (BeginWriteAttachmentsDeletion)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗаписьУдаленияВложений(<ОписаниеОповещения>, <Вложения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Вложения> (обязательный)

Тип: Массив.
Массив, содержащий имена файлов.

Описание:

Записывает в PDF-документ информацию об удалении вложения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

ЗаписьPDF, метод ЗаписатьУдалениеВложений
ЗаписьPDF, метод ЗаписатьУдалениеВложенийАсинх
