РезультатГлобальногоПоиска (GlobalSearchResult)

Доступен, начиная с версии 8.3.15.

Элементы коллекции:

ЭлементРезультатаГлобальногоПоиска
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.
Возможно обращение к элементам посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Представляет собой коллекцию результатов глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

См. также:

ЭлементПланаГлобальногоПоиска, свойство ИмяПроцедуры

Использование в версии:

Доступен, начиная с версии 8.3.15.
