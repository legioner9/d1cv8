ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Вставить (Insert)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Вставить(<Индекс>, <Имя>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Позиция, в которую необходимо вставить элемент коллекции.

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого поля.

<Тип> (необязательный)

Тип: Тип.
Тип вставляемого элемента.
Значение по умолчанию: Неопределено.

Описание:

Вставляет новый элемент коллекции на указанную позицию. Метод используется только для доступных полей описания временной таблицы (ДоступныеПоля).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
