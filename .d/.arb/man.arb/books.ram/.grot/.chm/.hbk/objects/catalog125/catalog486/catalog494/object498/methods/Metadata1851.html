ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: Перерасчет.

Описание:

Предоставляет доступ к объекту описания метаданных перерасчета. Другой путь получения того же значения - через свойство глобального контекста Метаданные. Например: Метаданные.РегистрыРасчета.ДополнительныеНачисленияСотрудниковОрганизации.Перерасчеты.ПерерасчетДополнительныхНачислений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
