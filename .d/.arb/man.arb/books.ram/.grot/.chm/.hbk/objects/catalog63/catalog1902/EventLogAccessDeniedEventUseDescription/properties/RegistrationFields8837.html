ПоляРегистрации (RegistrationFields)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Массив.
Описывает поля, которые в случае участия их в результате выборки запроса попадут в таблицу значений в поле Данные.Данные.
Поля регистрации задаются массивом, элементом которого является либо строка - имя поля, либо массив строк - имен полей.
Первый уровень массива определяет набор ключевых полей, все из которых будут зарегистрированы.
Если элементом массива является массив строк, то в этом массиве описываются альтернативные ключевые поля.
В случае возможности вывода нескольких альтернативных ключевых полей выводится поле с меньшим индексом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
