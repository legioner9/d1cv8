ОбластьЛегендыГеографическойСхемы (GeographicalSchemaLegendArea)

Доступен, начиная с версии 8.1.

Свойства:

МасштабнаяЛинейка (DrawingScale)
Прозрачный (Transparent)
ПрозрачныйФон (Transparent)
Рамка (Border)
Цвет (Color)
ЦветРамки (BorderColor)
ЦветФона (BackColor)
Шрифт (Font)
Элементы (Items)

Описание:

Используется для доступа к свойствам области легенды географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Схема.ОбластьЛегенды.Показывать = Истина;
Схема.ОбластьЛегенды.Прозрачность = Истина;
Схема.ОбластьЛегенды.МасштабнаяЛинейка = Ложь;

См. также:

ГеографическаяСхема, свойство ОбластьЛегенды

Использование в версии:

Доступен, начиная с версии 8.1.
