Форма.ОбработкаЗаписиНовогоОбъекта (Form.NewObjectWriteProcessing)

Форма (Form)

ОбработкаЗаписиНовогоОбъекта (NewObjectWriteProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаЗаписиНовогоОбъекта(<Объект>, <Источник>)

Параметры:

<Объект>

Тип: Ссылка на объект, либо ТекущаяСтрока, либо объект.
Если подчиненная форма (та, в которой запись выполнена) управляемая, то содержит ссылку на добавленный в подчиненной форме объект.
Если подчиненная форма обычная, то содержит сам объект.

<Источник>

Тип: Форма.
Форма - источник события.

Описание:

Возникает при записи объекта в одной из подчиненных форм или при вызове в одной из подчиненных форм метода ОповеститьОЗаписиНовогоОбъекта.

Доступность:

Толстый клиент.

См. также:

Форма, метод ОповеститьОЗаписиНовогоОбъекта

Использование в версии:

Доступен, начиная с версии 8.0.
