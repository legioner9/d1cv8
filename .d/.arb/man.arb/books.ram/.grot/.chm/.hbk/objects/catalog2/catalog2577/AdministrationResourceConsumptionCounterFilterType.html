АдминистрированиеТипОтбораСчетчикаПотребленияРесурсов (AdministrationResourceConsumptionCounterFilterType)

Доступен, начиная с версии 8.3.14.

Значения

Все (All)
ВсеВыбранные (AllSelected)
ВсеКромеВыбранных (AllButSelected)

Описание:

Определяет тип отбора счетчика потребления ресурсов.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
