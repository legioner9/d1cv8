ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

WebСервисы (WebServices)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: КоллекцияОбъектовМетаданных.
Коллекция объектов метаданных, которые описывают все WebСервисы, определенные в конфигурации.
Элементами коллекции являются объекты типа ОбъектМетаданных: WebСервис.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
