ОформлениеЗначений (ValueAppearance)

Получить (Get)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Получить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.
Ключ оформления.

Возвращаемое значение:

Тип: Произвольный, Неопределено.
Если запись с таким ключом не найдена, возвращает Неопределено.

Описание:

Выполняет поиск записи по ключу.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
