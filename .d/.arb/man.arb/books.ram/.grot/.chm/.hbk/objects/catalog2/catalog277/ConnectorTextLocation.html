ПоложениеТекстаСоединительнойЛинии (ConnectorTextLocation)

Доступен, начиная с версии 8.0.

Содержит варианты позиций текста относительно соединительной линии графической схемы.

Доступность:

Тонкий клиент, сервер, толстый клиент.
