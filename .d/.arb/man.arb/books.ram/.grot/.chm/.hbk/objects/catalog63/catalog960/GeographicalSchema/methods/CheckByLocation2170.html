Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьПоРасположению(<Объект>, <ГдеПроверять>, <КакПроверять>)

Параметры:

<Объект> (обязательный)

Тип: ТочечныйОбъектГеографическойСхемы, МноготочечныйОбъектГеографическойСхемы, ПолилинейныйОбъектГеографическойСхемы, ПолигональныйОбъектГеографическойСхемы.
Объект поиска.

<ГдеПроверять> (обязательный)

Тип: Массив.
Массив может содержать объекты следующих типов: ТочечныйОбъектГеографическойСхемы, МноготочечныйОбъектГеографическойСхемы, ПолилинейныйОбъектГеографическойСхемы, ПолигональныйОбъектГеографическойСхемы, ГеографическиеКоординаты,
ПрямоугольникГеографическойСхемы.

<КакПроверять> (обязательный)

Тип: ТипПоискаОбъектовГеографическойСхемы.
Определяет способ поиска объекта географической схемы.

Возвращаемое значение:

Тип: Булево.
Истина в противном случае.

Описание:

Проверяет, попадают ли указанные объекты в местоположение других.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
