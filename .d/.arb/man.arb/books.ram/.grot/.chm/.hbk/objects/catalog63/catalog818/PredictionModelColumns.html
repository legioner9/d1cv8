КолонкиМоделиПрогноза (PredictionModelColumns)

Доступен, начиная с версии 8.0.

Элементы коллекции:

КолонкаМоделиПрогноза
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.
Возможно обращение к колонкам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Содержит колонки модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Сериализуется.

См. также:

МодельПрогнозаПоискАссоциаций, свойство Колонки
МодельПрогнозаПоискПоследовательностей, свойство Колонки
МодельПрогнозаДеревоРешений, свойство Колонки

Использование в версии:

Доступен, начиная с версии 8.0.
