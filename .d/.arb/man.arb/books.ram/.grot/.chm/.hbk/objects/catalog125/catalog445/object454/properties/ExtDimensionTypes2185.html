ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ВидыСубконто (ExtDimensionTypes)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовВидыСубконто.<Имя плана счетов>.
Является специализированной табличной частью, в которой хранятся настройки видов субконто на счете.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
