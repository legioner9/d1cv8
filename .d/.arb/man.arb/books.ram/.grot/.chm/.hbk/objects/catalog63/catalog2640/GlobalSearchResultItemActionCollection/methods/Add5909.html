КоллекцияДействийЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemActionCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Добавить(<Значение>, <Текст>, <Картинка>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.
Значение действия элемента результата глобального поиска.
Значение по умолчанию: Неопределено.

<Текст> (необязательный)

Тип: Строка, ФорматированнаяСтрока.
Текст, отображаемый в действии элемента результата глобального поиска.
Значение по умолчанию: Пустая строка.

<Картинка> (необязательный)

Тип: Картинка.
Картинка, отображаемая в действии элемента результата глобального поиска.
Значение по умолчанию: Пустая картинка.

Возвращаемое значение:

Тип: ДействиеЭлементаРезультатаГлобальногоПоиска.

Описание:

Добавляет новое действие в конец коллекции действий элемента результата глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
