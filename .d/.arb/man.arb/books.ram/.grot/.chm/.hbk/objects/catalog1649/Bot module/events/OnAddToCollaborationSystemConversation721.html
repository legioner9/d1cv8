Синтаксис:

ПриДобавленииВОбсуждениеСистемыВзаимодействия(<ИдентификаторОбсуждения>, <ДополнительныеПараметры>)

Параметры:

<ИдентификаторОбсуждения>

Тип: ИдентификаторОбсужденияСистемыВзаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.3.23.
