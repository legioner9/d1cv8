ТипКнопкиКоманднойПанели.Разделитель (CommandBarButtonType.Separator)

ТипКнопкиКоманднойПанели (CommandBarButtonType)

Разделитель (Separator)

Доступен, начиная с версии 8.0.

Описание:

Визуально кнопка данного типа представляет собой горизонтальную или вертикальную (в зависимости от расположения командной панели) черту. Кнопки такого типа не имеют визуальных свойств (картинка, текст и т.д.). Используется для разделения групп кнопок в командной панели и панели инструментов.

Использование в версии:

Доступен, начиная с версии 8.0.
