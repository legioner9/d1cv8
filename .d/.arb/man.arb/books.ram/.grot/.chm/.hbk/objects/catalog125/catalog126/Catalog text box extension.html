Расширение поля ввода справочника (Catalog text box extension)

Доступен, начиная с версии 8.0.

Свойства:

БыстрыйВыбор (QuickChoice)
ВыборГруппИЭлементов (ChoiceFoldersAndItems)
ВыборПоВладельцу (ChoiceByOwner)

Описание:

Дополнительные свойства элемента управления ПолеВвода, предназначенного для ввода ссылки на элемент справочника.
Расширение предоставляет возможность автоматического поиска и подбора элемента справочника по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Пример:

Клиент = ЭлементыФормы.ПолеВводаКонтрагента;
Клиент.БыстрыйВыбор = Истина;
Клиент.ВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Группы;

Использование в версии:

Доступен, начиная с версии 8.0.
