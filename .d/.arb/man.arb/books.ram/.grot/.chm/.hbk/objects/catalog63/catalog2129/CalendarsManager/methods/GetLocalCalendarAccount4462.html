МенеджерКалендарей (CalendarsManager)

ПолучитьЛокальнуюУчетнуюЗаписьКалендарей (GetLocalCalendarAccount)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьЛокальнуюУчетнуюЗаписьКалендарей()

Возвращаемое значение:

Тип: УчетнаяЗаписьКалендарей.

Описание:

Получает локальную учетную запись календарей.

  • Для устройств на базе Android:
    • возвращает фиктивную учётную запись (с именем и типом "LOCAL"),
    • список учетных записей, возвращаемых методом ПолучитьУчетныеЗаписиКалендарей, всегда содержит фиктивную учетную запись.
  • Для устройств на базе iOS - использует штатный метод операционной системы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
