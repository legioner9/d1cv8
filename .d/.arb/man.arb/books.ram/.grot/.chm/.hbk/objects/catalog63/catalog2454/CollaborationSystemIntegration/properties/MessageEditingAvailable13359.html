ДоступноРедактированиеСообщения (MessageEditingAvailable)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Булево.
Определяет возможность изменения отправленных сообщений в обсуждениях данной интеграции:
  • Истина - изменение возможно,
  • Ложь - в противном случае (для интеграции типа WhatsApp).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
