Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

УстановитьГраницыВыделения (SetTextSelectionBounds)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьГраницыВыделения(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: ЗакладкаФорматированногоДокумента.
Новая позиция начала выделения.

<Конец> (обязательный)

Тип: ЗакладкаФорматированногоДокумента.
Новая позиция конца выделения.

Описание:

Устанавливает границы выделения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Метод можно использовать при открытом окне, в том числе в серверных вызовах.

Использование в версии:

Доступен, начиная с версии 8.2.
