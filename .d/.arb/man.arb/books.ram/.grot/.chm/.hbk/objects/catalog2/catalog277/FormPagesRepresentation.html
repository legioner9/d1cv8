ОтображениеСтраницФормы (FormPagesRepresentation)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
ЗакладкиСверху (TabsOnTop)
ЗакладкиСлеваГоризонтально (TabsOnLeftHorizontal)
ЗакладкиСнизу (TabsOnBottom)
ЗакладкиСправаГоризонтально (TabsOnRightHorizontal)
Нет (None)
Пролистывание (Swipe)

Описание:

Содержит варианты отображения закладок страниц в панели в форме клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение группы формы для страниц, свойство ОтображениеСтраниц

Использование в версии:

Доступен, начиная с версии 8.2.
