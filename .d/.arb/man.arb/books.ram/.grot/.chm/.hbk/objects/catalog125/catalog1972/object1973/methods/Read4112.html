ВнешнийИсточникДанныхТаблицаОбъект.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableObject.<External source name>.<External data source table name>)

Прочитать (Read)

Доступен, начиная с версии 8.2.

Синтаксис:

Прочитать()

Описание:

Читает данные таблицы.
Позволяет прочесть заново данные элемента таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Недопустим для нового объекта.

Пример:

Объект.Прочитать();

Использование в версии:

Доступен, начиная с версии 8.2.
