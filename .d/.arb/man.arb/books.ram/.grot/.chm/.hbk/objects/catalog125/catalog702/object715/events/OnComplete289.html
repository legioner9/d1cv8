ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриЗавершении (OnComplete)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗавершении(<ТочкаМаршрута>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса вида Завершение, на которой происходит завершение бизнес-процесса.

<Отказ>

Тип: Булево.
Признак отказа от завершения бизнес-процесса. Если необходимые условия для завершения не выполнены (например, когда не выполнены все необходимые действия, предусмотренные данным бизнес-процессом), то в теле процедуры-обработчика следует установить данному параметру значение Истина. В этом случае бизнес-процесса не будет завершен.
Значение по умолчанию: Ложь.

Описание:

Вызывается при переходе бизнес-процесса на указанную в параметре точку завершения до установки ему признака "Завершен". Обычно вызывается вследствие вызова метода ВыполнитьЗадачу у задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точки вида Завершение.
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
