Описание:

Тип: ТипКонтрольнойТочкиСхемыЗапроса.
Указывает тип контрольной точки итогов схемы запроса.
Контрольная точка представляет собой значение, для которого будут вычисляться значения итоговых полей

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
