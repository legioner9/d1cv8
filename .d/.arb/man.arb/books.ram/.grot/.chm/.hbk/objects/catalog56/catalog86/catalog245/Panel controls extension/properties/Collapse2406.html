Расширение элементов управления, расположенных на панели (Panel controls extension)

Свертка (Collapse)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: РежимСверткиЭлементаУправления.
Свойство используется для схлопывания элементов управления. При схлопывании одна из сторон "подтягивается" к противоположной стороне так, что элемент управления становится полностью невидимым, схлопнутым. Если установлены привязки, они отрабатываются и элементы управления формы "стремятся" заполнить образовавшееся свободное пространство.

Доступность:

Толстый клиент.

Примечание:

Свернуть можно следующие элементы управления и только в определенном направлении:
панель, поле ввода, табличный документ, табличное поле по вертикали и горизонтали; разделитель - по горизонтали, если он ориентирован по горизонтали, по вертикали, если он ориентирован по вертикали.

Использование в версии:

Доступен, начиная с версии 8.0.
