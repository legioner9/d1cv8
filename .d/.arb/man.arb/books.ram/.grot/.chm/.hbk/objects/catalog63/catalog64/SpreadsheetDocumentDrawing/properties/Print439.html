РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ВыводитьНаПечать (Print)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак вывода рисунка на печать.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - выводить рисунок на печать; Ложь - не выводить.
Не применимо для рисунков типа Группа.

Использование в версии:

Доступен, начиная с версии 8.0.
