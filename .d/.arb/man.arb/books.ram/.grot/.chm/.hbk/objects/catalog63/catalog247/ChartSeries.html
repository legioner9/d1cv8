Описание:

Используется для доступа к свойствам серии диаграммы. Позволяет установить название серии, цвет, стиль линий, задать расшифровку и другие свойства.
Если количество серий диаграммы уменьшилось и объект не вошел в измененный диапазон, при обращении к нему будет сгенерирована ошибка. Если для диаграммы интерактивно запрещена настройка серий (снят флаг "Настройка серий"), то при попытке изменить любые свойства будет сгенерирована ошибка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
