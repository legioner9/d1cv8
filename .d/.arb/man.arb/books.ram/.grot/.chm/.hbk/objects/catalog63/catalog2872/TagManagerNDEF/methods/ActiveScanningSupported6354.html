МенеджерМетокNDEF (TagManagerNDEF)

ПоддерживаетсяАктивноеСканирование (ActiveScanningSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяАктивноеСканирование()

Возвращаемое значение:

Тип: Булево.
Истина - если устройство поддерживает активное сканирование и NFC включено.
Ложь - в противном случае.

Описание:

Проверяет, что устройство поддерживает активное сканирование меток.
Для устройств на Android дополнительно проверяется, что пользователь включил NFC в настройках телефона.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
