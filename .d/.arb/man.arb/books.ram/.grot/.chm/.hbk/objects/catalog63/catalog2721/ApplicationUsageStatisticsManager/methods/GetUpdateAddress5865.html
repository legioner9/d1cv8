МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьАдресОбновления (GetUpdateAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьАдресОбновления()

Возвращаемое значение:

Тип: Строка.

Описание:

Возвращает адрес автоматического обновления настроек регистрации событий/экранов статистики использования приложения, установленный методом УстановитьАдресОбновления.
Если в последний раз метод УстановитьАдресОбновления был вызван со параметром Неопределено – адрес не установлен.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
