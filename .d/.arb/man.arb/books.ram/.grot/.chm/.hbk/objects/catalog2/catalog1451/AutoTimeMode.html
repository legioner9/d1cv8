РежимАвтоВремя (AutoTimeMode)

РежимАвтоВремя (AutoTimeMode)

Доступен, начиная с версии 8.0.

Значения

НеИспользовать (DontUse)
Первым (First)
Последним (Last)
ТекущееИлиПервым (CurrentOrFirst)
ТекущееИлиПоследним (CurrentOrLast)

Описание:

Определяет набор режимов записи документов, бизнес-процессов, задач.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

ДокументОбъект.<Имя документа>, метод УстановитьВремя
Расширение формы клиентского приложения для документа, свойство АвтоВремя
