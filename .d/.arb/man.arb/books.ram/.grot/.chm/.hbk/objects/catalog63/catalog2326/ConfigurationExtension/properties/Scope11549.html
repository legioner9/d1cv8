РасширениеКонфигурации.ОбластьДействия (ConfigurationExtension.Scope)

РасширениеКонфигурации (ConfigurationExtension)

ОбластьДействия (Scope)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: ОбластьДействияРасширенияКонфигурации, Неопределено.
Область действия расширения конфигурации.
Для нового объекта РасширениеКонфигурации значение свойства может быть прочитано как Неопределено. При записи такого расширения это значение будет автоматически заменено на одно из значений типа ОбластьДействияРасширенияКонфигурации по следующим правилам:
  • Если включено разделение расширений конфигурации, то будет выбрано значение РазделениеДанных.
  • В остальных случаях будет выбрано значение ИнформационнаяБаза.
Эти значения также можно установить явно.

Доступность:

Сервер, толстый клиент, внешнее соединение.
