Расширение таблицы формы для диаграммы Ганта (Form table extension for Gantt chart)

Доступен, начиная с версии 8.3.21.

Описание:

Дополнительные свойства элемента управления ТаблицаФормы, применимые для таблицы значений диаграммы Ганта.
В качестве значений для свойств ТекущаяСтрока, ТекущийРодитель и элемента коллекции, содержащейся в свойстве ВыделенныеСтроки, используется идентификатор типа ИдентификаторЗначенияДиаграммыГанта.
Свойство ТекущиеДанные и метод ДанныеСтроки возвращают объект типа ДанныеФормыСтруктура. Именами свойств структуры будут являться имена колонок диаграммы Ганта. Изменение этих данных не изменит диаграмму Ганта.
Следующие свойства становятся доступными только для чтения (при попытке записи генерирутся исключение):

  • АвтоМаксимальнаяВысота,
  • АвтоМаксимальнаяВысотаВСтрокахТаблицы,
  • ВариантУправленияВысотой,
  • ВертикальнаяПолосаПрокрутки,
  • Высота,
  • ВысотаВСтрокахТаблицы,
  • ВысотаЗаголовка,
  • Заголовок,
  • МаксимальнаяВысота,
  • МаксимальнаяВысотаВСтрокахТаблицы,
  • Отображение,
  • ПоложениеЗаголовка,
  • ПоложениеСостоянияПросмотра,
  • ПоложениеСтрокиПоиска,
  • ПоложениеУправленияПоиском,
  • ПутьКДанным,
  • РастягиватьПоВертикали,
  • ЦветТекстаЗаголовка,
  • ШрифтЗаголовка.
Необходимо использовать одноименные свойства "Расширение поля формы для поля диаграммы Ганта" вместо свойств:
  • АвтоМаксимальнаяВысота,
  • Высота,
  • МаксимальнаяВысота,
  • РастягиватьПоВертикали
.

В режиме совместимости Версия8_3_22 и ниже необходимо использовать одноименные события "Расширения поля формы для поля диаграммы Ганта" вместо событий таблицы формы:
  • Выбор,
  • ПриИзменении,
  • ПередРазворачиванием,
  • ПередСворачиванием
.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.23.
