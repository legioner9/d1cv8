ЧтениеHTML.ОткрытьПоток (HTMLReader.OpenStream)

ЧтениеHTML (HTMLReader)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОткрытьПоток(<Поток>, <ТипКодировки>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для чтения.

<ТипКодировки> (необязательный)

Тип: Строка.
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае файл будет прочитан в соответствующей кодировке.
Если кодировка не задана, будет использоваться кодировка, указанная в атрибуте 'encoding' в объявлении HTML-документа.
Если атрибут 'encoding' отсутствует, то для чтения файла будет использована кодировка ANSI.

Описание:

Открывает поток, содержащий HTML для чтения данным объектом. Если перед вызовом данного метода уже выполнялось чтение HTML из другого файла или строки, то чтение прекращается и объект инициализируется для чтения из указанного потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
