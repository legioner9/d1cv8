ИнтернетПрокси.Пользователь (InternetProxy.User)

ИнтернетПрокси (InternetProxy)

Пользователь (User)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Пользователь(<Протокол>)

Параметры:

<Протокол> (обязательный)

Тип: Строка.
Имя протокола, для которого запрашивается установленное имя пользователя. При указании пустой строки возвращается имя пользователя, установленное по умолчанию.
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: Строка.

Описание:

Возвращает имя пользователя, установленное для аутентификации на прокси-сервере для данного протокола.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
