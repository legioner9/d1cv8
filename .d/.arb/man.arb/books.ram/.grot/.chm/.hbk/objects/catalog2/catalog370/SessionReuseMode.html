РежимПовторногоИспользованияСеансов (SessionReuseMode)

Доступен, начиная с версии 8.3.9.

Значения

Использовать (Use)
ИспользоватьАвтоматически (AutoUse)
НеИспользовать (DontUse)

Описание:

Режим повторного использования сеансов при вызове веб-сервиса, http-сервиса или интерфейса OData.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
