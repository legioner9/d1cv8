ЭлементСоставаОбщегоРеквизита (CommonAttributeContentItem)

ЭлементСоставаОбщегоРеквизита (CommonAttributeContentItem)

Доступен, начиная с версии 8.2.

Свойства:

Использование (Use)
Метаданные (Metadata)
УсловноеРазделение (ConditionalSeparation)

Описание:

Определяет вхождение связанного с элементом объекта метаданных в состав общего реквизита, а также условия использования общего реквизита как разделителя данного объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
