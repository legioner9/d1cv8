ПолеВвода (TextBox)

АвтоВыборНезаполненного (AutoChoiceIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Свойство указывает, нужно ли формировать автоматически состояние свойства ВыборНезаполненного в зависимости от устанавливаемого или формируемого в поле ввода значения. Если Истина, тогда при установке и формировании значения в поле ввода будет меняться состояние свойства ВыборНезаполненного: если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то ВыборНезаполненного будет выставлено в Истина, иначе - в Ложь.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
