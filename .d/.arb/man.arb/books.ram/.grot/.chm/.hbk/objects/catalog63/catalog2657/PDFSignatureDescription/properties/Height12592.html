Высота (Height)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит высоту подписи (в миллиметрах) на странице PDF-документа.
Если при подписании не указана, то высота будет определена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
