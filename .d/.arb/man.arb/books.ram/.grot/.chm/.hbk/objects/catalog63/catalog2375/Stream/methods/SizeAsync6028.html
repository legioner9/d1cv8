Поток.РазмерАсинх (Stream.SizeAsync)

Поток (Stream)

РазмерАсинх (SizeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазмерАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - тип Число, размер данных в байтах.

Описание:

Асинхронная функция. Получает размер данных в байтах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если ДоступноИзменениеПозиции = Ложь, то при вызове этого метода будет сгенерировано исключение.
