УголНаклона (Angle)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Угол наклона вывода текста подписи в градусах (от -360 до 360).
Используется только, если свойство Ориентация установлено в значение ПроизвольныйУголНаклона.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.12.
