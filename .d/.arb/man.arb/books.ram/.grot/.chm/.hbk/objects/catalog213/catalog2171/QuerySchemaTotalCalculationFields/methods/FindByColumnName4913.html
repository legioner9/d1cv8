КонтрольныеТочкиИтоговСхемыЗапроса (QuerySchemaTotalCalculationFields)

НайтиПоИмениКолонки (FindByColumnName)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НайтиПоИмениКолонки(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: Строка.
Имя колонки для поиска.

Возвращаемое значение:

Тип: КонтрольнаяТочкаИтоговСхемыЗапроса.

Описание:

Выполняет поиск по свойству ИмяКолонки контрольных точек итогов схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
