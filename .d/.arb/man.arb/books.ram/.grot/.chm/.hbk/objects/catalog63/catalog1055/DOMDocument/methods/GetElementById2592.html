ДокументDOM (DOMDocument)

ПолучитьЭлементПоИдентификатору (GetElementById)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьЭлементПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: Строка.
Идентификатор искомого элемента.

Возвращаемое значение:

Тип: ЭлементDOM, Неопределено.
Узел, у которого имеется атрибут со свойством ЯвляетсяИдентификатором, установленным в Истина.

Описание:

Выбирает все элементы по идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
