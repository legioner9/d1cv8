РастягиваниеПоВертикалиДиаграммыГанта (GanttChartVerticalStretch)

Доступен, начиная с версии 8.0.

Значения

НеРастягивать (None)
РастягиватьСтроки (StretchRows)
РастягиватьСтрокиИДанные (StretchRowsAndData)

Описание:

Содержит варианты способов масштабирования строк.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChartVerticalStretch.

См. также:

ДиаграммаГанта, свойство РастягиваниеПоВертикали

Использование в версии:

Доступен, начиная с версии 8.0.
