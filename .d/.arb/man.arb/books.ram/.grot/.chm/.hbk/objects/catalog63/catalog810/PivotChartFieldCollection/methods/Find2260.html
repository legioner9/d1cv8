КоллекцияПолейСводнойДиаграммы.Найти (PivotChartFieldCollection.Find)

КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.
Имя поля, которое требуется найти.

Возвращаемое значение:

Тип: КоллекцияПолейСводнойДиаграммы, ПолеСводнойДиаграммы, Неопределено.
Если поле отсутствует, возвращается значение Неопределено.

Описание:

Осуществляет поиск поля по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
