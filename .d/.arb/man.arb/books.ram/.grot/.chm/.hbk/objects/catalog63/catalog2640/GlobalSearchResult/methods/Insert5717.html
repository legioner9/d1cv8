РезультатГлобальногоПоиска (GlobalSearchResult)

Вставить (Insert)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: По значению

Синтаксис:

Вставить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Расположение элемента в коллекции.

<Значение> (обязательный)

Тип: ЭлементРезультатаГлобальногоПоиска.
Вставляемое значение.

Вариант синтаксиса: По параметрам

Синтаксис:

Вставить(<Индекс>, <Значение>, <Представление>, <Картинка>, <ВидПоиска>, <Описание>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Расположение элемента в коллекции.

<Значение> (необязательный)

Тип: Произвольный.
Значение результата поиска.

<Представление> (необязательный)

Тип: Строка, ФорматированнаяСтрока.
Представление результата поиска.

<Картинка> (необязательный)

Тип: Картинка.
Картинка результата поиска.

<ВидПоиска> (необязательный)

Тип: СтандартныйВидГлобальногоПоиска, Строка.
Вид поиска.

<Описание> (необязательный)

Тип: Строка, ФорматированнаяСтрока.
Описание результата поиска.

Возвращаемое значение:

Тип: ЭлементРезультатаГлобальногоПоиска.

Описание:

Вставляет элемент в заданную позицию коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
