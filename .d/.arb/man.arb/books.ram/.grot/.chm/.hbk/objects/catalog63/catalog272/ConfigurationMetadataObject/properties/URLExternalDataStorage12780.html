ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ХранилищеВнешнихДанныхНавигационныхСсылок (URLExternalDataStorage)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: СтандартноеХранилищеНастроекМенеджер.
В данном свойстве задается хранилище настроек, в котором будут сохраняться внешние данные ссылок.
Если хранилище не установлено:
  • Если режим совместимости не используется, используется стандартное хранилище.
  • В режиме совместимости Версия8_3_18:
    • при обращении из встроенного языка выдается ошибка,
    • формы отчета и динамического списка не предоставляют возможность получать ссылки с полными настройками.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
