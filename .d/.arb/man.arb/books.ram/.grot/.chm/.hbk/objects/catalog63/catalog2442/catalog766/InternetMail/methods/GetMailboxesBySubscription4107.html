ИнтернетПочта.ПолучитьПочтовыеЯщикиПоПодписке (InternetMail.GetMailboxesBySubscription)

ИнтернетПочта (InternetMail)

ПолучитьПочтовыеЯщикиПоПодписке (GetMailboxesBySubscription)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПочтовыеЯщикиПоПодписке()

Возвращаемое значение:

Тип: Массив.
Массив строк с именами подписанных почтовых ящиков IMAP.

Описание:

Получить список всех почтовых ящиков, которые помечены как подписанные.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется для POP3 соединения.

Использование в версии:

Доступен, начиная с версии 8.2.
