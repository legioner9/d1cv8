ТипДиаграммы.ГрафикПоШагам (ChartType.Step)

ТипДиаграммы (ChartType)

ГрафикПоШагам (Step)

Доступен, начиная с версии 8.0.

Описание:

Линейный график ступенями. Значения внутри одной серии в соседних точках соединены между собой горизонтальными линиями. Строится на основании данных, аналогичных данным диаграммы вида График.

Использование в версии:

Доступен, начиная с версии 8.0.
