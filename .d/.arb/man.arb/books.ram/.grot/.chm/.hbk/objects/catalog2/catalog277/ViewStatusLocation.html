ПоложениеСостоянияПросмотра (ViewStatusLocation)

Доступен, начиная с версии 8.3.5.

Значения

Авто (Auto)
Верх (Top)
Нет (None)
Низ (Bottom)

Описание:

Определяет положение автоматически размещенного состояния просмотра таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
