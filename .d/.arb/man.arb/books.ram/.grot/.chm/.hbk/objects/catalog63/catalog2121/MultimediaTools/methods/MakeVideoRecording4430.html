СредстваМультимедиа.СделатьВидеозапись (MultimediaTools.MakeVideoRecording)

СредстваМультимедиа (MultimediaTools)

СделатьВидеозапись (MakeVideoRecording)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СделатьВидеозапись(<ТипКамеры>, <Качество>)

Параметры:

<ТипКамеры> (необязательный)

Тип: ТипКамерыУстройства.
Показывает, какой камерой будет сделана видеозапись.

<Качество> (необязательный)

Тип: КачествоВидеозаписи.
Определяет качество видеозаписи.
В устройствах под управлением iOS параметр игнорируется.

Возвращаемое значение:

Тип: ДанныеМультимедиа, Неопределено.
Неопределено - запись не выполнена.

Описание:

Выполняет видеозапись встроенной фото-видео камерой.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяВидеозапись.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваМультимедиа, метод ПоддерживаетсяВидеозапись

Использование в версии:

Доступен, начиная с версии 8.3.3.
