Файл (File)

ПолучитьКартинкуПредставленияФайлаБиблиотекиМобильногоУстройстваАсинх (GetMobileDeviceLibraryFileThumbnailAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьКартинкуПредставленияФайлаБиблиотекиМобильногоУстройстваАсинх()

Возвращаемое значение:

Тип: Обещание.
При использовании на мобильном устройстве под управлением ОС Windows возвращает Неопределено.

Описание:

Асинхронная функция. Работает только с файлами, расположенными в мультимедиа библиотеках картинок и видео, с путем начинающемся со схемы “content:”.

Доступность:

Веб-клиент, мобильный клиент, мобильное приложение (клиент).
