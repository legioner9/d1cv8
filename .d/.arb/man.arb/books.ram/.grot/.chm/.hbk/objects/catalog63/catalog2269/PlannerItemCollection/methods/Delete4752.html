КоллекцияЭлементовПланировщика.Удалить (PlannerItemCollection.Delete)

КоллекцияЭлементовПланировщика (PlannerItemCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число, ЭлементПланировщика;.
Индекс удаляемого элемента в коллекции или сам элемент.

Описание:

Удаляет элемент планировщика в заданной позиции коллекции элементов планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
