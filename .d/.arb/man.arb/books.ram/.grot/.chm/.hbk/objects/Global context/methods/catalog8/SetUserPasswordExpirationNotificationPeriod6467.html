Глобальный контекст (Global context)

УстановитьСрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей (SetUserPasswordExpirationNotificationPeriod)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьСрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей(<СрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей>)

Параметры:

<СрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей> (необязательный)

Тип: Число.
Срок предупреждения об истечении срока действия пароля в секундах.
Значение по умолчанию: 432000.

Описание:

Устанавливает срок предупреждения об источении срока действия пароля в секундах

Значение по умолчанию: 432000 (5 суток).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.22.
