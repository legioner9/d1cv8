Менеджер кластера.MainPort (IClusterManagerInfo.MainPort)

Менеджер кластера (IClusterManagerInfo)

MainPort (MainPort)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.
Основной порт менеджера кластера.
От 0 до 65535. Выбирается кластером серверов автоматически из диапазонов портов для динамического занятия, определенных для рабочего сервера.

Доступность:

Интеграция.

Примечание:

Для нового дополнительного менеджера кластера – 0.

Использование в версии:

Доступен, начиная с версии 8.2.
