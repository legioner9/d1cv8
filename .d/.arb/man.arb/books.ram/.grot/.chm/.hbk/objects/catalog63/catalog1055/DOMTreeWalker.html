ОбходДереваDOM (DOMTreeWalker)

Доступен, начиная с версии 8.1.

Свойства:

Корень (Root)
РаскрыватьСсылкиНаСущности (ExpandEntityReference)
ТекущийУзел (CurrentNode)
Фильтр (Filter)

Методы:

ПервыйДочерний (FirstChild)
ПоследнийДочерний (LastChild)
ПредыдущийСоседний (PreviousSibling)
ПредыдущийУзел (PreviousNode)
РодительскийУзел (ParentNode)
СледующийСоседний (NextSibling)
СледующийУзел (NextNode)

Конструкторы:

На основании документа DOM

Описание:

Иерархический итератор по дереву документа DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ДокументDOM, метод СоздатьОбходДерева
ДокументHTML, метод СоздатьОбходДерева

Использование в версии:

Доступен, начиная с версии 8.1.
