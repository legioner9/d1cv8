УстановитьИзменяемыеПоля (SetChangeableFields)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьИзменяемыеПоля(<Поля>)

Параметры:

<Поля> (обязательный)

Тип: Массив.
Массив имен полей.

Описание:

Позволяет установить поля, которые можно изменять и которые участвуют в записи объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.3.6.
