ТабличныйДокумент (SpreadsheetDocument)

ПоказатьУровеньГруппировокКолонок (ShowColumnGroupLevel)

Доступен, начиная с версии 8.1.

Синтаксис:

ПоказатьУровеньГруппировокКолонок(<Уровень>)

Параметры:

<Уровень> (обязательный)

Тип: Число.
Уровень, до которого необходимо раскрыть группировки колонок.

Описание:

Показывает группировки колонок до заданного уровня.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
