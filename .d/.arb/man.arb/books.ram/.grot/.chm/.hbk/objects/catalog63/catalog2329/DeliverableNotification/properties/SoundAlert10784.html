ДоставляемоеУведомление (DeliverableNotification)

ЗвуковоеОповещение (SoundAlert)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: ЗвуковоеОповещение, Строка.
Звуковое оповещение системного уведомления, которое появляется на устройстве под управлением iOS или Android, если мобильное приложение не активно.
Может быть задан либо в виде стандартного звукового оповещения, либо в виде текстовой строки с именем аудиофайла. Если оповещение задано именем файла, файл должен быть добавлен в ресурсы мобильного приложения.
Файл может быть указан без расширения. В этом случае на платформе iOS будет использован файл с расширением "caf", на платформе Android - с расширением "mp3".
Обработчику в конфигурации значение не передается.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - ПоУмолчанию.

Использование в версии:

Доступен, начиная с версии 8.3.6.
