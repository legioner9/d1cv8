Описание:

Значения параметров вывода для таблиц.
Содержит значения следующих параметров:
1. РасположениеИтогов (TotalsPlacement) - расположение итогов (тип РасположениеИтоговКомпоновкиДанных);
2. РасположениеПолейГруппировки (GroupFieldsPlacement) - расположение полей группировки (тип РасположениеПолейГруппировкиКомпоновкиДанных);
3. РасположениеГруппировки (GroupPlacement) - расположение группировки (тип РасположениеГруппировкиКомпоновкиДанных);
4. РасположениеРеквизитов (AttributePlacement) - расположение реквизитов (тип РасположениеРеквизитовКомпоновкиДанных);
5. РасположениеРесурсов (ResourcePlacement) - расположение ресурсов (тип РасположениеРесурсовКомпоновкиДанных);
6. ГоризонтальноеРасположениеОбщихИтогов (HorizontalOverallPlacement) - горизонтальное расположение общих итогов (тип РасположениеИтоговКомпоновкиДанных);
7. ВертикальноеРасположениеОбщихИтогов (VerticalOverallPlacement) - вертикальное расположение общих итогов (тип РасположениеИтоговКомпоновкиДанных);
8. ТипЗаголовкаПолей (FieldsTitleType) - тип заголовка полей (ТипЗаголовкаПолейКомпоновкиДанных);
9. ВыводЗаголовка (TitleOutput) - вывод заголовка (тип ТипВыводаТекстаКомпоновкиДанных);
10. Заголовок (Title) - заголовок (тип Строка);
11. МакетОформления (AppearanceTemplate) - имя макета оформления (тип Строка);
12. АвтоПозицияРесурсов (ResourcesAutoPosition) - авто позиция ресурсов (тип АвтоПозицияРесурсовКомпоновкиДанных).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/settings}. Имя типа XDTO: TableOutputParameterValues.
