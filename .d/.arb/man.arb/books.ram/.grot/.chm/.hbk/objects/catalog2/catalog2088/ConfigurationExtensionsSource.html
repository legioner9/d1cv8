ИсточникРасширенийКонфигурации (ConfigurationExtensionsSource)

Доступен, начиная с версии 8.3.8.

Значения

БазаДанных (Database)
СеансАктивные (SessionApplied)
СеансОтключенные (SessionDisabled)

Описание:

Определяет источник получения списка расширений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

МенеджерРасширенийКонфигурации, метод Получить

Использование в версии:

Доступен, начиная с версии 8.3.8.
