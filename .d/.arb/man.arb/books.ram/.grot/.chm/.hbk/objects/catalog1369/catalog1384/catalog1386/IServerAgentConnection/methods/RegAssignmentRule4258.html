Соединение с агентом сервера (IServerAgentConnection)

RegAssignmentRule (RegAssignmentRule)

Доступен, начиная с версии 8.3.1.

Синтаксис:

RegAssignmentRule(<Кластер>, <Сервер>, <Требование>, <Позиция>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, для которого нужно добавить или обновить требование.

<Сервер> (обязательный)

Тип: Рабочий сервер.
Рабочий сервер, для которого нужно добавить или обновить требование.

<Требование> (обязательный)

Тип: Требование назначения сервисов.
Добавляемое требование назначения сервисов или новое состояние изменяемого требования назначения cервисов.

<Позиция> (обязательный)

Тип: Число.
Порядковый номер в массиве требования данного рабочего сервиса, на который должно быть помещено новое или измененное требование. Нумерация с 0.

Описание:

Добавляет или обновляет (перемещает) требование <Требование> в позицию <Позиция> списка требований сервера <Сервер> кластера <Кластер>.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
