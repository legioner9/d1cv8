АдминистрированиеМенеджерКластера (AdministrationClusterManager)

Доступен, начиная с версии 8.3.14.

Свойства:

ГлавныйМенеджер (MainManager)
ИдентификаторМенеджераКластера (ClusterManagerID)
ИдентификаторПроцесса (ProcessID)
ИмяКомпьютера (HostName)
Описание (Description)
Порт (Port)

Описание:

Содержит свойства одного главного или дополнительного менеджера кластера.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

См. также:

АдминистрированиеКластер, метод ПолучитьМенеджерыКластера

Использование в версии:

Доступен, начиная с версии 8.3.14.
