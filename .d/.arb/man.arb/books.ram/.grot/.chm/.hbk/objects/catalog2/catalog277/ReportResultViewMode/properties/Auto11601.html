Авто (Auto)

Доступен, начиная с версии 8.3.13.

Описание:

Режим отображения результата отчета определяется автоматически:
  • В мобильной версии - Компактный,
  • В платформе для ПК - Обычный.

Использование в версии:

Доступен, начиная с версии 8.3.13.
