ПреобразованиеККаноническомуXML (ConversionToCanonicalXML)

Доступен, начиная с версии 8.3.9.

Методы:

КанонизироватьВПоток (CanonicalizeToStream)
КанонизироватьВСтроку (CanonicalizeToString)
КанонизироватьВФайл (CanonicalizeToFile)

Конструкторы:

По умолчанию

Описание:

Объект предназначен для преобразования документов DOM к каноническому виду.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
