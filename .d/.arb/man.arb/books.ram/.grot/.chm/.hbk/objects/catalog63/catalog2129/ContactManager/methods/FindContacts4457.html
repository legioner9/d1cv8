МенеджерКонтактов.НайтиКонтакты (ContactManager.FindContacts)

МенеджерКонтактов (ContactManager)

НайтиКонтакты (FindContacts)

Доступен, начиная с версии 8.3.4.

Синтаксис:

НайтиКонтакты(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: ОтборКомпоновкиДанных.
Задает условия отбора контактов.
Отбор может быть установлен по следующим полям:
  • Имя (FirstName),
  • Фамилия (LastName),
  • Отчество (MiddleName),
  • ПроизношениеИмени (PhoneticFirstName),
  • ПроизношениеФамилии (PhoneticLastName),
  • ПроизношениеОтчества (PhoneticMiddleName),
  • Прозвище (Nickname),
  • Префикс (Prefix),
  • Суффикс (Suffix),
  • Организация (Organization),
  • Отдел (Department),
  • Должность (JobTitle),
  • Заметка (Note),
  • НомераТелефонов (PhoneNumbers),
  • АдресаЭлектроннойПочты (Emails),
  • Адреса (Addresses),
  • АдресаМгновенныхСообщений (InstantMessageAddresses),
  • ВебАдреса (URLs),
  • ДеньРождения (Birthday),
  • УчетнаяЗапись (Account) – для отбора по объекту типа УчетнаяЗаписьКонтактов,
  • ИдентификаторУчетнойЗаписи (AccountID),
  • ИсточникУчетнойЗаписи (AccountSource).

Возвращаемое значение:

Тип: Массив.
Массив ЛокальныйКлючКонтакта найденных контактов в соответствии с отбором.
Если отбор не задан, возвращает все контакты.

Описание:

Осуществляет поиск контактов.
Если отбор не задан, возвращает все контакты.
Отбор может быть установлен по следующим полям:

  • Имя (FirstName),
  • Фамилия (LastName),
  • Отчество (MiddleName),
  • ПроизношениеИмени (PhoneticFirstName),
  • ПроизношениеФамилии (PhoneticLastName),
  • ПроизношениеОтчества (PhoneticMiddleName),
  • Прозвище (Nickname),
  • Префикс (Prefix),
  • Суффикс (Suffix),
  • Организация (Organization),
  • Отдел (Department),
  • Должность (JobTitle),
  • Заметка (Note),
  • НомераТелефонов (PhoneNumbers),
  • АдресаЭлектроннойПочты (Emails),
  • Адреса (Addresses),
  • АдресаМгновенныхСообщений (InstantMessageAddresses),
  • ВебАдреса (URLs),
  • ДеньРождения (Birthday).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Пример:

Фильтр = новый ОтборКомпоновкиДанных();
элементОтбора =
    Фильтр.Элементы.Добавить(Тип("ЭлементОтбораКомпоновкиДанных"));
элементОтбора.ЛевоеЗначение = Новый ПолеКомпоновкиДанных
                        ("ИдентификаторУчетнойЗаписи");
элементОтбора.ВидСравнения = ВидСравненияКомпоновкиДанных.Равно;
элементОтбора.ПравоеЗначение = ИдентификаторУчетнойЗаписи;
 
Контакты = Менеджер.НайтиКонтакты(Фильтр);
 
Для Каждого КлючКонтакта Из Контакты Цикл
    Контакт = Менеджер.ПолучитьКонтакт(КлючКонтакта);
КонецЦикла;
