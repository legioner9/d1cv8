ТаблицыДляИзмененияСхемыЗапроса (QuerySchemaTablesForUpdate)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ТаблицаДляИзмененияСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются таблицы для изменения.
Возможно обращение к таблице для изменения посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Список таблиц для изменения схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
