ЗаписьДанных.НачатьЗакрытие (DataWriter.BeginClose)

ЗаписьДанных (DataWriter)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Инициирует вызов метода СброситьБуферы. Если целевой поток был создан при создании объекта ЗаписьДанных, также вызывает начало закрытие целевого потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ЗаписьДанных, метод Закрыть
ЗаписьДанных, метод ЗакрытьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
