ДиалогВыбораПользователейИсторииДанных (DataHistoryUsersChooseDialog)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ДиалогВыбораПользователейИсторииДанных(<Данные>, <Пользователи>)

Параметры:

<Данные> (обязательный)

Тип: СправочникСсылка.<Имя справочника>, ДокументСсылка.<Имя документа>, РегистрСведенийКлючЗаписи.<Имя регистра сведений>, БизнесПроцессСсылка.<Имя бизнес-процесса>, ЗадачаСсылка.<Имя задачи>, ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>, ПланСчетовСсылка.<Имя плана счетов>.
Данные, по которым отбираются версии. Используется для формирования списка доступных пользователей-авторов версий.

<Пользователи> (необязательный)

Тип: Массив.
Массив полных имен пользователей.
Устанавливается, если необходимо заполнить параметры отбора при открытии диалога. Если диалог был закрыт нажатием кнопки "OK", параметр содержит результат выбора.
Значение по умолчанию: Пустой массив.

Описание:

Создает диалог отбора пользователей версий указанных данных.

Использование в версии:

Доступен, начиная с версии 8.3.13.
