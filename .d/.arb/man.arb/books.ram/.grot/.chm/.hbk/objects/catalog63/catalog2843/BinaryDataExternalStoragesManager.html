МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Доступен, начиная с версии 8.3.24.

Элементы коллекции:

МенеджерВнешнегоХранилищаДвоичныхДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются МенеджерВнешнегоХранилищаДвоичныхДанных.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя внешнего хранилища.

Методы:

Добавить (Add)
Количество (Count)
Найти (Find)
Удалить (Delete)

Описание:

Предоставляет доступ к внешним хранилищам двоичных данных. Может быть получен с помощью свойства глобального контекста ВнешниеХранилищаДвоичныхДанных.
Требуется наличие у текущего пользователя права "Администрирование".

Доступность:

Сервер.

Пример:

// обход всех внешних хранилищ (без ХранилищеДвоичныхДанных)

Для каждого Мнджр Из ВнешниеХранилищаДвоичныхДанных Цикл
    Дата = ТекущаяУниверсальнаяДата();
    Мнджр.ОчиститьНеиспользуемоеМестоПоУниверсальнойДате(Дата);
КонецЦикла;

// доступ к менеджеру по имени

Мнджр = ВнешниеХранилищаДвоичныхДанных["manager"];

См. также:

Глобальный контекст, свойство ВнешниеХранилищаДвоичныхДанных

Использование в версии:

Доступен, начиная с версии 8.3.24.
