ОбъявлениеЭлементаXS.НедопустимыеПодстановки (XSElementDeclaration.DisallowedSubstitutions)

ОбъявлениеЭлементаXS (XSElementDeclaration)

НедопустимыеПодстановки (DisallowedSubstitutions)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: ОбъединениеНедопустимыхПодстановкиXS.
Содержит виды недопустимых подстановок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Представляет значение атрибута disallowed substitutions (http://www.w3.org/TR/xmlschema-1/#e-exact) (вычисляется на основе атрибута block).

Использование в версии:

Доступен, начиная с версии 8.1.
