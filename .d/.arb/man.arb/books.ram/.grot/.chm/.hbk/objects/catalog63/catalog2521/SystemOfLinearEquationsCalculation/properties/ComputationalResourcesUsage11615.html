РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ИспользованиеВычислительныхРесурсов (ComputationalResourcesUsage)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.
В качестве количества потоков можно указать любое целое число, большее или равное 0. Значение по умолчанию - 0.
Если указано значение 0, прикладное решение будет использовать n-1 поток, где n – максимальное число неконкурирующих потоков, которые можно запустить на компьютере (обычно количество потоков равно числу ядер, при использовании HyperThreading – удвоенному числу ядер).
Не рекомендуется устанавливать значение больше максимального возможного числа неконкурирующих потоков – это приведет к замедлению работы прикладного решения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
