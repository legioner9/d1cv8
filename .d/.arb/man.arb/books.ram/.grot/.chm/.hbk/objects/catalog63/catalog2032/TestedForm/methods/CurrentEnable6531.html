ТекущаяДоступность (CurrentEnable)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ТекущаяДоступность()

Возвращаемое значение:

Тип: Булево.
Истина - объект доступен; Ложь в противном случае.

Описание:

Возвращает признак того, что пользователь может взаимодействовать с элементами формы.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
