Рабочий процесс.MemoryExcessTime (IWorkingProcessInfo.MemoryExcessTime)

Рабочий процесс (IWorkingProcessInfo)

MemoryExcessTime (MemoryExcessTime)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит время, в течение которого объем виртуальной памяти рабочего процесса превышает критическое значение, установленное для кластера, в секундах.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
