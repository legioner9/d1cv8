ГоризонтальноеПоложениеЭлемента (ItemHorizontalLocation)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
Лево (Left)
Право (Right)
Центр (Center)

Описание:

Содержит варианты горизонтального положение элемента управления.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПолеФормы, свойство ГоризонтальноеПоложение
ПолеФормы, свойство ГоризонтальноеПоложениеВШапке
ПолеФормы, свойство ГоризонтальноеПоложениеВПодвале
Расширение группы формы для командной панели, свойство ГоризонтальноеПоложение
Расширение декорации формы для надписи, свойство ГоризонтальноеПоложение

Использование в версии:

Доступен, начиная с версии 8.2.
