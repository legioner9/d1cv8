ИнтернетПочтовоеСообщение.Категории (InternetMailMessage.Categories)

ИнтернетПочтовоеСообщение (InternetMailMessage)

Категории (Categories)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Указываются категории, к которым относится сообщение. Категория может быть как одна, так и несколько, разделенных запятыми. Например:
Сообщение = Новый ИнтернетПочтовоеСообщение;
Сообщение.Категории = "Личное, Поздравления, Подарки";

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
