ИсключенияГруппПодстановкиXS (XSSubstitutionGroupExclusions)

Доступен, начиная с версии 8.1.

Значения

Все (All)
Ограничение (Restriction)
Расширение (Extension)

Описание:

Содержит виды исключения групп подстановки.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
