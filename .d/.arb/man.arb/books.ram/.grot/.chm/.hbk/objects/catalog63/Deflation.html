СжатиеДанных (Deflation)

Доступен, начиная с версии 8.0.

Конструкторы:

По степени сжатия

Описание:

Реализует алгоритм сжатия данных "Deflation". Может быть использован для сжатия данных объектом ХранилищеЗначения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Хранилище = Новый ХранилищеЗначения(Значение, Новый СжатиеДанных(9));

См. также:

ХранилищеЗначения, конструктор На основании значения

Использование в версии:

Доступен, начиная с версии 8.0.
