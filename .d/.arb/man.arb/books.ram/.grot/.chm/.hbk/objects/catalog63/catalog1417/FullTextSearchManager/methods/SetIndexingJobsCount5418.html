МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

УстановитьКоличествоЗаданийИндексирования (SetIndexingJobsCount)

Доступен, начиная с версии 8.3.12.

Синтаксис:

УстановитьКоличествоЗаданийИндексирования(<КоличествоЗаданий>)

Параметры:

<КоличествоЗаданий> (обязательный)

Тип: Число.
Устанавливаемое количество заданий:
  • Если значение больше 1, то дополнительно могут быть запущены фоновые задания для индексирования, и их количество не будет превышать значение параметра. При порционном обновлении индекса, если количество не проиндексированных данных больше чем на одну порцию, то тогда также могут быть запущены фоновые задания для индексирования очередной порции.
  • Если значение равно 0, количество заданий подбирается автоматически.

Значение по умолчанию: 0.

Описание:

Устанавливает максимальное количество фоновых заданий, которое может быть запущено одновременно для обновления индекса. Значение параметра может быть получено при помощи метода ПолучитьКоличествоЗаданийИндексирования.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В файловом варианте значение параметра не используется и дополнительные фоновые задания в процессе индексирования не запускаются.
Установка параметра возможна только при наличии у текущего пользователя информационной базы права "Администрирование".

Использование в версии:

Доступен, начиная с версии 8.3.12.
