ПараметрыПолученияАрхиваФайлов (GetFilesArchiveParameters)

Доступен, начиная с версии 8.3.15.

Свойства:

ИмяАрхива (ArchiveName)
Режим (Mode)

Конструкторы:

По имени архива и режиму
По умолчанию

Описание:

Содержит настройки получения файлов в виде архива. Используется для работы методов НачатьПолучениеФайловССервера и ПолучитьФайлыССервераАсинх.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
