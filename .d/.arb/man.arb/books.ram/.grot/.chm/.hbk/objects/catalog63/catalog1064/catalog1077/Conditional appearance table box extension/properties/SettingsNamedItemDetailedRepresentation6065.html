Расширение табличного поля условного оформления компоновки данных (Conditional appearance table box extension)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если свойство установлено в Ложь, именованные части условного оформления (свойство Представление не пустое) отображаются в табличном поле одной строкой с одной ячейкой, в которой выводится представление.
В противном случае представление редактируется в отдельной колонке, отображение именованных и неименованных элементов условного оформления не отличается.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
