Геозона (Geofence)

По умолчанию

Доступен, начиная с версии 8.3.10.

Синтаксис:

Новый Геозона(<Данные>, <Представление>, <Координаты>, <Радиус>)

Параметры:

<Данные> (обязательный)

Тип: Произвольный.
Данные геозоны. Параметр должен поддерживать сериализацию.

<Представление> (обязательный)

Представление геозоны, отображаемое для пользователя.

<Координаты> (обязательный)

Тип: ГеографическиеКоординаты.
Координаты геозоны.

<Радиус> (обязательный)

Тип: Число.
Радиус геозоны.

Описание:

Создает новый объект на основе переданных параметров.

Использование в версии:

Доступен, начиная с версии 8.3.10.
