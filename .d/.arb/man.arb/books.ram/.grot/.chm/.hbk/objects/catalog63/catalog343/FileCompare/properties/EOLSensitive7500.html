УчитыватьРазделителиСтрок (EOLSensitive)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает необходимость сравнения разделителей строк при сравнении текстов. Истина - разделители строк учитываются. Значение по умолчания - Ложь.

Доступность:

Толстый клиент.

Примечание:

При сравнении табличных документов и двоичном сравнении свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.1.
