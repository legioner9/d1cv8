ИзбранноеРаботыПользователя (UserWorkFavorites)

Доступен, начиная с версии 8.2.

Элементы коллекции:

ЭлементИзбранногоРаботыПользователя
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элемент избранного.
Возможно обращение к элементу избранного посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Предназначен для управления (получения и установки) избранным для пользователя информационной базы. Его можно получать и сохранять в настройках с помощью ХранилищеСистемныхНастроек.
Настройки хранятся в системных настройках по адресу: Общее/ИзбранноеРаботыПользователя (Common/UserWorkFavorites).
Для использования объекта требуется, чтобы для пользователя было доступно право СохранениеДанныхПользователя.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Пример:

Избранное = ХранилищеСистемныхНастроек.Загрузить("Общее/ИзбранноеРаботыПользователя");
Элемент = Новый ЭлементИзбранногоРаботыПользователя;
Элемент.НавигационнаяСсылка = "file://C:/Kro05.jpg";
Избранное.Добавить(Элемент);
ХранилищеСистемныхНастроек.Сохранить("Общее/ИзбранноеРаботыПользователя", "", Избранное);

Использование в версии:

Доступен, начиная с версии 8.2.
