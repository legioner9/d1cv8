АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

СоздатьБазуДанных (CreateDatabase)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает режим создания информационной базы:
  • Истина - при создании информационной базы создавать базу данных;
  • Ложь - при создании информационной базы базу данных не создавать.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.15.
