Описание:

Используется для доступа к диапазону IP-портов, который используется системой для назначения адресов рабочим процессам, создаваемым на сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
