ФормаКлиентскогоПриложения (ClientApplicationForm)

КлючНазначенияИспользования (PurposeUseKey)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Тип: Строка.
Определяет назначение использования формы.
См. также КлючНазначенияИспользования.
Если режим совместимости Версия8_3_4 или ниже не установлен, то при попытке открытия формы клиентского приложения с параметром КлючНазначенияИспользования, длина значения которого превышает 128 символов, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
