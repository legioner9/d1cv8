СвязьПоТипуКомпоновкиДанных (DataCompositionTypeLink)

ЭлементСвязи (LinkItem)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.
Элемент связи по типу.
Номер элемента объекта, с которым поле связано по типу. Например, если поле связано со ссылкой на счет, то номер элемента будет обозначать номер вида субконто, из которого будет взят тип для отработки связи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
