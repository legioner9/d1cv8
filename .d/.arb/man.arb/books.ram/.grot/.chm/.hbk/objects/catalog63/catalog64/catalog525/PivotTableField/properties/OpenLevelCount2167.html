ПолеСводнойТаблицы (PivotTableField)

КоличествоОткрытыхУровней (OpenLevelCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит количество открытых уровней измерения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если значение свойства 0, то измерение не расположено ни на одной оси.
Если поле добавляется в коллекцию строк/колонок сводной таблицы, то свойство устанавливается в 1.
Если для поля, которое было добавлено в строки/колонки сводной таблицы, устанавливается значение свойства 0, то поле удаляется из коллекции строк/колонок сводной таблицы.
Если поле не было добавлено в коллекцию строк/колонок сводной таблицы и ему устанавливается значение свойства, отличное от 0, то это поле будет автоматически добавлено в коллекцию строк сводной таблицы.

Использование в версии:

Доступен, начиная с версии 8.0.
