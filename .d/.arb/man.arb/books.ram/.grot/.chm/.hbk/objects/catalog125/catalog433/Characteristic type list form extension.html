Расширение формы списка видов характеристик (Characteristic type list form extension)

Расширение формы списка видов характеристик (Characteristic type list form extension)

Доступен, начиная с версии 8.0.

Свойства:

ПараметрВыборГруппИЭлементов (ChoiceFoldersAndItemsParameter)
ПараметрТекущаяСтрока (CurrentLineParameter)

Описание:

Дополнительные свойства объекта Форма списка видов характеристик, в том случае, если основной реквизит формы имеет тип ПланВидовХарактеристикСписок.<Имя плана видов характеристик>.

Доступность:

Толстый клиент.

Пример:

Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы;

Использование в версии:

Доступен, начиная с версии 8.0.
