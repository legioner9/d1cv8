СтандартнаяКомандаПланировщика (PlannerStandardCommand)

РедактироватьЭлемент (EditItem)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

  • РедактироватьЭлемент

Доступен, начиная с версии 8.3.16.

Описание:

Переход в режим редактирования выбранного элемента планировщика.
Используется только для источника типа Элементы, когда свойство Элементы содержит массив из одного элемента планировщика.
В контекстном меню команда будет иметь представление "Редактировать".

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
