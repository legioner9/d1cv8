ОбъектМетаданных: Поле (MetadataObject: Field)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.
Признак вывода отрицательного значения особым образом.
Истина - при выводе реквизита с отрицательным значением значение будет отображено особым образом.
Имеет смысл только для реквизитов типа Число.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
