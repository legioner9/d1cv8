Рабочий сервер.GetPortRanges (IWorkingServerInfo.GetPortRanges)

Рабочий сервер (IWorkingServerInfo)

GetPortRanges (GetPortRanges)

Доступен, начиная с версии 8.1.

Синтаксис:

GetPortRanges()

Возвращаемое значение:

Тип: COMSafeArray.
Метод возвращает COMSafeArray из объектов Диапазон портов.

Описание:

Получает выделенные диапазоны IP-портов для динамического занятия.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
