МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

ТекущийРежим (CurrentMode)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ТекущийРежим()

Возвращаемое значение:

Тип: РежимПрогрессивногоВебПриложения, Неопределено.
Для тонкого, толстого, мобильного клиентов возвращаемое значение всегда Неопределено.
Для веб-клиента:
  • Если прогрессивное веб-приложение установлено и открыто в отдельном окне, возвращаемое значение - ВОтдельномОкне.
  • Если прогрессивное веб-приложение установлено и открыто во вкладке браузера, возвращаемое значение - ВОкнеБраузера.
  • В остальных случаях - Неопределено.

Описание:

Определяет текущий режим прогрессивного веб-приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
