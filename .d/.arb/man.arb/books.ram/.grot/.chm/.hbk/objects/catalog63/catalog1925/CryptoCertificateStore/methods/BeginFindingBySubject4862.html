ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьПоискПоСубъекту (BeginFindingBySubject)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПоискПоСубъекту(<ОписаниеОповещения>, <СтрокаПоиска>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения поиска со следующими параметрами:
  • <Сертификаты> - массив объектов СертификатКриптографии.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<СтрокаПоиска> (обязательный)

Тип: Строка.
Строка поиска субъекта.
Данные субъекта должны содержать переданную в этом параметре строку.

Описание:

Начинает поиск сертификатов криптографии по переданному имени субъекта, которому был выдан сертификат.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
