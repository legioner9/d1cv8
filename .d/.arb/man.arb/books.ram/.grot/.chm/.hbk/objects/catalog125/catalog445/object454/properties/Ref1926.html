ПланСчетовОбъект.<Имя плана счетов>.Ссылка (ChartOfAccountsObject.<Chart of accounts name>.Ref)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовСсылка.<Имя плана счетов>.
Содержит ссылку на счет. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Например, данное свойство можно использовать, если ссылку на созданный и записанный счет нужно присвоить реквизиту другого счета.

Использование в версии:

Доступен, начиная с версии 8.0.
