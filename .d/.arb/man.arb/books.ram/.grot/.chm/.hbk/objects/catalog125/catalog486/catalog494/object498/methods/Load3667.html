ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Загрузить (Load)

Доступен, начиная с версии 8.1.

Синтаксис:

Загрузить(<ТаблицаЗначений>)

Параметры:

<ТаблицаЗначений> (обязательный)

Тип: ТаблицаЗначений.
Источник для заполнения набора записей.

Описание:

Очищает набор записей и заполняет его значениями из указанной таблицы значений. Заполняются значения доступных для записи свойств записей перерасчета, имена которых совпали c именами колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
