СпособКодированияНеASCIIСимволовИнтернетПочтовогоСообщения (InternetMailMessageNonASCIISymbolsEncodingMode)

БезКодирования (None)

Доступен, начиная с версии 8.1.

Описание:

Все поля заголовков, содержащие не ASCII символы кодироваться не будут. Все символы будут конвертироваться в кодировку, заданною свойством Кодировка.

Использование в версии:

Доступен, начиная с версии 8.1.
