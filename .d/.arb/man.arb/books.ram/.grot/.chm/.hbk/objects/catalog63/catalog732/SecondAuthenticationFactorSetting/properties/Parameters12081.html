НастройкаВторогоФактораАутентификации.Параметры (SecondAuthenticationFactorSetting.Parameters)

НастройкаВторогоФактораАутентификации (SecondAuthenticationFactorSetting)

Параметры (Parameters)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Соответствие.
Настраиваемые параметры, которые будут использованы в шаблоне при аутентификации. Элемент соответствия представляет собой пару из имени параметра и его значения.
Например:
Если задано соответствие "sms_phone_number, +71234567890", тогда все вхождения подстроки "sms_phone_number" внутри свойств объекта ШаблонНастройкиВторогоФактораАутентификации будут заменены на +71234567890.
Если в качестве шаблона используется объект типа ШаблонНастройкиВторогоФактораАутентификации, тогда будет выполнен поиск вхождений параметра в свойствах:
  • по структуре HTTPЗапросНаАутентификацию:
    • АдресРесурса,
    • Заголовки,
    • в теле файла, устанавливаемом любым возможным способом.
  • по тексту в МетодHTTPЗапросаНаАутентификацию;
  • по структуре МетодHTTPЗапросаНаПроверкуРезультатаАутентификации:
    • АдресРесурса,
    • Заголовки,
    • в теле файла, устанавливаемом любым возможным способом..
  • по тексту в МетодHTTPЗапросаНаПроверкуРезультатаАутентификации.
Зарезервированные переменные:
  • secret - на место этой переменной будет подставлен секретный токен второго фактора, сгенерированной платформой. Длина токена фиксирована и равна 5.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
