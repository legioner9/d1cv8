ВстроеннаяПокупка (InAppPurchase)

Описание (Description)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: Строка.
Содержит описание покупки на текущем языке мобильного устройства. Если соответствующая локализация названия покупки в магазине приложений отсутствует, то используется язык, указанные в магазине как язык по умолчанию.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
