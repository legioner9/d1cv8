ИнформацияОбОшибке (ErrorInfo)

ЯвляетсяОшибкойКатегории (IsErrorOfCategory)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ЯвляетсяОшибкойКатегории(<Категория>, <УчитыватьПричины>)

Параметры:

<Категория> (обязательный)

Тип: КатегорияОшибки.
Проверяемая категория ошибки.

<УчитыватьПричины> (необязательный)

Тип: Булево.
Если параметр имеет значение Истина, то учитывается вся цепочка причин у описания ошибки.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: Булево.
Для категории ВсеОшибки всегда возвращает Истина.

Описание:

Проверяет, что ошибка принадлежит к указанной категории.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
