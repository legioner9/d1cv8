СертификатКлиентаОС (OSClientCertificate)

По способу выбора сертификата

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый СертификатКлиентаОС(<СпособВыбораСертификатаОС>)

Параметры:

<СпособВыбораСертификатаОС> (необязательный)

Тип: СпособВыбораСертификатаОС.
Указывает способ выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы.

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы.

Использование в версии:

Доступен, начиная с версии 8.3.21.
