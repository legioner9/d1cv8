ПланыВидовРасчетаМенеджер (ChartsOfCalculationTypesManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПланВидовРасчетаМенеджер.<Имя плана видов расчета>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов расчета.
Возможно обращение к менеджеру плана видов расчета посредством оператора [...]. В качестве аргумента передается имя плана видов расчета, как оно задано в конфигурации.

Свойства:

<Имя плана видов расчета> (<Chart of calculation types name>)

Методы:

ТипВсеСсылки (AllRefsType)

Описание:

Предназначен для управления планами видов расчета и предоставляет доступ к значениям типа ПланВидовРасчетаМенеджер.<Имя плана видов расчета>. Доступ к объекту осуществляется через свойство глобального контекста ПланыВидовРасчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

См. также:

Глобальный контекст, свойство ПланыВидовРасчета

Использование в версии:

Доступен, начиная с версии 8.0.
