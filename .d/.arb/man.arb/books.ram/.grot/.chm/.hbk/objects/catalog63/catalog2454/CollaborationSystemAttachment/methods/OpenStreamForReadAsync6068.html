ОткрытьПотокДляЧтенияАсинх (OpenStreamForReadAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьПотокДляЧтенияАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа Поток для содержимого вложения.

Описание:

Асинхронная функция. Открывает поток для чтения содержимого вложения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
