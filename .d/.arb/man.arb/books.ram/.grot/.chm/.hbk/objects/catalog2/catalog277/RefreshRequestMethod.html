СпособЗапросаОбновления (RefreshRequestMethod)

СпособЗапросаОбновления (RefreshRequestMethod)

Доступен, начиная с версии 8.3.6.

Значения

Нет (None)
ПотянутьСверху (PullFromTop)
ПотянутьСверхуИлиСнизу (PullFromTopOrBottom)
ПотянутьСнизу (PullFromBottom)

Описание:

Содержит варианты способов инициации обновления таблицы на мобильной платформе.

Доступность:

Мобильный клиент, сервер, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
