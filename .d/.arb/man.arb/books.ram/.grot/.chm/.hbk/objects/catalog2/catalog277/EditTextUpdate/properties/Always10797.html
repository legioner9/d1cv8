ОбновлениеТекстаРедактирования (EditTextUpdate)

Всегда (Always)

Доступен, начиная с версии 8.3.8.

Описание:

Текст в поле ввода формы клиентского приложения обновляется при любой установке значения в поле (установка значения может быть выполнена системой автоматически и в том случае, когда значение другого реквизита было изменено программно).

Использование в версии:

Доступен, начиная с версии 8.3.8.
