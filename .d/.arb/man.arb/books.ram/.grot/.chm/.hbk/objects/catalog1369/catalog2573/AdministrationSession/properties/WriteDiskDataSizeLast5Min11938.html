ОбъемДанныхЗаписанныхНаДискЗа5Мин (WriteDiskDataSizeLast5Min)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.
Объем данных, записанных на диск данным сеансом за последние 5 минут (в байтах).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
