Поток.НачатьСбросБуферов (Stream.BeginFlush)

Поток (Stream)

НачатьСбросБуферов (BeginFlush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьСбросБуферов(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Начинает сбрасывать все промежуточные буферы и производить запись всех незаписанных данных в целевое устройство.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

Поток, метод СброситьБуферы
Поток, метод СброситьБуферыАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.
