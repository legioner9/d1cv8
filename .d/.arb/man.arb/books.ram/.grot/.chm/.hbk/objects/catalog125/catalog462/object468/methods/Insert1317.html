РегистрБухгалтерииСубконто.<Имя регистра бухгалтерии> (AccountingRegisterExtDimensions.<Accouting register name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ВидСубконто>, <Значение>)

Параметры:

<ВидСубконто> (обязательный)

Тип: ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>.
Вид субконто устанавливаемого элемента.

<Значение> (необязательный)

Значение устанавливаемого элемента (Характеристика.<Имя плана видов характеристик>). Если не указано, то будет вставлено Неопределено.

Описание:

Устанавливает значение элемента коллекции субконто. Если элемент с переданным значением существует, то его значение заменяется, в противном случае добавляется новый элемент.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
