РазрешениеКамерыУстройства (DeviceCameraResolution)

Доступен, начиная с версии 8.3.6.

Свойства:

Высота (Height)
Ширина (Width)

Конструкторы:

По умолчанию

Описание:

Описывает разрешение, с которым работает камера мобильного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваМультимедиа, метод СделатьФотоснимок
