КоллекцияИдентификаторовПользователейСистемыВзаимодействия (CollaborationSystemUserIDCollection)

Доступен, начиная с версии 8.3.10.

Элементы коллекции:

ИдентификаторПользователяСистемыВзаимодействия
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Содержит (Contains)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Предоставляет методы для работы с идентификаторами пользователей системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
