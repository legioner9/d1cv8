Описание:

Дополнительные свойства элемента управления ТабличноеПоле для записей регистра накопления. Для данного расширения свойство ТекущаяСтрока содержит значение типа РегистрНакопленияКлючЗаписи.<Имя регистра накопления>. Свойство ТекущиеДанные содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта РегистрНакопленияСписок.<Имя регистра накопления>.

Толстый клиент.
