Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСФайлами (AttachFileSystemExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

  • НачатьПодключениеРасширенияРаботыСФайлами

Доступен, начиная с версии 8.2.

Синтаксис:

ПодключитьРасширениеРаботыСФайлами()

Возвращаемое значение:

Тип: Булево.
В тонком клиенте результат всегда Истина.
Для режима запуска "Мобильное приложение (клиент)" результат всегда Истина.

Описание:

При первом обращении выполняет подключение расширения для работы с файлами.
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
