ИнтернетПочтовоеСообщение.СтатусРазбора (InternetMailMessage.ParseStatus)

ИнтернетПочтовоеСообщение (InternetMailMessage)

СтатусРазбора (ParseStatus)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: СтатусРазбораПочтовогоСообщения.
Содержит статус разбора исходного текста почтового сообщения.
Если свойство имеет значение ОбнаруженыОшибки, то это означает, что во время разбора в структуре сообщения были найдены ошибки и объект ИнтернетПочтовоеСообщение сформирован по данным, которые удалось успешно распознать.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При инициализации сообщения с помощью метода УстановитьИсходныеДанные установка значения свойства работает так же, как и при получении сообщения с сервера.

Использование в версии:

Доступен, начиная с версии 8.3.6.
