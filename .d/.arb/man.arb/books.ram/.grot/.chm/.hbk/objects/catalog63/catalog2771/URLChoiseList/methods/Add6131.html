СписокВыбораНавигационнойСсылки (URLChoiseList)

Добавить (Add)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Добавить(<Ключ>, <Представление>, <ЗапрашиватьПользовательскоеПредставление>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.
Ключ навигационной ссылки, который будет передан обработчику для получения навигационной ссылки.

<Представление> (обязательный)

Тип: Строка.
Представление ссылки, которое в форме получения ссылки будет использовано в списке выбора ссылки.

<ЗапрашиватьПользовательскоеПредставление> (необязательный)

Тип: Булево.
Указывает необходимость запросить представление ссылки у пользователя.
Если Истина - форма получения навигационной ссылки запросит у пользователя представление ссылки, которое будет использовано для сохранения внешних данных ссылки и для отображения ссылки, например, в списке избранного.
Если Ложь - для сохранения данных и отображения ссылки будет использовано значение свойства Представление.
Значение по умолчанию: Ложь.

Описание:

Добавляет новый элемент в конец коллекции.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
