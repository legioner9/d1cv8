ТабличноеПоле (TableBox)

Шрифт (Font)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Шрифт.
Содержит шрифт, используемый при отображении текста в ячейках табличного поля.

Доступность:

Толстый клиент.

Примечание:

Указанный шрифт используется, если для ячейки указан вид шрифта АвтоШрифт.

Использование в версии:

Доступен, начиная с версии 8.0.
