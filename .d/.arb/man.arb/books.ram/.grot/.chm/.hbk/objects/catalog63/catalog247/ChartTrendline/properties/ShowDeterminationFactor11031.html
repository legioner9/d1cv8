ЛинияТрендаДиаграммы (ChartTrendline)

ОтображатьКоэффициентДетерминации (ShowDeterminationFactor)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если свойство установлено в Истина, в области построения будет отображаться коэффициент детерминации (погрешность аппроксимации) линии тренда.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
