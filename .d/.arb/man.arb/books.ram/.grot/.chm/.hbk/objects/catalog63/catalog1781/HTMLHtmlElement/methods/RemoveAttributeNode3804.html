ЭлементРазметкаHTML.УдалитьУзелАтрибута (HTMLHtmlElement.RemoveAttributeNode)

ЭлементРазметкаHTML (HTMLHtmlElement)

УдалитьУзелАтрибута (RemoveAttributeNode)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьУзелАтрибута(<Атрибут>)

Параметры:

<Атрибут> (обязательный)

Тип: АтрибутHTML.
Удаляемый атрибут элемента.

Возвращаемое значение:

Тип: АтрибутHTML.
Удаленный атрибут.

Описание:

Удаляет атрибут из элемента.
В случае удаления атрибута, не явно указанного в элементе, а взятого из схемы документа, будет создан новый узел Атрибут, и в него будет установлено значение по умолчанию.
Причины вызова исключений:

  • указанный атрибут не является атрибутом данного элемента;
  • данный узел в режиме Только для чтения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
