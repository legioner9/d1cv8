Форма (Form)

КартинкаЗаголовка (CaptionPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Картинка.
Содержит картинку, отображаемую в заголовке формы.

Доступность:

Толстый клиент.

Примечание:

Если картинка задается программно, может быть установлена картина вида Пустая или ИзБиблиотеки.

Использование в версии:

Доступен, начиная с версии 8.0.
