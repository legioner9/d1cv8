ЗадачаОбъект.<Имя задачи>.АктивироватьИнтерактивно (TaskObject.<Task name>.ActivateInteractively)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

АктивироватьИнтерактивно (ActivateInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

АктивироватьИнтерактивно()

Описание:

Выполняет интерактивную активацию задачи. Это приводит к возникновению события ОбработкаИнтерактивнойАктивации задачи, а потом, если разрешена стандартная обработка, события ОбработкаИнтерактивнойАктивации у точки бизнес-процесса.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
