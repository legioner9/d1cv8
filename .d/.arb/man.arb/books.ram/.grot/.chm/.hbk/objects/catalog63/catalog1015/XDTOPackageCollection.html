КоллекцияПакетовXDTO (XDTOPackageCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ПакетXDTO
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются пакеты XDTO.
Возможно обращение к пакету XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Количество (Count)
Получить (Get)

Описание:

Коллекция пакетов XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПакетXDTO, свойство Зависимости
ФабрикаXDTO, свойство Пакеты

Использование в версии:

Доступен, начиная с версии 8.1.
