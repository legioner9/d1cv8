HTTPСервисЗапрос.ПолучитьТелоКакСтроку (HTTPServiceRequest.GetBodyAsString)

HTTPСервисЗапрос (HTTPServiceRequest)

ПолучитьТелоКакСтроку (GetBodyAsString)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьТелоКакСтроку(<Кодировка>)

Параметры:

<Кодировка> (необязательный)

Тип: КодировкаТекста, Строка.
Указывает кодировку, в которой должно интерпретироваться тело полученного запроса.
Если не задан, то кодировка определяется из HTTP-заголовка "Content-Type".
Если кодировка не указана в заголовке, то используется UTF8.
Кодировка может быть задана как в виде значения перечисления КодировкаТекста, так и в виде строки с указанием названия кодировки.

Возвращаемое значение:

Тип: Строка.

Описание:

Возвращает тело запроса в виде строки.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
