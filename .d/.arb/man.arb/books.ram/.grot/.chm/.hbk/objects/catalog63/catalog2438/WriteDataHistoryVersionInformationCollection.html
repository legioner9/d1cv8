КоллекцияИнформацииОЗаписиВерсииИсторииДанных (WriteDataHistoryVersionInformationCollection)

Доступен, начиная с версии 8.3.15.

Элементы коллекции:

ИнформацияОЗаписиВерсииИсторииДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.
Возможно обращение к элементам посредством оператора [...]. В качестве аргумента передается индекс элемента коллекции (нумерация с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Получить (Get)

Описание:

Представляет коллекцию версий истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
