ПоследовательностиМенеджер (SequencesManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПоследовательностьМенеджер.<Имя последовательности>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры последовательностей.
Возможно обращение к менеджеру последовательности посредством оператора [...]. В качестве аргумента передается имя последовательности, как оно задано в конфигурации.

Свойства:

<Имя последовательности> (<Sequence name>)

Методы:

Восстановить (Restore)

Описание:

Предназначен для управления последовательностями и предоставляет доступ к значениям типа ПоследовательностьМенеджер.<Имя последовательности>. Объект также позволяет восстанавливать сразу несколько последовательностей. Доступ к объекту осуществляется через свойство глобального контекста Последовательности.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:
