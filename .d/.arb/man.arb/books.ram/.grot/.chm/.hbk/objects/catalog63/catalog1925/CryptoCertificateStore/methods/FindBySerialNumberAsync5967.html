ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НайтиПоСерийномуНомеруАсинх (FindBySerialNumberAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НайтиПоСерийномуНомеруАсинх(<СерийныйНомер>, <Издатель>)

Параметры:

<СерийныйНомер> (обязательный)

Тип: ДвоичныеДанные.
Серийный номер искомого сертификата.

<Издатель> (необязательный)

Тип: Строка.
Строка поиска издателя сертификата.
Данные издателя должны содержать переданную в этом параметре строку.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - массив объектов СертификатКриптографии.

Описание:

Асинхронная функция. Осуществляет поиск сертификатов криптографии по серийному номеру и издателю.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:
