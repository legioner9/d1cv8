ОтображениеКнопки (ButtonRepresentation)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
Картинка (Picture)
КартинкаИТекст (PictureAndText)
Текст (Text)

Описание:

Содержит варианты отображения кнопки в форме клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

КнопкаФормы, свойство Отображение
КомандаФормы, свойство Отображение
ОбъектМетаданных: Команда, свойство Отображение
ОбъектМетаданных: ГруппаКоманд, свойство Отображение
ОбъектМетаданных: ОбщаяКоманда, свойство Отображение

Использование в версии:

Доступен, начиная с версии 8.2.
