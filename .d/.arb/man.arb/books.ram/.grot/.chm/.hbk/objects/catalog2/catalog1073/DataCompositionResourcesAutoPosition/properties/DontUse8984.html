АвтоПозицияРесурсовКомпоновкиДанных (DataCompositionResourcesAutoPosition)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.2.

Описание:

Поля-ресурсы выводятся в той позиции, где они расположены, т.е. располагаются также, как и другие поля.

Использование в версии:

Доступен, начиная с версии 8.2.
