СтатусВыгрузкиДанныхСистемыВзаимодействия (CollaborationSystemDataDumpStatus)

Доступен, начиная с версии 8.3.22.

Значения

Восстановление (Restoring)
Готово (Done)
Загрузка (Loading)
Ошибка (Error)
Создание (Creating)

Описание:

Содержит варианты статусов выгрузки данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/ecs/core. Имя типа XDTO:
