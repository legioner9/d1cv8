Расширение формы набора записей регистра накопления (Accumulation register record set form extension)

Расширение формы набора записей регистра накопления (Accumulation register record set form extension)

Доступен, начиная с версии 8.0.

Методы:

ЗаписатьВФорме (WriteInForm)

События:

ПередЗаписью (BeforeWrite)
ПослеЗаписи (AfterWrite)
ПриЗаписи (OnWrite)

Описание:

Предоставляет доступ к дополнительным методам и событиям объекта Форма в том случае, если набор записей регистра расчета является основным реквизитом формы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
