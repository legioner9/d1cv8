ОтдельноИТолькоВИтогах (SeparatelyAndInTotalsOnly)

Доступен, начиная с версии 8.0.

Описание:

Каждое измерение располагается в отдельной строке/колонке, а подписи располагаются только в итоговой строке.

Использование в версии:

Доступен, начиная с версии 8.0.
