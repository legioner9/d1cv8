ТестируемаяТаблицаФормы (TestedFormTable)

ПолучитьТекстЯчейки (GetCellText)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьТекстЯчейки(<Ячейка>)

Параметры:

<Ячейка> (необязательный)

Тип: Строка.
Имя элемента или индекс.

Возвращаемое значение:

Если параметр <Ячейка> не указан, то возвращает значение текущей ячейки, в которой находится курсор.

Описание:

Возвращает текст ячейки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
