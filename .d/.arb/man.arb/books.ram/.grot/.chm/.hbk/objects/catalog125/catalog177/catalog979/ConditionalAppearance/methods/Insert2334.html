УсловноеОформление.Вставить (ConditionalAppearance.Insert)

УсловноеОформление (ConditionalAppearance)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Имя>, <Представление>, <Позиция>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого оформления области.

<Представление> (необязательный)

Тип: Строка.
Представление добавляемого оформления.

<Позиция> (обязательный)

Тип: Число.
Позиция, в которую необходимо добавить оформление.

Возвращаемое значение:

Тип: ЭлементУсловногоОформления.

Описание:

Вставить область в указанную позицию коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
