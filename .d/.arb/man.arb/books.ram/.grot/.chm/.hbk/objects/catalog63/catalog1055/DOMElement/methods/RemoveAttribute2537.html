ЭлементDOM.УдалитьАтрибут (DOMElement.RemoveAttribute)

ЭлементDOM (DOMElement)

УдалитьАтрибут (RemoveAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

УдалитьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя атрибута.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

УдалитьАтрибут(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен удаляемого атрибута.

<ЛокальноеИмя> (обязательный)

Тип: Строка.
Локальное имя удаляемого атрибута.

Описание варианта метода:

В случае удаления атрибута не явно указанного в элементе, а взятого из схемы документа, новый узел Атрибут будет создан и в него будет установлено значение по умолчанию.
Причины вызова исключений:
  • данный узел в режиме Только для чтения.

Описание:

Удаляет атрибут из элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если данный узел в режиме Только для чтения, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
