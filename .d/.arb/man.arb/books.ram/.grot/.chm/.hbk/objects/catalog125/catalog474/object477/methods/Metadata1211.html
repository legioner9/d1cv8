ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: ПланВидовРасчета.

Описание:

Предоставляет доступ к объекту метаданных плана видов расчета. Другой путь получения того же значения - через свойство глобального контекста Метаданные. Например: Метаданные.ПланыВидовРасчета.ОсновныеНачисления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
