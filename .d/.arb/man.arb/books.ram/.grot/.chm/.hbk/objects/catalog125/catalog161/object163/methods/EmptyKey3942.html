РегистрСведенийМенеджер.<Имя регистра сведений> (InformationRegisterManager.<Information register name>)

ПустойКлюч (EmptyKey)

Доступен, начиная с версии 8.2.

Синтаксис:

ПустойКлюч()

Возвращаемое значение:

Тип: РегистрСведенийКлючЗаписи.<Имя регистра сведений>.

Описание:

Получает пустое значение ключа записи регистра сведений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
