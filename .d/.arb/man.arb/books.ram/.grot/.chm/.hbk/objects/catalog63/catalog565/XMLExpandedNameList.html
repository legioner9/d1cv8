СписокРасширенныхИменXML (XMLExpandedNameList)

Доступен, начиная с версии 8.1.

Элементы коллекции:

РасширенноеИмяXML
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются расширенные имена XML.
Возможно обращение к расширенному имени XML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)
Установить (Set)

Описание:

Коллекция расширенных имен XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
