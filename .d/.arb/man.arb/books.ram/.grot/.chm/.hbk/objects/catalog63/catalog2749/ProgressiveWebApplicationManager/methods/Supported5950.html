МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

Поддерживается (Supported)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Поддерживается()

Возвращаемое значение:

Тип: Булево.
В веб-клиенте возвращает значение Истина, если выполнены все условия:
  • веб-клиент опубликован по протоколу HTTPS (с валидным сертификатом),
  • веб-клиент запущен в одном из браузеров:
    • Google Chrome, начиная с версии 73,
    • Яндекс.Браузер, начиная с версии 73 Chromium,
    • Microsoft Edge, начиная с версии 73 Chromium.
Для тонкого, толстого, мобильного клиентов возвращаемое значение всегда Ложь.

Описание:

Определяет, поддерживается ли установка прогрессивного веб-приложения в веб-клиенте.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Описание изменено в версии 8.3.22.
