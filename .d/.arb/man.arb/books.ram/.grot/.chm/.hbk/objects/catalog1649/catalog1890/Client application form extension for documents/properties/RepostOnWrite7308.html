Расширение формы клиентского приложения для документа (Client application form extension for documents)

ПриЗаписиПерепроводить (RepostOnWrite)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак режима записи проведенного документа. Если установлено значение Истина, проведение будет автоматически выполняться при записи проведенного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

На клиенте, тонком клиенте и Веб-клиенте свойство доступно только для чтения.

Использование в версии:

Доступен, начиная с версии 8.2.
