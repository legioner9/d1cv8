РегистрСведенийКлючЗаписи.<Имя регистра сведений> (InformationRegisterRecordKey.<Information register name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.3.11.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: РегистрСведений.

Описание:

Предоставляет доступ к объекту описания метаданных регистра сведений. Другой путь получения того же значения - через свойство глобального контекста Метаданные. Например: Метаданные.РегистрыСведений.КурсыВалют.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
