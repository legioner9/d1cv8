СущностьDOM.СравнитьПозициюВДокументе (DOMEntity.CompareDocumentPosition)

СущностьDOM (DOMEntity)

СравнитьПозициюВДокументе (CompareDocumentPosition)

Доступен, начиная с версии 8.1.

Синтаксис:

СравнитьПозициюВДокументе(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: АтрибутDOM, ДокументDOM, ЭлементDOM, ОпределениеТипаДокументаDOM, НотацияDOM, СущностьDOM, ФрагментДокументаDOM, ТекстDOM, КомментарийDOM, СекцияCDATADOM, ИнструкцияОбработкиDOM, СсылкаНаСущностьDOM, ПространствоИменXPath.
Узел DOM, с которым происходит сравнение.

Возвращаемое значение:

Тип: ПозицияВДокументеDOM.

Описание:

Сравнивает данный узел с узлом, переданным в качестве параметра, для определения позиции относительно документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
