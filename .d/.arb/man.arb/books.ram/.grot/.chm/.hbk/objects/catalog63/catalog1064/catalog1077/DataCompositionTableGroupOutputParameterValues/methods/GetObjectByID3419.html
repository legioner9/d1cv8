ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных (DataCompositionTableGroupOutputParameterValues)

ПолучитьОбъектПоИдентификатору (GetObjectByID)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбъектПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: ИдентификаторКомпоновкиДанных.
Идентификатор объекта.

Возвращаемое значение:

Тип: ЗначениеПараметраКомпоновкиДанных, Неопределено.
Неопределено - объект, для которого был передан идентификатор, не найден или если он не является частью данного объекта.

Описание:

Получает значения параметров вывода группировки таблицы компоновки данных, соответствующие переданному идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
