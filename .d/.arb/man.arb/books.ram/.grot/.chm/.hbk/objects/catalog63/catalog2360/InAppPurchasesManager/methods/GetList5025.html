МенеджерВстроенныхПокупок.ПолучитьСписок (InAppPurchasesManager.GetList)

МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьСписок (GetList)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПолучитьСписок(<ИдентификаторыПокупок>, <ДоступныеДляПокупки>)

Параметры:

<ИдентификаторыПокупок> (обязательный)

Тип: Массив.
Массив идентификаторов покупок, информация о которых получается.

<ДоступныеДляПокупки> (необязательный)

Тип: Булево.
Если установлено значение Истина, в списке покупок отображаются только покупки, коорые можно приобрести в данный момент. Приобретенные постоянные покупки и активные подписки из списка исключаются.
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ВстроеннаяПокупка.

Описание:

Возвращает список покупок, доступных в приложении. Для работы данного метода наличие соединения с сетью Интернет не требуется. Данные получаются из истории покупок, хранящейся локально на устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
