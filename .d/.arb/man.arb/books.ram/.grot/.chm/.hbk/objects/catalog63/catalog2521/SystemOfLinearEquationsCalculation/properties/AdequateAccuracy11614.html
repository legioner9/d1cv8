РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ТребуемаяТочность (AdequateAccuracy)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Максимальная погрешность искомого решения, рассчитывается как ошибка, полученная в результате подстановки решения.
При достижении требуемой точности расчет останавливается.
Значение по умолчанию – 0.000001.
Корректное значение – любое положительное число.
Не рекомендуется устанавливать значение меньше реально необходимого – вычисления могут занимать длительное время.
Оптимальные значения:
  • 0.1-1 для целых чисел,
  • 0.0000001-0.001 для денежных значений.
Рекомендуется для получения верного i-того знака после запятой выставлять точность 10^(-(i+1)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
