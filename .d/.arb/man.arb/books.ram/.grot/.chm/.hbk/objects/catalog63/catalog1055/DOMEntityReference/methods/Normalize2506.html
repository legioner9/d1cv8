СсылкаНаСущностьDOM.Нормализовать (DOMEntityReference.Normalize)

СсылкаНаСущностьDOM (DOMEntityReference)

Нормализовать (Normalize)

Доступен, начиная с версии 8.1.

Синтаксис:

Нормализовать()

Описание:

Приводит все узлы Текст на полной глубине поддерева данного узла, включая узлы Атрибут, в "нормальную" форму, где только структуры (например, элементы, комментарии, инструкции обработки, секции CDATA и ссылки на сущности) разделяются текстовыми узлами, т.е. смежные текстовые узлы отсутствуют.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
