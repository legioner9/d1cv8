ПолучитьДопустимыеСтраны (GetAvailableCountries)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьДопустимыеСтраны()

Возвращаемое значение:

Тип: ФиксированныйМассив.
Фиксированный массив объектов типа ДопустимаяСтранаПолученияЛицензий, отсортированный по представлениям.

Описание:

Возвращает список допустимых стран, отсортированный по наименованиям. Предназначен только для построения подсказки вариантов значений свойства Страна объекта ЗапросНаПолучениеЛицензии и не может считаться исчерпывающим списком стран при реализации бизнес-логики.

Доступность:

Тонкий клиент, сервер, внешнее соединение.
