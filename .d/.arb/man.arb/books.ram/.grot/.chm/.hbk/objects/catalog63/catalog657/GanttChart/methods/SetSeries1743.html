ДиаграммаГанта.УстановитьСерию (GanttChart.SetSeries)

ДиаграммаГанта (GanttChart)

УстановитьСерию (SetSeries)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСерию(<Значение>, <ЗначениеРодителя>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.
Значение серии.

<ЗначениеРодителя> (необязательный)

Тип: Произвольный.
Значение родительской серии.

Возвращаемое значение:

Тип: СерияДиаграммыГанта.

Описание:

Устанавливает серию диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если серия с ключом <Значение> существует, она возвращается вне зависимости от параметра <ЗначениеРодителя>. Если серия с ключом <Значение> не существует, но существует серия с ключом <ЗначениеРодителя>, то создаётся новая серия с ключом <Значение>, подчинённая серия с ключом <ЗначениеРодителя>. Иначе (т.е. не существуют серии с ключами <Значение> и <ЗначениеРодителя>), создаётся новая серия с ключом <Значение> на верхнем уровне (т.е. серия с ключом <ЗначениеРодителя> автоматически не создаётся никогда).

Использование в версии:

Доступен, начиная с версии 8.0.
