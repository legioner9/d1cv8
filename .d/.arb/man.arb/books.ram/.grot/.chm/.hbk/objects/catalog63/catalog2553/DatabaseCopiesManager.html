МенеджерКопийБазыДанных (DatabaseCopiesManager)

МенеджерКопийБазыДанных (DatabaseCopiesManager)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

МенеджерКопииБазыДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются объекты типа МенеджерКопииБазыДанных.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя копии.

Методы:

Добавить (Add)
Количество (Count)
Найти (Find)
Обновить (Update)
ПолучитьВерсиюСтандартнойРепликации (GetStandardReplicationVersion)
Удалить (Delete)
УстановитьВерсиюСтандартнойРепликации (SetStandardReplicationVersion)

Описание:

Предназначен для взаимодействия с копиями базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Копия1 = КопииБазыДанных.Добавить("Копия1", ТипРепликацииКопииБазыДанных.Стандартная, ТипСУБДКопииБазыДанных.PostgreSQL, "localhost:5432", "DbCopy1", "postgres", "postgres");

Копия1.Состав.Добавить(Метаданные.Справочники.Товары);
Копия1.Состав.Добавить(Метаданные.Документы.РасходТовара);
Копия1.Состав.Добавить(Метаданные.РегистрыНакопления.ТоварныеЗапасы);
Копия1.Записать();

КопииБазыДанных.Обновить(Копия1);

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.21.
