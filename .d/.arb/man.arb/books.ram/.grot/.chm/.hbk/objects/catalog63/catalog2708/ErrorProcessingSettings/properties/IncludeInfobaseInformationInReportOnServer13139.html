НастройкиОбработкиОшибок (ErrorProcessingSettings)

ВключатьСведенияОбИнформационнойБазеВОтчетНаСервере (IncludeInfobaseInformationInReportOnServer)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: РежимОтправкиИнформацииОбОшибке.
Указывает необходимость добавлять в отчет об ошибке на сервере сведения об информационной базе.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - Авто.

Использование в версии:

Доступен, начиная с версии 8.3.21.
