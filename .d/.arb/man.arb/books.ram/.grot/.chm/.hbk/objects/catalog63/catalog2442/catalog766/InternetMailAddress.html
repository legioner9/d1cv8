ИнтернетПочтовыйАдрес (InternetMailAddress)

Доступен, начиная с версии 8.0.

Свойства:

Адрес (Address)
Кодировка (Encoding)
ОтображаемоеИмя (DisplayName)
Пользователь (User)
Сервер (Server)

Описание:

Адрес для обмена почтовыми сообщениями.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Сообщение = Новый ИнтернетПочтовоеСообщение;
Адрес = Сообщение.Получатели.Добавить("user@server.ru");
Адрес.ОтображаемоеИмя="Иванов Иван Иванович";
