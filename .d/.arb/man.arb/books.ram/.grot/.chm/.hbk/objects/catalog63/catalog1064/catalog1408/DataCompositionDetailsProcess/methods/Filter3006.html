ОбработкаРасшифровкиКомпоновкиДанных (DataCompositionDetailsProcess)

Отфильтровать (Filter)

Доступен, начиная с версии 8.1.

Синтаксис:

Отфильтровать(<ИдентификаторРасшифровки>, <Поле>)

Параметры:

<ИдентификаторРасшифровки> (обязательный)

Тип: ИдентификаторРасшифровкиКомпоновкиДанных.
Идентификатор обрабатываемой расшифровки.

<Поле> (обязательный)

Тип: ПолеКомпоновкиДанных.
Поле, по значению которого должен быть отфильтрован результат.

Возвращаемое значение:

Тип: НастройкиКомпоновкиДанных, ПользовательскиеНастройкиКомпоновкиДанных.
Если в настройках используются пользовательские элементы (см. свойства ИдентификаторПользовательскойНастройки различных объектов) и изменения настроек затрагивают только пользовательские элементы, будут возвращены ПользовательскиеНастройкиКомпоновкиДанных, иначе НастройкиКомпоновкиДанных.

Описание:

Перестраивает настройки, позволяя отфильтровать результат по заданному полю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
