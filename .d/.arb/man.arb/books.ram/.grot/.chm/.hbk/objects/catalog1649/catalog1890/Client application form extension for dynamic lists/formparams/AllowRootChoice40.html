Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

РазрешитьВыборКорня (AllowRootChoice)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.
Определяет возможность выбора корня в форме с динамическим списком, отображаемом в виде дерева.
При открытии формы списка со значением параметра Истина и параметром РежимВыбора=Истина для всех таблиц, связанных с основным реквизитом типа ДинамическийСписок, автоматически будут переопределены значения некоторых свойств:
  • свойство ОтображатьКорень будет установлено в Истина;
  • свойство РазрешитьВыборКорня будет установлено в Истина;
  • свойство Отображение будет установлено в Дерево, если оно было не Дерево.
Команда "Переместить в группу" при открытии формы передает параметр со значением Истина.
В конструкторах форм списков для всех списков, связанных с основным реквизитом, свойству НачальноеОтображениеДерева устанавливается значение РаскрыватьВерхнийУровень.

Использование в версии:

Доступен, начиная с версии 8.2.
