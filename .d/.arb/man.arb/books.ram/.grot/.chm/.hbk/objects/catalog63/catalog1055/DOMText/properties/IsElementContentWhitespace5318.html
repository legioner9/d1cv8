Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит признак пробельных символов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - текст является пробельными символами узла Элемент. Ложь - в противном случае (игнорируемые пробельные символы).

Доступен, начиная с версии 8.1.
