ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: ПерерасчетЗапись.<Имя перерасчета>.
Запись, индекс которой требуется определить.

Возвращаемое значение:

Тип: Число.
Индекс указанной записи в наборе. Если не найдено, то возвращается -1.

Описание:

Получает индекс записи в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
