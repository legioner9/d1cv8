ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

ПрикрепляемыеДанные (ClipData)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства, Неопределено.
Содержит прикрепляемые данные, связанные с этим запуском приложения мобильного устройства. Прикрепляемые данные в запуске не используются для уточнения поиска мобильного приложения.
Семантически похоже на свойство
ДополнительныеДанные, используемое для передачи дополнительных данных с помощью ЗапускПриложенияМобильногоУстройства. Основная особенность использования этого свойства в сравнении со свойством ДополнительныеДанные заключается в том, что системные флаги Intent.FLAG_GRANT_READ_URI_PERMISSION и Intent.FLAG_GRANT_WRITE_URI_PERMISSION, которые всегда устанавливаются объекту ЗапускПриложенияМобильногоУстройства, будут работать с любыми элементами со ссылками на ресурсы, включенными в прикрепляемые данные.
Таким образом, при запуске мобильного приложения можно передать несколько ссылок на ресурсы со схемой "content:", для которых получатель может не иметь глобального разрешения на доступ к поставщику контента. Если данное свойство содержит элементы, которые сами являются объектами ЗапускПриложенияМобильногоУстройства, любые установленные флаги в этих запусках будут проигнорированы.
Учитываются только флаги основного объекта ЗапускПриложенияМобильногоУстройства(верхнего уровня) и они будут применены ко всем элементам со ссылками на ресурсы или объектам ЗапускПриложенияМобильногоУстройства в ПрикрепляемыхДанных (или под элементами ПрикрепляемыхДанных).
Тип, метка и значок MIME в свойстве ПрикрепляемыеДанные напрямую не используются объектом ЗапускПриложенияМобильногоУстройства. Приложения обычно должны учитывать MIME-тип самого объекта ЗапускПриложенияМобильногоУстройства, а не то, что находится в прикрепляемых данных.
Обычно для передачи объекта ЗапускПриложенияМобильногоУстройства следует создать объект ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства с типом MIME "*/*".
Аналог методов setClipData и getClipData у класса Intent в операционной системе Android.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Значение по умолчанию - Неопределено.

Использование в версии:

Доступен, начиная с версии 8.3.23.
