МенеджерКриптографии.ПолучитьХранилищеСертификатовАсинх (CryptoManager.GetCertificateStoreAsync)

МенеджерКриптографии (CryptoManager)

ПолучитьХранилищеСертификатовАсинх (GetCertificateStoreAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьХранилищеСертификатовАсинх(<ТипХранилища>, <РасположениеХранилища>)

Параметры:

<ТипХранилища> (необязательный)

Тип: ТипХранилищаСертификатовКриптографии.
Определяет тип хранилища сертификатов, который должен вернуть метод.
Если параметр не задан, возвращает хранилище, содержащее сертификаты всех доступных типов.

<РасположениеХранилища> (необязательный)

Тип: РасположениеХранилищаСертификатовКриптографии.
Определяет, к какому хранилищу предоставить доступ: к хранилищу сертификатов пользователя операционной системы или к хранилищу сертификатов локального компьютера.
Значение по умолчанию: ДанныеПользователяОС.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект ХранилищеСертификатовКриптографии.

Описание:

Асинхронная функция. Получает хранилище сертификатов, содержащее сертификаты заданного типа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МенеджерКриптографии, метод НачатьПолучениеХранилищаСертификатов
МенеджерКриптографии, метод ПолучитьХранилищеСертификатов

Использование в версии:

Доступен, начиная с версии 8.3.18.
