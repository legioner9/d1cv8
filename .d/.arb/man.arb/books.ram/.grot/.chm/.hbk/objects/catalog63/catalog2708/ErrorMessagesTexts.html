ТекстыСообщенийОбОшибках (ErrorMessagesTexts)

Доступен, начиная с версии 8.3.17.

Элементы коллекции:

КлючИЗначение
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции КлючИЗначение. Ключом является значение типа КатегорияОшибки, значением - ТекстыСообщенияОбОшибке.
Возможно обращение к значениям посредством оператора [...]. В качестве аргумента передается значение типа КатегорияОшибки.

Методы:

Вставить (Insert)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Содержит тексты сообщений об ошибках по категориям.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.22.
