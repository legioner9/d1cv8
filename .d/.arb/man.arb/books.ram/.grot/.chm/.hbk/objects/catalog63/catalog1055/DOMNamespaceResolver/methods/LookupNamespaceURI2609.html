РазыменовательПространствИменDOM.НайтиURIПространстваИмен (DOMNamespaceResolver.LookupNamespaceURI)

РазыменовательПространствИменDOM (DOMNamespaceResolver)

НайтиURIПространстваИмен (LookupNamespaceURI)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиURIПространстваИмен(<Префикс>)

Параметры:

<Префикс> (обязательный)

Тип: Строка.
Префикс пространства имен.

Возвращаемое значение:

Тип: Строка, Неопределено.
URI пространства имен, соответствующее указанному префиксу, или Неопределено.

Описание:

Производит поиск URI пространства имен для указанного префикса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
