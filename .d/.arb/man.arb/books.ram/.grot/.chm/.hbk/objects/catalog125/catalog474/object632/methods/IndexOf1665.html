ВедущиеВидыРасчета.<Имя плана видов расчета> (LeadingCalculationTypes.<Chart of calculation types name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: ВедущиеВидыРасчетаСтрока.<Имя плана видов расчета>.
Строка табличной части, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: Число.
Индекс указанной строки в табличной части. Если не найдено, то возвращается -1.

Описание:

Получает индекс строки в табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
