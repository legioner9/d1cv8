ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

ОписаниеСистемыЛинейныхУравнений
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.
Возможно обращение к к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Представляет собой коллекцию объектов типа ОписаниеСистемыЛинейныхУравнений.

Доступность:

Сервер, толстый клиент, внешнее соединение.
