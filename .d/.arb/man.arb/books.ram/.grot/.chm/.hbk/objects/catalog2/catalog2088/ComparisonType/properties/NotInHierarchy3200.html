ВидСравнения.НеВИерархии (ComparisonType.NotInHierarchy)

ВидСравнения (ComparisonType)

НеВИерархии (NotInHierarchy)

Доступен, начиная с версии 8.0.

Описание:

Проверка того, что сравниваемое значение не принадлежит иерархии заданного значения.
Имеет смысл при сравнении для иерархических объектов (справочников, планов видов характеристик, планов счетов).

Использование в версии:

Доступен, начиная с версии 8.0.
