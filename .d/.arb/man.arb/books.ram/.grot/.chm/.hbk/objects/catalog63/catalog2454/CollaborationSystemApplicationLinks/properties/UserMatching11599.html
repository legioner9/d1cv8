СовместноеИспользованиеПриложенийСистемыВзаимодействия (CollaborationSystemApplicationLinks)

СопоставлениеПользователей (UserMatching)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Задает способ сопоставления пользователей приложений системы взаимодействия.
Возможные значения:
  • Имя (Name) - сопоставление выполняется по именам пользователей,
  • ПолноеИмя (FullName) - сопоставление выполняется по полным именам пользователей,
  • КлючСопоставления (MatchingKey) - сопоставление выполняется по ключам сопоставления пользователей,
  • Пустая строка - сопоставление пользователей не выполняется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
