На основе набора схем XML
По умолчанию

Описание:

Фабрика типов XDTO.
Содержит определения всех типов, позволяет осуществлять чтения/запись данных XDTO в XML и JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
