КоллекцияВстроенныхТаблиц.Найти (EmbeddedTableCollection.Find)

КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяТаблицы>)

Параметры:

<ИмяТаблицы> (обязательный)

Тип: Строка.
Имя встроенной таблицы, как оно задано в табличном документе.

Возвращаемое значение:

Тип: СводнаяТаблица, Неопределено.
Если не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск сводной таблицы по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
