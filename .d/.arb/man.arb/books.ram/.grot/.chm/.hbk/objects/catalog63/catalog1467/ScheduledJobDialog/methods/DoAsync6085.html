ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Расписание, если диалог был закрыт нажатием кнопки "OK", Неопределено - в противном случае.

Описание:

Асинхронная функция. Открывает немодальный диалог расписания.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
