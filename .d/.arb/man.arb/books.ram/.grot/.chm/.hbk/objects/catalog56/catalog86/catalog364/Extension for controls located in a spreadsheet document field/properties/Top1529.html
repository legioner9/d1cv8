Расширение элементов управления, расположенных в поле табличного документа (Extension for controls located in a spreadsheet document field)

Верх (Top)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит положение верхней границы элемента управления, отсчитываемое от верхней границы табличного документа (в миллиметрах).

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
