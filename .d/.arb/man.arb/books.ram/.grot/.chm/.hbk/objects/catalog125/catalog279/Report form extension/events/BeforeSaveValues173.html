Расширение формы отчета (Report form extension)

ПередСохранениемЗначений (BeforeSaveValues)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСохранениемЗначений(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.
Признак сохранения значений формы. Если в теле процедуры-обработчика установить данному параметру значение Истина, сохранение не будет выполнено.
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением сохранения значений формы из списка сохраняемых значений.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
