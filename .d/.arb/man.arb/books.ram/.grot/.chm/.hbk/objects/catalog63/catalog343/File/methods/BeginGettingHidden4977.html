Файл.НачатьПолучениеНевидимости (File.BeginGettingHidden)

Файл (File)

НачатьПолучениеНевидимости (BeginGettingHidden)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеНевидимости(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Содержит описание процедуры, которая будет вызвана после завершения получения атрибута "Скрытый" со следующими параметрами:
  • <Невидимый> - помещается признак невидимости, тип Булево. Истина, если файл невидимый.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Начинает получение атрибута файла "Скрытый" ("Hidden").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
