ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Содержит полное имя файла, в который нужно записать пакет отображаемых документов.

<ТипФайла> (обязательный)

Тип: ТипФайлаПакетаОтображаемыхДокументов.
Содержит формат файла, в котором будет сохранен пакет отображаемых документов.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания:
  • Истина - если запись была инициирована с помощью расширения браузера и пользователь согласился на его получение, то есть запись файла была завершена;
  • Ложь - если запись была инициирована с помощью расширения браузера, но пользователь не согласился на его получение, то есть запись файла не была завершена;
  • Неопределено - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

Описание:

Асинхронная функция. Записывает пакет отображаемых документов в файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
