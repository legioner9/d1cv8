Виртуальный каталог.Alias (ISecurityProfileVirtualDirectory.Alias)

Виртуальный каталог (ISecurityProfileVirtualDirectory)

Alias (Alias)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Логический URL каталога. Этот URL можно использовать в конфигурации для обращения к файлам и подкаталогам данного виртуального каталога. Перед сравнением логические URL приводятся к каноническому виду. Для этого выполняется: удаление лишних space символов; перевод регистронезависимых частей URL в нижний регистр; замена символов '\\' на '/' в пути к ресурсу; преобразование пути – удаление пустых секций и интерпретация секций "." и "..". Если в контексте информационной базы, области данных, сеанса или режима безопасного исполнения кода конфигурация или платформа выполняет любую операцию с файловой системой по URL, начало которого не совпадает со значением свойства Alias одного из виртуальных каталогов, то генерируется исключение. Существуют несколько предопределенных логических URL:
  • /bin – каталог загрузочных модулей текущей версии 1С:Предприятия (может быть получен методом КаталогПрограммы),
  • /temp – каталог временных файлов (может быть получен методом КаталогВременныхФайлов).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
