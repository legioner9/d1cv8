ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ОбработкаПроверкиВыполнения (OnCheckExecuteProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаПроверкиВыполнения(<ТочкаМаршрута>, <Задача>, <Результат>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса, в которой выполняется проверка завершения.

<Задача>

Тип: ЗадачаСсылка.<Имя задачи>.
Задача, при выполнении которой приходит событие.

<Результат>

Тип: Булево.
Признак завершения задачи. Если в параметре возвращается Ложь, то задача не будет отмечена как завершенная.
Значение по умолчанию: Истина.

Описание:

Возникает при вызове метода ПроверитьВыполнение у задачи. Не вызывается, если задача уже выполнена. Вызывается после вызова обработчика события ОбработкаПроверкиВыполнения у самой задачи и если в теле этого обработчика для параметра <Результат> установлено значение Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида Действие.
Допускаются интерактивные вызовы (открытие форм, предупреждений).

Использование в версии:

Доступен, начиная с версии 8.0.
