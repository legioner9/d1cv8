РедактироватьАсинх (EditAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РедактироватьАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Период - диалог был закрыт нажатием кнопки "OK"; Неопределено - в противном случае.

Описание:

Асинхронная функция. Начинает интерактивное редактирование.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
