СтрокиДинамическогоСписка (DynamicListRows)

Доступен, начиная с версии 8.3.10.

Элементы коллекции:

СтрокаДинамическогоСписка
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа КлючИЗначение, где Ключ - ключ строки, Значение - СтрокаДинамическогоСписка.
Возможно обращение к значению элемента коллекции посредством оператора [...]. В качестве аргумента передается ключ строки.

Методы:

Количество (Count)
Получить (Get)
ПолучитьКлючи (GetKeys)

Описание:

Представляет собой коллекцию строк динамического списка.
К содержимому объекта можно обращаться только из обработчика ПриПолученииДанныхНаСервере. По окончании работы обработчика попытка обращения к содержимому приведет к вызову исключения.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
