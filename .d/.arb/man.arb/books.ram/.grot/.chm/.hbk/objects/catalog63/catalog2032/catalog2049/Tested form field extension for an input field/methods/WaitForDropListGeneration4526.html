Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ОжидатьФормированияВыпадающегоСписка (WaitForDropListGeneration)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ОжидатьФормированияВыпадающегоСписка(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: Число.
Интервал, в течение которого метод проверяет, заполнен ли список данными.
Значение по умолчанию: 60.

Возвращаемое значение:

Тип: Булево.
Истина - список заполнен сформированными данными,
Ложь - список не сформировал данные или таймаут истёк.

Описание:

Ожидает окончания формирования данных выпадающего списка в течение указанного интервала времени.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
