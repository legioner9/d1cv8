Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Отбор.
Содержит объект Отбор, по которому осуществляется текущая фильтрация задач в списке. Имена элементов отбора совпадают с именами реквизитов, реквизитов адресации задачи и критериев отбора, затрагивающими задачу, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
