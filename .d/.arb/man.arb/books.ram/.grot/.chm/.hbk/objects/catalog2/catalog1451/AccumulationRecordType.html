ВидДвиженияНакопления (AccumulationRecordType)

Доступен, начиная с версии 8.1.

Значения

Приход (Receipt)
Расход (Expense)

Описание:

Виды движений регистра накопления.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise}. Имя типа XDTO: AccumulationRecordType.

См. также:
