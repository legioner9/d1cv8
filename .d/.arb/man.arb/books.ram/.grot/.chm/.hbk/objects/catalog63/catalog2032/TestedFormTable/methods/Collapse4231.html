Свернуть (Collapse)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Свернуть(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: Соответствие.
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.

Описание:

Выполняет попытку свернуть узел для текущей строки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
