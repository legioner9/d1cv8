СредстваПочты (MailTools)

Послать (Send)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Послать(<Сообщение>, <ОбрабатыватьТексты>)

Параметры:

<Сообщение> (обязательный)

Тип: ИнтернетПочтовоеСообщение.
Почтовое сообщение.
Внимание! Тип текстов почтовых сообщений РазмеченныйТекст не поддерживается.

<ОбрабатыватьТексты> (необязательный)

Тип: ОбработкаТекстаИнтернетПочтовогоСообщения.
Управляет обработкой текстов перед отправкой:
  • Обрабатывать - Локальные картинки из текста, помещаются в сообщение в виде вложений. Содержимое текста письма изменяется таким образом, чтобы теги <img> указывали на картинки из вложений.
  • НеОбрабатывать - Текст посылается без внутренней обработки.
  • Загрузка картинок при отправке HTML-документов отключена.

Значение по умолчанию: Обрабатывать.

Описание:

Отправляет почтовое сообщение.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяОтправка.
Внимание! Тип текстов почтовых сообщений РазмеченныйТекст не поддерживается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваПочты, метод ПоддерживаетсяОтправка

Использование в версии:

Доступен, начиная с версии 8.3.5.
