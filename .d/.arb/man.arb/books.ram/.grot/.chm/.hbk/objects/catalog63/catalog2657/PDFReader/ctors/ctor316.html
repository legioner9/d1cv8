ЧтениеPDF (PDFReader)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЧтениеPDF()

Описание:

Формирует неинициализированный объект ЧтениеPDF. Для инициализации объекта используется метод Открыть или НачатьОткрытие.

Использование в версии:

Доступен, начиная с версии 8.3.16.
