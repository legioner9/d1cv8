Описание:

Описывает команду поля ввода. Используется для наполнения командами контекстного меню поля ввода.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Расширение поля формы для поля ввода, событие ОбработкаФормированияКоманд
