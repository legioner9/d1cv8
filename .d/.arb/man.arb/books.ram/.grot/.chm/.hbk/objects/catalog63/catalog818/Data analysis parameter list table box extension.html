Расширение табличного поля списка параметров анализа данных (Data analysis parameter list table box extension)

Расширение табличного поля списка параметров анализа данных (Data analysis parameter list table box extension)

Доступен, начиная с версии 8.0.

Свойства:

НастройкаПараметров (ParametersSetting)

Описание:

Дополнительные свойства элемента управления ТабличноеПоле для списка параметров анализа данных. Для данного расширения свойство ТекущаяСтрока возвращает значение типа ПараметрАнализаДанных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
