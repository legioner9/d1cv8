БизнесПроцессОбъект.<Имя бизнес-процесса>.ПолучитьСсылкуНового (BusinessProcessObject.<Business process name>.GetNewObjectRef)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: БизнесПроцессСсылка.<Имя бизнес-процесса>.

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом УстановитьСсылкуНового значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

БизнесПроцессМенеджер.<Имя бизнес-процесса>, метод ПолучитьСсылку

Использование в версии:

Доступен, начиная с версии 8.0.
