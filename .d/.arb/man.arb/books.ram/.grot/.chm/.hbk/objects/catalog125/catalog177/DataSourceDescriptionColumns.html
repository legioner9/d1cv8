КолонкиОписанияИсточникаДанных (DataSourceDescriptionColumns)

Доступен, начиная с версии 8.0.

Элементы коллекции:

КолонкаОписанияИсточникаДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.
Возможно обращение к колонке описания посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

<Имя колонки> (<Column name>)

Методы:

Индекс (IndexOf)
Количество (Count)
Получить (Get)

Описание:

Содержит колонки описания источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ОписаниеИсточникаДанных, свойство Колонки

Использование в версии:

Доступен, начиная с версии 8.0.
