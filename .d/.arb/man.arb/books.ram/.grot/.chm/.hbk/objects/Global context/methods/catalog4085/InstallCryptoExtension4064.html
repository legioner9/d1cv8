Глобальный контекст (Global context)

УстановитьРасширениеРаботыСКриптографией (InstallCryptoExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

  • НачатьУстановкуРасширенияРаботыСКриптографией

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьРасширениеРаботыСКриптографией()

Описание:

Устанавливает расширение для работы с криптографией в браузер.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска толстого и тонкого клиентов, а также на клиенте мобильного приложения информирует о том, что расширение установлено.
Если для конфигурации свойство РежимИспользованияМодальности установлено в НеИспользовать, следует использовать метод НачатьУстановкуРасширенияРаботыСКриптографией.
