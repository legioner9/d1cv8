Соединение с рабочим процессом.UpdateInfoBase (IWorkingProcessConnection.UpdateInfoBase)

Соединение с рабочим процессом (IWorkingProcessConnection)

UpdateInfoBase (UpdateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateInfoBase(<ИнформационнаяБаза>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: Информационная база.
Информационная база с новыми значениями параметров. Новые значения могут иметь только следующие свойства объекта:
  • Descr,
  • SessionsDenied,
  • DBMS,
  • dbServerName,
  • dbName,
  • dbUser,
  • dbPassword,
  • DeniedFrom,
  • DeniedTo,
  • DeniedMessage,
  • DeniedParameter,
  • ExternalSessionManagerConnectionString,
  • ExternalSessionManagerRequired,
  • LicenseDistributionAllowed,
  • PermissionCode,
  • SafeModeSecurityProfileName,
  • ScheduledJobsDenied,
  • SecurityProfileName
.

Описание:

Устанавливает новые параметры существующей информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
