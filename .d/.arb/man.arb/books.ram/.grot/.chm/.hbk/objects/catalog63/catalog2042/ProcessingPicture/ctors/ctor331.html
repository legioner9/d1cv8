ОбрабатываемаяКартинка (ProcessingPicture)

На основании массива картинок и формата

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ОбрабатываемаяКартинка(<Кадры>, <Формат>)

Параметры:

<Кадры> (необязательный)

Массив с картинками, которые станут кадрами новой картинки.

<Формат> (необязательный)

Формат будущей картинки.
Если не указан, то формат устанавливется по первому кадру.

Описание:

Создает объект на основании массива картинок и указанного формата:

  • Картинки используются в качестве кадров, собирается многокадровая картинка.
  • Если в качестве элемента массива была добавлена многокадровая картинка, то каждый ее кадр будет добавлен в новую отдельно.
  • Если формат не позволяет создать много кадровую каринку, то будет сделана однокадровая с первым кадром.
