ИдентификаторОбсужденияСистемыВзаимодействия (CollaborationSystemConversationID)

Из строки

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый ИдентификаторОбсужденияСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Строковое представление идентификатора обсуждения системы взаимодействия, созданное из идентификатора обсуждения с помощью метода Строка.

Описание:

Создает идентификатор обсуждения из указанной строки.
Данный конструктор можно использовать, если нужно произвести действие над обсуждением, идентификатор которого хранится в информационной базе в виде строки.
Корректность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.11.
