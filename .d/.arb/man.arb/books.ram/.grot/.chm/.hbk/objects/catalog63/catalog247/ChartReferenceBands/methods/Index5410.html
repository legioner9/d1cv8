ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Индекс (Index)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Индекс(<Интервал>)

Параметры:

<Интервал> (обязательный)

Тип: ИнформационныйИнтервалДиаграммы.
Интервал, индекс которого в коллекции требуется получить.

Возвращаемое значение:

Тип: Число.
Индекс указанного элемента коллекции.
Если элемент не принадлежит этой коллекции, генерируется исключение.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
