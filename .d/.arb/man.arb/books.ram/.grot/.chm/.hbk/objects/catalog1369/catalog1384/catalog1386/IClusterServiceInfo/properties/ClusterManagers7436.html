Сервис кластера.ClusterManagers (IClusterServiceInfo.ClusterManagers)

Сервис кластера (IClusterServiceInfo)

ClusterManagers (ClusterManagers)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: COMSafeArray.
Содержит массив объектов Менеджер кластера, на которые назначен данный сервис. Каждый из сервисов может быть назначен только на один менеджер кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
