ЧтениеУзловDOM (DOMNodeReader)

ЭтоАтрибутПоУмолчанию (IsDefaultAttribute)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит признак включения значения текущего узла как значения по умолчанию, полученного из DTD или XMLSchema.
Имеет смысл только для узла Атрибут.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - значение атрибута было получено подстановкой значения по умолчанию из DTD или XMLSchema, Ложь - значение было указано явно.

Использование в версии:

Доступен, начиная с версии 8.1.
