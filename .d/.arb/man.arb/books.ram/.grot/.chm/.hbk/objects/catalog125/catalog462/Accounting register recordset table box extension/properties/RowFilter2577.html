Расширение табличного поля набора записей регистра бухгалтерии (Accounting register recordset table box extension)

ОтборСтрок (RowFilter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Отбор.
Позволяет установить отбор по реквизитам набора записей. Имена элементов отбора совпадают с именами колонок набора записей.

Доступность:

Толстый клиент.

Примечание:

Отбор выполняется по условию вхождения значения. Например, если в колонке есть ячейки со значением "Строка 1", "Строка 2", ... "Строка 10", "Строка 11", ... и устанавливается отбор по значению "Строка 1", то в таблицу будут отобраны строки со значениями "Строка 1", "Строка 10", "Строка 11" и т.д.

Использование в версии:

Доступен, начиная с версии 8.0.
