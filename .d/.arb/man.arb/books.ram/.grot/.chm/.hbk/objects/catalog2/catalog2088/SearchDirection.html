НаправлениеПоиска (SearchDirection)

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).

Значения

СКонца (FromEnd)
СНачала (FromBegin)

Описание:

Определяет направление поиска подстроки в строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

Глобальный контекст, метод Найти

Использование в версии:

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).
