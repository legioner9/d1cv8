СертификатКриптографии.Выгрузить (CryptoCertificate.Unload)

СертификатКриптографии (CryptoCertificate)

Выгрузить (Unload)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: В двоичные данные

Синтаксис:

Выгрузить()

Описание варианта метода:

Возвращает двоичные данные выгруженного сертификата.

Вариант синтаксиса: В файл

Синтаксис:

Выгрузить(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла, в который будет выгружен сертификат.

Описание варианта метода:

Выгружает сертификат в файл. Возвращает Неопределено.

Вариант синтаксиса: В поток

Синтаксис:

Выгрузить(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток, в который будет записан сертификат.

Описание варианта метода:

Выгружает сертификат в поток.

Возвращаемое значение:

Тип: ДвоичныеДанные, Неопределено.
Если выгрузка производится в файл или поток, то возвращает Неопределено. Если в двоичные данные, то возвращает ДвоичныеДанные.

Описание:

Выгружает сертификат.
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В случае, если сертификат связан с закрытым ключом, выгрузка закрытого ключа не производится.
