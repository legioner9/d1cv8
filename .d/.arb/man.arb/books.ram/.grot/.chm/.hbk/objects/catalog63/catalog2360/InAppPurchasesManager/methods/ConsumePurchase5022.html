МенеджерВстроенныхПокупок (InAppPurchasesManager)

ИзрасходоватьПокупку (ConsumePurchase)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ИзрасходоватьПокупку(<Покупка>)

Параметры:

<Покупка> (обязательный)

Тип: ВстроеннаяПокупка, Строка.
Покупка, которую необходимо израсходовать.

Возвращаемое значение:

Тип: Булево.

Описание:

Покупка помечается как потраченная:

  • На Android - метод удаляет информацию о том, что покупка уже приобреталась, выполняя запрос к серверу-магазину, после чего она снова становится доступной для приобретения;
  • На iOS генерируется исключение.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяРасходованиеПокупок.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Внимание! Все покупки требуют подтверждения, в том числе подписки. В противном случае, покупка будет аннулирована и деньги возвращены. Для расходуемых покупок убедитесь, что вы используете этот метод, который действует как неявное подтверждение покупки, или вы можете явно подтвердить покупку с помощью метода ПодтвердитьПокупку.

См. также:

МенеджерВстроенныхПокупок, метод ПоддерживаетсяРасходованиеПокупок
