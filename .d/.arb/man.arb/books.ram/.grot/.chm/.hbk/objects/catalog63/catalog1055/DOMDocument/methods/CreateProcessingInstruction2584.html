ДокументDOM.СоздатьИнструкциюОбработки (DOMDocument.CreateProcessingInstruction)

ДокументDOM (DOMDocument)

СоздатьИнструкциюОбработки (CreateProcessingInstruction)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьИнструкциюОбработки(<Цель>, <Данные>)

Параметры:

<Цель> (обязательный)

Тип: Строка.
Цель инструкции обработки.

<Данные> (обязательный)

Тип: Строка.
Данные инструкции обработки.

Возвращаемое значение:

Тип: ИнструкцияОбработкиDOM.

Описание:

Создает узел инструкции обработки DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
