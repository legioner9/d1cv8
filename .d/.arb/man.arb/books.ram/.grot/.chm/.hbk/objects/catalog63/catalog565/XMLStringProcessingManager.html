МенеджерОбработкиСтрокиXML (XMLStringProcessingManager)

Доступен, начиная с версии 8.3.20.

Методы:

НайтиНедопустимыеСимволыXML (FindDisallowedXMLCharacters)

Описание:

Предназначен для работы с текстовым представлением XML-документов. Позволяет обнаружить и удалить из текстового представления символы, недопустимые в XML-документе (не соответствующие стандарту XML 1.0 или 1.1).
Доступ к объекту осуществляется через свойство глобального контекста ОбработкаСтрокиXML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.
