БизнесПроцессыМенеджер (BusinessProcessesManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

БизнесПроцессМенеджер.<Имя бизнес-процесса>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры бизнес-процессов.
Возможно обращение к менеджеру бизнес-процесса посредством оператора [...]. В качестве аргумента передается имя бизнес-процесса, как оно задано в конфигурации.

Свойства:

<Имя бизнес-процесса> (<Business process name>)

Методы:

ТипВсеСсылки (AllRefsType)
ТипВсеСсылкиТочекМаршрутаБизнесПроцессов (RoutePointsAllRefsType)

Описание:

Предназначен для управления бизнес-процессами и предоставляет доступ к значениям типа БизнесПроцессыМенеджер. Доступ к объекту осуществляется через свойство глобального контекста БизнесПроцессы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.

Пример:

// менеджер бизнес-процессов предоставляет доступ к менеджеру

// бизнес-процесса ВыпискаСчета.

БизнесПроцесс = БизнесПроцессы.ВыпискаСчета.СоздатьБизнесПроцесс();

См. также:

Глобальный контекст, свойство БизнесПроцессы

Использование в версии:

Доступен, начиная с версии 8.0.
