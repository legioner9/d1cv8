КоллекцияВстроенныхТаблиц.Добавить (EmbeddedTableCollection.Add)

КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ТипВстроеннойТаблицы>, <Область>)

Параметры:

<ТипВстроеннойТаблицы> (обязательный)

Тип: Тип.
Допускается только тип СводнаяТаблица.

<Область> (необязательный)

Тип: ОбластьЯчеекТабличногоДокумента.
Встроенная таблица будет располагаться в области, координаты левого верхнего угла которой совпадают с соответствующими координатами указанной в параметре области.

Возвращаемое значение:

Тип: СводнаяТаблица.

Описание:

Добавляет встроенную таблицу в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

СводТаб = ТабДокумент.ВстроенныеТаблицы.Добавить(Тип("СводнаяТаблица"));

Использование в версии:

Доступен, начиная с версии 8.0.
