КоллекцияВложенийPDF.Добавить (PDFAttachmentCollection.Add)

КоллекцияВложенийPDF (PDFAttachmentCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Добавить(<ИмяФайла>, <Содержимое>, <ТипСодержимого>, <ТипСвязи>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя создаваемого файла-вложения.

<Содержимое> (обязательный)

Тип: ДвоичныеДанные.
Содержимое создаваемого вложения.

<ТипСодержимого> (необязательный)

Тип: Строка.
Если не указан, то используется тип "application/octet-stream".

<ТипСвязи> (необязательный)

Тип: ТипСвязиВложенияPDF.
Если не указан, то используется НеУстановлено.

Описание:

Добавляет элемент в коллекцию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
