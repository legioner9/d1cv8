ОператорыСхемыЗапроса (QuerySchemaOperators)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ОператорВыбратьСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются операторы выбрать схемы запроса.
Возможно обращение к оператору выбрать схемы запроса посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция операторов языка запросов.
Каждый оператор представляет собой запрос ВЫБРАТЬ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
