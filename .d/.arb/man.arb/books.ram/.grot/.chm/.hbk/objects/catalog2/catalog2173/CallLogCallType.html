ТипЗвонкаЖурналаЗвонков (CallLogCallType)

Доступен, начиная с версии 8.3.5.

Значения

Входящий (Incoming)
Исходящий (Outgoing)
Пропущенный (Missed)

Описание:

Тип звонка из записи журнала звонков.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/data/mob. Имя типа XDTO: CallLogCallType.

Использование в версии:

Доступен, начиная с версии 8.3.5.
