Расширение поля формы для поля картинки (Form field extension for a picture field)

КартинкаЗначений (ValuesPicture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Картинка.
Поле вида ПолеКартинки может отображать данные типов Число и Булево. В этом случае значение представляет собой индекс в наборе картинок, задаваемых этим свойством. Для типа Булево значение Истина означает индекс 0, а значение Ложь означает индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида Пустая или ИзБиблиотеки.

Использование в версии:

Доступен, начиная с версии 8.2.
