ТестируемаяГруппаФормы (TestedFormGroup)

ТекущаяДоступность (CurrentEnable)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущаяДоступность()

Возвращаемое значение:

Тип: Булево.
Истина - объект доступен; Ложь в противном случае.

Описание:

Возвращает признак того, что пользователь может активизировать элемент без настройки формы, но путем переключения закладок и полос прокрутки.
Если объект является колонкой таблицы и таблица не находится в режиме редактирования, при вызове метода генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
