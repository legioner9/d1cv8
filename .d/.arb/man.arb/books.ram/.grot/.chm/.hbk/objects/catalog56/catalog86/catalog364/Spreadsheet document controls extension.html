Расширение элементов управления принадлежащих табличному документу (Spreadsheet document controls extension)

Доступен, начиная с версии 8.0.

Методы:

ПолучитьДействие (GetAction)
УстановитьДействие (SetAction)

Описание:

Расширение для элементов управления, встроенных в ячейку, или рисунок табличного документа (свойство ячейки ЭлементУправления).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
