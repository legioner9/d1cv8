ИнтервалДиаграммыГанта (GanttChartInterval)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СвязьДиаграммыГанта
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются связи.
Возможно обращение к связям посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

Значение (Value)
Идентификатор (ID)
Конец (End)
Начало (Begin)
Расшифровка (Details)
Текст (Text)
Цвет (Color)
ЦветТекста (TextColor)

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Удалить (Delete)

Описание:

Используется для доступа к свойствам значения интервала диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
