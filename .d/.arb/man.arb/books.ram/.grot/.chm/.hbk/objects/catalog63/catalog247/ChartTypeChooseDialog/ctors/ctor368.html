ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ДиалогВыбораТипаДиаграммы()

Описание:

Создает новый диалог выбора типа диаграммы.
По умолчанию свойство ДоступныеТипыДиаграммы содержит коллекцию со всеми типами диаграммы, и свойство ТипДиаграммы имеет значение ГистограммаОбъемная.

Пример:

ВыборТипаДиаграммы = Новый ДиалогВыбораТипаДиаграммы;

Использование в версии:

Доступен, начиная с версии 8.3.23.
