Соединение.dbProcInfo (IInfoBaseConnectionInfo.dbProcInfo)

Соединение (IInfoBaseConnectionInfo)

dbProcInfo (dbProcInfo)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Если в момент получение списка соединений информационной базы методом GetInfoBaseConnections данное соединение выполняло обращение к серверу баз данных, то свойство содержит идентификатор процесса соединения с СУБД, выполняющего это обращение. Идентификатор выдается в терминах самой СУБД. В противном случае – пустая строка. В частности, идентификатор соединения с СУБД совпадает со значением SPID в Microsoft SQL Server Enterprise Manager.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
