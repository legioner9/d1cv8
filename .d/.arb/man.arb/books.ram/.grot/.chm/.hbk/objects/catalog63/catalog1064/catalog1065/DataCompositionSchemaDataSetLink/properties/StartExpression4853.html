СвязьНаборовДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetLink)

НачальноеВыражение (StartExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит выражение, значение которого будет использоваться как начальное значение источника связи. Применяется в случае необходимости организации связи набора данных самого к себе для указания начального значения связи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
