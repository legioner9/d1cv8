Авто (Auto)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание:

Значение по умолчанию. Время актуальности будет выведено в заголовок отчета, если полученные данные не актуальны. При получении актуальных данных время актуальности не выводится.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
