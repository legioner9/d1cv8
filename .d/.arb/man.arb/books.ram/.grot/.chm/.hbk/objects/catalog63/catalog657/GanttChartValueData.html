ДанныеЗначенияДиаграммыГанта (GanttChartValueData)

Доступен, начиная с версии 8.3.21.

Элементы коллекции:

Произвольный
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения колонки таблицы значений для текущего значения диаграммы Ганта.
Возможно обращение к значению колонки таблицы значений для текущего значения диаграммы Ганта посредством оператора [...]. В качестве аргумента передается индекс колонки (нумерация с 0).

Свойства:

<Имя колонки> (<Column name>)

Методы:

Получить (Get)
Установить (Set)

Описание:

Представляет собой коллекцию объектов произвольного типа, являющихся значениями колонок таблицы значений для текущего значения диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.3.21.
