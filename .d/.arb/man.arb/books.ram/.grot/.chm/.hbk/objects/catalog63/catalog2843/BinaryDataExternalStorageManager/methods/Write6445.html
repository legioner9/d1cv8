МенеджерВнешнегоХранилищаДвоичныхДанных.Записать (BinaryDataExternalStorageManager.Write)

МенеджерВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageManager)

Записать (Write)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Записать()

Возвращаемое значение:

Нет

Описание:

Записывает изменения менеджера внешнего хранилища двоичных данных в базу данных. Если у объекта меняется только ПараметрыДоступа, то монопольный доступ к информационной базе не потребуется.
При изменении других параметров (Имя, ПараметрыПодключения, РежимЧтенияЗаписи, либо МинимальныйРазмерЗаписываемыхДанных), то при записи объекта потребуется монопольный доступ. Монопольный доступ устанавливается автоматически на время вызова метода.

Доступность:

Сервер.

Примечание:

Если РежимЧтенияЗаписи на каком-то внешнем, либо внутреннем хранилище двоичных данных установлен в ЧтениеИЗапись, то при попытке установить в РежимЧтенияЗаписи в ЧтениеИЗапись будет сгенерировано исключение, а операция отклонена.
МинимальныйРазмерЗаписываемыхДанных должен быть не менее 2048 байт.

Пример:

СтрПодключения = Новый 
    ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
СтрПодключения.URL = "https://bucket.example.com";
СтрПодключения.ТипURL =
    ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

Доступ = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных();
Доступ.ИдентификаторДоступа = "EXAMPLE";
Доступ.СекретныйКлюч = "wJalrXUtnFEMI/K7MDENG/bPxRfiCY";
Доступ.Регион = "US";

Мнджр = ВнешниеХранилищаДвоичныхДанных
            .Добавить("manager", СтрПодключения, Доступ);

Мнджр.МинимальныйРазмерЗаписываемыхДанных = 4096;
Мнджр.Имя = "OtherName";
РежимЗаписи = РежимЧтенияЗаписиХранилищаДвоичныхДанных.ЧтениеЗапись;
Мнджр.РежимЧтенияЗаписи = РежимЗаписи;
Мнджр.Записать();
