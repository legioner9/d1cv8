МенеджерПроверкиВстроенныхПокупок (InAppPurchasesValidationManager)

МенеджерПроверкиВстроенныхПокупок (InAppPurchasesValidationManager)

Доступен, начиная с версии 8.3.8.

Методы:

ПроверитьКвитанциюВстроеннойПокупки (ValidateInAppPurchaseReceipt)
ПроверитьКвитанциюВстроеннойПокупкиНаМобильномУстройстве (ValidateInAppPurchaseReceiptAtMobileDevice)

Описание:

Используется для доступа к проверке квитанций о выполненных встроенных покупках.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
