МенеджерКриптографии.ПолучитьДанныеИзПодписи (CryptoManager.GetDataFromSignature)

МенеджерКриптографии (CryptoManager)

ПолучитьДанныеИзПодписи (GetDataFromSignature)

Доступен, начиная с версии 8.3.24.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.
Если параметр типа Строка, то обозначает имя файла с подписью.
Если параметр типа ДвоичныеДанные или Поток, ПотокВПамяти, ФайловыйПоток, то он задает данные подписи.

Описание варианта метода:

Данные возвращаются методом как ДвоичныеДанные.

Вариант синтаксиса: Сохранение данных в файл или запись их в поток

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.
Если параметр типа Строка, то обозначает имя файла с подписью.
Если параметр типа ДвоичныеДанные или Поток, ПотокВПамяти, ФайловыйПоток, то он задает данные подписи.

<ВыходныеДанные> (обязательный)

Тип: Строка, Поток, ПотокВПамяти, ФайловыйПоток.
Имя файла или поток, в который будут выгружены данные, содержащиеся в подписи.

Возвращаемое значение:

Тип: ДвоичныеДанные, Неопределено.
В зависимости от варианта метода, либо ДвоичныеДанные, либо Неопределено.

Описание:

Извлекает подписанные данные из электронной подписи.
Если данные отсутствуют в подписи, т.е. она является отсоединенной, то будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МенеджерКриптографии, метод ПолучитьДанныеИзПодписиАсинх

Использование в версии:

Доступен, начиная с версии 8.3.24.
