ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

УстановитьПараметрыСоединенияПользователя (SetUserConnectionParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьПараметрыСоединенияПользователя(<ПользовательИБ>, <ПараметрыСоединения>)

Параметры:

<ПользовательИБ> (обязательный)

Тип: Строка.
Имя пользователя информационной базы, для которого устанавливаются параметры соединения.

<ПараметрыСоединения> (обязательный)

Тип: ПараметрыСоединенияВнешнегоИсточникаДанных.
Устанавливаемые параметры соединения.

Описание:

Устанавливает параметры соединения для пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если у пользователя нет права на изменение свойства и значение свойства изменено, то выдается исключение. Например, у пользователя нет права ИзменениеСтандартнойАутентификации и в параметре было установлено свойство ИмяПользователя, то будет выдано исключение.
Для свойств ИмяПользователя, Пароль, ПарольУстановлен проверяется права ИзменениеСтандартнойАутентификации.

Использование в версии:

Доступен, начиная с версии 8.2.
