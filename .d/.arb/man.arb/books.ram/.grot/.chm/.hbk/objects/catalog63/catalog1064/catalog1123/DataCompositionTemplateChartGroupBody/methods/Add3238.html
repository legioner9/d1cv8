ТелоГруппировкиДиаграммыМакетаКомпоновкиДанных (DataCompositionTemplateChartGroupBody)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.
Тип добавляемого элемента. Элементами коллекции могут быть МакетГруппировкиДиаграммыМакетаКомпоновкиДанных, ГруппировкаДиаграммыМакетаКомпоновкиДанных, ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных.

Возвращаемое значение:

Тип: МакетГруппировкиДиаграммыМакетаКомпоновкиДанных, ГруппировкаДиаграммыМакетаКомпоновкиДанных, ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных.

Описание:

Добавляет новый элемент коллекции и возвращает его. В случае, если тип нового элемента не подходит для данной коллекции, добавления не происходит, и метод возвращает Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
