КонстантаКлючЗначения.<Имя константы> (ConstantValueKey.<Constant name>)

Пустой (IsEmpty)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Пустой()

Возвращаемое значение:

Тип: Булево.
Истина - все поля ключа имеют значения по умолчанию.

Описание:

Определяет, есть ли у константы значение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
