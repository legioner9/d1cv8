HTTPМетод.MKCOL (HTTPMethod.MKCOL)

HTTPМетод (HTTPMethod)

MKCOL (MKCOL)

Доступен, начиная с версии 8.3.5.

Описание:

Соответствует HTTP-методу MKCOL.
Является частью протокола WebDAV.
Описан в стандарт RFC 2518 (http://tools.ietf.org/html/rfc2518).

Использование в версии:

Доступен, начиная с версии 8.3.5.
