КонтейнерКлючейКриптографии.НачатьВыгрузку (CryptoKeysContainer.BeginUnload)

КонтейнерКлючейКриптографии (CryptoKeysContainer)

НачатьВыгрузку (BeginUnload)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:
  • ВыгруженныеДанные - данные типа ДвоичныеДанные;
  • ДополнительныеПараметры - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание варианта метода:

Передает выгруженные данные в параметре типа ДвоичныеДанные.

Вариант синтаксиса: Данные в файл

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:
  • ИмяФайла - имя файла, в который выгружен сертификат;
  • ДополнительныеПараметры - значение, которое было указано при создании объекта ОписаниеОповещения.

<ИмяФайла> (необязательный)

Тип: Строка.
Имя файла, в который будет выгружен контейнер ключей криптографии.

Описание варианта метода:

Выгружает контейнер ключей криптографии в файл.

Описание:

Возвращает данные, использованные при создании или инициализации объекта.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
