ПотокВПамяти (MemoryStream)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьАсинх(<Буфер>, <ПозицияВБуфере>, <Количество>)

Параметры:

<Буфер> (обязательный)

Тип: БуферДвоичныхДанных.
Буфер, данные которого получаются для записи.

<ПозицияВБуфере> (обязательный)

Тип: Число.
Позиция в буфере, начиная с которой требуется получить данные для записи в поток.

<Количество> (обязательный)

Тип: Число.
Количество байт, которые требуется записать.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Записывает в поток заданное количество байтов из буфера по заданному смещению.

Запись в поток возможна, только если поток поддерживает запись. В противном случае, при вызове метода будет сгенерировано исключение.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
