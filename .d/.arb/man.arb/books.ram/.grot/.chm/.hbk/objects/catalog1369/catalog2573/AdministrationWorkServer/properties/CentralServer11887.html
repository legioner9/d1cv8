АдминистрированиеРабочийСервер (AdministrationWorkServer)

ЦентральныйСервер (CentralServer)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Истина - указывает, что сервер является центральным.
В этом случае на него назначается сервис конфигурации кластера, и адрес менеджера кластера может использоваться клиентскими приложениями для соединения с кластером.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
