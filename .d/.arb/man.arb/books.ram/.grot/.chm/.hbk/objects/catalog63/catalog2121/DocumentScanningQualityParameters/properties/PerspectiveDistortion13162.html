ПараметрыКачестваСканированияДокументов (DocumentScanningQualityParameters)

ПерспективныеИскажения (PerspectiveDistortion)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: Число.
Показывает, насколько мало перспективных искажений, хорошо ли расположен документ по отношению к плоскости камеры.
Значение данного свойства учитывается при расчете значения ОбобщенноеКачество, если для в параметре <ПараметрыСканирования> методов ПоказатьСканированиеДокументов и СканироватьДокументыАсинх используется свойство ПрямоугольныйДокумент.
Значение – число, нормированное от 1 до 10:
  • [1.0, 4.0) – низкое качество, устройство расположено не параллельно документу или под большим углом относительно документа. Какие-то области документа находятся на много дальше других от устройства и поэтому очень плохо видны.
  • [4.0, 7.0) – среднее качество, устройство расположено почти параллельно плоскости документа. Области документа, расположенные дальше от камеры, видны хуже.
  • [7.0, 10.0] – высокое качество, документ снят прямо. Все области документа видны отлично.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
