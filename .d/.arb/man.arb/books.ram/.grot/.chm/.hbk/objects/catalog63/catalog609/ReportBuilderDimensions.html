ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ИзмерениеПостроителяОтчета
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются измерения.
Возможно обращение к измерению посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция измерений построителя отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.
