ПолеКалендаря.КонецПериодаОтображения (CalendarBox.EndOfDisplayPeriod)

ПолеКалендаря (CalendarBox)

КонецПериодаОтображения (EndOfDisplayPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Дата, Неопределено.
Верхняя граница периода видимых в календаре дат. За указанную границу перемещение в календаре невозможно.
Для снятия ограничения следует указать значение Неопределено.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
