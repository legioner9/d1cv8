ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

ВыраженияИтогов (TotalExpressions)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: ВыраженияИтоговСхемыЗапроса.
Содержит выражения итогов схемы запроса.
Соответствует выражениям итогов в конструкции ИТОГИ языка запросов.
Доступно только, если у схемы запроса выключен режим компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
