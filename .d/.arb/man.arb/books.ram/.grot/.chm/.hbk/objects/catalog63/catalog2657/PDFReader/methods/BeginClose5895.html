ЧтениеPDF.НачатьЗакрытие (PDFReader.BeginClose)

ЧтениеPDF (PDFReader)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Закрывает открытый файл. Закрытие файла выполняется автоматически при удалении объекта ЧтениеPDF.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

ЧтениеPDF, метод Закрыть
ЧтениеPDF, метод ЗакрытьАсинх

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
