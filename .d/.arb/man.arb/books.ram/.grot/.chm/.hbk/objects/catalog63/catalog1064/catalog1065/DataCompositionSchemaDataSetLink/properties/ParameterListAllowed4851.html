Описание:

Тип: Булево.
Возможность использования в качестве значения параметра список значений. Список значений допустим в случае, если параметр используется в операции В.
Истина - список значений допускается к использованию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
