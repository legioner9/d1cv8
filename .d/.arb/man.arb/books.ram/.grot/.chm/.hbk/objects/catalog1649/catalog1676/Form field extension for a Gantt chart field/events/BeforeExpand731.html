Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ПередРазворачиванием (BeforeExpand)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПередРазворачиванием(<ИдентификаторЗначения>, <Отказ>)

Параметры:

<ИдентификаторЗначения>

Тип: ИдентификаторЗначенияДиаграммыГанта.
Содержит идентификатор значения, точка/серия которого будет развернута.

<Отказ>

Тип: Булево.
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение Истина, то стандартной обработки события не будет.
Значение по умолчанию: Ложь.

Описание:

Вызывается перед разворачиванием точки/серии диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
