АдминистрированиеТипТребованияНазначения (AdministrationAssignmentRuleType)

Доступен, начиная с версии 8.3.14.

Значения

Авто (Auto)
Назначать (Assign)
НеНазначать (DontAssign)

Описание:

Определяет действие требования назначения.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
