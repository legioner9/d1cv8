ОбъектМетаданных: ПланВидовХарактеристик.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (MetadataObject: ChartOfCharacteristicTypes.ExecuteAfterWriteDataHistoryVersionProcessing)

ОбъектМетаданных: ПланВидовХарактеристик (MetadataObject: ChartOfCharacteristicTypes)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если имеет значение Истина, то в обработчике события ОбработкаПослеЗаписиВерсийИсторииДанных можно обработать запись версии после обновления истории.
При записи данных значение, указанное в конфигураторе, устанавливается в качестве значения свойства ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных. Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию: Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.15.
