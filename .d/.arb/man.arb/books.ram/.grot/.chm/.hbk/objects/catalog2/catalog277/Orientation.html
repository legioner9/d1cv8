Ориентация (Orientation)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
Вертикально (Vertical)
Горизонтально (Horizontal)

Описание:

Содержит варианты ориентации элементов управления. Используется при определении свойства Ориентация.

Доступность:

Толстый клиент.
