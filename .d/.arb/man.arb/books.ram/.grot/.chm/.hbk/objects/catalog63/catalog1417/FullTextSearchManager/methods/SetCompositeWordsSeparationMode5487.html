МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

УстановитьРежимРазделенияСоставныхСлов (SetCompositeWordsSeparationMode)

Доступен, начиная с версии 8.3.15.

Синтаксис:

УстановитьРежимРазделенияСоставныхСлов(<РежимРазделенияСоставныхСлов>)

Параметры:

<РежимРазделенияСоставныхСлов> (необязательный)

Тип: РежимРазделенияСоставныхСлов.
Выбранный режим для механизма разделения составных слов.

Описание:

Устанавливает режим использования разделения составных слов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МенеджерПолнотекстовогоПоиска, метод ПолучитьРежимРазделенияСоставныхСлов

Использование в версии:

Доступен, начиная с версии 8.3.15.
