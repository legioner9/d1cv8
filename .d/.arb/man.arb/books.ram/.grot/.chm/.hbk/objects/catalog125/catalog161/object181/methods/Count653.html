РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество записей в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Для Индекс = 0 По НаборКурсов.Количество()-1  Цикл
    ДатаКурса = НаборКурсов[Индекс].Период;
    ТекКурс   = НаборКурсов[Индекс].Курс; 
    Сообщить("Курс на " + ДатаКурса + " - " + ТекКурс + " руб.");
КонецЦикла;
