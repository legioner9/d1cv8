ФормаКлиентскогоПриложения.ВыборДобавлением (ClientApplicationForm.ChoiceByAdding)

ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыборДобавлением (ChoiceByAdding)

Доступен, начиная с версии 8.3.23.

Использование:

Ключевой.

Описание:

Тип: Булево.
Определяет способ обработки для выбранных значений в поле ввода в режиме расширенного редактирования множественных значений.

Параметр принимает значение Истина, если поле ввода находится в режиме расширенного редактирования множественных значений и форма открывается в обработчике НачалоВыбора, когда СтандартнаяОбработка=Истина.

В остальных случаях параметр принимает значение Ложь.
