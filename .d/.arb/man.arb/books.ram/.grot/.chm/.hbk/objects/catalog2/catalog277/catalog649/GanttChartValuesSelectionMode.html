РежимВыделенияЗначенийДиаграммыГанта (GanttChartValuesSelectionMode)

Доступен, начиная с версии 8.3.23.

Значения

Авто (Auto)
Множественный (Multiple)
Нет (None)
Одиночный (Single)

Описание:

Содержит варианты выделения значений диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/data/chart. Имя типа XDTO: GanttChartValuesSelectionMode.

Использование в версии:

Доступен, начиная с версии 8.3.23.
