Низкая (Low)

Описание:

Проблема применения расширения конфигурации с низкой важностью. Наличие такой проблемы может возникнуть, например, после установки свойства Активно в значение Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.12.
