ЧтениеДанных.ЗакрытьАсинх (DataReader.CloseAsync)

ЧтениеДанных (DataReader)

ЗакрытьАсинх (CloseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗакрытьАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Дальнейшая работа с объектом после вызова этого метода невозможна. Если объект является владельцем нижележащего потока, то поток также закрывается.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

ЧтениеДанных, метод Закрыть
ЧтениеДанных, метод НачатьЗакрытие

Использование в версии:

Доступен, начиная с версии 8.3.18.
