ПолеФормы.ФиксацияВТаблице (FormField.FixingInTable)

ПолеФормы (FormField)

ФиксацияВТаблице (FixingInTable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ФиксацияВТаблице.
Определяет, будет ли фиксироваться колонка таблицы.
Если значение свойства установлено в Лево, то колонка прижимается к левому краю таблицы и фиксируется.
Если значение установлено в Право, то колонка прижимается к правому краю и фиксируется.
Если значение установлено в значение Нет, то колонка выводится обычным образом.
Если свойство установлено в Лево или Право для колонки в группе, то колонка переносится на верхний уровень таблицы и фиксируется так, как описано выше.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Свойство имеет смысл, если поле является колонкой таблицы управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.2.
