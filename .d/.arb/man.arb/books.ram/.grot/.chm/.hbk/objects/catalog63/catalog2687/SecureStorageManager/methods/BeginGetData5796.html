МенеджерБезопасногоХранилища.НачатьПолучениеДанных (SecureStorageManager.BeginGetData)

МенеджерБезопасногоХранилища (SecureStorageManager)

НачатьПолучениеДанных (BeginGetData)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьПолучениеДанных(<Ключ>, <Обработчик>, <СообщениеПользователю>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.
Ключ, под которым были сохранены данные.

<Обработчик> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после получения данных со следующими параметрами:
  • <ОтмененоПользователем> - тип Булево.
  • <Ключ> - тип Строка. Переданный ключ.
  • <Данные> - тип Произвольный. Данные, хранимые под переданным ключом.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
При возникновении ошибки во время выполнения процедуры, будет вызван обработчик ошибки.

<СообщениеПользователю> (необязательный)

Тип: Строка.
Текст сообщения, отображаемый пользователю при проверке доступа.

Возвращаемое значение:

Если переданный ключ не существует, возвращается ошибка.

Описание:

Получает данные из защищенного хранилища при выполнении условний доступа, переданных при сохранении данных.
Вызов метода не блокирует исполнение кода языка, все результаты возвращаются в переданный обработчик оповещения.
Если при помещении данных <СпособЗащитыДоступа> имеет значение ТребуетсяДополнительнаяПроверкаПользователя, то при получении выполняется проверка пользователя в соответствии с заданным способом проверки.
Выполняемая операция может быть отменена пользователем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
