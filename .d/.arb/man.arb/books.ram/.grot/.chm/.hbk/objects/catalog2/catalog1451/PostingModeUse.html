ИспользованиеРежимаПроведения (PostingModeUse)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
Неоперативный (AnyTime)
Оперативный (RealTime)

Описание:

Определяет набор вариантов использования режима проведения в форме документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение формы документа, свойство ИспользоватьРежимПроведения
Расширение формы клиентского приложения для документа, свойство ИспользоватьРежимПроведения

Использование в версии:

Доступен, начиная с версии 8.0.
