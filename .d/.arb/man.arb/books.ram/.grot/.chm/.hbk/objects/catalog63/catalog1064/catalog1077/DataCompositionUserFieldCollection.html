КоллекцияПользовательскихПолейКомпоновкиДанных (DataCompositionUserFieldCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ПользовательскоеПолеВыражениеКомпоновкиДанных, ПользовательскоеПолеВыборКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются пользовательские поля.
Возможно обращение к пользовательскому полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция пользовательских полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПользовательскиеПоляКомпоновкиДанных, свойство Элементы
ПользовательскиеНастройкиКомпоновкиДанных, свойство Элементы

Использование в версии:

Доступен, начиная с версии 8.1.
