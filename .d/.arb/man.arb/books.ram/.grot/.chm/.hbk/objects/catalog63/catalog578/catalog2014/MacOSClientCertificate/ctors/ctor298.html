СертификатКлиентаMacOS (MacOSClientCertificate)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый СертификатКлиентаMacOS()

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы macOS. Сертификат подбирается автоматически.

Использование в версии:

Доступен, начиная с версии 8.3.13.
