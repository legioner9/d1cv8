ВариантыТочкиМаршрутаБизнесПроцесса (BusinessProcessRoutePointCases)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ВариантТочкиМаршрутаБизнесПроцесса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты.
Возможно обращение к варианту посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента в коллекции.

Свойства:

<Имя варианта> (<Variant name>)

Методы:

Количество (Count)
Получить (Get)

Описание:

Представляет коллекцию вариантов точки маршрута бизнес-процесса.
Только для точек вида ВыборВарианта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Процедура ВыборСогласованияОбработкаВыбораВарианта(
        ТочкаВыбораВарианта, 
        Результат)
    Если Случай = 1 Тогда
        Результат = ТочкаВыбораВарианта.Варианты.УпрощенныйВариант;
    Иначе 
        Если Случай = 2 Тогда
            Результат = ТочкаВыбораВарианта.Варианты.ПолныйВариант;
        Иначе 
            Если Случай = 3 Тогда
                Результат = ТочкаВыбораВарианта.Варианты.ВариантДиректора;
            Иначе        
                Результат = ТочкаВыбораВарианта.Варианты.ВариантБезСогласования;
            КонецЕсли;
        КонецЕсли;
    КонецЕсли;
КонецПроцедуры
