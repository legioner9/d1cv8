ПараметрыЗаписиJSON (JSONWriterSettings)

По описанию параметров записи

Доступен, начиная с версии 8.3.6.

Синтаксис:

Новый ПараметрыЗаписиJSON(<ПереносСтрок>, <СимволыОтступа>, <ИспользоватьДвойныеКавычки>, <ЭкранированиеСимволов>, <ЭкранироватьУгловыеСкобки>, <ЭкранироватьРазделителиСтрок>, <ЭкранироватьАмперсанд>, <ЭкранироватьОдинарныеКавычки>, <ЭкранироватьСлеш>)

Параметры:

<ПереносСтрок> (необязательный)

Тип: ПереносСтрокJSON.
Определяет способ переноса строк, который будет использован при записи данных JSON.
Значение по умолчанию: Авто.

<СимволыОтступа> (необязательный)

Тип: Строка.
Определяет символы отступа, используемые при записи данных JSON.
Применяется только, если значение ПереносСтрокJSON отлично от Нет.
Значение по умолчанию: " ".

<ИспользоватьДвойныеКавычки> (необязательный)

Тип: Булево.
Определяет, будут ли при записи имена свойств JSON записываться в двойных кавычках.
Значение по умолчанию: Истина.

<ЭкранированиеСимволов> (необязательный)

Тип: ЭкранированиеСимволовJSON.
Определяет используемый способ экранирования (замены) символов при записи данных JSON.
Значение по умолчанию: Нет.

<ЭкранироватьУгловыеСкобки> (необязательный)

Тип: Булево.
Определяет, будут ли при записи экранироваться символы "<" и ">".
Значение по умолчанию: Ложь.

<ЭкранироватьРазделителиСтрок> (необязательный)

Тип: Булево.
Определяет, будут ли экранироваться разделители строк - U+2028 (line-separator) и U+2029 (page-separator).
Значение по умолчанию: Истина.

<ЭкранироватьАмперсанд> (необязательный)

Тип: Булево.
Определяет, будет ли при записи экранироваться символ амперсанда "&".
Значение по умолчанию: Ложь.

<ЭкранироватьОдинарныеКавычки> (необязательный)

Тип: Булево.
Определяет, будут ли экранироваться одинарные кавычки.
Устанавливается в значение Истина, если ИспользоватьДвойныеКавычки установлено в Ложь.
Значение по умолчанию: Ложь.

<ЭкранироватьСлеш> (необязательный)

Тип: Булево.
Определяет, будет ли экранироваться слеш (косая черта) при записи значения.
Значение по умолчанию: Ложь.

Описание:

Создает объект параметров записи JSON.

Использование в версии:

Доступен, начиная с версии 8.3.6.
