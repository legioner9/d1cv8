ПолучениеЭлементаДанных (DataItemReceive)

ПолучениеЭлементаДанных (DataItemReceive)

Доступен, начиная с версии 8.0.

Значения

Авто (Auto)
Игнорировать (Ignore)
Принять (Accept)

Описание:

Содержит варианты значений выходных параметров обработчиков событий ПриПолученииДанныхОтГлавного и ПриПолученииДанныхОтПодчиненного объектов ПланОбменаОбъект.<Имя плана обмена>. Данное значение определяет поведение системы при получении элементов данных из сообщения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

См. также:

ПланОбменаОбъект.<Имя плана обмена>, событие ПриПолученииДанныхОтПодчиненного
ПланОбменаОбъект.<Имя плана обмена>, событие ПриПолученииДанныхОтГлавного

Использование в версии:

Доступен, начиная с версии 8.0.
