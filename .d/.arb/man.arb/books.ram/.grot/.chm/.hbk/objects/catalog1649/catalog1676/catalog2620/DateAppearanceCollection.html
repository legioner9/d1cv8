КоллекцияОформленийДат (DateAppearanceCollection)

КоллекцияОформленийДат (DateAppearanceCollection)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ОформлениеДаты
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается дата.

Методы:

Количество (Count)
Получить (Get)

Описание:

Коллекция, содержащая оформление дат календаря. Коллекция является перечислимой. Доступ к элементам осуществляется по значению даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

ОформлениеПериода, свойство Даты

Использование в версии:

Доступен, начиная с версии 8.0.
