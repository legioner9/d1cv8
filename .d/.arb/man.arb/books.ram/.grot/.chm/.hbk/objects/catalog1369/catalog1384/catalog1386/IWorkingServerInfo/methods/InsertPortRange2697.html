Рабочий сервер.InsertPortRange (IWorkingServerInfo.InsertPortRange)

Рабочий сервер (IWorkingServerInfo)

InsertPortRange (InsertPortRange)

Доступен, начиная с версии 8.1.

Синтаксис:

InsertPortRange(<Диапазон>)

Параметры:

<Диапазон> (необязательный)

Тип: Диапазон портов.
Добавляет новый диапазон IP-портов к списку диапазонов IP-портов для динамического распределения.

Описание:

Добавляет диапазон портов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
