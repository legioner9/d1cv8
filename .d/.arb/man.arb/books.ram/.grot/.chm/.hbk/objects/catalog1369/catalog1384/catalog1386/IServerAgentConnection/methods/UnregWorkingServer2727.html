Соединение с агентом сервера.UnregWorkingServer (IServerAgentConnection.UnregWorkingServer)

Соединение с агентом сервера (IServerAgentConnection)

UnregWorkingServer (UnregWorkingServer)

Доступен, начиная с версии 8.1.

Синтаксис:

UnregWorkingServer(<Кластер>, <Сервер>)

Параметры:

<Кластер> (необязательный)

Тип: Кластер серверов.
Кластер, в котором должна быть отменена регистрация рабочего сервера.

<Сервер> (необязательный)

Тип: Рабочий сервер.
Рабочий сервер, регистрация которого должна быть отменена.

Описание:

Отменяет регистрацию в кластере рабочего сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
