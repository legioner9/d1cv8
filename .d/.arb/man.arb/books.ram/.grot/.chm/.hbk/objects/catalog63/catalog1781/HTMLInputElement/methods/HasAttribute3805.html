ЭлементВводаHTML.ЕстьАтрибут (HTMLInputElement.HasAttribute)

ЭлементВводаHTML (HTMLInputElement)

ЕстьАтрибут (HasAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

ЕстьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя атрибута.

Возвращаемое значение:

Тип: Булево.
Истина, если атрибут с указанным именем явно присутствует в элементе, или имеет значение по умолчанию; Ложь - в противном случае.

Описание:

Определяет наличие атрибута с указанным именем в данном элементе.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
