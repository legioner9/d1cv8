ПоложениеЗаголовкаЭлементаФормы (FormItemTitleLocation)

ПоложениеЗаголовкаЭлементаФормы (FormItemTitleLocation)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
Верх (Top)
Лево (Left)
Нет (None)
Низ (Bottom)
Право (Right)

Описание:

Содержит варианты положения подчиненного элемента относительно элемента формы. В зависимости от вида элемента и его размещения, не все варианты могут отрабатываться.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПолеФормы, свойство ПоложениеЗаголовка
ТаблицаФормы, свойство ПоложениеЗаголовка
ТаблицаФормы, свойство ПоложениеКоманднойПанели

Использование в версии:

Доступен, начиная с версии 8.2.
