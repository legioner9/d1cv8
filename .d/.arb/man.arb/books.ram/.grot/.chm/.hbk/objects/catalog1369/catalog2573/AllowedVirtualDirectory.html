РазрешенныйВиртуальныйКаталог (AllowedVirtualDirectory)

Доступен, начиная с версии 8.3.14.

Свойства:

ЗаписьРазрешена (WritingAllowed)
Описание (Description)
Псевдоним (Alias)
Путь (Path)
ЧтениеРазрешено (ReadingAllowed)

Методы:

Записать (Write)
Удалить (Delete)

Описание:

Используется для доступа к свойствам и методам виртуального каталога.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.
