Описание:

Тип: ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>.
Набор свойств содержит табличные части видов расчета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>).
Предопределенные табличные части ВедущиеВидыРасчетаСтрока.<Имя плана видов расчета>, БазовыеВидыРасчетаСтрока.<Имя плана видов расчета> и ВытесняющиеВидыРасчетаСтрока.<Имя плана видов расчета> доступны через специальные соответствующие свойства объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
