Почта.Выбрать (Mail.Get)

Почта (Mail)

Выбрать (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать(<НепрочитанныеСообщения>, <Конверты>)

Параметры:

<НепрочитанныеСообщения> (необязательный)

Тип: Булево.
Истина - выбирать только непрочитанные сообщения.
Значение по умолчанию: Истина.

<Конверты> (необязательный)

Тип: Булево.
Истина - выбирать только конверты сообщений (вложения не переносятся в возвращаемые сообщения).
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ПочтовоеСообщение.

Описание:

Используется для получения сообщений с сервера.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

Почта = Новый Почта;
Почта.Подключиться();
Сообщения = Почта.Выбрать(Ложь, Ложь);
Почта.Отключиться();

Использование в версии:

Доступен, начиная с версии 8.0.
