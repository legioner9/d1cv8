Требование назначения сервисов.RuleType (IAssignmentRule.RuleType)

Требование назначения сервисов (IAssignmentRule)

RuleType (RuleType)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Определяет действие данного требования. Возможные значения:
  • 0 – авто. Объект требования будет назначен на данный рабочий сервер наравне с теми рабочими серверами, на которых для данного объекта не найдено подходящего требования;
  • 1 – назначать. Объект требования будет назначен на один из тех серверов, для которых были найдены требования с действием "назначать";
  • 2 – не назначать. Объект требования будет назначен на один из тех серверов, для которых не были найдены требования с действием "не назначать".

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
