Тип: Булево.
Если это свойство установлено в значение Истина, то справка по данному объекту метаданных (свойство Справка) будет включена в общую справку по конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.1.
