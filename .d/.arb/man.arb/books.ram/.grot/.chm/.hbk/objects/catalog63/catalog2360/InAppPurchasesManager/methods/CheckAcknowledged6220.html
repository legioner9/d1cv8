МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПроверитьПодтверждение (CheckAcknowledged)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьПодтверждение(<Покупка>)

Параметры:

<Покупка> (обязательный)

Тип: ВстроеннаяПокупка, Строка.
Покупка, которая была приобретена, и подтверждение которой необходимо проверить.

Возвращаемое значение:

Тип: Булево.

Описание:

Возвращает информацию, подтверждена ли указанная покупка:

  • На Android возвращает Истина, если покупка была подтверждена.
  • На других ОС всегда возвращает Истина.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
