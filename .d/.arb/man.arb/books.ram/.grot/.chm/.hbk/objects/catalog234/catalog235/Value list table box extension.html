Расширение табличного поля списка значений (Value list table box extension)

Доступен, начиная с версии 8.0.

Свойства:

ИзменятьПорядокСтрок (ChangeRowOrder)

Описание:

Дополнительные свойства элемента управления ТабличноеПоле, предназначенные для интерактивной работы со списком значений. Для данного расширения свойство ТекущаяСтрока имеет значение типа ЭлементСпискаЗначений.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
