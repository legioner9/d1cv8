ПланСчетовСсылка.<Имя плана счетов>.ПолучитьПорядокКода (ChartOfAccountsRef.<Chart of accounts name>.GetCodeOrder)

ПланСчетовСсылка.<Имя плана счетов> (ChartOfAccountsRef.<Chart of accounts name>)

ПолучитьПорядокКода (GetCodeOrder)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьПорядокКода()

Возвращаемое значение:

Тип: Строка.

Описание:

Преобразует код счета в соответствии с маской.
Если маска представлена в виде "@@@.@@@.@@", то для счета "41.1" метод возвратит строку " 41. 1". При этом к символам счета будут добавлены пробелы, а числовые фрагменты кода прижимаются вправо.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для масок, которые используют другие символы, возвращается собственно значение кода счета.

Пример:

Процедура ПередЗаписью(Отказ)
    Порядок = ПолучитьПорядокКода();
КонецПроцедуры

Использование в версии:

Доступен, начиная с версии 8.0.
