ТаблицаФормы.ПриИзменении (FormTable.OnChange)

ТаблицаФормы (FormTable)

ПриИзменении (OnChange)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриИзменении()

Описание:

Возникает при изменении данных.
Если таблица формы клиентского приложения связана с динамическим списком, в следующих случаях:

  • После записи объекта в форме;
  • После непосредственного удаления;
  • После установки или снятия пометки удаления;
  • После проведения и отмены проведения;
  • После переноса в группу;
  • После чтения или записи изменений списка плана обмена.
Если таблица формы клиентского приложения связана с набором записей регистра, после выполнения команды "Переключить активность".

Если событие вызвано из-за начала редактирования, то в обработчике данного события нельзя использовать серверные методы формы с директивой компиляции НаСервере, а также изменять свойства, которые могут привести к серверному вызову.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
