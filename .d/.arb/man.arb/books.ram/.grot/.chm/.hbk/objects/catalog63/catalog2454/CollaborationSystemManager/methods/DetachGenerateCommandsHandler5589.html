МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтключитьОбработчикФормированияКоманд (DetachGenerateCommandsHandler)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ОтключитьОбработчикФормированияКоманд(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Описание оповещения, заданное при вызове метода ПодключитьОбработчикФормированияКоманд.

Описание:

Отключает обработчик, установленный вызовом метода ПодключитьОбработчикФормированияКоманд.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
