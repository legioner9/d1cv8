РасширениеКонфигурации.ГлавныйУзел (ConfigurationExtension.MasterNode)

РасширениеКонфигурации (ConfigurationExtension)

ГлавныйУзел (MasterNode)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Неопределено, ПланОбменаСсылка.<Имя плана обмена>.
Содержит узел распределенной информационной базы, в котором создано данное расширение конфигурации. Если текущая информационная база не является узлом распределенной информационной базы или расширение создано локально, то содержит Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

РасширениеКонфигурации, свойство ИспользуетсяВРаспределеннойИнформационнойБазе
ОбъектМетаданных: ПланОбмена, свойство ВключатьРасширенияКонфигурации

Использование в версии:

Доступен, начиная с версии 8.3.12.
