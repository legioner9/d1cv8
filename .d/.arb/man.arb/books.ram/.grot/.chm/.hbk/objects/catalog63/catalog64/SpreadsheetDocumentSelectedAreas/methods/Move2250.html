ВыделенныеОбластиТабличногоДокумента (SpreadsheetDocumentSelectedAreas)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Область>, <Смещение>)

Параметры:

<Область> (обязательный)

Тип: ОбластьЯчеекТабличногоДокумента, РисунокТабличногоДокумента.
Сдвигаемая область в коллекции выделенных областей.

<Смещение> (обязательный)

Тип: Число.
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает область в коллекции на указанное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
