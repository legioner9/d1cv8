КнопкаКоманднойПанели.ПорядокКнопок (CommandBarButton.ButtonOrder)

КнопкаКоманднойПанели (CommandBarButton)

ПорядокКнопок (ButtonOrder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ПорядокКнопокКоманднойПанели.
Признак, определяющий сортировку кнопок по тексту подменю в пределах разделителей.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только для кнопок, у которых установлено значение свойства ТипКнопки равное Подменю.

Использование в версии:

Доступен, начиная с версии 8.0.
