ПреобразованиеXSL (XSLTransform)

ЗагрузитьТаблицуСтилейXSLИзСтроки (LoadXSLStylesheetFromString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЗагрузитьТаблицуСтилейXSLИзСтроки(<СтрокаXSL>)

Параметры:

<СтрокаXSL> (необязательный)

Тип: Строка.
Строка, содержащая описание преобразования XSL.

Описание:

Загружает описание преобразования XSL из строки.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
