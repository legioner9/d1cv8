ДиаграммаГанта.РазвернутаТочка (GanttChart.PointExpanded)

ДиаграммаГанта (GanttChart)

РазвернутаТочка (PointExpanded)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутаТочка(<Точка>)

Параметры:

<Точка> (обязательный)

Тип: ТочкаДиаграммыГанта.
Точка диаграммы Ганта.

Возвращаемое значение:

Тип: Булево.
Истина - точка развернута; Ложь - в противном случае.

Описание:

Определяет, развернута ли точка диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
