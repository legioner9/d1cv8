МенеджерОбработкиОшибок (ErrorProcessingManager)

КатегорияОшибкиДляПользователя (ErrorCategoryForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

КатегорияОшибкиДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: ИнформацияОбОшибке.
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: КатегорияОшибки.
Возвращает одно из следующих значений:
  • ИсключениеВызванноеИзВстроенногоЯзыка,
  • НарушениеПравДоступа,
  • НеправильныйПароль,
  • ОтсутствиеРазрешенияДляИспользованияФункциональности,
  • ОшибкаВнешнегоИсточникаДанных,
  • ОшибкаДоступаКЛокальномуФайлу,
  • ОшибкаКонфигурации,
  • ОшибкаКопииБазыДанных,
  • ОшибкаНастроекКомпоновкиДанных,
  • ОшибкаПереходаПоНавигационнойСсылке,
  • ОшибкаПолнотекстовогоПоиска,
  • ОшибкаПреобразованияДокумента,
  • ОшибкаПроверкиПодписи,
  • ОшибкаРаботыСПринтером,
  • ОшибкаРаботыСРечью,
  • ОшибкаСеанса,
  • ОшибкаСети,
  • ОшибкаСистемыВзаимодействия,
  • ОшибкаСредствМультимедиа,
  • ОшибкаТабличногоПространстваБазыДанных,
  • ОшибкаХранимыхДанных,
  • ПринудительноеЗавершениеРаботы,
  • ПрочаяОшибка.
Не возвращает:
  • ВсеОшибки,
  • ОшибкаВоВремяВыполненияВстроенногоЯзыка,
  • ОшибкаИспользованияВстроенногоЯзыка,
  • ОшибкаКомпиляцииВстроенногоЯзыка.

Описание:

Возвращает категорию для пользователя. По этой категории будет формироваться сообщение в методе СообщениеОбОшибкеДляПользователя.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.19.
