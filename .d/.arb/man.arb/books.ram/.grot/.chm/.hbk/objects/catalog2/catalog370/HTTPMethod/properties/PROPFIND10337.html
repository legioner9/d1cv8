HTTPМетод.PROPFIND (HTTPMethod.PROPFIND)

HTTPМетод (HTTPMethod)

PROPFIND (PROPFIND)

Доступен, начиная с версии 8.3.5.

Описание:

Соответствует HTTP-методу PROPFIND.
Является частью протокола WebDAV.
Описан в стандарте RFC 2518 (http://tools.ietf.org/html/rfc2518).

Использование в версии:

Доступен, начиная с версии 8.3.5.
