Глобальный контекст (Global context)

ПолучитьДвоичныеДанныеИзБуфераДвоичныхДанных (GetBinaryDataFromBinaryDataBuffer)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДвоичныеДанныеИзБуфераДвоичныхДанных(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: БуферДвоичныхДанных.
Буфер двоичных данных.

Возвращаемое значение:

Тип: ДвоичныеДанные.

Описание:

Преобразует буфер двоичных данных в значение типа ДвоичныеДанные.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
