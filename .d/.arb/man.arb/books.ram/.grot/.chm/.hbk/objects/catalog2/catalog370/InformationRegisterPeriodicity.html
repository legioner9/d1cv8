ПериодичностьРегистраСведений (InformationRegisterPeriodicity)

Доступен, начиная с версии 8.0.

Значения

Год (Year)
День (Day)
Квартал (Quarter)
Месяц (Month)
Непериодический (Nonperiodical)
ПозицияРегистратора (RecorderPosition)
Секунда (Second)

Описание:

Определяет набор вариантов периодичности регистра сведений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
