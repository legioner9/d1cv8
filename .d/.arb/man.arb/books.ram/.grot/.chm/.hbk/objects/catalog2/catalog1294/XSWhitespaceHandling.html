ОбработкаПробельныхСимволовXS (XSWhitespaceHandling)

Доступен, начиная с версии 8.1.

Значения

Заменять (Replace)
Сворачивать (Collapse)
Сохранять (Preserve)

Описание:

Содержит варианты способов обработки пробельных символов.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
