КонструкторФорматнойСтроки (FormatStringWizard)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: Булево.
Истина - конструктор закрыли по кнопке "OK"; Ложь - в противном случае.

Описание:

Открывает модально конструктор форматной строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если для конфигурации свойство РежимИспользованияМодальности установлено в НеИспользовать, следует использовать метод Показать.

См. также:

КонструкторФорматнойСтроки, метод ОткрытьАсинх

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
