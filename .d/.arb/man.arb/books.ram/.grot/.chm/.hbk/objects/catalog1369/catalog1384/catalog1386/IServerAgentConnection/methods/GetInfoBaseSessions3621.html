Соединение с агентом сервера (IServerAgentConnection)

GetInfoBaseSessions (GetInfoBaseSessions)

Доступен, начиная с версии 8.2.

Синтаксис:

GetInfoBaseSessions(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер серверов, для которого необходимо получить список сеансов. Должен содержать объект Кластер серверов, полученный методом GetClusters.

<ИнформационнаяБаза> (обязательный)

Тип: Описание информационной базы.
Информационная база, для которой необходимо получить список сеансов. Должен содержать объект, полученный методом GetInfoBases.

Возвращаемое значение:

Тип: COMSafeArray.
Массив, содержащий объекты Сеанс.

Описание:

Получить список сеансов информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
