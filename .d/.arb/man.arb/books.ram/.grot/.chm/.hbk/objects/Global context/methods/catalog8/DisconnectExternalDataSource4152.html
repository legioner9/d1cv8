Глобальный контекст (Global context)

РазорватьСоединениеСВнешнимИсточникомДанных (DisconnectExternalDataSource)

Доступен, начиная с версии 8.2.

Синтаксис:

РазорватьСоединениеСВнешнимИсточникомДанных(<ИмяВнешнегоИсточникаДанных>)

Параметры:

<ИмяВнешнегоИсточникаДанных> (необязательный)

Тип: Строка.
Имя внешнего источника данных, с которым нужно разорвать соединение.

Описание:

Выполняет разрыв соединения с внешним источником данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
