Глобальный контекст (Global context)

УстановитьПреимущественноеИспользованиеОсновногоСервера (SetPreferableMainServerUse)

Доступен, начиная с версии 8.3.16.

Синтаксис:

УстановитьПреимущественноеИспользованиеОсновногоСервера(<Включить>)

Параметры:

<Включить> (обязательный)

Тип: Булево.
Определяет, будет ли включен режим преимущественного использования основного сервера:
  • Истина - режим включен;
  • Ложь - режим выключен.

Описание:

После вызова данного метода мобильный клиент с автономным режимом использует основной сервер, вне зависимости от приоритета установленного в конфигурации.
Метод УстановитьИспользуемыйСервер имеет более высокий приоритет.

Доступность:

Мобильный клиент, мобильный автономный сервер.
