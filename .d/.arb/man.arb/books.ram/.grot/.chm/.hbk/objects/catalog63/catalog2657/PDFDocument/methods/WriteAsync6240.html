ДокументPDF.ЗаписатьАсинх (PDFDocument.WriteAsync)

ДокументPDF (PDFDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <Пароль>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла для записи PDF-документа.

<Пароль> (необязательный)

Тип: Строка.
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: ТипФайлаДокументаPDF, Неопределено.
Тип файла документа PDF. Если содержит Неопределено, то в качестве значения параметра используется свойство ТипФайла объекта ДокументPDF.

Описание варианта метода:

Асинхронная функция. Записывает документ PDF.
Если документ имеет электронную цифровую подпись и выполняется запись с паролем, то генерируется исключение.

Доступность:
Тонкий клиент, толстый клиент, веб-клиент.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <Пароль>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для записи PDF-документа.

<Пароль> (необязательный)

Тип: Строка.
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: ТипФайлаДокументаPDF, Неопределено.
Тип файла документа PDF. Если содержит Неопределено, то в качестве значения параметра используется свойство ТипФайла объекта ДокументPDF.

Описание варианта метода:

Асинхронная функция. Записывает изменения в документ PDF.
Если документ имеет электронную цифровую подпись и выполняется запись с паролем, то генерируется исключение.

Доступность:
Тонкий клиент, толстый клиент, веб-клиент.

Возвращаемое значение:

Тип: Обещание.

Описание:

Асинхронная функция. Записывает изменения в PDF-документ.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
