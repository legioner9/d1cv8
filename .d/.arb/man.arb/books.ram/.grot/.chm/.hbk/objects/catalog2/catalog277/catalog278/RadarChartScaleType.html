ТипШкалыРадарнойДиаграммы (RadarChartScaleType)

Доступен, начиная с версии 8.0.

Значения

Окружность (Circle)
Полигон (Polygon)

Описание:

Определяет способ отображения линий в шкале радарных диаграмм.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: RadarChartScaleType.
