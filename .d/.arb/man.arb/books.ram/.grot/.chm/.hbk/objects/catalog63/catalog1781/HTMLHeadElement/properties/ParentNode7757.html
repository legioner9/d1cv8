ЭлементЗаголовокHTML.РодительскийУзел (HTMLHeadElement.ParentNode)

ЭлементЗаголовокHTML (HTMLHeadElement)

РодительскийУзел (ParentNode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ЭлементЗаголовокHTML, Неопределено.
Родительский узел узла HTML или Неопределено, в случае отсутствия такового.
Все узлы, за исключением Атрибут, Документ, ФрагментДокумента, могут иметь родительский узел.
Если узел только что был создан и еще не добавлен в дерево или был уже удален из него, родительский узел отсутствует.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
