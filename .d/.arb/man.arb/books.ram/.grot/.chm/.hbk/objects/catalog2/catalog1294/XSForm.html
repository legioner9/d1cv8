ФормаПредставленияXS (XSForm)

ФормаПредставленияXS (XSForm)

Доступен, начиная с версии 8.1.

Значения

Квалифицированная (Qualified)
Неквалифицированная (Unqualified)

Описание:

Содержит варианты форм представления элементов или атрибутов.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ОбъявлениеАтрибутаXS, свойство Форма
ОбъявлениеЭлементаXS, свойство Форма
СхемаXML, свойство ФормаАтрибутовПоУмолчанию
СхемаXML, свойство ФормаЭлементовПоУмолчанию

Использование в версии:

Доступен, начиная с версии 8.1.
