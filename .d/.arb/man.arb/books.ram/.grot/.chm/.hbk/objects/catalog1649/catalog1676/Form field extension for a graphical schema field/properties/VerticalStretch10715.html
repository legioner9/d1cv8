Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

РастягиватьПоВертикали (VerticalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Указывает возможность растягивания календаря по вертикали.
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
