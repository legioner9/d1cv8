Automation сервер.NewObject (Automation server.NewObject)

Automation сервер (Automation server)

NewObject (NewObject)

Доступен, начиная с версии 8.1.

Синтаксис:

NewObject(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя объекта, объявленного в конфигураторе.

Возвращаемое значение:

Тип: COMОбъект.

Описание:

Создает объект, для которого предусмотрен конструктор, и возвращает ссылку на него.

Доступность:

Интеграция.

Пример:

connector = Новый COMObject("V83.Application");
connector.Connect("File=c:\InfoBases\Trade;Usr=Director;")
ТДокумент = connector.NewObject("ТекстовыйДокумент");
ТДокумент.ВставитьСтроку(1, "Новая строка");
Сообщить(ТДокумент. КоличествоСтрок());

Использование в версии:

Доступен, начиная с версии 8.1.
