МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьВыгрузкуДанныхАсинх (GetDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьВыгрузкуДанныхАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания – ВыгрузкаДанныхСистемыВзаимодействия.

Описание:

Асинхронная функция. Получает выгрузку данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

МенеджерСистемыВзаимодействия, метод ПолучитьВыгрузкуДанных
