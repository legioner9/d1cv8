ЧтениеДанных.РазделитьНаЧастиПо (DataReader.SplitInPartsOf)

ЧтениеДанных (DataReader)

РазделитьНаЧастиПо (SplitInPartsOf)

Доступен, начиная с версии 8.3.9.

Синтаксис:

РазделитьНаЧастиПо(<РазмерЧасти>)

Параметры:

<РазмерЧасти> (обязательный)

Тип: Число.
Размер части данных в байтах.

Возвращаемое значение:

Тип: Массив.
Массив значений типа РезультатЧтенияДанных.

Описание:

Разделяет остаток данных на части заданного размера. Размер части указывается в байтах.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ЧтениеДанных, метод РазделитьНаЧастиПоАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.
