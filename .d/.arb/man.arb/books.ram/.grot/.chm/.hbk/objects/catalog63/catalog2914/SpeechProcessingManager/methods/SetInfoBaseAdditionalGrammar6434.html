МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьДополнительнуюГрамматикуИнформационнойБазы (SetInfoBaseAdditionalGrammar)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьДополнительнуюГрамматикуИнформационнойБазы(<Ключ>, <ДополнительнаяГрамматика>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.
Идентификатор грамматики. В случае если грамматика с таким ключом уже есть – грамматика будет заменена.

<ДополнительнаяГрамматика> (обязательный)

Тип: Массив.
Массив строк, дополняющих или заменяющих грамматику модели.

Описание:

Устанавливает дополнительную грамматику распознавания речи в информационной базе (с учетом используемых разделителей сеанса) в сервис распознавания речи. Дополнительная грамматика будет сразу доступна для использования.
Дополнительная грамматика хранится в служебной таблице базы данных с учетом установленных разделителей.
Требует наличия у текущего пользователя права "Администрирование данных".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
