МенеджерРасширенийКонфигурации.РасширенияКонфигурацииИзменены (ConfigurationExtensionsManager.ConfigurationExtensionsChanged)

МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

РасширенияКонфигурацииИзменены (ConfigurationExtensionsChanged)

Доступен, начиная с версии 8.3.22.

Синтаксис:

РасширенияКонфигурацииИзменены()

Возвращаемое значение:

Тип: Булево.

Описание:

Возвращает Истина, если у нового сеанса, запущенного в той же области данных, набор расширений отличается, т.е. с момента запуска сеанса был изменен состав активных расширений либо загружена новая версия одного из расширений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
