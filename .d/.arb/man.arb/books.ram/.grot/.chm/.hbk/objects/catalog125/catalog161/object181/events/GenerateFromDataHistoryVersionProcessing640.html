РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

ОбработкаФормированияПоВерсииИсторииДанных (GenerateFromDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОбработкаФормированияПоВерсииИсторииДанных(<ДанныеВерсии>, <НомерВерсии>, <КлючЗаписи>, <ИсключитьДанные>, <СтандартнаяОбработка>)

Параметры:

<ДанныеВерсии>

Тип: ФиксированнаяСтруктура.
Фиксированная структура с данными версии, по которой выполняется формирование.

<НомерВерсии>

Тип: Число.
Номер версии, по которой выполняется формирование.

<КлючЗаписи>

Тип: РегистрСведенийКлючЗаписи.<Имя регистра сведений>.
Ключ записи регистра сведений.

<ИсключитьДанные>

Тип: Массив.
Массив с именами полей, которые будут исключены из стандартной обработки.

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.
Значение по умолчанию: Истина.

Описание:

Возникает при формировании объекта по версии истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

МенеджерИсторииДанных, метод СформироватьПоВерсии

Использование в версии:

Доступен, начиная с версии 8.3.13.
