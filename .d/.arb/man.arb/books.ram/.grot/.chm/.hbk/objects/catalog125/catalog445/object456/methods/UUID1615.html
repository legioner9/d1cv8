ПланСчетовСсылка.<Имя плана счетов>.УникальныйИдентификатор (ChartOfAccountsRef.<Chart of accounts name>.UUID)

ПланСчетовСсылка.<Имя плана счетов> (ChartOfAccountsRef.<Chart of accounts name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: УникальныйИдентификатор.

Описание:

Получает уникальный идентификатор ссылки.
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера ПолучитьСсылку.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ПланСчетовМенеджер.<Имя плана счетов>, метод ПолучитьСсылку

Использование в версии:

Доступен, начиная с версии 8.0.
