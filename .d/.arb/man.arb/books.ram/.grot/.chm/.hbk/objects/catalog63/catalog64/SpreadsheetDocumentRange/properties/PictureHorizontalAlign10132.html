Описание:

Тип: ГоризонтальноеПоложение.
Определяет способ горизонтального выравнивания картинки внутри ячейки табличного документа. По умолчанию – Лево.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
