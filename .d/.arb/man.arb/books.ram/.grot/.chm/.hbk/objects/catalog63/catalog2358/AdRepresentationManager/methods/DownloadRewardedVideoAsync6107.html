МенеджерОтображенияРекламы (AdRepresentationManager)

ЗагрузитьВидеообъявлениеСВознаграждениемАсинх (DownloadRewardedVideoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗагрузитьВидеообъявлениеСВознаграждениемАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - тип Булево, признак загрузки видеообъявления.

Описание:

Асинхронная функция. Загружает из сети видеообъявление. По завершению загрузки вызывается обработчик.
После загрузки, при вызове метода ПоказатьВидеообъявлениеСВознаграждением видеообъявление будет показано без задержки на загрузку.
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
