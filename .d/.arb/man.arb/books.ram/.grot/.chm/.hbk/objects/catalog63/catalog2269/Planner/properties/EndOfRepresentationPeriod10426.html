Планировщик.КонецПериодаОтображения (Planner.EndOfRepresentationPeriod)

Планировщик (Planner)

КонецПериодаОтображения (EndOfRepresentationPeriod)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Дата.
Содержит конечную границу периода в планировщике. За указанную границу перемещение в планировщике невозможно. Если значение равно Неопределено, конечная граница периода планировщика не установлена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
