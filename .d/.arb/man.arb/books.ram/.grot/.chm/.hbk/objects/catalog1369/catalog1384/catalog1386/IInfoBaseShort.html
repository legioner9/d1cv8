Описание информационной базы (IInfoBaseShort)

Доступен, начиная с версии 8.1.

Свойства:

Descr (Descr)
Name (Name)

Описание:

Содержит параметры информационной базы, причем только те, для получения которых не требуется аутентификация пользователей информационных баз.
Представляет собой объект с интерфейсом IInfoBaseShort.

Доступность:

Интеграция.

Соединение с агентом сервера, метод UpdateInfoBase
Соединение с агентом сервера, метод GetInfoBaseConnections
Соединение с агентом сервера, метод GetInfoBaseSessions
Сеанс, свойство InfoBase
