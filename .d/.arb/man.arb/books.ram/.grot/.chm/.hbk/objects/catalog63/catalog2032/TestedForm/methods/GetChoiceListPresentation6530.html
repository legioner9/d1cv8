ТестируемаяФорма (TestedForm)

ПолучитьПредставлениеСпискаВыбора (GetChoiceListPresentation)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПолучитьПредставлениеСпискаВыбора()

Возвращаемое значение:

Тип: ФиксированныйМассив.
Возвращает фиксированный массив объектов типа Структура.
Имена полей структуры зависят от языка конфигурации менеджера тестирования.
Поля структуры:
  • ПредставлениеДанных (DataPresentation) - представление значения,
  • ОтображаемыйТекст (DisplayedText) – заданное представление значения.

Описание:

Метод возвращает массив соответствий представлений значений элементов списка выбора с их отображаемыми значениями.
Если представление не задано для какого-либо из элементов, в качестве представления передается представление самого значения.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
