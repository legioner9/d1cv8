СертификатКлиентаФайл (FileClientCertificate)

Доступен, начиная с версии 8.3.1.

Свойства:

ИмяФайла (FileName)

Конструкторы:

На основании файла

Описание:

Объект клиентского сертификата OpenSSL (с приватным ключом), загружаемого из файла.
Формат файла определяется по расширению имени файла:
*.PFX; *.P12 - файл сертификата формата P12;
*.PEM - файл сертификата формата PEM.
по умолчанию - файл сертификата формата PEM.
Также существует возможность указать в качестве сертификата клиента СертификатКриптографии, который можно выбрать в хранилище по отпечатку. Это может быть альтернативой использованию данного объекта.

Примечание 1. Использование https-соединения с клиентскими сертификатами не доступно на устройствах под управлением Windows 8.
Примечание 2. PXF-файлы во внутреннем формате КриптоПро не поддерживаются.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
