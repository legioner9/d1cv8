ЧтениеPDF.Закрыть (PDFReader.Close)

ЧтениеPDF (PDFReader)

Закрыть (Close)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый файл.
Закрытие файла выполняется автоматически при удалении объекта ЧтениеPDF.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

ЧтениеPDF, метод ЗакрытьАсинх
ЧтениеPDF, метод НачатьЗакрытие

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
