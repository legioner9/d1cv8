НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияОтбором
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки отбора.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Набор элементов управления отбором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля отбора, свойство НастройкаОтбора

Использование в версии:

Доступен, начиная с версии 8.0.
