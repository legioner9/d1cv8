Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Значение кодировки вычисляется из настроек конфигурационного файла. Если настройки в конфигурационном файле отсутствуют, используется значение UTF8.

Использование в версии:

Доступен, начиная с версии 8.3.9.
