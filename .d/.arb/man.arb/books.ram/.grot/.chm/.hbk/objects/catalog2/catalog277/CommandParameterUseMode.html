РежимИспользованияПараметраКоманды (CommandParameterUseMode)

Доступен, начиная с версии 8.2.

Значения

Множественный (Multiple)
Одиночный (Single)

Описание:

Содержит варианты режимов использования параметра команды.
Если свойствах команды РежимИспользованияПараметра и РежимИспользованияПараметра имеет значение Множественный, то в обработчик команды передается массив значений, а не одно значение даже, если в конкретном случае значение одно.
Массив значений может быть передан в том случае, если источником для параметра команды является динамический список в режиме множественного выделения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
