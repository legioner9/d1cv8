Расширение табличного поля значений параметров компоновки данных (Data composition available fields table box extension)

ОграниченияИспользования (UseRestrictions)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ОграниченияИспользованияДоступныхПараметровКомпоновкиДанных.
Содержит коллекцию ограничений для использования доступных параметров в данном табличном поле.
С помощью данной коллекции разработчик может исключить часть доступных полей из данного табличного поля. Таким образом, можно фильтровать набор доступных полей и не предоставлять пользователю те поля, использование которых нецелесообразно в данном конкретном случае.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
