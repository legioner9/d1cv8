ЗаписьДанных.ЗаписатьЦелое16 (DataWriter.WriteInt16)

ЗаписьДанных (DataWriter)

ЗаписатьЦелое16 (WriteInt16)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьЦелое16(<Число>, <ПорядокБайтов>)

Параметры:

<Число> (обязательный)

Тип: Число.
Число, которое будет записано в целевой поток.
Значение числа должно находиться в диапазоне от 0 до 65535.

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, который будет использован для кодировки числа при записи. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта ЗаписьДанных.
Значение по умолчанию: Неопределено.

Описание:

Записывает 16-разрядное число в целевой поток.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

ЗаписьДанных, метод НачатьЗаписьЦелого16
ЗаписьДанных, метод ЗаписатьЦелое16Асинх

Использование в версии:
