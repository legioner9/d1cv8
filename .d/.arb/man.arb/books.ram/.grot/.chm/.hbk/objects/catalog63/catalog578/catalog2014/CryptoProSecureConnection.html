ЗащищенноеСоединениеКриптоПро (CryptoProSecureConnection)

ЗащищенноеСоединениеКриптоПро (CryptoProSecureConnection)

Доступен, начиная с версии 8.3.24.

Свойства:

РежимПроверкиОтзываTLSСертификатаСервера (ServerTLSCertificateRevocationCheckMode)
СертификатКлиента (ClientCertificate)
СертификатыУдостоверяющихЦентров (CertificationAuthorityCertificates)

Конструкторы:

По объектам используемых сертификатов

Описание:

Объект защищенного соединения КриптоПро.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

ssl1 = Новый ЗащищенноеСоединениеКритоПро(
            Новый СертификатКлиентаОС,
            Новый СертификатыУдостоверяющихЦентровОС,
            РежимПроверкиОтзываСертификатовУдостоверяющихЦентров.Строгий);

ssl2 = Новый ЗащищенноеСоединениеКритоПро(
            Новый СертификатКлиентаОС(СпособВыбораСертификатаОС.Авто),
            Новый СертификатыУдостоверяющихЦентровФайл("C:/certs/myca.pem"));

ssl3 = Новый ЗащищенноеСоединениеКритоПро(
            Новый СертификатКлиентаФайл("C:/certs/user.p12", "password" ),
            Новый СертификатыУдостоверяющихЦентровФайл("C:/certs/myca.pem"));

ssl4 = Новый ЗащищенноеСоединениеКритоПро(
            неопределено,
            неопределено);
