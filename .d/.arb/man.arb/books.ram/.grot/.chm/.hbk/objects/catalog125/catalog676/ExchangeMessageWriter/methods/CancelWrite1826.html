ЗаписьСообщенияОбмена.ПрерватьЗапись (ExchangeMessageWriter.CancelWrite)

ЗаписьСообщенияОбмена (ExchangeMessageWriter)

ПрерватьЗапись (CancelWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПрерватьЗапись()

Описание:

Прерывает запись сообщения. Сообщение при этом считается не отправленным. Запись сообщения обмена может быть завершена либо обращением к методу ЗакончитьЗапись, либо обращением к методу ПрерватьЗапись.
Если запись была начата и не была завершена тем или иным способом перед удалением объекта ЗаписьСообщенияОбмена, то перед удалением производится неявный вызов метода ПрерватьЗапись. Аналогично, если не была завершена запись сообщения и было обращение к методу НачатьЗапись, то запись предыдущего сообщения прерывается.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ЗаписьСообщения = ПланыОбмена.СоздатьЗаписьСообщения();
ЗаписьXML = Новый ЗаписьXML;
ЗаписьXML.ОткрытьФайл(ИмяФайла);
ЗаписьСообщения.НачатьЗапись(ЗаписьXML, Получатель);
// Запись тела сообщения

Если Неудача Тогда
    ЗаписьСообщения.ПрерватьЗапись();
Иначе
    ЗаписьСообщения.ЗакончитьЗапись();
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
