ОбластьМакетаОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateArea)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ЭлементОбластиМакетаОформленияКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы области макета оформления.
Возможно обращение к элементу области макета оформления посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Используется для описания предопределенной области и является коллекцией элементов области макета оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/appearance-template}. Имя типа XDTO: AppearanceTemplateArea.

Использование в версии:

Доступен, начиная с версии 8.1.
