НеНазначать (DontAssign)

Доступен, начиная с версии 8.3.14.

Описание:

Рабочий сервер, для которого создано данное требование, не будет назначен для обслуживания объекта требования, подходящего под условия, заданные в требовании.

Использование в версии:

Доступен, начиная с версии 8.3.14.
