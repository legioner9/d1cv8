КоллекцияВариантовПользовательскогоПоляВыборКомпоновкиДанных (DataCompositionUserFieldCaseVariantCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ВариантПользовательскогоПоляВыборКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты пользовательского поля.
Возможно обращение к вариантам пользовательского поля посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция вариантов пользовательского поля выбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
