БизнесПроцессМенеджер.<Имя бизнес-процесса>.ПолучитьДанныеВыбора (BusinessProcessManager.<Business process name>.GetChoiceData)

БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ПолучитьДанныеВыбора (GetChoiceData)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьДанныеВыбора(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: Структура.
Содержит параметры для формирования списка значений.
Структура, в которой обязательно должны содержаться следующие ключи:
  • Отбор (Filter) - тип Структура. Отбор, используемый для поиска данных. Значения элементов отбора могут иметь тип Массив, СписокЗначений, ФиксированныйМассив или иной тип, не являющийся коллекцией; для типов Массив, СписокЗначений, ФиксированныйМассив сравнение выполняется с видом сравнения ВСписке, иначе вид сравнения Равно;
  • СтрокаПоиска (SearchString) - содержит строку, используемую при поиске данных. Для быстрого выбора содержит значение Неопределено;
  • ВыборГруппИЭлементов (ChoiceFoldersAndItems) (необязательный) - тип ИспользованиеГруппИЭлементов. Указывает использование групп и элементов;
  • СпособПоискаСтроки (StringSearchMode) (необязательный) - тип СпособПоискаСтрокиПриВводеПоСтроке. Указывает способ поиска при вводе по строке;
  • ПолнотекстовыйПоиск (FullTextSearch) (необязательный) - тип ПолнотекстовыйПоискПриВводеПоСтроке. Указывает необходимость использования полнотекстового поиска при вводе по строке;
  • ОграничиватьСтрокуПоиска (LimitSearchString) (необязательный) - тип Булево. Указывает необходимость применения ограничений на искомую строку при вводе по строке.

Возвращаемое значение:

Тип: СписокЗначений.

Описание:

Формирует список значений данных выбора при вводе по строке, автоподборе текста и быстром выборе.
Если требуется выполнить поиск более 32 слов или 1000 символов, то свойство ОграничиватьСтрокуПоиска требуется установить в значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Вызов метода выполняет обращение к серверу.

Примечание:

Если в модуле менеджера в методе указано два параметра, то будет вызван метод ПолучитьДанныеВыбора глобального контекста.
Выполнение метода приводит к вызову события ОбработкаПолученияДанныхВыбора.

В режиме совместимости конфигурации Версия8_3_17 и ниже рекомендуется устанавливать свойство ОграничиватьСтрокуПоиска в значение Истина, чтобы при выполнении сложных запросов генерировалась ошибка.

Использование в версии:

Доступен, начиная с версии 8.2.
