Описание:

Отображает точку маршрута бизнес-процесса ВыборВарианта, имеющую несколько выходов (вариантов), из которых в зависимости от значения возвращаемого параметра <Вариант> обработчика события ОбработкаВыбораВарианта выбирается только один.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
