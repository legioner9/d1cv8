Рабочий сервер.WorkingProcessMemoryLimit (IWorkingServerInfo.WorkingProcessMemoryLimit)

Рабочий сервер (IWorkingServerInfo)

WorkingProcessMemoryLimit (WorkingProcessMemoryLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит ограничение суммарного объема оперативной памяти, занятой рабочими процессами на этом сервере. Если суммарный объем оперативной памяти, занятой рабочими процессами, превысил эту величину, то на этом сервере новые рабочие процессы запускаться не будут.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
