Расширение поля формы для поля планировщика (Form field extension for a planner)

Перетаскивание (Drag)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Перетаскивание(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Дата>, <ЗначенияИзмерений>)

Параметры:

<ПараметрыПеретаскивания>

Тип: ПараметрыПеретаскивания.
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: Булево.
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.
Значение по умолчанию: Истина.

<Дата>

Тип: Дата.
Дата в планировщике, на которую произведено перетаскивание.

<ЗначенияИзмерений>

Тип: ФиксированноеСоответствие.
Коллекция значений измерений. В качестве ключа выступает Значение измерения. В качестве значения — Значение элемента измерения .

Описание:

Возникает при окончании перетаскивания в поле - приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
