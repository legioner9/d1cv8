ТочкиДиаграммы.Добавить (ChartPoints.Add)

ТочкиДиаграммы (ChartPoints)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.
Название точки.

Описание:

Добавляет точку в конец коллекции точек диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Диаграмма.Точки.Добавить("Март");

Использование в версии:

Доступен, начиная с версии 8.0.
