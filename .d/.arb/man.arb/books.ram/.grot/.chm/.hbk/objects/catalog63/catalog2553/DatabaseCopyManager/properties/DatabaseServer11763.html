МенеджерКопииБазыДанных (DatabaseCopyManager)

СерверБазыДанных (DatabaseServer)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Адрес сервера базы данных копии. Может содержать значение порта после имени (или IP-адреса) хоста и двоеточия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.16.
