ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ПредставленияПолейДополнительныхДанных (AdditionalDataFieldsPresentations)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Соответствие.
Представление полей, табличных частей, реквизитов табличных частей.
Элемент соответствия описывает представление поля, табличной части, реквизита табличной части:
  • Ключ – путь к полю, табличной части или реквизиту табличной части. Например, реквизит "Цена" табличной части "ПозицииЗаказа" будет иметь путь "ПозицииЗаказа.Цена", к которому через ":" добавлен код языка, на котором указано представление. Если код языка не указан, то представление, используемое по умолчанию, при отсутствии представления на необходимом языке.
  • Значение – Представление поля, табличной части или реквизита табличной части на указанном языке.
Если в соответствие включено представление поля, по которому нет данных, то генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
