ДопустимаяСтранаПолученияЛицензий (AvailableLicenseAcquisitionCountry)

Представление (Presentation)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: ФиксированныйМассив.
Содержит список вариантов наименования страны (тип Строка).
Элемент с индексом 0 - основное наименование страны.
Если допускаются альтернативные наименования, они перечислены в последующих элементах.
Например:
0 - Российская Федерация, 1 – Россия, 2 – РФ.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
