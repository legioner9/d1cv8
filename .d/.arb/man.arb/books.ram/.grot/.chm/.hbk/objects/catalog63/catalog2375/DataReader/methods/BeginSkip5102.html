ЧтениеДанных.НачатьПропуск (DataReader.BeginSkip)

ЧтениеДанных (DataReader)

НачатьПропуск (BeginSkip)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьПропуск(<ОписаниеОповещения>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <Количество> - тип Число. Фактическое число пропущенных байтов.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Количество> (обязательный)

Тип: Число.
Количество байтов, которые требуется пропустить.

Описание:

Начинает пропуск при чтении указанного количества байтов в потоке.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
