ТипОтображенияСерииСлояГеографическойСхемы (GeographicalSchemaLayerSeriesShowMode)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Описание:

Серия будет отображаться в виде картинки, которая выбирается по значению серии.
Значение серии должно иметь тип Картинка.

Использование в версии:

Доступен, начиная с версии 8.0.
