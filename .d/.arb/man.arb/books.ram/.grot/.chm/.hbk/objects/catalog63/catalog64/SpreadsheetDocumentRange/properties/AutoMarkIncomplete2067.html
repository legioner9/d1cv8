ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет, необходимо ли выделять ячейку, в которой не заполнено значение, либо оно выставлено в значение по умолчанию.
Если нет ни одной заполненной ячейки, будет установлено ОтметкаНезаполненного в Истина, иначе Ложь.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
