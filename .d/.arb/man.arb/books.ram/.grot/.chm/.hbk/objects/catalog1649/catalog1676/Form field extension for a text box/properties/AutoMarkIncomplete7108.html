Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Указывает, нужно ли формировать автоматически состояние свойства ОтметкаНезаполненного в зависимости от устанавливаемого или формируемого в поле ввода значения.
Если Истина, тогда при установке и формировании значения в поле ввода будет меняться состояние свойства ОтметкаНезаполненного: если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то ОтметкаНезаполненного будет выставлено в Истина, иначе - в Ложь.
В случае, если для поля в свойствах метаданных "Автоотметка незаполненного" установлено "Авто":
  • если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Выдавать ошибку", то свойство принимает значение Истина, и соответственно, ОтметкаНезаполненного должно зависеть от того, является ли значение реквизита значением по умолчанию;
  • если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Не проверять", то свойство ОтметкаНезаполненного должно принимать значение Ложь, и соответственно, ОтметкаНезаполненного не зависит от того, является ли значение реквизита значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
