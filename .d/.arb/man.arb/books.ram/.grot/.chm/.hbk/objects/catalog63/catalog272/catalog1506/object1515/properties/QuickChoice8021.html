БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ИспользованиеБыстрогоВыбора.
Свойство устанавливает способ выбора значения в поле ввода формы клиентского приложения, если свойство Быстрый Выбор поля формы клиентского приложения установлено в значение Авто.

Если свойство БыстрыйВыбор константы установлено в значение Авто, то при выборе значения учитывается свойство Быстрый выбор константы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
