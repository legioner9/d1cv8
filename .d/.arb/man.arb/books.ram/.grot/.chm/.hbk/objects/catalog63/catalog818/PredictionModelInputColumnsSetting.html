НастройкаВходныхКолонокМоделиПрогноза (PredictionModelInputColumnsSetting)

Доступен, начиная с версии 8.0.

Элементы коллекции:

НастройкаВходнойКолонкиМоделиПрогноза
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются настройки входных колонок.
Возможно обращение к настройке посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

<Имя колонки прогноза> (<Forecast column name>)

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Содержит соответствия колонок источника данных колонкам модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

МодельПрогнозаПоискПоследовательностей, свойство НастройкаВходныхКолонок
МодельПрогнозаДеревоРешений, свойство НастройкаВходныхКолонок

Использование в версии:

Доступен, начиная с версии 8.0.
