ПоведениеОбычнойГруппы (UsualGroupBehavior)

Авто (Auto)

Доступен, начиная с версии 8.3.20.

Описание:

Для формы отчета интепретируется следующим образом:
  • В тонком и веб-клиентах используется значение Обычная,
  • В мобильном клиенте:
    • Если задан заголовок группы, то интерпретируется как Свертываемая, при этом не учитывается значение свойства группы ОтображатьЗаголовок;
    • Если не задан заголовок группы, то интерпретируется как Обычная.
Для остальных форм используется значение Обычная.

Использование в версии:

Доступен, начиная с версии 8.3.20.
