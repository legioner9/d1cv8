ИзвлечениеТекста (TextExtraction)

Доступен, начиная с версии 8.1.

Свойства:

ИмяФайла (FileName)

Методы:

Записать (Write)
ПолучитьТекст (GetText)

Конструкторы:

По имени файла

Описание:

Предназначен для извлечения текста из файлов произвольного формата. Используется только для операционных систем MS Windows NT 4.0, 2000, XP, Vista, 7 и выше.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
