РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ЭлементУправления (Control)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Содержит элемент управления рисунка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Доступно только для рисунков типа Текст, если в рисунке хранится значение (свойство СодержитЗначение равно Истина).

Использование в версии:

Доступен, начиная с версии 8.0.
