СсылкаНаСущностьDOM.РодительскийУзел (DOMEntityReference.ParentNode)

СсылкаНаСущностьDOM (DOMEntityReference)

РодительскийУзел (ParentNode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: СсылкаНаСущностьDOM, Неопределено.
Родительский узел узла DOM или Неопределено в случае отсутствия такового.
Все узлы, за исключением Атрибут, Документ, ФрагментДокумента, Сущность и Нотация могут иметь родительский узел.
В любом случае если узел только что был создан и еще не добавлен в дерево или был уже удален из него, родительский узел отсутствует.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
