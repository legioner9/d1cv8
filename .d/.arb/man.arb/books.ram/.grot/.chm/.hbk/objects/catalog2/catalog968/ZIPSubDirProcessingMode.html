РежимОбработкиПодкаталоговZIP (ZIPSubDirProcessingMode)

Доступен, начиная с версии 8.0.

Значения

НеОбрабатывать (DontProcess)
ОбрабатыватьРекурсивно (ProcessRecursively)

Описание:

Содержит варианты обработки масок файлов в именах для упаковки в ZIP архив.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЗаписьZipФайла, метод Добавить

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
