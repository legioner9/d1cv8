Расширение поля формы для поля ввода (Form field extension for a text box)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Предоставляет доступ к управлению режимом выбора значения. Если свойство установлено в Истина, то выбор будет выполняться из выпадающего списка. Неопределено - автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
