ФорматированнаяСтрока (FormattedString)

На основании описания форматирования

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый ФорматированнаяСтрока(<Содержимое>, <Шрифт>, <ЦветТекста>, <ЦветФона>, <Ссылка>)

Параметры:

<Содержимое> (обязательный)

Тип: Строка, ФорматированнаяСтрока, Картинка, Массив.
Содержимое строки.
Если тип параметра Массив, то элементами массива должны быть значения типа Строка, ФорматированнаяСтрока или Картинка.
Результирующая строка является конкатенацией элементов массива.
Картинка должна содержаться в объекте БиблиотекаКартинок.

<Шрифт> (необязательный)

Тип: Шрифт.
Шрифт строки.

<ЦветТекста> (необязательный)

Тип: Цвет.
Цвет текста строки.

<ЦветФона> (необязательный)

Тип: Цвет.
Цвет фона строки.

<Ссылка> (необязательный)

Тип: Строка.
Если параметр задан и его значение не пустая строка, то конструируемая строка представляет собой гиперссылку, по нажатию на которой происходит переход по ссылке, заданной в этом параметре.

Описание:

Создает значение на основании описания форматирования.

Пример:

Текст = Новый ФорматированнаяСтрока("Предупреждение!", , 
        WebЦвета.Красный);

Использование в версии:

Доступен, начиная с версии 8.3.1.
