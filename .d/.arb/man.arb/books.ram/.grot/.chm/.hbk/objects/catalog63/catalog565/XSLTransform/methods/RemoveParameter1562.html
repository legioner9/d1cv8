ПреобразованиеXSL.УдалитьПараметр (XSLTransform.RemoveParameter)

ПреобразованиеXSL (XSLTransform)

УдалитьПараметр (RemoveParameter)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По полному имени

Синтаксис:

УдалитьПараметр(<ПолноеИмя>)

Параметры:

<ПолноеИмя> (обязательный)

Тип: Строка.
Полное имя параметра.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

УдалитьПараметр(<ЛокальноеИмя>, <URIПространстваИмен>)

Параметры:

<ЛокальноеИмя> (обязательный)

Тип: Строка.
Локальное имя параметра.

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен параметра.

Описание:

Удаляет значение параметра.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Отменяет ранее заданное значение параметра. Параметр принимает значение, определенное в описании XSL-преобразования.

Пример:

Преобразование.УдалитьПараметр("low_sales", 30000);

Использование в версии:

Доступен, начиная с версии 8.0.
