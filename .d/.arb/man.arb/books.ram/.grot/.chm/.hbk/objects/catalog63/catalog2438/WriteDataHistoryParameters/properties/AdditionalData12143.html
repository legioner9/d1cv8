ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ДополнительныеДанные (AdditionalData)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Структура.
Данные полей и табличных частей, добавляемые в версию истории данных.
Элемент структуры описывает добавляемое поле или табличную часть:
  • Ключ – имя поля или табличной части.
  • Значение – Значение поля или ТаблицаЗначений для табличной части. Если в переданной таблице значений есть колонка "НомерСтроки" или "LineNumber", то строки в таблице значений сортируются по данной колонке, выполняется проверка совпадения значения, полученного добавлением единицы к индексу строки, и значения в колонке номера строки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
