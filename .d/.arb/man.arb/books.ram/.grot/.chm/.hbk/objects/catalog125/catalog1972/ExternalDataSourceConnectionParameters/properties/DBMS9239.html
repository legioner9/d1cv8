ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

СУБД (DBMS)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.
Содержит тип используемой СУБД. Одно из:
  • MSSQLServer;
  • PostgreSQL;
  • IBMDB2;
  • OracleDatabase;
  • MySQL.
Также может содержать пустую строку. В этом случае используется тип СУБД "Прочее".
Также может содержать значение Неопределено. В таком случае тип СУБД получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
