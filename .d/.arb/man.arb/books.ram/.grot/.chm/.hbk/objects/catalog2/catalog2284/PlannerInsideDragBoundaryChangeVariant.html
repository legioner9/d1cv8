Доступен, начиная с версии 8.3.18.

Значения

Конец (End)
Начало (Begin)

Описание:

Описывает, какие границы изменяются при перетаскивании внутри поля планировщика.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
