ЧтениеJSON.Прочитать (JSONReader.Read)

ЧтениеJSON (JSONReader)

Прочитать (Read)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Прочитать()

Возвращаемое значение:

Тип: Булево.
Истина - очередной узел прочитан;
Ложь - достигнут конец текста JSON.

Описание:

Выполняет чтение значения JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.ОткрытьФайл("c:/docs/data.json");
ЧтениеJSON.Прочитать()

Использование в версии:

Доступен, начиная с версии 8.3.6.
