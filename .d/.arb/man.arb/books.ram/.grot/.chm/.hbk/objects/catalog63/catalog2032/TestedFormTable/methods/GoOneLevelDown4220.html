ПерейтиНаУровеньВниз (GoOneLevelDown)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПерейтиНаУровеньВниз(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: Соответствие.
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.
Значение по умолчанию: Неопределено.

Описание:

Выполняет переход на уровень вниз в иерархическом списке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
