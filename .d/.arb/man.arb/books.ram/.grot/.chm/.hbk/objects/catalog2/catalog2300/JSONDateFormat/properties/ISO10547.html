ФорматДатыJSON.ISO (JSONDateFormat.ISO)

ФорматДатыJSON (JSONDateFormat)

ISO (ISO)

Доступен, начиная с версии 8.3.6.

Описание:

Формат даты ISO вида: "ГГГГ-ММ-ДДTЧЧ:ММ:ССZ".
Например: "2009-02-15T00:00:00Z".
Дата сериализуется в дату стандарта UTC.

Использование в версии:

Доступен, начиная с версии 8.3.6.
