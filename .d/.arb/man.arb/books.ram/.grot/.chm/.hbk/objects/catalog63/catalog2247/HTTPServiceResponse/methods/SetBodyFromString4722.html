HTTPСервисОтвет.УстановитьТелоИзСтроки (HTTPServiceResponse.SetBodyFromString)

HTTPСервисОтвет (HTTPServiceResponse)

УстановитьТелоИзСтроки (SetBodyFromString)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТелоИзСтроки(<Тело>, <Кодировка>, <ИспользоватьBOM>)

Параметры:

<Тело> (обязательный)

Тип: Строка.
Строка, которая будет записана в тело ответа.

<Кодировка> (необязательный)

Тип: КодировкаТекста, Строка.
Определяет кодировку, в которой будет отправлено тело HTTP-ответа. Если параметр не задан, то тело отправляется в кодировке UTF8.
Установка этого параметра не вызывает автоматической установки соответствующего HTTP-заголовка.
Кодировка может быть задана как в виде значения перечисления КодировкаТекста, так и в виде строки с указанием названия кодировки.

<ИспользоватьBOM> (необязательный)

Тип: ИспользованиеByteOrderMark.
Использовать метку порядка байтов (BOM) для кодировок, которые ее поддерживают: UTF-8, UTF-16 и др.
В режиме совместимости конфигурации значение по умолчанию - Использовать.
Значение по умолчанию: Авто.

Описание:

Устанавливает тело ответа из строки.

Доступность:

Сервер, толстый клиент.
