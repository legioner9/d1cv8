Доступен, начиная с версии 8.0.

Значения

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)

Описание:

Содержит варианты границ элементов управления. Используется для управления привязками.

Доступность:

Толстый клиент.
