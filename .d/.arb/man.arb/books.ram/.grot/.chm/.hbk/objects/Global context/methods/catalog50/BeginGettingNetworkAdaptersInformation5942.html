Глобальный контекст (Global context)

НачатьПолучениеИнформацииОСетевыхАдаптерах (BeginGettingNetworkAdaptersInformation)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьПолучениеИнформацииОСетевыхАдаптерах(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения получения информации со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ИнформацияОСетевомАдаптере.

Описание:

Процедура заполняет массив объектов типа ИнформацияОСетевомАдаптере и передает в функцию, указанную в обработчике <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
