ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Сгруппировать (Group)

Доступен, начиная с версии 8.0.

Синтаксис:

Сгруппировать(<ИмяГруппы>, <РасположениеЗаголовка>)

Параметры:

<ИмяГруппы> (необязательный)

Тип: Строка.
Название группы.

<РасположениеЗаголовка> (необязательный)

Тип: РасположениеЗаголовкаГруппировкиТабличногоДокумента.
Расположение заголовка группировки.
Значение по умолчанию: Авто.

Описание:

Группирует область, состоящую из строк или колонок.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Область.Сгруппировать("Товар");

См. также:

ОбластьЯчеекТабличногоДокумента, метод Разгруппировать

Использование в версии:

Доступен, начиная с версии 8.0.
