Глобальный контекст (Global context)

ПолучитьВремяЗасыпанияПассивногоСеанса (GetPassiveSessionHibernateTime)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьВремяЗасыпанияПассивногоСеанса()

Возвращаемое значение:

Тип: Число.

Описание:

Получить интервал времени в секундах, по завершении которого неактивный сеанс переводится в спящий режим.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
