ПланыВидовХарактеристикМенеджер (ChartsOfCharacteristicTypesManager)

ПланыВидовХарактеристикМенеджер (ChartsOfCharacteristicTypesManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов характеристик.
Возможно обращение к менеджеру плана видов характеристик посредством оператора [...]. В качестве аргумента передается имя плана видов характеристик, как оно задано в конфигурации.

Свойства:

<Имя плана видов характеристик> (<Chart of characteristic types name>)

Методы:

ТипВсеСсылки (AllRefsType)

Описание:

Предназначен для управления планами видов характеристик и предоставляет доступ к значениям типа ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик>. Доступ к объекту осуществляется через свойство глобального контекста ПланыВидовХарактеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

Глобальный контекст, свойство ПланыВидовХарактеристик

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
