ГлубинаЦвета (ColorDepth)

ГлубинаЦвета (ColorDepth)

Доступен, начиная с версии 8.3.14.

Значения

БитНаПиксел1 (BitPerPixel1)
БитНаПиксел24 (BitPerPixel24)
БитНаПиксел32 (BitPerPixel32)
БитНаПиксел4 (BitPerPixel4)
БитНаПиксел8 (BitPerPixel8)

Описание:

Описывает поддерживаемые глубины цвета картинки.
При работе в режиме совместимости Версия8_3_13 поддерживается только английский вариант терма.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
