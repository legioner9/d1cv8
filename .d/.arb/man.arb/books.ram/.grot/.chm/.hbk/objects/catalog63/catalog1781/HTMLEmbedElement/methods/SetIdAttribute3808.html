ЭлементВставкаHTML.УстановитьАтрибутИдентификатор (HTMLEmbedElement.SetIdAttribute)

ЭлементВставкаHTML (HTMLEmbedElement)

УстановитьАтрибутИдентификатор (SetIdAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьАтрибутИдентификатор(<Имя>, <ЭтоИД>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя атрибута.

<ЭтоИД> (обязательный)

Тип: Булево.
Признак идентификатора.

Описание:

Метод устанавливает признак идентификатора атрибуту по его имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не меняет установленной в документе схемы.
Причины вызова исключений:
  • данный узел в режиме Только для чтения;
  • указанный атрибут не является атрибутом данного элемента.

Использование в версии:

Доступен, начиная с версии 8.2.
