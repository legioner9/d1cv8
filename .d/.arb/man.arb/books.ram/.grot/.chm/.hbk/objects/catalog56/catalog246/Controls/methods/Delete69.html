ЭлементыФормы (Controls)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: Число; Элемент формы.
Индекс элемента в коллекции элементов управления формы или сам элемент формы.

Описание:

Удаляет элемент управления с формы.

Доступность:

Толстый клиент.

Пример:

Для каждого Элемент Из ЭлементыФормы Цикл
    Если Найти(ВРег(Элемент.Имя),"ТМП") > 0 Тогда
        ЭлементыФормы.Удалить(Элемент);
    КонецЕсли;
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
