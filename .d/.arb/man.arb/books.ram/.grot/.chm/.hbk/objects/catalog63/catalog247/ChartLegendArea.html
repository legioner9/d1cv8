Доступен, начиная с версии 8.1.

Свойства:

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)
ПрозрачныйФон (Transparent)
Прокрутка (Scrolling)

ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области легенды диаграммы для настройки параметров оформления. Легенда диаграммы предназначена для описания серий диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
