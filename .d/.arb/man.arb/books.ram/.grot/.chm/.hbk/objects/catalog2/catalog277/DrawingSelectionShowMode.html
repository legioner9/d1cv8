РежимОтображенияВыделенияРисунков (DrawingSelectionShowMode)

Доступен, начиная с версии 8.3.17.

Значения

Авто (Auto)
НеОтображать (DontShow)
Отображать (Show)

Описание:

Содержит варианты режима отображения выделения рисунков.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/managed-application/logform. Имя типа XDTO: DrawingSelectionShowMode.

Использование в версии:

Доступен, начиная с версии 8.3.17.
