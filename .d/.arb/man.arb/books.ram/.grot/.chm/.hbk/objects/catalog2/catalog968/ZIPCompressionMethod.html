МетодСжатияZIP (ZIPCompressionMethod)

Доступен, начиная с версии 8.0.

Значения

BZIP2 (BZIP2)
Копирование (Copy)
Сжатие (Deflate)

Описание:

Содержит варианты методов сжатия записываемых ZIP файлов.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЗаписьZipФайла, метод Открыть
ЗаписьZipФайла, конструктор На основании имени файла

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
