Расширение табличного поля списка колонок анализа данных (Data analysis column list table box extension)

Доступен, начиная с версии 8.0.

Свойства:

НастройкаКолонок (ColumnsSetting)

Описание:

Дополнительные свойства элемента управления ТабличноеПоле, предназначенные для интерактивной работы со списком колонок анализа данных. Для данного расширения свойство ТекущаяСтрока возвращает значение типа КолонкаАнализаДанных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
