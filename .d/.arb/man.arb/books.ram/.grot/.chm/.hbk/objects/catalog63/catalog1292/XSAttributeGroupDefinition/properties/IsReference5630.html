ЭтоСсылка (IsReference)

Доступен, начиная с версии 8.1.

Только чтение.

Описание:

Тип: Булево.
Указывает, является ли данный компонент ссылкой на глобальную группу атрибутов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - компонент является ссылкой.

Использование в версии:

Доступен, начиная с версии 8.1.
