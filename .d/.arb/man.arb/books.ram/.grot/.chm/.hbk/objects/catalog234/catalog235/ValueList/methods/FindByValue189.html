СписокЗначений.НайтиПоЗначению (ValueList.FindByValue)

СписокЗначений (ValueList)

НайтиПоЗначению (FindByValue)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоЗначению(<ИскомоеЗначение>)

Параметры:

<ИскомоеЗначение> (обязательный)

Тип: Произвольный.
Выражение со значением, которое требуется найти.

Возвращаемое значение:

Тип: ЭлементСпискаЗначений, Неопределено.
Если ни один элемент не хранит значения, равного искомому, то возвращается значение Неопределено.

Описание:

Осуществляет поиск значения элемента списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Если ПолеСписка1.НайтиПоЗначению(ВыбЗначение) = Неопределено Тогда
    Сообщить("Значение не найдено!");
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
