УровеньСжатияZIP (ZIPCompressionLevel)

Доступен, начиная с версии 8.0.

Значения

Максимальный (Maximum)
Минимальный (Minimum)
Оптимальный (Optimal)

Описание:

Содержит варианты степени сжатия ZIP файлов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЗаписьZipФайла, метод Открыть
ЗаписьZipФайла, конструктор На основании имени файла

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
