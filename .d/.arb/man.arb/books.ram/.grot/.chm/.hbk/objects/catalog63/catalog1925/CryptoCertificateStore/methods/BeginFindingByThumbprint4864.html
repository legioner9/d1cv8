ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьПоискПоОтпечатку (BeginFindingByThumbprint)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПоискПоОтпечатку(<ОписаниеОповещения>, <Отпечаток>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения поиска со следующими параметрами:
  • <Сертификат> - СертификатКриптографии или Неопределено, если сертификат не найден.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Отпечаток> (обязательный)

Тип: ДвоичныеДанные.
Отпечаток, по которому производится поиск.

Описание:

Начинает поиск сертификатов в хранилище по заданному отпечатку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
