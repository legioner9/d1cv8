ТипДанныхXML (XMLDataType)

Доступен, начиная с версии 8.0.

Свойства:

URIПространстваИмен (NamespaceURI)
ИмяТипа (TypeName)

Конструкторы:

По имени типа XML и URI пространства имен

Описание:

Тип данных XML представляется двумя строковыми величинами: именем типа и URI пространства имен. Данный объект объединяет две эти величины.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером. Сериализуется.
