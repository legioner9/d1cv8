СоединениеИнформационнойБазы (InfoBaseConnection)

Пользователь (User)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: ПользовательИнформационнойБазы.
Пользователь текущего соединения с информационной базой.
Если пользователь для текущего соединения не задан, то в качестве пользователя соединения будет указан пользователь с пустым именем и идентификатором объекта, полученного методом НайтиПоИмени с указанием пустого имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
