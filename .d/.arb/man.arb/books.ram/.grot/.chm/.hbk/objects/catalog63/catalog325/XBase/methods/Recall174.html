XBase.Восстановить (XBase.Recall)

XBase (XBase)

Восстановить (Recall)

Доступен, начиная с версии 8.0.

Синтаксис:

Восстановить()

Описание:

Снимает с текущей записи пометку удаления.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применение метода для непомеченной на удаление записи не приводит ни к каким изменениям.

Пример:

// Восстановление всех удаленных записей

xB.ОтображатьУдаленные = истина;
xB.Первая();
Пока Не xB.ВКонце() Цикл
    xB.Восстановить();
    xB.Следующая();
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
