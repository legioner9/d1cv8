МакетКомпоновкиДанных.ТребуемоеВремяАктуальностиДанных (DataCompositionTemplate.RequiredDataRelevanceTime)

МакетКомпоновкиДанных (DataCompositionTemplate)

ТребуемоеВремяАктуальностиДанных (RequiredDataRelevanceTime)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Число, Дата.
Минимальное допустимое время актуальности данных, которые необходимо получить.
Свойство имеет смысл только если ТребуемаяАктуальностьДанных равна Авто или Любые.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.16.
