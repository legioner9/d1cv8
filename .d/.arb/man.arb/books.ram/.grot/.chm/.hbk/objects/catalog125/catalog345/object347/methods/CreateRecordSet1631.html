ПоследовательностьМенеджер.<Имя последовательности>.СоздатьНаборЗаписей (SequenceManager.<Sequence name>.CreateRecordSet)

ПоследовательностьМенеджер.<Имя последовательности> (SequenceManager.<Sequence name>)

СоздатьНаборЗаписей (CreateRecordSet)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьНаборЗаписей()

Возвращаемое значение:

Тип: ПоследовательностьНаборЗаписей.<Имя последовательности>.

Описание:

Создает набор записей последовательности.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Набор записей создается пустым.

Использование в версии:

Доступен, начиная с версии 8.0.
