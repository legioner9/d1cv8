КоллекцияДействийЭлементаПланировщика.Добавить (PlannerItemActionCollection.Add)

КоллекцияДействийЭлементаПланировщика (PlannerItemActionCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Добавить(<Значение>, <Текст>, <Картинка>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.
Значение действия планировщика.

<Текст> (необязательный)

Тип: Строка, ФорматированнаяСтрока.
Текст, отображаемый в действии планировщика.

<Картинка> (необязательный)

Тип: Картинка.
Картинка, отображаемая в действии планировщика.

Возвращаемое значение:

Тип: ДействиеЭлементаПланировщика.

Описание:

Добавляет новое действие в конец коллекции действий элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
