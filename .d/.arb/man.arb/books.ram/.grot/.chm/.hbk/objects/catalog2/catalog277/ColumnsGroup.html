ГруппировкаКолонок (ColumnsGroup)

Доступен, начиная с версии 8.2.

Значения

Вертикальная (Vertical)
ВЯчейке (InCell)
Горизонтальная (Horizontal)

Описание:

Содержит варианты группировки колонок в группе вида ГруппаКолонок формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение группы формы для группы колонок, свойство Группировка

Использование в версии:

Доступен, начиная с версии 8.2.
