Доступен, начиная с версии 8.3.5.

Элементы коллекции:

СоединениеИсточникаЗапросаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются соединения источника.
Возможно обращение к соединению источника посредством оператора [...]. В качестве аргумента передается индекс.

Добавить (Add)
Индекс (IndexOf)
Количество (Count)

Получить (Get)
Удалить (Delete)

Описание:

Коллекция соединений источников схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
