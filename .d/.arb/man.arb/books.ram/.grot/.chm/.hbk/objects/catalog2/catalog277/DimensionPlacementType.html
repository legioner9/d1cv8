ТипРазмещенияИзмерений (DimensionPlacementType)

Доступен, начиная с версии 8.0.

Значения

Вместе (Together)
Отдельно (Separately)
ОтдельноИТолькоВИтогах (SeparatelyAndInTotalsOnly)

Описание:

Содержит варианты расположения измерений в сводной таблице и построителе отчета.
Для объекта СводнаяТаблица значения Отдельно и ОтдельноИТолькоВИтогах интерпретируются как одно и то же значение - ОтдельноИТолькоВИтогах.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.

См. также:

СводнаяТаблица, свойство РазмещениеИзмеренийВСтроках
СводнаяТаблица, свойство РазмещениеИзмеренийВКолонках
ПостроительОтчета, свойство РазмещениеИзмеренийВСтроках
ПостроительОтчета, свойство РазмещениеИзмеренийВКолонках

Использование в версии:

Доступен, начиная с версии 8.0.
