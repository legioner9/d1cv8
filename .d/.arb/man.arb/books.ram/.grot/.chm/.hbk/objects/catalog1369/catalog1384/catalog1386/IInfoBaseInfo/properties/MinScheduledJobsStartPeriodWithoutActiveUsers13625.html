Информационная база.MinScheduledJobsStartPeriodWithoutActiveUsers (IInfoBaseInfo.MinScheduledJobsStartPeriodWithoutActiveUsers)

Информационная база (IInfoBaseInfo)

MinScheduledJobsStartPeriodWithoutActiveUsers (MinScheduledJobsStartPeriodWithoutActiveUsers)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Определяет минимальный период запуска каждого регламентного задания информационной базы в секундах, если в информационной базе нет ни одного неспящего сеанса пользователей.
Значение по умолчанию: 0.
Значение должно находиться в диапазоне от 0 до 4294967.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
