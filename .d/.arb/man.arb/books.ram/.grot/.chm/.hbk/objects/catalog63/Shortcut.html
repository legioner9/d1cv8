СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.0.

Свойства:

Alt (Alt)
Ctrl (Ctrl)
Shift (Shift)
Клавиша (Key)

Конструкторы:

На основании сочетания клавиш

Описание:

Предназначен для получения и хранения сочетания клавиш быстрого доступа к элементам управления. Используется для определения свойств других объектов, имеющих тип СочетаниеКлавиш.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: ShortCutType.
