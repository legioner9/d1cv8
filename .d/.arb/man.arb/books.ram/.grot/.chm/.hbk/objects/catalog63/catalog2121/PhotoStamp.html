Доступен, начиная с версии 8.3.15.

Свойства:

ОтметкаДаты (DateStamp)
Текст (Text)
ФорматДаты (DateFormat)

Конструкторы:

По умолчанию

Описание:

Объект, описывающий подпись, которая будет размещена на фотографии. Дата от произвольного текста отделяется символом перевода строки. Цвет надписи - оранжевый.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваМультимедиа, метод СделатьФотоснимок
