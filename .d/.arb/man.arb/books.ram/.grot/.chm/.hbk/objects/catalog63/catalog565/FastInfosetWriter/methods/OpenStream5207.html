ЗаписьFastInfoset.ОткрытьПоток (FastInfosetWriter.OpenStream)

ЗаписьFastInfoset (FastInfosetWriter)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОткрытьПоток(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для записи.

Описание:

Устанавливает поток для записи данных в формате Fast Infoset.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
