Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ИзменятьАвтоОбновление (ChangeAutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если свойство установлено как Ложь, то пользователь не имеет доступа к интерактивному управлению параметрами автообновления, устанавливаемыми при конфигурировании на закладке "Прочее" окна редактирования списка.

Доступность:

Толстый клиент.
