ИдентификаторПодписчикаДоставляемыхУведомлений (DeliverableNotificationSubscriberID)

Доступен, начиная с версии 8.3.6.

Свойства:

ИдентификаторИнформационнойБазы (InfoBaseID)
ИдентификаторПриложения (ApplicationID)
ИдентификаторУстройства (DeviceID)
ТипПодписчика (SubscriberType)

Описание:

Определяет подписчика доставляемых уведомлений: конкретную информационную базу в определенном приложении на конкретном устройстве.
Может быть получен с помощью метода ПолучитьИдентификаторПодписчикаУведомлений.
Используется в методе Отправить.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен (http://v8.1c.ru/8.3/data/ext). Имя типа XDTO: DeliverableNotificationSubscriberID.

Использование в версии:

Доступен, начиная с версии 8.3.6.
