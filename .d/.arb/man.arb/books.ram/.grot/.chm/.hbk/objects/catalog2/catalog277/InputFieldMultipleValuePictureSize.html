РазмерКартинкиМножественногоЗначенияПоляВвода (InputFieldMultipleValuePictureSize)

Доступен, начиная с версии 8.3.23.

Значения

Авто (Auto)
Крупный (Large)
Маленький (Small)
Средний (Medium)

Описание:

Содержит варианты размеров области отображения картинки множественного значения поля ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
