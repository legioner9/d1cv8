МенеджерФайловыхПотоков.Открыть (FileStreamsManager.Open)

МенеджерФайловыхПотоков (FileStreamsManager)

Открыть (Open)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: С указанием режима открытия

Синтаксис:

Открыть(<РежимОткрытия>, <ИмяФайла>, <РазмерБуфера>)

Параметры:

<РежимОткрытия> (обязательный)

Тип: РежимОткрытияФайла.
Режим открытия файла.

<ИмяФайла> (обязательный)

Тип: Строка.
Имя открываемого файла.

<РазмерБуфера> (необязательный)

Тип: Число.
Размер буфера для операций с файлом.

Описание варианта метода:

Открывает файл в заданном режиме с возможностью чтения и записи.
Файл открывается в режиме разделяемого чтения.

Вариант синтаксиса: С указанием режимов открытия и доступа

Синтаксис:

Открыть(<РежимДоступа>, <ИмяФайла>, <РежимОткрытия>, <РазмерБуфера>)

Параметры:

<РежимДоступа> (обязательный)

Тип: ДоступКФайлу.
Режим доступа к файлу.

<ИмяФайла> (обязательный)

Тип: Строка.
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: РежимОткрытияФайла.
Режим открытия файла.

<РазмерБуфера> (необязательный)

Тип: Число.
Размер буфера для операций с файлом.

Описание варианта метода:

Открыть файл в выбранном режиме, с заданным уровнем доступа, с общим доступом на чтение.

Возвращаемое значение:

Тип: ФайловыйПоток.

Описание:

Открывает файл в заданном режиме с возможностью чтения и записи.
Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
