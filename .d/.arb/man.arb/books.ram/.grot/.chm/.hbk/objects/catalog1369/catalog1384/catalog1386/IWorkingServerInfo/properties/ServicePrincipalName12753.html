Рабочий сервер.ServicePrincipalName (IWorkingServerInfo.ServicePrincipalName)

Рабочий сервер (IWorkingServerInfo)

ServicePrincipalName (ServicePrincipalName)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Имя сервиса (Service Principal Name, SPN) сервера 1С:Предприятия. Используется в процессе аутентификации пользователя информационной базы средствами операционной системы по протоколу Kerberos.
По умолчанию - Пустая строка.
Установка непустого значения требуется при настройке аутентификации операционной системы http-клиентов с ограниченным делегированием пользователя клиента на веб-сервере (delegation to specified services only), если используется сервер 1С:Предприятия под управлением Windows.

Доступность:

Интеграция.
