Доступен, начиная с версии 8.1.

Элементы коллекции:

ЯчейкаМакетаЗаголовкаКоллекцииЗначенийОбластиКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ячейки.
Возможно обращение к ячейке посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция ячеек заголовка коллекции значений области компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
