ЗаписьFastInfoset.ЗаписатьТекст (FastInfosetWriter.WriteText)

ЗаписьFastInfoset (FastInfosetWriter)

ЗаписатьТекст (WriteText)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Текст, помещаемый в элемент XML.

Описание:

Записывает текст элемента. Перед вызовом данного метода должен быть вызван метод ЗаписьFastInfoset.ЗаписатьНачалоЭлемента, так как иначе непонятно, к какому элементу относится текст. Для одного элемента данный метод может вызываться неоднократно. То есть текст элемента может записываться по частям.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Запись.ЗаписатьНачалоЭлемента("fruits");
Запись.ЗаписатьНачалоЭлемента("apple");
Запись.ЗаписатьТекст("Очень вкусно");
Запись.ЗаписатьКонецЭлемента();
Запись.ЗаписатьКонецЭлемента();

Использование в версии:

Доступен, начиная с версии 8.1.
