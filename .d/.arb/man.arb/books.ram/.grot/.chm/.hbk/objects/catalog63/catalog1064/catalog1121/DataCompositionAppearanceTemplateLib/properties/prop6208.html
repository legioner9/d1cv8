Использование:

Только чтение.

Описание:

Тип: МакетОформленияКомпоновкиДанных.
Набор свойств содержит макеты оформления, созданные при конфигурировании прикладной задачи и содержащиеся в папке "Общие макеты" дерева метаданных, а также предопределенные макеты оформления. Имена свойств совпадают с именами макетов, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
