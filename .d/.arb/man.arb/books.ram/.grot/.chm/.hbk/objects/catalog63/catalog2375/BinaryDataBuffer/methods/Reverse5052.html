БуферДвоичныхДанных.Перевернуть (BinaryDataBuffer.Reverse)

БуферДвоичныхДанных (BinaryDataBuffer)

Перевернуть (Reverse)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Перевернуть()

Возвращаемое значение:

Тип: БуферДвоичныхДанных.

Описание:

Создает новый буфер, содержащий элементы текущего буфера в противоположном порядке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
