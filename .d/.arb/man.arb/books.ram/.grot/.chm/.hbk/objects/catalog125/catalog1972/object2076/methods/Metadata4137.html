Метаданные (Metadata)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: ТаблицаИзмерения.

Описание:

Предоставляет доступ к объекту описания метаданных таблицы измерения куба внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
