ЧтениеHTML.УстановитьСтроку (HTMLReader.SetString)

ЧтениеHTML (HTMLReader)

УстановитьСтроку (SetString)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьСтроку(<СтрокаHTML>, <Кодировка>)

Параметры:

<СтрокаHTML> (обязательный)

Тип: Строка.
Строка, содержащая текст HTML.

<Кодировка> (необязательный)

Тип: Строка.
Указывает кодировку, которая будет использована в механизме разбора HTML для преобразования.
Значение по умолчанию: UTF8.

Описание:

Устанавливает строку, содержащую текст HTML для чтения данным объектом. Если перед вызовом данного метода уже производилось чтение HTML из другого файла или строки, то чтение прекращается и объект инициализируется для чтения из указанной строки.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ЧтениеHTML = Новый ЧтениеHTML;
ЧтениеHTML.УстановитьСтроку(СтрHTML);

Использование в версии:

Доступен, начиная с версии 8.2.
