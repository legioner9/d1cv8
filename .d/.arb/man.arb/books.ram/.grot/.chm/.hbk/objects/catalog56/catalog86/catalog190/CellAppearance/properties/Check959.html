ОформлениеЯчейки.Флажок (CellAppearance.Check)

ОформлениеЯчейки (CellAppearance)

Флажок (Check)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число, Булево.
Содержит значение флажка.

Доступность:

Толстый клиент.

Примечание:

Флажок со значением типа Булево может иметь только два состояния (Истина - установлен, Ложь - снят), со значением типа Число может иметь три состояния (1 - установлен, 0 - снят, 2 - не определено).

Использование в версии:

Доступен, начиная с версии 8.0.
