РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии> (AccountingRegisterManager.<Accounting register name>)

УстановитьМинимальныйПериодРассчитанныхИтогов (SetMinTotalsPeriod)

Доступен, начиная с версии 8.3.3.

Синтаксис:

УстановитьМинимальныйПериодРассчитанныхИтогов(<Период>)

Параметры:

<Период> (обязательный)

Тип: Дата.
Минимальный период, начиная с которого будут рассчитаны итоги. Можно передать любую дату, она будет приведена к началу месяца.
Если указана дата 01.01.2000 то это значит, что будут рассчитаны итоги на 00:00:00 01.01.2000 и последующие периоды.

Описание:

Устанавливает период. начиная с которого рассчитаны итоги. Влияет на производительность получения остатков и оборотов по данному регистру, а также на производительность регламентных операция типа восстановления базы данных. При установке может выполнятся расчет итогов за новые открытые периоды. Периодичность расчета месяц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Регистры.Остатки.УстановитьМинимальныйПериодРассчитанныхИтогов(ВыбДата);
