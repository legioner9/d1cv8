ПараметрыОбменаДанными (DataExchangeParameters)

Доступен, начиная с версии 8.0.

Свойства:

Загрузка (Load)
Отправитель (Sender)
Получатели (Recipients)

Описание:

Объект не может быть создан самостоятельно. Объект данного типа представляет собой значение свойства ОбменДанными объектов, представляющих данные, по которым ведется учет изменений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Объект.ОбменДанными.Получатели.Заполнить();

См. также:

УдалениеОбъекта, свойство ОбменДанными

Использование в версии:

Доступен, начиная с версии 8.0.
