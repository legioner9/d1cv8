ПриЧтенииНаСервере (OnReadAtServer)

Расширение формы клиентского приложения для записи таблицы внешнего источника данных

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: РегистрСведенийМенеджерЗаписи.<Имя регистра сведений>.
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
