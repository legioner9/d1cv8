СерияДиаграммы.ФорматВТаблицеДанных (ChartSeries.DataTableFormat)

СерияДиаграммы (ChartSeries)

ФорматВТаблицеДанных (DataTableFormat)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Задает формат значений этой серии в таблице данных. Если формат пуст, то используется формат общий для диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
