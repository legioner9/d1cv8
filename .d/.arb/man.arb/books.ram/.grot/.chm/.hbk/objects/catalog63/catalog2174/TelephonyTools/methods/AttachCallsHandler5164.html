СредстваТелефонии (TelephonyTools)

ПодключитьОбработчикЗвонков (AttachCallsHandler)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПодключитьОбработчикЗвонков(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: Строка, ОписаниеОповещения.
Имя процедуры, подключаемой в качестве обработчика.

Описание:

Выполняет подключение обработчика звонков.
Обработчик вызывается со следующими параметрами:

  • НомерТелефона - тип Строка. Номер телефона, с которого приходит вызов или на который отправляется вызов.
  • Дата - тип Дата. Время возникновения события.
  • ВариантСобытия - тип ВариантСобытияЗвонкаСредствТелефонии. Одно из значений перечисления.
  • ТипЗвонка - тип ТипЗвонкаЖурналаЗвонков: входящий, исходящий или пропущенный (если вызов был завершен при входящем звонке).
  • Дополнительный параметр (если используется ОписаниеОповещения).
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяОбработкаЗвонков.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
