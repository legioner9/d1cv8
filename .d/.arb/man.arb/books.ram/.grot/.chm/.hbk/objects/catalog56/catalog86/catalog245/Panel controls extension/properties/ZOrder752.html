Расширение элементов управления, расположенных на панели (Panel controls extension)

ПорядокОтображения (ZOrder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит порядок отображения элемента управления. Совокупность порядков отображения элементов управления, расположенных на одной панели, определяет, какой элемент окажется "сверху" в случае, если элементы наложены друг на друга.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
