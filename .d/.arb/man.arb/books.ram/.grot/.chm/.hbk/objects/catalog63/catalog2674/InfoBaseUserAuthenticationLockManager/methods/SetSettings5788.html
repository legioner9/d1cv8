МенеджерБлокировкиАутентификацииПользователейИнформационнойБазы (InfoBaseUserAuthenticationLockManager)

УстановитьНастройки (SetSettings)

Доступен, начиная с версии 8.3.16.

Синтаксис:

УстановитьНастройки(<НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы>)

Параметры:

<НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы> (необязательный)

Тип: НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы.
Устанавливаемые настройки блокировки аутентификации пользователей.

Описание:

Устанавливает настройки блокировки аутентификации пользователей для информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
