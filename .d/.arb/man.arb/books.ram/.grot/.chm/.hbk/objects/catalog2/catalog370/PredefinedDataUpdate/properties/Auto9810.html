ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Авто (Auto)

Доступен, начиная с версии 8.3.3.

Описание:

Фактическое значение вычисляется автоматически.
Для главного узла информационной базы - значение будет равно ОбновлятьАвтоматически, для периферийного узла информационной базы будет равно НеОбновлятьАвтоматически.

Использование в версии:

Доступен, начиная с версии 8.3.3.
