Доступен, начиная с версии 8.3.5.

Элементы коллекции:

КонтрольнаяТочкаИтоговСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются контрольные точки итогов.
Возможно обращение к контрольной точке итогов посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
НайтиПоВыражению (FindByExpression)
НайтиПоИмениКолонки (FindByColumnName)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция контрольных точек итогов.
Представляет собой список полей итогов из выражения ИТОГИ ПО языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
