ОбъектМетаданных: КаналСервисаИнтеграции (MetadataObject: IntegrationServiceChannel)

НаправлениеСообщения (MessageDirection)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: НаправлениеСообщенияКаналаСервисаИнтеграции.
Если Отправка - канал сервиса интеграции доступен для отправки сообщений и не доступен для получения сообщений. Если для канала установлен обработчик полученных сообщений, при использовании конфигурации выводится сообщение об ошибке.
Если Получение - канал сервиса интеграции доступен для получения сообщения и не доступен для отправки сообщения. Если для канала не установлен обработчик полученных сообщений, при использовании конфигурации выводится сообщение об ошибке, а при вызове метода ОтправитьСообщение генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
