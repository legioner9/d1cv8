Планировщик (Planner)

ФиксироватьЗаголовокИзмерений (FixDimensionsHeader)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Определяет, будет ли зафиксирован заголовок измерений планировщика при прокрутке. Изменение данного свойства также распространяется и на фиксацию перенесенных заголовков шкалы времени.
Неопределено - фиксация заголовка измерений планировщика определяется автоматически.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение Неопределено заполняется по умолчанию, в том числе и для уже добавленных на форму реквизитов типа Планировщик.
В веб-клиенте под управлением Internet Explorer свойство всегда имеет значение Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.15.
