ИсточникиСхемыЗапроса (QuerySchemaSources)

Удалить (Delete)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс источника, который требуется удалить.

Описание варианта метода:

Удаляет источник по индексу.

Вариант синтаксиса: По псевдониму

Синтаксис:

Удалить(<Псевдоним>)

Параметры:

<Псевдоним> (обязательный)

Тип: Строка.
Псевдоним источника, который требуется удалить.

Описание варианта метода:

Удаляет источник по псевдониму.

Описание:

Удаляет элемент из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
