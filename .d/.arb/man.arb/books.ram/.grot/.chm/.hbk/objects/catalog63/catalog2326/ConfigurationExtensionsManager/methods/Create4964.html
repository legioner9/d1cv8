МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

Создать (Create)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Создать(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: ДвоичныеДанные.
Данные файла формата *.cfe.

Возвращаемое значение:

Тип: РасширениеКонфигурации.

Описание:

Создает новый объект типа РасширениеКонфигурации.
Для объектов, созданных с указанием параметра Данные:

  • свойства Версия, Имя, Назначение, ОсновныеРоли, Синоним и ХешСумма имеют такое же значение, какое будет у расширения конфигурации, установленного из файла с идентичными двоичными данными,
  • метод ИзменяетСтруктуруДанных возвращает Истина, если после установки расширение будет влиять на структуру данных конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
