ДинамическийСписок (DynamicList)

Поля (Fields)

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).

Использование:

Только чтение.

Описание:

Тип: ПоляНабораДанныхСхемыКомпоновкиДанных.
Содержит описания полей (набора данных) динамического списка.

Свойства Роль, ВыражениеПредставления, НаборДанныхПроверкиИерархии и ПараметрНабораДанныхПроверкиИерархии не поддерживаются в динамическом списке.

Кроме того, при установленной основной таблице:
  • для многих стандартных полей (например, Ссылка, Наименование, ПометкаУдаления) не поддерживается установка значение свойства Поле в Истина,
  • для ключевых полей, стандартных полей сортировки и поля Родитель дополнительно не поддерживается установка значений свойств Условие и Порядок в Истина.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).
