ВариантыНастроекКомпоновкиДанных (DataCompositionSettingsVariants)

Доступен, начиная с версии 8.2.

Элементы коллекции:

ВариантНастроекКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты настроек.
Возможно обращение к варианту настроек посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция вариантов настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

СхемаКомпоновкиДанных, свойство ВариантыНастроек

Использование в версии:

Доступен, начиная с версии 8.2.
