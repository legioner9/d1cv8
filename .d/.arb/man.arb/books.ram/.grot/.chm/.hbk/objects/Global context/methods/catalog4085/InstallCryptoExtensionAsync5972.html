Глобальный контекст (Global context)

УстановитьРасширениеРаботыСКриптографиейАсинх (InstallCryptoExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьРасширениеРаботыСКриптографиейАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Устанавливает расширение для работы с криптографией в браузер.
В режиме запуска толстого и тонкого клиентов, а так же в клиенте мобильного приложения информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
