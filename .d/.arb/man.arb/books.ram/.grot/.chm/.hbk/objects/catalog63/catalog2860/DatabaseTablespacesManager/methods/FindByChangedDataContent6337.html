МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

НайтиПоИзмененномуСоставуДанных (FindByChangedDataContent)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиПоИзмененномуСоставуДанных(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: ЭлементСоставаТабличногоПространстваБазыДанных, ОбъектМетаданных.
Объект метаданных, для которого выполняется поиск табличного пространства.

Возвращаемое значение:

Тип: МенеджерТабличногоПространстваБазыДанных.

Описание:

Возвращает табличное пространство, которое будет хранить данные таблиц объекта метаданных после обновления размещения таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
