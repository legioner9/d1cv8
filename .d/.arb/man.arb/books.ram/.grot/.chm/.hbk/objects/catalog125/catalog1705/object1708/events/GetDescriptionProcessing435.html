ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ОбработкаПолученияОписания (GetDescriptionProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаПолученияОписания(<КлючОбъекта>, <КлючНастроек>, <ОписаниеНастроек>, <Пользователь>)

Параметры:

<КлючОбъекта>

Тип: Произвольный.
Ключ объекта, для которого получается описание.

<КлючНастроек>

Тип: Произвольный.
Ключ настроек.

<ОписаниеНастроек>

Тип: ОписаниеНастроек.
Описание настроек.

<Пользователь>

Тип: Строка.
Идентификатор пользователя, для которого получается описание.
Если не указан или представляет пустую строку, то следует загружать настройки для текущего пользователя.

Описание:

Возникает при получении описания настроек.
В данном обработчике необходимо реализовать получение описания настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
