РегистрНакопленияСписок.<Имя регистра накопления> (AccumulationRegisterList.<Accumulation register name>)

Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Порядок.
Содержит объект Порядок, по которому осуществляется текущее упорядочивание записей регистра в списке. В качестве свойств объекта могут выступать предопределенные поля Период, Регистратор, НомерСтроки, Активность, а также измерения, ресурсы и реквизиты регистра. Имена свойств совпадают с именами полей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
