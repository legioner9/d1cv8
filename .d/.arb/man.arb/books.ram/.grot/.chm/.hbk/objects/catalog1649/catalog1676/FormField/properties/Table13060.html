ПолеФормы.Таблица (FormField.Table)

ПолеФормы (FormField)

Таблица (Table)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: ТаблицаФормы.
Для объекта с значением свойства Вид, равным ПолеДиаграммыГанта, возвращается объект типа ТаблицаФормы, содержащий предопределенную таблицу поля, для остальных - выбрасывается исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
