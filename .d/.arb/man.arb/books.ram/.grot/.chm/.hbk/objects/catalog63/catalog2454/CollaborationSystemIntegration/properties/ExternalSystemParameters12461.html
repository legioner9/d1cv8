ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

ПараметрыВнешнейСистемы (ExternalSystemParameters)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: Соответствие, Неопределено.
Параметры интеграции с внешним мессенджером. Значение свойства зависит от типа внешней системы.
Если интеграция получена без права "Администрирование данных" (или не в привилегированном режиме), то содержит Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
