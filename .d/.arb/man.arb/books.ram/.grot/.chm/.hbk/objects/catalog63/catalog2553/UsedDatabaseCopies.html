ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Доступен, начиная с версии 8.3.19.

Элементы коллекции:

ИспользуемаяКопияБазыДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются .
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя копии.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Представляет собой коллекцию объектов типа ИспользуемаяКопияБазыДанных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
