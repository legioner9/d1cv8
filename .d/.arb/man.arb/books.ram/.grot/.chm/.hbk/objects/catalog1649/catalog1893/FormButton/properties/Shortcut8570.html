КнопкаФормы.СочетаниеКлавиш (FormButton.Shortcut)

КнопкаФормы (FormButton)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: СочетаниеКлавиш.
Содержит сочетание клавиш кнопки. Действие сочетания клавиш приводит к выполнению команды, связанной с данной кнопкой. Если у команды назначено сочетание клавиш и у кнопки, связанной с командой, назначено сочетание клавиш, то для выполнения команды можно выполнить любое из этих сочетаний клавиш.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
