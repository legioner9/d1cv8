ФорматКартинки (PictureFormat)

Доступен, начиная с версии 8.0.

Значения

BMP (BMP)
EMF (EMF)
GIF (GIF)
Icon (Icon)
JPEG (JPEG)
PNG (PNG)
SVG (SVG)
TIFF (TIFF)
WMF (WMF)
НеизвестныйФормат (UnknownFormat)

Описание:

Содержит варианты форматов хранения картинки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: PictureFormat.
