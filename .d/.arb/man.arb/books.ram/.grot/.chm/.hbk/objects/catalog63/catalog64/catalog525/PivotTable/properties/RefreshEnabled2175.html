СводнаяТаблица (PivotTable)

Обновление (RefreshEnabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Признак, отвечающий за необходимость обновления данных табличного документа при изменениях состава измерений на осях и ресурсов.
Истина - требуется обновление.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Перед установкой колонок рекомендуется установить Ложь.
После завершения настройки установите Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
