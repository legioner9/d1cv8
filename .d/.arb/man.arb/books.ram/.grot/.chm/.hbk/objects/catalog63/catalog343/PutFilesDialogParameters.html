ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

Доступен, начиная с версии 8.3.15.

Свойства:

Заголовок (Title)
ИндексФильтра (FilterIndex)
МножественныйВыбор (MultipleChoice)
Фильтр (Filter)

Конструкторы:

По заголовку, множественному выбору, фильтру и индексу фильтра
По умолчанию

Описание:

Содержит параметры диалога помещения файлов.
Используется в качестве параметра <ПараметрыДиалога> методов НачатьПомещениеФайлаНаСервер и НачатьПомещениеФайловНаСервер.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
