ЧтениеДанных.НачатьКопированиеВ (DataReader.BeginCopyTo)

ЧтениеДанных (DataReader)

НачатьКопированиеВ (BeginCopyTo)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьКопированиеВ(<ОписаниеОповещения>, <Поток>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Описание процедуры, которая будет вызвана после завершения операции.
Процедура должна содержать следующие параметры:
  • <Количество> - тип Число. Фактическое количество скопированных байтов;
  • <Дополнительные параметры> - значение, которое было указано при создании объекта ОписаниеОповещения

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток: ЗаписьДанных.
Поток или объект ЗаписьДанных, в который требуется скопировать данные.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется скопировать. Если не задано, то будут скопированы все данные до конца вложенного потока.

Описание:

Начинает копирование заданного количества байтов из объекта ЧтениеДанных в указанный поток или объект ЗаписьДанных.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ЧтениеДанных, метод КопироватьВ
ЧтениеДанных, метод КопироватьВАсинх

Использование в версии:

Доступен, начиная с версии 8.3.10.
