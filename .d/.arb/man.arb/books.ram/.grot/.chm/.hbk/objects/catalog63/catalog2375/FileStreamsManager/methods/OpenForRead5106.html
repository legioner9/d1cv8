МенеджерФайловыхПотоков.ОткрытьДляЧтения (FileStreamsManager.OpenForRead)

МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьДляЧтения (OpenForRead)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ОткрытьДляЧтения(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя открываемого файла.

Возвращаемое значение:

Тип: ФайловыйПоток.

Описание:

Открывает существующий файл для чтения с общим доступом на чтение.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерФайловыхПотоков, метод НачатьОткрытиеДляЧтения
МенеджерФайловыхПотоков, метод ОткрытьДляЧтенияАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
