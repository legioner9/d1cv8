Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

ПолучитьРекомендуемуюШиринуСжатия (GetRecommendedReductionWidth)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьРекомендуемуюШиринуСжатия()

Возвращаемое значение:

Тип: Число.

Описание:

Возвращает ширину поля табличного документа в средних символах табличного документа.
Результат данного метода рекомендуется использовать в качестве параметра метода табличного документа ПреобразоватьВКомпактныйДокумент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

ТабличныйДокумент, метод ПреобразоватьВКомпактныйДокумент

Использование в версии:

Доступен, начиная с версии 8.3.14.
