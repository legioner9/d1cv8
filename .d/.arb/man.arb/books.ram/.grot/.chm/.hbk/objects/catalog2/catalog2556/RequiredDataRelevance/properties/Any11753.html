Любые (Any)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание:

Требование по актуальности уточняется с помощью параметра ТребуемоеВремяАктуальностиДанных. Запрос может выполняться как на основной базе данных, так и на копии.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
