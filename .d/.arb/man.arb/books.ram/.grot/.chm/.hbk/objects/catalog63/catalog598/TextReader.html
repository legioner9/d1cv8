ЧтениеТекста (TextReader)

Доступен, начиная с версии 8.1.

Методы:

Закрыть (Close)
Открыть (Open)
Прочитать (Read)
ПрочитатьСтроку (ReadLine)

Конструкторы:

На основании потока
По имени файла
Формирование неинициализированного объекта

Описание:

Предназначен для последовательного чтения текстовых файлов (большой длины).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Текст = Новый ЧтениеТекста("d:\win.txt", КодировкаТекста.ANSI);
Стр = Текст.ПрочитатьСтроку();
Пока Стр <> Неопределено Цикл // строки читаются до символа перевода строки

    Сообщить(Стр);
    Стр = Текст.ПрочитатьСтроку();
КонецЦикла;
