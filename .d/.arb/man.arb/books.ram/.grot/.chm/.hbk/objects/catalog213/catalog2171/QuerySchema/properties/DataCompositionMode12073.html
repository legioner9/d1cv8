СхемаЗапроса (QuerySchema)

РежимКомпоновкиДанных (DataCompositionMode)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Управляет режимом схемы запроса.
Если Истина, то для каждого запроса в пакете запросов схемы запроса будут недоступны поля ОбщиеИтоги, КонтрольныеТочкиИтогов и ВыраженияИтогов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
