ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьПолучениеВсех (BeginGettingAll)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеВсех(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения получения сертификатов со следующими параметрами:
  • <Сертификаты> - массив объектов СертификатКриптографии.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Начинает получение массива всех сертификатов криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
