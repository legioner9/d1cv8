Automation сервер.Connect (Automation server.Connect)

Automation сервер (Automation server)

Connect (Connect)

Доступен, начиная с версии 8.1.

Синтаксис:

Connect(<СтрокаСоединения>)

Параметры:

<СтрокаСоединения> (обязательный)

Тип: Строка.
Строка параметров (Строка соединения), используемая 1С:Предприятием для соединения с информационной базой.

Возвращаемое значение:

Тип: Булево.
Истина - инициализация прошла удачно, Ложь - в противном случае.

Описание:

Выполняет соединение системы 1С:Предприятие с информационной базой.

Доступность:

Интеграция.

Пример:

// Пример приводится на языке MS Visual Basic:

Dim connector As Object
Set connector=CreateObject("V83.Application")
result=connector.Connect("File=c:\InfoBases\Trade;Usr=Director;")

Использование в версии:

Доступен, начиная с версии 8.1.
