Глобальный контекст (Global context)

ПолучитьФункциональнуюОпциюИнтерфейса (GetInterfaceFunctionalOption)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьФункциональнуюОпциюИнтерфейса(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя функциональной опции.

Возвращаемое значение:

Тип: Произвольный.

Описание:

Получает значение функциональной опции применительно к командному интерфейсу.
Для получения значения опции используются параметры, установленные с помощью метода УстановитьПараметрыФункциональныхОпцийИнтерфейса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
Вызов метода выполняет обращение к серверу.

Примечание:

Результат выполнения не кэшируется, вызов сервера осуществляется при каждом вызове метода.
Если в качестве параметра функциональной опции интерфейса была передана ссылка на группу, а реквизит, в котором хранится значение функциональной опции, не используется для группы, то при получении значения функциональной опции будет возращено Null.

Использование в версии:

Доступен, начиная с версии 8.2.
