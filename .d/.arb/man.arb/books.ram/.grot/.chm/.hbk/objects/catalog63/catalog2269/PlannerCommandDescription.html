ОписаниеКомандыПоляПланировщика (PlannerCommandDescription)

Доступен, начиная с версии 8.3.23.

Свойства:

Доступность (Enabled)
Картинка (Picture)
Команда (Command)
Пометка (Check)
Представление (Presentation)

Конструкторы:

По умолчанию

Описание:

Описывает команду планировщика. Необходим для наполнения командами контекстного меню планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Расширение поля формы для поля планировщика
