Планировщик (Planner)

ТекущиеПериодыОтображения (CurrentRepresentationPeriods)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: КоллекцияТекущихПериодовОтображенияПланировщика.
Коллекция, содержащая элементы типа ПериодОтображенияПланировщика. Диапазон дат, отображаемый планировщиком, определяется, исходя из элементов этой коллекции. Некоторый диапазон будет отображён, если он входит в хотя бы один из периодов отображения, находящихся в коллекции.
По умолчанию в коллекции содержится один элемент с датами, вычисленными на основании текущей даты, а также свойств ЕдиницаПериодическогоВарианта и КратностьПериодическогоВарианта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Планировщик, свойство КратностьПериодическогоВарианта
Планировщик, свойство ЕдиницаПериодическогоВарианта

Использование в версии:

Доступен, начиная с версии 8.3.6.
