ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Разблокировать (Unlock)

Доступен, начиная с версии 8.0.

Синтаксис:

Разблокировать()

Описание:

Выполняет разблокировку счета, если он заблокирован данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Объект.Разблокировать();

Использование в версии:

Доступен, начиная с версии 8.0.
