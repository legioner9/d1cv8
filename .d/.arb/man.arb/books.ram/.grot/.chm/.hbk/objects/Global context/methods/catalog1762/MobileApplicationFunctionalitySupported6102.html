Глобальный контекст (Global context)

ПоддерживаетсяФункциональностьМобильногоПриложения (MobileApplicationFunctionalitySupported)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПоддерживаетсяФункциональностьМобильногоПриложения(<ЗапрашиваемаяФункциональностьМобильногоПриложения>)

Параметры:

<ЗапрашиваемаяФункциональностьМобильногоПриложения> (обязательный)

Тип: ФункциональностьМобильногоПриложения.
Функциональность мобильного приложения, поддержка которой проверяется.

Возвращаемое значение:

Тип: Булево.
И на клиенте, и на сервере возвращает одно и то же (и при разных конфигурациях) значение, объединяя установки на клиенте и сервере "по И".
Для толстого, тонкого и веб-клиентов всегда возвращает Ложь.

Описание:

Проверяет, была ли включена запрошенная функциональность в мобильном приложении или мобильном клиенте во время сборки.
В мобильном клиенте также сопоставляет с информацией на сервере. Если на сервере данная функциональность выключена, то и на клиенте метод вернет значение Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
