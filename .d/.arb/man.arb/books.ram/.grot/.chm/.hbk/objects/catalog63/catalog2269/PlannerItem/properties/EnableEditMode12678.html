ЭлементПланировщика (PlannerItem)

РежимРазрешенияРедактирования (EnableEditMode)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: РежимРазрешенияРедактированияЭлементаПланировщика.
Определяет режим редактирования элемента планировщика. Значение свойства не влияет на копирование элемента планировщика и на перетаскивание элемента планировщика вовне и извне поля планировщика. При копировании элемента планировщика данное свойство заполняется значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
