ЭлементБуфераОбмена (ClipboardItem)

По умолчанию

Доступен, начиная с версии 8.3.24.

Синтаксис:

Новый ЭлементБуфераОбмена(<ФорматДанных>, <Данные>)

Параметры:

<ФорматДанных> (обязательный)

Тип: СтандартныйФорматДанныхБуфераОбмена, Строка.
Вариант стандартного формата данных буфера обмена или строка, устанавливающая произвольный формат данных.

В веб-клиенте значение параметра типа Строка должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

<Данные> (обязательный)

Тип: Строка, ДвоичныеДанные, Картинка.
Данные типа Строка могут быть использованы только, если указан формат данных типа Текст или HTML.

Данные типа ДвоичныеДанные могут быть использованы только, если указан формат данных типа Строка.

Данные типа Картинка могут быть использованы только, если указан формат данных типа Картинка.

Использование в версии:

Доступен, начиная с версии 8.3.24.
