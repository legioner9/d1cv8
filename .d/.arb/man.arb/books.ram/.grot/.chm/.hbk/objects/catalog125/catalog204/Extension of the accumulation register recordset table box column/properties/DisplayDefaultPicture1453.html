Расширение колонки набора записей регистра накопления (Extension of the accumulation register recordset table box column)

ОтображатьСтандартнуюКартинку (DisplayDefaultPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак отображения в колонке стандартной картинки, обозначающей запись регистра накопления и ее состояние.

Доступность:

Толстый клиент.

Примечание:

Если Истина, то при выводе свойство КартинкиСтрок колонки табличного поля не используется.

Использование в версии:

Доступен, начиная с версии 8.0.
