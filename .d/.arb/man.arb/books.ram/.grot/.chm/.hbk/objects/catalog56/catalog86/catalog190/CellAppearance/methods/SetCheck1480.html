УстановитьФлажок (SetCheck)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьФлажок(<Флажок>)

Параметры:

<Флажок> (обязательный)

Тип: Булево, Число.
Значение флажка.

Устанавливает свойство Флажок объекта. Устанавливает свойство ОтображатьФлажок в значение Истина.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
