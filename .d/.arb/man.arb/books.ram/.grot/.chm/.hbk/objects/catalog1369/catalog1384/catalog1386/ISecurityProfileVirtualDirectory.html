Виртуальный каталог (ISecurityProfileVirtualDirectory)

Доступен, начиная с версии 8.3.3.

Свойства:

Alias (Alias)
AllowedRead (AllowedRead)
AllowedWrite (AllowedWrite)
Descr (Descr)
PhysicalPath (PhysicalPath)

Описание:

Относится к определенному профилю безопасности и определяет один виртуальный каталог, доступный информационной базе.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
