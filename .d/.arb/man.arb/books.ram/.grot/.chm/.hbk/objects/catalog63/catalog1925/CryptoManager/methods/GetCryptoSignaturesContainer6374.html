МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии (CryptoManager.GetCryptoSignaturesContainer)

МенеджерКриптографии (CryptoManager)

ПолучитьКонтейнерПодписейКриптографии (GetCryptoSignaturesContainer)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьКонтейнерПодписейКриптографии(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись для разбора.
Если параметр типа Строка, то обозначает имя файла с подписью.
Если параметр типа ДвоичныеДанные или Поток, ПотокВПамяти, ФайловыйПоток, то он задает данные подписи.

Возвращаемое значение:

Тип: КонтейнерПодписейКриптографии.

Описание:

Выполняет разбор двоичных данных подписи.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

МенеджерКриптографии.Подписать(
    ИсходныеДанные, РезультатПодписи, Сертификат);
Контейнер =
    МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии(
        РезультатПодписи);
