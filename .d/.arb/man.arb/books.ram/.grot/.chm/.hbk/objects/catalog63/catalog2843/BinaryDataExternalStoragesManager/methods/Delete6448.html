МенеджерВнешнихХранилищДвоичныхДанных.Удалить (BinaryDataExternalStoragesManager.Delete)

МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Удалить (Delete)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Удалить(<ВнешнееХранилищеДвоичныхДанных>)

Параметры:

<ВнешнееХранилищеДвоичныхДанных> (обязательный)

Тип: МенеджерВнешнегоХранилищаДвоичныхДанных.
Менеджер внешнего хранилища двоичных данных.

Возвращаемое значение:

Нет

Описание:

Удаляет внешнее хранилище двоичных данных.
Требуется режим монопольного доступа к информационной базе.

Доступность:

Сервер.

Примечание:

При попытке повторного удаления будет сгенерировано исключение.

Удаленное внешнее хранилище можно восстановить: для этого необходимо вызвать метод Добавить) с такими же параметрами <Имя>, <ПараметрыПодключения> и <ПараметрыДоступа> - в таком случае все BLOB-объекты будут перезаписаны и станут доступны из внешнего хранилища.

Пример:

cp = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cp.URL = "https://bucket.example.com";
cp.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

ap = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных();
ap.ИдентификаторДоступа = "EXAMPLE";
ap.СекретныйКлюч = "SecretKey";
ap.Регион = "US";

Мнджр = ВнешниеХранилищаДвоичныхДанных.Добавить("manager", cp, ap);

Мнджр = ВнешниеХранилищаДвоичныхДанных["manager"];
ВнешниеХранилищаДвоичныхДанных.Удалить(Мнджр);
