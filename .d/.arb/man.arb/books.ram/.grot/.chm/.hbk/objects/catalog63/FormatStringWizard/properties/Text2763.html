КонструкторФорматнойСтроки.Текст (FormatStringWizard.Text)

КонструкторФорматнойСтроки (FormatStringWizard)

Текст (Text)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Свойство содержит форматную строку в виде текста.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Для русского, украинского и казахского языка интерфейса объект будет генерировать строку, используя русские спецификаторы форматной строки, для всех остальных языков интерфейса будут использованы английские спецификаторы.

Использование в версии:

Доступен, начиная с версии 8.0.
