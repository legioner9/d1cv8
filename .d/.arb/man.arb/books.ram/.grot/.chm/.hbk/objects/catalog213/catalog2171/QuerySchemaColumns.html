КолонкиСхемыЗапроса (QuerySchemaColumns)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

КолонкаСхемыЗапроса, КолонкаВложеннаяТаблицаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.
Возможно обращение к колонке посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция колонок схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
