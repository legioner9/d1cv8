МенеджерИсторииДанных.ПолучитьМетаданные (DataHistoryManager.GetMetadata)

МенеджерИсторииДанных (DataHistoryManager)

ПолучитьМетаданные (GetMetadata)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: Получить метаданные версии

Синтаксис:

ПолучитьМетаданные(<Данные>, <НомерВерсии>)

Параметры:

<Данные> (обязательный)

Тип: СправочникСсылка.<Имя справочника>, ДокументСсылка.<Имя документа>, РегистрСведенийКлючЗаписи.<Имя регистра сведений>, БизнесПроцессСсылка.<Имя бизнес-процесса>, ЗадачаСсылка.<Имя задачи>, ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>, ПланСчетовСсылка.<Имя плана счетов>, ПланОбменаСсылка.<Имя плана обмена>, ПланВидовРасчетаСсылка.<Имя плана видов расчета>, КонстантаКлючЗначения.<Имя константы>.
Отбор версий метаданных по ссылке на объект или ключу записи регистра сведений.

<НомерВерсии> (обязательный)

Тип: Число.
Версия метаданных.

Описание варианта метода:

Получить метаданные версии с указанным номером.

Вариант синтаксиса: Получить актуальные метаданные

Синтаксис:

ПолучитьМетаданные(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: ОбъектМетаданных: Справочник, ОбъектМетаданных: Документ, ОбъектМетаданных: РегистрСведений, ОбъектМетаданных: БизнесПроцесс, ОбъектМетаданных: Задача, ОбъектМетаданных: ПланСчетов, ОбъектМетаданных: ПланВидовХарактеристик, ОбъектМетаданных: ПланОбмена, ОбъектМетаданных: ПланВидовРасчета, ОбъектМетаданных: Константа.
Объект метаданных.

Описание варианта метода:

Получить актуальные метаданные.

Вариант синтаксиса: Получить метаданные версий указанных данных

Синтаксис:

ПолучитьМетаданные(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: СправочникСсылка.<Имя справочника>, ДокументСсылка.<Имя документа>, РегистрСведенийКлючЗаписи.<Имя регистра сведений>, БизнесПроцессСсылка.<Имя бизнес-процесса>, ЗадачаСсылка.<Имя задачи>, ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>, ПланСчетовСсылка.<Имя плана счетов>, ПланОбменаСсылка.<Имя плана обмена>, ПланВидовРасчетаСсылка.<Имя плана видов расчета>, КонстантаКлючЗначения.<Имя константы>.
Отбор версий метаданных по ссылке на объект или ключу записи регистра сведений.

Описание варианта метода:

Получить актуальную версию метаданных, расширенную описаниями полей и табличных частей дополнительных данных, добавленных хотя бы в одну из версий данных, созданных по указанным данным.

Возвращаемое значение:

Тип: Структура.
Структура вида:
  • Представление (Presentation) - тип Строка. Обязательный. Представление объекта метаданных.
  • Поля (Fields) - тип Структура. Содержит все поля версии метаданных. Ключом структуры выступает имя поля, значением - представление поля. Если метаданные версии не содержат реквизиты, элемент отсутствует.
  • ТабличныеЧасти (TabularSections) - тип Структура. Содержит табличные части версии метаданных. Ключ структуры - имя табличной части. Значение - структура табличной части. Если метаданные версии не содержат табличных частей, элемент отсутствует.

Структура табличной части содержит поля:
  • Представление (Presentation) - тип Строка. Представление табличной части.
  • Поля (Fields) - тип Структура. Содержит все поля табличной части версии метаданных. Ключ - имя поля, значение - представление поля. Если метаданные версии не содержат полей табличной части, элемент отсутствует.

Описание:

Получает метаданные на момент создания версии данных или актуальные метаданные.
В версии хранятся синонимы данных на всех языках конфигурации. При получении метаданных возвращается представление на языке, установленном для пользователя.
Имя стандартного реквизита возвращается в варианте встроенного языка конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Вызов метода выполняет обращение к серверу.

Пример:

// Выбор метаданных версии


МетаданныеВерсии = ИсторияДанных.ПолучитьМетаданные(
    Справочники.Валюты.НайтиПоНаименованию("Рубль"), 1);

// Выбор актуальных метаданных


АктуальныеМетаданные = ИсторияДанных.ПолучитьМетаданные(
    Метаданные.Справочники.Валюты);

Использование в версии:

Доступен, начиная с версии 8.3.10.
