ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

НайтиПоНомеру (FindByNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоНомеру(<НомерЗадачи>)

Параметры:

<НомерЗадачи> (обязательный)

Тип: Число, Строка.
Номер искомой задачи. Тип передаваемого значения зависит от установки в конфигураторе представления номера задачи.

Возвращаемое значение:

Тип: ЗадачаСсылка.<Имя задачи>, Неопределено.
Ссылка на найденную задачу. Если задача не найдена, возвращается пустая ссылка.
Если номер не задан (длина = 0), то будет возвращено Неопределено.

Описание:

Осуществляет поиск задачи по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
