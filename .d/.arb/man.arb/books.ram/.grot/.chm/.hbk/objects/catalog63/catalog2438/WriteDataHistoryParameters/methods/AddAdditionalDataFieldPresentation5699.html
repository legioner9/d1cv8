ПараметрыЗаписиИсторииДанных.ДобавитьПредставлениеПоляДополнительныхДанных (WriteDataHistoryParameters.AddAdditionalDataFieldPresentation)

ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ДобавитьПредставлениеПоляДополнительныхДанных (AddAdditionalDataFieldPresentation)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: Добавление представление на указанном языке

Синтаксис:

ДобавитьПредставлениеПоляДополнительныхДанных(<Имя>, <КодЯзыка>, <Представление>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого поля или табличной части.

<КодЯзыка> (обязательный)

Тип: Строка.
Код языка, на котором добавляется представление.

<Представление> (необязательный)

Тип: Строка.
Представление на указанном языке.

Вариант синтаксиса: Добавление представления по умолчанию

Синтаксис:

ДобавитьПредставлениеПоляДополнительныхДанных(<Имя>, <Представление>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя добавляемого поля или табличной части.

<Представление> (обязательный)

Тип: Строка.
Представление, используемое по умолчанию. При отсутствии представления на требуемом языке, поведение отлично от конфигуратора, в котором нет возможности указать представление по умолчанию.

Описание варианта метода:

Добавление представления, используемого по умолчанию.

Описание:

Добавляет представление поля дополнительных данных в запись истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
