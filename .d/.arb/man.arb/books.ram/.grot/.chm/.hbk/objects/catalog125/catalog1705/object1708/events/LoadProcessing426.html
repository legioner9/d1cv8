ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ОбработкаЗагрузки (LoadProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаЗагрузки(<КлючОбъекта>, <КлючНастроек>, <Настройки>, <ОписаниеНастроек>, <Пользователь>)

Параметры:

<КлючОбъекта>

Тип: Произвольный.
Ключ объекта, для которого загружаются настройки.

<КлючНастроек>

Тип: Произвольный.
Ключ настроек.

<Настройки>

Тип: Произвольный.
Содержит загруженные настройки.

<ОписаниеНастроек>

Тип: ОписаниеНастроек, Неопределено.
Описание настроек.
Неопределено - описание настроек сохранять не нужно.

<Пользователь>

Тип: Строка.
Идентификатор пользователя, для которого загружаются настройки.
Если не указан или представляет пустую строку, то будут загружаться настройки для текущего пользователя.

Описание:

Возникает при загрузке настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
