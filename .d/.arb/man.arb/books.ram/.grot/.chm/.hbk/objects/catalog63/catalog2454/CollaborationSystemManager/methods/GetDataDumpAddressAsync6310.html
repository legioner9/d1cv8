МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьАдресВыгрузкиДанныхАсинх (GetDataDumpAddressAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьАдресВыгрузкиДанныхАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания – Строка.
Ссылка (адрес) на скачивание выгрузки, если она расположена в хранилище, доступ к которому предоставлен по протоколу Amazon S3 (https://aws.amazon.com/ru/s3/). (Полученная ссылка имеет ограниченное время жизни, которое определяется настройками хранилища S3).
Или путь к файлу в файловой системе сервера системы взаимодействия.

Описание:

Асинхронная функция. Получает адрес выгрузки данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

МенеджерСистемыВзаимодействия, метод ПолучитьАдресВыгрузкиДанных

Использование в версии:

Доступен, начиная с версии 8.3.22.
