Расширение формы списка записей регистра накопления (Accumulation register record list form extension)

Доступен, начиная с версии 8.0.

Свойства:

ПараметрОтборПоРегистратору (FilterByRecorderParameter)
ПараметрТекущаяСтрока (CurrentLineParameter)

Описание:

Дополнительные свойства объекта Форма, в том случае, если основной реквизит формы имеет тип РегистрНакопленияСписок.<Имя регистра накопления>.

Доступность:

Толстый клиент.

Пример:

Приходные = Документы.ПриходнаяНакладная;
Док = Приходные.НайтиПоНомеру("00003",ТекущаяДата());
Остатки = РегистрыНакопления.УчетНоменклатуры;
Форма = Остатки.ПолучитьФормуСписка();
Форма.ПараметрОтборПоРегистратору = Док;
Форма.Открыть();

Использование в версии:

Доступен, начиная с версии 8.0.
