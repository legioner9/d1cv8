СхемаЗапроса.УстановитьТекстЗапроса (QuerySchema.SetQueryText)

СхемаЗапроса (QuerySchema)

УстановитьТекстЗапроса (SetQueryText)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТекстЗапроса(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Текст запроса в виде строки.

Описание:

Устанавливает текст запроса, переданный строкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если у схемы запроса свойство РежимКомпоновкиДанных имеет значение Истина, то при установке схеме запроса текста запроса, в котором используются предложение ИТОГИ, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
