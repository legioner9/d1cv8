КонтейнерПодписейКриптографии (CryptoSignaturesContainer)

Доступен, начиная с версии 8.3.20.

Свойства:

Подписи (Signatures)

Описание:

Контейнер подписей криптографии. Содержит массив элементов ПодписьКриптографии.
Объект может быть получен с помощью метода ПолучитьКонтейнерПодписейКриптографии.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

КонтейнерПодписей = МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии(РезультатПодписи);
Для каждого Подпись Из КонтейнерПодписей.Подписи Цикл
    Сообщить(Подпись.ТипПодписи);
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.23.
