РегистрРасчетаНаборЗаписей.<Имя регистра расчета>.Метаданные (CalculationRegisterRecordSet.<Calculation register name>.Metadata)

РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: ОбъектМетаданных: РегистрРасчета.

Описание:

Предоставляет доступ к объекту описания метаданных регистра расчета. Другой путь получения того же значения - через свойство глобального контекста Метаданные. Например: Метаданные.РегистрыРасчета.РасчетСреднегоЗаработка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
