Расширение табличного поля журнала документов (Document journal table box extension)

Доступен, начиная с версии 8.0.

Свойства:

НастройкаОтбора (FilterSettings)
НастройкаПорядка (OrderSetting)
ПроверкаОтображенияНовойСтроки (NewRowShowCheck)
РазрешитьНастройкуПериода (EnablePeriodSetting)
СтандартныйПериод (StandardPeriod)

События:

ПередИзменениемДаты (BeforeDateChange)
ПередОтменойПроведения (BeforeUndoPosting)
ПередПроведением (BeforePosting)
ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Описание:

Дополнительные свойства и события элемента управления ТабличноеПоле для журнала документов. Для данного расширения свойство ТекущаяСтрока содержит значение типа ДокументСсылка.<Имя документа>. Свойство ТекущиеДанные содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта ЖурналДокументовСписок.<Имя журнала документов>.

Доступность:

Толстый клиент.

Пример:

// Пример установки интервала просмотра журнала

НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод;
НастройкаПериода.ВариантНачала = ВариантГраницыПериода.Месяц;
НастройкаПериода.ВариантОкончания = ВариантГраницыПериода.БезОграничения;
НастройкаПериода.РедактироватьКакИнтервал = Истина;
НастройкаПериода.РедактироватьКакПериод = Ложь;
// последнее задает то, что при нажатии кнопки 

// "Установить интервал просмотра" будет вызвано окно 

// с одной закладкой Интервал


Использование в версии:

Доступен, начиная с версии 8.0.
