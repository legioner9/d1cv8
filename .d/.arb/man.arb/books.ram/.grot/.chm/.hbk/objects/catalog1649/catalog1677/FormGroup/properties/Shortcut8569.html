ГруппаФормы.СочетаниеКлавиш (FormGroup.Shortcut)

ГруппаФормы (FormGroup)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: СочетаниеКлавиш.
Содержит сочетание клавиш группы. Для разных видов групп предполагается разное поведение:
  • обычная группа - действие сочетания клавиш приводит к активности первый элемент группы;
  • командная панель – действие сочетания клавиш приводит к активности первой кнопки командной панели;
  • группа страницы - действие сочетания клавиш приводит к активизации текущей страницы;
  • группа страница - действие сочетания клавиш приводит к выбору заданной страницы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
