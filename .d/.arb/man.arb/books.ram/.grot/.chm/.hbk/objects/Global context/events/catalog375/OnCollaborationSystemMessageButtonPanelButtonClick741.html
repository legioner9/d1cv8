Глобальный контекст (Global context)

ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (OnCollaborationSystemMessageButtonPanelButtonClick)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия(<Сообщение>, <Кнопка>)

Параметры:

<Сообщение>

Тип: СообщениеСистемыВзаимодействия.
Сообщение, в котором была нажата кнопка.

<Кнопка>

Тип: КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.
Кнопка, которая была нажата.

Описание:

Обработчик, вызываемый при нажатии кнопки панели кнопок сообщения системы взаимодействия, если действие кнопки ОбработатьНаКлиенте.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
