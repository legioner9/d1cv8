КонструкторФорматнойСтроки (FormatStringWizard)

ДоступныеТипы (AvailableTypes)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ОписаниеТипов.
Свойство содержит описание типов, для которых форматные строки будут редактироваться с помощью конструктора форматной строки.
Например, если в описание типов входят описание строки, числа или булево, то соответствующие закладки конструктора будут отображаться. Если в описание типов входит только некоторые из этих трех типов, будут отображаться только соответствующие им закладки. Если описание типов не содержит ни один из этих типов, будут отображаться ВСЕ закладки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
