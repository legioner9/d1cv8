ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.
При установке значения данного параметра в Истина будет отменена запись объекта.
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта. Процедура-обработчик вызывается после записи объекта в базу данных, но до окончания транзакции записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
