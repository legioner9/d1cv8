СообщениеВнешнегоСайта (MessageFromExternalSite)

Доступен, начиная с версии 8.3.16.

Свойства:

Данные (Data)
Источник (Origin)

Описание:

Предназначен для хранения параметров сообщения от внешнего сайта, когда веб-клиент интегрирован в него.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

МенеджерОкнаВнешнегоСайта, метод ПодключитьОбработчикСообщений
МенеджерОкнаВнешнегоСайта, метод ОтключитьОбработчикСообщений

Использование в версии:

Доступен, начиная с версии 8.3.16.
