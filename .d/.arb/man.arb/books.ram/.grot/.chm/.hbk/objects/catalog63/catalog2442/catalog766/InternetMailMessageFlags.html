ФлагиИнтернетПочтовогоСообщения (InternetMailMessageFlags)

Доступен, начиная с версии 8.3.12.

Свойства:

Новое (New)
Отвеченное (Answered)
Отмененное (Canceled)
Пересланное (Forwarded)
Помеченное (Flagged)
Прочитанное (Seen)
Удаленное (Deleted)

Описание:

Набор флагов IMAP, установленных для почтового сообщения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, интеграция.
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.3.12.
