ОбъектМетаданных: ПланСчетов.ДлинаПорядка (MetadataObject: ChartOfAccounts.OrderLength)

ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ДлинаПорядка (OrderLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.
Максимальная длина поля Порядок плана счетов. Если значение этого свойства равно 0, то по умолчанию список счетов упорядочивается по полю Код. В противном случае - по полю Порядок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
