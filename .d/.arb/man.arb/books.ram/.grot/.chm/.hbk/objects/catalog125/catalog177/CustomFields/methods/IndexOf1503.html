ПоляНастройки.Индекс (CustomFields.IndexOf)

ПоляНастройки (CustomFields)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: ПолеНастройки.
Искомое поле настройки.

Возвращаемое значение:

Тип: Число.
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
