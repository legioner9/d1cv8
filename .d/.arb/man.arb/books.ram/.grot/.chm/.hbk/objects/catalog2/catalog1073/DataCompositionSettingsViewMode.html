РежимОтображенияНастроекКомпоновкиДанных (DataCompositionSettingsViewMode)

Доступен, начиная с версии 8.1.

Значения

БыстрыйДоступ (QuickAccess)
Все (All)

Описание:

Содержит режимы отображения настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение табличного поля отбора компоновки данных, свойство РежимОтображения
Расширение таблицы формы для отбора компоновки данных, свойство РежимОтображения
Расширение таблицы формы для пользовательских настроек компоновки данных, свойство РежимОтображения

Использование в версии:

Доступен, начиная с версии 8.1.
