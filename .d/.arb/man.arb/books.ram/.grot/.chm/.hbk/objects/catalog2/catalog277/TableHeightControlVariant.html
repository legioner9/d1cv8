ВариантУправленияВысотойТаблицы (TableHeightControlVariant)

Доступен, начиная с версии 8.3.9.

Значения

Авто (Auto)
ВСтрокахТаблицы (UseHeightInTableRows)
ВСтрокахФормы (UseHeightInFormRows)
ПоСодержимому (UseContentHeight)

Описание:

Содержит варианты управления высотой таблицы формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
