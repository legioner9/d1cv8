Элементы коллекции:

Описание:

Содержит список дополнительных данных в виде пар: "Ключ – Значение", где:

  • Ключ – тип Строка, уникальный идентификатор данных,
  • Значение – объект типа ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
