РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

РасчетСистемЛинейныхУравнений

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый РасчетСистемЛинейныхУравнений()

Описание:

Создает объект и устанавливает следующие значения свойств:

  • ИсточникДанныхУзлов - Неопределено,
  • ИсточникДанныхСвязей – Неопределено,
  • КолонкаУравненияВСвязях – Неопределено,
  • КолонкаПеременныеВСвязях – Неопределено,
  • КолонкаУравненияВУзлах – Неопределено,
  • ОписанияСистем – новый объект типа ОписанияСистемЛинейныхУравнений (пустая коллекция),
  • КоличествоИтераций = 200,
  • ТребуемаяТочность = 0.000001,
  • ИспользованиеВычислительныхРесурсов = 0,
  • ГраницаИзмененияАлгоритмаРешения = 10000.

Использование в версии:

Доступен, начиная с версии 8.3.14.
