ГеографическиеКоординаты (GeographicCoordinates)

По умолчанию

Доступен, начиная с версии 8.3.3.

Синтаксис:

Новый ГеографическиеКоординаты(<Широта>, <Долгота>, <Высота>)

Параметры:

<Широта> (обязательный)

Тип: Число.
Широта. Задается в градусах.

<Долгота> (обязательный)

Тип: Число.
Долгота. Задается в градусах.

<Высота> (необязательный)

Тип: Число.
Высота. Задается в метрах.

Описание:

Создает объект по географическим координатам.

Использование в версии:

Доступен, начиная с версии 8.3.3.
