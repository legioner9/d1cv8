ДополнениеЭлементаФормы (FormItemAddition)

ВертикальноеПоложениеВГруппе (VerticalAlignInGroup)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: ВертикальноеПоложениеЭлемента.
Устанавливает вертикальное расположение элемента внутри группы.
Если имеет значение Авто, то будет испльзовано значение свойства ВертикальноеПоложение родительской группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.7.
