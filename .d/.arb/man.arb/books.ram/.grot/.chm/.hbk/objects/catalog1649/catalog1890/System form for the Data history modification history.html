Системная форма истории изменений истории данных (System form for the Data history modification history)

Доступен, начиная с версии 8.3.13.

Параметры формы:

Данные (Data)

Описание:

Форма история изменений истории данных. Предоставляет список версий данных, открывается при вызове команды "История изменений".

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.21.
