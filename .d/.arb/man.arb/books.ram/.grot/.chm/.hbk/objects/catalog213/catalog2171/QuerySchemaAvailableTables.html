ДоступныеТаблицыСхемыЗапроса (QuerySchemaAvailableTables)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ГруппаДоступныхТаблицСхемыЗапроса, ДоступнаяТаблицаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются группа доступных таблиц или доступная таблица.
Возможно обращение к группе доступных таблиц или к доступной таблице посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Коллекция доступных таблиц схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
