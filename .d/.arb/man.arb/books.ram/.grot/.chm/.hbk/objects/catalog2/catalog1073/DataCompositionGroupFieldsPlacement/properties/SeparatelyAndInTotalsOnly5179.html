РасположениеПолейГруппировкиКомпоновкиДанных (DataCompositionGroupFieldsPlacement)

ОтдельноИТолькоВИтогах (SeparatelyAndInTotalsOnly)

Доступен, начиная с версии 8.1.

Описание:

Каждая группировка располагается в отдельной области. Группировки располагаются друг за другом слева направо. Выводимые поля выводятся только в данной группировке.

Использование в версии:

Доступен, начиная с версии 8.1.
