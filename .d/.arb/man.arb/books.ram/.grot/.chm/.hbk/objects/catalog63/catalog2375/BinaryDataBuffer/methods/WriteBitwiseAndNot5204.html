БуферДвоичныхДанных (BinaryDataBuffer)

ЗаписатьПобитовоеИНе (WriteBitwiseAndNot)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЗаписатьПобитовоеИНе(<Позиция>, <Буфер>, <Количество>)

Параметры:

<Позиция> (обязательный)

Тип: Число.
Начальная позиция в буфере.

<Буфер> (обязательный)

Тип: БуферДвоичныхДанных.
Буфер, с которым выполняется объединение.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые требуется объединить.
Значение по умолчанию: Неопределено.

Описание:

Объединить заданное количество байтов, начиная с заданной позиции с байтами из заданного буфера, используя побитовое И НЕ. Если количество байтов не указано, то объединяются все байты до конца буфера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
