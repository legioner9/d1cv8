ДопустимыйНомерСообщения (AllowedMessageNo)

Больший (Greater)

Доступен, начиная с версии 8.0.

Описание:

Данное значение показывает, что принимаемое сообщение может иметь любой номер больший, чем максимальный номер принятого сообщения. Максимальный номер принятого сообщения можно узнать, обратившись к реквизиту НомерПринятого узла плана обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
