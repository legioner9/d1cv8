СеансИнформационнойБазы (InfoBaseSession)

ПолучитьФоновоеЗадание (GetBackgroundJob)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьФоновоеЗадание()

Возвращаемое значение:

Тип: ФоновоеЗадание.

Описание:

Если сеанс был создан в рамках выполнения фонового задания, возвращает это фоновое задание, в противном случае - Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
