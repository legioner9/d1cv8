ПараметрыМакетаТекстовогоДокумента (TextDocumentTemplateParameters)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: Произвольный.
Объект, из свойств которого будут заполняться параметры макета текстового документа.

Описание:

Заполняет параметры значениями свойств переданного объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
