УдалитьЭлементы (DeleteItems)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

  • УдалитьЭлементы

Доступен, начиная с версии 8.3.16.

Описание:

Удалить выбранные элементы. Используется только для источника типа Элементы.
В контекстном меню команда будет иметь представление "Удалить".

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
