НастройкаНастройкиОформления (AppearanceSettingSettings)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияНастройкойНастройкиОформления
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Набор элементов управления настройкой оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля настроек оформления, свойство НастройкаНастройкиОформления

Использование в версии:

Доступен, начиная с версии 8.0.
