ПоведениеЭлементовПланировщикаПриНедостаткеМеста (PlannerItemsBehaviorOnLackOfSpace)

СворачиватьЭлементы (CollapseItems)

Доступен, начиная с версии 8.3.15.

Описание:

При недостатке места в планировщике элементы, которые не вмещаются по высоте, сворачиваются в подменю "Ещё". При этом высоты всех строк планировщика одинаковы.

Использование в версии:

Доступен, начиная с версии 8.3.15.
