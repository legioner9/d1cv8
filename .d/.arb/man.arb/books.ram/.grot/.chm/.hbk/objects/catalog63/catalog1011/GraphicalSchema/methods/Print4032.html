ГрафическаяСхема.Напечатать (GraphicalSchema.Print)

ГрафическаяСхема (GraphicalSchema)

Напечатать (Print)

Доступен, начиная с версии 8.2.

Синтаксис:

Напечатать(<ДиалогПечати>)

Параметры:

<ДиалогПечати> (необязательный)

Тип: РежимИспользованияДиалогаПечати.
Указывает необходимость показа диалога печати перед отправкой документа на принтер.
Значение по умолчанию: НеИспользовать.

Описание:

Выводит графическую схему на печать.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Только для графической схемы, вставленной в поле графической схемы.

Использование в версии:

Доступен, начиная с версии 8.2.
