МенеджерСистемыВзаимодействия.НачатьСозданиеВыгрузкиДанных (CollaborationSystemManager.BeginCreateDataDump)

МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьСозданиеВыгрузкиДанных (BeginCreateDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

НачатьСозданиеВыгрузкиДанных(<Описание>)

Параметры:

<Описание> (обязательный)

Тип: Строка.
Описание выгрузки данных.

Возвращаемое значение:

Тип: ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.

Описание:

Начинает создание выгрузки данных системы взаимодействия.
Вызов метода возвращает управление при начале процесса создания выгрузки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерСистемыВзаимодействия, метод НачатьСозданиеВыгрузкиДанныхАсинх

Использование в версии:

Доступен, начиная с версии 8.3.22.
