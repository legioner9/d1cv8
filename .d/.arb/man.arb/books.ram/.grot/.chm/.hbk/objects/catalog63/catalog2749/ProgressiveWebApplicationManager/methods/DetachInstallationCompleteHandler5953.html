МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

ОтключитьОбработчикОкончанияУстановки (DetachInstallationCompleteHandler)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОтключитьОбработчикОкончанияУстановки(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, заданной при вызове метода ПодключитьОбработчикОкончанияУстановки.

Описание:

Отключает ранее подключенную процедуру-обработчик.
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

  • ИмяПроцедуры,
  • ИмяПроцедурыОбработкиОшибки,
  • Модуль,
  • МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
