WSСсылкаМенеджер.<Имя WS-Ссылки> (WSReferenceManager.<WS-reference name>)

Доступен, начиная с версии 8.1.

Методы:

ПолучитьWSОпределения (GetWSDefinitions)
СоздатьWSПрокси (CreateWSProxy)

Описание:

Предназначен для управления менеджером ссылки на внешний Web-сервис. Через данный менеджер можно создавать объекты-прокси для обращения к операциям, предоставляемым внешним Web-сервисом, а также получать WSОпределения, содержащие набор определений, описывающих Web-сервис.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

WSСсылкиМенеджер, свойство <Имя WS-ссылки>

Использование в версии:

Доступен, начиная с версии 8.1.
