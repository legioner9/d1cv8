ОтправкаЭлементаДанных.Удалить (DataItemSend.Delete)

ОтправкаЭлементаДанных (DataItemSend)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Описание:

При указании этого значения вместо самих данных помещается элемент, который при приеме сообщения получателем вызовет удаление этих данных. Для объектов это УдалениеОбъекта, а для наборов записей набор записей с тем же отбором, но не содержащий ни одной записи.

Использование в версии:

Доступен, начиная с версии 8.0.
