Описание:

Элемент графической схемы "Декоративная линия". Предназначен для оформления графической схемы. Может соединять между собой как элементы графической схемы, отображающие точки маршрута бизнес-процесса, так и элементы графической схемы декорации. Также декоративная линия может просто размещаться на схеме, без присоединения к чему-либо.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
