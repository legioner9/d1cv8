КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ПолеСводнойТаблицы, КоллекцияПолейСводнойТаблицы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной таблицы или коллекция полей сводной таблицы.
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс элемента (нумерация с 0).

Свойства:

<Имя поля> (<Field name>)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит коллекцию полей сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
