Описание:

Содержит объектную модель выражения схемы запроса. Поддерживает преобразование к строке и сравнение с другим выражением (Сравнение выполняется по строке).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
