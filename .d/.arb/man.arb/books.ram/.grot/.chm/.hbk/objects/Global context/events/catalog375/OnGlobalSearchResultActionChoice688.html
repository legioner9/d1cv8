Глобальный контекст (Global context)

ПриВыбореДействияРезультатаГлобальногоПоиска (OnGlobalSearchResultActionChoice)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПриВыбореДействияРезультатаГлобальногоПоиска(<ЭлементРезультата>, <Действие>)

Параметры:

<ЭлементРезультата>

Тип: ЭлементРезультатаГлобальногоПоиска.
Элемент результата глобального описка, в котором нажата гиперссылка действия.

<Действие>

Тип: Произвольный.
Значение, заданное для нажатой гиперссылки.

Описание:

Вызывается при нажатии на гиперссылку в элементе результата глобального поиска, заданную в свойстве Действия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.18.
