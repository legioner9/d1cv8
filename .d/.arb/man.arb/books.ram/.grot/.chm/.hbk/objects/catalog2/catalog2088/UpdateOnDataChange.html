ОбновлениеПриИзмененииДанных (UpdateOnDataChange)

Доступен, начиная с версии 8.2.

Значения

Авто (Auto)
НеОбновлять (DontUpdate)

Описание:

Содержит варианты режимов обновления элемента формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Расширение таблицы формы для динамического списка, свойство ОбновлениеПриИзмененииДанных

Использование в версии:

Доступен, начиная с версии 8.2.
