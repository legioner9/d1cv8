НастройкаПараметровАнализаДанных.Получить (DataAnalysisParametersSetting.Get)

НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: Число.
Номер элемента управления параметрами.

Возвращаемое значение:

Тип: ЭлементУправленияПараметрамиАнализаДанных.

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
