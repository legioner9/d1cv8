КолонкиАнализаДанных (DataAnalysisColumns)

<Имя колонки> (<Column name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: КолонкаАнализаДанных.
Набор свойств содержит значения полей колонок анализа данных. Доступ к значению осуществляется по имени колонки. Имена свойств совпадают с именами колонок анализа данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
