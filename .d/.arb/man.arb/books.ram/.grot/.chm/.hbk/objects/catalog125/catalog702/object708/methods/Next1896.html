БизнесПроцессВыборка.<Имя бизнес-процесса>.Следующий (BusinessProcessSelection.<Business process name>.Next)

БизнесПроцессВыборка.<Имя бизнес-процесса> (BusinessProcessSelection.<Business process name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: Булево.
Истина - следующий бизнес-процесс выбрана; Ложь - достигнут конец выборки.

Описание:

Получает очередной бизнес-процесс из выборки. Для обхода списка бизнес-процессов нужно после получения выборки использовать данный метод для позиционирования на первый и последующие бизнес-процессы выборки до тех пор, пока он не вернет значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Выборка = БизнесПроцессы.ОбработкаПоступлений.Выбрать();
Пока Выборка.Следующий() Цикл
    // обработка выборки

КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
