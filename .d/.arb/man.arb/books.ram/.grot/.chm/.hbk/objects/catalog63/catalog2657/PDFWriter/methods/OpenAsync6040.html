ЗаписьPDF.ОткрытьАсинх (PDFWriter.OpenAsync)

ЗаписьPDF (PDFWriter)

ОткрытьАсинх (OpenAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Открыть файл

Синтаксис:

ОткрытьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: Строка.
Имя файла для записи PDF-документа.

Вариант синтаксиса: Установить поток

Синтаксис:

ОткрытьАсинх(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для записи PDF-файла.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Открывает PDF-документ для записи.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ЗаписьPDF, метод Открыть
ЗаписьPDF, метод НачатьОткрытие

Использование в версии:

Доступен, начиная с версии 8.3.18.
