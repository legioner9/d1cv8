HTTPМетод.DELETE (HTTPMethod.DELETE)

HTTPМетод (HTTPMethod)

DELETE (DELETE)

Доступен, начиная с версии 8.3.5.

Описание:

Соответствует HTTP-методу DELETE.
Предназначен для удаления данных.
Описан в стандарте RFC 2616 (http://tools.ietf.org/html/rfc2616).

Использование в версии:

Доступен, начиная с версии 8.3.5.
