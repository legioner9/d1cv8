ХарактеристикиКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionCharacteristics)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

ХарактеристикаКомпоновкиДанныхСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания характеристик.
Возможно обращение к описанию характеристик посредством оператора [...]. В качестве аргумента передается индекс элемента коллекции.

Методы:

Добавить (Add)
Изменить (Change)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Индексированная коллекция элементов типа ХарактеристикаКомпоновкиДанныхСхемыЗапроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
