Администратор.PasswordAuthAllowed (IRegUserInfo.PasswordAuthAllowed)

Администратор (IRegUserInfo)

PasswordAuthAllowed (PasswordAuthAllowed)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак разрешения администратору аутентификации паролем.
Истина - аутентификация паролем. В этом случае при правильном вводе пароля администратор будет считаться аутентифицированным. Ложь - запрещена.

Доступность:

Интеграция.

Примечание:

Среди всех администраторов должен быть хотя бы один, которому разрешена аутентификация паролем.

Использование в версии:

Доступен, начиная с версии 8.1.
