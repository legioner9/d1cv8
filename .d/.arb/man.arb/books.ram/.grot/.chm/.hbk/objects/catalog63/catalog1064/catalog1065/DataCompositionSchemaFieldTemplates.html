МакетыПолейСхемыКомпоновкиДанных (DataCompositionSchemaFieldTemplates)

Доступен, начиная с версии 8.1.

Элементы коллекции:

МакетПоляСхемыКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются макеты полей.
Возможно обращение к макету поля посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит макеты полей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

СхемаКомпоновкиДанных, свойство МакетыПолей
