КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных (DataCompositionSettingStructureItemCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ГруппировкаКомпоновкиДанных, ТаблицаКомпоновкиДанных, ДиаграммаКомпоновкиДанных, НастройкиВложенногоОбъектаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются группировки.
Возможно обращение к группировке посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

ИдентификаторПользовательскойНастройки (UserSettingID)
ПредставлениеПользовательскойНастройки (UserSettingPresentation)
РежимОтображения (ViewMode)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит элементы структуры настроек.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Использование в версии:
