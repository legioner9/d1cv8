Менеджер COM-соединений (COM connector)

Доступен, начиная с версии 8.2.

Свойства:

HighBoundDefault (HighBoundDefault)
LowBoundDefault (LowBoundDefault)
MaxConnections (MaxConnections)
PoolCapacity (PoolCapacity)
PoolTimeout (PoolTimeout)
RAgentPortDefault (RAgentPortDefault)
RMngrPortDefault (RMngrPortDefault)

Методы:

Connect (Connect)
ConnectAgent (ConnectAgent)
ConnectWorkingProcess (ConnectWorkingProcess)

Описание:

С помощью данного объекта выполняется установка внешнего соединения с информационной базой 1С:Предприятия 8 и администрирование кластера серверов. С помощью одного экземпляра объекта может быть установлено неограниченное число соединений.
Установка соединения с информационной базой 1С:Предприятия 8 осуществляется с помощью метода Connect.
Администрирование кластера серверов выполняется с помощью методов ConnectAgent и ConnectWorkingProcess.

Доступность:

Интеграция.

Пример:

// Пример создания объекта

Соединитель = Новый COMObject("V83.COMConnector");

Использование в версии:

Доступен, начиная с версии 8.2.
