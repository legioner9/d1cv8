МенеджерРаботыСРечью (SpeechProcessingManager)

ВыполнитьРегистрациюСеанса (RegisterSession)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВыполнитьРегистрациюСеанса(<Пользователь>, <Пароль>, <АдресСервера>)

Параметры:

<Пользователь> (обязательный)

Тип: Строка.
Логин пользователя в сервисе работы с речью.

<Пароль> (обязательный)

Тип: Строка.
Пароль пользователя в сервисе работы с речью.

<АдресСервера> (необязательный)

Тип: Строка.
Адрес сервиса работы с речью.
Значение по умолчанию зависит от региона использования.

Описание:

Формирует данные подключения к внешнему сервису работы с речью и сохраняет их для текущего сеанса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
