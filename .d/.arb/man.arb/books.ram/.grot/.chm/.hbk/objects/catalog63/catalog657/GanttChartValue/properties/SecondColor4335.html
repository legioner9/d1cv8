ЗначениеДиаграммыГанта (GanttChartValue)

ДополнительныйЦвет (SecondColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Цвет.
Содержит цвет перекрывающихся интервалов. Допустимо значение АвтоЦвет. В этом случае цвет определяется в зависимости от значения свойства ПриоритетЦвета серии или точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
