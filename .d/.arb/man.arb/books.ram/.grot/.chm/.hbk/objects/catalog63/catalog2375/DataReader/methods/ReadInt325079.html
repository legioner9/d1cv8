ЧтениеДанных (DataReader)

ПрочитатьЦелое32 (ReadInt32)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьЦелое32(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, используемый при чтении числа.
Если не задан, используется порядок, определенный для текущего экземпляра ЧтениеДанных.
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Число.
Неопределено - если не удалось прочитать число, поскольку был достигнут конец потока.

Описание:

Прочитать 32-битное целое число из потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ЧтениеДанных, метод НачатьЧтениеЦелого32
ЧтениеДанных, метод ПрочитатьЦелое32Асинх

Использование в версии:

Доступен, начиная с версии 8.3.9.
