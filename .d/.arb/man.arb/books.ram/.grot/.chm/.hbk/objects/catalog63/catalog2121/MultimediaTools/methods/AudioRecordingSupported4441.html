СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяАудиозапись (AudioRecordingSupported)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоддерживаетсяАудиозапись()

Возвращаемое значение:

Тип: Булево.
Возвращает Истина в следующих случаях:
  • на Android - всегда,
  • на iOS - всегда,
  • на Windows Phone - при наличии микрофона.

Описание:

Определяет, поддерживается ли аудиозапись системой.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваМультимедиа, метод СделатьАудиозапись

Доступен, начиная с версии 8.3.3.
