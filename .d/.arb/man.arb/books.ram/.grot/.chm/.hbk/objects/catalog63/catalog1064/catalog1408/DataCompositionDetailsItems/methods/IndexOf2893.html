ЭлементыРасшифровкиКомпоновкиДанных.Индекс (DataCompositionDetailsItems.IndexOf)

ЭлементыРасшифровкиКомпоновкиДанных (DataCompositionDetailsItems)

Индекс (IndexOf)

Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<ЭлементРасшифровки>)

Параметры:

<ЭлементРасшифровки> (необязательный)

Тип: ЭлементРасшифровкиКомпоновкиДанныхПоля, ЭлементРасшифровкиКомпоновкиДанныхГруппировка.
Элемент, для которого нужно получить индекс.

Возвращаемое значение:

Тип: Число.
Если элемент не найден возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
