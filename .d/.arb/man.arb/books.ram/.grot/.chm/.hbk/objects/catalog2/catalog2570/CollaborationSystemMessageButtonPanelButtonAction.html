ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonAction)

Доступен, начиная с версии 8.3.25.

Значения

ЗапроситьМестоположение (RequestLocation)
ЗапроситьТелефон (RequestPhone)
ОбработатьБотом (ProcessByBot)
ОбработатьНаКлиенте (ProcessOnClient)
ОтправитьСообщение (SendMessage)
ОтправитьСообщениеСДанными (SendMessageWithData)
ПерейтиПоНавигационнойСсылке (GotoURL)

Описание:

Содержит варианты действия кнопки панели кнопок сообщения системы взаимодействия.
Если действие не поддерживается в системе, тогда кнопка, выполняющая данное действие, не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
