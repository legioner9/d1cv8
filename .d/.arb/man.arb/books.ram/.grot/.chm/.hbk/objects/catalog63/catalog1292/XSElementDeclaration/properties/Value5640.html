ОбъявлениеЭлементаXS.Значение (XSElementDeclaration.Value)

ОбъявлениеЭлементаXS (XSElementDeclaration)

Значение (Value)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Содержит значение атрибутов fixed или default.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Объявления атрибута (http://www.w3.org/TR/xmlschema-1/#a-value_constraint) или объявления элемента (http://www.w3.org/TR/xmlschema-1/#e-value_constraint).

Использование в версии:

Доступен, начиная с версии 8.1.
