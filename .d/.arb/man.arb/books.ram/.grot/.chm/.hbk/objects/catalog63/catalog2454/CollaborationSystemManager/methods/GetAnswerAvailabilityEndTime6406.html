МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьВремяОкончанияДоступностиОтвета (GetAnswerAvailabilityEndTime)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьВремяОкончанияДоступностиОтвета(<Обсуждение>)

Параметры:

<Обсуждение> (обязательный)

Тип: ИдентификаторОбсужденияСистемыВзаимодействия.
Идентификатор обсуждения.

Возвращаемое значение:

Тип: Дата, Неопределено.
Дата возвращается в часовом поясе текущего сеанса.
Для интеграции других типов возвращает Неопределено.

Описание:

Для интеграции типа WhatsApp получает время окончания 24-х часового окна для ответа.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
