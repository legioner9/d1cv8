МенеджерОбработкиОшибок.УстановитьОбщиеНастройки (ErrorProcessingManager.SetCommonSettings)

МенеджерОбработкиОшибок (ErrorProcessingManager)

УстановитьОбщиеНастройки (SetCommonSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьОбщиеНастройки(<Настройки>)

Параметры:

<Настройки> (обязательный)

Тип: НастройкиОбработкиОшибок.
Устанавливаемые настройки.

Описание:

Устанавливает общие настройки обработки ошибки. Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.17.
