ПланСчетовВыборка.<Имя плана счетов>.Следующий (ChartOfAccountsSelection.<Chart of accounts name>.Next)

ПланСчетовВыборка.<Имя плана счетов> (ChartOfAccountsSelection.<Chart of accounts name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: Булево.
Истина - следующий элемент выбран; Ложь - достигнут конец выборки.

Описание:

Получает следующий элемент из выборки. Для обхода плана счетов нужно после получения выборки вызвать данный метод для позиционирования на первый элемент и далее вызывать до тех пор, пока не будет возвращено значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Выборка = ПланыСчетов.Основной.Выбрать();
Пока Выборка.Следующий() Цикл
    // обработка выборки

КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
