Диаграмма.ПроцентПолупрозрачности (Chart.SemitransparencyPercent)

Диаграмма (Chart)

ПроцентПолупрозрачности (SemitransparencyPercent)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит степень прозрачности, указанную в процентах. Целочисленное значение от 0 до 100. 100% - полностью прозрачный.
Если указано 0, для каждого вида диаграммы используется собственное оптимальное значение по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
