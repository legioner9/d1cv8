ТестируемаяФорма (TestedForm)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбъект(<Тип>, <ТекстЗаголовка>, <Имя>, <Таймаут>)

Параметры:

<Тип> (необязательный)

Тип: Тип.
Тип объекта, который нужно найти.
Значение по умолчанию: Неопределено.

<ТекстЗаголовка> (необязательный)

Тип: Строка.
Заголовок объекта, если это применимо. Может содержать знаки подстановки "*" и "?".

<Имя> (необязательный)

Тип: Строка.
Имя объекта, если это применимо. Может содержать знаки подстановки "*" и "?".

<Таймаут> (необязательный)

Тип: Число.
Время в секундах, в течение которого выполняется поиск объекта. Если значение параметра не задано, время поиска не ограничено.
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: ТестируемаяФорма, ТестируемаяГруппаФормы, ТестируемаяКнопкаФормы, ТестируемаяТаблицаФормы, ТестируемоеПолеФормы, ТестируемаяДекорацияФормы, Неопределено.
Первый найденный объект. Если объект не найден, генерируется исключительная ситуация.

Описание:

Выполняет поиск объекта среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
