РасширениеКонфигурации.ПолучитьДанные (ConfigurationExtension.GetData)

РасширениеКонфигурации (ConfigurationExtension)

ПолучитьДанные (GetData)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьДанные()

Возвращаемое значение:

Тип: ДвоичныеДанные.
Для нового расширения возвращает Неопределено либо ДвоичныеДанные, если они были переданы при создании объекта.

Описание:

Возвращает текущие используемые данные расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерРасширенийКонфигурации, метод Создать

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
