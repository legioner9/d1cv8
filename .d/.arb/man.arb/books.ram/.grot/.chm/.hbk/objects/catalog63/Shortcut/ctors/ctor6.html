СочетаниеКлавиш (Shortcut)

На основании сочетания клавиш

Доступен, начиная с версии 8.0.

Синтаксис:

Новый СочетаниеКлавиш(<Клавиша>, <Alt>, <Control>, <Shift>)

Параметры:

<Клавиша> (обязательный)

Тип: Клавиша.
Основная клавиша сочетания.

<Alt> (необязательный)

Тип: Булево.
Истина - комбинация клавиш включает Alt, Ложь - иначе.

<Control> (необязательный)

Тип: Булево.
Истина - комбинация клавиш включает Control, Ложь - иначе.

<Shift> (необязательный)

Тип: Булево.
Истина - комбинация клавиш включает Shift, Ложь - иначе.

Описание:

Формирует сочетание из заданной клавиши и клавиш Alt, Control и Shift.

Использование в версии:

Доступен, начиная с версии 8.0.
