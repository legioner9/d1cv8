Доступен, начиная с версии 8.1.

Элементы коллекции:

ЯчейкаМакетаКоллекцииЗначенийОбластиКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ячейки.
Возможно обращение к ячейке посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Коллекция ячеек для вывода в коллекцию значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
