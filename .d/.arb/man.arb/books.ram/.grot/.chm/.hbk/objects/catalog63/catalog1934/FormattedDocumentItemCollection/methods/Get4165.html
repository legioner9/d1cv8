КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Получить (Get)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс элемента.

Возвращаемое значение:

Тип: ПараграфФорматированногоДокумента, ТекстФорматированногоДокумента, ПереводСтрокиФорматированногоДокумента, КартинкаФорматированногоДокумента.

Описание:

Получает элемент форматированного документа по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
