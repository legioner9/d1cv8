КоллекцияЭлементовУсловногоОформленияКомпоновкиДанных (DataCompositionConditionalAppearanceItemCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ЭлементУсловногоОформленияКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы условного оформления.
Возможно обращение к элементам условного оформления посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция элементов условного оформления.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
