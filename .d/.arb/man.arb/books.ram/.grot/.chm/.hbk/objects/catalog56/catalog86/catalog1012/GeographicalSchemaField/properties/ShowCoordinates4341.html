ПолеГеографическойСхемы (GeographicalSchemaField)

ОтображатьКоординаты (ShowCoordinates)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак показа дополнительной информационной панели, в которой отображаются координаты точки на схеме под курсором мыши.

Доступность:

Толстый клиент.

Примечание:

Если Истина, то координаты точки отображаются на схеме. Значение по умолчанию Ложь.

Использование в версии:

Доступен, начиная с версии 8.0.
