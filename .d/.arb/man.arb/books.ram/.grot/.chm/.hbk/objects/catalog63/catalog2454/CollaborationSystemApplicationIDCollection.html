КоллекцияИдентификаторовПриложенийСистемыВзаимодействия (CollaborationSystemApplicationIDCollection)

Доступен, начиная с версии 8.3.13.

Элементы коллекции:

ИдентификаторПриложенияСистемыВзаимодействия
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Содержит (Contains)
Удалить (Delete)

Описание:

Предоставляет методы для работы с приложениями в системе взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
