РежимПроведенияДокумента (DocumentPostingMode)

РежимПроведенияДокумента (DocumentPostingMode)

Доступен, начиная с версии 8.0.

Значения

Неоперативный (AnyTime)
Оперативный (RealTime)

Описание:

Определяет набор режимов проведения документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Возможен обмен с сервером.

См. также:

ДокументОбъект.<Имя документа>, метод Записать
ДокументОбъект.<Имя документа>, событие ОбработкаПроведения
ДокументОбъект.<Имя документа>, событие ПередЗаписью
Расширение формы документа, событие ПередЗаписью
Расширение формы клиентского приложения для документа, событие ПередЗаписью
Расширение формы клиентского приложения для документа, событие ПередЗаписьюНаСервере

Использование в версии:

Доступен, начиная с версии 8.0.
