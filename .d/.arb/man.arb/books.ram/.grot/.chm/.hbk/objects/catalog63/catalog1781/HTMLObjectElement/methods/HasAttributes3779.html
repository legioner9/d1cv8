ЭлементОбъектHTML.ЕстьАтрибуты (HTMLObjectElement.HasAttributes)

ЭлементОбъектHTML (HTMLObjectElement)

ЕстьАтрибуты (HasAttributes)

Доступен, начиная с версии 8.2.

Синтаксис:

ЕстьАтрибуты()

Возвращаемое значение:

Тип: Булево.

Описание:

Определяет наличие атрибутов у данного узла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
