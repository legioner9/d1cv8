ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

По умолчанию

Доступен, начиная с версии 8.3.5.

Синтаксис:

Новый ЗапускПриложенияМобильногоУстройства(<Действие>, <Данные>)

Параметры:

<Действие> (необязательный)

Тип: Строка.
Описывает выполняемое действие.

<Данные> (необязательный)

Тип: Строка.
Содержит URI-ссылку на данные.

Описание:

Запускает приложение на мобильном устройстве.

Использование в версии:

Доступен, начиная с версии 8.3.5.
