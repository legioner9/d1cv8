МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтключитьОбработчикПослеОтправкиСообщения (DetachAfterMessageSendHandler)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ОтключитьОбработчикПослеОтправкиСообщения(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, заданной при вызове метода ПодключитьОбработчикПослеОтправкиСообщения.

Описание:

Отключает ранее подключенную процедуру-обработчик.
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

  • ИмяПроцедуры,
  • ИмяПроцедурыОбработкиОшибки,
  • Модуль,
  • МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.11.
