КоллекцияОформленийДат.Количество (DateAppearanceCollection.Count)

КоллекцияОформленийДат (DateAppearanceCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество дат в коллекции оформлений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.0.
