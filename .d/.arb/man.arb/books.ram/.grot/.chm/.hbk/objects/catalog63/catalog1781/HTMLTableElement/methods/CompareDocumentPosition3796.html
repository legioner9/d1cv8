ЭлементТаблицаHTML.СравнитьПозициюВДокументе (HTMLTableElement.CompareDocumentPosition)

ЭлементТаблицаHTML (HTMLTableElement)

СравнитьПозициюВДокументе (CompareDocumentPosition)

Доступен, начиная с версии 8.2.

Синтаксис:

СравнитьПозициюВДокументе(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: АтрибутHTML, ЭлементHTML, ЭлементКнопкаHTML, ЭлементВводаHTML, ЭлементЗаголовокHTML, ЭлементРазметкаHTML, ЭлементПлавающийФреймHTML, ЭлементВставкаHTML, ЭлементФреймHTML, ЭлементНаборФреймовHTML, ЭлементМетаHTML, ЭлементОбъектHTML, ЭлементСкриптHTML, ЭлементТаблицаHTML, ЭлементСтрокаТаблицыHTML, ЭлементЯчейкаТаблицыHTML, ЭлементКолонкаТаблицыHTML, ЭлементЗаголовокТаблицыHTML, ЭлементБлокHTML, ЭлементЛинияHTML, ЭлементФорматированногоТекстаHTML, ЭлементКартинкаHTML, ЭлементСвязьHTML, ЭлементЯкорьHTML, ЭлементАплетHTML, ЭлементФормаHTML, ДокументHTML, ТекстHTML, КомментарийHTML.
Узел HTMLDOM, с которым происходит сравнение.

Описание:

Сравнивает данный узел с узлом, переданным в качестве параметра, для определения позиции относительно документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
