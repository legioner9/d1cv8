МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьПользователя (GetUser)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьПользователя(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: ИдентификаторПользователяСистемыВзаимодействия.
Идентификатор получаемого пользователя.

Возвращаемое значение:

Тип: ПользовательСистемыВзаимодействия.

Описание:

Возвращает пользователя системы взаимодействия по его идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если в качестве параметра используется значение ВсеПользователиПриложения, будет сгенерирована ошибка времени выполнения.
