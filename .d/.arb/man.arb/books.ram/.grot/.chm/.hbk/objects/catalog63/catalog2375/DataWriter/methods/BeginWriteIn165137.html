ЗаписьДанных.НачатьЗаписьЦелого16 (DataWriter.BeginWriteIn16)

ЗаписьДанных (DataWriter)

НачатьЗаписьЦелого16 (BeginWriteIn16)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗаписьЦелого16(<ОписаниеОповещения>, <Число>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Число> (обязательный)

Тип: Число.
Число, которое будет записано в поток.
Значение числа должно находиться в диапазоне от 0 до 65535.

<ПорядокБайтов> (необязательный)

Тип: ПорядокБайтов.
Порядок байтов, который будет использован для кодировки числа перед записью в поток. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта ЗаписьДанных.
Значение по умолчанию: Неопределено.

Описание:

Вызывает начало записи 16-разрядного целого числа в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

ЗаписьДанных, метод ЗаписатьЦелое16
