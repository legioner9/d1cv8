ДокументPDF.ПолучитьОписанияПодписей (PDFDocument.GetSignatureDescriptions)

ДокументPDF (PDFDocument)

ПолучитьОписанияПодписей (GetSignatureDescriptions)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПолучитьОписанияПодписей(<МенеджерКриптографии>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: МенеджерКриптографии.
Менеджер криптографии.

Возвращаемое значение:

Тип: Массив.
Массив объектов типа ОписаниеПодписиPDF.

Описание:

Получает массив, содержащий описания присоединенных к PDF-документу электронных подписей.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
