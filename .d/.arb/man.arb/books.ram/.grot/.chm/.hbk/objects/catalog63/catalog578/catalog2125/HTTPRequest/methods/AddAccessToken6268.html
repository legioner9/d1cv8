HTTPЗапрос.ДобавитьТокенДоступа (HTTPRequest.AddAccessToken)

HTTPЗапрос (HTTPRequest)

ДобавитьТокенДоступа (AddAccessToken)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ДобавитьТокенДоступа(<ТокенДоступа>)

Параметры:

<ТокенДоступа> (необязательный)

Тип: ТокенДоступа.
Токен доступа.

Описание:

Добавляет токен доступа к HTTP-запросу. Устанавливает заголовок Authorization.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.21.
