Глобальный контекст (Global context)

УстановитьМинимальныйСрокДействияПаролейПользователей (SetUserPasswordMinEffectivePeriod)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьМинимальныйСрокДействияПаролейПользователей(<МинимальныйСрокДействияПаролей>)

Параметры:

<МинимальныйСрокДействияПаролей> (необязательный)

Тип: Число.
Минимальный срок действия пароля.
Значение по умолчанию: 0.

Описание:

Устанавливает минимальный срок действия паролей пользователей.
Минимальный срок действия пароля не может превышать максимальный срок действия пароля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
