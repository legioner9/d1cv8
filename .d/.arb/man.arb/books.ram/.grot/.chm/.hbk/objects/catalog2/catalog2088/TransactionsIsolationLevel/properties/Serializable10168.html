УровеньИзоляцииТранзакций (TransactionsIsolationLevel)

Упорядочиваемость (Serializable)

Доступен, начиная с версии 8.3.5.

Описание:

Уровень изоляции транзакций внешнего источника данных, при котором операция чтения возвращает данные и выборку строк, зафиксированные в других транзакциях на момент начала транзакции. Операция записи на измененных в других транзакциях данных завершается с ошибкой.

Использование в версии:

Доступен, начиная с версии 8.3.5.
