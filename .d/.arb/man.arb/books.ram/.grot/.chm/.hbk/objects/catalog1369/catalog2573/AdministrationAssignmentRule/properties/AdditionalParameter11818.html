АдминистрированиеТребованиеНазначения (AdministrationAssignmentRule)

ДополнительныйПараметр (AdditionalParameter)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит имя приложения с уточнением. Дополнительные признаки объектов требований Connection и SessionDataService, которые могут использоваться для выбора подходящего требования:
  • Для фонового и регламентного задания - BackgroundJob.CommonModule.<Имя модуля>.<Имя метода>;
  • Для отчета - BackgroundJob.Report.<Имя отчета>;
  • Для фоновой реструктуризации – SystemBackgroundJob;
  • Для клиентского приложения:
    • 1CV8 – толстый клиент;
    • 1CV8CDirect – тонкий клиент – непосредственное соединение;
    • Designer – конфигуратор;
    • COMConnection – COM-соединение;
    • WebServerExtension – веб-клиент, тонкий клиент через веб-сервер, веб-сервисы.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
