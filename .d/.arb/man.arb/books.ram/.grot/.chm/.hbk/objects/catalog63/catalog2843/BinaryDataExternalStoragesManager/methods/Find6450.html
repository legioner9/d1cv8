МенеджерВнешнихХранилищДвоичныхДанных.Найти (BinaryDataExternalStoragesManager.Find)

МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Найти (Find)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя внешнего хранилища двоичных данных.

Возвращаемое значение:

Тип: Неопределено, МенеджерВнешнегоХранилищаДвоичныхДанных.
Если менеджер существует и не удален, вернет МенеджерВнешнегоХранилищаДвоичныхДанных, иначе - Неопределено.

Описание:

Возвращает внешнее хранилище по имени.

Доступность:

Сервер.

Пример:

cp = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cp.URL = "https://bucket.example.com";
cp.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

ap = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных();
ap.ИдентификаторДоступа = "EXAMPLE";
ap.СекретныйКлюч = "SecretKey";
ap.Регион = "US";

ВнешниеХранилищаДвоичныхДанных.Добавить("manager", cp, ap);

Мнджр = ВнешниеХранилищаДвоичныхДанных.Найти("manager");

Использование в версии:

Доступен, начиная с версии 8.3.24.
