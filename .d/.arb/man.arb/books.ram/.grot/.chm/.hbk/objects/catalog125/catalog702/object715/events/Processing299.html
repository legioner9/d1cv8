ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

Обработка (Processing)

Доступен, начиная с версии 8.0.

Синтаксис:

Обработка(<ТочкаМаршрута>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса, в которой выполняется обработка.

Описание:

Вызывается для выполнения автоматических действий при приходе бизнес-процесса на точку обработки. После завершения обработчика бизнес-процесс переходит к следующей точке маршрута в соответствии с картой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида Обработка.
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
