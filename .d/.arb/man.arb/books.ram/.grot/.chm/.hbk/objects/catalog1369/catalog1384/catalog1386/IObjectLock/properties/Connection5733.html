Блокировка.Connection (IObjectLock.Connection)

Блокировка (IObjectLock)

Connection (Connection)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Описание соединения, Неопределено.
Содержит объект с информацией о соединении с информационной базой, установившем данную блокировку, или Неопределено, если источником блокировки не является соединение с информационной базой.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
