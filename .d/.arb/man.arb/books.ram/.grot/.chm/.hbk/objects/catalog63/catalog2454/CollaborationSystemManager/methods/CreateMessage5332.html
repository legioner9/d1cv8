МенеджерСистемыВзаимодействия (CollaborationSystemManager)

СоздатьСообщение (CreateMessage)

Доступен, начиная с версии 8.3.10.

Синтаксис:

СоздатьСообщение(<Обсуждение>)

Параметры:

<Обсуждение> (обязательный)

Тип: ИдентификаторОбсужденияСистемыВзаимодействия.
Идентификатор обсуждения.

Возвращаемое значение:

Тип: СообщениеСистемыВзаимодействия.

Описание:

Создает новое сообщение в обсуждении. Для отправки сообщения необходимо вызвать метод Записать.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
