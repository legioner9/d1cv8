ЖурналыДокументовМенеджер (DocumentJournalsManager)

<Имя журнала документов> (<Document journal name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ЖурналДокументовМенеджер.<Имя журнала документов>.
Набор свойств содержит менеджеры отдельных журналов документов. Имена свойств совпадают с именами журналов документов, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
