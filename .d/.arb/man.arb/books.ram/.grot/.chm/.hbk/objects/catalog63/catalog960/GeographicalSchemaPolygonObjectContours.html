КонтурыПолигональногоОбъектаГеографическойСхемы (GeographicalSchemaPolygonObjectContours)

Доступен, начиная с версии 8.1.

Элементы коллекции:

КонтурПолигональногоОбъектаГеографическойСхемы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Предоставляет доступ к коллекции контуров полигонального объекта (замкнутых линий), из которых состоит полигон.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ПолигональныйОбъектГеографическойСхемы, свойство Контуры

Использование в версии:

Доступен, начиная с версии 8.1.
