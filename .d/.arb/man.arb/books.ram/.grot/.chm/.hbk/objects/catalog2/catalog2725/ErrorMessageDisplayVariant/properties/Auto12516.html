ВариантОтображенияСообщенияОбОшибке (ErrorMessageDisplayVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.17.

Описание:

Если используется в качестве параметра метода ПоказатьИнформациюОбОшибке, то означает, что будет использовано значение из настроек данного пользователя.
Если же &Авто получено из настроек, то в режиме совместимости с 8.3.16 означает КраткоеПредставлениеОшибки, иначе - СообщениеОбОшибкеДляПользователя.

Использование в версии:

Доступен, начиная с версии 8.3.17.
