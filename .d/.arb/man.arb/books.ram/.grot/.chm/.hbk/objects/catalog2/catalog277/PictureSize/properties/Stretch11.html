РазмерКартинки (PictureSize)

Растянуть (Stretch)

Доступен, начиная с версии 8.0.

Описание:

Растянуть или сжать картинку таким образом, чтобы она занимала всю область, отведенную для ее отображения.

Использование в версии:

Доступен, начиная с версии 8.0.
