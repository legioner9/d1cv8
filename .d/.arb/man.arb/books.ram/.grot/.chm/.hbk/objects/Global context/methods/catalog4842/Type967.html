Глобальный контекст.Тип (Global context.Type)

Глобальный контекст (Global context)

Тип (Type)

Доступен, начиная с версии 8.0.

Синтаксис:

Тип(<ИмяТипа>)

Параметры:

<ИмяТипа> (обязательный)

Тип: Строка.
Имя получаемого типа.

Возвращаемое значение:

Тип: Тип.
Полученный тип значения.

Описание:

Получает тип по его строковому имени.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Если ТипЗнч(ТекОбъект) = Тип("СправочникСсылка.Номенклатура") Тогда
    ...
КонецЕсли;
