КнопкиКоманднойПанели.Вставить (CommandBarButtons.Insert)

КнопкиКоманднойПанели (CommandBarButtons)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Имя>, <Тип>, <Текст>, <Действие>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Позиция в коллекции кнопок, в которую необходимо вставить кнопку.

<Имя> (необязательный)

Тип: Строка.
Имя кнопки - должно соответствовать правилам имен. Уникально в пределах командной панели.

<Тип> (необязательный)

Тип: ТипКнопкиКоманднойПанели.
Тип кнопки командной панели (разделитель, действие, подменю).

<Текст> (необязательный)

Тип: Строка.
Текст, который будет выводиться на кнопке.

<Действие> (необязательный)

Тип: Действие.
Действие, которое необходимо выполнить при нажатии на данную кнопку. Для типов кнопок Разделитель и Подменю, действие указывать не нужно.

Возвращаемое значение:

Тип: КнопкаКоманднойПанели.

Описание:

Вставляет кнопку в указанную позицию коллекции кнопок командной панели.

Доступность:

Толстый клиент.

Пример:

КП = ЭлементыФормы.КоманднаяПанель;
НовоеДействие = Новый Действие("МояПроцедураВМодулеФормы");
ТипКнопки = ТипКнопкиКоманднойПанели.Действие;
НоваяКнопка = КП.Кнопки.Вставить(0, "Действие1", ТипКнопки,, НовоеДействие);

Использование в версии:

Доступен, начиная с версии 8.0.
