ПарольУстановлен (PasswordIsSet)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Булево.
Показывает, установлен ли пароль у пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - пароль установлен; Ложь - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.0.
