ЗаписьДвиженийПриПроведении (RegisterRecordsWritingOnPost)

Доступен, начиная с версии 8.2.

Значения

ЗаписыватьВыбранные (WriteSelected)
ЗаписыватьМодифицированные (WriteModified)

Описание:

Содержит варианты поведения системы при создании движений во время проведения документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
