СерииКодов (CodeSeries)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: СерииКодовПланаВидовХарактеристик, СерииКодовПланаСчетов, СерииКодовСправочника.
Определяет варианты диапазона проверки кода на уникальность и автоматического присвоения кодов ( например, ВоВсемСправочнике, ВПределахПодчинения, ВПределахПодчиненияВладельцу).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
