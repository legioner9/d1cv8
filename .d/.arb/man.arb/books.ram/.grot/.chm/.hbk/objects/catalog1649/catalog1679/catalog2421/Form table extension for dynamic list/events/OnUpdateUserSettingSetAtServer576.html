Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ПриОбновленииСоставаПользовательскихНастроекНаСервере (OnUpdateUserSettingSetAtServer)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПриОбновленииСоставаПользовательскихНастроекНаСервере(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: Булево.
Если в обработчике установить данному параметру значение Ложь, то после обработчика не будет выполняться стандартная обработка данного события.
В стандартной обработке события расширение формы динамического списка при помощи метода СоздатьЭлементыФормыПользовательскихНастроек создает в указанной группе элементы формы для быстрого редактирования пользовательских настроек.
Значение по умолчанию: Истина.

Описание:

Вызывается при обновлении состава пользовательских настроек:

  • При получении формы;
  • После загрузки настроек;
  • После изменения настроек, если изменился состав настроек.
.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
