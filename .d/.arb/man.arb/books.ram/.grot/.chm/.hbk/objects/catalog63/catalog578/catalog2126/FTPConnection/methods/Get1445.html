FTPСоединение.Получить (FTPConnection.Get)

FTPСоединение (FTPConnection)

Получить (Get)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Получить в файл

Синтаксис:

Получить(<АдресРесурса>, <ИмяВыходногоФайла>)

Параметры:

<АдресРесурса> (обязательный)

Тип: Строка.
Относительный адрес ресурса на FTP-сервере, из которого будут прочитаны данные.

<ИмяВыходногоФайла> (обязательный)

Тип: Строка.
Имя файла, в который будут помещены данные полученного ресурса.

Описание варианта метода:

Записывает данные из указанного ресурса в файл.

Вариант синтаксиса: Получить в поток

Синтаксис:

Получить(<АдресРесурса>, <ВыходнойПоток>)

Параметры:

<АдресРесурса> (обязательный)

Тип: Строка.
Относительный адрес ресурса на FTP-сервере, из которого будут прочитаны данные.

<ВыходнойПоток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток, в который записываются данные из ресурса.

Описание варианта метода:

Записывает данные из указанного ресурса в поток.

Описание:

Получает ресурс из указанного адреса.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
