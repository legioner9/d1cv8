МенеджерКалендарей.ИзменитьСобытие (CalendarsManager.UpdateEvent)

МенеджерКалендарей (CalendarsManager)

ИзменитьСобытие (UpdateEvent)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ИзменитьСобытие(<Событие>, <Данные>)

Параметры:

<Событие> (обязательный)

Тип: ЛокальныйКлючСобытияКалендаря.
Локальный ключ события календаря.

<Данные> (обязательный)

Тип: ДанныеСобытияКалендаря.
Данные события календаря.

Описание:

Изменяет событие по локальному ключу в соответствии с новыми данными.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяИзменениеСобытий.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерКалендарей, метод ПоддерживаетсяИзменениеСобытий

Использование в версии:

Доступен, начиная с версии 8.3.4.
