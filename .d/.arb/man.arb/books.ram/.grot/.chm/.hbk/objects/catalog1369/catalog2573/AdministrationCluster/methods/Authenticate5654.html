АдминистрированиеКластер (AdministrationCluster)

ВыполнитьАутентификацию (Authenticate)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ВыполнитьАутентификацию(<ИмяПользователя>, <Пароль>)

Параметры:

<ИмяПользователя> (необязательный)

Тип: Строка.
Имя пользователя кластера.

<Пароль> (необязательный)

Тип: Строка.
Пароль пользователя для аутентификации пользователя. Пользователь будет аутентифицирован, если его имя и пароль соответствуют.

Описание:

Выполняет аутентификацию администратора кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
