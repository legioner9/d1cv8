ЭлементыЛегендыГеографическойСхемы (GeographicalSchemaLegendItems)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс объекта, который необходимо переместить.

<Смещение> (обязательный)

Тип: Число.
Количество позиций, на которое необходимо переместить объект в коллекции. Положительное значение соответствует смещению к концу коллекции, отрицательное - к началу.

Описание:

Сдвигает элемент из указанного индекса на указанное смещение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
