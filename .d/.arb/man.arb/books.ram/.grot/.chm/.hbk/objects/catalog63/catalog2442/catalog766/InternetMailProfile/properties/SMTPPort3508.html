ИнтернетПочтовыйПрофиль (InternetMailProfile)

ПортSMTP (SMTPPort)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Порт протокола SMTP.
0 - означает использовать 465 порт для SSL соединений и 25 - для незащищенных.
Значение по умолчанию: 0.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ИнтернетПочтовыйПрофиль, свойство АдресСервераSMTP

Использование в версии:

Доступен, начиная с версии 8.0.
