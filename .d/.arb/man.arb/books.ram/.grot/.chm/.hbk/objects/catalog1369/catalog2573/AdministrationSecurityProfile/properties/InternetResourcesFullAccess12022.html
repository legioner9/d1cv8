Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет использование сети Интернет.
Истина – разрешено обращение к любым ресурсам Интернета. Ложь – разрешено обращение только к ресурсам сети Интернет, перечисленным в списке разрешенных ресурсов интернет профиля.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
