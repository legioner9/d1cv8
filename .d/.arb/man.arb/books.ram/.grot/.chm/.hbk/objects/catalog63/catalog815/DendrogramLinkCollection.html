СвязиДендрограммы (DendrogramLinkCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СвязьДендрограммы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются связи коллекции.
Возможно обращение к связи коллекции посредством оператора [...]. В качестве аргумента передается индекс связи (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция связей дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
