Планировщик.Напечатать (Planner.Print)

Планировщик (Planner)

Напечатать (Print)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Напечатать(<ДиалогПечати>)

Параметры:

<ДиалогПечати> (необязательный)

Тип: РежимИспользованияДиалогаПечати.
Указывает необходимость показа диалога печати перед отправкой документа на принтер.
Значение по умолчанию: НеИспользовать.

Описание:

Выводит данные планировщика на печать.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент, внешнее соединение.

Примечание:

Только для данных планировщика, которые отображаются в поле планировщика.

Использование в версии:

Доступен, начиная с версии 8.3.6.
