ОбновлениеКонфигурацииБазыДанных (DataBaseConfigurationUpdate)

Приостановлено (Suspended)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: Булево.
Показывает, было ли приостановлено фоновое обновление конфигурации базы данных.
Истина - приостановлено.
Фоновое обновление конфигурации базы данных может быть приостановлено интерактивно, из командной строки, а также вынужденными факторами, такими как аварийное завершение рабочего процесса, проводящего фоновое обновление.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
