ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageConnectionParameters)

Доступен, начиная с версии 8.3.24.

Свойства:

URL (URL)
ТипURL (URLType)

Конструкторы:

По умолчанию

Описание:

Параметры подключения внешнего хранилища двоичных данных.

Доступность:

Сервер.

Пример:

// Настройка подключения типа Path-style

cn1 = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cn1.URL = "https://example.com/bucket-name";
cn1.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных
                .ХостS3СБакетомВСоставеПути;

// Настройка подключения типа Virtual-hosted-style

cn2 = Новый ПараметрыПодключениеВнешнегоХранилищаДвоичныхДанных();
cn2.URL = "https://bucket.example.com";
cn2.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;

Использование в версии:

Доступен, начиная с версии 8.3.24.
