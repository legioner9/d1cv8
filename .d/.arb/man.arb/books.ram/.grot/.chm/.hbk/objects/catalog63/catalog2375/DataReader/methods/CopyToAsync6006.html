ЧтениеДанных.КопироватьВАсинх (DataReader.CopyToAsync)

ЧтениеДанных (DataReader)

КопироватьВАсинх (CopyToAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КопироватьВАсинх(<Поток>, <Количество>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток, ЗаписьДанных.
Поток или объект ЗаписьДанных, в который требуется скопировать данные.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые необходимо скопировать. Если не задано, будут скопированы все данные до конца потока.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - фактическое количество скопированных байтов типа Число.

Описание:

Асинхронная функция. Копирует заданное количество байтов из объекта ЧтениеДанных в указанный поток или объект ЗаписьДанных.
Если количество не указано, копируются все данные из объекта.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

ЧтениеДанных, метод НачатьКопированиеВ
ЧтениеДанных, метод КопироватьВ

Использование в версии:

Доступен, начиная с версии 8.3.18.
