СистемнаяИнформация (SystemInfo)

ТипПлатформы (PlatformType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ТипПлатформы, Неопределено.
Содержит тип платформы, в рамках которой осуществляется работа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме веб-клиента:
  • Windows_x86– для 32-х разрядных браузеров в среде Windows вне зависимости от разрядности операционной системы,
  • Windows_x86_64 – для 64-х разрядных браузеров в среде 64-х разрядной Windows,
  • Linux_x86 – для 32-х разрядных браузеров в среде Linux вне зависимости от разрядности операционной системы,
  • Linux_x86_64 – для 64-х разрядных браузеров в среде 64-разрядной Linux,
  • Неопределено - в остальных случаях.
В мобильной версии "1С:Предприятия" в режиме совместимости Версия8_3_22 и ниже содержит Неопределено.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
