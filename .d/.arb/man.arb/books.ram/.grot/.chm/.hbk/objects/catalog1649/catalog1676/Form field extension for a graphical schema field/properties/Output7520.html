Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

Вывод (Output)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеВывода.
Управляет ограничением вывода на печать, сохранения, работы с буфером обмена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Может иметь значение, отличное от значения свойства Вывод документа, с которым это поле связано.

Использование в версии:

Доступен, начиная с версии 8.2.
