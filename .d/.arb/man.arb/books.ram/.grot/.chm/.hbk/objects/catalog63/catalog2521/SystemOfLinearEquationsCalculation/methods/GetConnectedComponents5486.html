РасчетСистемЛинейныхУравнений.ПолучитьКомпонентыСвязности (SystemOfLinearEquationsCalculation.GetConnectedComponents)

РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ПолучитьКомпонентыСвязности (GetConnectedComponents)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьКомпонентыСвязности(<СпособПолученияКомпонент>)

Параметры:

<СпособПолученияКомпонент> (необязательный)

Тип: СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.
Определяет способ получения компонент связности.
В режиме совместимости Версия8_3_14 значение по умолчанию – КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент.
Значение по умолчанию: КомпонентыСлабойСвязности.

Возвращаемое значение:

Тип: ТаблицаЗначений.
Таблица значений с двумя колонками: в первой колонке содержится номер узла, а во второй - номер компоненты, которой принадлежит узел.

Описание:

Возвращает компоненты связности.
Для корректной работы метода требуется:

  • установить значения свойств ИсточникДанныхСвязей, КолонкаУравненияВСвязях, КолонкаПеременныеВСвязях,
  • в ОписанияСистемЛинейныхУравнений добавлен объект ОписаниеСистемыЛинейныхУравнений, в котором указана только КолонкаКоэффициентовВСвязях.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
