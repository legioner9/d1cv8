ОтображениеЗаголовкаШкалыДиаграммы (ShowChartScaleTitle)

ОтображениеЗаголовкаШкалыДиаграммы (ShowChartScaleTitle)

Доступен, начиная с версии 8.3.10.

Значения

Авто (Auto)
НеОтображать (DontShow)
Отображать (Show)

Описание:

Определяет варианты отображения заголовка шкалы диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ShowChartScaleTitle.

Использование в версии:

Доступен, начиная с версии 8.3.10.
