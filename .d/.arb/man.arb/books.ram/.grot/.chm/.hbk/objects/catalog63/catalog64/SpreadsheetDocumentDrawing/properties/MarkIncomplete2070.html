ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Булево.
Необходимость помечать рисунок.
Истина - отметка выведена, Ложь - в противном случае.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо для рисунков типа Текст, Примечание.

Использование в версии:

Доступен, начиная с версии 8.0.
