НаборСхемXML.Проверить (XMLSchemaSet.Validate)

НаборСхемXML (XMLSchemaSet)

Проверить (Validate)

Доступен, начиная с версии 8.1.

Синтаксис:

Проверить(<РазыменовательСхем>)

Параметры:

<РазыменовательСхем> (необязательный)

Тип: НаборСхемXML.
Набор схем XML, используемый для разыменования ссылок на схему из проверяемого набора.

Возвращаемое значение:

Тип: Булево.
Истина - успешная проверка набора схем; Ложь - в противном случае.

Описание:

Проверяет набора схем XML на правильность в соответствии со стандартом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
