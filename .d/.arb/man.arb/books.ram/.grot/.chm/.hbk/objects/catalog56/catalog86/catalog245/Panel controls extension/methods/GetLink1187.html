Расширение элементов управления, расположенных на панели (Panel controls extension)

ПолучитьПривязку (GetLink)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьПривязку(<Граница>, <ПервыйЭлемент>, <ГраницаПервогоЭлемента>, <ВторойЭлемент>, <ГраницаВторогоЭлемента>)

Параметры:

<Граница> (обязательный)

Тип: ГраницаЭлементаУправления.
Граница, привязки которой вернет данный метод.

<ПервыйЭлемент> (обязательный)

Элемент управления, к которому жестко привязана указанная граница.

<ГраницаПервогоЭлемента> (обязательный)

Тип: ГраницаЭлементаУправления.
Граница элемента управления, к которой жестко привязана указанная граница.

<ВторойЭлемент> (обязательный)

Элемент управления, к которому пропорционально привязана указанная граница.

<ГраницаВторогоЭлемента> (обязательный)

Тип: ГраницаЭлементаУправления.
Граница элемента управления, к которой пропорционально привязана указанная граница.

Описание:

Получает привязку границы элемента.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
