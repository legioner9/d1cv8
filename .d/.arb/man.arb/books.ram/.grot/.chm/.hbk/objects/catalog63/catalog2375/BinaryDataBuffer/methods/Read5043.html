БуферДвоичныхДанных.Прочитать (BinaryDataBuffer.Read)

БуферДвоичныхДанных (BinaryDataBuffer)

Прочитать (Read)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Прочитать(<Позиция>, <Количество>)

Параметры:

<Позиция> (обязательный)

Тип: Число.
Позиция, начиная с которой требуется прочитать байты.

<Количество> (обязательный)

Тип: Число.
Количество байтов, которое требуется прочитать.

Возвращаемое значение:

Тип: БуферДвоичныхДанных.

Описание:

Выполняет чтение байтов из буфера и помещает их в новый буфер.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
