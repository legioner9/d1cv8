МенеджерКалендарей.НайтиСобытия (CalendarsManager.FindEvents)

МенеджерКалендарей (CalendarsManager)

НайтиСобытия (FindEvents)

Доступен, начиная с версии 8.3.4.

Синтаксис:

НайтиСобытия(<Отбор>, <Календарь>)

Параметры:

<Отбор> (необязательный)

Тип: ОтборКомпоновкиДанных.
Отбор компоновки данных.

<Календарь> (необязательный)

Тип: ЛокальныйКлючКалендаря.
Локальный ключ календаря.

Возвращаемое значение:

Тип: Массив.
Массив ЛокальныйКлючСобытияКалендаря, соответствующих отбору и принадлежащих календарю.
Если календарь не задан, поиск осуществляется по всем календарям. Если отбор не задан, возвращает все события.

Описание:

Осуществляет поиск событий.
Отбор может быть установлен по следующим полям:

  • Тема(Subject),
  • Начало(Start),
  • Конец(End),
  • Место(Location),
  • Описание(Description),
  • ВесьДень(AllDay),
  • Организатор(Organizer),
  • ЧасовойПояс(TimeZone).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
