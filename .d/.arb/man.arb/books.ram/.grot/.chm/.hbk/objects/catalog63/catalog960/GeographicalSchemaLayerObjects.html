ОбъектыСлояГеографическойСхемы (GeographicalSchemaLayerObjects)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ТочечныйОбъектГеографическойСхемы, ПолилинейныйОбъектГеографическойСхемы, ПолигональныйОбъектГеографическойСхемы, МноготочечныйОбъектГеографическойСхемы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Методы:

Добавить (Add)
Количество (Count)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Предоставляет доступ к коллекции объектов, входящих в слой географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

СлойГеографическойСхемы, свойство Объекты

Использование в версии:

Доступен, начиная с версии 8.1.
