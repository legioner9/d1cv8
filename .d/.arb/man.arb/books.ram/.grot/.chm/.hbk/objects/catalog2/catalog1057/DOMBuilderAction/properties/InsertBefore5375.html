ДействиеПостроителяDOM (DOMBuilderAction)

ВставитьПеред (InsertBefore)

Доступен, начиная с версии 8.1.

Описание:

Вставить до. Результат чтения добавляется непосредственно до узла контекста документа. В качестве узла контекста могут выступать узлы, имеющие в качестве родительского: Элемент или ФрагментДокумента.

Использование в версии:

Доступен, начиная с версии 8.1.
