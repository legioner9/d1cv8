МетодКластеризации.ДальняяСвязь (ClusterizationMethod.FurthestNeighbor)

МетодКластеризации (ClusterizationMethod)

ДальняяСвязь (FurthestNeighbor)

Доступен, начиная с версии 8.0.

Описание:

Метод "Дальняя связь". Метод кластеризации, в котором объект присоединяется к той группе, для которой расстояние до наиболее дальнего объекта минимально.

Использование в версии:

Доступен, начиная с версии 8.0.
