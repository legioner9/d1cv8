Шрифт.Масштаб (Font.Scale)

Шрифт (Font)

Масштаб (Scale)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: Число.
Задает увеличение/уменьшение размера шрифта на указанное количество процентов относительно исходного.
Значение по умолчанию - 100%.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.1.
