Расширение поля ввода для описания типа (Text box for type description extension)

РазрешитьСоставнойТип (TypeDomainEnabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Разрешает или запрещает выбор составного описания типов. Если Ложь, то описание типов состоит из одного типа.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
