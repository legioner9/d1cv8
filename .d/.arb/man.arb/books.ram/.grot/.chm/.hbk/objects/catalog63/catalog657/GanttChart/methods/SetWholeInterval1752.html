ДиаграммаГанта.УстановитьПолныйИнтервал (GanttChart.SetWholeInterval)

ДиаграммаГанта (GanttChart)

УстановитьПолныйИнтервал (SetWholeInterval)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПолныйИнтервал(<ДатаНачала>, <ДатаОкончания>)

Параметры:

<ДатаНачала> (обязательный)

Тип: Дата.
Дата начала интервала. Не может быть меньше даты "01.01.1000".

<ДатаОкончания> (обязательный)

Тип: Дата.
Дата окончания интервала. Не может быть больше даты "01.01.3000".

Описание:

Устанавливает полный интервал.
Следует использовать при АвтоОпределениеПолногоИнтервала = Ложь.
ВНИМАНИЕ! Установку полного интервала можно производить только с помощью этого метода. Свойства НачалоПолногоИнтервала и КонецПолногоИнтервала только для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ДиаграммаГанта, свойство НачалоПолногоИнтервала
ДиаграммаГанта, свойство КонецПолногоИнтервала
ДиаграммаГанта, свойство АвтоОпределениеПолногоИнтервала

Использование в версии:

Доступен, начиная с версии 8.0.
