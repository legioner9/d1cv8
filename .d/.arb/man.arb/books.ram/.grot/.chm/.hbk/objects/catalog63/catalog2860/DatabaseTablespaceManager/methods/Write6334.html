МенеджерТабличногоПространстваБазыДанных (DatabaseTablespaceManager)

Записать (Write)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Записать()

Описание:

Записывает табличное пространство в базу данных.
Если данные или индексы объекта метаданных уже включены в другое записанное пространство, генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
