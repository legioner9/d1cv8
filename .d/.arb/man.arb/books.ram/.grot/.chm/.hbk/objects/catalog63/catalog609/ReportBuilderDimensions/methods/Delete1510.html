ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Измерение>)

Параметры:

<Измерение> (обязательный)

Тип: ИзмерениеПостроителяОтчета.
Удаляемое измерение.

Описание:

Удаляет измерение из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Пример 1

Для Индекс=0 По ПостроительОтчета.ИзмеренияСтроки.Количество()-1 Цикл
    ТекущаяСтрока=ПостроительОтчета.ИзмеренияСтроки[Индекс];
    Если ТекущаяСтрока.Имя = ВыбИмя Тогда
        ПутьКДанным = ТекущаяСтрока.ПутьКДанным;
        ПостроительОтчета.ИзмеренияСтроки.Удалить(ТекущаяСтрока);
        Прервать;
    КонецЕсли;
КонецЦикла;

// Пример 2

Если ПостроительОтчета.ИзмеренияКолонки.Количество()>0 Тогда
    ТекущаяКолонка=ПостроительОтчета.ИзмеренияКолонки[0];
    ПостроительОтчета.ИзмеренияКолонки.Удалить(ТекущаяКолонка);
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
