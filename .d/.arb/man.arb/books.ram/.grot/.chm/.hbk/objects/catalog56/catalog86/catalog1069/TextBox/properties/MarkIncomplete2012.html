ПолеВвода.ОтметкаНезаполненного (TextBox.MarkIncomplete)

ПолеВвода (TextBox)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Свойство указывает, нужно ли сейчас в поле ввода отображать особую отметку или нет. Если свойство имеет значение Истина, то независимо от хранимых в поле ввода данных будет отображаться особая отметка, если Ложь, то независимо от хранимых в поле ввода данных особая отметка отображаться не будет.
При начале редактирования свойство сбрасывается, даже если АвтоОтметкаНезаполненного = Ложь.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
