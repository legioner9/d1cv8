РегистрНакопленияСписок.<Имя регистра накопления> (AccumulationRegisterList.<Accumulation register name>)

Доступен, начиная с версии 8.0.

Свойства:

Колонки (Columns)
Отбор (Filter)
Порядок (Order)

Методы:

Обновить (Refresh)

Описание:

Предназначен для управления списком записей регистра накопления, отображаемым в табличном поле.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Форма = РегистрыНакопления.УчетНоменклатуры.ПолучитьФормуСписка();
Форма.СписокРегистра.Порядок.Установить("Склад");
Форма.СписокРегистра.Отбор.Номенклатура.Установить(ВыбТовар);
Форма.Открыть();

Использование в версии:

Доступен, начиная с версии 8.0.
