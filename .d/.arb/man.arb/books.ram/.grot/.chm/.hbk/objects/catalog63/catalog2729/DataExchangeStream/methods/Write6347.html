ПотокОбменаДанными (DataExchangeStream)

Записать (Write)

Доступен, начиная с версии 8.3.22.

Синтаксис:

Записать(<Значение>)

Параметры:

<Значение> (необязательный)

Записываемое значение, например: Справочник.Объект, Справочник.Ссылка, и т.д.

В обработчиках ПередНачаломОтправкиДанныхГлавному, ПередНачаломОтправкиДанныхПодчиненному можно передать объект данных типа ФиксированнаяСтруктура, для передачи сырых данных и обработать в обработчиках ПриПолученииДанныхОтГлавного, ПриПолученииДанныхОтПодчиненного.

Описание:

Записывает объект, который может быть передан в обработчики ПередСозданиемНачальногоОбраза, ПередНачаломОтправкиДанныхПодчиненному для записи объектов в обмене данными.

Доступность:

Мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
