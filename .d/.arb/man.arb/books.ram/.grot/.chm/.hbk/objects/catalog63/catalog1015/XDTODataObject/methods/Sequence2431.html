ОбъектXDTO.Последовательность (XDTODataObject.Sequence)

ОбъектXDTO (XDTODataObject)

Последовательность (Sequence)

Доступен, начиная с версии 8.1.

Синтаксис:

Последовательность()

Возвращаемое значение:

Тип: ПоследовательностьXDTO.

Описание:

Получает последовательность объекта.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
