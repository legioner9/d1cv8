Глобальный контекст (Global context)

НачатьСозданиеДвоичныхДанныхИзФайла (BeginCreateBinaryDataFromFile)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьСозданиеДвоичныхДанныхИзФайла(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:
  • <ДвоичныеДанные> - Экземпляр объекта ДвоичныеДанные, содержащий данные из файла.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла, из которого надо создать ДвоичныеДанные.

Описание:

Начинает создание двоичных данных из файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

При вызове этого метода в веб-клиенте запрашивается разрешение пользователя на выполнение операции.

См. также:

Глобальный контекст, метод СоздатьДвоичныеДанныеИзФайлаАсинх

Использование в версии:

Доступен, начиная с версии 8.3.9.
