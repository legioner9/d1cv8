НастройкиНачальнойСтраницы (HomePageSettings)

ПолучитьСоставФорм (GetForms)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьСоставФорм()

Возвращаемое значение:

Тип: СоставФормНачальнойСтраницы.

Описание:

Получает список форм начальной страницы. Список формируется из состава форм, заданных в конфигураторе, к которому применены пользовательские настройки.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
