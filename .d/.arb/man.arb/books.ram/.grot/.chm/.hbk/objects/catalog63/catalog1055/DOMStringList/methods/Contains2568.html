СписокСтрокDOM.Содержит (DOMStringList.Contains)

СписокСтрокDOM (DOMStringList)

Содержит (Contains)

Доступен, начиная с версии 8.1.

Синтаксис:

Содержит(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Проверяемая строка.

Возвращаемое значение:

Тип: Булево.
Истина - строка входит в коллекцию; Ложь - в противном случае.

Описание:

Определяет наличие строки в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
