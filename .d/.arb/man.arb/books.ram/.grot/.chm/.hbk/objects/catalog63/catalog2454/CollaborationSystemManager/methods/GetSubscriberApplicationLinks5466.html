МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьСовместноеИспользованиеПриложенийАбонента (GetSubscriberApplicationLinks)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьСовместноеИспользованиеПриложенийАбонента()

Возвращаемое значение:

Тип: Массив.
Массив элементов типа СовместноеИспользованиеПриложенийСистемыВзаимодействия.
В свойстве Приложения объекта СовместноеИспользованиеПриложенийСистемыВзаимодействия всегда возвращается массив из двух элементов.

Описание:

Возвращает описание совместного использования приложений системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
