XBase.Ключ (XBase.Key)

XBase (XBase)

Ключ (Key)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: КлючXBase.
Предоставляет доступ к объекту КлючXBase. Состав свойств объекта повторяет набор свойств полей объекта XBase. Значения свойств используются для вычисления выражения индекса при использовании метода НайтиПоКлючу.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

XBase, метод НайтиПоКлючу

Использование в версии:

Доступен, начиная с версии 8.0.
