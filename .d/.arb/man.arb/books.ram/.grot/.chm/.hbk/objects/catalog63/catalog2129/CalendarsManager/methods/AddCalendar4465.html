МенеджерКалендарей.ДобавитьКалендарь (CalendarsManager.AddCalendar)

МенеджерКалендарей (CalendarsManager)

ДобавитьКалендарь (AddCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ДобавитьКалендарь(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: ДанныеКалендаряУчетнойЗаписи.
Данные календаря учетной записи.

Возвращаемое значение:

Тип: ЛокальныйКлючКалендаря.

Описание:

Добавляет новый календарь в мобильном устройстве и возвращает его локальный ключ.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяДобавлениеКалендарей.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

МенеджерКалендарей, метод ПоддерживаетсяДобавлениеКалендарей

Использование в версии:

Доступен, начиная с версии 8.3.4.
