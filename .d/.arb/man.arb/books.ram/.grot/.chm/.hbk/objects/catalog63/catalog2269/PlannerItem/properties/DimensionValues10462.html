ЗначенияИзмерений (DimensionValues)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: ФиксированноеСоответствие.
Коллекция значений измерений. В качестве ключа выступает Значение измерения. В качестве значения - Значение элемента измерения (ИзмерениеПланировщика.<Элемент>.Значение).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
