СериализаторXDTO.XMLЗначение (XDTOSerializer.XMLValue)

СериализаторXDTO (XDTOSerializer)

XMLЗначение (XMLValue)

Доступен, начиная с версии 8.1.

Синтаксис:

XMLЗначение(<Тип>, <СтрокаXML>)

Параметры:

<Тип> (обязательный)

Тип: Тип.
Тип, значение которого надо получить при преобразовании из строки.

<СтрокаXML> (обязательный)

Тип: Строка.
Строка, содержащая строковое представление значения, полученное из XML.

Возвращаемое значение:

Тип: Null, Булево, Число, Строка, Дата, УникальныйИдентификатор, ДвоичныеДанные, ХранилищеЗначения; значения перечислений, значения системных перечислений (ВидСчета, ВидДвиженияБухгалтерии, ВидДвиженияНакопления, ДопустимыйЗнак, ДопустимаяДлина, ЧастиДаты), Неопределено или все ссылки на объекты базы данных.

Описание:

Выполняет преобразование из строки, полученной из текста элемента или значения атрибута XML, в значение в соответствии с указанным типом.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполняет действие, обратное действию метода XMLСтрока.

Использование в версии:

Доступен, начиная с версии 8.1.
