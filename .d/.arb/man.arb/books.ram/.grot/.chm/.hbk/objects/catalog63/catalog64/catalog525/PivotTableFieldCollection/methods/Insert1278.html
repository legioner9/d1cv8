КоллекцияПолейСводнойТаблицы.Вставить (PivotTableFieldCollection.Insert)

КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Поле>, <Позиция>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, ПолеСводнойТаблицы, КоллекцияПолейСводнойТаблицы.
Вставляемое поле.

<Позиция> (обязательный)

Тип: Число.
Позиция, в которую будет вставлено поле или коллекция.

Возвращаемое значение:

Тип: ПолеСводнойТаблицы, КоллекцияПолейСводнойТаблицы.

Описание:

Вставляет поле или другую коллекцию полей в указанную позицию коллекции полей сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
