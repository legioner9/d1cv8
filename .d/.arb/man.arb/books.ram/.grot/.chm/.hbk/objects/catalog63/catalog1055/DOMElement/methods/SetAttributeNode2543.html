ЭлементDOM.УстановитьУзелАтрибута (DOMElement.SetAttributeNode)

ЭлементDOM (DOMElement)

УстановитьУзелАтрибута (SetAttributeNode)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьУзелАтрибута(<НовыйАтрибут>)

Параметры:

<НовыйАтрибут> (обязательный)

Тип: АтрибутDOM.
Устанавливаемый атрибут DOM.

Возвращаемое значение:

Тип: АтрибутDOM, Неопределено.
Возвращает атрибут, который был замещен новым атрибутом или Неопределено, если такого нет.

Описание:

Устанавливает атрибут в элемент.
Если значение локального имени и URI пространства имен совпадают с одним из существующих атрибутов, то данный атрибут заменяется на новый.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:
  • узел-атрибут был создан другим документом;
  • данный узел в режиме Только для чтения;
  • устанавливаемый атрибут является уже узлом-атрибутом другого узла Элемент, необходимо явное клонирование атрибута.

Использование в версии:

Доступен, начиная с версии 8.1.
