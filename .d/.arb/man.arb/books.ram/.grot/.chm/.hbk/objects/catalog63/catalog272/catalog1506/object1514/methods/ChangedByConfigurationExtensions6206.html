ОбъектМетаданных: Интерфейс.ЕстьИзмененияРасширениямиКонфигурации (MetadataObject: Interface.ChangedByConfigurationExtensions)

ОбъектМетаданных: Интерфейс (MetadataObject: Interface)

ЕстьИзмененияРасширениямиКонфигурации (ChangedByConfigurationExtensions)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЕстьИзмененияРасширениямиКонфигурации()

Описание:

Возвращает Истина, если сам объект или один из его дочерних объектов изменен (в том числе, и добавлением нового дочернего объекта) расширениями в текущем сеансе.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
