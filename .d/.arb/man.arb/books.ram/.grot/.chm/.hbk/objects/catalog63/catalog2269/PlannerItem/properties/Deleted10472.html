ЭлементПланировщика.Удаленный (PlannerItem.Deleted)

ЭлементПланировщика (PlannerItem)

Удаленный (Deleted)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено;.
Если элемент является замещающим в расписании, свойство указывает, что замещенный элемент был удален из расписания. Иначе - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЭлементПланировщика, свойство Расписание
ЭлементПланировщика, свойство ДатаЗамещения
ЭлементПланировщика, свойство ЗамещающиеЭлементы
ЭлементПланировщика, свойство Родитель

Доступен, начиная с версии 8.3.6.
