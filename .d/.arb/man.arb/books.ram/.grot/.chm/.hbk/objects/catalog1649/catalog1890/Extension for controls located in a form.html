Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

Доступен, начиная с версии 8.2.

События:

<Имя события> (<Event name>)

Описание:

Дополнительные свойства и методы элемента управления в форме. Эти свойства и методы появляются у любого элемента управления, расположенного на форме.

ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы все свойства расширения становятся недоступны.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
