ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриСозданииВложенныхБизнесПроцессов (OnCreateSubBusinessProcesses)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриСозданииВложенныхБизнесПроцессов(<ТочкаМаршрута>, <ФормируемыеБизнесПроцессы>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка бизнес-процесса, на которой происходит создание вложенных бизнес-процессов.

<ФормируемыеБизнесПроцессы>

Тип: Массив.
Массив формируемых бизнес-процессов.

<Отказ>

Тип: Булево.
Признак отказа от записи и старта вложенных бизнес-процессов. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись и старт вложенных бизнес-процессов произведен не будет.
Значение по умолчанию: Ложь.

Описание:

Возникает после формирования списка ведущих задач для вложенных бизнес-процессов. Список ведущих задач может быть сформирован либо платформой автоматически, либо переопределен разработчиком в обработчике события ПриСозданииЗадач.
Этот же обработчик предназначен для того, чтобы изменить списки ведущих задач и вложенных бизнес-процессов, сформированные платформой. Например, добавить к ним новые бизнес-процессы и ведущие задачи, или удалить ненужные.
Если вложенный бизнес-процесс имеет несколько точек старта, то в этом обработчике можно записать и стартовать вложенный бизнес-процесс с нужной точки старта.
Запись и старт вложенных бизнес-процессов будет осуществлен после завершения обработчика.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида ВложенныйБизнесПроцесс.
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода Сообщить.

Использование в версии:

Доступен, начиная с версии 8.0.
