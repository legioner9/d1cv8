ПорядокКнопокКоманднойПанели (CommandBarButtonOrder)

ПорядокКнопокКоманднойПанели (CommandBarButtonOrder)

Доступен, начиная с версии 8.0.

Значения

Возр (Asc)
НеУпорядочивать (DontOrder)
Убыв (Desc)

Описание:

Содержит варианты порядка кнопок командной панели в подменю.
Кнопки упорядочиваются по тексту в пределах разделителей.

Доступность:

Толстый клиент.

ПорядокКнопокКоманднойПанели, свойство Возр
ПорядокКнопокКоманднойПанели, свойство Убыв

Доступен, начиная с версии 8.0.
