ТипРезультатаDOMXPath (DOMXPathResultType)

Доступен, начиная с версии 8.1.

Значения

Булево (Boolean)
Любой (Any)
ЛюбойНеупорядоченныйУзел (AnyUnorderedNode)
НеупорядоченныйИтераторУзлов (UnorderedNodeIterator)
НеупорядоченныйСнимокУзлов (UnorderedNodeSnapshot)

Описание:

Содержит варианты типов результата при вычислении выражения XPath.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
