МенеджерКриптографии.ДобавитьАрхивнуюМеткуВремениАсинх (CryptoManager.AddArchiveTimestampAsync)

МенеджерКриптографии (CryptoManager)

ДобавитьАрхивнуюМеткуВремениАсинх (AddArchiveTimestampAsync)

Доступен, начиная с версии 8.3.20.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ДобавитьАрхивнуюМеткуВремениАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как ДвоичныеДанные, Поток, ПотокВПамяти или ФайловыйПоток.

Описание варианта метода:

Данные подписи возвращаются методом как Обещание с результатом ДвоичныеДанные.

Вариант синтаксиса: Данные подписи сохраняются в файл

Синтаксис:

ДобавитьАрхивнуюМеткуВремениАсинх(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как ДвоичныеДанные, Поток, ПотокВПамяти или ФайловыйПоток.

<ВыходныеДанные> (обязательный)

Тип: Строка, Поток, ПотокВПамяти, ФайловыйПоток.
Имя файла или поток, в который будут выгружены данные подписи.
Если Неопределено, то результат работы метода вернется в возвращаемом значении типа ДвоичныеДанные.

Описание варианта метода:

Данные подписи записываются в файл. Возвращает Обещание с результатом Неопределено.

Возвращаемое значение:

Тип: Обещание.
Обещание. Результат выполнения обещания – двоичные данные подписи с добавленной архивной меткой времени (если ВыходныеДанные = Неопределено) иначе Неопределено.

Описание:

Асинхронная функция. Добавляет в подпись архивную метку времени версии 3.
При добавлении метки выполняется обращение к серверам меток времени, указанным в поле АдресаСерверовМетокВремени.

Для добавления архивной метки времени подпись должна иметь тип CAdES-BES или выше.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.
