ТабличныйДокумент.НомерПервойСтраницы (SpreadsheetDocument.FirstPageNumber)

ТабличныйДокумент (SpreadsheetDocument)

НомерПервойСтраницы (FirstPageNumber)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит число, используемое в качестве начального номера при нумерации страниц.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - 1.

Использование в версии:

Доступен, начиная с версии 8.3.12.
