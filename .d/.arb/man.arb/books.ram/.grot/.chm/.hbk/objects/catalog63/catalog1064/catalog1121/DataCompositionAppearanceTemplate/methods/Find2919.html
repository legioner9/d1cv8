МакетОформленияКомпоновкиДанных.Найти (DataCompositionAppearanceTemplate.Find)

МакетОформленияКомпоновкиДанных (DataCompositionAppearanceTemplate)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: Строка.
Идентификатор параметра.

Возвращаемое значение:

Тип: ЗначениеПараметраКомпоновкиДанных или ЗначениеПараметраНастроекКомпоновкиДанных, Неопределено.
Если не найден, возвращается Неопределено.

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
