ФорматированныйДокумент.Вставить (FormattedDocument.Insert)

ФорматированныйДокумент (FormattedDocument)

Вставить (Insert)

Доступен, начиная с версии 8.2.

Синтаксис:

Вставить(<Закладка>, <Значение>, <ТипЭлемента>)

Параметры:

<Закладка> (обязательный)

Тип: ЗакладкаФорматированногоДокумента.
Определяет позицию вставки, в которую необходимо вставить элемент форматированного документа.

<Значение> (необязательный)

Тип: Произвольный.
Содержит вставляемое в форматированный документ значение.
В зависимости от значения параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.
Если значение параметра ТекстФорматированногоДокумента, то в указанную позицию документа будет вставлено строковое представление переданного значения.
Если значение КартинкаФорматированногоДокумента, то тип параметра должен быть Картинка, ДвоичныеДанные или Строка, содержащая URL картинки. В противном случае вызывается исключение. Картинка будет вставлена в указанное место.
Если значение параметра ПереводСтрокиФорматированногоДокумента, то независимо от значения параметра в указанную позицию вставляется перевод строки.
Значение по умолчанию: Неопределено.

<ТипЭлемента> (необязательный)

Тип: Тип.
Указывает тип элемента, который нужно вставить. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.
Элементами коллекции могут быть:
1) ТекстФорматированногоДокумента
2) КартинкаФорматированногоДокумента
3) ПереводСтрокиФорматированногоДокумента
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.
Значение по умолчанию: ТекстФорматированногоДокумента.

Возвращаемое значение:

Тип: ТекстФорматированногоДокумента, ПереводСтрокиФорматированногоДокумента, КартинкаФорматированногоДокумента.
Возращает элемент, вставленный в форматированный документ.

Описание:

Вставляет строку, картинку или перевод строки в указанную позицию форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Вставка картинки доступна только на сервере.
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
