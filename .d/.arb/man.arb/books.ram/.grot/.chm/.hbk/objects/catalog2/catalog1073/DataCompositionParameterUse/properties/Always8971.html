ИспользованиеПараметраКомпоновкиДанных (DataCompositionParameterUse)

Всегда (Always)

Доступен, начиная с версии 8.2.

Описание:

Значение параметра всегда получается из настроек компоновки данных. Даже если в настройках компоновки данных использование параметра выключено.

Использование в версии:

Доступен, начиная с версии 8.2.
