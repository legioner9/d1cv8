ТочкиДиаграммы (ChartPoints)

Вставить (Insert)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Вставить(<Индекс>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Если индекс больше текущего количества точек, то новая точка вставляется в конец коллекции. Если индекс меньше нуля, то новая точка вставляется в начало коллекции.

<Текст> (необязательный)

Тип: Строка.
Название точки.

Возвращаемое значение:

Тип: ТочкаДиаграммы.
Возвращает вставленную точку.

Описание:

Вставляет новую точку по указанному индексу. Все точки с индексом больше, либо равным переданному, сдвигаются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
