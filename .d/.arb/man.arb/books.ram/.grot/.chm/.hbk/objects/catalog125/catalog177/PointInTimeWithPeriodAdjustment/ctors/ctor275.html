МоментВремениСУточнениемПериода (PointInTimeWithPeriodAdjustment)

На основании даты, уточнения периода и ссылки

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый МоментВремениСУточнениемПериода(<Дата>, <УточнениеПериода>, <Ссылка>)

Параметры:

<Дата> (обязательный)

Тип: Дата.
Дата и время.

<УточнениеПериода> (необязательный)

Тип: Число.
Значение уточнения периода. Неотрицательное целое число. Максимальное значение определяется длиной уточнения периода регистра, для которого будет использовано значение.
Значение по умолчанию: 0.

<Ссылка> (необязательный)

Тип: Ссылка на объект информационной базы.
Ссылка на объект базы данных.

Описание:

Создает момент времени с уточнением периода по дате, уточнению периода и ссылке на объект базы данных.

Использование в версии:

Доступен, начиная с версии 8.3.9.
