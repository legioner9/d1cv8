Лево (Left)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Число.
Содержит положение левой границы элемента графической схемы, отсчитываемое от левой границы графической схемы (в логических единицах).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
