ДокументDOM.СоздатьАтрибут (DOMDocument.CreateAttribute)

ДокументDOM (DOMDocument)

СоздатьАтрибут (CreateAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

СоздатьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.
Имя создаваемого атрибута.

Описание варианта метода:

Причины вызова исключений:
  • Имя атрибута содержит символы, недопустимые стандартом XML.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

СоздатьАтрибут(<URIПространстваИмен>, <КвалифицированноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен.

<КвалифицированноеИмя> (обязательный)

Тип: Строка.
Квалифицированное имя атрибута.

Описание варианта метода:

Таблица заполнения значений свойств атрибута:
ДокументDOM.ИмяУзла - по значению параметра <КвалифицированноеИмя>;
ДокументDOM.URIПространстваИмен - по значению параметра <URIПространстваИмен>;
ДокументDOM.Префикс - префикс, извлеченный из параметра <КвалифицированноеИмя>, или пустая строка, если префикс отсутствует;
ДокументDOM.ЛокальноеИмя - локальное имя, извлеченное из параметра <КвалифицированноеИмя>;
ИмяЭлемента - по значению параметра <КвалифицированноеИмя>;
ДокументDOM.ЗначениеУзла - пустая строка.
Причины вызова исключений:
  • квалифицированное имя содержит символы, недопустимые стандартом XML данного документа;
  • квалифицированное имя содержит префикс, значение которого недопустимо для данного значения URI пространства имен.

Возвращаемое значение:

Тип: АтрибутDOM.
В качастве имени узла (nodeName) устанавливается name. Локальное имя (localName), префикс (prefix) и URI пространства имен (namespaceURI) не устанавливаются.
Значение атрибута также не устанавливается.

Описание:

Создает новый узел DOM атрибут XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
