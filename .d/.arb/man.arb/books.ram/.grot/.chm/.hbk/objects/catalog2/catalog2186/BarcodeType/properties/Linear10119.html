ТипШтрихКода.Линейный (BarcodeType.Linear)

ТипШтрихКода (BarcodeType)

Линейный (Linear)

Доступен, начиная с версии 8.3.5.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа.
Поддерживаются штрихкоды следующих типов: UPCA, UPCE, EAN13, EAN8, Codabar, Code93, Code128, RSS14, RSSExpanded, Code39, ITF.

Использование в версии:

Доступен, начиная с версии 8.3.5.
