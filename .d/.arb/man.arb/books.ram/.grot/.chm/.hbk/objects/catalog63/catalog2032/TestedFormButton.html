ТестируемаяКнопкаФормы (TestedFormButton)

Доступен, начиная с версии 8.3.1.

Свойства:

Вид (Type)
Имя (Name)
ТекстЗаголовка (TitleText)

Методы:

Активизировать (Activate)
Нажать (Click)
ТекущаяВидимость (CurrentVisible)
ТекущаяДоступность (CurrentEnable)
ТекущееПометка (CurrentCheck)

Описание:

Представляет абстракцию элемента управления КнопкаФормы. Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством Вид.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
