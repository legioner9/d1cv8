COM-класс.FileName (ISecurityProfileCOMClass.FileName)

COM-класс (ISecurityProfileCOMClass)

FileName (FileName)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит имя файла, по которому создается объект методом ПолучитьCOMОбъект с пустым значением второго параметра. В общем случае строка используется как текстовое представление моникера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
