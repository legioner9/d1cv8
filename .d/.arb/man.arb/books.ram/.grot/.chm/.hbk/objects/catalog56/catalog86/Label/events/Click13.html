Надпись.Нажатие (Label.Click)

Надпись (Label)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на надпись, если у надписи свойство Гиперссылка имеет значение Истина.

Доступность:

Толстый клиент.

См. также:

Надпись, свойство Гиперссылка

Использование в версии:

Доступен, начиная с версии 8.0.
