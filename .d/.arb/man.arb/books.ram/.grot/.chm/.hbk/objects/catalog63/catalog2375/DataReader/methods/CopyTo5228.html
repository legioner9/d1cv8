ЧтениеДанных.КопироватьВ (DataReader.CopyTo)

ЧтениеДанных (DataReader)

КопироватьВ (CopyTo)

Доступен, начиная с версии 8.3.10.

Синтаксис:

КопироватьВ(<Поток>, <Количество>)

Параметры:

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток: ЗаписьДанных.
Поток или объект ЗаписьДанных, в который требуется скопировать данные.

<Количество> (необязательный)

Тип: Число.
Количество байтов, которые необходимо скопировать. Если не задано, будут скопированы все данные до конца потока.

Возвращаемое значение:

Тип: Число.
Фактически записанное количество байтов.

Описание:

Копирует заданное количество байтов из объекта ЧтениеДанных в указанный поток или объект ЗаписьДанных.
Если количество не указано, копируются все данные из объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ЧтениеДанных, метод НачатьКопированиеВ
ЧтениеДанных, метод КопироватьВАсинх

Использование в версии:

Доступен, начиная с версии 8.3.10.
