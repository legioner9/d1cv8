ОбъектМетаданныхКонфигурация.ЕстьИзмененияРасширениямиКонфигурации (ConfigurationMetadataObject.ChangedByConfigurationExtensions)

ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ЕстьИзмененияРасширениямиКонфигурации (ChangedByConfigurationExtensions)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЕстьИзмененияРасширениямиКонфигурации()

Описание:

Возвращает Истина, если конфигурация или один из ее дочерних объектов изменены (в том числе и добавлением нового дочернего объекта) расширениями в текущем сеансе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
