Группа (Group)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ГруппаКоманд.
Группа команд, в которую входит общая команда. Устанавливает расположение команды в глобальном командном интерфейсе приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
