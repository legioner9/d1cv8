ИтераторУзловDOM (DOMNodeIterator)

Доступен, начиная с версии 8.1.

Свойства:

Корень (Root)
РаскрыватьСсылкиНаСущности (ExpandEntityReference)
Фильтр (Filter)

Методы:

Отсоединить (Detach)
ПредыдущийУзел (PreviousNode)
СледующийУзел (NextNode)

Конструкторы:

На основании документа DOM

Описание:

Итератор узлов DOM.
Позволяет обходить узлы иерархии узлов DOM в прямом и обратном порядке.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ДокументDOM, метод СоздатьИтераторУзлов
ДокументHTML, метод СоздатьИтераторУзлов

Использование в версии:

Доступен, начиная с версии 8.1.
