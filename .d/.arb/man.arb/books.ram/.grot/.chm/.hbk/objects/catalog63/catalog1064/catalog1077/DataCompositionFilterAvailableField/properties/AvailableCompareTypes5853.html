ДоступноеПолеОтбораКомпоновкиДанных (DataCompositionFilterAvailableField)

ДоступныеВидыСравнения (AvailableCompareTypes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: СписокЗначений.
Содержит список видов сравнения, которые могут быть использованы при построении элемента отбора с использованием этого поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
