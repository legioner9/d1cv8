СерииСлояГеографическойСхемы (GeographicalSchemaLayerSeries)

Доступен, начиная с версии 8.1.

Элементы коллекции:

СерияДанныхСлояГеографическойСхемы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Свойства:

<Имя серии> (<Series name>)

Методы:

Добавить (Add)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Представляет собой коллекцию, содержащую серии слоя географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
