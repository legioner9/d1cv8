ИспользованиеПолосыПрокрутки (ScrollBarUse)

Доступен, начиная с версии 8.0.

Значения

ИспользоватьАвтоматически (AutoUse)
ИспользоватьВсегда (UseAlways)
НеИспользовать (DontUse)

Описание:

Содержит варианты использования полосы прокрутки. Используется для определения признаков использования вертикальной и горизонтальной полосы прокрутки у элементов формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

См. также:

ТабличноеПоле, свойство ВертикальнаяПолосаПрокрутки
ТаблицаФормы, свойство ВертикальнаяПолосаПрокрутки
ТабличноеПоле, свойство ГоризонтальнаяПолосаПрокрутки
ТаблицаФормы, свойство ГоризонтальнаяПолосаПрокрутки

Использование в версии:

Доступен, начиная с версии 8.0.
