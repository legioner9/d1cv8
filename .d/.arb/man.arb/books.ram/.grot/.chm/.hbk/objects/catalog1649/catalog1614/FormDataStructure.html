Описание:

Структура данных для моделирования данных, редактируемых в форме клиентского приложения. Моделирует объект с набором свойств.
Для текущих данных таблицы формы клиентского приложения обращение к значениям осуществляется по именам колонок.
Для параметров формы клиентского приложения обращение к значениям осуществляется по именам параметров.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
