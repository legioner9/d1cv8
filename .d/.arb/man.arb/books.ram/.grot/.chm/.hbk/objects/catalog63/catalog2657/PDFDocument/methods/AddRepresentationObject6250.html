ДокументPDF.ДобавитьОтображаемыйОбъект (PDFDocument.AddRepresentationObject)

ДокументPDF (PDFDocument)

ДобавитьОтображаемыйОбъект (AddRepresentationObject)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ДобавитьОтображаемыйОбъект(<ОписаниеОтображаемогоОбъекта>)

Параметры:

<ОписаниеОтображаемогоОбъекта> (обязательный)

Тип: ОписаниеОтображаемогоОбъектаPDF.
Объект, содержащий описание отображения объекта в PDF-документе.

Описание:

Располагает отображение объекта (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
