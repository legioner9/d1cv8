ПеречислениеСписок.<ИмяПеречисления> (EnumList.<Enum name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Отбор.
Состоит из единственного элемента управления отбором "Ссылка". Для данного элемента осмысленными являются четыре варианта отбора: равно, не равно, в списке, не в списке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если указать любой другой вариант отбора, список перечисления окажется пустым.

Использование в версии:

Доступен, начиная с версии 8.0.
