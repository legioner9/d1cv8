ИнтернетПрокси.НеИспользоватьПроксиДляАдресов (InternetProxy.BypassProxyOnAddresses)

ИнтернетПрокси (InternetProxy)

НеИспользоватьПроксиДляАдресов (BypassProxyOnAddresses)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Массив.
Список адресов, соединение с которыми осуществляется без использования прокси.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
