ФормаКлиентскогоПриложения.ЗаблокироватьДанныеФормыДляРедактирования (ClientApplicationForm.LockFormDataForEdit)

ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗаблокироватьДанныеФормыДляРедактирования (LockFormDataForEdit)

Доступен, начиная с версии 8.2.

Синтаксис:

ЗаблокироватьДанныеФормыДляРедактирования()

Описание:

Устанавливает блокировку сохраняемых данных формы в информационной базе для редактирования.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
