ПользовательСистемыВзаимодействия (CollaborationSystemUser)

Внешний (External)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если Истина, пользователь является внешним.
Установить свойство можно только для нового объекта.
При записи нового внешнего объекта должно быть записано свойство АдресЭлектроннойПочтыДляАутентификации. Если уже есть внешний пользователь с таким же значением свойства АдресЭлектроннойПочтыДляАутентификации, будет сгенерировано исключение.
Значение по умолчанию - Ложь.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
