Соединение с агентом сервера.GetLocks (IServerAgentConnection.GetLocks)

Соединение с агентом сервера (IServerAgentConnection)

GetLocks (GetLocks)

Доступен, начиная с версии 8.1.

Синтаксис:

GetLocks(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер серверов, из которого должен быть получен массив зарегистрированных информационных баз. Может быть получено при помощи метода GetServerWorkingProcesses.

Возвращаемое значение:

Тип: COMSafeArray.
Массив блокировок объектов, установленных клиентами кластера. Каждая блокировка представлена объектом с интерфейсом Блокировка.

Описание:

Получает массив блокировок, установленных клиентами кластера, и хранящихся в кластере. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера. Предоставляемая информация о блокировках не требует аутентификации пользователей информационных баз.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
