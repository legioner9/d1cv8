ОбрабатываемаяКартинка (ProcessingPicture)

УстановитьПлотность (SetDensity)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьПлотность(<ПоГоризонтали>, <ПоВертикали>)

Параметры:

<ПоГоризонтали> (обязательный)

Тип: Число.
Количество пикселей на дюйм изображения по горизонтали.

<ПоВертикали> (необязательный)

Тип: Число.
Количество пикселей на дюйм изображения по вертикали.

Описание:

Влияет на изменение метаинформации, предназначенной для определения соответствия между размером в пикселях и реальным размером картинки.
Для форматов, которые не хранят эту информацию, изменение будет формально успешным, но не приведет к изменениям.
Если не задана отдельно плотность по вертикали, она принимается равной плотности по горизонтали.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
