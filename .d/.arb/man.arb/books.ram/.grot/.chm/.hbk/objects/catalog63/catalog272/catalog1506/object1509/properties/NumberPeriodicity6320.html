ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

ПериодичностьНомера (NumberPeriodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: ПериодичностьНомераБизнесПроцесса, ПериодичностьНомераДокумента.
Устанавливает пределы контроля уникальности номеров и период повторяемости номеров (например, Год, Квартал, Месяц, День).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
