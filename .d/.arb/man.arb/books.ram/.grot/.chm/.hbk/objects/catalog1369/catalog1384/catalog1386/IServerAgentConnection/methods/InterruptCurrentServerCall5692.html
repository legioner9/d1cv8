Соединение с агентом сервера (IServerAgentConnection)

InterruptCurrentServerCall (InterruptCurrentServerCall)

Доступен, начиная с версии 8.3.13.

Синтаксис:

InterruptCurrentServerCall(<Кластер>, <Сеанс>, <СообщениеОбОшибке>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, из которого будет удален сеанс. Должен содержать объект, полученный методом GetClusters.

<Сеанс> (обязательный)

Тип: Сеанс.
Сеанс, который будет удален. Должен содержать объект, полученный методом GetSessions или GetInfoBaseSessions.

<СообщениеОбОшибке> (необязательный)

Тип: Строка.
Сообщение для пользователя, которое содержит причину прерывания текущего серверного вызова.
Значение по умолчанию: "Выполнение текущей операции прервано администратором".

Описание:

Прерывает выполнение текущего серверного вызова.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.13.
