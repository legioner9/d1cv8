ГеографическаяСхема (GeographicalSchema)

Показать (Show)

Доступен, начиная с версии 8.0.

Синтаксис:

Показать(<Заголовок>, <ИмяФайла>)

Параметры:

<Заголовок> (необязательный)

Тип: Строка.
Содержит заголовок окна.

<ИмяФайла> (необязательный)

Тип: Строка.
Имя файла, в который может быть сохранен документ.

Описание:

Открывает окно для показа и редактирования географической схемы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
