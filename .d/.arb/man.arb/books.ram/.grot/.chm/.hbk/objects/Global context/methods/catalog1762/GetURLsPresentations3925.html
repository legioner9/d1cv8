Глобальный контекст (Global context)

ПолучитьПредставленияНавигационныхСсылок (GetURLsPresentations)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПредставленияНавигационныхСсылок(<МассивСсылок>)

Параметры:

<МассивСсылок> (обязательный)

Тип: Массив.
Массив ссылок.
Пример ссылки:
e1cib/data/Справочник.Товары?ref="ae92005056c0000811dcf650677a5575" указывает на элемент справочника Товары с наименованием "Гвозди".

Возвращаемое значение:

Тип: Массив.
Возвращает массив представлений ссылок.
При работе через HTTP абсолютные ссылки со схемой "e1c" интерпретируются как ссылки, относящиеся к текущей информационной базе.
Например, для ссылки, указанной в элементе массива (передаваемом в параметре) строк, возвращает объект ПредставлениеНавигационнойСсылки, у которого свойство Текст="Деньги".
Неопределено - объект отсутствует в информационной базе (удален или не относится к текущей базе).

Описание:

Получает массив представлений ссылок ПредставлениеНавигационнойСсылки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Примеры ссылок:
Навигационная: e1cib/navigationpoint/Продажи/Документ.РасходТовара.Команда.ОткрытьСписок
На объект: e1cib/data/Документ.РасходТовара?ref="8ca0000d8843cd1b11dc8d043d71007f"
На отчет или обработку: e1cib/app/Отчет.ДинамикаПродаж

Использование в версии:

Доступен, начиная с версии 8.2.
