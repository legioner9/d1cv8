РегистрСведенийКлючЗаписи.<Имя регистра сведений> (InformationRegisterRecordKey.<Information register name>)

<Имя общего реквизита> (<Common attribute name>)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Произвольный.
Имя общего реквизита - разделителя, у которого уровень разделения установлен в НезависимоИСовместно. Количество атрибутов определяется количеством таких общих реквизитов, в состав которых входит регистр.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
