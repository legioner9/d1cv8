ПолеФормы.ГоризонтальноеПоложениеВШапке (FormField.HeaderHorizontalAlign)

ПолеФормы (FormField)

ГоризонтальноеПоложениеВШапке (HeaderHorizontalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: ГоризонтальноеПоложениеЭлемента.
Определяет способ горизонтального выравнивания текста в заголовке таблицы. Данное свойство имеет смысл только для полей, расположенных в таблице (представляющих собой колонки).
Запрещено использовать значение Авто.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
