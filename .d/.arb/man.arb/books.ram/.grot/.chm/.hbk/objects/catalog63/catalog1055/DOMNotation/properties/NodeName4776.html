НотацияDOM.ИмяУзла (DOMNotation.NodeName)

НотацияDOM (DOMNotation)

ИмяУзла (NodeName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Имя узла DOM. Имя узла присутствует не у каждого типа узла DOM.
Узел Имя узла
  • Атрибут - Attribute.name;
  • СекцияCDATA - "#cdata-section";
  • Комментарий - "#comment";
  • Документ - "#document";
  • ФрагментДокумента - "#document-fragment";
  • ОпределениеТипаДокумента - DocumentType.name;
  • Элемент - Element.tagName;
  • Сущность - имя сущности;
  • СсылкаНаСущность - имя ссылки на сущность;
  • Нотация - имя нотации;
  • ИнструкцияОбработки - ProcessingInstruction.target;
  • Текст - "#text".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
