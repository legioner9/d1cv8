НайтиПоПолю (FindByField)

Доступен, начиная с версии 8.3.3.

Синтаксис:

НайтиПоПолю(<ИмяПоля>, <ЗначениеПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.
Имя поля, как оно задано в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме ХранилищеЗначения и строк произвольной длины.

<ЗначениеПоля> (обязательный)

Тип: Произвольный.
Значение поля, по которому должен выполняться поиск.

Возвращаемое значение:

Тип: ВнешнийИсточникДанныхКубТаблицаИзмеренияСсылка.<Имя внешнего источника>.<Имя куба>.<Имя таблицы измерения>.

Описание:

Осуществляет поиск элемента по значению поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
