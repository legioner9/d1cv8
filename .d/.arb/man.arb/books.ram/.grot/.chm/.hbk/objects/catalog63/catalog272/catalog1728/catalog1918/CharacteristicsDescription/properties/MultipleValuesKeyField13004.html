ОписаниеХарактеристик (CharacteristicsDescription)

ПолеКлючаМножественныхЗначений (MultipleValuesKeyField)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: Поле.
Содержит имя поля таблицы значений характеристик, в котором хранится ключ множественности значения характеристики.
Если указано, то для немножественных характеристик значения фильтруются по значению по умолчанию для типа поля. Таким образом устраняется "задваивание" данных после изменения характеристики с множественной на немножественную.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
