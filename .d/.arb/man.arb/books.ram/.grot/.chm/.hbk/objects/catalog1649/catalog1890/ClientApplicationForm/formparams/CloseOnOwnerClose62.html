ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗакрыватьПриЗакрытииВладельца (CloseOnOwnerClose)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.
Указывает необходимость закрытия формы при закрытии владельца.
При создании формы значение данного параметра устанавливается в свойство ЗакрыватьПриЗакрытииВладельца.
Истина - форма будет закрыта при закрытии владельца.

Использование в версии:

Доступен, начиная с версии 8.2.
