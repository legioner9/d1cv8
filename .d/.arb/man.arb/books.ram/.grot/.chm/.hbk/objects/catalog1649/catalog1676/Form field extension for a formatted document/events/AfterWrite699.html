Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ПослеЗаписи (AfterWrite)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПослеЗаписи()

Описание:

Вызывается после интерактивного сохранения форматированного документа.

В веб-клиенте событие вызывается только при установленном расширении для работы с файлами

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
