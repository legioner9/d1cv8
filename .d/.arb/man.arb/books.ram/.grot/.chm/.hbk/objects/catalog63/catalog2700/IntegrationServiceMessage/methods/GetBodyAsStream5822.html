СообщениеСервисаИнтеграции.ПолучитьТелоКакПоток (IntegrationServiceMessage.GetBodyAsStream)

СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: Поток.
Возвращает поток со следующими свойствами в зависимости от состояния сообщения:
  • При создании сообщения - ДоступнаЗапись имеет значение Истина, ДоступноЧтение и ДоступноИзменениеПозиции имеют значение Ложь.
  • При отправке и для отправленного сообщения - ДоступноЧтение имеет значение Истина, ДоступнаЗапись и ДоступноИзменениеПозиции имеют значение Ложь.
  • Для полученного сообщения - ДоступноЧтение имеет значение Истина, ДоступнаЗапись и ДоступноИзменениеПозиции имеют значение Ложь.

Описание:

Получить тело сообщения как поток, доступный для чтения и записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
