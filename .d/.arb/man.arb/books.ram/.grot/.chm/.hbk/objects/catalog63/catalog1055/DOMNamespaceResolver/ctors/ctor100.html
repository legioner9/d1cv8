РазыменовательПространствИменDOM (DOMNamespaceResolver)

На основании соответствия

Доступен, начиная с версии 8.1.

Синтаксис:

Новый РазыменовательПространствИменDOM(<Соответствие>)

Параметры:

<Соответствие> (обязательный)

Тип: Соответствие.
Ключом в соответствии является префикс, значением - URI пространства имен.

Описание:

Создает объект разыменователя URI пространств имен на основе Соответствие.

Использование в версии:

Доступен, начиная с версии 8.1.
