<Имя колонки> (<Column name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.
Набор свойств содержит значения в колонках таблицы значений по конкретной строке. Доступ к значению осуществляется по имени колонки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
