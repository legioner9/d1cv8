ПланГлобальногоПоиска.Вставить (GlobalSearchPlan.Insert)

ПланГлобальногоПоиска (GlobalSearchPlan)

Вставить (Insert)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: Вставка значения

Синтаксис:

Вставить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Расположение элемента в коллекции.

<Значение> (обязательный)

Тип: ЭлементПланаГлобальногоПоиска.
Вставляемое значение.

Описание варианта метода:

Вставляет элемент (значение) в заданную позицию коллекции.

Вариант синтаксиса: По виду поиска

Синтаксис:

Вставить(<Индекс>, <ВидПоиска>, <Порядок>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Расположение элемента в коллекции.

<ВидПоиска> (обязательный)

Тип: СтандартныйВидГлобальногоПоиска.
Вид поиска.

<Порядок> (необязательный)

Тип: Число.
Порядок выдачи результатов поиска.
Значение по умолчанию: 0.

Описание варианта метода:

Вставляет элемент (вид поиска) в заданную позицию коллекции.

Вариант синтаксиса: По обработчику

Синтаксис:

Вставить(<Индекс>, <ИмяПроцедуры>, <Модуль>, <НаСервере>, <Фоновый>, <Порядок>, <ДополнительныеПараметры>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Расположение элемента в коллекции.

<ИмяПроцедуры> (обязательный)

Тип: Строка.
Имя экспортируемой процедуры общего модуля.

<Модуль> (обязательный)

Тип: Строка.
Имя общего модуля.

<НаСервере> (обязательный)

Тип: Булево.
Признак выполнения поиска на сервере.

<Фоновый> (необязательный)

Тип: Булево.
Признак выполнения поиска в фоновом задании.
Значение по умолчанию: Ложь.

<Порядок> (необязательный)

Тип: Число.
Порядок выдачи результатов поиска.
Значение по умолчанию: 0.

<ДополнительныеПараметры> (необязательный)

Тип: Произвольный.
Дополнительные параметры, передаваемые в вызываемую процедуру.
Значение по умолчанию: Неопределено.

Описание:

Вставляет элемент в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
