ТипТекстаНовыхЭлементовПланировщика (NewPlannerItemsTextType)

Доступен, начиная с версии 8.3.18.

Значения

Строка (String)
ФорматированнаяСтрока (FormattedString)

Описание:

Содержит тип текста новых элементов планировщика. Влияет на тип параметра <Текст> события ПередСозданием, а также тип элементов планировщика, создаваемых через методы Добавить и Вставить.

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML.

Использование в версии:

Доступен, начиная с версии 8.3.18.
