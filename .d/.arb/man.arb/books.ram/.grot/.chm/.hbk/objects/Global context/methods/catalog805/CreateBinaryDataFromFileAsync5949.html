Глобальный контекст (Global context)

СоздатьДвоичныеДанныеИзФайлаАсинх (CreateBinaryDataFromFileAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СоздатьДвоичныеДанныеИзФайлаАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла, из которого надо создать ДвоичныеДанные.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа ДвоичныеДанные, содержащий данные из файла.

Описание:

Асинхронная функция. Создает двоичные данные из файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

Глобальный контекст, метод НачатьСозданиеДвоичныхДанныхИзФайла

Использование в версии:

Доступен, начиная с версии 8.3.18.
