СписокXDTO (XDTOList)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ОбъектXDTO, ЗначениеXDTO
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значение элемента списка.
Возможно обращение к элементу списка посредством оператора [...]. В качестве аргумента передается индекс элемента.

Свойства:

Владелец (Owner)
ВладеющееСвойство (OwningProperty)

Методы:

Вставить (Insert)
Добавить (Add)
Количество (Count)
Очистить (Clear)
Получить (Get)
ПолучитьXDTO (GetXDTO)
Удалить (Delete)
Установить (Set)

Описание:

Список элементов данных XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ОбъектXDTO, метод ПолучитьСписок
