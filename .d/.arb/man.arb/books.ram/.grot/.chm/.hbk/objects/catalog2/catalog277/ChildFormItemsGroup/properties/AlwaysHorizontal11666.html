ГруппировкаПодчиненныхЭлементовФормы (ChildFormItemsGroup)

ГоризонтальнаяВсегда (AlwaysHorizontal)

Доступен, начиная с версии 8.3.12.

Описание:

Горизонтальная группировка независимо от наличия достаточного места по ширине формы.
Если места недостаточно:
  • В версии для персонального компьютера на форме отображается полоса прокрутки.
  • На мобильном устройстве часть содержимого становится недоступна.

Использование в версии:

Доступен, начиная с версии 8.3.12.
