ЧтениеСообщенияОбмена (ExchangeMessageReader)

ЧтениеXML (XMLReader)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ЧтениеУзловDOM, ЧтениеFastInfoset, ЧтениеXML, Неопределено.
Содержит объект, с помощью которого осуществляется чтение сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу НачатьЧтение и после обращения к методу ЗакончитьЧтение или ПрерватьЧтение значением данного свойства будет Неопределено.

Использование в версии:

Доступен, начиная с версии 8.0.
