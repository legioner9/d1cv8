ЗаписьДанных.ЗаписатьБайтАсинх (DataWriter.WriteByteAsync)

ЗаписьДанных (DataWriter)

ЗаписатьБайтАсинх (WriteByteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьБайтАсинх(<Число>)

Параметры:

<Число> (обязательный)

Тип: Число.
Целое число, которое будет записано в целевой поток. Значение числа должно находиться в диапазоне от 0 до 255.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Записывает один байт в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

ЗаписьДанных, метод НачатьЗаписьБайта
ЗаписьДанных, метод ЗаписатьБайт

Использование в версии:

Доступен, начиная с версии 8.3.18.
