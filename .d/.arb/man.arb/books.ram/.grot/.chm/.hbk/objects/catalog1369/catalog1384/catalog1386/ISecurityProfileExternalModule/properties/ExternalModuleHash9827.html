Внешний модуль.ExternalModuleHash (ISecurityProfileExternalModule.ExternalModuleHash)

Внешний модуль (ISecurityProfileExternalModule)

ExternalModuleHash (ExternalModuleHash)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит контрольную сумму внешнего модуля, вычисленная алгоритмом SHA-1, преобразованная к строке base64. Для построения контрольной суммы можно использовать:
  • объект ХешированиеДанных с указанием хеш-функции SHA1;
  • метод глобального контекста Base64Строка.
Для расширений конфигурации может быть получена из свойства ХешСумма.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
