Доступен, начиная с версии 8.1.

Элементы коллекции:

ОграничениеИспользованияДоступногоПоляКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ограничения использования полей.
Возможно обращение к ограничениям использования полей посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Индекс (IndexOf)

Очистить (Clear)
ПолеДоступно (FieldEnabled)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция ограничений использования доступных полей системы компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
