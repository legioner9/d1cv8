Системная форма данных версии истории данных (System form for the Data history version)

Доступен, начиная с версии 8.3.13.

Параметры формы:

Данные (Data)
НомерВерсии (VersionNumber)

Описание:

Форма данных версии истории данных, отображает отчет "Данные версии".

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.22.
