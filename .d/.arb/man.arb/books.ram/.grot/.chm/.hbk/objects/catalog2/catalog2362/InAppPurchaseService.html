СервисВстроенныхПокупок (InAppPurchaseService)

Доступен, начиная с версии 8.3.8.

Значения

AppleInAppPurchase (AppleInAppPurchase)
GooglePlayInAppBilling (GooglePlayInAppBilling)
HuaweiInAppPurchase (HuaweiInAppPurchase)
WindowsInAppPurchase (WindowsInAppPurchase)

Описание:

Содержит варианты используемых сервисов встроенных покупок.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен (http://v8.1c.ru/8.3/data/ext). Имя типа XDTO: InAppPurchaseService.

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.25.
