ЗадачаОбъект.<Имя задачи>.ПолучитьСсылкуНового (TaskObject.<Task name>.GetNewObjectRef)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: ЗадачаСсылка.<Имя задачи>.

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом УстановитьСсылкуНового значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ЗадачаМенеджер.<Имя задачи>, метод ПолучитьСсылку

Использование в версии:

Доступен, начиная с версии 8.0.
