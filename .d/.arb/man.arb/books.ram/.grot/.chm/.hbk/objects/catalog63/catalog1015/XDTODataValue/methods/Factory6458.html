ЗначениеXDTO.Фабрика (XDTODataValue.Factory)

ЗначениеXDTO (XDTODataValue)

Фабрика (Factory)

Доступен, начиная с версии 8.3.22.

Синтаксис:

Фабрика()

Возвращаемое значение:

Тип: ФабрикаXDTO.

Описание:

Возвращает экземпляр фабрики XDTO, использовавшийся при создании данного значения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
