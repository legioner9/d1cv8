ЭлементыZipФайла (ZipFileEntries)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементZipФайла
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элемент коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Методы:

Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Содержит коллекцию элементов ZIP файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЧтениеZipФайла, свойство Элементы

Использование в версии:

Доступен, начиная с версии 8.0.
