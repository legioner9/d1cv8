ПоследовательностьXDTO.УстановитьЗначение (XDTOSequence.SetValue)

ПоследовательностьXDTO (XDTOSequence)

УстановитьЗначение (SetValue)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьЗначение(<Индекс>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс в последовательности.

<Элемент> (обязательный)

Тип: ОбъектXDTO, ЗначениеXDTO, Неопределено.
Элемент данных XDTO.

Описание:

Устанавливает значение по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если по указанному индексу в последовательности содержится текст, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
