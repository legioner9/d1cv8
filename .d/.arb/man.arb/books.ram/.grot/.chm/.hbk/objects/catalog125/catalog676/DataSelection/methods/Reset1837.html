ВыборкаДанных.Сбросить (DataSelection.Reset)

ВыборкаДанных (DataSelection)

Сбросить (Reset)

Доступен, начиная с версии 8.0.

Синтаксис:

Сбросить()

Описание:

Позиционирует выборку в начало.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

После вызова метода Следующий выборка спозиционируется на первую запись в ней.

Пример:

Выб.Сбросить();
Пока Выб.Следующий() Цикл
    Объект = Выб.Получить();
    ЗаписатьXML(Запись, Объект);
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
