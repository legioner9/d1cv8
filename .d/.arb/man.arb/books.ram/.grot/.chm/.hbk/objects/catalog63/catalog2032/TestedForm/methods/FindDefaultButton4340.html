НайтиКнопкуПоУмолчанию (FindDefaultButton)

Доступен, начиная с версии 8.3.3.

Синтаксис:

НайтиКнопкуПоУмолчанию()

Возвращаемое значение:

Тип: Неопределено, ТестируемаяКнопкаФормы.
Неопределено - кнопка по умолчанию отсутствует.

Описание:

Получает кнопку по умолчанию.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
