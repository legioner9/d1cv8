КвитанцияВстроеннойПокупки (InAppPurchaseReceipt)

Доступен, начиная с версии 8.3.8.

Свойства:

Данные (Data)
Сервис (Service)

Конструкторы:

По умолчанию

Описание:

Полученная квитанция о встроенной покупке, сделанной в мобильном приложении.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен (http://v8.1c.ru/8.3/data/ext). Имя типа XDTO: InAppPurchaseReceipt.

Использование в версии:

Доступен, начиная с версии 8.3.8.
