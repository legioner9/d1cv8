ВидФлажка (CheckBoxType)

Доступен, начиная с версии 8.3.2.

Значения

Авто (Auto)
Выключатель (Switch)
Тумблер (Tumbler)
Флажок (CheckBox)

Описание:

Содержит варианты режимов отображения поля вида ПолеФлажка в форме клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.2.
