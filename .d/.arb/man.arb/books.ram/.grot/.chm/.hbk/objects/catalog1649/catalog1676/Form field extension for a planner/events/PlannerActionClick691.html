Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаДействиеПланировщика (PlannerActionClick)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НажатиеНаДействиеПланировщика(<ЭлементПланировщика>, <ДействиеЭлементаПланировщика>)

Параметры:

<ЭлементПланировщика>

Тип: ЭлементПланировщика.
Элемент планировщика, которому принадлежит нажатое действие, на которое было произведено нажатие.

<ДействиеЭлементаПланировщика>

Тип: ДействиеЭлементаПланировщика.
Действие элемента планировщика, на которое было произведено нажатие.

Описание:

Возникает при нажатии левой кнопкой мыши на действие элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
