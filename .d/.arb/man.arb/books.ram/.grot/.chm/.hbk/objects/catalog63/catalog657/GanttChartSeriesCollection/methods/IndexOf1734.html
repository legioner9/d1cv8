СерииДиаграммыГанта.Индекс (GanttChartSeriesCollection.IndexOf)

СерииДиаграммыГанта (GanttChartSeriesCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Серия>)

Параметры:

<Серия> (обязательный)

Тип: СерияДиаграммыГанта.
Серия диаграммы Ганта.

Возвращаемое значение:

Тип: Число.
Индекс указанной серии в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс серии в коллекции серий диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
