НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияКолонкамиАнализаДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки колонок анализа данных.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Описание:

Коллекция элементов настройки колонок анализа данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля списка колонок анализа данных, свойство НастройкаКолонок

Использование в версии:

Доступен, начиная с версии 8.0.
