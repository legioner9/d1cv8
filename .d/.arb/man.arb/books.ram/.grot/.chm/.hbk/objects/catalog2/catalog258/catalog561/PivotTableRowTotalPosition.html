ПоложениеИтоговСтрокСводнойТаблицы (PivotTableRowTotalPosition)

Доступен, начиная с версии 8.0.

Значения

Верх (Top)
Низ (Bottom)

Описание:

Содержит варианты положения итоговых строк в строках сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется.

См. также:

СводнаяТаблица, свойство ПоложениеИтоговСтрок

Использование в версии:

Доступен, начиная с версии 8.0.
