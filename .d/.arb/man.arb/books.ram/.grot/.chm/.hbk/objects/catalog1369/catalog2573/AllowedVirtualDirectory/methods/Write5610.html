РазрешенныйВиртуальныйКаталог (AllowedVirtualDirectory)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового виртуального каталога или обновление существующего. Виртуальный каталог идентифицируется логическим URL-адресом.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
