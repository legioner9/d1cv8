МоментВремени (PointInTime)

На основании даты и ссылки

Доступен, начиная с версии 8.0.

Синтаксис:

Новый МоментВремени(<Дата>, <Ссылка>)

Параметры:

<Дата> (обязательный)

Тип: Дата.
Дата и время.

<Ссылка> (необязательный)

Тип: Ссылка на объект информационной базы.
Ссылка на любой объект в базе данных.

Описание:

Создает момент времени по дате и ссылке на объект в базе данных.
Также может быть создан с использованием функциональной формы оператора Новый.

Пример:

Параметры = Новый Массив(2);
Параметры[0] = '20020703151902';
Параметры[1] = ТекДокумент;
Момент = Новый (Тип("МоментВремени"),Параметры);

Использование в версии:

Доступен, начиная с версии 8.0.
