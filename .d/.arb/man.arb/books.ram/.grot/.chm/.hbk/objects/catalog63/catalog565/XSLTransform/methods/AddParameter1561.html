ПреобразованиеXSL.ДобавитьПараметр (XSLTransform.AddParameter)

ПреобразованиеXSL (XSLTransform)

ДобавитьПараметр (AddParameter)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Задать значение параметра с полным именем

Синтаксис:

ДобавитьПараметр(<ПолноеИмя>, <Значение>)

Параметры:

<ПолноеИмя> (обязательный)

Тип: Строка.
Полное имя параметра.

<Значение> (обязательный)

Тип: Булево, Число, Строка.
Значение параметра.

Вариант синтаксиса: Задать значение параметра с локальным именем и URI пространства имен

Синтаксис:

ДобавитьПараметр(<ЛокальноеИмя>, <URIПространстваИмен>, <Значение>)

Параметры:

<ЛокальноеИмя> (обязательный)

Тип: Строка.
Локальное имя параметра.

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен параметра.

<Значение> (обязательный)

Тип: Булево, Число, Строка.
Значение параметра.

Описание:

Задает значение параметра преобразования.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если значение параметра с указанным именем уже было задано, то оно переопределяется.

Пример:

Преобразование = Новый ПреобразованиеXSL;
Преобразование.ЗагрузитьИзФайла("c:/transforms/transform.xsl");
Преобразование.ДобавитьПараметр("low_sales", 30000);

Использование в версии:

Доступен, начиная с версии 8.0.
