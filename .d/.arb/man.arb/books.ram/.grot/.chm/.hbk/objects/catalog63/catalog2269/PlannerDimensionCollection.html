КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Доступен, начиная с версии 8.3.6.

Элементы коллекции:

ИзмерениеПланировщика
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются измерения планировщика.
Возможно обращение к измерению планировщика посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Представляет собой коллекцию измерений планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
