КонструкторЗапроса.ОткрытьАсинх (QueryWizard.DoAsync)

КонструкторЗапроса (QueryWizard)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - текст, если конструктор был закрыт нажатием кнопки "OK".

Описание:

Асинхронная функция. Открывает немодально конструктор запроса.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

КонструкторЗапроса, метод Показать

Использование в версии:

Доступен, начиная с версии 8.3.18.
