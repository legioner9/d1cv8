ЧтениеFastInfoset.Прочитать (FastInfosetReader.Read)

ЧтениеFastInfoset (FastInfosetReader)

Прочитать (Read)

Доступен, начиная с версии 8.1.

Синтаксис:

Прочитать()

Возвращаемое значение:

Тип: Булево.
Истина - очередной узел прочитан; Ложь - текст XML завершился.

Описание:

Считывает очередной узел XML. При этом свойствам ЧтениеFastInfoset.ТипУзла, ЧтениеFastInfoset.Имя, ЧтениеFastInfoset.ЛокальноеИмя, ЧтениеFastInfoset.Префикс, ЧтениеFastInfoset.URIПространстваИмен и ЧтениеFastInfoset.Значение присваиваются значения, соответствующие прочитанным данным.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Чтение.ОткрытьФайл("c:/docs/data.xml");
Пока Чтение.Прочитать() Цикл
    // Обработка узла XML

КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.1.
