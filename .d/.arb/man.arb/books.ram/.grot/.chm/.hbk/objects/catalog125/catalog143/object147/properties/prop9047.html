ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

<Имя общего реквизита> (<Common attribute name>)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Произвольный.
Набор свойств содержит значения общих реквизитов, входящих в объект метаданных. Но только тех общих реквизитов, которые или не являются разделителями (РазделениеДанных = НеИспользовать), или являются разделителями (РазделениеДанных = Разделять) и имеют настройку разделения ИспользованиеРазделяемыхДанных = НезависимоИСовместно.
Доступ к значению осуществляется по имени, как оно задано в Конфигураторе.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
