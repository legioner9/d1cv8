СертификатКлиентаОС (OSClientCertificate)

Доступен, начиная с версии 8.3.21.

Свойства:

СпособВыбораСертификатаОС (OSCertificateSelectMode)

Конструкторы:

По способу выбора сертификата
По умолчанию

Описание:

Объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы:

  • В ОС Windows работает полностью аналогично объекту СертификатКлиентаWindows.
  • В ОС Linux работает полностью аналогично объекту СертификатКлиентаLinux.
  • В ОС macOS работает полностью аналогично объекту СертификатКлиентаMacOS.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
