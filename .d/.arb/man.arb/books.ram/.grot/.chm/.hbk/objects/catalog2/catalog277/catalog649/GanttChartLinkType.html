ТипСвязиДиаграммыГанта (GanttChartLinkType)

Доступен, начиная с версии 8.0.

Значения

КонецКонец (EndEnd)
КонецНачало (EndBegin)
НачалоКонец (BeginEnd)
НачалоНачало (BeginBegin)

Описание:

Тип связи между интервалами диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChartLinkType.
