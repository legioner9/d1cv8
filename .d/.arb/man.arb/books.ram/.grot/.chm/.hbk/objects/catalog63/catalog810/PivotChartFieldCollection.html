КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

КоллекцияПолейСводнойДиаграммы, ПолеСводнойДиаграммы
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной диаграммы или коллекция полей сводной диаграммы.
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс (нумерация с 0) поля.

Свойства:

<Имя поля> (<Field name>)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Содержит коллекцию полей сводной диаграммы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
