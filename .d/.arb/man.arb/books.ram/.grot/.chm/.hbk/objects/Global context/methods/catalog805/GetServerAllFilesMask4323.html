Глобальный контекст (Global context)

ПолучитьМаскуВсеФайлыСервера (GetServerAllFilesMask)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМаскуВсеФайлыСервера()

Возвращаемое значение:

Тип: Строка.
Маска возвращается в виде строки. Например: "*.*".

Описание:

Получает файловую маску, принятую для выбора всех файлов в операционной системе, в которой работает сервер 1С:Предприятия, обрабатывающий вызов.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вызывается исключение, если код выполняется на клиенте в клиент-серверном варианте.

Использование в версии:

Доступен, начиная с версии 8.3.3.
