СерииДиаграммы.Удалить (ChartSeriesCollection.Delete)

СерииДиаграммы (ChartSeriesCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Если индекс больше текущего количества серий или меньше нуля, то метод не выполняет никаких действий.

Описание:

Удаляет серию по индексу. Все серии с индексом больше переданного сдвигаются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
