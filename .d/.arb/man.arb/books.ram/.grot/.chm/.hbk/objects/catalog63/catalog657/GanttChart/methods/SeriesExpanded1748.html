ДиаграммаГанта.РазвернутаСерия (GanttChart.SeriesExpanded)

ДиаграммаГанта (GanttChart)

РазвернутаСерия (SeriesExpanded)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутаСерия(<Серия>)

Параметры:

<Серия> (обязательный)

Тип: СерияДиаграммыГанта.
Серия диаграммы Ганта.

Возвращаемое значение:

Тип: Булево.
Истина - серия развернута; Ложь - в противном случае.

Описание:

Определяет, развернута ли серия диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
