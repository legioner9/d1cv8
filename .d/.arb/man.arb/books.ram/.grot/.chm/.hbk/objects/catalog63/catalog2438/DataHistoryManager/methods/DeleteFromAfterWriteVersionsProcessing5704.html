МенеджерИсторииДанных.УдалитьИзОбработкиПослеЗаписиВерсий (DataHistoryManager.DeleteFromAfterWriteVersionsProcessing)

МенеджерИсторииДанных (DataHistoryManager)

УдалитьИзОбработкиПослеЗаписиВерсий (DeleteFromAfterWriteVersionsProcessing)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: Удаление из обработки записей версий указанных объектов метаданных

Синтаксис:

УдалитьИзОбработкиПослеЗаписиВерсий(<ВключитьМетаданные>, <ИсключитьМетаданные>, <Дата>)

Параметры:

<ВключитьМетаданные> (необязательный)

Тип: Неопределено, Массив.
Объекты метаданных, по данным которых выполняется удаление из обработки записей версий.
Если значение параметра Неопределено или пустой массив, то выполняется удаление из обработки всех записей версий, за исключением указанных в параметре <ИсключитьМетаданные>.
Значение по умолчанию: Неопределено.

<ИсключитьМетаданные> (необязательный)

Тип: Неопределено, Массив.
Объекты метаданных, по данным которых не выполняется удаление из обработки о записи версий.
Значение по умолчанию: Неопределено.

<Дата> (необязательный)

Тип: Дата.
Дата. Метод удаляет из обработки записи все версии указанных объектов метаданных, созданные ранее этой даты.

Описание варианта метода:

Если метод вызывается из сеанса, в котором разделитель не используется:
  • Удаляются версии данных из всех областей данных.
  • Требуется право "Администрирование".
Если метод вызывается из сеанса, в котором разделитель используется:
  • Удаляются версии данных текущей области.
  • Требуется право "Администрирование данных".
  • При определении области данных учитываются все независимые разделители конфигурации.

Вариант синтаксиса: Удаление из обработки записей версий указанных данных

Синтаксис:

УдалитьИзОбработкиПослеЗаписиВерсий(<Данные>, <НомерВерсии>)

Параметры:

<Данные> (обязательный)

Тип: СправочникСсылка.<Имя справочника>, ДокументСсылка.<Имя документа>, РегистрСведенийКлючЗаписи.<Имя регистра сведений>, БизнесПроцессСсылка.<Имя бизнес-процесса>, ЗадачаСсылка.<Имя задачи>, ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>, ПланСчетовСсылка.<Имя плана счетов>, ПланОбменаСсылка.<Имя плана обмена>, ПланВидовРасчетаСсылка.<Имя плана видов расчета>, КонстантаКлючЗначения.<Имя константы>.
Данные, по которым создана версия. Значение соответствует значению колонки "Данные" таблицы значений, возвращаемой методом ВыбратьВерсии.

<НомерВерсии> (необязательный)

Тип: Число.
Номер созданной версии. Значение аналогично значению в колонке "Данные" таблицы значений, возвращаемой методов ВыбратьВерсии.
Если номер не установлен, то из обработки удаляются все версии по указанным данным.

Описание:

Удаляет версии из обработки после записи версий.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
