СервисИнтеграцииМенеджер.<Имя сервиса интеграции> (IntegrationServiceManager.<Integration service name>)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьНастройки()

Возвращаемое значение:

Тип: НастройкиСервисаИнтеграции.
Если настройки в режиме "1С:Предприятие" не выполнялись, то возвращает новый объект НастройкиСервисаИнтеграции.

Описание:

При получении настроек выполняются следующие действия:

  • Если в сервисе интеграции определены разделители:
    • Генерируется исключение, если выполняется одно из перечисленных условий:
      • В сеансе используется разделитель, который не разделяет сервис интеграции.
      • В сеансе не используется разделитель, который разделяет сервис интеграции.
    • Возвращаются настройки сервиса интеграции для области данных.
  • Если сервисе интеграции не определены разделители:
    • Генерируется исключение, если в сеансе используется любой разделитель.
    • Возвращаются настройки сервиса интеграции для информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
