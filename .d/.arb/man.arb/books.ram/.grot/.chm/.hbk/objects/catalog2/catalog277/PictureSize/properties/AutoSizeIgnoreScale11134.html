РазмерКартинки (PictureSize)

АвтоРазмерБезУчетаМасштаба (AutoSizeIgnoreScale)

Доступен, начиная с версии 8.3.10.

Описание:

Если предоставленного для картинки места достаточно, используется режим РеальныйРазмер, в противном случае используется режим Пропорционально.
При определении размера картинки не учитывается масштаб формы.

Использование в версии:

Доступен, начиная с версии 8.3.10.
