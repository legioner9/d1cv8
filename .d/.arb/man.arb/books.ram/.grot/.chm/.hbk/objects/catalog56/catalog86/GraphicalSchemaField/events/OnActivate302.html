ПолеГрафическойСхемы.ПриАктивизации (GraphicalSchemaField.OnActivate)

ПолеГрафическойСхемы (GraphicalSchemaField)

ПриАктивизации (OnActivate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриАктивизации()

Описание:

Возникает при смене текущего элемента графической схемы с помощью указателя мыши или нажатии клавиши TAB.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
