ПараметрыЧтенияXML.На основе набора параметров

ПараметрыЧтенияXML (XMLReaderSettings)

На основе набора параметров

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ПараметрыЧтенияXML(<Версия>, <Язык>, <ПробельныеСимволы>, <ТипПроверкиПравильности>, <ИгнорироватьОбъявлениеXML>, <ИгнорироватьТипДокумента>, <ИгнорироватьИнструкцииОбработки>, <ИгнорироватьКомментарии>, <ИгнорироватьПробельныеСимволы>, <СекцииCDATAКакТекст>, <ИспользоватьИгнорируемыеПробельныеСимволы>)

Параметры:

<Версия> (необязательный)

Тип: Строка.
Версия спецификации XML, используемой при чтении.
Может принимать значения: "1.0" или "1.1".
Значение по умолчанию: 1.0.

<Язык> (необязательный)

Тип: Строка.
Язык документа XML - начальное значение атрибута xml:lang для документа XML.
Значение данного свойства является начальным значением свойства Язык объекта ПараметрыЧтенияXML.

<ПробельныеСимволы> (необязательный)

Тип: ПробельныеСимволыXML.
Значение по умолчанию атрибута xml:space, используемое при чтении XML.
Влияет на свойство ПробельныеСимволы объекта ПараметрыЧтенияXML.
Значение по умолчанию: ПоУмолчанию.

<ТипПроверкиПравильности> (необязательный)

Тип: ТипПроверкиXML.
Тип проверки данных XML на правильность при чтении.
Значение по умолчанию: НетПроверки.

<ИгнорироватьОбъявлениеXML> (необязательный)

Тип: Булево.
Признак игнорирования объявления XML при чтении.
Значение по умолчанию: Истина.

<ИгнорироватьТипДокумента> (необязательный)

Тип: Булево.
Признак игнорирования определения типа документа (DTD).
Значение по умолчанию: Истина.

<ИгнорироватьИнструкцииОбработки> (необязательный)

Тип: Булево.
Признак игнорирования инструкций обработки.
Значение по умолчанию: Ложь.

<ИгнорироватьКомментарии> (необязательный)

Тип: Булево.
Признак игнорирования комментариев.
Значение по умолчанию: Ложь.

<ИгнорироватьПробельныеСимволы> (необязательный)

Тип: Булево.
Признак игнорирования пробельных символов при чтении.
Значение по умолчанию: Истина.

<СекцииCDATAКакТекст> (необязательный)

Тип: Булево.
Признак интерпретации секций CDATA как текста при чтении.
Значение по умолчанию: Ложь.

<ИспользоватьИгнорируемыеПробельныеСимволы> (необязательный)

Тип: Булево.
Определяет использование узлов ПробельныеСимволы при чтении XML.
Значение по умолчанию: Ложь.

Описание:

Создает экземпляр объекта ПараметрыЧтенияXML.

Использование в версии:

Доступен, начиная с версии 8.1.
