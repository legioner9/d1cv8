ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

РазборПоКопиям (Collate)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.
Указывает на необходимость разобрать документы по копиям при печати. Неопределено - разбор по копиям определяется родительским пакетом. Если родительского пакета нет, то считается Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
