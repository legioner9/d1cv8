ОписаниеПомещенногоФайла.Адрес (StoredFileDescription.Address)

ОписаниеПомещенногоФайла (StoredFileDescription)

Адрес (Address)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.
Адрес файла во временном хранилище.
Если файл не был помещен (параметр <Отказ> имеет значение Истина), то данное свойство имеет значение Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
