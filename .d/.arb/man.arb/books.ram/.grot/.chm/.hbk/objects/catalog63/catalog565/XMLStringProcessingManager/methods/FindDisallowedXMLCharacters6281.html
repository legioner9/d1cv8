МенеджерОбработкиСтрокиXML.НайтиНедопустимыеСимволыXML (XMLStringProcessingManager.FindDisallowedXMLCharacters)

МенеджерОбработкиСтрокиXML (XMLStringProcessingManager)

НайтиНедопустимыеСимволыXML (FindDisallowedXMLCharacters)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НайтиНедопустимыеСимволыXML()

Возвращаемое значение:

Тип: Число.
В случае обнаружения некорректного символа будет возвращена его позиция в строке, если же подобного символа не найдено – возвращаемое значение 0.

Описание:

Проверяет символы указанной строки на соответствие рекомендации XML 1.0 и 1.1. Проверка строки начинается с символа в позиции, указанной в параметре <ПозицияНачала>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерОбработкиСтрокиXML, метод УдалитьНедопустимыеСимволыXML

Использование в версии:

Доступен, начиная с версии 8.3.20.
