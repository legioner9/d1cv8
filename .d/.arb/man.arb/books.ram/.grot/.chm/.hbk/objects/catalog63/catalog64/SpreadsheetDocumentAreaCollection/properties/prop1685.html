КоллекцияОбластейТабличногоДокумента (SpreadsheetDocumentAreaCollection)

<Имя области> (<Area name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ОбластьЯчеекТабличногоДокумента, РисунокТабличногоДокумента.
Набор свойств содержит области и рисунки табличного документа. Доступ к области или рисунку осуществляется по имени. Имена свойств совпадают с именами областей или рисунков табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
