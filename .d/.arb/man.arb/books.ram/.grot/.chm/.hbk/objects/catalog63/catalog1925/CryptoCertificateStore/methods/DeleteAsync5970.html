ХранилищеСертификатовКриптографии (CryptoCertificateStore)

УдалитьАсинх (DeleteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УдалитьАсинх(<Сертификат>)

Параметры:

<Сертификат> (обязательный)

Тип: СертификатКриптографии, КонтейнерКлючейКриптографии.
Сертификат, который нужно удалить из хранилища.
На мобильной платформе вместо сертификата может быть использован контейнер ключей криптографии.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Удаляет сертификат (или контейнер ключей криптографии) из хранилища.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе под ОС Windows в хранилище с типом "Персональные сертификаты" метод удаляет только сертификаты и контейнеры ключей криптографии с истекшим сроком действия.
