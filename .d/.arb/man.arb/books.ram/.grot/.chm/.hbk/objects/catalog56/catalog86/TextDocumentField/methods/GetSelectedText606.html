ПолучитьВыделенныйТекст (GetSelectedText)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.2 и последующими).

Синтаксис:

ПолучитьВыделенныйТекст()

Возвращаемое значение:

Тип: Строка.
Выделенный текст в виде строки.

Описание:

Получает выделенный текст поля текстового документа.

Доступность:

Толстый клиент.

Пример:

ВыделенныйТекст = ЭлементыФормы.ПолеТекста.ПолучитьВыделенныйТекст();

Использование в версии:

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.2 и последующими).

Не рекомендуется использовать, начиная с версии 8.2.
