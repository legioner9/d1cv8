РегистрыБухгалтерииМенеджер (AccountingRegistersManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров бухгалтерии.
Возможно обращение к менеджеру регистра бухгалтерии посредством оператора [...]. В качестве аргумента передается имя регистра бухгалтерии, как оно задано в конфигурации.

Свойства:

<Имя регистра бухгалтерии> (<Accounting register name>)

Описание:

Доступ к объекту осуществляется через свойство глобального контекста РегистрыБухгалтерии. Используется для доступа к объектам, управляющим регистрами бухгалтерии.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

Пример:

ОсновнойРегистр = РегистрыБухгалтерии.Основной;
Форма = ОсновнойРегистр.ПолучитьФорму("ФормаСписка");

См. также:

Глобальный контекст, свойство РегистрыБухгалтерии

Использование в версии:

Доступен, начиная с версии 8.0.
