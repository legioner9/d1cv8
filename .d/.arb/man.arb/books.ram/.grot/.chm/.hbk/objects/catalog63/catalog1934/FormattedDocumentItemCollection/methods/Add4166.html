КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Добавить(<Значение>, <ТипЭлемента>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.
Элемент, добавляемый в коллекцию.
В зависимости от параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.
Если значение параметра ТекстФорматированногоДокумента, то в коллекцию будет добавлено строковое представление переданного значения.
Если значение параметра КартинкаФорматированногоДокумента, то тип параметра <Значение> должен быть Картинка, ДвоичныеДанные или Строка, содержащая URL картинки. В противном случае вызывается исключение.
Если значение параметра ПереводСтрокиФорматированногоДокумента, то значение параметра игнорируется и в коллекцию добавляется перевод строки.
Если значение параметров Неопределено, а также коллекция элементов форматированного документа содержит только параграфы, то в коллекцию добавляется пустой параграф.
Значение по умолчанию: Неопределено.

<ТипЭлемента> (необязательный)

Тип: Тип.
Тип элемента, который нужно добавить в коллекцию. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.
Элементами коллекции могут быть:
1) ТекстФорматированногоДокумента
2) КартинкаФорматированногоДокумента
3) ПереводСтрокиФорматированногоДокумента
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.
Значение по умолчанию: ТекстФорматированногоДокумента.

Возвращаемое значение:

Тип: ПараграфФорматированногоДокумента, ТекстФорматированногоДокумента, ПереводСтрокиФорматированногоДокумента, КартинкаФорматированногоДокумента.
Возвращает добавленный элемент.

Описание:

Добавляет элемент в конец коллекции элементов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Добавление картинки доступно только на сервере.
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.21.
