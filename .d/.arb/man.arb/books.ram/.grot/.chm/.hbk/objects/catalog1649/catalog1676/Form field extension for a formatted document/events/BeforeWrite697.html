Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: Булево.
Истина, если выполняется команда "Сохранить как".

<Отказ>

Тип: Булево.
Установка данного параметра в Истина позволяет отказаться от записи.
Значение по умолчанию: Ложь.

Описание:

Возникает перед интерактивной записью форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
