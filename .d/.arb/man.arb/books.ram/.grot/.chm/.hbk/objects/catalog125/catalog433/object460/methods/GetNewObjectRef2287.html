ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>.

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом УстановитьСсылкуНового значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

ПланВидовРасчетаМенеджер.<Имя плана видов расчета>, метод ПолучитьСсылку

Использование в версии:

Доступен, начиная с версии 8.0.
