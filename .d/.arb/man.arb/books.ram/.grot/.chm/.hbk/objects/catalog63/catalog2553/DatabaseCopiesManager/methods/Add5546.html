МенеджерКопийБазыДанных.Добавить (DatabaseCopiesManager.Add)

МенеджерКопийБазыДанных (DatabaseCopiesManager)

Добавить (Add)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Вариант синтаксиса: По имени копии

Синтаксис:

Добавить(<ИмяКопии>)

Параметры:

<ИмяКопии> (необязательный)

Тип: Строка.
Имя копии. Используется для идентификации копии.
Имя должно состоять из одного слова, начинаться с буквы и не содержать специальных символов, кроме "_".

Описание варианта метода:

Добавляет копию базы данных, использующую встроенный Дата акселератор по имени копии.

Вариант синтаксиса: По имени копии и параметрам базы данных

Синтаксис:

Добавить(<ИмяКопии>, <ТипРепликации>, <ТипСУБД>, <СерверБазыДанных>, <БазаДанных>, <ПользовательБазыДанных>, <ПарольПользователяБазыДанных>, <СоздатьБазуДанных>)

Параметры:

<ИмяКопии> (обязательный)

Тип: Строка.
Имя копии. Используется для идентификации копии.
Имя должно состоять из одного слова, начинаться с буквы и не содержать специальных символов, кроме "_".

<ТипРепликации> (обязательный)

Тип: ТипРепликацииКопииБазыДанных.
Тип репликации копии.

<ТипСУБД> (обязательный)

Тип: ТипСУБДКопииБазыДанных.
Тип СУБД копии базы данных.

<СерверБазыДанных> (обязательный)

Тип: Строка.
Адрес сервера базы данных копии. Может содержать значение порта после имени (или ip-адреса) хоста и двоеточия.

<БазаДанных> (обязательный)

Тип: Строка.
Имя базы данных копии.

<ПользовательБазыДанных> (необязательный)

Тип: Строка.
Пользователь базы данных копии.

<ПарольПользователяБазыДанных> (необязательный)

Тип: Строка.
Пароль пользователя базы данных копии.

<СоздатьБазуДанных> (необязательный)

Тип: Булево.
Создавать базу данных копии, если она отсутствует на сервере базы данных копии.
Значение по умолчанию: Истина.

Описание варианта метода:

Добавляет копию базы данных, использующую Дата акселератор по имени копии и параметрам базы данных.

Возвращаемое значение:

Тип: МенеджерКопииБазыДанных.

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
