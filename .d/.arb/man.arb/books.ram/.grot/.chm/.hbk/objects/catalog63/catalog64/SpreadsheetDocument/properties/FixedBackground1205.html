ТабличныйДокумент.ФиксированныйФон (SpreadsheetDocument.FixedBackground)

ТабличныйДокумент (SpreadsheetDocument)

ФиксированныйФон (FixedBackground)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак зафиксированности фона табличного документа. Зафиксированный фон остается неподвижным при прокрутке таблицы.
Истина - фон зафиксирован.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
