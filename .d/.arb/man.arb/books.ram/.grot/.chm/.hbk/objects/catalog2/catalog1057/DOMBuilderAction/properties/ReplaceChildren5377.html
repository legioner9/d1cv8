ДействиеПостроителяDOM.ЗаменитьДочерние (DOMBuilderAction.ReplaceChildren)

ДействиеПостроителяDOM (DOMBuilderAction)

ЗаменитьДочерние (ReplaceChildren)

Доступен, начиная с версии 8.1.

Описание:

Заменить дочерние узлы. Результат чтения заменяет дочерние узлы контекста документа. В качестве узла контекста могут выступать: Элемент, Документ или ФрагментДокумента.

Использование в версии:

Доступен, начиная с версии 8.1.
