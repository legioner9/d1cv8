Расширение поля формы для поля планировщика (Form field extension for a planner)

РазвернутЭлементИзмерения (DimensionItemExpanded)

Доступен, начиная с версии 8.3.17.

Синтаксис:

РазвернутЭлементИзмерения(<ЗначенияИзмерений>)

Параметры:

<ЗначенияИзмерений> (обязательный)

Тип: ФиксированноеСоответствие.
Фиксированное соответствие, однозначно определяющее элемент измерения, состояние которого нужно получить. В качестве ключа выступает Значение измерения. В качестве значения - Значение элемента измерения.

Возвращаемое значение:

Тип: Булево.
Возвращает значение Истина, если элемент измерения развернут, и Ложь в противном случае.

Описание:

Определяет, развернут ли элемент измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
