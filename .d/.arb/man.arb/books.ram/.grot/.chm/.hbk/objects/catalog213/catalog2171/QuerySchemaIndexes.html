ИндексыСхемыЗапроса (QuerySchemaIndexes)

Доступен, начиная с версии 8.3.25.

Элементы коллекции:

ИндексСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются таблицы для изменения.
Возможно обращение к таблице для изменения посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Переместить (MoveTo)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Представляет собой коллекцию описаний индексов схемы запроса (объекты типа ИндексСхемыЗапроса).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
