Цвет (Color)

На основе цветовых составляющих

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Цвет(<Красный>, <Зеленый>, <Синий>)

Параметры:

<Красный> (обязательный)

Тип: Число.
Значение красной составляющей цвета.

<Зеленый> (обязательный)

Тип: Число.
Значение зеленой составляющей цвета.

<Синий> (обязательный)

Тип: Число.
Значение синей составляющей цвета.

Описание:

Формирует абсолютный цвет из трех цветовых составляющих.

Пример:

Для Индекс=1 По Элемент.ШиринаТаблицы Цикл
    Элемент.Область(1,Индекс).ЦветФона=Новый Цвет(255,255,255);
КонецЦикла;

КрасныйЦвет = Новый Цвет (255,0,0);
ЧерныйЦвет = Новый Цвет(0,0,0);

Использование в версии:

Доступен, начиная с версии 8.0.
