ОписаниеКомандыВходящегоЗапросаПоделиться (IncomingShareRequestCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: СтандартнаяКомандаВходящегоЗапросаПоделиться, ОписаниеОповещения, Неопределено.
Содержит описание команды:
  • СтандартнаяКомандаВходящегоЗапросаПоделиться - стандартная команда;
  • ОписаниеОповещения - процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
    • <Данные> - тип ДанныеЗапросаПоделиться,
    • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения;
    ;
  • Неопределено - разделитель;
  • Любой другой тип - пункт меню не отображается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
