ИнтернетСоединение (InternetConnection)

Доступен, начиная с версии 8.0.

Свойства:

ИмяСоединения (ConnectionName)
Установлено (IsConnected)

Методы:

Разорвать (Disconnect)
Установить (Connect)

Конструкторы:

По умолчанию

Описание:

Предназначен для доступа к Интернет-соединениям. Доступен только на компьютерах с установленным Internet Explorer версии 5.0 или выше.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
