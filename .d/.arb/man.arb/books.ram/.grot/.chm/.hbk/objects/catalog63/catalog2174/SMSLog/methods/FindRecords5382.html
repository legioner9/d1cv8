ЖурналSMS.НайтиЗаписи (SMSLog.FindRecords)

ЖурналSMS (SMSLog)

НайтиЗаписи (FindRecords)

Доступен, начиная с версии 8.3.11.

Синтаксис:

НайтиЗаписи(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: ОтборКомпоновкиДанных.
Задает условия поиска.
В качестве условий отбора могут быть указаны следующие поля:
  • НомераТелефонов – тип массив строк, Строка. Допускаются виды сравнения:
    • Для массива: ВСписке, НеВСписке.
    • Для строки: Равно, НеРавно, Содержит, НеСодержит.
  • ДатаПолучения – тип Дата. Допускаются виды сравнения: Больше, Меньше, Равно, БольшеИлиРавно, МеньшеИлиРавно, НеРавно.
  • ДатаОтправки – тип Дата. Допускаются виды сравнения: Больше, Меньше, Равно, БольшеИлиРавно, МеньшеИлиРавно, НеРавно.
  • Текст – тип Строка. Допускаются виды сравнения: Равно, НеРавно, Содержит, НеСодержит.
  • ТипСообщения – тип значение перечисления ТипSMSСредствТелефонии. Допускаются виды сравнения: Равно, НеРавно.
  • Прочитано – тип Булево. Допускаются виды сравнения: Равно, НеРавно.

Возвращаемое значение:

Тип: Массив.
Массив элементов типа ЗаписьЖурналаSMS.

Описание:

Выполняет поиск записей в журнале SMS-сообщений, отвечающих заданному отбору.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.11.
