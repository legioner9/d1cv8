Значения

ВремяНачала (BeginTime)
ВремяНачалаИКонца (BeginAndEndTime)

Описание:

Содержит варианты отображения времени элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. PlannerItemsTimeRepresentation.

Использование в версии:

Доступен, начиная с версии 8.3.6.
