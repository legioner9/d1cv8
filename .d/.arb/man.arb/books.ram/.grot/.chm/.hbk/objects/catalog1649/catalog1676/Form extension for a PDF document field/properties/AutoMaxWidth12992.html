Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

АвтоМаксимальнаяШирина (AutoMaxWidth)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет использование автоматического ограничения максимальной ширины элемента.
Истина - используется автоматическое ограничение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.21.
