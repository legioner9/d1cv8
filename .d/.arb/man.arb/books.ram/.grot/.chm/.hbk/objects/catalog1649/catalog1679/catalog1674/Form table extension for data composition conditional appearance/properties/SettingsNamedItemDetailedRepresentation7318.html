Расширение таблицы формы для условного оформления компоновки данных (Form table extension for data composition conditional appearance)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Свойство появляется у таблицы, если она связана с данными типа УсловноеОформлениеКомпоновкиДанных.
Ложь - именованные элементы (свойство Отображение не пустое) отображаются в таблице одной строкой с одной ячейкой, в которой выводится представление.
В противном случае, представление редактируется в отдельной колонке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
