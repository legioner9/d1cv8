МенеджерПанелиЗадачОС.УстановитьКнопки (OSTaskbarManager.SetButtons)

МенеджерПанелиЗадачОС (OSTaskbarManager)

УстановитьКнопки (SetButtons)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УстановитьКнопки(<Кнопки>)

Параметры:

<Кнопки> (обязательный)

Тип: Массив.
Массив должен содержать объекты типа Структура со следующими
ключами:
  • Картинка (Picture) – тип Картинка. Картинка на кнопке. Рекомендуется использовать картинку размера 16х16 пикселей в формате "ico", а также желательно иметь в наличии картинки размеров 20х20 и 24х24 px.
  • Заголовок (Title) – тип Строка. Всплывающая подсказка для кнопки.
  • Действие (Action) – тип ОписаниеОповещения. Описание процедуры, которая будет вызвана после нажатия на кнопку под областью предпросмотра со следующим параметром:
    • <ДополнительныеПараметры> – значение, которое было указано при создании объекта ОписаниеОповещения .

Описание:

Добавляет кнопки под областью предпросмотра в панели задач.
Работает только под Windows 7 и выше. Под остальными ОС не производит никаких действий.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

При повторном вызове замещает установленные ранее кнопки.

Пример:

Кнопки = Новый Массив();
    
Кнопка1 = Новый Структура();
Кнопка1.Вставить("Картинка", БиблиотекаКартинок.Иконка1);
Кнопка1.Вставить("Заголовок", "Кнопка1");
Оп = Новый ОписаниеОповещения("Тест", ТестОповещений, "сообщение1");
Кнопка1.Вставить("Действие", Оп);
    
Кнопка2 = Новый Структура();
Кнопка2.Вставить("Картинка", БиблиотекаКартинок.Иконка2);
Кнопка2.Вставить("Заголовок", "Кнопка2");
Оп2 = Новый ОписаниеОповещения("Тест", ТестОповещений, "сообщение2");
Кнопка2.Вставить("Действие", Оп2);
    
Кнопки.Добавить(Кнопка1);
Кнопки.Добавить(Кнопка2);
    
ПанельЗадачОС.УстановитьКнопки(Кнопки);

Использование в версии:

Доступен, начиная с версии 8.3.8.
