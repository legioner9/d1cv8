ЧтениеXML.ЛокальноеИмя (XMLReader.LocalName)

ЧтениеXML (XMLReader)

ЛокальноеИмя (LocalName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.
Содержит локальное имя для текущего узла XML. Имя зависит от типа текущего узла:
  • Атрибут - локальное имя атрибута;
  • ОпределениеТипаДокумента - имя типа документа;
  • НачалоЭлемента - локальное имя элемента;
  • КонецЭлемента - локальное имя элемента;
  • СсылкаНаСущность - имя ссылки на сущность;
  • ИнструкцияОбработки - адрес инструкции обработки;
  • ОбъявлениеXML - строка xml.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для узлов, для которых локальное имя не определено, – пустая строка.

Использование в версии:

Доступен, начиная с версии 8.1.
