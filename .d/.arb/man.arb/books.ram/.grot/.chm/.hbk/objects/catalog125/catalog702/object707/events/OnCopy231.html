БизнесПроцессОбъект.<Имя бизнес-процесса>.ПриКопировании (BusinessProcessObject.<Business process name>.OnCopy)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПриКопировании (OnCopy)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриКопировании(<ОбъектКопирования>)

Параметры:

<ОбъектКопирования>

Тип: БизнесПроцессОбъект.<Имя бизнес-процесса>.
Исходный бизнес-процесс, который является источником копирования.

Описание:

Возникает при создании бизнес-процесса копированием.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

См. также:

БизнесПроцессОбъект.<Имя бизнес-процесса>, метод Скопировать

Использование в версии:

Доступен, начиная с версии 8.0.
