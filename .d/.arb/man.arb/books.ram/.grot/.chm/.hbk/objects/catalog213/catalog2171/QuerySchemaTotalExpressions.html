Элементы коллекции:

ВыражениеИтогаСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения итогов.
Возможно обращение к выражению итога посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Описывает объектную модель выражений итогов схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
