ВыделенныеОбластиТабличногоДокумента.Очистить (SpreadsheetDocumentSelectedAreas.Clear)

ВыделенныеОбластиТабличногоДокумента (SpreadsheetDocumentSelectedAreas)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все области из коллекции выделенных областей.
После очистки выделенной областью становится ячейка R1:C1.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
