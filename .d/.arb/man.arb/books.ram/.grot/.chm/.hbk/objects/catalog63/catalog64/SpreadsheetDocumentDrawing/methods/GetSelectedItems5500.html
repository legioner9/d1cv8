РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ПолучитьВыделенныеЭлементы (GetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьВыделенныеЭлементы()

Возвращаемое значение:

Тип: Массив.
Элементами массива являются значения следующих типов: ТочкаДиаграммы, СерияДиаграммы, ЗначениеДиаграммы.

Описание:

Возвращает массив выделенных элементов для рисунка типа Диаграмма.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
