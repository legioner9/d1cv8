ЭлементЛегендыГеографическойСхемы (GeographicalSchemaLegendItem)

Доступен, начиная с версии 8.1.

Свойства:

Картинка (Picture)
ОтображатьШкалу (ShowScale)
Серия (Series)
ТекстОбразца (SampleText)
ТекстПодписи (LabelText)
ЦветОбразца (SampleColor)
ШрифтОбразца (SampleFont)

Описание:

Используется для доступа к элементу легенды географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

// Создание в слое географической схемы серии для отображения данных

СерияСуммаОборотаПоРегионам = СлойРегионов.Серии.Добавить();
СерияСуммаОборотаПоРегионам.Текст = "Сумма оборота";
ТипОтображения = ТипОтображенияСерииСлояГеографическойСхемы.Гистограмма;
СерияСуммаОборотаПоРегионам.ТипОтображения = ТипОтображения;

// Добавление в легенду

ЭлементЛегенды = Схема.ОбластьЛегенды.Элементы.Добавить();
ЭлементЛегенды.Серия = СерияСуммаОборотаПоРегионам;
ЭлементЛегенды.ТекстОбразца = "Сумма по регионам";
