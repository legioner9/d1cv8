Сдвинуть (Move)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Сдвинуть(<Ключ>, <Смещение>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.
Ключ сдвигаемого элемента в коллекции.

<Смещение> (обязательный)

Тип: Число.
Количество элементов, на которое необходимо сдвинуть элемент. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание:

Сдвигает элемент коллекции с соответствующим ключом на заданное смещение.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
