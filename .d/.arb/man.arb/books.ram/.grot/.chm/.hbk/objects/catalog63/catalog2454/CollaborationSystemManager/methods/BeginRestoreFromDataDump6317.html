МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьВосстановлениеИзВыгрузкиДанных (BeginRestoreFromDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

НачатьВосстановлениеИзВыгрузкиДанных(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.
Идентификатор выгрузки данных.

Описание:

Начинает восстановление данных системы взаимодействия из выгрузки данных с указанным идентификатором.
Вызов метода возвращает управление при начале восстановления данных из выгрузки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерСистемыВзаимодействия, метод НачатьВосстановлениеИзВыгрузкиДанныхАсинх

Использование в версии:

Доступен, начиная с версии 8.3.22.
