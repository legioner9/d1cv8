ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

ВключатьРасширенияКонфигурации (IncludeConfigurationExtensions)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если это свойство установлено в значение Истина, то в распределенной информационной базе, организованной данным планом обмена, будут передаваться расширения конфигурации, у которых свойство ИспользуетсяВРаспределеннойИнформационнойБазе установлено в значение Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

РасширениеКонфигурации, свойство ГлавныйУзел

Использование в версии:

Доступен, начиная с версии 8.3.12.
