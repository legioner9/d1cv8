РезультатЧтенияДанных (ReadDataResult)

ПолучитьБуферДвоичныхДанныхАсинх (GetBinaryDataBufferAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьБуферДвоичныхДанныхАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа БуферДвоичныхДанных. Необходимо учитывать, что при этом данные будут полностью загружены в оперативную память. Если требуется избежать загрузки оперативной памяти, следует использовать другие методы получения двоичных данных.

Описание:

Асинхронная функция. Получает обещание, содержащее буфер двоичных данных.

Доступность:

Тонкий клиент, толстый клиент.
