ПланСчетовВидыСубконто.<Имя плана счетов> (ChartOfAccountsExtDimensionTypes.<Chart of accounts name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: ПланСчетовВидыСубконтоСтрока.<Имя плана счетов>.
Строка табличной части, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: Число.
Индекс указанной строки табличной части. Если не найдено, то возвращается -1.

Описание:

Получает индекс строки в табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

ИндексТекСтроки = Состав.Индекс(ЭлементыФормы.Состав.ТекущаяСтрока);

Использование в версии:

Доступен, начиная с версии 8.0.
