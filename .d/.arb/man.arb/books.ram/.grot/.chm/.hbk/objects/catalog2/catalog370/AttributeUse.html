ИспользованиеРеквизита (AttributeUse)

Доступен, начиная с версии 8.0.

Значения

ДляГруппы (ForFolder)
ДляГруппыИЭлемента (ForFolderAndItem)
ДляЭлемента (ForItem)

Описание:

Определяет набор вариантов использования реквизита справочника для групп и элементов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ПеречислимыеСвойстваОбъектовМетаданных, свойство ИспользованиеРеквизита
