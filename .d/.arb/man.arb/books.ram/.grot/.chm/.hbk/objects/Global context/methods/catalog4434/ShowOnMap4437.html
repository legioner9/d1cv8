Глобальный контекст (Global context)

ПоказатьНаКарте (ShowOnMap)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьНаКарте(<Местоположение>)

Параметры:

<Местоположение> (обязательный)

Тип: ГеографическиеКоординаты, СписокЗначений.
Местоположение, которое нужно показать.
Для отображения нескольких точек используется СписокЗначений, в котором значение элемента списка - это географическая координата, а представление - текст, отображаемый в точке (на булавке).

Описание:

Отображает на карте переданные координаты.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяОтображениеКарты.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

Глобальный контекст, метод ПоддерживаетсяОтображениеКарты

Использование в версии:

Доступен, начиная с версии 8.3.3.
