РежимРазрешенияРедактированияЭлементаПланировщика (PlannerItemEnableEditMode)

Доступен, начиная с версии 8.3.18.

Значения

ЗапретитьПеретаскиваниеИРастягивание (DisableDragAndStretch)
ЗапретитьРастягивание (DisableStretch)
ЗапретитьРедактирование (DisableEdit)
РазрешитьРедактирование (EnableEdit)

Описание:

Определяет варианты режима редактирования элемента планировщика. Значение свойства РежимРазрешенияРедактирования не влияет на копирование элемента планировщика и на перетаскивание элемента планировщика вовне и извне поля планировщика. При копировании элемента планировщика данное свойство заполняется значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
