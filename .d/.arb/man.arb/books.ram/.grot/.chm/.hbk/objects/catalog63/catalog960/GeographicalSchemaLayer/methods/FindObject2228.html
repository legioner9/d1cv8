СлойГеографическойСхемы (GeographicalSchemaLayer)

НайтиОбъект (FindObject)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиОбъект(<Серия>, <Значение>)

Параметры:

<Серия> (обязательный)

Тип: СерияДанныхСлояГеографическойСхемы.
Серия, по которой ищется объект.

<Значение> (обязательный)

Тип: Произвольный.
Значение, по которому ищется объект.

Возвращаемое значение:

Тип: ТочечныйОбъектГеографическойСхемы, МноготочечныйОбъектГеографическойСхемы, ПолилинейныйОбъектГеографическойСхемы, ПолигональныйОбъектГеографическойСхемы, Неопределено.
Неопределено - элемент не найден.

Описание:

Осуществляет поиск объекта в слое.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
