ДокументHTML.СоздатьТекстовыйУзел (HTMLDocument.CreateTextNode)

ДокументHTML (HTMLDocument)

СоздатьТекстовыйУзел (CreateTextNode)

Доступен, начиная с версии 8.2.

Синтаксис:

СоздатьТекстовыйУзел(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Строка, представляющая собой значение текстового узла.

Возвращаемое значение:

Тип: ТекстDOM.

Описание:

Создает узел DOM - текст.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
