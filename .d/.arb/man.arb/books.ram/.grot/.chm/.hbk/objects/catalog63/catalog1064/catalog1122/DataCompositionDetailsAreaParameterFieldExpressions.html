ВыражениеПоляПараметраОбластиРасшифровкаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения.
Возможно обращение к выражению посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция выражений полей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
