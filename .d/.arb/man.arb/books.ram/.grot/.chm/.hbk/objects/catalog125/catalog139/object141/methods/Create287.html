ОбработкаМенеджер.<Имя обработки>.Создать (DataProcessorManager.<Data processor name>.Create)

ОбработкаМенеджер.<Имя обработки> (DataProcessorManager.<Data processor name>)

Создать (Create)

Доступен, начиная с версии 8.0.

Синтаксис:

Создать()

Возвращаемое значение:

Тип: ОбработкаОбъект.<Имя обработки>.

Описание:

Создает новый экземпляр обработки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

ОбработкаДокументов = Обработки.ОбработкаДокументов.Создать();

Использование в версии:

Доступен, начиная с версии 8.0.
