Соединение с агентом сервера.GetAgentAdmins (IServerAgentConnection.GetAgentAdmins)

Соединение с агентом сервера (IServerAgentConnection)

GetAgentAdmins (GetAgentAdmins)

Доступен, начиная с версии 8.1.

Синтаксис:

GetAgentAdmins()

Возвращаемое значение:

Тип: COMSafeArray.
Массив администраторов центрального сервера. Каждый администратор центрального сервера является объектом с интерфейсом Администратор.

Описание:

Получает массив администраторов центрального сервера. Для успешного выполнения метода необходима аутентификация одного из администраторов центрального сервера. Если не зарегистрирован ни один администратор центрального сервера, то перед выполнением этого метода следует выполнить метод AuthenticateAgent("", "").

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
