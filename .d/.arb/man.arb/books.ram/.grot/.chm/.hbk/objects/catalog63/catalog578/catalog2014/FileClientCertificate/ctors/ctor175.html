СертификатКлиентаФайл (FileClientCertificate)

На основании файла

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый СертификатКлиентаФайл(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.
Полное имя файла клиентского сертификата.

<Пароль> (необязательный)

Тип: Строка.
Пароль файла клиентского сертификата.

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из указанного файла.

Использование в версии:

Доступен, начиная с версии 8.3.1.
