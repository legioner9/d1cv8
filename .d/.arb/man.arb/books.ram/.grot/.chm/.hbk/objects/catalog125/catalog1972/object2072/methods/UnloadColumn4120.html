ВнешнийИсточникДанныхКубНаборЗаписей.<Имя внешнего источника>.<Имя куба> (ExternalDataSourceCubeRecordSet.<External source name>.<Cube name>)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ВыгрузитьКолонку(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.
Имя поля записей набора, значения из которого будут выгружены в массив.

Возвращаемое значение:

Тип: Массив.

Описание:

Создает массив и копирует в него значения заданного поля записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения записей выгружаются в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.3.3.
