НастройкиКомпоновкиДанных.НаличиеУсловногоОформленияУЭлемента (DataCompositionSettings.HasItemConditionalAppearance)

НастройкиКомпоновкиДанных (DataCompositionSettings)

НаличиеУсловногоОформленияУЭлемента (HasItemConditionalAppearance)

Доступен, начиная с версии 8.2.

Синтаксис:

НаличиеУсловногоОформленияУЭлемента(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: НастройкиКомпоновкиДанных, ГруппировкаКомпоновкиДанных, ТаблицаКомпоновкиДанных, ДиаграммаКомпоновкиДанных, НастройкиВложенногоОбъектаКомпоновкиДанных, ГруппировкаДиаграммыКомпоновкиДанных, ГруппировкаТаблицыКомпоновкиДанных , КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных, КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных.
Элемент структуры настроек.

Возвращаемое значение:

Тип: Булево.

Описание:

Определяет наличие собственного условного оформления у переданного элемента структуры настроек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
