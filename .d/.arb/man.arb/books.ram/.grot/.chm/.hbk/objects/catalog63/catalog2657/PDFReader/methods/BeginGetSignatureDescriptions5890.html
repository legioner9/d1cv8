ЧтениеPDF.НачатьПолучениеОписанияПодписей (PDFReader.BeginGetSignatureDescriptions)

ЧтениеPDF (PDFReader)

НачатьПолучениеОписанияПодписей (BeginGetSignatureDescriptions)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПолучениеОписанияПодписей(<ОписаниеОповещения>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ОписанияПодписей> - тип Массив. В параметре возвращается массив, содержащий объекты типа ОписаниеПодписиPDF.
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.
<ОписанияПодписей> - тип Массив. В параметре возвращается массив, содержащий объекты типа ОписаниеПодписиPDF.
<ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<МенеджерКриптографии> (обязательный)

Тип: МенеджерКриптографии.
Менеджер криптографии.

Описание:

Инициирует получение массива, содержащего описания присоединенных к PDF-документу электронных подписей. В веб-клиенте требуется установка расширения для работы с криптографией.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
