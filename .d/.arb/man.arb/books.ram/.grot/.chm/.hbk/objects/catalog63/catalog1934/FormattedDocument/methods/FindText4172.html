ФорматированныйДокумент.НайтиТекст (FormattedDocument.FindText)

ФорматированныйДокумент (FormattedDocument)

НайтиТекст (FindText)

Доступен, начиная с версии 8.3.1.

Синтаксис:

НайтиТекст(<Строка>, <Начало>, <Конец>, <ИскатьВперед>, <ИгнорироватьРегистр>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Искомый текст.

<Начало> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на начало диапазона поиска. Если не задан, поиск будет выполнен с начала форматиорванного документа.

<Конец> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на конец диапазона поиска. Если не задан, поиск будет выполнен до конца форматиорванного документа.

<ИскатьВперед> (необязательный)

Тип: Булево.
Определяет направление поиска: Истина - искать вперед; Ложь - искать назад.
Значение по умолчанию: Истина.

<ИгнорироватьРегистр> (необязательный)

Тип: Булево.
Учет регистра.
Истина - без учета регистра, Ложь - поиск с учетом регистра.
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: ОбластьФорматированногоДокумента, Неопределено.
ОбластьФорматированногоДокумента - содержит закладки начала и конца ближайшего найденного текста.
Если текст не найден, то возвращается Неопределено.

Описание:

Осуществляет поиск текста в форматированном документе.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
