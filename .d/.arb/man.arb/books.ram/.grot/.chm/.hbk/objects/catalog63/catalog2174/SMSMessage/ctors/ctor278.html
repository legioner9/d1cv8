SMSСообщение (SMSMessage)

На основании записи в журнале СМС

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый SMSСообщение(<ЗаписьЖурналаSMS>, <КопироватьВложения>)

Параметры:

<ЗаписьЖурналаSMS> (обязательный)

Запись в журнале SMS-сообщений.

<КопироватьВложения> (необязательный)

Тип: Булево.
Указывает, будут ли скопированы вложения и поля Вложения.
Истина - вложения будут скопированы.
Значение по умолчанию: Ложь.

Описание:

Создает SMS-сообщение на основе записи журнала SMS. Свойства объекта заполняются по следующим принципам:

  • Получатели - из поля НомераТелефонов копируются все цифровые номера. Имена сервисов, на которые невозможно отправить сообщения не копируются;
  • Текст - получается из поля Текст;
  • Вложения - вложения копируются из поля Вложения, если параметр <КопироватьВложения> установлен в значение Истина;
  • Поля Отправитель и ДатаПолучения не заполняются.

Использование в версии:

Доступен, начиная с версии 8.3.11.
