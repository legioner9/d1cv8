Расширение формы клиентского приложения для компоновщика настроек (Client application form extension for settings composer)

Доступен, начиная с версии 8.3.3.

Описание:

Содержит стандартные параметры формы, которые применимы для формы компоновщика настроек.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
