Профиль безопасности.ModulesNotAvailableForExtension (ISecurityProfile.ModulesNotAvailableForExtension)

Профиль безопасности (ISecurityProfile)

ModulesNotAvailableForExtension (ModulesNotAvailableForExtension)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: Строка.
Используется, когда расширение всех модулей разрешено.
Содержит список полных имен объектов конфигурации или модулей, расширение которых не разрешено, разделенных ";".
Указание полного имени объекта конфигурации запрещает расширение всех модулей объекта.
Указание полного имени модуля запрещает расширение конкретного модуля.
Примеры имен:
  • ОбщийМодуль.ОбщийМодуль1,
  • Справочник.Справочник1.Форма.ФормаЭлемента,
  • Справочник.Справочник1.МодульОбъекта,
  • Конфигурация.ДемоКонфигурация.МодульСеанса

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
