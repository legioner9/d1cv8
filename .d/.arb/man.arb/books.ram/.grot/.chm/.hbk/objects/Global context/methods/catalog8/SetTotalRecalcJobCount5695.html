Глобальный контекст (Global context)

УстановитьКоличествоЗаданийПересчетаИтогов (SetTotalRecalcJobCount)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьКоличествоЗаданийПересчетаИтогов(<КоличествоЗаданийПересчетаИтогов>)

Параметры:

<КоличествоЗаданийПересчетаИтогов> (необязательный)

Тип: Число.
Количество заданий пересчета итогов.
Значение по умолчанию: 4.

Описание:

Устанавливает количество заданий пересчета итогов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
