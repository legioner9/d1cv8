АдминистрированиеСервис (AdministrationService)

ПоддерживаетсяТолькоГлавнымМенеджером (SupportedOnlyByMainManager)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.
Содержит признак поддержки главным и дополнительным менеджером кластера:
  • Истина - если сервис поддерживается только главным менеджером кластера и не может быть перенесен в процесс дополнительного менеджера кластера.
  • Ложь - если сервис может поддерживаться как главным, так и дополнительным менеджером кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
