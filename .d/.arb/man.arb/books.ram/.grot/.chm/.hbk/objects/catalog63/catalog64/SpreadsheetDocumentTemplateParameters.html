ПараметрыМакетаТабличногоДокумента (SpreadsheetDocumentTemplateParameters)

Доступен, начиная с версии 8.0.

Свойства:

<Имя параметра> (<Parameter name>)

Методы:

Заполнить (Fill)
Количество (Count)
Получить (Get)
Установить (Set)

Описание:

Представляет собой коллекцию параметров, используемых в ячейках макета табличного документа.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ТабличныйДокумент, свойство Параметры

Доступен, начиная с версии 8.0.
