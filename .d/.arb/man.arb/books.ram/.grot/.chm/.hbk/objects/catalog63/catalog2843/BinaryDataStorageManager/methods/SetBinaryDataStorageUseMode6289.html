МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьРежимИспользованияХранилищаДвоичныхДанных (SetBinaryDataStorageUseMode)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьРежимИспользованияХранилищаДвоичныхДанных(<РежимИспользованияХранилищаДвоичныхДанных>)

Параметры:

<РежимИспользованияХранилищаДвоичныхДанных> (обязательный)

Тип: РежимИспользованияХранилищаДвоичныхДанных.
Режим использования хранилища двоичных данных.
Для созданной информационной базы использование хранилище двоичных данных выключено (установлено в Запретить).

Описание:

Устанавливает режим использования хранилища двоичных данных.
Требуется режим монопольного доступа к информационной базе.
Метод доступен только из сеанса, в котором не используется ни один разделитель.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
