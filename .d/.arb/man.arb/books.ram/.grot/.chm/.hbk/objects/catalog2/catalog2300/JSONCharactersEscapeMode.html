ЭкранированиеСимволовJSON (JSONCharactersEscapeMode)

ЭкранированиеСимволовJSON (JSONCharactersEscapeMode)

Доступен, начиная с версии 8.3.6.

Значения

Нет (None)
СимволыВнеASCII (NotASCIISymbols)
СимволыВнеBMP (SymbolsNotInBMP)

Описание:

Содержит варианты способов экранирования символов JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
