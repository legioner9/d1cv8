ОформлениеЯчейки (CellAppearance)

УстановитьКартинку (SetPicture)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьКартинку(<Картинка>)

Параметры:

<Картинка> (обязательный)

Тип: Картинка.
Картинка.

Описание:

Устанавливает свойство Картинка объекта ОформлениеЯчейки в значение, переданное в параметре <Картинка>, и устанавливает свойство ОтображатьКартинку в значение Истина.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
