ОбъектМетаданных: РегистрСведений.ДополнительнаяФормаСписка (MetadataObject: InformationRegister.AuxiliaryListForm)

ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ДополнительнаяФормаСписка (AuxiliaryListForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ОбъектМетаданных: Форма , Неопределено.
Описывает форму, которая будет вызвана системой для просмотра списка регистра сведений, в случае если основная форма не установлена или не может быть использована в конкретном варианте использования 1С:Предприятия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
