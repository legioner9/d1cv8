ОбрабатываемаяКартинка (ProcessingPicture)

УстановитьРазмер (SetSize)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьРазмер(<Ширина>, <Высота>)

Параметры:

<Ширина> (необязательный)

Тип: Число.
Устанавливаемая ширина картинки.

<Высота> (необязательный)

Тип: Число.
Высота картинки в пикселях.

Описание:

Устанавливает новые размеры картинки. Если один из них не задан, то он будет рассчитан относительно другого с сохранением пропорции картинки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
