ИнтернетПочта (InternetMail)

ПереместитьВПочтовыйЯщик (MoveToMailbox)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПереместитьВПочтовыйЯщик(<МассивИдентификаторовСообщений>, <Имя>)

Параметры:

<МассивИдентификаторовСообщений> (обязательный)

Тип: Массив.
Массив серверных идентификаторов (UIDL) сообщений.

<Имя> (обязательный)

Тип: Строка.
Имя почтового ящика IMAP. Ящик должен существовать на сервере. Имя ящика может содержать иерархию. В качестве разделителя будет использован символ, содержащийся в свойстве СимволРазделитель. В большинстве случаев в качестве разделителя используется символ "/".

Описание:

Перемещает письма из одного почтового ящика IMAP в другой.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Примечание:

Доступен только при подключении по протоколу IMAP.
Если почтовый сервер не поддерживает расширение UIDPLUS, при вызове метода будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.12.
