Глобальный контекст (Global context)

АктивноеОкно (ActiveWindow)

Доступен, начиная с версии 8.2.

Синтаксис:

АктивноеОкно()

Возвращаемое значение:

Тип: ОкноКлиентскогоПриложения, Неопределено.
В режиме совместимости Версия8_3_19 и ниже, если активна начальная страница, метод возвращает:
  • в тонком клиенте - Неопределено,
  • в веб-клиенте - объект типа ОкноКлиентскогоПриложения, у которого свойство НачальнаяСтраница имеет значение Ложь.
В режиме совместимости выше Версия8_3_19 в тонком и веб-клиенте, в интерфейсе Такси возвращает объект типа ОкноКлиентскогоПриложения, у которого свойство НачальнаяСтраница имеет значение Истина. И свойство Содержимое содержит формы начальной страницы.
Если активна начальная страница то и данный метод, и свойство Окно формы начальной страницы возвращают один и тот же объект.

Описание:

Получает текущее активное окно.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска клиентского приложения "Обычное приложение" возвращает Неопределено.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
