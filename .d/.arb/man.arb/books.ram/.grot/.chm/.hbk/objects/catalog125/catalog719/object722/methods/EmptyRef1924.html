ЗадачаМенеджер.<Имя задачи>.ПустаяСсылка (TaskManager.<Task name>.EmptyRef)

ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: ЗадачаСсылка.<Имя задачи>.

Описание:

Получает пустое значение ссылки на задачу данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
