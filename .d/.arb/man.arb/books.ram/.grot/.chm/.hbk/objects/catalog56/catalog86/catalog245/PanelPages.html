СтраницыПанели (PanelPages)

СтраницыПанели (PanelPages)

Доступен, начиная с версии 8.0.

Элементы коллекции:

СтраницаПанели
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс страницы (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Предназначен для доступа к методам коллекции страниц панели.

Доступность:

Толстый клиент.

Панель, свойство Страницы
