ПланГлобальногоПоиска (GlobalSearchPlan)

Доступен, начиная с версии 8.3.15.

Элементы коллекции:

ЭлементПланаГлобальногоПоиска
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения элементов коллекции.
Возможно обращение к значению элемента посредством оператора [...]. В качестве аргумента передается индекс значения (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Описывает план выполнения глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

См. также:

МенеджерГлобальногоПоиска, метод УстановитьПлан
Глобальный контекст, событие ПриГлобальномПоиске

Использование в версии:

Доступен, начиная с версии 8.3.15.
