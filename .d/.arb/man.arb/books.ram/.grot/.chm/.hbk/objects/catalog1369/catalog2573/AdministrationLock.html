АдминистрированиеБлокировка (AdministrationLock)

Доступен, начиная с версии 8.3.14.

Свойства:

ВремяНачалаБлокировки (LockBeginTime)
ИдентификаторОбъекта (ObjectID)
ИдентификаторСеанса (SessionID)
ИдентификаторСоединения (ConnectionID)
Описание (Description)

Описание:

Используется для доступа к свойствам блокировки объекта.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

См. также:

АдминистрированиеКластер, метод ПолучитьБлокировки

Доступен, начиная с версии 8.3.14.
