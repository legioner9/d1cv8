ОписаниеКомандыПоляВвода (InputFieldCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: СтандартнаяКомандаПоляВвода, ОписаниеОповещения, Массив, Неопределено.
Содержит описание команды:
  • СтандартнаяКомандаПоляВвода – стандартная команда;
  • ОписаниеОповещения – процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
    • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения;
  • Массив – задает подменю. Массив может содержать элементы обозначенных выше типов;
  • Неопределено – разделитель;
  • Любой другой тип - пункт меню не отображается.

Если свойство содержит значение типа СтандартнаяКомандаПоляВвода, игнорируются прочие свойства объекта.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
