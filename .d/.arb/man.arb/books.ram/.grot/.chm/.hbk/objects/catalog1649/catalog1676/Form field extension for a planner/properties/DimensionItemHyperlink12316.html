Расширение поля формы для поля планировщика (Form field extension for a planner)

ГиперссылкаЭлементаИзмерения (DimensionItemHyperlink)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет внешний вид и поведение при нажатии левой кнопкой мыши на элемент измерения:
  • Истина - вызывается событие ОбработкаФормированияКоманд.
  • Ложь - никакие действия не выполняются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
