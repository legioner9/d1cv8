  • ДанныеПереходаПоНавигационнойСсылкеЗапуска

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: ДанныеПереходаПоНавигационнойСсылке, Неопределено.
Используется для доступа к ссылке, по которой был произведен запуск приложения.
Свойство доступно во всех клиентских обработчиках событий, начиная с обработчика события ПередНачаломРаботыСистемы.
В мобильной платформе помимо внешней ссылки запуск может быть осуществлен с помощью ассоциативной ссылки со специальной схемой (maps://, mailto://, tel:, и т.д.).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
