Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ЗакончитьРедактированиеТекущейОбласти (EndEditCurrentArea)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ЗакончитьРедактированиеТекущейОбласти(<ОтменаРедактирования>)

Параметры:

<ОтменаРедактирования> (необязательный)

Тип: Булево.
Признак завершения редактирования с отменой редактирования или принятием изменений.
Значение по умолчанию: Ложь.

Описание:

Завершает редактирование текущей области табличного документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
