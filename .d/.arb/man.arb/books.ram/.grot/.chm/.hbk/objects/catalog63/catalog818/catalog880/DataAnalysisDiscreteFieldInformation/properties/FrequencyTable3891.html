ИнформацияДискретногоПоляАнализаДанных (DataAnalysisDiscreteFieldInformation)

ТаблицаЧастот (FrequencyTable)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ТаблицаЗначений.
Таблица значений, содержащая частоту появления значений. Содержит следующие колонки:
  • Значение (Value) - значение поля;
  • Частота (Frequency) - частота возникновения значения (Число);
  • ОтносительнаяЧастота (RelativeFrequency) - относительная частота возникновения значения (Число);
  • НакопленнаяЧастота (CumulativeFrequency) - накопленная частота (Число);
  • НакопленнаяОтносительнаяЧастота (CumulativeRelativeFrequency) - накопленная относительная частота (Число).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
