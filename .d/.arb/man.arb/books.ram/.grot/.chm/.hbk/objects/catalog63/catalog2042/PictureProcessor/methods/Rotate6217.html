ОбработкаКартинок.Повернуть (PictureProcessor.Rotate)

ОбработкаКартинок (PictureProcessor)

Повернуть (Rotate)

Не рекомендуется использовать, начиная с версии 8.3.14.

Доступен, начиная с версии 8.3.19.

Синтаксис:

Повернуть(<Картинка>, <Угол>)

Параметры:

<Картинка> (необязательный)

Тип: Картинка.
Картинка, которую нужно повернуть.

<Угол> (необязательный)

Тип: Число.
Угол (в градусах), на который нужно повернуть картинку. Угол должен быть кратен 90.

Возвращаемое значение:

Тип: Картинка.
Повернутая картинка.

Описание:

Поворачивает картинку на заданный угол.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.

Не рекомендуется использовать, начиная с версии 8.3.14.
