ЗаписьPDF.НачатьОткрытие (PDFWriter.BeginOpen)

ЗаписьPDF (PDFWriter)

НачатьОткрытие (BeginOpen)

Доступен, начиная с версии 8.3.16.

Вариант синтаксиса: Открыть файл

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<ИмяФайла> (обязательный)

Тип: Строка.
Имя файла для чтения PDF-документа.

Вариант синтаксиса: Открыть поток

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <Поток>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: ОписаниеОповещения.
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

<Поток> (обязательный)

Тип: Поток, ПотокВПамяти, ФайловыйПоток.
Поток для записи PDF-документа.

Описание:

Открывает файл PDF-документа для записи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
