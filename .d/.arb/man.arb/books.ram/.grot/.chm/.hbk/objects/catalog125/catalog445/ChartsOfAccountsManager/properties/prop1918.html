ПланыСчетовМенеджер (ChartsOfAccountsManager)

<Имя плана счетов> (<Chart of accounts name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовМенеджер.<Имя плана счетов>.
Набор свойств содержит менеджеры отдельных планов счетов. Доступ к менеджеру осуществляется по имени, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
