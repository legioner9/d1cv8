АдминистрированиеПриоритетВыбораПроцесса (AdministrationProcessChoicePriority)

Доступен, начиная с версии 8.3.14.

Значения

ПоПамяти (ByMemory)
ПоПроизводительности (ByPerformance)

Описание:

Определяет приоритет выбора рабочего процесса.
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
