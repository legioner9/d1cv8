Верх (Top)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Содержит положение верхней границы подписи (в миллиметрах) на странице PDF-документа с учетом полей страницы. Начало координат страницы – левый верхний угол.
Если при подписании не указано, то подпись будет расположена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
