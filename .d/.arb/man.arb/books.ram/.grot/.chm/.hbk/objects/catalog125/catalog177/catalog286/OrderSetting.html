НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементУправленияПорядком
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки порядка.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)

Конструкторы:

По умолчанию

Описание:

Коллекция элементов управления порядком.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

Расширение табличного поля порядка, свойство НастройкаПорядка

Использование в версии:

Доступен, начиная с версии 8.0.
