СущностьDOM.УдалитьДочерний (DOMEntity.RemoveChild)

СущностьDOM (DOMEntity)

УдалитьДочерний (RemoveChild)

Доступен, начиная с версии 8.1.

Синтаксис:

УдалитьДочерний(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: АтрибутDOM, ДокументDOM, ЭлементDOM, ОпределениеТипаДокументаDOM, НотацияDOM, СущностьDOM, ФрагментДокументаDOM, ТекстDOM, КомментарийDOM, СекцияCDATADOM, ИнструкцияОбработкиDOM, СсылкаНаСущностьDOM, ПространствоИменXPath.
Удаляемый узел DOM.

Возвращаемое значение:

Тип: АтрибутDOM, ДокументDOM, ЭлементDOM, ОпределениеТипаДокументаDOM, НотацияDOM, СущностьDOM, ФрагментДокументаDOM, ТекстDOM, КомментарийDOM, СекцияCDATADOM, ИнструкцияОбработкиDOM, СсылкаНаСущностьDOM, ПространствоИменXPath.
Удаляемый узел.

Описание:

Удаляет узел из списка дочерних узлов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Исключение вызывается в случаях:
  • если данный узел в режиме Только для чтения;
  • если удаляемый узел не найден;
  • если удаляемый узел является дочерним узлом узла Документ, исключение генерируется в случае, если реализация не поддерживает удаление узлов ОпределениеТипаДокумента и Элемент.

Использование в версии:

Доступен, начиная с версии 8.1.
