РазрешеннаяВнешняяКомпонента (AllowedAddIn)

ХешСумма (HashSum)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит контрольную сумму разрешенной внешней компоненты, вычисленную алгоритмом SHA-1 и преобразованную к строке base64.
Для построения контрольной суммы можно использовать:
  • объект ХешированиеДанных с указанием хеш-функции SHA1;
  • метод глобального контекста Base64Строка
.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
