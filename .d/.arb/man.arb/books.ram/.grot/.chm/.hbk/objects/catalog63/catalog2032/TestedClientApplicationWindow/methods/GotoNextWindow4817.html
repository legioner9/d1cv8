ПерейтиКСледующемуОкну (GotoNextWindow)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПерейтиКСледующемуОкну()

Описание:

Для объекта ТестируемоеОкноКлиентскогоПриложения, у которого свойство Основное установлено в значение Истина, выполняет переход к следующему открытому окну.
Для остальных окон генерирует исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
