WSСсылкаМенеджер.<Имя WS-Ссылки>.СоздатьWSПрокси (WSReferenceManager.<WS-reference name>.CreateWSProxy)

WSСсылкаМенеджер.<Имя WS-Ссылки> (WSReferenceManager.<WS-reference name>)

СоздатьWSПрокси (CreateWSProxy)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьWSПрокси(<URIПространстваИменСервиса>, <ИмяСервиса>, <ИмяПорта>, <ИнтернетПрокси>, <Таймаут>, <ЗащищенноеСоединение>, <Местоположение>, <ИспользоватьАутентификациюОС>)

Параметры:

<URIПространстваИменСервиса> (обязательный)

Тип: Строка.
URI пространства имен Web-сервиса.

<ИмяСервиса> (обязательный)

Тип: Строка.
Локальное имя Web-сервиса.

<ИмяПорта> (обязательный)

Тип: Строка.
Имя порта Web-сервиса.

<ИнтернетПрокси> (необязательный)

Тип: ИнтернетПрокси.
Используется для осуществления HTTP-соединения с веб-сервисом объектом WSПрокси.
Значение по умолчанию: Неопределено.

<Таймаут> (необязательный)

Тип: Число.
Определяет время ожидания соединения и операций, осуществляемых объектом WSПрокси, в секундах.
0 - не устанавливать таймаут.
Значение по умолчанию: 0.

<ЗащищенноеСоединение> (необязательный)

Тип: ЗащищенноеСоединениеOpenSSL, Неопределено.
Объект защищенного соединения для осуществления HTTPS соединения с веб-сервисом объектом WSПрокси.
Значение по умолчанию: Неопределено.

<Местоположение> (необязательный)

Тип: Строка.
Позволяет указать фактический адрес сервиса.
Если параметр не задан или задана пустая строка, то в качестве адреса сервиса используется значение свойства Местоположение.
Значение по умолчанию: Неопределено.

<ИспользоватьАутентификациюОС> (необязательный)

Тип: Булево.
Указывает на использование аутентификации NTLM или Negotiate на сервере.
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: WSПрокси.

Описание:

Создает прокси для обращения к внешнему Web-сервису.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметры заданы неправильно (не позволяют идентифицировать сервис и порт), то будет инициировано исключение.

Пример:

Прокси = WSСсылки.Заказы.СоздатьWSПрокси(
    "http://orders.org", "orderService", "orderPort");

Использование в версии:

Доступен, начиная с версии 8.1.
