ЭлементФреймHTML.ТекстовоеСодержимое (HTMLFrameElement.TextContent)

ЭлементФреймHTML (HTMLFrameElement)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Текстовое содержимое узла HTML.
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство ЭтоПробельныеСимволыЭлемента).
Принцип формирования текстового значения узла:
Элемент, Атрибут, ФрагментДокумента - объединение текстового содержимого каждого из дочерних узлов, исключая Комментарий. Пустая строка, если дочерних узлов нет;
Текст, Комментарий - значение свойства ЗначениеУзла;
Документ, ОпределениеТипаДокумента - Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
