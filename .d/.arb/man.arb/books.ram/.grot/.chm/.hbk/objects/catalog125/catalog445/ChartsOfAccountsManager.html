ПланыСчетовМенеджер (ChartsOfAccountsManager)

ПланыСчетовМенеджер (ChartsOfAccountsManager)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПланСчетовМенеджер.<Имя плана счетов>
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов счетов.
Возможно обращение к менеджеру плана счетов посредством оператора [...]. В качестве аргумента передается имя плана счетов, как оно задано в конфигурации.

Свойства:

<Имя плана счетов> (<Chart of accounts name>)

Методы:

ТипВсеСсылки (AllRefsType)

Описание:

Предназначен для управления планами счетов и предоставляет доступ к значениям типа ПланСчетовМенеджер.<Имя плана счетов>. Доступ к объекту осуществляется через свойство глобального контекста ПланыСчетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером.

См. также:

Глобальный контекст, свойство ПланыСчетов

Использование в версии:

Доступен, начиная с версии 8.0.
