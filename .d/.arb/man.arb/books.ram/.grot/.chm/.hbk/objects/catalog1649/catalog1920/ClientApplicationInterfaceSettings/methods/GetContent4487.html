НастройкиИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceSettings)

ПолучитьСостав (GetContent)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьСостав()

Возвращаемое значение:

Тип: НастройкиСоставаИнтерфейсаКлиентскогоПриложения.

Описание:

Получает список панелей, размещенных в основном окне приложения. Список формируется из состава и положения панелей, заданных в конфигураторе, к которым применены пользовательские настройки.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
