Профиль безопасности.UnSafeExternalModuleFullAccess (ISecurityProfile.UnSafeExternalModuleFullAccess)

Профиль безопасности (ISecurityProfile)

UnSafeExternalModuleFullAccess (UnSafeExternalModuleFullAccess)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет разрешение запуска внешнего кода (внешние отчеты, внешние обработки, оператор Выполнить, Вычислить, расширения конфигурации) без безопасного режима, если это указано в конфигурации.
Истина – для любого кода. Ложь – разрешен запуск без включения безопасного режима только внешних модулей, перечисленных в списке разрешенных внешних модулей.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
