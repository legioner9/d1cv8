Расширение таблицы формы для отбора компоновки данных (Form table extension for data composition filter)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Свойство появляется у таблицы, если она связана с данными типа ОтборКомпоновкиДанных.
Если свойство установлено в Ложь, именованные элементы (свойство Представление не пустое) отображаются в табличном поле одной строкой с одной ячейкой, в которой выводится представление.
В противном случае, представление редактируется в отдельной колонке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
