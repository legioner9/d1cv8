ФигураКнопки (ButtonShape)

Авто (Auto)

Доступен, начиная с версии 8.3.8.

Описание:

Фигура кнопки определяется автоматически:
  • Для стандартной команды используется значение, заданное для команды.
  • В остальных случаях используется значение Обычная.

Использование в версии:

Доступен, начиная с версии 8.3.8.
