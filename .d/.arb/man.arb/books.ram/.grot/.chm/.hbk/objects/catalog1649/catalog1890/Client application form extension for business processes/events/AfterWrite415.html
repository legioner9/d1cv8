Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

ПослеЗаписи (AfterWrite)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписи(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи>

Структура, содержащая параметры записи. Для формы бизнес-процесса существует один предопределенный параметр Старт. При отработке команд "Старт", "Стартовать и закрыть" он устанавливается в Истина.

Описание:

Возникает на клиенте после записи объекта и после окончания транзакции записи, то есть к моменту вызова этой процедуры запись объекта полностью завершена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В обработчике данного события нельзя использовать серверные методы формы с директивой компиляции &НаСервер, а также изменять свойства, которые могут привести к серверному вызову.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.15.
