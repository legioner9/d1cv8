СредстваБуфераОбмена.СодержитДанныеАсинх (ClipboardTools.ContainsDataAsync)

СредстваБуфераОбмена (ClipboardTools)

СодержитДанныеАсинх (ContainsDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СодержитДанныеАсинх(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: СтандартныйФорматДанныхБуфераОбмена, Строка.
Проверяемый формат данных.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - тип Булево.

Описание:

Асинхронная функция. Возвращает признак того, что буфер обмена содержит данные указанного формата. Данный метод рекомендуется использовать перед вызовом метода ПолучитьДанныеАсинх.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте может быть отображен запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается Обещание, результат которого – Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.24.
