СлойГеографическойСхемы (GeographicalSchemaLayer)

НайтиПоЗначению (FindByValue)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоЗначению(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.
Искомое значение.

Возвращаемое значение:

Тип: ТочечныйОбъектГеографическойСхемы, МноготочечныйОбъектГеографическойСхемы, ПолилинейныйОбъектГеографическойСхемы, ПолигональныйОбъектГеографическойСхемы, Неопределено.
Если значение не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск объекта в слое, сравнивая по значению параметра со свойством Значение всех объектов данного слоя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
