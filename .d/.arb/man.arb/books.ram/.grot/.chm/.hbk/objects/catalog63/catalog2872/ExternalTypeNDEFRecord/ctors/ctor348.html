ЗаписьNDEFВнешнегоТипа (ExternalTypeNDEFRecord)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ЗаписьNDEFВнешнегоТипа(<Домен>, <ТипЗаписи>, <Данные>, <ИдентификаторЗаписи>)

Параметры:

<Домен> (обязательный)

Тип: Строка.
Домен организации.
Суммарный размер параметров <Домен> и <ТипЗаписи> не должен превышать 254 байт.

<ТипЗаписи> (обязательный)

Тип: Строка.
Имя типа. Строка должна состоять только из латинских символов.
Суммарный размер параметров <Домен> и <ТипЗаписи> не должен превышать 254 байт.

<Данные> (обязательный)

Тип: ДвоичныеДанные.
Двоичные данные.
Размер не должен превышать 10 Мбайт.

<ИдентификаторЗаписи> (необязательный)

Тип: Строка.
Строка размером не более 255 байт.
Обычно не указывается для экономии места на метке.

Описание:

Если в параметрах обнаружена ошибка, будет сгенерировано исключение с описанием ошибки.

Использование в версии:

Доступен, начиная с версии 8.3.23.
