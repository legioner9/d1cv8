ИзмеренияПостроителяОтчета.Вставить (ReportBuilderDimensions.Insert)

ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Тип>, <Область>, <ОбластьИерархия>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: Строка.
Путь к данным для измерения.

<Имя> (необязательный)

Тип: Строка.
Имя измерения.

<Тип> (необязательный)

Тип: ТипИзмеренияПостроителяОтчета.
Тип измерения.

<Область> (необязательный)

Тип: ТабличныйДокумент, ПолеТабличногоДокумента.
Область для вывода.

<ОбластьИерархия> (необязательный)

Тип: ТабличныйДокумент, ПолеТабличногоДокумента.
Область для иерархического вывода.

<Позиция> (обязательный)

Тип: Число.
Позиция, в которую необходимо вставить измерение.

Возвращаемое значение:

Тип: ИзмерениеПостроителяОтчета.

Описание:

Вставляет измерение в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

СтрПутьКДанным = ПостроительОтчета.ИзмеренияКолонки[Индекс].ПутьКДанным;
Измерение = ПостроительОтчета.ИзмеренияКолонки[Индекс];
ПостроительОтчета.ИзмеренияКолонки.Удалить(Измерение);
ПостроительОтчета.ИзмеренияКолонки.Вставить(СтрПутьКДанным,,,,,Индекс);

Использование в версии:

Доступен, начиная с версии 8.0.
