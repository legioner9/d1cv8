ПоложениеДействияЭлементаПланировщика (PlannerItemActionLocation)

Доступен, начиная с версии 8.3.18.

Значения

ВКонцеЭлемента (EndOfItem)
ПослеТекста (EndOfText)

Описание:

Содержит варианты положения действия внутри элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML.
