Авто (Auto)

Доступен, начиная с версии 8.3.20.

Описание:

Для формы настроек отчета, формы редактирования варианта отчета:
  • В тонком и веб-клиентах используется ЗакладкиСверху,
  • На мобильных клиентах используется значение ЗакладкиСлеваГоризонтально.
Для остальных форм используется значение ЗакладкиСверху.

Использование в версии:

Доступен, начиная с версии 8.3.20.
