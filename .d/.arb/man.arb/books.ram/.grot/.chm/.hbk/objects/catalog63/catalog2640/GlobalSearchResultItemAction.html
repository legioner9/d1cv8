ДействиеЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemAction)

Доступен, начиная с версии 8.3.18.

Свойства:

Значение (Value)
Картинка (Picture)
Текст (Text)

Описание:

Представляет собой действие элемента результата глобального поиска. Отображаются в виде гиперссылок при отображении результата поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

См. также:
