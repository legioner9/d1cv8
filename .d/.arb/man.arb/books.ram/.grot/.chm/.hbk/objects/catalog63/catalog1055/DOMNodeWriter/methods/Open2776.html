ЗаписьУзловDOM.Открыть (DOMNodeWriter.Open)

ЗаписьУзловDOM (DOMNodeWriter)

Открыть (Open)

Доступен, начиная с версии 8.1.

Синтаксис:

Открыть(<ДокументDOM>)

Параметры:

<ДокументDOM> (обязательный)

Тип: ДокументDOM.
ДокументDOM, который будет использован для создания узлов DOM при записи данных.

Описание:

Открывает объект записи данных XML в объектную модель DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
