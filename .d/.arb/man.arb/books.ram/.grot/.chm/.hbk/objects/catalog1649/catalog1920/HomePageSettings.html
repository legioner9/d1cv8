НастройкиНачальнойСтраницы (HomePageSettings)

Доступен, начиная с версии 8.2.

Методы:

ПолучитьСоставФорм (GetForms)
УстановитьСоставФорм (SetForms)

Конструкторы:

По умолчанию

Описание:

Содержит настройки начальной страницы (состав и расположение форм на начальной странице).

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.18.
