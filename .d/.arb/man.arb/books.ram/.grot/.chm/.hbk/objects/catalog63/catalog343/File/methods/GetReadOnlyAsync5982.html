Файл.ПолучитьТолькоЧтениеАсинх (File.GetReadOnlyAsync)

Файл (File)

ПолучитьТолькоЧтениеАсинх (GetReadOnlyAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьТолькоЧтениеАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Истина, если у файла установлен атрибут "Только чтение", в противном случае - Ложь.

Описание:

Асинхронная функция. Получает значение атрибута файла "Только чтение" ("Read only").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Применим только к файлам или каталогам, которые реально существуют.
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
