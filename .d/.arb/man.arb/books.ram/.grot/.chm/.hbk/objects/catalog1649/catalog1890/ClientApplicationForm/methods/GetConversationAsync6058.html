ФормаКлиентскогоПриложения (ClientApplicationForm)

ПолучитьОбсуждениеАсинх (GetConversationAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьОбсуждениеАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа ИдентификаторОбсужденияСистемыВзаимодействия или Неопределено, если у формы нет навигационной ссылки (например, форма нового незаписанного объекта).

Описание:

Асинхронная функция. Выполняет получение контекстного обсуждения формы.
Если обсуждение с указанным идентификатором не найдено, оно будет создано.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

ФормаКлиентскогоПриложения, метод НачатьПолучениеОбсуждения

Использование в версии:

Доступен, начиная с версии 8.3.18.
