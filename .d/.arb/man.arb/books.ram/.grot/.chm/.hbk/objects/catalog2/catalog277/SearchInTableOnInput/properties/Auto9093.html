ПоискВТаблицеПриВводе (SearchInTableOnInput)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

При вводе символов в таблицу, связанную с данными типа ДинамическийСписок, выполняется поиск в таблице.
При вводе символов в таблицу, связанную с ДанныеФормыКоллекция или ДанныеФормыСтруктураСКоллекцией, поиск выполняется только в режиме совместимости конфигурации Версия8_2_13.

Использование в версии:

Доступен, начиная с версии 8.2.
