Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ПередЗагрузкойПользовательскихНастроекНаСервере (BeforeLoadUserSettingsAtServer)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПередЗагрузкойПользовательскихНастроекНаСервере(<Элемент>, <Настройки>, <ИспользуютсяСтандартныеНастройки>)

Параметры:

<Элемент>

Тип: ТаблицаФормы.
Таблица, для которой выполняется сохранение настроек.

<Настройки>

Тип: ПользовательскиеНастройкиКомпоновкиДанных.
Загружаемые настройки.

<ИспользуютсяСтандартныеНастройки>

Тип: Булево.
Если имеет значение Истина, настройки были сформированы на базе основных настроек динамического списка. Если Ложь - настройки загружены из хранилища.

Описание:

Вызывается при восстановлении пользовательских настроек, перед помещением настроек в компоновщик настроек. В обработчике можно изменить значения загружаемых настроек.
Обработчик вызывается при получении формы, а также при интерактивной загрузке настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
