Расширение элементов управления, расположенных в табличном поле (Extension for controls located in a table box)

Доступен, начиная с версии 8.0.

Свойства:

Значение (Value)
ТипЗначения (ValueType)

Методы:

ПолучитьДействие (GetAction)
УстановитьДействие (SetAction)

События:

<Имя события> (<Event name>)

Описание:

Дополнительные параметры событий элемента управления в табличном поле. Эти параметры появляются у любого элемента управления, расположенного в табличном поле.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
