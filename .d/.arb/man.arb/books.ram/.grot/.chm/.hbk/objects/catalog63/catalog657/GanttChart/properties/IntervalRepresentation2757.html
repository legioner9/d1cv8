ДиаграммаГанта.ОтображениеИнтервала (GanttChart.IntervalRepresentation)

ДиаграммаГанта (GanttChart)

ОтображениеИнтервала (IntervalRepresentation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: ОтображениеИнтервалаДиаграммыГанта.
Указывает вариант отображения интервала диаграммы Ганта.
В режиме совместимости Версия8_3_22 и ниже используется значение Градиент.
В режиме совместимости выше Версия8_3_22 используется значение Плоский.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.23.
