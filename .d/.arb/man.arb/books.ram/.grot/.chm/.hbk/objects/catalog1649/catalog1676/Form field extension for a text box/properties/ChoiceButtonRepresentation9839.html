Расширение поля формы для поля ввода (Form field extension for a text box)

ОтображениеКнопкиВыбора (ChoiceButtonRepresentation)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: ОтображениеКнопкиВыбора.
Если указано значение Авто, то значение интерпретируется в зависимости от типа, редактируемого в поле ввода значения:
  • Ссылочный тип (в том числе пользовательское перечисление) - ОтображатьВВыпадающемСписке;
  • Для остальных типов - ОтображатьВПолеВвода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
