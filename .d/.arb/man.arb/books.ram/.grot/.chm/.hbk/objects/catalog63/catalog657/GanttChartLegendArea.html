ОбластьЛегендыДиаграммыГанта (GanttChartLegendArea)

Доступен, начиная с версии 8.1.

Свойства:

Верх (Top)
Лево (Left)
Низ (Bottom)
Право (Right)
ПрозрачныйФон (Transparent)
Прокрутка (Scrolling)
Рамка (Border)
Расположение (Placement)
ЦветРамки (BorderColor)
ЦветТекста (TextColor)
ЦветФона (BackColor)
Шрифт (Font)

Описание:

Используется для доступа к области легенды диаграммы Ганта для настройки параметров оформления. Легенда предназначена для описания серий диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

ДиаграммаГанта, свойство ОбластьЛегенды

Использование в версии:

Доступен, начиная с версии 8.1.
