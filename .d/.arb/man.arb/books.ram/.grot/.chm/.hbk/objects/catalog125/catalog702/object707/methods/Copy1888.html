БизнесПроцессОбъект.<Имя бизнес-процесса>.Скопировать (BusinessProcessObject.<Business process name>.Copy)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: БизнесПроцессОбъект.<Имя бизнес-процесса>.

Описание:

Создает новый бизнес-процесс копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

Объект = ВыбБизнесПроцесс.Скопировать();

См. также:

БизнесПроцессОбъект.<Имя бизнес-процесса>, событие ПриКопировании

Использование в версии:

Доступен, начиная с версии 8.0.
