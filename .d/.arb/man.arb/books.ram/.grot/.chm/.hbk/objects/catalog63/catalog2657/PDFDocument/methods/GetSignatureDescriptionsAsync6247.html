ДокументPDF.ПолучитьОписанияПодписейАсинх (PDFDocument.GetSignatureDescriptionsAsync)

ДокументPDF (PDFDocument)

ПолучитьОписанияПодписейАсинх (GetSignatureDescriptionsAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПолучитьОписанияПодписейАсинх(<МенеджерКриптографии>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: МенеджерКриптографии.
Менеджер криптографии.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - массив объектов типа ОписаниеПодписиPDF, содержащий описания присоединенных к PDF-документу электронных подписей.

Описание:

Асинхронная функция. Получает массив, содержащий описания присоединенных к PDF-документу электронных подписей.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
