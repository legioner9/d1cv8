ПланОбменаСсылка.<Имя плана обмена>.Скопировать (ExchangePlanRef.<Exchange plan name>.Copy)

ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: ПланОбменаОбъект.<Имя плана обмена>.

Описание:

Создает новый узел копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.0.
