ПланВидовРасчетаМенеджер.<Имя плана видов расчета>.ПолучитьИнициализациюПредопределенныхДанных (ChartOfCalculationTypesManager.<Chart of calculation types name>.GetPredefinedDataInitialization)

ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

ПолучитьИнициализациюПредопределенныхДанных (GetPredefinedDataInitialization)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИнициализациюПредопределенныхДанных(<Область>)

Параметры:

<Область> (необязательный)

Тип: Соответствие.
Данный параметр позволяет задать область объекта, в которой необходимо получить признак проинициализированности.
Область описывается соответствием: разделитель - значение. Могут быть использованы только разделители, разделяющие данные в режиме НезависимоИСовместно.
Ключом соответствия может быть объект метаданных или имя разделителя.
Если разделители, переданные в этом параметре, установлены и используются в сеансе, то они игнорируются.
Значение по умолчанию: Неопределено.

Описание:

Получает признак того, что предопределенные данные области инициализированы.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
