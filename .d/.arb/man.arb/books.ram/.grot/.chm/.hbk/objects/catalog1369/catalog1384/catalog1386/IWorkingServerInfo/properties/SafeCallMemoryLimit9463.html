Рабочий сервер.SafeCallMemoryLimit (IWorkingServerInfo.SafeCallMemoryLimit)

Рабочий сервер (IWorkingServerInfo)

SafeCallMemoryLimit (SafeCallMemoryLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.
Определяет размер безопасного расхода памяти за один вызов.
Объем памяти в байтах, использование которого в процессе вызова сервера считается безопасным и не приводит к прекращению вызова даже если достигнут максимальный объем памяти рабочих процессов сервера.
Допустимы значения от -1 до 9223372036854775807.
Если значение 0, то безопасный расход памяти определяется автоматически, как 5% максимального объема памяти рабочих процессов на данном рабочем сервере.
Если значение -1, любой вызов считается опасным, если достигнут максимальный объем памяти рабочих процессов.

Доступность:

Интеграция.

См. также:

Рабочий сервер, свойство SafeWorkingProcessesMemoryLimit

Использование в версии:

Доступен, начиная с версии 8.3.1.
