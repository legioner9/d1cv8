ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

Отказ (Cancel)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Если установлено значение Истина, то запись данных не отражается в истории данных.
Если данные не используются в истории, то данное свойство игнорируется.
Не влияет на работу ЗаписатьВерсию, поскольку при вызове метода запись данных не выполняется.
Значение по умолчанию - Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
