ВыраженияСхемыЗапроса (QuerySchemaExpressions)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

ВыражениеСхемыЗапроса
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения схемы запроса.
Возможно обращение к выражению схемы запроса посредством оператора [...]. В качестве аргумента передается индекс.

Методы:

Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Удалить (Delete)
Установить (Set)

Описание:

Содержит коллекцию выражений схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
