Использование:

Только чтение.

Описание:

Тип: ИспользованиеГруппИЭлементов.
Устанавливает вид данных, которые могут быть использованы в качестве значения признака учета плана счетов: Элементы, Группы, Группы и элементы.
Имеет смысл для ссылочных типов данных, описывающих ссылки на справочники или планы видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
