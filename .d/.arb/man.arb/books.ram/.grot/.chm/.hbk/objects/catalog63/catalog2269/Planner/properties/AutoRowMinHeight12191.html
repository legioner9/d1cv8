Планировщик (Planner)

АвтоМинимальнаяВысотаСтроки (AutoRowMinHeight)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Определяет использование автоматического ограничения минимальной высоты строки.
Если значение Ложь, автоматическое ограничение не используется, и минимальная высота определяется свойством МинимальнаяВысотаСтроки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.15.
