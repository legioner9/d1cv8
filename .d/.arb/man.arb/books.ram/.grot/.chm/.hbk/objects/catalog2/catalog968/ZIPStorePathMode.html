РежимСохраненияПутейZIP (ZIPStorePathMode)

Доступен, начиная с версии 8.0.

Значения

НеСохранятьПути (DontStorePath)
СохранятьОтносительныеПути (StoreRelativePath)
СохранятьПолныеПути (StoreFullPath)

Описание:

Содержит варианты методов сохранения путей файлов при помещении их в архив.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

ЗаписьZipФайла, метод Добавить

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
