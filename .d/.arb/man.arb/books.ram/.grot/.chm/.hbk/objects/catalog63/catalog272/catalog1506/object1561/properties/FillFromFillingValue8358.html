ОбъектМетаданных: РеквизитАдресации (MetadataObject: AddressingAttribute)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.
Свойство определяет возможность заполнения реквизита адресации из данных заполнения. Используется для заполнения реквизитов из значений текущего отбора списков. Если установлено Истина, то процедура стандартной обработки заполнения будет заполнять реквизит адресации из данных заполнения. Если в данных заполнения нет данных для заполнения этого реквизита или свойство имеет значение Ложь, то ресурс будет заполнен из свойства ЗначениеЗаполнения, если оно установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
