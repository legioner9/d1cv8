СоставКопииБазыДанных (DatabaseCopyContent)

Количество (Count)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.

Описание:

Получает количество элементов состава.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
