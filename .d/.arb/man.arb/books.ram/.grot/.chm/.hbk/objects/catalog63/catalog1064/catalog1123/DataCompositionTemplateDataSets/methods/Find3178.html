НаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSets)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.
Имя искомого элемента.

Возвращаемое значение:

Тип: НаборДанныхЗапросМакетаКомпоновкиДанных, НаборДанныхОбъектМакетаКомпоновкиДанных, НаборДанныхОбъединениеМакетаКомпоновкиДанных, Неопределено.
Если элемент не найден, будет возвращено значение Неопределено.

Описание:

Осуществляет поиск элемента коллекции по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
