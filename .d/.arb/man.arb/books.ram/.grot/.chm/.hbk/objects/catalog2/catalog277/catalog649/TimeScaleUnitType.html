ТипЕдиницыШкалыВремени (TimeScaleUnitType)

Доступен, начиная с версии 8.0.

Значения

Год (Year)
День (Day)
Квартал (Quarter)
Месяц (Month)
Минута (Minute)
Неделя (Week)
Секунда (Second)
Час (Hour)

Описание:

Содержит варианты единиц измерения шкалы времени диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: TimeScaleUnitType.
