ФильтрОбработкиСканированияДокументов (DocumentScanningProcessingFilter)

Текст (Text)

Доступен, начиная с версии 8.3.23.

Описание:

Тип: Строка.
Фильтр для обработки документа, содержащего только текст.
Данный фильтр максимально упрощает отображение картинок в документе.
Если предполагается, что документ будут обрабатывать вручную и в документе не ожидается ничего кроме текста, таблиц, печатей, рамок, подчеркиваний и аналогичной графической информации, то можно указывать этот фильтр в качестве фильтра по умолчанию, но рекомендуется добавлять Нет в качестве дополнительного варианта.

Использование в версии:

Доступен, начиная с версии 8.3.23.
