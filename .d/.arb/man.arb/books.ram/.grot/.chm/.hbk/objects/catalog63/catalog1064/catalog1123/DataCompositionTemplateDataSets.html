НаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSets)

Доступен, начиная с версии 8.1.

Элементы коллекции:

НаборДанныхЗапросМакетаКомпоновкиДанных, НаборДанныхОбъектМакетаКомпоновкиДанных, НаборДанныхОбъединениеМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы.
Возможно обращение к набору посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция наборов данных макета компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
