ПараметрыАнализаДанных (DataAnalysisParameters)

ПараметрыАнализаДанных (DataAnalysisParameters)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ПараметрАнализаДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры.
Возможно обращение к параметру посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

<Имя параметра> (<Parameter name>)

Методы:

Индекс (IndexOf)
Количество (Count)
Найти (Find)
Получить (Get)

Описание:

Содержит параметры анализа данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
Сериализуется.
