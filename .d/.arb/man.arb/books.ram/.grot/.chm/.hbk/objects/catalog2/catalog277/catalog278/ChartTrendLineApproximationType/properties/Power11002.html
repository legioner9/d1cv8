Степенной (Power)

Доступен, начиная с версии 8.3.10.

Описание:

Степенная аппроксимация.
Все значения данных по оси Y и по оси X должны быть строго больше нуля, в противном случае линия тренда не будет построена.

Использование в версии:

Доступен, начиная с версии 8.3.10.
