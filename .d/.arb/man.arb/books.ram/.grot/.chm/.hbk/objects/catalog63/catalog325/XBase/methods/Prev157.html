XBase.Предыдущая (XBase.Prev)

XBase (XBase)

Предыдущая (Prev)

Доступен, начиная с версии 8.0.

Синтаксис:

Предыдущая()

Возвращаемое значение:

Тип: Булево.
Истина - получена предыдущая запись; Ложь - в противном случае.

Описание:

Позиционирует объект на предыдущей записи. В зависимости от того, установлен ли текущий индекс, позиционирование происходит либо в порядке убывания физического номера записи, либо в порядке убывания значения выражения текущего индекса (если индекс создавался с установленным флагом "Убывание", то в порядке возрастания).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

// Подсчет суммы в поле COST

СуммаВсего = 0;
Пока Истина Цикл
    СуммаВсего = СуммаВсего + xB.COST;
    Если Не xB.Предыдущая() Тогда
        Прервать;
    КонецЕсли;
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
