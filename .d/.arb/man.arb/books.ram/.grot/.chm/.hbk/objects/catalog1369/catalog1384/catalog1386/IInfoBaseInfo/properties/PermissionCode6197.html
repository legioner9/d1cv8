Информационная база.PermissionCode (IInfoBaseInfo.PermissionCode)

Информационная база (IInfoBaseInfo)

PermissionCode (PermissionCode)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Код разрешения, разрешающий начало сеанса вопреки блокировке сеансов. Этот код должен быть задан в параметре /UC командной строки запуска клиентского приложения или в параметре UC строки соединения объектов V83.COMConnector или V83.Application, чтобы начать сеанс с информационной базой во время действия блокировки сеансов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
