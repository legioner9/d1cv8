ПотокВПамяти (MemoryStream)

Размер (Size)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Размер()

Возвращаемое значение:

Тип: Число.

Описание:

Получает размер данных в байтах.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Примечание:

Если ДоступноИзменениеПозиции установлено в Ложь, то при вызове этого метода будет сгенерировано исключение.
