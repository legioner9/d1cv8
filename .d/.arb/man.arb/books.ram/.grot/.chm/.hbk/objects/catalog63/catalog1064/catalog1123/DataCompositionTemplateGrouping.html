ГруппировкиМакетаКомпоновкиДанных (DataCompositionTemplateGrouping)

ГруппировкиМакетаКомпоновкиДанных (DataCompositionTemplateGrouping)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ЭлементГруппировкиМакетаКомпоновкиДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы группировки макета компоновки данных.
Возможно обращение к элементу группировки макета компоновки данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Коллекция группировок макета компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
