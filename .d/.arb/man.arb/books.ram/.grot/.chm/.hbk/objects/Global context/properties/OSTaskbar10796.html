ПанельЗадачОС (OSTaskbar)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: МенеджерПанелиЗадачОС.
Используется для управления кнопкой программы на панели задач.
Функционал работает в ОС Windows 7 и выше, а также в Ubuntu Unity. Не влияет на поведение в других операционных системах.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
