Планировщик (Planner)

ТипТекстаНовыхЭлементов (NewItemsTextType)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: ТипТекстаНовыхЭлементовПланировщика.
Содержит тип текста новых элементов планировщика. Влияет на тип параметра <Текст> события ПередСозданием, а также тип элементов планировщика, создаваемых через методы Добавить и Вставить.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение по умолчанию - Строка.

Использование в версии:

Доступен, начиная с версии 8.3.18.
