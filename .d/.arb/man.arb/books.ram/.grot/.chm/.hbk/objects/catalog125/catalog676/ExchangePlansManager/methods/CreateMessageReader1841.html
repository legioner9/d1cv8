ПланыОбменаМенеджер.СоздатьЧтениеСообщения (ExchangePlansManager.CreateMessageReader)

ПланыОбменаМенеджер (ExchangePlansManager)

СоздатьЧтениеСообщения (CreateMessageReader)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьЧтениеСообщения()

Возвращаемое значение:

Тип: ЧтениеСообщенияОбмена.

Описание:

Создает объект, с помощью которого может быть произведено чтение одного или нескольких сообщений обмена данными.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

Сообщение = ПланыОбмена.СоздатьЧтениеСообщения();

Использование в версии:

Доступен, начиная с версии 8.0.
