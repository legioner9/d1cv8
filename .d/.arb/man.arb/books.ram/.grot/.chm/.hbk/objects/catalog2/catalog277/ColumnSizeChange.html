ИзменениеРазмераКолонки (ColumnSizeChange)

Доступен, начиная с версии 8.0.

Значения

Изменять (Change)
НеИзменять (DontChange)

Описание:

Определяет набор режимов изменения размера колонки. Используется для определения свойства ИзменениеРазмера колонки табличного поля.

Доступность:

Толстый клиент.

См. также:

КолонкаТабличногоПоля, свойство ИзменениеРазмера

Использование в версии:

Доступен, начиная с версии 8.0.
