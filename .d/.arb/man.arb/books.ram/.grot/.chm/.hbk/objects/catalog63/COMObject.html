COMОбъект (COMObject)

COMОбъект (COMObject)

Доступен, начиная с версии 8.0.

Конструкторы:

По имени приложения

Описание:

Создает COM-объект (например, Word, Excel и т.д.). Методы и свойства СОМ-объектов в дальнейшем становятся доступными через данный объект.

В веб-клиенте данный объект поддерживается только в браузере Internet Explorer.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
