ТабличныйДокумент.Защита (SpreadsheetDocument.Protection)

ТабличныйДокумент (SpreadsheetDocument)

Защита (Protection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак защиты таблицы от редактирования и копирования. В режиме защиты таблица может быть только распечатана.
Истина - защита включена.
Внимание! Значение данного свойства не сохраняется в следующих случаях:
- при сохранении документа в двоичные данные,
- при сохранении документа в файл.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.23.
