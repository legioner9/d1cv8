XBase.Удалить (XBase.Delete)

XBase (XBase)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Ставит пометку "удалена" на текущую запись. Физически запись из таблицы базы данных не удаляется и может быть впоследствии восстановлена. Полное удаление записей из таблицы базы данных с освобождением дискового пространства, занятого ими, производится вызовом методов Сжать и ОчиститьФайл.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

// Удаление всех записей:

xB = Новый xBase;
xB.ОткрытьФайл("c:\test\test.dbf", "c:\test\test.cdx");
xB.Первая();
Пока Не xB.ВКонце() Цикл
    xB.Удалить();
    xB.Следующая();
КонецЦикла;

Использование в версии:

Доступен, начиная с версии 8.0.
