ИнтернетПочта (InternetMail)

ТекущийПочтовыйЯщик (CurrentMailbox)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Содержит текущий почтовый ящик на сервере, с которым выполняется работа.
Данное свойство влияет на методы ПолучитьЗаголовки, ПолучитьИдентификаторы, УдалитьСообщения, ПолучитьКоличествоСообщений, Послать для случая, когда используется протокол отправки IMAP.
Если свойство не задано, то действие будет производиться с IMAP почтовым ящиком "Входящие" ("Inbox").

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
