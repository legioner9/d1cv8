ОбластьФорматированногоДокумента (FormattedDocumentRange)

Доступен, начиная с версии 8.3.1.

Свойства:

ЗакладкаКонца (EndBookmark)
ЗакладкаНачала (BeginBookmark)

Описание:

Содержит область форматированного документа. Определяет некоторый диапазон в ФорматированныйДокумент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/formatted-document}. Имя типа XDTO: FormattedDocumentRange.

Использование в версии:

Доступен, начиная с версии 8.3.1.
