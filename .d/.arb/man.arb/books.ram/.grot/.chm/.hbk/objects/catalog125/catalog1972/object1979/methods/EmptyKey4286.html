ПустойКлюч (EmptyKey)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПустойКлюч()

Возвращаемое значение:

Тип: ВнешнийИсточникДанныхТаблицаКлючЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>.

Описание:

Получает пустой ключ записи таблицы внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных НеобъектныеДанные.

Использование в версии:

Доступен, начиная с версии 8.3.3.
