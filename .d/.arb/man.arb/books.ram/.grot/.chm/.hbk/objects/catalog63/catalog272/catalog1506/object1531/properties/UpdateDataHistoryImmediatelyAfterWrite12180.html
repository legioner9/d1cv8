ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

ОбновлятьИсториюДанныхСразуПослеЗаписи (UpdateDataHistoryImmediatelyAfterWrite)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Булево.
Если установлено значение Истина, то после завершения транзакции
выполняется обновление истории данных по записанным данным,
в фоновом задании без ожидания завершения фонового задания.
Значение по умолчанию - Ложь.
При записи данных значение, указанное в конфигураторе, записывается в свойство ОбновлятьИсториюСразуПослеЗаписи. Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
