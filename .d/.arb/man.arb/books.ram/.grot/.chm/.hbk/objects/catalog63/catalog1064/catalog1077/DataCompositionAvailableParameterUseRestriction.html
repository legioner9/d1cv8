ОграничениеИспользованияДоступногоПараметраКомпоновкиДанных (DataCompositionAvailableParameterUseRestriction)

Доступен, начиная с версии 8.2.

Свойства:

Доступность (Enabled)
Параметр (Parameter)

Описание:

Описание ограничения использования конкретного параметра в табличном поле доступных полей компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
