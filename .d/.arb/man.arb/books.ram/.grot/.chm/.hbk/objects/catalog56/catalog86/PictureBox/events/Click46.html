ПолеКартинки (PictureBox)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на поле картинки, если у него свойство Гиперссылка имеет значение Истина.

Доступность:

Толстый клиент.

См. также:

ПолеКартинки, свойство Гиперссылка

Использование в версии:

Доступен, начиная с версии 8.0.
