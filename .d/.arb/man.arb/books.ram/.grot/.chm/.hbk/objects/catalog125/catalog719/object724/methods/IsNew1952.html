ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: Булево.
Истина - изменяется еще ни разу не записанная задача; Ложь - задача уже записана.

Описание:

Определяет, записана ли задача в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

Если Объект.ЭтоНовый() Тогда
    Объект.Наименование = "Новая задача";
КонецЕсли;

Использование в версии:

Доступен, начиная с версии 8.0.
