ОбъектМетаданных: ПараметрФункциональныхОпций (MetadataObject: FunctionalOptionsParameter)

Использование (Use)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: ОбъектМетаданных: Измерение, ОбъектМетаданных: Справочник.
Указывает на объект, значение которого будет являться параметром для получения значения функциональной опции.
Параметры используется в случае, если для функциональной опции в свойстве Хранение указан реквизит справочника или ресурс регистра сведений, т.е. в случае, когда в одном объекте (справочнике или регистре сведений) может храниться набор значений опций.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство должно быть заполнено.

Использование в версии:

Доступен, начиная с версии 8.2.
