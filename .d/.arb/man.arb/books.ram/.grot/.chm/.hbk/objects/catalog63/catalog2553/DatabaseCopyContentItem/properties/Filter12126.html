ЭлементСоставаКопииБазыДанных (DatabaseCopyContentItem)

Отбор (Filter)

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: Отбор.
Поддерживается только для элементов состава копии, у которых свойство Метаданные содержит периодический регистр свдеений или регистр накопления.
Отбор содержит один элемент "Период". Для элемента отбора доступны следующие виды сравнения:
  • Больше,
  • БольшеИлиРавно,
  • Интервал,
  • ИнтервалВключаяГраницы,
  • ИнтервалВключаяНачало,
  • ИнтервалВключаяОкончание,
  • Меньше,
  • МеньшеИлиРавно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).
