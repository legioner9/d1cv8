Защита (Protection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.
Содержит признак защиты рисунка от изменений.
Истина - защита включена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Не применимо для рисунков типа Группа.

Использование в версии:

Доступен, начиная с версии 8.0.
