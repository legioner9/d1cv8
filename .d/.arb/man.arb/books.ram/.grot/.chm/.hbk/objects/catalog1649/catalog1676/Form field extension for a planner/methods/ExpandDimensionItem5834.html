Расширение поля формы для поля планировщика (Form field extension for a planner)

РазвернутьЭлементИзмерения (ExpandDimensionItem)

Доступен, начиная с версии 8.3.17.

Синтаксис:

РазвернутьЭлементИзмерения(<ЗначенияИзмерений>, <СПодчиненными>)

Параметры:

<ЗначенияИзмерений> (обязательный)

Тип: ФиксированноеСоответствие.
Фиксированное соответствие, однозначно определяющее элемент измерения, который нужно развернуть. В качестве ключа выступает Значение измерения. В качестве значения - Значение элемента измерения.

<СПодчиненными> (необязательный)

Тип: Булево.
Определяет необходимость разворачивания элемента измерения вместе с подчиненными элементами измерения. Истина – развернуть вместе с подчиненными.
Значение по умолчанию: Ложь.

Описание:

Разворачивает элемент измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
