Расширение табличного поля списка плана счетов.НастройкаПорядка (Chart of accounts list table box extension.OrderSetting)

Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: НастройкаПорядка.
Предоставляет доступ к настройкам порядка списка счетов. Представляет собой коллекцию значений типа ЭлементУправленияПорядком. Свойства коллекции содержат объекты для управления элементом порядка. Имена свойств совпадают с именами доступных для плана счетов элементов упорядочивания.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
