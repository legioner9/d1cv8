КонструкторФорматнойСтроки (FormatStringWizard)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый КонструкторФорматнойСтроки(<ФорматнаяСтрока>)

Параметры:

<ФорматнаяСтрока> (необязательный)

Тип: Строка.
Форматная строка, которая будет редактироваться конструктором.
Значение по умолчанию: Пустая строка.

Описание:

Создает конструктор форматной строки.

Использование в версии:

Доступен, начиная с версии 8.0.
