СоставПланаОбмена (ExchangePlanContent)

Доступен, начиная с версии 8.0.

Элементы коллекции:

ЭлементСоставаПланаОбмена
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы состава плана обмена.

Методы:

Количество (Count)
Найти (Find)
Содержит (Contains)

Описание:

Коллекция элементов состава объекта метаданных плана обмена.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
