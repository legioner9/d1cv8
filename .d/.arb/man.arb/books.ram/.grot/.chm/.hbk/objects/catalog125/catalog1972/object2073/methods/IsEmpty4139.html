Доступен, начиная с версии 8.3.3.

Синтаксис:

Пустой()

Возвращаемое значение:

Тип: Булево.
Истина - все поля ключа имеют значения по умолчанию.

Описание:

Определяет, все ли поля ключа имеют значения по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
