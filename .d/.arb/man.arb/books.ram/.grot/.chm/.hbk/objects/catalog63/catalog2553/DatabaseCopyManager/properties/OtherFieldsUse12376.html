ИспользованиеПрочихПолей (OtherFieldsUse)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: ИспользованиеПоляЭлементаСоставаКопииБазыДанных.
Определяет режим использования полей, которые не являются табличными частями, не имеют тип ХранилищеЗначения или Строка неограниченной длины.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
