ФайловыйПоток.СброситьБуферыАсинх (FileStream.FlushAsync)

ФайловыйПоток (FileStream)

СброситьБуферыАсинх (FlushAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СброситьБуферыАсинх()

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Сбрасывает все промежуточные буферы и производит запись всех незаписанных данных в целевое устройство.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

Поток, метод НачатьСбросБуферов
Поток, метод СброситьБуферы

Использование в версии:

Доступен, начиная с версии 8.3.18.
