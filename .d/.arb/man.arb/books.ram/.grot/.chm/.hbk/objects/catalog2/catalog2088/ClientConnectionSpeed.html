СкоростьКлиентскогоСоединения (ClientConnectionSpeed)

Доступен, начиная с версии 8.2.

Значения

Низкая (Low)
Обычная (Normal)

Описание:

Содержит варианты скорости клиентского соединения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Глобальный контекст, метод ПолучитьСкоростьКлиентскогоСоединения
