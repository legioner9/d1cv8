КоллекцияФасетовXDTO (XDTOFacetCollection)

Доступен, начиная с версии 8.1.

Элементы коллекции:

ФасетXDTO
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются фасеты XDTO.
Возможно обращение к фасету XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Свойства:

Образцы (Patterns)
Перечисления (Enumerations)

Методы:

Количество (Count)
Получить (Get)

Описание:

Коллекция фасетов типа простого значения XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
