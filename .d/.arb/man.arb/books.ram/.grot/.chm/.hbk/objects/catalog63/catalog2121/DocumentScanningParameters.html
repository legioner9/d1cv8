ПараметрыСканированияДокументов (DocumentScanningParameters)

Доступен, начиная с версии 8.3.23.

Свойства:

БелыйФонДокумента (WhiteBackgroundDocument)
ПроверкаКачества (QualityCheck)
ПрямоугольныйДокумент (RectangularDocument)
РазрешитьАвтоЗахват (AllowAutoCapture)
РежимОпределенияОриентации (OrientationDetectionMode)
СканированиеСериями (ScanSeries)
ФильтрыОбработки (ProcessingFilters)

Конструкторы:

По умолчанию

Описание:

Определяет набор параметров, используемых в методах ПоказатьСканированиеДокументов и СканироватьДокументыАсинх.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/data/ext. Имя типа XDTO: DocumentScanningParameters.

Использование в версии:

Доступен, начиная с версии 8.3.23.
