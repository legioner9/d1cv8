МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьПараметрыПривязкиККомпьютеру (GetComputerFingerprintParameters)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьПараметрыПривязкиККомпьютеру(<АдресАгента>, <Порт>)

Параметры:

<АдресАгента> (необязательный)

Тип: Строка.
Адрес агента сервера 1С:Предприятия.
Если пустая строка, то будут получены данные текущего процесса. В этом случае значение параметра <Порт> игнорируется.
Значение по умолчанию: Пустая строка.

<Порт> (необязательный)

Тип: Число.
IP-порт агента сервера 1С:Предприятия.
Если 0, то используется значение "1540".
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: ПараметрыПривязкиККомпьютеруПолученияЛицензий.

Описание:

Возвращает параметры компьютера, участвующие в привязке лицензии, и составные части представления параметров текущего компьютера или компьютера с указанным адресом агента сервера 1С:Предприятия.
В случае ошибки обращения к агенту сервера генерируется исключение.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
