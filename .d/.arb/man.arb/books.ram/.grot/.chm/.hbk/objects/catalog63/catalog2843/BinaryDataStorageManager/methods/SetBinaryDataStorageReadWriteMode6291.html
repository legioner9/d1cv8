МенеджерХранилищаДвоичныхДанных.УстановитьРежимЧтенияЗаписиХранилищаДвоичныхДанных (BinaryDataStorageManager.SetBinaryDataStorageReadWriteMode)

МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьРежимЧтенияЗаписиХранилищаДвоичныхДанных (SetBinaryDataStorageReadWriteMode)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьРежимЧтенияЗаписиХранилищаДвоичныхДанных(<РежимЧтенияЗаписи>)

Параметры:

<РежимЧтенияЗаписи> (обязательный)

Тип: РежимЧтенияЗаписиХранилищаДвоичныхДанных.
Режим работы хранилища двоичных данных, который требуется установить.

Описание:

Устанавливает режим хранилища двоичных данных (только для чтения, чтение и запись).

Для изменения режима чтения и записи хранилища двоичных данных требуется монопольный доступ к информационной базе (устанавливается с помощью метода УстановитьМонопольныйРежим), иначе будет сгенерировано исключение.
Метод доступен только из сеанса, в котором не используется ни один разделитель.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
