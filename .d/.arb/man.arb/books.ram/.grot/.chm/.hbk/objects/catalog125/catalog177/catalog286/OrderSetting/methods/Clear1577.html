НастройкаПорядка.Очистить (OrderSetting.Clear)

НастройкаПорядка (OrderSetting)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Очищает коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не доступен для настройки отбора динамических списков (свойство НастройкаПорядка объектов СправочникСписок.<Имя справочника>, ДокументСписок.<Имя документа> и т.д.).

Использование в версии:

Доступен, начиная с версии 8.0.
