ТипКолонкиАнализаДанныхДеревоРешений (DataAnalysisColumnTypeDecisionTree)

ТипКолонкиАнализаДанныхДеревоРешений (DataAnalysisColumnTypeDecisionTree)

Доступен, начиная с версии 8.0.

Значения

Входная (Input)
НеИспользуемая (NotUsed)
Прогнозируемая (Predictable)

Описание:

Содержит варианты типов колонок анализа данных - дерево решений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: DataAnalysisColumnTypeDecisionTree.

Использование в версии:

Доступен, начиная с версии 8.0.
