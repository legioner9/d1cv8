Описание:

Значения параметров вывода для группировок.
Содержит значения следующих параметров:

  • АвтоПозицияРесурсов (ResourcesAutoPosition) - автопозиция ресурсов (тип АвтоПозицияРесурсовКомпоновкиДанных);
  • ВариантИспользованияГруппировки (GroupUseVariant) - вариант использования группировки (тип ВариантИспользованияГруппировкиКомпоновкиДанных).
  • ВертикальноеРасположениеОбщихИтогов (VerticalOverallPlacement) - вертикальное расположение общих итогов (тип РасположениеИтоговКомпоновкиДанных);
  • ВыводитьЗаголовок (TitleOutput) - выводить заголовок (тип ТипВыводаТекстаКомпоновкиДанных);
  • ВыводитьОтбор (FilterOutput) - выводить отбор (тип ТипВыводаТекстаКомпоновкиДанных);
  • ГоризонтальноеРасположениеОбщихИтогов (HorizontalOverallPlacement) - горизонтальное расположение общих итогов (тип РасположениеИтоговКомпоновкиДанных);
  • Заголовок (Title) - заголовок (тип Строка);
  • КоличествоЗаписей (RecordsCount) - количество записей, которое необходимо выводить в группировке (тип Число);
  • МакетОформления (AppearanceTemplate) - имя макета оформления (тип Строка);
  • ПроцентЗаписей (RecordsPercent) - процент от общего количества записей, выводимый в группировке (тип Число);
  • РасположениеГруппировки (GroupPlacement) - расположение группировки (тип РасположениеГруппировкиКомпоновкиДанных);
  • РасположениеИтогов (TotalsPlacement) - расположение итогов (тип РасположениеИтоговКомпоновкиДанных);
  • РасположениеПолейГруппировки (GroupFieldsPlacement) - расположение полей группировки (тип РасположениеПолейГруппировкиКомпоновкиДанных);
  • РасположениеРеквизитов (AttributePlacement) - расположение реквизитов (тип РасположениеРеквизитовКомпоновкиДанных);
  • РасположениеРесурсов (ResourcePlacement) - расположение ресурсов (тип РасположениеРесурсовКомпоновкиДанных);
  • ТипЗаголовкаПолей (FieldsTitleType) - тип заголовка полей (тип ТипЗаголовкаПолейКомпоновкиДанных);
  • ТипДиаграммы (ChartType) - тип диаграммы (тип ТипДиаграммы);
  • ТипДиаграммы.АвтоРаздвижениеСерий (ChartType.AutoSeriesSeparation) - режим раздвижения (тип АвтоРаздвижениеСерий);
  • ТипДиаграммы.БазовоеЗначение (ChartType.BaseValue) – базовое значение (тип Число);
  • ТипДиаграммы.ВидПодписей (ChartType.LabelType) - состав подписей (тип ВидПодписейКДиаграмме);
  • ТипДиаграммы.ВлияниеРазмераНаПузырек (ChartType.ChartBubbleSizing) - определяет зависимость размера пузырька от значения (тип ВлияниеРазмераНаПузырекДиаграммы);
  • ТипДиаграммы.ВнутреннийРадиусКольцевойДиаграммы (ChartType.DonutChartInnerRadius) - настройка размера внутреннего радиуса кольцевой диаграммы (тип Число).
  • ТипДиаграммы.ГлубинаОбъемнойДиаграммы (ChartType.Chart3DDepth) - глубина объемной диаграммы. Используется в круговой, кольцевой и объемных гистограммах (тип Число);
  • ТипДиаграммы.Градиент (ChartType.Gradient) – градиент (тип Булево);
  • ТипДиаграммы.ДополнительнаяОсьЗначений (ChartType.AdditionalValuesAxis) - настройка дополнительной оси значений (тип ОсьДиаграммы);
  • ТипДиаграммы.ДополнительнаяШкалаЗначений (ChartType.AdditionalValuesScale) - настройка дополнительной шкалы значений (тип ШкалаДиаграммы);
  • ТипДиаграммы.ЗаполнениеПодсказкиЗначений (ChartType.ValuesToolTipFillType) - настройка режима заполнения всплывающей подсказки диаграммы (тип ЗаполнениеПодсказкиЗначенийДиаграммы);
  • ТипДиаграммы.ИнформационныеИнтервалыЗначений (ChartType.ValuesReferenceBands) - информационные интервалы значений в диаграмме (тип ИнформационныеИнтервалыДиаграммы);
  • ТипДиаграммы.ИнформационныеИнтервалыТочек (ChartType.PointsReferenceBands) - информационные интервалы точек в диаграмме (тип ИнформационныеИнтервалыДиаграммы);
  • ТипДиаграммы.ИнформационныеЛинииЗначений (ChartType.ValuesReferenceLines) - информационные линии значений в диаграмме (тип ИнформационныеЛинииДиаграммы);
  • ТипДиаграммы.ИнформационныеЛинииТочек (ChartType.PointsReferenceLines) - информационные линии точек в диаграмме (тип ИнформационныеЛинииДиаграммы);
  • ТипДиаграммы.ИспользованиеНечисловыхЗначений (ChartType.NonnumericValuesUse) - настройка использования значений, которые не приводятся к числу (тип ИспользованиеНечисловыхЗначенийДиаграммы);
  • ТипДиаграммы.ИсточникЗначенийОсиТочек (ChartType.ChartPointsAxisValuesSource) - определяет вид значений на оси точек (тип ИсточникЗначенийОсиТочекДиаграммы);
  • ТипДиаграммы.ИсточникЗначенияРазмераПузырька (ChartType.BubbleSizeValueSource) - источник значения размера пузырьков (тип ИсточникЗначенияРазмераПузырькаДиаграммы);
  • ТипДиаграммы.КлючСохраненияЦветов (ChartType.SaveColorsKey) - ключ, по которому в системном хранилище настроек лежит СохранениеОформленияКомпоновкиДанных. Если по ключу ничего нет, то создает новую запись. Если по ключу лежит объект другого типа, то параметр ТипДиаграммы.КлючСохраненияЦветов игнорируется (тип Строка).
  • ТипДиаграммы.КонечныйУгол (ChartType.EndAngle) - угол конца отображения секторов. Используется в круговой и кольцевой диагарммах (тип Число);
  • ТипДиаграммы.ЛинииСоединенияЗначенийПоСериям (ChartType.ValuesBySeriesConnectionLines) - стиль и толщина линий соединений (тип Линия);
  • ТипДиаграммы.ЛинииШкал (ChartType.ScaleLines) - стиль линий шкалы (тип Линия);
  • ТипДиаграммы.МаксимальноеЗначение (ChartType.MaxValue) - максимальное значение (тип Число);
  • ТипДиаграммы.МаксимальныйРазмерПузырька (ChartType.MaxBubbleSize) - максимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения) если для размера пузырьков указана серия значений (тип Число);
  • ТипДиаграммы.МинимальноеЗначение (ChartType.MinValue) - минимальное значение (тип Число);
  • ТипДиаграммы.МинимальныйРазмерПузырька (ChartType.MinBubbleSize) - минимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения), если для размера пузырьков указана серия значений (тип Число);
  • ТипДиаграммы.НатяжениеСглаживания (ChartType.SplineStrain) - целочисленное значение определяет "натяжение" кривой для режима сглаживания ГладкаяКривая;
  • ТипДиаграммы.НачальныйУгол (ChartType.BeginAngle) - угол начала отображения секторов по часовой стрелке от вертикальной оси. Используется в круговой и кольцевой диагарммах (тип Число);
  • ТипДиаграммы.ОбщаяСерияРазмераПузырька (ChartType.BubbleSizeCommonSeries) - серия, значения которой будут использоваться для размеров пузырьков для всех серий по оси ординат (тип Произвольн);
  • ТипДиаграммы.Окантовка (ChartType.Outline) – окантовка (тип Булево);
  • ТипДиаграммы.ОписаниеПалитрыЦветов (ChartType.ColorPaletteDescription) - настройка палитры цветов серий в диаграмме (тип ОписаниеПалитрыЦветовДиаграммы);
  • ТипДиаграммы.ОписаниеПалитрыЦветовИнформационныхИнтервалов (ChartType.ReferenceBandsColorPaletteDescription) - настройка палитры цветов для информационных интервалов (тип ОписаниеПалитрыЦветовДиаграммы)
  • ТипДиаграммы.ОсьЗначений (ChartType.ValuesAxis) - настройки оси значений в диаграмме (тип ОсьДиаграммы);
  • ТипДиаграммы.ОсьТочек (ChartType.PointsAxis) - настройки оси точек в диаграмме (тип ОсьДиаграммы);
  • ТипДиаграммы.ОтображатьТаблицуДанных (ChartType.ShowDataTable) - отображать таблицу данных (тип Булево);
  • ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииЗначений (ChartType.ShowValuesPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале значений при показе подсказки (тип ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы);
  • ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииТочек (ChartType.ShowPointsPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале точек при показе подсказки диаграммы (тип ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы);
  • ТипДиаграммы.ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы (ChartType.BubbleChartNegativeValuesShowMode) - указывает вариант работы с отрицательными значениями размера (тип ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы);
  • ТипДиаграммы.ПорядокОтображенияТочекГоризонтальнойГистограммы (ChartType.BarChartPointsOrder) - порядок вывода точек в горизонтальных диаграммах (тип ПорядокОтображенияТочекГоризонтальнойГистограммы);
  • ТипДиаграммы.ПорядокСерийВЛегенде (ChartType.SeriesOrderInLegend) - определяет порядок отображения серий в легенде диаграммы (тип ПорядокСерийВЛегендеДиаграммы);
  • ТипДиаграммы.ПропускатьБазовоеЗначение (ChartType.HideBaseValue) - пропускать точки с базовым значением (тип Булево);
  • ТипДиаграммы.РазмерПузырькаПоУмолчанию (ChartType.DefaultBubbleSize) - указывает базовый диаметр пузырька в процентах от минимума из ширины и высоты области построения (тип Число);
  • ТипДиаграммы.РасположениеЛегенды (ChartType.LegendPlacement) - расположение легенды в диаграммы (тип РасположениеЛегендыДиаграммыКомпоновкиДанных);

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/settings}. Имя типа XDTO: GroupOutputParameterValues.
