Файл (File)

УстановитьНевидимостьАсинх (SetHiddenAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьНевидимостьАсинх(<Невидимость>)

Параметры:

<Невидимость> (обязательный)

Тип: Булево.
Истина - установить атрибут "Скрытый", Ложь - снять атрибут "Скрытый".

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Устанавливает значение атрибута файла "Скрытый". Применима только к файлам или каталогам, которые реально существуют. В ОС Linux метод не работает.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:
