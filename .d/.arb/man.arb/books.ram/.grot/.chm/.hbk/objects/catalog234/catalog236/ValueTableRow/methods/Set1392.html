СтрокаТаблицыЗначений (ValueTableRow)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс колонки, в которую будет установлено значение.

<Значение> (обязательный)

Тип: Произвольный.
Устанавливаемое значение.

Описание:

Устанавливает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
