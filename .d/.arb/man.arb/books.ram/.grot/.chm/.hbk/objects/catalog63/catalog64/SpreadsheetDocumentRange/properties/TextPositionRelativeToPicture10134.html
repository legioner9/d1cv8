ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ПоложениеТекстаОтносительноКартинки (TextPositionRelativeToPicture)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: ПоложениеТекстаОтносительноКартинки.
Определяет расположение текста в ячейке относительно картинки. Если свойство РазмерКартинки содержит значение Черепица или Растянуть, то игнорируется и считается равным Поверх.
Значение по умолчанию – Авто.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
