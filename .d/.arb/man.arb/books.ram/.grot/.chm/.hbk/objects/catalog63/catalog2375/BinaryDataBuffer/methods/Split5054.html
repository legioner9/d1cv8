БуферДвоичныхДанных.Разделить (BinaryDataBuffer.Split)

БуферДвоичныхДанных (BinaryDataBuffer)

Разделить (Split)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: По двоичному буферу

Синтаксис:

Разделить(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: БуферДвоичныхДанных.
Разделитель.

Описание варианта метода:

Разделяет буфер на части, используя двоичный буфер-разделитель.

Вариант синтаксиса: По массиву разделителей

Синтаксис:

Разделить()

Описание варианта метода:

Разделить буфер на части по заданным разделителям. Элементы массива имеют тип БуферДвоичныхДанных.

Вариант синтаксиса: По массиву разделителей

Синтаксис:

Разделить(<Разделители>)

Параметры:

<Разделители> (обязательный)

Тип: Массив.
Массив разделителей. Может содержать значения типа БуферДвоичныхДанных.

Описание варианта метода:

Разделить буфер на части по заданным разделителям.
Элементы массива должны иметь тип БуферДвоичныхДанных.

Возвращаемое значение:

Тип: Массив.
Тип элемента массива - БуферДвоичныхДанных.

Описание:

Разделить буфер на части по заданному разделителю.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
