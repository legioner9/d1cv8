ЧтениеДанных.РазделитьНаЧастиПоАсинх (DataReader.SplitInPartsOfAsync)

ЧтениеДанных (DataReader)

РазделитьНаЧастиПоАсинх (SplitInPartsOfAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазделитьНаЧастиПоАсинх(<РазмерЧасти>)

Параметры:

<РазмерЧасти> (обязательный)

Тип: Число.
Размер части данных в байтах.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - массив значений типа РезультатЧтенияДанных.

Описание:

Асинхронная функция. Делит остаток данных на части заданного размера.
Размер части задается в байтах.

Доступность:

Тонкий клиент, толстый клиент.

ЧтениеДанных, метод РазделитьНаЧастиПо

Доступен, начиная с версии 8.3.18.
