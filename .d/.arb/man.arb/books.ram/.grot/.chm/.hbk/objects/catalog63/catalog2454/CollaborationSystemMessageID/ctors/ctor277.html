ИдентификаторСообщенияСистемыВзаимодействия (CollaborationSystemMessageID)

По умолчанию

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый ИдентификаторСообщенияСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Строковое представление идентификатора.

Описание:

Создает идентификатор сообщения из указанной строки.
Данный конструктор можно использовать, если нужно произвести действие над сообщением, идентификатор которого хранится в информационной базе в виде строки.
Корректность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.11.
