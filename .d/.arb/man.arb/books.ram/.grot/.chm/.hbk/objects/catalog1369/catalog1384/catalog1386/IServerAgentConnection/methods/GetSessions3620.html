Соединение с агентом сервера (IServerAgentConnection)

GetSessions (GetSessions)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessions(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: Кластер серверов.
Кластер, для которого необходимо получить список сеансов. Должен быть получен методом GetClusters.

Возвращаемое значение:

Тип: COMSafeArray.
Массив, который содержит список объектов Сеанс.

Описание:

Получает список сеансов, работающих с данным кластером.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
