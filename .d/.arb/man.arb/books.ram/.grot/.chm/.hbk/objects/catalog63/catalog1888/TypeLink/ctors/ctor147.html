СвязьПоТипу (TypeLink)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СвязьПоТипу(<ПутьКДанным>, <ЭлементСвязи>)

Параметры:

<ПутьКДанным> (необязательный)

Тип: Строка.
Путь к полю формы клиентского приложения, или объекту метаданных, или свойству объекта метаданных, откуда будут браться данные для отбора.
Значение по умолчанию: Пустая строка.

<ЭлементСвязи> (необязательный)

Тип: Число.
Содержит номер вида субконто для случая, если реквизит, с которым выбрана связь по типу, имеет значение типа Ссылка на план счетов.
Значение по умолчанию: 0.

Описание:

Создает объект и заполняет его свойства значениями из параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
