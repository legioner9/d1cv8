ЭлементГрафическойСхемыДействие (GraphicalSchemaItemActivity)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.
Точка маршрута бизнес-процесса, которой соответствует данный элемент графической схемы.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Имеет смысл только для элемента графической схемы, полученной методами ПолучитьКартуМаршрута и ПолучитьКартуМаршрута.

Использование в версии:

Доступен, начиная с версии 8.0.
