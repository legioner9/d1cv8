ОформлениеЗначений.Очистить (ValueAppearance.Clear)

ОформлениеЗначений (ValueAppearance)

Очистить (Clear)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Очистить(<Тип>, <НеУдалятьЗаданныеНепосредственно>)

Параметры:

<Тип> (необязательный)

Тип: Тип.
Тип очищаемых значений.

<НеУдалятьЗаданныеНепосредственно> (необязательный)

Тип: Булево.
Указывает необходимость оставлять нечисловые значения.
Значение по умолчанию: Ложь.

Описание:

Если тип указан, очищает ключи указанного типа.
Если тип не указан, очищает ключи для всех типов.
Если НеУдалятьЗаданныеНепосредственно имеет значение Истина, удаляет только числовые значения.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
