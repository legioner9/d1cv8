КонструкторЗапроса (QueryWizard)

Показать (Show)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: ОписаниеОповещения.
Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет. Если параметр указан, то содержит описание процедуры, которая будет вызвана после закрытия диалога конструктора со следующими параметрами:
  • <Текст> - Текст, если конструктор закрыли по кнопке "OK", Неопределено - в противном случае;
  • <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Открывает немодально конструктор запроса.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

КонструкторЗапроса, метод ОткрытьАсинх

Использование в версии:
