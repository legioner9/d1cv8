WSКоллекцияСервисов.Получить (WSServiceCollection.Get)

WSКоллекцияСервисов (WSServiceCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс коллекции.

Вариант синтаксиса: По имени

Синтаксис:

Получить(<ПространствоИмен>, <ЛокальноеИмя>)

Параметры:

<ПространствоИмен> (обязательный)

Тип: Строка.
Пространство имен сервиса.

<ЛокальноеИмя> (обязательный)

Тип: Строка.
Локальное имя сервиса.

Возвращаемое значение:

Тип: WSСервис.

Описание:

Получает сервис.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
