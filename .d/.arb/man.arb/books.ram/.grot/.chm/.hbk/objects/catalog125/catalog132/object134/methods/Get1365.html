ПеречислениеМенеджер.<Имя перечисления> (EnumManager.<Enumeration name>)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.
Индекс значения перечисления в перечислении.

Возвращаемое значение:

Тип: ПеречислениеСсылка.<Имя перечисления>.

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
