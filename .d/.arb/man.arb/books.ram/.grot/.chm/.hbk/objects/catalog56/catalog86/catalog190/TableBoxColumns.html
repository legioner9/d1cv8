КолонкиТабличногоПоля (TableBoxColumns)

Доступен, начиная с версии 8.0.

Элементы коллекции:

КолонкаТабличногоПоля
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс колонки (нумерация с 0).

Свойства:

<Имя колонки> (<Column name>)

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Представляет собой коллекцию колонок табличного поля.

Доступность:

Толстый клиент.

См. также:

ТабличноеПоле, свойство Колонки

Использование в версии:

Доступен, начиная с версии 8.0.
