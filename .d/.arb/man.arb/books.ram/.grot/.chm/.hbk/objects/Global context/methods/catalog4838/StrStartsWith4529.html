Глобальный контекст.СтрНачинаетсяС (Global context.StrStartsWith)

Глобальный контекст (Global context)

СтрНачинаетсяС (StrStartsWith)

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Синтаксис:

СтрНачинаетсяС(<Строка>, <СтрокаПоиска>)

Параметры:

<Строка> (обязательный)

Тип: Строка.
Строка, начало которой проверяется на совпадение с подстрокой поиска.

<СтрокаПоиска> (обязательный)

Тип: Строка.
Строка, содержащая предполагаемое начало строки.
В случае если переданное значение является пустой строкой генерируется исключительная ситуация.

Возвращаемое значение:

Тип: Булево.
Истина - строка начинается с указанной подстроки, Ложь - в противном случае.

Описание:

Определяет, что строка начинается с указанной подстроки. Определение выполняется с учетом регистра, аналогично функции Найти.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

Глобальный контекст, метод СтрЗаканчиваетсяНа

Использование в версии:

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Описание изменено в версии 8.3.15.
