ФорматированныйДокумент (FormattedDocument)

ПолучитьТекст (GetText)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьТекст(<Начало>, <Конец>)

Параметры:

<Начало> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на начало текстового фрагмента. Если закладка не задана, то текст будет выбран с самого начала документа.

<Конец> (необязательный)

Тип: ЗакладкаФорматированногоДокумента.
Закладка, указывающая на конец текстового фрагмента. Если закладка не задана, то текст будет выбран до самого конца документа.

Возвращаемое значение:

Тип: Строка.

Описание:

Получает фрагмент текста, заданный закладками.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
