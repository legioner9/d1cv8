Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ВвестиТекст (InputText)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВвестиТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.
Вводимый текст.

Описание:

Осуществляет ввод текста в поле ввода. Переданное значение целиком замещает весь текст, ранее содержащийся в поле ввода.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
