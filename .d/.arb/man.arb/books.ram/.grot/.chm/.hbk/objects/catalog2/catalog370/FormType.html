ТипФормы (FormType)

Доступен, начиная с версии 8.2.

Значения

Обычная (Ordinary)
Управляемая (Managed)

Описание:

Тип формы (обычная или управляемая). Задается при создании формы и впоследствии не может быть изменен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
