ВыделенныеОбластиТабличногоДокумента (SpreadsheetDocumentSelectedAreas)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Область>, <Позиция>)

Параметры:

<Область> (обязательный)

Тип: ОбластьЯчеекТабличногоДокумента, РисунокТабличногоДокумента.
Добавляемая область к коллекции выделенных областей.

<Позиция> (обязательный)

Тип: Число.
Позиция, в которую будет добавлена область.

Описание:

Вставляет область в указанную позицию коллекции выделенных областей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
