ЭлементВводаHTML.Форма (HTMLInputElement.Form)

ЭлементВводаHTML (HTMLInputElement)

Форма (Form)

Доступен, начиная с версии 8.2.

Синтаксис:

Форма()

Возвращаемое значение:

Тип: ЭлементФормаHTML.

Описание:

Возвращает форму, которой принадлежит данный элемент ввода.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
