МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографииАсинх (CryptoManager.GetCryptoSignaturesContainerAsync)

МенеджерКриптографии (CryptoManager)

ПолучитьКонтейнерПодписейКриптографииАсинх (GetCryptoSignaturesContainerAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьКонтейнерПодписейКриптографииАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, ДвоичныеДанные, Поток, ПотокВПамяти, ФайловыйПоток.
Подпись для разбора.
Если параметр типа Строка, то обозначает имя файла с подписью.
Если параметр типа ДвоичныеДанные или Поток, ПотокВПамяти, ФайловыйПоток, то он задает данные подписи.

Возвращаемое значение:

Тип: Обещание.
Результат выполнения обещания - объект типа КонтейнерПодписейКриптографии.

Описание:

Выполняет разбор двоичных данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
