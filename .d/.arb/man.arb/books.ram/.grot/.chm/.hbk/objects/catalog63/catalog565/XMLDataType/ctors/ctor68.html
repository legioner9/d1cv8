ТипДанныхXML (XMLDataType)

По имени типа XML и URI пространства имен

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ТипДанныхXML(<ИмяТипа>, <URIПространстваИмен>)

Параметры:

<ИмяТипа> (обязательный)

Тип: Строка.
Имя типа XML.

<URIПространстваИмен> (обязательный)

Тип: Строка.
URI пространства имен, к которому относится тип. Если тип не относится ни к какому пространству имен, то должна быть указана пустая строка.

Описание:

Создает объект ТипДанныхXML по имени типа XML и URI пространства имен, к которому относится тип.

Использование в версии:

Доступен, начиная с версии 8.0.
