ОтборКомпоновкиДанных (DataCompositionFilter)

ИдентификаторПользовательскойНастройки (UserSettingID)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.
Предназначен для идентификации пользовательской настройки.
Если идентификатор не пустой, объект включается в состав пользовательских настроек.
Начальный состав пользовательской настройки отбора определяется составом элементов данного отбора. В него попадут все элементы отбора, за исключением:
элементов и групп, которые вынесены в пользовательскую настройку (определяется по ИдентификаторПользовательскойНастройки);
элементов и групп, у которых РежимОтображения=Обычный.
Если элемент или группа вложен в данный отбор не напрямую, а в составе группы, то его РежимОтображения игнорируется, а его использование в пользовательской настройке определяется режимом отображения группы.
При применении пользовательской настройки состав элементов данного отбора будет изменен: элементы, которые не вошли в пользовательские настройки, не будут изменены; остальные элементы будут заменены на состав пользовательского отбора.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
