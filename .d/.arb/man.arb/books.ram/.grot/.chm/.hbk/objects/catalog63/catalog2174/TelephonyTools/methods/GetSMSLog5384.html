СредстваТелефонии.ПолучитьЖурналSMS (TelephonyTools.GetSMSLog)

СредстваТелефонии (TelephonyTools)

ПолучитьЖурналSMS (GetSMSLog)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПолучитьЖурналSMS()

Возвращаемое значение:

Тип: ЖурналSMS, Неопределено.
Если журнал сообщений не доступен, возвращает Неопределено.

Описание:

Получает журнал SMS-сообщений.
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод ПоддерживаетсяЖурналSMS.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

СредстваТелефонии, метод ПоддерживаетсяЖурналSMS

Использование в версии:

Доступен, начиная с версии 8.3.11.
