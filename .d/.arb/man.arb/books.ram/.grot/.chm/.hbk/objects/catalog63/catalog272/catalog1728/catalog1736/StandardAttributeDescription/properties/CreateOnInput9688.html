СозданиеПриВводе (CreateOnInput)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: СозданиеПриВводе.
Определяет режим создания при вводе. Если значение равно Авто, то участие в механизме создания при вводе определяется значением свойства СозданиеПриВводе объекта метаданных этого типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
