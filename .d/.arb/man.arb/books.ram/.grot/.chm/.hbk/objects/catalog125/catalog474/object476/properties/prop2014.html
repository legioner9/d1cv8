<Имя предопределенного объекта> (<The name of a predefined object.>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланВидовРасчетаСсылка.<Имя плана видов расчета>.
Содержит ссылки на предопределенный объект плана видов расчета. Доступ к значению осуществляется по имени, как они заданы в Конфигураторе. Имена имеют только предопределенные виды расчета. Количество таких свойств задается списком предопределенных объектов плана видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
