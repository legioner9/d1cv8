СоставКопииБазыДанных (DatabaseCopyContent)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Элементы коллекции:

ЭлементСоставаКопииБазыДанных
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ЭлементСоставаКопииБазыДанных.
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента.

Методы:

Вставить (Insert)
Добавить (Add)
Индекс (IndexOf)
Количество (Count)
Найти (Find)
Очистить (Clear)
Получить (Get)
Сдвинуть (Move)
Удалить (Delete)

Описание:

Представляет собой коллекцию объектов копии базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

МенеджерКопииБазыДанных, свойство Состав

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
