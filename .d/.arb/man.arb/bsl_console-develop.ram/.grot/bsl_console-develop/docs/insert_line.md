# Функция *insertLine*
## Назначение функции
Вставляет текст в строку с указанным номером

## Параметры функции
* **lineNumber** - *integer*, номер строки, в которую необходимо вставить текст
* **text** - *string*, текст для вставки

## Пример вызова
```javascript
insertLine(5, "Таблица = Новый ТаблицаЗначений();");
```
